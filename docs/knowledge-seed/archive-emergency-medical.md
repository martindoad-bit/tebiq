# 我入院 / 遇到灾害，档案能拿出什么

## 这事跟我有关吗？
你或家人突然入院、需要紧急就诊、或所在地区受 地震 / 台風 / 水害 影响被指定为 災害救助法 適用区域。紧急情况下，市役所、医院、避难所、保险公司窗口都会同时要求出示「你现在的身份」。档案能在几分钟内调出当前在留資格、健康保険 类型、紧急联系人，比翻钱包翻邮箱快得多。

## 这是什么
紧急 / 医疗 / 灾害场景下，档案能即时调出的核心数据：

**入院 / 急诊**
- 当前 健康保険 类型：社保（全国健康保険協会 / 組合健保）/ 国民健康保険 / 後期高齢者医療制度
- 健康保険証 记号番号（マイナンバーカード 健康保険証 利用 也支持）
- 在留カード 信息（在留資格 / 在留期限 / 住所）
- 高額療養費 限度額認定証 申请状态：未申请 / 已申请 / 認定証号码
- 出産育児一時金 / 傷病手当金 申请历史（健保通过雇主或保険者）
- 直近健康診断 结果（如档案保存过）
- 紧急联系人姓名 + 关系 + 电话（用户自填）

**灾害**
- 被災証明書 申请状态：档案标记「YYYY-MM-DD 申请，自治体 = ○○市」
- 罹災証明書（与 被災証明書 区别在于损坏程度评定，房屋有实质损坏时申请）
- 災害救助法 適用区域 在留資格 期限延长措施记录（重大灾害时入管会发布临时延长通知）
- 火災保険 / 地震保険 申请凭证
- 当前住居地址、避難先 临时住所

档案不替你向医院办理手续，也不替你向自治体提交 被災証明書 申请，但能在 30 秒内打开「当前身份快照」页，把上述全部数据列成可直接出示的一屏。

## 你应该做什么
1. 入院时携带：健康保険証（或 マイナンバーカード）+ 在留カード + 印鑑 + 现金 1 万円程度。挂号时出示档案中「当前身份快照」截图，加快办理。
2. 入院 1 周内或自费金额预估超过 自己負担限度額（70 岁未满一般所得者约 8 万円）时，立即向健保 申请 限度額認定証。档案保存申请受理日 + 認定証号码后，医院会按限度額直接结算，无需事后申请高額療養費 还付。
3. 灾害发生后：先在自治体公式网站确认是否被指定为 災害救助法 適用区域；适用则前往 市役所 申请 罹災証明書 / 被災証明書。档案标记申请日，避免重复申请。
4. 入管在重大灾害后会公布在留期限自动延长 / 申请延期、再入国期限延期等措施，档案会同步入管公告，在「待办」区列出。

## 不做的后果
紧急情况下找不到健康保険証，医院需先按 全額自費（10 割）结算，事后凭 領収書 + 健保証 在 2 年内申请 還付。社保 / 国保 都有 2 年时効，过期不还付。

高額療養費 限度額認定証 不事先申请，医院会按全额收费，自费金额可能瞬间过 30 万円。事后申请高額療養費 还付要 3 个月以上，资金流压力直接冲个人。

灾害后不申请 罹災証明書 / 被災証明書，无法享受 災害減免法（所得税減免）、住民税 減免、国保保険料 減免、災害救助金 等措施，永住 / 续签材料里也无法说明本年度收入异常的原因。

## 什么时候你应该找专家
医院窗口、市役所 防災課、保険会社 客服 处理日常紧急事务足够。以下场景建议咨询行政書士或相关专业人士：（1）大额医疗费 + 跨境保险理赔；（2）灾害后 在留資格 期限到期、入管未公告自动延长，需个案申请延期；（3）家人在国外，需紧急办理短期来日 看病 / 探病。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
