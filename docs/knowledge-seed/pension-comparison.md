## 这事跟我有关吗？

你是 20 岁以上 60 岁未満、住所在日本的人——无论国籍、无论 在留資格——都**强制加入**公的年金，没法自由选择。如果你看工资条发现「厚生年金保険料」每月扣一笔不小的钱想知道这钱去哪了；或者你是 自営業 / 留学生 / 待业，每月收到 国民年金 納付書，最近经济困难想申请减免；或者你打算申请永住 / 帰化、担心年金记录会不会扯后腿——这篇适合你。

## 这是什么

日本公的年金采取「**2 阶建て**」（两层结构）：

- **第 1 阶 国民年金（基礎年金）**：覆盖所有居住在日本、20 岁以上 60 岁未満的人。
- **第 2 阶 厚生年金**：在国民年金之上，针对会社员、公務員 等受雇者追加加入。

会社员同时加入两层，将来同时领取**老齢基礎年金** + **老齢厚生年金**；自営業者只加入第 1 阶，将来只领取 老齢基礎年金。除老齢給付外，还提供**障害年金**（因病或事故导致一定程度障害时支付）和**遺族年金**（被保険者死亡时支付给遺族）。运营主体为**日本年金機構**，窗口为各地的**年金事務所**。

国民年金将被保険者分三类：

- **第 1 号被保険者**：自営業者、自由职业者、学生、無業者 等。**国民年金**保険料 2025 年度月額定額约 **17,510 日元**（每年度调整），需自行缴纳。
- **第 2 号被保険者**：加入 厚生年金 的会社员、公務員。**保険料**按**標準報酬月額** + 賞与，由本人与雇主对半负担、公司从工资代扣。已包含国民年金部分，**不需另缴**。
- **第 3 号被保険者**：第 2 号被保険者 的被扶養配偶（年収原则 130 万日元以下等条件）。原则上不需本人缴纳保険料，但需通过配偶的勤務先 办「第 3 号被保険者関係届」。

**对在留 / 永住 / 帰化 审查的影响**：出入国在留管理庁 与 法務省 将「公的義務的履行」作为重要要件，最近**2 年程度的年金缴纳记录**（含保険料缴纳・免除・猶予 的处理状况）是核心审查材料。如有未加入期間或长期滞納，即便其他要件满足，也可能被判定「素行要件」「独立生計要件」未达标。

**对所得税 / 住民税 的减税效果**：已缴纳的 国民年金・厚生年金 保険料 在 確定申告 中可作为**社会保険料控除**全额扣除。

## 你应该做什么

1. **入境后办住民登録**：市役所信息会转送日本年金機構，第 1 号被保険者 会收到加入通知与缴纳书。
2. **入职公司**：人事会办「健康保険・厚生年金保険被保険者資格取得届」，**不需要自己跑年金事務所**。
3. **离职 / 转 自営業**：失去 厚生年金 资格后 14 天内，到市役所办**国民年金第 1 号資格取得届**，转回第 1 号。
4. **配偶就业 / 失业**：通过雇主办「第 3 号被保険者関係届」（成为 / 退出第 3 号）。
5. **第 1 号缴纳方式**：口座振替、クレジットカード、Pay-easy、コンビニ。**前納**（半年・1 年・2 年一次性预缴）有折扣。
6. **经济困难（第 1 号）**：尽早申请**保険料免除制度**（全額・4 分の 3・半額・4 分の 1 免除）、**納付猶予制度**（50 岁未満低收入者）、**学生納付特例**。**已被认可的免除・猶予期間在审查上被作为「已经适当处理」对待，与单纯滞納有本质区别**。
7. **每年定期检查**：日本年金機構 寄送的**ねんきん定期便**会记载至今的加入履歴和将来给付見込額。建议结合**マイナンバー**绑定的**ねんきんネット**定期确认記録是否正确。
8. **离开日本前**：考虑「**脱退一時金**」请求（厚生年金 / 国民年金 缴纳 6 个月以上、2 年内可请求）。

## 不做的后果

滞納会先后经历：

- **督促状**寄出
- **延滞金**附加
- **最終催告状**
- **督促状（強制徴収予告）**
- **強制徴収**（給与差押、預金差押、財産差押）

一旦进入 強制徴収 阶段：

- 记录长期保留，对永住・帰化 审查产生明显不利影响
- 将来可领取的 老齢年金額 减少
- 可能丧失 障害年金・遺族年金 的请求权（突发重病 / 事故 / 死亡时家属拿不到钱）

## 什么时候你应该找专家

**简单情形**（自己处理即可）：单纯加入 / 退出、按时缴纳、经济困难时申请免除 / 猶予 / 学生特例、在线确认 ねんきんネット 记录。

**复杂情形**（建议咨询）：

- 加入分类判定（多重身份、海外公司雇佣、家族经营事业所）
- 长期未加入 / 滞納 即将申请永住・帰化，需要评估对审查的实际影响
- ねんきん定期便 上记录有遗漏 / 错误，需要追溯订正
- 跨国就业 / 社会保障協定 国适用证明（避免双重缴纳）
- 离开日本前的 脱退一時金 请求 + 受領后的所得税源泉徴収退还
- 障害 / 遺族年金 请求
