# 收入水平 / 独立生计

## 这是什么
入管法 在多种 在留資格 中要求「日本人と同等以上の報酬」或「独立生計」要件。具体数额因 在留資格 / 地域 / 業種 而异，没有全国统一线，但有事实上的下限标准。特定技能 在「日本人と同等以上の報酬」+「受入機関 雇用契約 規定」双重约束下被审查 —— 同職場 同業務 日本人従業員 的报酬作为对照基準，且 雇用契約 中的 報酬額 不得低于行業 ガイドライン 規定値。特定技能 2 号 被允许携带家族 → 独立生計 范围扩大到家族单位。

## 为什么重要
- 在留期間更新：受入機関 給与 低于行業基準 → 续签可能短期化或不许可，受入機関 也面临处分风险。
- 転職 case：跨 受入機関 转职时新 雇用契約 報酬 须达标，且行業 切换需重新通过 技能評価試験。
- 特定技能 2 号：可携家族 + 永住路径开放 → 報酬水位直接影响后续 永住申請 的「独立生計」要件。

## 判断标准
- **绿**：q1 ≥ 中段 + q2 == yes + q3 != no（一致或更高 + 覆盖家族）
- **黄**：q1 偏低 OR q2/q3 unknown
- **红**：q1 = under_300 OR q2 == no OR q3 == no

## 处理建议
### 绿
档案归档直近 3 年 源泉徴収票 / 給与明細 / 課税証明書 / 雇用契約書。续签前 90 天复检。

### 黄
1. 对照档案 直近 3 年 源泉徴収票 + 課税証明書 与申请时声明。
2. 給与 在行業 ガイドライン 下限附近 → 雇用契約書 + 同職場 日本人従業員 報酬規定 比对说明。
3. 残業時間 大幅变动导致 年収 波动 → 申請理由書 简单说明即可。

### 红
1. 受入機関 給与 不足行業基準 → 立即与 受入機関 / 登録支援機関 確認 雇用契約書 + 給与改定 + 申請理由書 经過说明。
2. 特定技能 2 号 + 家族同伴 case 給与 不足覆盖家族 → 配偶 就労 / 預貯金 補足 + 申請理由書。
3. 受入機関 倒産 / 経営悪化 致 給与 大幅下落 → 评估转 受入機関 + 重新提交 在留資格変更。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 源泉徴収票（直近 3 年）
- 給与明細（直近 12 个月）
- 課税証明書（直近 3 年，含所得 + 控除项目）
- 雇用契約書（特定技能 専用書式）+ 給与改定通知
- 同職場 日本人従業員 報酬規定（必要时）
- 登録支援機関 支援計画書（1 号 case）

## 价值 expiry 说明
expiry_days = 90。月薪 / 年収变化按月度走，但每年 6 月 課税証明書 切換期需重新上传新年度数据。
