# 所得税 / 確定申告 状况

## 这是什么
所得税 是个人就一年（1/1-12/31）所得缴纳的国税，所得税法 规定。特定技能 持有者多为 給与所得者，通过 源泉徴収 + 年末調整 完成大多数情况；但派遣切換 / 転籍 / 中途入退社 易出现「年中两处给与」未合并 年末調整 → 须做 確定申告。海外汇款（送金）超 100 万円 同样触发 確定申告。帰国時 未完成 申告 → 5 年期间申告权时效 + 出国后追跡困难。

## 为什么重要
申告漏れ・無申告 / 過少申告 → 加算税 + 延滞税。出入国在留管理庁 续签 / 1 号 → 2 号切換 审查直接调取 納税証明書（其 1 / 其 2），有違反记录是消极评价。特定技能 雇用主交替时易出现 源泉徴収票 拼接错漏。帰国時送金 未做 国外送金等調書 → 后续 還付 / 追跡 困难。

## 判断标准
- **绿**：q1=no（无义务） 或 q1=yes + q2=yes + q3=no（按期完成且无补正）
- **黄**：q1 或 q2 unknown（不确定是否需做或是否完成）
- **红**：q1=yes + q2=no（漏报）OR q3=yes（已收处罚通知）

## 处理建议
### 绿
档案归档直近 3 年 確定申告書 控（写） + 全部 源泉徴収票（含跳槽前公司）。下次续签前 90 天复检。雇用主单一 + 全年無切換 → 通常 年末調整 即足。

### 黄
1. 登录 [e-Tax](https://www.e-tax.nta.go.jp/) 查申告履历，或到所辖 税務署 调取过去 3 年记录。
2. 雇用主交替年份 → 旧公司 + 新公司 双方 源泉徴収票 都要齐，年末調整 仅由当年最后一家公司做，前家所得未合并 → 须 確定申告。
3. 派遣会社切換 / 転籍 case 同样适用。

### 红
1. 漏报 → 期限後申告 立即补做，可减轻 無申告加算税（自首減額）。
2. 帰国時送金 → 国外送金等調書 关联；帰国前完成 確定申告 + 還付申告（多缴回收）。
3. 涉及海外汇款超 100 万円 → 国外送金等調書 同期补。
4. 收到处罚通知不回应 = 升级 强制執行。建议咨询税理士 + 行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 確定申告書 控（直近 3 年）
- 源泉徴収票（全部雇用主，含跳槽前）
- 給与支払報告書
- 納税証明書 其 1 / 其 2 / 其 3（直近 3 年）
- e-Tax 受信通知 / 利用者識別番号
- 海外送金 凭证 / 国外送金等調書 控

## 价值 expiry 说明
expiry_days = 90。確定申告 按年度走（次年 3/15 期限），但每年 6 月 課税証明書 切換后档案需复检。雇用主交替 / 派遣切換 时即时核对 源泉徴収票 齐全。
