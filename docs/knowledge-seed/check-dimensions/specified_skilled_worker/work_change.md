# 工作变更 / 所属机构届出

## 这是什么
入管法第 19-16 条规定，中长期在留者所属机构发生消灭、离脱、移籍时，事由发生 14 日内须向 出入国在留管理庁 提交「契約機関に関する届出」。特定技能 的特殊点：转籍受限严格，离职后须在合理期间（一般 3 个月内，超出有在留資格 取消风险）转入同一特定产业分野的另一家受入机构 + 重新签订 雇用契約 + 重新提交 在留資格変更（同业种転職）申请或所属機関 届出。

## 为什么重要
特定技能 持有者的转籍受 12-16 个特定产业分野（介護 / 建設 / 外食業 / 宿泊 等）严格限定，跨业种転職原则上不允许。届出义务履行 + 业种匹配性 + 受入机构资格 三项缺一不可。受入机构本身需向 出入国在留管理庁 报告 雇用状況、四半期 报告，单方违反会连累在留者。漏报或转入非登录支援机构 / 非届出受入机构 在续签时直接驳回。

## 判断标准
- **绿**：q1=no（无变更） 或 q1=yes + q2=yes + q3=yes（已届出 + 同业种 + 新公司具备受入机构资格）
- **黄**：q1=yes 但 q2 或 q3 unknown（届出记录或业种 / 受入机构资格不确定）
- **红**：q1=yes 且 q2=no（未届出）或 q3=no（跨业种 或 新公司非受入机构）

## 处理建议
### 绿
档案归档「最近一次届出受理票」+ 雇用契約書 + 新公司「特定技能所属機関 登録票」+ 登録支援機関 委託契約书（如委託）。下次续签前 90 天复检即可。

### 黄
1. 登录 [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 查届出受理记录。
2. 对照原 在留資格 认定的特定产业分野，比对当前 雇用契約書 内业务内容。
3. 在 出入国在留管理庁 公开的「特定技能所属機関」名单中确认新公司是否在列。
4. 离职状态超过 3 个月需主动联系 入国管理局 说明求职状况，避免在留資格 被取消。

### 红
1. 14 日窗口已过 → 立即补办「契約機関に関する届出」。同业种転職 须同时提交 在留資格変更 申请。
2. 跨业种 → 原则上不可继续以特定技能 在留，需要重新接受技能評価試験 + 日本語試験 後 再入境申请。
3. 新公司非受入机构 → 立即停止该雇用，转入合规受入机构。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
