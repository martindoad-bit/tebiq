# 健康保険 / 年金 缴纳状况

## 这是什么
日本「国民皆保険」/「国民皆年金」体制。雇员体系：社会保険（健康保険 + 厚生年金 一套打包）。其他居住者：国民健康保険 + 国民年金。20-60 岁 + 住所在日 强制加入年金，3 个月以上居留强制加入医疗保険。

特定技能 受入機関（雇主）按 出入国在留管理庁「特定技能所属機関 基準」必须为外国人办理 社会保険（健康保険 + 厚生年金）加入手続。2024-10 社会保険 適用拡大（被保険者数 51 人以上 事業所 → パート・短時間労働者 也纳入）后，特定技能者 几乎全数 社会保険 适用。

## 为什么重要
未缴 / 滞納 → 督促状 → 催告状 → 強制徴収（給与差押 / 預金差押）。出入国在留管理庁 续签 / 永住・帰化 审查取「直近 2 年缴纳记录」。2024-06-21 公布的入管法改正法（法律第 60 号）将「故意不缴 公租公課」追加为永住資格 取消事由（施行日由政令定，预期 2027 内）。

特定技能 续签 / 1 号→2 号 移行时，入管 同时审查 受入機関 + 本人 双方履行状况。受入機関 未办 社会保険 加入 = 受入機関 基準不適合 → 雇用継続不能。本人 短期出国 / 转入受入機関 期间 也可能出现 国保 漏缴。

## 判断标准
- **绿**：q2=yes + q3=no（按期 + 无异常）
- **黄**：q2 或 q3 unknown
- **红**：q2=no（漏缴） OR q3=yes（有滞納 / 督促 / 強制徴収予告）

## 处理建议
### 绿
档案归档：ねんきん定期便（年度）+ 健康保険証 + 直近 12 个月給与明細（標準報酬月額 可见）+ 受入機関 発行「社会保険加入証明」。下次续签前 90 天复检。

### 黄
1. [ねんきんネット](https://www.nenkin.go.jp/n_net/) 查个人年金缴纳记录（中文 / 英文等多言語界面 也有）。
2. 受入機関 担当者 / 登録支援機関 取「健康保険・厚生年金 加入証明」。
3. 受入機関 移行 / 短期出国 期间若曾走 国保 + 国民年金，市役所 国保課 取「保険料 納付済通知書」。

### 红
1. 経济困難 → 申请 国民年金 保険料免除制度（全額・3/4・半額・1/4）/ 納付猶予制度。被认可的免除期间不算消极评价。
2. 受入機関 未加入 社会保険 → 立即通过 登録支援機関 / 行政書士 协调办理 新規適用届 + 遡及加入。
3. 単純漏缴 → 立即补缴 + 索取 納付済通知書。
4. 強制徴収予告 阶段 = 续签 / 1 号→2 号 移行 高风险。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 90。月度缴纳，90 天内变化有限。但 受入機関 変更 / 帰国一時離脱 / 算定基礎届 提出（7 月）后 標準報酬 切換需复检。1 号→2 号 移行 申请前 必检。
