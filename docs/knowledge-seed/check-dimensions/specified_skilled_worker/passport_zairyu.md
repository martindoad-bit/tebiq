# 护照 / 在留カード 有效期

## 这是什么
- 护照（パスポート）：本国大使馆 / 領事館 发行的身份证明 + 出入国 証明。日本 在留資格 持有期间须保持有效。
- 在留カード：出入国在留管理庁 发行的中长期在留者身份证明，载在留資格 / 期限 / 就労制限 等。
- 特定技能 1 号 在留期間 多档：4 个月 / 1 年 / 6 个月 / 不定期，通算上限 5 年。
- 特定技能 2 号 在留期間：6 个月 / 1 年 / 3 年（更新可），无通算上限。
- 短档（4 个月 / 6 个月）高频出现，续签节奏极密。

## 为什么重要
- 护照失效 → 出入国不可 / 续签材料无法核证 / 银行口座 等手续受阻。
- 在留カード 期限到 = 在留期間 同时到 → 不法滞在 风险。特定技能 1 号 通算 5 年内频繁续签，每次续签前 D-90 已是常态，期限管理失败影响最大。
- 在留カード 上信息不一致（住址 / 所属机构 / 姓名）= 续签时被「申請事項違背」消极评价。特定技能 持有者一旦转職未届出 / 受入機関 倒産 / 解雇 14 日以内未届出，续签时按既往运用难以通过。
- 受入機関 与 在留カード 所属机关 不一致 = 即时不利。

## 判断标准
- **绿**：护照 > 12 个月有效 + 在留カード ≥ 6 个月 + 信息全部更新
- **黄**：护照 6-12 个月 OR 在留カード 3-6 个月 OR 信息更新状况不确定
- **红**：护照 < 6 个月 OR 在留カード < 3 个月 OR 信息错误

## 处理建议
### 绿
档案归档 护照号 + 在留カード 番号 + 各自有效期 + 直近一次更新日 + 通算在留 期间累計（特定技能 1 号）。续签前 30 天复检。1 号 持有者额外标注通算 5 年到期日。

### 黄
1. 护照 6-12 个月 → 在出国前更新（多数国家要求入境时 ≥ 6 个月）。本国大使馆 / 領事館 在地方都市处理时间偏长。
2. 在留カード 3-6 个月 → 续签申请窗口已开（D-90），开始准备材料：受入機関 / 登録支援機関 提供 雇用契約書 / 報酬要件 関連書類 / 支援計画。
3. 信息更新不确定 → 持卡到 出入国在留管理庁 / 入管局 確認现行记载，住址变更须 14 日以内届出 市区町村役所 + 出入国在留管理庁。

### 红
1. 护照 < 6 个月 → 立即到本国大使馆 / 領事館 更新；多数大使馆需 1-3 周。
2. 在留カード < 3 个月 → 本周内启动续签，避免「申請受理票 + 特例期間」紧迫。受入機関 与登録支援機関 协调材料是常见瓶颈，提早联络。1 号 通算 5 年接近期满 → 必须并行准备 2 号移行 / 别在留資格 切换。
3. 信息错误 → 持卡到 入管 申请「変更届」或「再交付申請」。涉及受入機関 倒産 / 转職 / 国籍变更 → 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 30。护照 / 在留カード 期限按月计算，30 天内变化对续签 / 出国 影响显著，必须高频复检。特定技能 短档申请高发，建议从 D-30 起每周复检一次。
