# 課税証明書 / 納税証明書 准备

## 这是什么
課税証明書：证明该年度课税额（所得 + 控除项目 + 税额）。納税証明書：证明已缴 / 未缴金额（其 1 = 納付状況、其 2 = 所得金額、其 3 = 完納証明）。発行权属当年 1 月 1 日时点居住的 市役所 / 区役所 / 都税事務所。受入機関 法人税 / 消費税 / 源泉所得税 納税証明書 在所轄税務署 取得。

## 为什么重要
出入国在留管理庁 在 特定技能 在留期間更新申請 时调取这两份证明判断「公租公課按期缴纳」+「収入安定性」。特定技能 1 号 续签通常 直近 1 年，特定技能 2 号 / 高度化 转向准备 须 直近 3 年。同时 受入機関（雇用先）法人 納税証明書 是 特定技能 雇用契約 审查必备——受入機関滞納会被判定不適格 → 在籍 特定技能外国人 续签连带受影响。納税証明書 其 1 显示督促 / 滞納 / 強制徴収 历史，本人 + 受入機関 双方都不能有。

## 判断标准
- **绿**：明确知道需要哪几年 + 知道取得窗口 + 1/1 居住地与现在一致 + 受入機関無滞納確認
- **黄**：不确定取得窗口或 1/1 居住地
- **红**：完全不知道需要哪些 / 1/1 居住地不在现在 / 受入機関有滞納 → 需邮寄申请 + 雇主沟通（耗时长）

## 处理建议
### 绿
档案归档当前年度 + 直近 1-2 年証明書。受入機関 同时确认 法人侧 納税証明書（其 1 / 其 2）由雇主负责取得提交。

### 黄
1. 个人侧：持 在留カード 到所辖 市役所 / 区役所 申请，当天可拿。手数料约 300 円 / 通。
2. 受入機関 法人側：由雇主 持登記事項証明書 / 法人代表者印 到所轄税務署 取得，或 e-Tax 在线申請。提示雇主提前 30 日准备。
3. 代理人取需 委任状（本人署名 + 印鑑）。
4. 6 月切換新年度，前 1-5 月只能取去年度。
5. 転職 / 受入機関変更 case：旧雇主期间所得仍体现于本人 課税証明書，无需向旧雇主索取。

### 红
1. 1 月 1 日 时点居住地与现在不同 → 须到旧居住地市町村申请。多数支持邮寄申請（公式样式 + 切手 + 返信用封筒），耗时 1-2 周。
2. 受入機関滞納 → 立即与雇主沟通完納清算 + 取得 完納証明書。雇主若拒不清算 → 转職検討（特定技能 在留中可同分野転職）。
3. 提前 30 日以上启动准备，受入機関协调耗时。
4. 涉及多市町村跨地区 / 受入機関滞納 / 転職連動，建议咨询行政書士確認流程。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（本人確認）
- 印鑑（部分自治体可签名替代）
- 委任状（代理人申请）
- 旧住所証明（搬迁后）
- 受入機関 法人 納税証明書（雇主提供）
- 雇用契約書 / 雇用条件書（受入機関確認）

## 价值 expiry 说明
expiry_days = 30。証明書 内容固定，但 6 月切換新年度 + 续签材料"3 个月内発行"要求双重约束，30 日内重检最稳。受入機関 法人侧同步刷新更稳。
