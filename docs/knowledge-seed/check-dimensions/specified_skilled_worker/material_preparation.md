# 续签 / 永住 材料 准备进度

## 这是什么
特定技能 1 号 / 2 号 在留期間更新申請 必备材料分两部分：① 受入機関 侧（受入計画 写 / 雇用契約書 / 1 号特定技能外国人支援計画書 / 受入機関 履歴事項全部証明書 / 直近 決算書 等）② 個人侧（在留カード / パスポート / 課税証明書 / 納税証明書 / 住民票 / 技能評価試験 合格証 / 日本語能力試験 合格証 等）。受入機関 协助材料较多，准备时间一般 3-4 周。1 号转 2 号 case 还需 技能 2 号評価試験 合格証。

## 为什么重要
- 入管 受理窗口 D-90 启动；最迟 D-Day 前递交可启动 特例期間。
- 特定技能 case 受入機関 適合性 + 支援体制 是审查重点，受入計画 / 支援計画書 缺失 = 不許可 风险。
- 1 号 通算 5 年上限 → 1 号転 2 号 / 1 号転 技人国 case 需早期准备 技能 2 号評価試験 合格 / 学歴 整合性 説明。

## 判断标准
- **绿**：D-90 ~ D-120 + 受入機関 材料 已请 + 个人材料齐全
- **黄**：D-30 ~ D-90 OR 受入機関 / 个人 材料其中一类不确定
- **红**：D-30 内 OR 受入機関 未请 OR 个人材料缺

## 处理建议
### 绿
档案归档完整清单 + 各项材料取得日期。续签前 14 天复检（材料更新状况 + 1 号 残期间）。

### 黄
1. 对照档案 + 本签证 必备材料清单（在 TEBIQ 知识中心查看 特定技能 续签材料文章）逐项核对。
2. 受入機関 材料（受入計画 / 支援計画書 / 履歴事項全部証明書 等）→ 担当者 协助，提前 1 个月请准备。
3. 登録支援機関 委託 case → 支援計画書 由 登録支援機関 作成。
4. 市役所材料（住民票 / 課税証明書 / 納税証明書）当天可拿，但 6 月切換新年度时点要注意。
5. 1 号转 2 号 → 技能 2 号評価試験 合格証 是 必須。

### 红
1. D-30 内 → 即使材料未齐也优先到 入管 递交 申請受理票，启动 特例期間（最长 到期后 2 个月）。
2. 受入機関 侧 缺 → 立刻联系受入機関 担当者 + 提供详细清单。建议按 受入計画 → 雇用契約書 → 支援計画書 → 履歴事項全部証明書 顺序请。
3. 个人侧 缺（如 課税証明書 跨自治体）→ 邮寄申请，需提前 1-2 周。
4. 特定技能 case 受入機関 / 登録支援機関 协调复杂，复杂情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

**受入機関 侧**：
- 特定技能雇用契約書（写）
- 受入計画書（特定技能所属機関 概要 + 受入予定数 + 業務区分）
- 1 号特定技能外国人支援計画書（1 号 case 必須）
- 受入機関 履歴事項全部証明書（3 个月以内発行）
- 受入機関 直近 決算書（損益計算書 / 貸借対照表）
- 労働保険料 納付証明書
- 社会保険料 納入証明書
- 登録支援機関 との支援委託契約書（委託 case）

**个人侧**：
- 在留カード（正反面写）
- パスポート（写 + 出入国印章页）
- 申請書（最新样式从 [出入国在留管理庁](https://www.moj.go.jp/isa/applications/procedures/) 下载）
- 課税証明書（直近 1 年）
- 納税証明書（其 1 / 其 2 / 其 3）
- 住民票（直近 3 个月以内発行）
- 技能評価試験 合格証（1 号 → 1 号 case 維持確認 / 1 号 → 2 号 case 2 号合格証）
- 日本語能力試験 合格証（N4 以上 / 業務区分 别要件）
- 写真 4×3cm（3 个月以内）
- 健康診断書（受入機関 要請 case）
- 手数料 4,000 円分 収入印紙（許可时購入）

## 价值 expiry 说明
expiry_days = 14。续签材料准备期高频变化（受入機関 進度 / 試験 合格証 取得 / 市役所材料 新発行），14 天内复检最稳。
