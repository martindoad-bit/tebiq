# 出入国记录 / 连续居住

## 这是什么
出入国記録 是 出入国在留管理庁 保存的本人入出境完整履历。再入国許可（包括 みなし再入国 1 年 / 正規 最長 5 年）= 在留資格 持有人出国期间不丧失资格的法定手续。永住者 同适用。

## 为什么重要
- 在留期間更新：特定技能 1 号 / 2 号 持有人长期返国 = 雇用契約 履行不稳，受入機関（雇主）须协助说明出国期间未实际离职。1 号有累计 5 年上限的硬性约束，长期离开日本会浪费在日本的累计时长。
- 永住申請：特定技能 2 号 可参与永住「引き続き 10 年」累计；1 号期间不计入。出国天数依然按 90 日 / 100 日 标准评估。
- 全签证：出国时未办再入国許可 = 在留資格 自动失效。

## 判断标准
- **绿**：1 年内单次出国 ≤ 90 日 + 累计 ≤ 100 日 + 全程办了再入国許可
- **黄**：天数 / 手续不确定
- **红**：单次或累计超 + 或未办手续

## 处理建议
### 绿
档案归档每次出入国时点 + ED カード 控（写）/ みなし选项确认。续签 / 永住 前自查累计天数。特定技能 应与受入機関 / 登録支援機関 同步出国安排。

### 黄
1. 向 [出入国在留管理庁](https://www.moj.go.jp/isa/) 申请「出入国記録に係る開示請求」（书面，回答 2-4 周）。
2. 对照档案 出入国 履历 + 護照入出国印章。

### 红
1. 单次 / 累计超期 → 永住「引き続き」可能中断，需重新累计。续签 也可能给较短期。1 号 累计 5 年上限内出国天数过长 = 实际在职时间不足。
2. 未办再入国許可 → 在留資格 已自动失效，回国后只能以新申请，雇用契約 中止。
3. 受入機関 须协助提交雇用継続証明書 / 出国理由書 说明出国期间不属实质离职。
4. 严重情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 出入国記録 開示請求 結果
- 護照（入出国印章页）
- 再入国許可書 控（写）/ ED カード 副本
- 在留カード（含「再入国許可」栏，如有正規许可）
- 受入機関 雇用継続証明書 / 給与支払 履历

## 价值 expiry 说明
expiry_days = 30。出入国 状况按出行随时变化，30 日内须复检；永住申請 关键节点前重新核对完整记录。
