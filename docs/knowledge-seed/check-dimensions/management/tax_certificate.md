# 課税証明書 / 納税証明書 准备

## 这是什么
課税証明書：证明该年度课税额（所得 + 控除项目 + 税额）。納税証明書：证明已缴 / 未缴金额（其 1 = 納付状況、其 2 = 所得金額、其 3 = 完納証明）。発行权属当年 1 月 1 日时点居住的 市役所 / 区役所 / 都税事務所。法人税 / 消費税 / 源泉所得税 的 納税証明書 在所辖 税務署 取得。

## 为什么重要
出入国在留管理庁 在 在留期間更新申請 时调取这两份证明判断「公租公課按期缴纳」+「収入安定性」。経営・管理 必须 法人 + 个人双轨提交：法人侧 法人税 / 消費税 / 源泉所得税 納税証明書（税務署発行）；个人侧 役員報酬 課税証明書 + 住民税 納税証明書（市役所発行）。永住申請 须直近 5 年完整。納税証明書 其 1 显示督促 / 滞納 / 強制徴収 历史，法人滞納直接判定「事業安定性」不足。

## 判断标准
- **绿**：明确知道需要哪几年 + 知道取得窗口（市役所 + 税務署）+ 1/1 居住地与现在一致
- **黄**：不确定取得窗口或 1/1 居住地
- **红**：完全不知道需要哪些 / 1/1 居住地不在现在 / 法人滞納 → 需邮寄申请 + 滞納清算（耗时长）

## 处理建议
### 绿
档案归档当前年度 + 直近 1-2 年証明書。法人 + 个人 双套并行管理。

### 黄
1. 个人侧：持 在留カード 到所辖 市役所 / 区役所 申请，当天可拿。手数料约 300 円 / 通。
2. 法人侧：法人代表者 持登記事項証明書 / 法人印鑑証明書 到 所轄税務署 申请 法人税・消費税・源泉所得税 納税証明書（其 1 / 其 2）。e-Tax 経由可在线申請。
3. 代理人取需 委任状（本人署名 + 印鑑）。法人代理需 法人代表者印 委任状。
4. 6 月切換新年度，前 1-5 月只能取去年度。

### 红
1. 1 月 1 日 时点居住地与现在不同 → 个人 課税証明書 须到旧居住地市町村申请。多数支持邮寄申請（公式样式 + 切手 + 返信用封筒），耗时 1-2 周。
2. 法人滞納情形 → 先到 税務署 完納清算 + 取得 完納証明書 / 滞納処理証明書，否则续签直接危险。
3. 提前 30 日以上启动准备，経営・管理 双轨材料量大需更早。
4. 涉及多市町村跨地区情形 / 法人税滞納清算，建议咨询行政書士確認流程。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（本人確認）
- 印鑑（部分自治体可签名替代）
- 法人代表者印 / 法人印鑑証明書（法人税納税証明書取得用）
- 委任状（代理人申请）
- 旧住所証明（搬迁后）
- 登記事項証明書（法人确认）

## 价值 expiry 说明
expiry_days = 30。証明書 内容固定，但 6 月切換新年度 + 续签材料"3 个月内発行"要求双重约束，30 日内重检最稳。経営・管理 双轨同步更稳。
