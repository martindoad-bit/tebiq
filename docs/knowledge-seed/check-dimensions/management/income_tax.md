# 所得税 / 確定申告 状况

## 这是什么
所得税 是个人就一年（1/1-12/31）所得缴纳的国税，所得税法 规定。経営・管理 签持有者 99% 为代表取締役 / 役員，役員報酬 走 源泉徴収 + 年末調整，但常因 法人决算 / 配当 / 役員賞与 / 退職金 / 不動産 等触发 確定申告 必要。法人税 申告（決算後 2 个月内）与个人 確定申告（次年 3/15）双轨同期管理。

## 为什么重要
経営・管理 续签是审查 法人 经营实态 + 个人税务合规 双重证明。役員報酬 申告漏れ / 法人决算 与个人申告口径不一致 → 续签直接消极。重大違反追究刑责（所得税法第 238 条）。永住・帰化 审查取 納税証明書 其 3（所得税额 + 完納事实），含 役員報酬 + 配当 等综合所得。

## 判断标准
- **绿**：q1=no（无义务） 或 q1=yes + q2=yes + q3=no（按期完成且无补正）
- **黄**：q1 或 q2 unknown（不确定是否需做或是否完成）
- **红**：q1=yes + q2=no（漏报）OR q3=yes（已收处罚通知）

## 处理建议
### 绿
档案归档直近 3 年 確定申告書 控（写） + 法人决算書 + 役員報酬 一覧。下次续签前 90 天复检。法人税 / 消費税 / 源泉所得税 / 個人所得税 四条线分别核对。

### 黄
1. 登录 [e-Tax](https://www.e-tax.nta.go.jp/) 查个人申告履历，法人申告查 [国税庁 法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/) 关联档案。
2. 顾问税理士 处取过去 3 年 個人 + 法人 申告 控。
3. 役員報酬 改定（事業年度開始 3 个月内）记录核对，超期改定无法损金算入。

### 红
1. 漏报 → 期限後申告 立即补做，可减轻 無申告加算税（自首減額）。
2. 涉及海外资产（5000 万円超）→ 国外財産調書 须同期补；涉及海外汇款 → 国外送金等調書 关联。
3. 法人决算书 与个人申告口径不一致 → 修正申告 双轨同步。
4. 收到处罚通知不回应 = 升级 强制執行。建议咨询税理士 + 行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 確定申告書 控（直近 3 年）
- 源泉徴収票 / 給与支払報告書（役員報酬）
- 法人決算書 / 法人税 申告書 控（直近 3 年）
- 納税証明書 其 1 / 其 2 / 其 3（直近 3 年）
- e-Tax 受信通知 / 利用者識別番号
- 配当 / 不動産 / 海外汇款 收支凭证

## 价值 expiry 说明
expiry_days = 90。確定申告 按年度走（次年 3/15 期限）+ 法人决算 各社事業年度，每年 6 月 課税証明書 切換后档案需复检。
