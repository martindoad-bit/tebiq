# 护照 / 在留カード 有效期

## 这是什么
- 护照（パスポート）：本国大使馆 / 領事館 发行的身份证明 + 出入国 証明。日本 在留資格 持有期间须保持有效。
- 在留カード：出入国在留管理庁 发行的中长期在留者身份证明，载在留資格 / 期限 / 就労制限 等。経営・管理 在留期間 多为 1 年 / 3 年 / 5 年，初次取得多为 1 年档。

## 为什么重要
- 护照失效 → 出入国不可 / 续签材料无法核证 / 银行口座 / 法人代表者口座 等手续受阻。
- 在留カード 期限到 = 在留期間 同时到 → 不法滞在 风险，法人代表者一旦失格，登記簿 役員记载需即时变更。
- 在留カード 上信息不一致（住址 / 所属机构 / 姓名）= 续签时被「申請事項違背」消极评价。
- 経営・管理 2024 年改正経過措置 期限为 2028-10-16，期满后续签审查（资本金 3000 万円 / 常勤 1 名以上等）升级；在留カード 期限届至此节点附近的申請者需提前规划。

## 判断标准
- **绿**：护照 > 12 个月有效 + 在留カード ≥ 6 个月 + 信息全部更新
- **黄**：护照 6-12 个月 OR 在留カード 3-6 个月 OR 信息更新状况不确定
- **红**：护照 < 6 个月 OR 在留カード < 3 个月 OR 信息错误

## 处理建议
### 绿
档案归档 护照号 + 在留カード 番号 + 各自有效期 + 直近一次更新日。续签前 30 天复检。経営・管理 持有者建议同步对照 法人 履歴事項全部証明書 上代表者住址 / 姓名 与 在留カード 一致性。

### 黄
1. 护照 6-12 个月 → 在出国前更新（多数国家要求入境时 ≥ 6 个月）。
2. 在留カード 3-6 个月 → 续签申请窗口已开（D-90），开始准备材料：直近 2 期决算書 / 事業計画書 / 損益計算書。
3. 信息更新不确定 → 持卡到 出入国在留管理庁 / 入管局 確認现行记载。

### 红
1. 护照 < 6 个月 → 立即到本国大使馆 / 領事館 更新；多数大使馆需 1-3 周。
2. 在留カード < 3 个月 → 本周内启动续签，避免「申請受理票 + 特例期間」紧迫。経営・管理 受理后审查时间在 1-3 个月，特例期間内需正常营业、保留出勤記録。
3. 信息错误 → 持卡到 入管 申请「変更届」或「再交付申請」。法人代表者姓名 / 住所 变动需联动 法務局 登記。涉及国籍变更 → 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 护照（含申请取得日 / 有效期至）
- 在留カード（正反面）
- 「在留カード 記載事項に変更が生じた場合の届出」受理票（如有）
- 「在留カード 再交付申請」受理票（如有）
- 法人 履歴事項全部証明書（与在留カード 信息对照用）

## 价值 expiry 说明
expiry_days = 30。护照 / 在留カード 期限按月计算，30 天内变化对续签 / 出国 影响显著，必须高频复检。
