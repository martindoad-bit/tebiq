# 健康保険 / 年金 缴纳状况

## 这是什么
日本「国民皆保険」/「国民皆年金」体制。雇员体系：社会保険（健康保険 + 厚生年金 一套打包）。其他居住者：国民健康保険 + 国民年金。20-60 岁 + 住所在日 强制加入年金，3 个月以上居留强制加入医疗保険。

経営・管理 持有者作为法人 役員（代表取締役 / 取締役）：法人即使是 1 人公司、即使 自宅兼办公室，役員 报酬若有支付，原则上 必须加入 社会保険（健康保険 + 厚生年金）。这是 厚生年金保険法 第 6 条「適用事業所」规定。

## 为什么重要
未缴 / 滞納 → 督促状 → 催告状 → 強制徴収（給与差押 / 預金差押）。出入国在留管理庁 续签 / 永住・帰化 审查取「直近 2 年缴纳记录」。2024-06-21 公布的入管法改正法（法律第 60 号）将「故意不缴 公租公課」追加为永住資格 取消事由（施行日由政令定，预期 2027 内）。

経営・管理 续签时，入管 同时审查 法人 + 個人 的社会保険加入状况。法人未办 健保・年金 加入手続 = 重大缺陷。日本年金機構 近年加强 社会保険 加入指导，未加入 法人 会收到「加入指導通知」→「立入検査」→「職権適用」。

## 判断标准
- **绿**：q2=yes + q3=no（按期 + 无异常）
- **黄**：q2 或 q3 unknown
- **红**：q2=no（漏缴） OR q3=yes（有滞納 / 督促 / 強制徴収予告）

## 处理建议
### 绿
档案归档：ねんきん定期便（年度）+ 健康保険証 + 法人 健保・厚年 標準報酬月額 通知書 + 月次 納付書 控え。下次续签前 90 天复检。

### 黄
1. [ねんきんネット](https://www.nenkin.go.jp/n_net/) 查个人年金缴纳记录。
2. 协会けんぽ / 健保組合 取「健康保険 被保険者 標準報酬決定通知書」。
3. 法人侧：年金事務所 取「社会保険料 領収済通知書」（直近 24 个月分）。
4. 役員 報酬 0 円 期间 = 不適用，但需有 株主総会議事録 / 報酬決定議事録 留底。

### 红
1. 法人未加入 社会保険 → 立即办理「健康保険・厚生年金保険 新規適用届」。遡及加入 最长 2 年保険料 一次性补缴风险。
2. 経济困難 → 法人侧申请 厚生年金 納付の猶予（換価の猶予 等）；個人 国民年金期間 申请 免除制度。
3. 強制徴収予告 阶段 = 续签 高风险。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

expiry_days = 90。月度缴纳，90 天内变化有限。但 算定基礎届 提出（7 月）/ 月額変更届 后 標準報酬 切換需复检。役員報酬 改定（事業年度開始 3 ヶ月以内）期间也需重检。
