# 续签 / 永住 材料 准备进度

## 这是什么
経営・管理 在留期間更新申請 必备材料是 双轨：① 法人侧（履歴事項全部証明書 / 定款 / 直近 決算書 / 株主名簿 / 事業計画書 / 損益計算書 / 貸借対照表 等）② 個人侧（在留カード / パスポート / 課税証明書 / 納税証明書 / 住民票 / 役員報酬 改定 議事録 / 源泉徴収票 等）。法人材料涉及 顧問税理士 / 司法書士 协助，准备时间一般 3-4 周。

## 为什么重要
- 入管 受理窗口 D-90 启动；最迟 D-Day 前递交可启动 特例期間。
- 経営・管理 case 法人 持续性 + 役員報酬 妥当性 是核心审查点，决算書 / 議事録 缺失 = 直接 補正通知 / 不許可 风险。
- 法人材料涉及多方（税理士 / 司法書士 / 银行 残高証明）→ 准备期比一般 受雇签证 长。

## 判断标准
- **绿**：D-90 ~ D-120 + 法人材料 已开始 + 个人材料齐全
- **黄**：D-30 ~ D-90 OR 法人 / 个人 材料其中一类不确定
- **红**：D-30 内 OR 法人材料未开始 OR 个人材料缺

## 处理建议
### 绿
档案归档完整清单 + 各项材料取得日期。续签前 14 天复检（决算 / 議事録 状况）。

### 黄
1. 对照档案 + 本签证 必备材料清单（在 TEBIQ 知识中心查看 経営・管理 续签材料文章）逐项核对。
2. 法人材料 决算書 / 損益計算書 / 貸借対照表 → 顧問税理士 配合，需 直近 1 期决算 完成后才能取得。
3. 履歴事項全部証明書 → 法務局 当天可拿（600 円 / 通），但 役員 / 本店所在地 变更后须 直近 3 个月以内発行。
4. 市役所材料（住民票 / 課税証明書 / 納税証明書）当天可拿，但 6 月切換新年度时点要注意。

### 红
1. D-30 内 → 即使材料未齐也优先到 入管 递交 申請受理票，启动 特例期間（最长 到期后 2 个月）。
2. 法人侧 缺 → 立刻联系 顧問税理士 / 司法書士 + 提供详细清单。建议按 履歴事項全部証明書 → 直近 決算書 → 事業計画書 → 役員報酬 議事録 顺序请。
3. 个人侧 缺（如 課税証明書 跨自治体）→ 邮寄申请，需提前 1-2 周。
4. 経営・管理 case 涉及 法人実態 / 事業継続性 説明，复杂情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

**法人侧**：
- 履歴事項全部証明書（直近 3 个月以内発行，法務局）
- 定款（写）
- 直近 決算書（損益計算書 / 貸借対照表 / 株主資本等変動計算書）
- 株主名簿
- 事業計画書（次期 売上 / 利益 / 雇用 計画）
- 法人税 確定申告書（写）
- 法人 納税証明書（其 1 / 其 2）
- 銀行 残高証明書（如有）
- 事務所 賃貸借契約書（写）

**个人侧**：
- 在留カード（正反面写）
- パスポート（写 + 出入国印章页）
- 申請書（最新样式从 [出入国在留管理庁](https://www.moj.go.jp/isa/applications/procedures/) 下载）
- 課税証明書（直近 1 年；役員報酬 大幅变动 case 可能 3 年）
- 納税証明書（其 1 / 其 2 / 其 3）
- 住民票（直近 3 个月以内発行）
- 役員報酬 改定 議事録（直近 1 期内的株主総会 / 取締役会）
- 源泉徴収票（直近 1 年）
- 写真 4×3cm（3 个月以内）
- 手数料 4,000 円分 収入印紙（許可时購入）

## 价值 expiry 说明
expiry_days = 14。续签材料准备期高频变化（决算書 確定 / 議事録 作成 / 法人材料 取得 進度），14 天内复检最稳。
