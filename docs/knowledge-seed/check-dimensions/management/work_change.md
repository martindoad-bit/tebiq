# 工作变更 / 所属机构届出

## 这是什么
入管法第 19-16 条规定，中长期在留者所属机构发生消灭、离脱、移籍时，事由发生 14 日内须向 出入国在留管理庁 提交「契約機関に関する届出」。経営・管理 持有者的特殊点：自己作为代表者的会社也算「所属機関」，公司本店所在地变更、商号变更、解散、新设分公司、役員 / 取締役 加入或退任，均属届出对象。

## 为什么重要
経営・管理 在留資格 与所属公司绑定极紧。届出义务履行情况是续签审查的素行善良 + 公的義務履行 评价项。2024 年 経営・管理 改正后，常勤職員 范围、事務所要件、出資額（500 万円→3000 万円方向）的审查趋严，公司基础信息有任何变化但未届出，会在续签时直接质疑事業の継続性 与 安定性。

## 判断标准
- **绿**：q1=no（无变更） 或 q1=yes + q2=yes + q3=yes（公司变更已届出 + 仍在経営・管理 框架）
- **黄**：q1=yes 但 q2 或 q3 unknown（届出记录或岗位匹配性不确定）
- **红**：q1=yes 且 q2=no（未届出）或 q3=no（实际不再担任经营 / 管理职务，转为受雇员工）

## 处理建议
### 绿
档案归档「最近一次届出受理票」+ 登記事項証明書（最新）+ 役員報酬決議書。下次续签前 90 天复检即可。

### 黄
1. 登录 [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 查届出受理记录。
2. 对照原 在留資格 认定的活动范围，比对当前 役員登記 状态、事業内容、事務所所在地。
3. 役員 变动（如新增取締役、监查役任期到期未续聘）也属变更事项，常被忽略。

### 红
1. 14 日窗口已过 → 立即补办「契約機関に関する届出」。本店移転、商号変更、解散、合併 等分别有专用样式。
2. 岗位偏离 → 若已不再实际经营，转为他社受雇者，需要 在留資格変更（経営・管理 → 技人国 等）。
3. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（含「就労制限」栏）
- 登記事項証明書（最新 + 変更履歴）
- 定款 / 株主名簿
- 役員報酬 議事録
- 事務所 賃貸借契約書（变更时）
- 届出受理票（電子届出 PDF 或纸质回執）
- 直近年度 決算書（续签时使用）

## 价值 expiry 说明
expiry_days = 90。続签材料准备期一般 3-5 个月前启动，90 天能覆盖整个准备窗口；超过 90 天若公司情况未变，状态仍可信。
