# 收入水平 / 独立生计

## 这是什么
入管法 在多种 在留資格 中要求「日本人と同等以上の報酬」或「独立生計」要件。具体数额因 在留資格 / 地域 / 業種 而异，没有全国统一线，但有事实上的下限标准。経営・管理 的核心是「役員報酬」—— 实务上一般 250-500 万円 / 年 起步，地域 + 業種 + 事業 規模 决定具体水位。役員報酬 不可低于事业可持续运营的金额；過低 会触发「事業の継続性」与「独立生計」双重质疑。

## 为什么重要
- 在留期間更新：役員報酬 长期低于 250 万円 → 入管 怀疑事業 实质性，续签可能短期化或不许可。
- 役員報酬 改定：每年「事業年度開始から 3 ヶ月以内」由 株主総会 / 取締役会 議事録 决定 —— 议事录缺失或金额不一致直接成审查瑕疵。
- 永住申請：法定「独立生計」要件，5 年内 課税証明 显示稳定 役員報酬 是关键。

## 判断标准
- **绿**：q1 ≥ 中段 + q2 == yes + q3 != no（一致或更高 + 覆盖家族）
- **黄**：q1 偏低 OR q2/q3 unknown
- **红**：q1 = under_300 OR q2 == no OR q3 == no

## 处理建议
### 绿
档案归档直近 3 年 源泉徴収票 / 課税証明書 / 役員報酬 議事録 / 決算書。续签前 90 天复检。

### 黄
1. 对照档案 直近 3 年 源泉徴収票 + 課税証明書 与申请时声明。
2. 役員報酬 在 250-500 万円 区间且事業 仍在投入期 → 申請理由書 + 事業計画書 + 決算 实绩 补强。
3. 確認 役員報酬 改定 議事録 与実際 給与額 一致。

### 红
1. 役員報酬 显著下降或不足 250 万円 → 准备客观证据（役員報酬 改定 議事録 / 決算書 / 売上推移）+ 申請理由書 说明原因（事業 投入期 / 売上 季節変動 / 設備投資 阶段 等）。
2. 配偶担保 case 担保人 役員報酬 不足 → 评估配偶独立収入或本人转 在留資格。
3. 永住 case 不足独立生計 → 等到稳定 役員報酬 1-2 年再递。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 源泉徴収票（直近 3 年，役員報酬 来源）
- 課税証明書（直近 3 年，含所得 + 控除项目）
- 役員報酬 改定 議事録（株主総会 / 取締役会）
- 決算書（直近 2 期，B/S + P/L）
- 法人 都道府県民税 / 市町村民税 課税証明書

## 价值 expiry 说明
expiry_days = 90。役員報酬 一年改定 1 次，但每年 6 月 課税証明書 切換期需重新上传新年度数据。
