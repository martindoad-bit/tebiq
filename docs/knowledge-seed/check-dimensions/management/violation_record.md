# 违规记录 / 罚金 / 入管警告

## 这是什么
日本签证审查中的「素行善良」要件，依据 入管法 / 国籍法。経営・管理 在留資格 范围内，对象包括 役員 个人 + 所属法人。範围包括：刑事处罚（罚金以上）/ 行政处罚（道交法违反、労基法 違反、税法違反、風俗営業 違反、不適正受入 等）/ 入管法 上的警告 / 出頭通知 / 補正 通知 / 不許可 通知 / 在留資格 取消手続。

## 为什么重要
出入国在留管理庁 / 法務省 在 在留期間更新 / 永住申請 / 帰化申請 时通过 法務省内 共有データベース 调取记录。永住許可ガイドライン 明示「素行善良」为法定要件之一。重大违反 → 直接 不許可 / 取消。轻微违反 → 时间 + 全部信息开示后可能通过。

経営・管理 特有的双重审查：
- 役員 个人 违反（飲酒運転 / 罚金 等）= 个人 在留資格 续签消极。
- 法人 违反（脱税 / 申告漏れ / 労基法 違反 / 不適正受入 / 風俗営業 違反 / 名義貸し）= 法人事業 持续性 + 个人 経営 適格性 双重负面评价。
- 法人 税务申告漏れ 被 税務調査 指摘 = 即使無刑事処分，経営状況 可信度受损，与「事業 継続性」要件冲突。

## 判断标准
- **绿**：q1=no + q2=no + q3=no（无任何记录）
- **黄**：q1=no + q2=no + q3=yes（仅有入管系警告 / 補正通知 / 不许可 历史）
- **红**：q1=yes 或 q2=yes（个人 / 法人 任一有刑事 / 行政处罚）

## 处理建议
### 绿
档案保留「无记录」状态。法人 决算 / 税务 / 労基 合规年度自查。下次续签前 90 天复检。即使无记录，仍须填写 申請書 「過去の犯罪歴」栏诚实申告。

### 黄
1. 整理过去的 不許可通知書 / 補正通知 / 警告 文件原件 + 已采取的应对记录。
2. 续签 / 永住 申请书「申請理由書」中说明 + 申告 過去の経緯。
3. 不许可 / 補正 未回应 = 续签前必须先解决。

### 红
1. 罚金以上（个人 or 法人代表者）→ 不主动告知 = 不許可。判決書 / 略式命令 / 反則金 凭证档案归档。
2. 法人 税务违反（脱税 / 重加算税 / 申告漏れ）→ 修正申告 + 完納 凭证档案；事業 継続性 説明書 必备。
3. 労基法 違反 / 不適正受入（不法就労 助長罪 等）= 事業 適格性 直接消极；可能引发 在留資格 取消。
4. 風俗営業 違反 / 名義貸し = 経営・管理 在留資格 取消高风险。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 90。违反记录是历史事实，状态稳定；但法人 税务 / 労基 / 営業 合规状况随业务运营随时可能产生新违反，90 天内须复检。
