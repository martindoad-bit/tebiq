# 收入水平 / 独立生计

## 这是什么
入管法 在多种 在留資格 中要求「日本人と同等以上の報酬」或「独立生計」要件。具体数额因 在留資格 / 地域 / 業種 而异，没有全国统一线，但有事实上的下限标准。永住申請 重「独立生計」要件 —— 入管 ガイドライン 实务上要求直近 3-5 年 課税証明 显示稳定収入，单身一般 年収 300 万円 以上、家族 同伴 一般 400 万円 以上为基本线。高度人材 70 分 case 需 年収 300 万円 以上、80 分 case 需 1000 万円 以上 等档位匹配。

## 为什么重要
- 永住申請：法定「独立生計」要件，5 年内 課税証明 显示稳定收入是关键 —— 単年的下落会被审查重点关注。
- 高度人材 加速通道：70 分 / 80 分 申請需对应 年収 阈值，不达标即不能走 1 年 / 3 年 短缩通道。
- 在留期間更新：永住申請 受理需「3 年」在留期間 在手 → 收入下降导致续签短期化 → 永住 申請时机推迟。

## 判断标准
- **绿**：q1 ≥ 中段 + q2 == yes + q3 != no（一致或更高 + 覆盖家族）
- **黄**：q1 偏低 OR q2/q3 unknown
- **红**：q1 = under_300 OR q2 == no OR q3 == no

## 处理建议
### 绿
档案归档直近 5 年 源泉徴収票 / 課税証明書 / 給与明細 / 役員報酬 議事録。永住 申請前 90 天复检 5 年連続性。

### 黄
1. 对照档案 直近 5 年 源泉徴収票 + 課税証明書 与申请时声明。
2. 单年 年収 在 300-400 万円 + 家族 同伴 → 預貯金 残高証明 / 不動産 所有証明 補強。
3. 高度人材 ポイント 在临界点 → 重新计算 ポイント 表 + 補強 学歴 / 職歴 / 資格 等其他维度。

### 红
1. 直近 5 年 内 年収 不足 300 万円 单年 → 准备客观证据（雇用契約書 / 給与变动通知 / 役員報酬 改定 議事録）+ 申請理由書 说明原因（育児休業 / 転職 / 起業 阶段 / 経済周期 等）。
2. 高度人材 70 / 80 分 不达年収阈值 → 等到稳定収入达標 1-2 年再递永住 申請。
3. 配偶担保 case 不足独立生計 → 评估自己独立収入或本人转 在留資格。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 源泉徴収票（直近 5 年）
- 給与明細（直近 12 个月）
- 課税証明書（直近 5 年，含所得 + 控除项目）
- 役員報酬 改定 議事録（自営 / 法人代表 case）
- 預貯金 残高証明 / 不動産 所有証明（必要时）
- 高度人材 ポイント 計算表（高度人材 case）

## 价值 expiry 说明
expiry_days = 90。月薪 / 年収变化按月度走，但每年 6 月 課税証明書 切換期需重新上传新年度数据，永住 申請所需 5 年連続性须每年滚动更新。
