# 住民票 / 搬家记录

## 这是什么
住民票 是市町村 住民基本台帳 系统中本人記録 的副本。中长期在留者（在留期間 3 个月以上）和特别永住者强制登载。搬家时：旧居住地 市役所 取 転出証明書 → 新居住地 14 日内 提交 転入届 → 在留カード 背面更新住址。

永住申請 准备阶段，「在日 10 年（含 5 年就労）」要件 = 看 出入国記録 + 住民票 履历完整性。跨多市町村搬家时档案归档全部 転入 / 転出 履历，缺一段就需要市役所发行「不在住証明」或「住民票除票」补缺。

## 为什么重要
- 入管法 第 19 条 7-9：住居地届出 14 日义务，违反 = 在留資格 取消事由（90 日以上不届出 = 直接取消）。
- 続签 / 永住 申請：调取 住民票 / 住民票記載事項証明書 验证 居住状况 + 世帯构成。
- 在留カード 与 住民票 不一致 = 银行口座 / 携带通信 / 公的手续 全部受阻。
- 永住ガイドライン（令和 8 年 2 月 24 日改訂版）「素行善良 + 国益適合」要件中 公的義務 履行 = 含 住居地届出。10 年期间 履历断点 / 届出遅延 = 消极评价。

## 判断标准
- **绿**：q1=no（无搬家） 或 q1=yes + q2=yes + q3=yes（已届出 + 一致）
- **黄**：q2 或 q3 unknown（届出 / 一致状况不确定）
- **红**：q1=yes + q2=no（未届出）OR q3=no（信息不一致）

## 处理建议
### 绿
档案归档直近一次「住民票（世帯全員 + 国籍 + 在留期間）」+ 5 年内全部 旧住所変更履歴。永住申請时 申請書 履歴欄 与 住民票 履历交叉比对，无缺口。

### 黄
1. 持 在留カード 到所辖 市役所 / 区役所 取「住民票」+「住民票記載事項証明書」，当天可拿，手数料约 300 円 / 通。
2. 旧居住地市役所 申请「住民票除票」+「除かれた住民票記載事項証明書」（保管期限 5 年，超期可能取不到 → 改取「不在住証明」）。
3. 对照档案 已识别住址 与官方记载，差异时尽快 修正。

### 红
1. 14 日 窗口已过 → 立即到 市役所 提交 転入届 / 住居地届出（市役所自动通过 「市町村 → 出入国在留管理庁」 通知系统更新 在留カード）。
2. 90 日 以上未届出 = 在留資格 取消事由（入管法 第 22 条の 4）→ 永住 申請前必须先解决。永住审查 5-10 年期间任何 90 日以上未届出 = 几乎确定不許可。
3. 在留カード 住址错误 → 持卡到 入管 申请「在留カード 記載事項に変更が生じた場合の届出」（一般同时通过 市役所 完成）。
4. 過去 5 年内 住居地届出 違反 履历存在时 → 永住申請 时机推后到違反履历距今 5 年以上为安全线。
5. 严重情形（多次搬家未届出 / 90 日以上 / 多市町村履历缺口）建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 90。住民票 一般 3 个月内有効，永住申請材料 通常要求「3 个月以内発行」。永住申請 → 結果通知 期間 6-12 个月内若再搬家须补提交新 住民票。
