# 健康保険 / 年金 缴纳状况

## 这是什么
日本「国民皆保険」/「国民皆年金」体制。雇员体系：社会保険（健康保険 + 厚生年金 一套打包）。其他居住者：国民健康保険 + 国民年金。20-60 岁 + 住所在日 强制加入年金，3 个月以上居留强制加入医疗保険。

申请永住时，「公的義務 履行」是 永住許可ガイドライン 明记的独立审查项目。2019-05 起入管运用变更，永住申请 提出 直近 2 年（特定情形 5 年）保険料 缴纳记录，按月核对。

## 为什么重要
未缴 / 滞納 → 督促状 → 催告状 → 強制徴収（給与差押 / 預金差押）。

**核心风险**：2024-06-21 公布的入管法改正法（法律第 60 号）将「故意に 公租公課（税・社会保険料・国民年金保険料）を 納付しない」追加为永住資格 取消事由（入管法 第 22 条の 4）。施行日由政令定，预期 2027 内。已取得永住者 也在适用范围内 — 取消后 转换为 定住者 / 1 年 在留期間 / 退去強制 等处分。

直近 2 年记录中 即使一次「期限後納付」也会被记入审查。完整 5 年无瑕疵 是稳妥线。

## 判断标准
- **绿**：q2=yes + q3=no（按期 + 无异常）
- **黄**：q2 或 q3 unknown
- **红**：q2=no（漏缴） OR q3=yes（有滞納 / 督促 / 強制徴収予告）

## 处理建议
### 绿
直近 5 年 完整档案归档：ねんきん定期便 + ねんきんネット 全期間 履历 PDF + 健康保険証（履歴含む）+ 給与明細（24-60 个月）+ 国保 納付済通知書（如有）。永住申请 提出前 90 天再次确认。

### 黄
1. [ねんきんネット](https://www.nenkin.go.jp/n_net/) 下载「年金記録」全期間 PDF，按月核对。
2. 转职 / 离职 空白期 → 市役所 国保課 取「保険料 納付済通知書」（最长 5 年遡及）。
3. 社会保険 → 协会けんぽ / 健保組合 取「健康保険 加入歴 証明」。
4. 出现「期限後納付」标记的月份，准备 上申書 说明事由。

### 红
1. 直近 2 年内 滞納 → 永住申请 阶段 等于不通过。先完納 + 等待 2 年清白期 后再提出。
2. 経济困難 → 申请 国民年金 保険料免除制度 / 納付猶予制度。**認定済 期间** 不算消极评价；漏办免除 直接放着 = 重大瑕疵。
3. 強制徴収予告 / 差押歴 = 永住申请 高风险。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

- 健康保険証（含履歴）
- 国民健康保険料 納付済通知書（市役所，最长 5 年）

- 上申書（期限後納付 / 漏缴月份的事由说明）

## 价值 expiry 说明
expiry_days = 90。月度缴纳，90 天内变化有限。永住申请 提出 时点 直近 2 年（特定情形 5 年）记录都会被取出，**任何一次新滞納 = 起算重置**。提出前 90 天必检。
