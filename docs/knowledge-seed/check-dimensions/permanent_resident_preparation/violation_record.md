# 违规记录 / 罚金 / 入管警告

## 这是什么
日本签证审查中的「素行善良」要件，依据 入管法 / 国籍法。永住申請 中「素行善良」为法定要件，永住許可ガイドライン 明示。範围包括：刑事处罚（罚金以上）/ 行政处罚（道交法违反、风俗営業 等）/ 入管法 上的警告 / 出頭通知 / 補正 通知 / 不許可 通知 / 在留資格 取消手続。

## 为什么重要
出入国在留管理庁 / 法務省 在 在留期間更新 / 永住申請 / 帰化申請 时通过 法務省内 共有データベース 调取记录。永住許可ガイドライン 明示「素行善良」为法定要件之一。重大违反 → 直接 不許可 / 取消。轻微违反 → 时间 + 全部信息开示后可能通过。

永住申請 准备者特有强约束：
- 永住改正法（2024-06-21 公布 法律第 60 号）追加 故意 公租公課（税 / 年金 / 健康保険）未繳 + 特定刑罰法令 違反 为永住 取消 事由。施行日 政令 未定，预期 2027 年内施行。永住取得後 也持续审查。
- 永住許可ガイドライン「直近 5 年 素行善良」= 罚金以上 / 道交法 重大違反 5 年 内出现 = 直接 不許可 几乎确定。
- 永住申請 「身元保証人」对申请人 素行 也间接背书，違反 历史共有受影响。

## 判断标准
- **绿**：q1=no + q2=no + q3=no（无任何记录）
- **黄**：q1=no + q2=no + q3=yes（仅有入管系警告 / 補正通知 / 不许可 历史）
- **红**：q1=yes 或 q2=yes（有刑事 / 行政处罚）

## 处理建议
### 绿
档案保留「无记录」状态。永住申請 前 5 年 / 10 年 履历整体复检。即使无记录，仍须填写 申請書 「過去の犯罪歴」栏诚实申告。永住取得後 改正法 施行后仍持续合规。

### 黄
1. 整理过去的 不許可通知書 / 補正通知 / 警告 文件原件 + 已采取的应对记录。
2. 永住 申請理由書 主動説明 過去の経緯 + 改善経過。
3. 不许可 / 補正 未回应 = 永住申請 前必须先解决。

### 红
1. 罚金以上 → 永住 5 年内 不許可 几乎确定。最早可申请时点 = 罚金完納 后 5 年経過。判決書 / 略式命令 / 反則金 凭证档案归档。
2. 道交法重大违反（飲酒運転 / 当て逃げ / 二重制限速度 等）= 即使已完納，5 年内 永住 不許可 高确率。
3. 风俗営業 違反 / 偽装結婚 嫌疑等 = 在留資格 取消风险，永住 申請資格 直接丧失。
4. 永住改正法 施行后，故意 税 / 年金 / 健康保険 未繳 也可能触发 永住 取消，须长期合规。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
