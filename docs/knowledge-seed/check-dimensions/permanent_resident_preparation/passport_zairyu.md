# 护照 / 在留カード 有效期

## 这是什么
- 护照（パスポート）：本国大使馆 / 領事館 发行的身份证明 + 出入国 証明。日本 在留資格 持有期间须保持有效。
- 在留カード：出入国在留管理庁 发行的中长期在留者身份证明，载在留資格 / 期限 / 就労制限 等。
- 永住申請 时点的 在留カード 有效期 = 「最長の在留期間」要件检查项之一。出入国在留管理庁 实务上要求申请时持有 3 年 / 5 年档（或对应签证最长档），1 年档原则上不受理。

## 为什么重要
- 护照失效 → 出入国不可 / 永住申請 添付資料 无法核证 / 银行口座 等手续受阻。
- 在留カード 期限到 = 在留期間 同时到 → 永住申請 受理途中如期限到来，须先续签现签证再继续审查。
- 在留カード 上信息不一致（住址 / 所属机构 / 姓名）= 永住申請 时被「申請事項違背」消极评价，住民税 / 年金 / 社保 履历追溯将不一致。
- 永住審査 通常 6-12 个月，护照 / 在留カード 有效期需覆盖审查全期。

## 判断标准
- **绿**：护照 > 12 个月有效 + 在留カード ≥ 6 个月 + 信息全部更新
- **黄**：护照 6-12 个月 OR 在留カード 3-6 个月 OR 信息更新状况不确定
- **红**：护照 < 6 个月 OR 在留カード < 3 个月 OR 信息错误

## 处理建议
### 绿
档案归档 护照号 + 在留カード 番号 + 各自有效期 + 直近一次更新日。永住申請 前 30 天复检。准备 永住申請者 优先确认 在留カード 是否为本签证种类的「最长档」。非最长档 → 申請前先续签到最长档。

### 黄
1. 护照 6-12 个月 → 在永住申請 前更新（审查 6-12 个月内护照需保持有效）。
2. 在留カード 3-6 个月 → 永住申請 与 在留期間更新申請 同时提出（実務上一般受理；审查中 在留期間 続行）。
3. 信息更新不确定 → 持卡到 出入国在留管理庁 / 入管局 確認现行记载，永住申請 资料中的住所历需与 住民票 一致。

### 红
1. 护照 < 6 个月 → 立即到本国大使馆 / 領事館 更新；多数大使馆需 1-3 周。永住申請 前必须完成。
2. 在留カード < 3 个月 → 本周内启动现签证续签，待续签完成 / 取得最长档后再提出永住申請。
3. 信息错误 → 持卡到 入管 申请「変更届」或「再交付申請」。永住申請 涉及住所履歴 / 所属机构履歴 全期追溯，错误履历会延长审查或导致不许可，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
