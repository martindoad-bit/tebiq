# 续签 / 永住 材料 准备进度

## 这是什么
永住許可申請 必备材料是 续签的 5-10 倍体量：① 会社侧（在職証明書 / 履歴事項全部証明書 / 源泉徴収票 直近 5 年）② 配偶侧 / 身元保証人 侧（戸籍謄本 / 在職証明書 / 課税証明書 / 身元保証書）③ 個人侧（直近 5 年 全部 課税 / 納税 / 年金 / 健康保険 履歴 + 在留カード / パスポート / 写真 / 理由書 / 履歴書 等）。永住 case 直近 5 年材料量大 1-3 个月准备时间必要。

## 为什么重要
- 永住申請 受理 = 审查期 6 个月 ~ 1 年。即使审查中 在留期間 到期，也需按现签证 续签 → 双轨进行。
- 材料量大 + 年度跨度 5 年 → 任何 1 项缺失 = 補正通知 拖延 1-2 个月。
- 直近 2 年（公的義務 履行）+ 直近 5 年（収入安定 / 居住継続）是审查核心，年金 / 健康保険 缴纳记录 缺一不可。

## 判断标准
- **绿**：D-90 ~ D-120 + 会社 / 身元保証人 侧 已开始 + 个人 5 年材料齐全
- **黄**：D-30 ~ D-90 OR 会社 / 身元保証人 / 个人 材料其中一类不确定
- **红**：D-30 内 OR 会社 / 身元保証人 侧未开始 OR 个人材料缺

## 处理建议
### 绿
档案归档完整清单 + 各项材料取得日期 + 直近 5 年材料 时序图。永住申请前 14 天复检（材料更新 / 年度切換）。

### 黄
1. 对照档案 + 永住 必备材料清单（在 TEBIQ 知识中心查看 永住申請材料文章）逐项核对。
2. 直近 5 年 課税 / 納税証明書 → 各年度 市役所，跨自治体 case 邮寄申请 1-2 周 / 通。
3. 年金 履歴 → 年金事務所 取「ねんきん定期便」或「被保険者記録照会回答票」直近 2 年。
4. 健康保険 履歴 → 国保 case 市役所、社保 case 会社 / 健康保険組合，直近 2 年。
5. 身元保証人 → 日本人 / 永住者 1 名（亲戚 / 同事 / 友人），需取得 在職証明書 / 課税証明書 / 住民票。

### 红
1. D-30 内永住未递交 → 优先按现签证 续签 + 同时 / 后续 申请永住（永住申请 不影响 现签证 续签）。
2. 会社侧 缺 → 立刻联系人事 + 提供清单（含 直近 5 年 源泉徴収票）。
3. 身元保証人 缺 → 联系候补人选 + 说明 身元保証人 不承担法律责任仅道义保証。
4. 永住申請 case 材料组合最复杂，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

**会社侧**：
- 在職証明書（職務内容 + 給与 + 雇用形态）
- 履歴事項全部証明書（雇主公司，3 个月以内）
- 雇用契約書（写）
- 源泉徴収票（直近 5 年）

**配偶侧 / 身元保証人 侧**：
- 配偶 戸籍謄本 / 結婚証明書（配偶签证 申請永住 case）
- 身元保証書（身元保証人 自筆 + 印鑑）
- 身元保証人 在職証明書 / 課税証明書 / 住民票

**个人侧（直近 5 年）**：
- 在留カード（正反面写）
- パスポート（写 + 出入国印章页 全部）
- 申請書（永住版，最新样式从 [出入国在留管理庁](https://www.moj.go.jp/isa/applications/procedures/) 下载）
- 課税証明書（直近 5 年）
- 納税証明書 其 1 / 其 2 / 其 3（直近 5 年）
- 年金 履歴（直近 2 年「被保険者記録照会回答票」）
- 健康保険 履歴（直近 2 年 缴纳记録）
- 住民票（世帯全員 + 続柄記載，3 个月以内）
- 写真 4×3cm（3 个月以内）
- 理由書（永住 希望理由，自筆推奨）
- 履歴書（学歴 + 職歴）
- 家族 写真（如有 配偶 / 子）
- 手数料 8,000 円分 収入印紙（許可时購入）

## 价值 expiry 说明
expiry_days = 14。永住材料准备期高频变化（直近 5 年 课税 / 納税 / 年金 / 健保 状况 + 会社 / 身元保証人 侧 進度），14 天内复检最稳。
