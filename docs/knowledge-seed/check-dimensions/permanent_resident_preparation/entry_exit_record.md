# 出入国记录 / 连续居住

## 这是什么
出入国記録 是 出入国在留管理庁 保存的本人入出境完整履历。再入国許可（包括 みなし再入国 1 年 / 正規 最長 5 年）= 在留資格 持有人出国期间不丧失资格的法定手续。永住者 同适用。

## 为什么重要
- 永住申請：「引き続き 10 年」（或高度人材 1 年 / 3 年、配偶者 3 年等短缩特例）要件 = 看出入国記録 而非物理时长。这是永住审查的核心要件之一。
- 单次出国 90 日以上 / 累计 100 日以上（年间）在审查中通常被视为「引き続き」中断或消极信号（待书士确认具体口径），可能直接导致重新累计。
- 全签证：出国时未办再入国許可 = 在留資格 自动失效，永住「引き続き」直接归零。

## 判断标准
- **绿**：1 年内单次出国 ≤ 90 日 + 累计 ≤ 100 日 + 全程办了再入国許可
- **黄**：天数 / 手续不确定
- **红**：单次或累计超 + 或未办手续

## 处理建议
### 绿
档案归档每次出入国时点 + ED カード 控（写）/ みなし选项确认。永住申請 提交前必须重新核对全部 10 年（或短缩特例对应年数）出入国記録。

### 黄
1. 向 [出入国在留管理庁](https://www.moj.go.jp/isa/) 申请「出入国記録に係る開示請求」（书面，回答 2-4 周）。永住申請 强烈建议在提交前以此客观文档自查。
2. 对照档案 出入国 履历 + 護照入出国印章。

### 红
1. 单次 / 累计超期 → 永住「引き続き」可能中断，需重新累计。出国天数较多者建议先与行政書士确认「引き続き」是否实质中断、自起算点重新累计起。
2. 未办再入国許可 → 在留資格 已失效，回国后只能以新申请，「引き続き」从零计。
3. 永住 是核心要件级判断，强烈建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 出入国記録 開示請求 結果（永住申請 自查必备）
- 護照（入出国印章页，含旧护照）
- 再入国許可書 控（写）/ ED カード 副本
- 在留カード（含「再入国許可」栏，如有正規许可）
- 出国理由说明书（长期出国时手写陈述书 + 客观证据）

## 价值 expiry 说明
expiry_days = 30。出入国 状况按出行随时变化，30 日内须复检；永住申請 关键节点前重新核对完整记录。
