# 課税証明書 / 納税証明書 准备

## 这是什么
課税証明書：证明该年度课税额（所得 + 控除项目 + 税额）。納税証明書：证明已缴 / 未缴金额（其 1 = 納付状況、其 2 = 所得金額、其 3 = 完納証明）。発行权属当年 1 月 1 日时点居住的 市役所 / 区役所 / 都税事務所。

## 为什么重要
永住許可申請 是 5 项书类中权重最高的「公租公課履行」核心材料。出入国在留管理庁 要求 直近 5 年完整 課税証明書 + 納税証明書（5 通 / 5 年），用以判断长期収入安定 + 税金按期缴纳。任何 1 年滞納 / 督促 / 遅延履歴 都会直接拒绝。住所変更 跨多市町村，需对应数量份。永住 是 課税証明書 维度最严苛的签证。

## 判断标准
- **绿**：明确知道需要 直近 5 年 + 知道取得窗口（多市町村）+ 各年 1/1 居住地清晰
- **黄**：不确定取得窗口或某年度 1/1 居住地
- **红**：完全不知道需要哪几年 / 跨多市町村未确认 / 有滞納履歴 → 需邮寄申请 + 清算（耗时极长）

## 处理建议
### 绿
档案归档 直近 5 年完整証明書（5 张課税 + 5 张納税）。永住申請 时直接调出。强烈建议每年新年度（6 月後）取得当年份归档。

### 黄
1. 持 在留カード 到所辖 市役所 / 区役所 申请，当天可拿。手数料约 300 円 / 通 × 10 通 ≈ 3000 円。
2. 多市町村場合：每个 1/1 居住地市町村各自取对应年份。
3. 代理人取需 委任状（本人署名 + 印鑑）。
4. 6 月切換新年度，前 1-5 月只能取直近 4 年（最新年度尚未確定）。
5. 永住申請 推荐 7 月以後 提交，可包含最新年度。

### 红
1. 5 年内住所変更跨多市町村 → 须分别到各旧居住地市町村申请。多数支持邮寄申請（公式样式 + 切手 + 返信用封筒），每个市町村耗时 1-2 周。
2. 任何年度有滞納履歴 → 立即到 市役所 / 税務署 完納清算 + 取得 完納証明書，且通常需"清算後 1 年以上"才稳妥再申永住。
3. 提前 60 日以上启动准备，永住 5 年材料量大且邮寄回收慢。
4. 跨多市町村 / 滞納履歴 / 海外滞在期間，建议咨询行政書士確認流程。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（本人確認）
- 印鑑（部分自治体可签名替代）
- 委任状（代理人申请）
- 旧住所証明（搬迁后 / 多市町村跨地区）
- 住民票除票（已迁出地确认）
- 完納証明書（滞納清算後）

## 价值 expiry 说明
expiry_days = 30。証明書 内容固定，但 6 月切換新年度 + 永住申請材料"3 个月内発行"要求双重约束，30 日内重检最稳。永住 5 年份須 同时点取得保持新鮮度。
