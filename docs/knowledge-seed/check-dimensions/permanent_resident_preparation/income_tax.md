# 所得税 / 確定申告 状况

## 这是什么
所得税 是个人就一年（1/1-12/31）所得缴纳的国税，所得税法 规定。給与所得者 通过 源泉徴収 + 年末調整 完成大多数情况；满足以下任一须自行 確定申告：年収 2000 万円超 / 副業 雑所得 20 万円超 / 不動産所得 / 海外汇款超 100 万円 / 経営者・自営業 等。申告期限：次年 2/16-3/15。永住申请直近 5 年完整缴纳记录是审查必要条件。

## 为什么重要
永住许可申请取 納税証明書 其 3（所得税额 + 完納事实），出入国在留管理庁 ガイドライン 明示「公的義務（納税・公的年金・公的医療保険）の履行」是 5 年期完整审查项。直近 5 年中任意年度 申告漏れ / 期限後申告 / 加算税 / 延滞税 → 永住审查直接消极，常见不许可理由。即使后来补缴，期限後 + 加算税 痕迹保留在 納税証明書 上。

## 判断标准
- **绿**：q1=no（无义务） 或 q1=yes + q2=yes + q3=no（按期完成且无补正）
- **黄**：q1 或 q2 unknown（不确定是否需做或是否完成）
- **红**：q1=yes + q2=no（漏报）OR q3=yes（已收处罚通知）

## 处理建议
### 绿
档案归档直近 5 年 確定申告書 控（写） + 源泉徴収票 + 納税証明書 其 1 / 其 2 / 其 3。永住申请前 90 天复检。直近 5 年連続無漏 是绿状态硬性条件。

### 黄
1. 登录 [e-Tax](https://www.e-tax.nta.go.jp/) 查申告履历，或到所辖 税務署 调取过去 5 年记录（永住审查需 5 年）。
2. 对照档案 已识别凭证（給与 / 副業 / 不動産 / 海外汇款）核对。
3. 任何 期限後申告 / 修正申告 痕迹 → 标记并准备说明書。

### 红
1. 漏报 → 期限後申告 立即补做，可减轻 無申告加算税（自首減額）；但 永住审查 痕迹保留，建议补缴后等 1-2 年再申请。
2. 涉及海外资产（5000 万円超）→ 国外財産調書 须同期补；涉及海外汇款 → 国外送金等調書 关联。
3. 永住申请前发现 申告漏れ → **不要先递永住**，先 期限後申告 + 完納，再评估递交时机。
4. 收到处罚通知不回应 = 升级 强制執行 + 永住决定的消极。建议咨询税理士 + 行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 確定申告書 控（直近 5 年）
- 源泉徴収票 / 給与支払報告書（直近 5 年）
- 納税証明書 其 1 / 其 2 / 其 3（直近 5 年，永住申请必须）
- 課税証明書（直近 5 年，市区町村）
- e-Tax 受信通知 / 利用者識別番号
- 副業 / 海外汇款 / 不動産 收支凭证

## 价值 expiry 说明
expiry_days = 90。確定申告 按年度走（次年 3/15 期限），但每年 6 月 課税証明書 切換后档案需复检。永住申请窗口期 90 天内 納税証明書 须重新取得。
