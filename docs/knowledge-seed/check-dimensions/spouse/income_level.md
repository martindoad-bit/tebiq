# 收入水平 / 独立生计

## 这是什么
入管法 在多种 在留資格 中要求「日本人と同等以上の報酬」或「独立生計」要件。具体数额因 在留資格 / 地域 / 業種 而异，没有全国统一线，但有事实上的下限标准。配偶者 在留 的「独立生計」要件按家族单位评估 —— 担保人（日本人配偶 / 永住者配偶）+ 申请人本人 的合算収入须能覆盖全家生活。担保人 年収 350 万円 以上一般较稳，本人若有自己的給与収入更稳定。

## 为什么重要
- 在留期間更新：配偶担保人収入下降 → 续签可能短期化（1 年）或不许可。
- 永住申請：日本人配偶者 / 永住者配偶者 申请 永住 时「独立生計」要件被严格审查，5 年家族合算 課税証明 是关键。
- 配偶死亡 / 离婚：配偶身份消失后 在留資格 取消风险，本人有独立収入更易转 定住者 等。

## 判断标准
- **绿**：q1 ≥ 中段 + q2 == yes + q3 != no（一致或更高 + 覆盖家族）
- **黄**：q1 偏低 OR q2/q3 unknown
- **红**：q1 = under_300 OR q2 == no OR q3 == no

## 处理建议
### 绿
档案归档直近 3 年 担保人 + 本人 源泉徴収票 / 給与明細 / 課税証明書。续签前 90 天复检。

### 黄
1. 对照档案 直近 3 年 担保人 + 本人 源泉徴収票 + 課税証明書 与申请时声明。
2. 担保人 年収 在 300-350 万円 + 子女多 → 申請理由書 中说明家计规划 / 储蓄 / 親族支援 等補足要素。
3. 本人在 育児休業 等导致収入空白 → 申請理由書 简单说明即可。

### 红
1. 担保人 年収 不足 300 万円 + 家族多 → 准备 預貯金 残高証明 / 親族 経済支援書 / 不動産 所有証明 等補足证据 + 申請理由書 说明家计実態。
2. 担保人 失业 / 重大収入变动 → 雇用契約書 / 雇用保険 受給資格者証 / 転職証明 + 申請理由書。
3. 永住 case 不足独立生計 → 等到家族合算 年収 稳定 1-2 年再递。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 担保人 + 本人 源泉徴収票（直近 3 年）
- 担保人 + 本人 給与明細（直近 12 个月）
- 担保人 + 本人 課税証明書（直近 3 年，含所得 + 控除项目）
- 雇用契約書 + 給与改定通知（如有变动）
- 預貯金 残高証明 / 親族 経済支援書（必要时）

## 价值 expiry 说明
expiry_days = 90。月薪 / 年収变化按月度走，但每年 6 月 課税証明書 切換期需重新上传新年度数据。
