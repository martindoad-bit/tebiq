# 住民票 / 搬家记录

## 这是什么
住民票 是市町村 住民基本台帳 系统中本人記録 的副本。中长期在留者（在留期間 3 个月以上）和特别永住者强制登载。搬家时：旧居住地 市役所 取 転出証明書 → 新居住地 14 日内 提交 転入届 → 在留カード 背面更新住址。

日本人配偶者 / 永住者配偶者 case，配偶 住民票 也要查；別居 / 単身赴任 状况须 申請理由書 说明。世帯 构成 + 配偶 住所 直接影响婚姻真实性审查，世帯主 / 続柄 记载 + 双方住址是否同址 是关键审查点。

## 为什么重要
- 入管法 第 19 条 7-9：住居地届出 14 日义务，违反 = 在留資格 取消事由（90 日以上不届出 = 直接取消）。
- 続签 / 永住 申請：调取 住民票 / 住民票記載事項証明書 验证 居住状况 + 世帯构成。
- 在留カード 与 住民票 不一致 = 银行口座 / 携带通信 / 公的手续 全部受阻。
- 配偶者ビザ：「世帯全員 + 続柄」記載的 住民票 是核心審查資料，別居住址 = 婚姻実体怀疑触发点。

## 判断标准
- **绿**：q1=no（无搬家） 或 q1=yes + q2=yes + q3=yes（已届出 + 一致）
- **黄**：q2 或 q3 unknown（届出 / 一致状况不确定）
- **红**：q1=yes + q2=no（未届出）OR q3=no（信息不一致）

## 处理建议
### 绿
档案归档直近一次「住民票（世帯全員 / 続柄 / 国籍 / 在留期間 全記載）」+ 旧住所変更履歴。续签前 90 天复检。配偶 住民票 同步取一份比对。

### 黄
1. 持 在留カード 到所辖 市役所 / 区役所 取「住民票（世帯全員）」+「住民票記載事項証明書」，当天可拿，手数料约 300 円 / 通。
2. 对照档案 已识别住址 与官方记载，差异时尽快 修正。
3. 配偶住址 不同时 → 准备 申請理由書 说明 単身赴任 / 介護 / 就労等具体事由。

### 红
1. 14 日 窗口已过 → 立即到 市役所 提交 転入届 / 住居地届出（市役所自动通过 「市町村 → 出入国在留管理庁」 通知系统更新 在留カード）。
2. 90 日 以上未届出 = 在留資格 取消事由（入管法 第 22 条の 4）→ 续签 / 永住 申請前必须先解决。
3. 在留カード 住址错误 → 持卡到 入管 申请「在留カード 記載事項に変更が生じた場合の届出」（一般同时通过 市役所 完成）。
4. 配偶 別居 + 未届出 双重情形 = 婚姻実体审查警戒级 → 申請理由書 + 同居実態 補足資料（共通光熱費 / 家計同一 等）档案归档。
5. 严重情形（多次搬家未届出 / 90 日以上 / 別居 + 信息不一致）建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 90。住民票 一般 3 个月内有效，续签材料 / 银行 / 公的手续 通常要求「3 个月以内発行」。配偶者 case 建议本人 + 配偶 同时取，确保世帯构成记载一致。
