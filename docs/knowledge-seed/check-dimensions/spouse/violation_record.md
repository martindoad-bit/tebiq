# 违规记录 / 罚金 / 入管警告

## 这是什么
日本签证审查中的「素行善良」要件，依据 入管法 / 国籍法。配偶者 在留資格 范围内，DV 嫌疑 / 偽装結婚 嫌疑 是配偶者 类签证特有的高敏感事项。範围包括：刑事处罚（罚金以上）/ 行政处罚（道交法违反、DV 接近禁止命令、风俗営業 違反 等）/ 入管法 上的警告 / 出頭通知 / 補正 通知 / 不許可 通知 / 在留資格 取消手続。

## 为什么重要
出入国在留管理庁 / 法務省 在 在留期間更新 / 永住申請 / 帰化申請 时通过 法務省内 共有データベース 调取记录。永住許可ガイドライン 明示「素行善良」为法定要件之一。重大违反 → 直接 不許可 / 取消。轻微违反 → 时间 + 全部信息开示后可能通过。

配偶者 在留資格 特有的取消事由：
- DV 嫌疑（配偶者暴力相談支援センター 介入 / 接近禁止命令）= 婚姻 实态 + 素行善良 双重消极。
- 偽装結婚 嫌疑（同居実態 不存在 / 金銭 授受 / 第三者 通報）= 入管 立件 → 在留資格 取消第一原因。
- 配偶者 与日本人/永住者 离婚 / 死別 6 ヶ月 经过未通知 = 入管法 19 条の16 申告義務 違反。

## 判断标准
- **绿**：q1=no + q2=no + q3=no（无任何记录）
- **黄**：q1=no + q2=no + q3=yes（仅有入管系警告 / 補正通知 / 不许可 / 偽装結婚 嫌疑调查 历史）
- **红**：q1=yes 或 q2=yes（有刑事 / 行政处罚）

## 处理建议
### 绿
档案保留「无记录」状态。同居 凭证（共同口座 / 公共料金 / 賃貸借契約 / 住民票 同一世帯）持续归档以对冲未来 偽装結婚 调查风险。下次续签前 90 天复检。即使无记录，仍须填写 申請書 「過去の犯罪歴」栏诚实申告。

### 黄
1. 整理过去的 不許可通知書 / 補正通知 / 警告 / 偽装結婚 嫌疑 调查记录原件 + 已采取的应对记录。
2. 续签 / 永住 申请书「申請理由書」中说明 + 申告 過去の経緯。
3. 不许可 / 補正 未回应 = 续签前必须先解决。
4. 同居 + 婚姻 实态 凭证（写真 / SNS 履历 / 共同口座 / 公共料金 / 旅行 同伴）档案化。

### 红
1. 罚金以上 → 不主动告知 = 不許可。判決書 / 略式命令 / 反則金 凭证档案归档。
2. DV 接近禁止命令 / 配偶者暴力 立件 = 婚姻関係 破綻 推定，在留資格 取消高风险。
3. 偽装結婚 嫌疑 立件 = 在留資格 取消 + 退去強制 高风险。
4. 風俗営業 違反 / 資格外活動 違反 = 在留資格 取消 + 配偶者 婚姻関係 信頼性 双重消极。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
