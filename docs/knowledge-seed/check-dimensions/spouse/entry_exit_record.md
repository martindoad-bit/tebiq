# 出入国记录 / 连续居住

## 这是什么
出入国記録 是 出入国在留管理庁 保存的本人入出境完整履历。再入国許可（包括 みなし再入国 1 年 / 正規 最長 5 年）= 在留資格 持有人出国期间不丧失资格的法定手续。永住者 同适用。

## 为什么重要
- 在留期間更新：配偶者 持有人长期出国 + 与配偶 不同居 → 婚姻 真実性 疑虑，续签 可能要求加补 同居証明 / 共同生活记录 / SNS 履历 / 通话记录等证据。
- 永住申請：「引き続き」要件 = 看出入国記録 而非物理时长。配偶者 走永住「引き続き 3 年」最短路径时，长期出国会直接打断累计。
- 全签证：出国时未办再入国許可 = 在留資格 自动失效。

## 判断标准
- **绿**：1 年内单次出国 ≤ 90 日 + 累计 ≤ 100 日 + 全程办了再入国許可
- **黄**：天数 / 手续不确定
- **红**：单次或累计超 + 或未办手续

## 处理建议
### 绿
档案归档每次出入国时点 + ED カード 控（写）/ みなし选项确认。续签 / 永住 前自查累计天数。建议同时保留婚姻 共同生活 痕迹（住民票世帯合併、共同口座、共同账单）。

### 黄
1. 向 [出入国在留管理庁](https://www.moj.go.jp/isa/) 申请「出入国記録に係る開示請求」（书面，回答 2-4 周）。
2. 对照档案 出入国 履历 + 護照入出国印章。

### 红
1. 单次 / 累计超期 → 永住「引き続き」可能中断，需重新累计。续签 也可能给较短期，且会触发婚姻真実性 加查。
2. 长期出国 + 不同居 → 准备 SNS / LINE / 通话 / 汇款 等远程交流证据，证明婚姻关系实质继续。
3. 未办再入国許可 → 在留資格 已失效，回国后只能以新申请。
4. 严重情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 出入国記録 開示請求 結果
- 護照（入出国印章页）
- 再入国許可書 控（写）/ ED カード 副本
- 在留カード（含「再入国許可」栏，如有正規许可）
- 婚姻真実性 补强：住民票（世帯全員）、共同口座流水、SNS / LINE 履历、共同生活照片

## 价值 expiry 说明
expiry_days = 30。出入国 状况按出行随时变化，30 日内须复检；永住申請 关键节点前重新核对完整记录。
