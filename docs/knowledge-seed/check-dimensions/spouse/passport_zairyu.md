# 护照 / 在留カード 有效期

## 这是什么
- 护照（パスポート）：本国大使馆 / 領事館 发行的身份证明 + 出入国 証明。日本 在留資格 持有期间须保持有效。
- 在留カード：出入国在留管理庁 发行的中长期在留者身份证明，载在留資格 / 期限 / 就労制限 等。日本人配偶者 / 永住者配偶者 在留期間 1 年 / 3 年 / 5 年，婚姻初期多为 1 年档，5 年后通常稳定到 3 年 / 5 年。

## 为什么重要
- 护照失效 → 出入国不可 / 续签材料无法核证 / 银行口座 等手续受阻。
- 在留カード 期限到 = 在留期間 同时到 → 不法滞在 风险。
- 在留カード 上信息不一致（住址 / 所属机构 / 姓名）= 续签时被「申請事項違背」消极评价。配偶者 持有者婚姻状态变化（离婚 / 配偶死亡）时，14 日以内需「配偶者に関する届出」，否则续签不利。
- 离婚后未届出 → 6 个月内未取得其他在留資格 = 在留資格取消 风险。

## 判断标准
- **绿**：护照 > 12 个月有效 + 在留カード ≥ 6 个月 + 信息全部更新
- **黄**：护照 6-12 个月 OR 在留カード 3-6 个月 OR 信息更新状况不确定
- **红**：护照 < 6 个月 OR 在留カード < 3 个月 OR 信息错误

## 处理建议
### 绿
档案归档 护照号 + 在留カード 番号 + 各自有效期 + 直近一次更新日。续签前 30 天复检。配偶者 持有者建议同步对照 戸籍謄本（日本人配偶者）/ 配偶者 在留カード（永住者配偶者） 上信息一致性。

### 黄
1. 护照 6-12 个月 → 在出国前更新（多数国家要求入境时 ≥ 6 个月）。
2. 在留カード 3-6 个月 → 续签申请窗口已开（D-90），开始准备材料：戸籍謄本 / 住民票（世帯全員）/ 配偶者 課税証明 / 納税証明。
3. 信息更新不确定 → 持卡到 出入国在留管理庁 / 入管局 確認现行记载。

### 红
1. 护照 < 6 个月 → 立即到本国大使馆 / 領事館 更新；多数大使馆需 1-3 周。
2. 在留カード < 3 个月 → 本周内启动续签，避免「申請受理票 + 特例期間」紧迫。
3. 信息错误 → 持卡到 入管 申请「変更届」或「再交付申請」。婚姻状态变更未届出 → 立即补「配偶者に関する届出」。涉及姓名变更 / 国籍变更 / 离婚后在留资格变更 → 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 护照（含申请取得日 / 有效期至）
- 在留カード（正反面）
- 「在留カード 記載事項に変更が生じた場合の届出」受理票（如有）
- 「在留カード 再交付申請」受理票（如有）
- 「配偶者に関する届出」受理票（如有）

expiry_days = 30。护照 / 在留カード 期限按月计算，30 天内变化对续签 / 出国 影响显著，必须高频复检。
