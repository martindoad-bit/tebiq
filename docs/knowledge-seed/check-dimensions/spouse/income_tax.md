# 所得税 / 確定申告 状况

## 这是什么
所得税 是个人就一年（1/1-12/31）所得缴纳的国税，所得税法 规定。日本人配偶者 / 永住者配偶者 多为 給与所得者，通过 源泉徴収 + 年末調整 完成大多数情况；满足以下任一须自行 確定申告：年収 2000 万円超 / 副業 雑所得 20 万円超 / 不動産所得 / 海外汇款超 100 万円 等。配偶控除（配偶年収 ≤103 万円）/ 配偶特別控除（≤201.6 万円）须在 申告 / 年末調整 中正确填写。

## 为什么重要
申告漏れ・無申告 / 過少申告 → 加算税 + 延滞税。重大違反追究刑责（所得税法第 238 条）。配偶签 续签 / 永住 审查中常调取 双方 納税証明書（夫 + 妻），日本人配偶担保 case 双方申告完整都要查；配偶控除 误用 → 修正申告 + 加算税。

## 判断标准
- **绿**：q1=no（无义务） 或 q1=yes + q2=yes + q3=no（按期完成且无补正）
- **黄**：q1 或 q2 unknown（不确定是否需做或是否完成）
- **红**：q1=yes + q2=no（漏报）OR q3=yes（已收处罚通知）

## 处理建议
### 绿
档案归档直近 3 年 確定申告書 控（写） + 源泉徴収票（夫 + 妻）。下次续签前 90 天复检。配偶控除 已申用记录核对。

### 黄
1. 登录 [e-Tax](https://www.e-tax.nta.go.jp/) 查申告履历，或到所辖 税務署 调取过去 3 年记录。
2. 对照档案 已识别凭证（給与 / 副業 / 不動産 / 海外汇款）核对。
3. 配偶年収 临近 103 万 / 201.6 万 阈值 → 重新核算配偶控除适用区间。

### 红
1. 漏报 → 期限後申告 立即补做，可减轻 無申告加算税（自首減額）。
2. 配偶控除 误用（配偶实际年収超阈值）→ 修正申告 + 補税；本人配偶双方都涉及。
3. 涉及海外资产（5000 万円超）→ 国外財産調書 须同期补；涉及海外汇款 → 国外送金等調書 关联。
4. 收到处罚通知不回应 = 升级 强制執行。建议咨询税理士 + 行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 確定申告書 控（直近 3 年，夫 + 妻）
- 源泉徴収票 / 給与支払報告書（双方）
- 納税証明書 其 1 / 其 2 / 其 3（直近 3 年）
- 配偶控除 / 配偶特別控除 关连凭证
- e-Tax 受信通知 / 利用者識別番号

## 价值 expiry 说明
expiry_days = 90。確定申告 按年度走（次年 3/15 期限），但每年 6 月 課税証明書 切換后档案需复检。配偶就职 / 退职 引发年収变动 → 控除区间需即时复算。
