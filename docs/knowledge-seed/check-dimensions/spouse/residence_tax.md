# 住民税 缴纳状况

## 这是什么
住民税 由 都道府県民税 + 市町村民税 构成（東京 23 区为「特別区民税 + 都民税」），地方税法 第 41 条以下规定。课税基准日为当年 1 月 1 日，按前年所得课税。来日次年 6 月起开始负担。

日本人配偶者等 / 永住者配偶者等 持有人本人的 住民税 缴纳是基本审查项；同时由于此类签证以 配偶（日本人 / 永住者）作为身份基础与生计担保，配偶 一方的 住民税 缴纳状态在续签时会作为「世帯単位」的稳定性参考被一并审视——尤其当 申請人本人无收入或低收入、由 配偶 担保生计 时。

## 为什么重要
公租公课 缴纳履历是 在留期間更新 / 永住申請 / 帰化申請 的核心审查项。出入国在留管理庁 在 永住許可ガイドライン（令和 8 年 2 月 24 日改訂版）中明示「公租公課按期缴纳」为 国益適合 要件之一，逾期缴纳「原则上消极评价」。配偶者 签证递交 在留期間更新 时，入管要求附 申請人本人 + 配偶 双方 的 課税証明書 / 納税証明書（最新年度），任一方有积欠都会拉低审查印象。

## 判断标准
- **绿**：本人 + 配偶 直近 1-2 年全部按期缴纳，无 督促状
- **黄**：本人 或 配偶 缴纳履历不确定，需到 市役所 取 納税証明書 比对
- **红**：本人 或 配偶 有 滞納 / 督促状 / 延滞金 / 差押予告 历史

## 处理建议
### 绿
档案归档本人 + 配偶 直近年度 課税証明書 + 納税証明書，下次续签前 90 天复检。

### 黄
1. 本人 持 在留カード、配偶 持 マイナンバーカード / 運転免許証 等身份证明，到 市役所 / 区役所 各自申请「納税証明書」（覆盖直近 3 年），手数料约 300 円 / 通。
2. 配偶 一方需 配偶 本人去取，或本人持「委任状」代办；不同自治体格式略异。
3. 对照档案 与官方记载差异。
4. 6 月切換新年度，前 1-5 月只能取去年度的；递交续签前注意时点。

### 红
1. 联系 市役所 確認本人 + 配偶 滞納金額 + 状態（一般徴収 / 督促 / 強制徴収）。
2. 递交续签 / 永住 申請前双方完納；困难时申请 分納計画。
3. 督促状 已发 / 差押予告 → 续签风险显著上升，配偶 一方滞納尤其影响「身元保証人」适格性。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 課税証明書（直近 3 年，本人 + 配偶）
- 納税証明書（直近 3 年，本人 + 配偶）
- 住民税 决定通知書（雇主代扣 / 普通徴収）
- 督促状 / 催告状（如有，作为风险记录）

## 价值 expiry 说明
expiry_days = 90。住民税 按月度走，状态稳定时 90 天内变化有限；超过 90 天若涉及年度切換（5-6 月）或 配偶 收入大幅变动需重检。
