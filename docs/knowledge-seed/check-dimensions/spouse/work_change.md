# 工作变更 / 所属机构届出

## 这是什么
入管法第 19-16 条规定，中长期在留者所属机构发生消灭、离脱、移籍时，事由发生 14 日内须向 出入国在留管理庁 提交届出。配偶者 在留資格 属於身份系，没有就労制限，但仍属「所属機関に関する届出」对象（如本人在职）。配偶者签证不要求工作内容契合特定领域，q3 主要核对是否存在双重雇用、虚假雇用等异常情形。

## 为什么重要
配偶者签证审查重点不在工作内容，而在 婚姻実態 + 生計要件。届出义务履行情况虽不像就労系那样直接关联岗位匹配，但作为公的義務履行 仍会被记录。续签审查会通过 課税証明書 + 雇用契約書 + 届出履历 综合判定生計の安定性—独立收入或配偶担保都需要稳定的雇用证据链。

## 判断标准
- **绿**：q1=no（无变更） 或 q1=yes + q2=yes + q3=yes（已届出 + 雇用关系真实）
- **黄**：q1=yes 但 q2 或 q3 unknown（届出记录不确定）
- **红**：q1=yes 且 q2=no（未届出）。q3=no 在配偶者 签证下罕见，但若涉及虚假雇用证明则按红处理

## 处理建议
### 绿
档案归档「最近一次届出受理票」+ 雇用契約書 副本 + 直近 課税証明書。下次续签前 90 天复检即可。

### 黄
1. 登录 [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 查届出受理记录。
2. 配偶者 签证下，无就労就无届出义务；如本人是專業主婦/夫 + 配偶担保生計，无需届出。
3. 对照 雇用契約書 与 給与振込記録，确保收入真实可证。

### 红
1. 14 日窗口已过 → 立即补办「所属機関に関する届出」。
2. 若长期未届出 + 续签临近，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
