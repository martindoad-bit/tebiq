# 续签 / 永住 材料 准备进度

## 这是什么
日本人配偶者 / 永住者配偶者 在留期間更新申請 必备材料分两部分：① 配偶侧（戸籍謄本（婚姻記載） / 結婚証明書（外国侧） / 質問書 / 共同生活照片 等）② 個人侧（在留カード / パスポート / 課税証明書 / 納税証明書 / 住民票（世帯全員） / 配偶 在職証明書 / 課税証明書 等）。質問書 双方写 + 婚姻経緯 + 共同生活 是审查重点。

## 为什么重要
- 入管 受理窗口 D-90 启动；最迟 D-Day 前递交可启动 特例期間。
- 配偶签证 case 婚姻実態 是核心审查点，質問書 / 共同生活照片 / 戸籍謄本 缺失 = 偽装結婚 怀疑 = 不許可 风险。
- 戸籍謄本 仅 配偶（日本人 / 永住者）能取，需配偶配合。

## 判断标准
- **绿**：D-90 ~ D-120 + 配偶材料 已开始 + 个人材料齐全
- **黄**：D-30 ~ D-90 OR 配偶 / 个人 材料其中一类不确定
- **红**：D-30 内 OR 配偶材料未开始 OR 个人材料缺

## 处理建议
### 绿
档案归档完整清单 + 各项材料取得日期。续签前 14 天复检（戸籍 / 質問書 状况）。

### 黄
1. 对照档案 + 本签证 必备材料清单（在 TEBIQ 知识中心查看 配偶签证 续签材料文章）逐项核对。
2. 戸籍謄本 → 配偶 本籍地 市役所，当天可拿（450 円 / 通）；本籍地远的 case 邮寄需 1-2 周。
3. 結婚証明書（外国侧）→ 中国 公証処 取得 + 翻译者 署名 翻译，1 个月程度。
4. 質問書 双方写 + 共同生活照片 推荐 5-10 张（婚礼 / 旅行 / 日常）。

### 红
1. D-30 内 → 即使材料未齐也优先到 入管 递交 申請受理票，启动 特例期間（最长 到期后 2 个月）。
2. 配偶侧 缺 → 立刻请配偶取 戸籍謄本（本籍地 市役所）+ 双方写 質問書。
3. 个人侧 缺（如 課税証明書 跨自治体）→ 邮寄申请，需提前 1-2 周。
4. 配偶签证 case 涉及 婚姻実態 説明，質問書 写法 / 共同生活照片 选择 影响审查结果，复杂情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

**配偶侧**：
- 戸籍謄本（婚姻記載，配偶 本籍地 市役所，3 个月以内）
- 結婚証明書（外国侧，中国 公証処 + 翻译者 署名 翻译）
- 質問書（双方写，婚姻経緯 / 知合経緯 / 共同生活）
- 共同生活照片（5-10 张，婚礼 / 旅行 / 日常）
- 配偶 在職証明書（収入面の安定性）
- 配偶 課税証明書 / 納税証明書（直近 1 年）
- 配偶 住民票（世帯全員）

**个人侧**：
- 在留カード（正反面写）
- パスポート（写 + 出入国印章页）
- 申請書（最新样式从 [出入国在留管理庁](https://www.moj.go.jp/isa/applications/procedures/) 下载）
- 課税証明書（直近 1 年）
- 納税証明書（其 1 / 其 2 / 其 3）
- 住民票（世帯全員 + 続柄記載，直近 3 个月以内発行）
- 写真 4×3cm（3 个月以内）
- 手数料 4,000 円分 収入印紙（許可时購入）

## 价值 expiry 说明
expiry_days = 14。续签材料准备期高频变化（戸籍 / 質問書 / 配偶材料 取得 進度），14 天内复检最稳。
