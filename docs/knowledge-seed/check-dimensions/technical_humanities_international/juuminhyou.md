# 住民票 / 搬家记录

## 这是什么
住民票 是市町村 住民基本台帳 系统中本人記録 的副本。中长期在留者（在留期間 3 个月以上）和特别永住者强制登载。搬家时：旧居住地 市役所 取 転出証明書 → 新居住地 14 日内 提交 転入届 → 在留カード 背面更新住址。

技人国 case，搬家后 通勤距離 显著变化时 雇用契約 + 通勤手当 也需更新，雇用契約書 上「就業場所」与 住民票 住址不一致会影响续签时的雇用実態 审查。

## 为什么重要
- 入管法 第 19 条 7-9：住居地届出 14 日义务，违反 = 在留資格 取消事由（90 日以上不届出 = 直接取消）。
- 続签 / 永住 申請：调取 住民票 / 住民票記載事項証明書 验证 居住状况 + 世帯构成。
- 在留カード 与 住民票 不一致 = 银行口座 / 携带通信 / 公的手续 全部受阻。

## 判断标准
- **绿**：q1=no（无搬家） 或 q1=yes + q2=yes + q3=yes（已届出 + 一致）
- **黄**：q2 或 q3 unknown（届出 / 一致状况不确定）
- **红**：q1=yes + q2=no（未届出）OR q3=no（信息不一致）

## 处理建议
### 绿
档案归档直近一次「住民票」（含 世帯全員 / 国籍 / 在留期間）+ 旧住所変更履歴。续签前 90 天复检。雇用契約書「就業場所」与 住民票 住址同步核对。

### 黄
1. 持 在留カード 到所辖 市役所 / 区役所 取「住民票」+「住民票記載事項証明書」，当天可拿，手数料约 300 円 / 通。
2. 对照档案 已识别住址 与官方记载，差异时尽快 修正。

### 红
1. 14 日 窗口已过 → 立即到 市役所 提交 転入届 / 住居地届出（市役所自动通过 「市町村 → 出入国在留管理庁」 通知系统更新 在留カード）。
2. 90 日 以上未届出 = 在留資格 取消事由（入管法 第 22 条の 4）→ 续签 / 永住 申請前必须先解决。
3. 在留カード 住址错误 → 持卡到 入管 申请「在留カード 記載事項に変更が生じた場合の届出」（一般同时通过 市役所 完成）。
4. 严重情形（多次搬家未届出 / 90 日以上）建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

- 住民票記載事項証明書
- 転入届 / 転出届 受理票

- 雇用契約書（就業場所记载，确认与住址匹配）

## 价值 expiry 说明
expiry_days = 90。住民票 一般 3 个月内有效，续签材料 / 银行 / 公的手续 通常要求「3 个月以内発行」。
