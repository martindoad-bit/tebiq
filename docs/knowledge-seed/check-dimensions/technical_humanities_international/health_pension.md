# 健康保険 / 年金 缴纳状况

## 这是什么
日本「国民皆保険」/「国民皆年金」体制。雇员体系：社会保険（健康保険 + 厚生年金 一套打包）。其他居住者：国民健康保険 + 国民年金。20-60 岁 + 住所在日 强制加入年金，3 个月以上居留强制加入医疗保険。

技人国 持有者通常受雇于日本企业，按规则应加入 社会保険（健康保険 + 厚生年金），保険料从工资天引（源泉徴収）。

## 为什么重要
未缴 / 滞納 → 督促状 → 催告状 → 強制徴収（給与差押 / 預金差押）。出入国在留管理庁 续签 / 永住・帰化 审查取「直近 2 年缴纳记录」。2024-06-21 公布的入管法改正法（法律第 60 号）将「故意不缴 公租公課」追加为永住資格 取消事由（施行日由政令定，预期 2027 内）。

技人国 续签时，入管 审查重点是「公的義務 履行状況」。试用期未加入 社会保険 的灰色期、转职空白期 自己未补办 国保，都会形成档案缺口。

## 判断标准
- **绿**：q2=yes + q3=no（按期 + 无异常）
- **黄**：q2 或 q3 unknown
- **红**：q2=no（漏缴） OR q3=yes（有滞納 / 督促 / 強制徴収予告）

## 处理建议
### 绿
档案归档：ねんきん定期便（年度）+ 健康保険証 + 直近 12 个月给与明细（標準報酬月額 可见）。下次续签前 90 天复检。

### 黄
1. [ねんきんネット](https://www.nenkin.go.jp/n_net/) 查个人年金缴纳记录。
2. 公司人事 / 协会けんぽ 取「健康保険 加入証明」。
3. 试用期 / 转职空白 期间若曾走 国保 + 国民年金，市役所 国保課 取「保険料 納付済通知書」。

### 红
1. 経济困難 → 申请 国民年金 保険料免除制度（全額・3/4・半額・1/4）/ 納付猶予制度。被认可的免除期间不算消极评价。
2. 単純漏缴 → 立即补缴 + 索取 納付済通知書。
3. 強制徴収予告 阶段 = 续签 高风险。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- ねんきん定期便 / ねんきんネット 履历
- 健康保険証
- 厚生年金 標準報酬月額 通知書（社会保険）
- 国民健康保険料 納付済通知書（市役所，转职空白期适用）
- 免除 / 猶予 認定通知書（如适用）

## 价值 expiry 说明
expiry_days = 90。月度缴纳，90 天内变化有限。但年度 課税基礎 切換期（6 月）若有变需复检。转职 / 离职 / 入社新公司时立即重检。
