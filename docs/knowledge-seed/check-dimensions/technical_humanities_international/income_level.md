# 收入水平 / 独立生计

## 这是什么
入管法 在多种 在留資格 中要求「日本人と同等以上の報酬」或「独立生計」要件。具体数额因 在留資格 / 地域 / 業種 而异，没有全国统一线，但有事实上的下限标准。技人国 在「日本人と同等以上の報酬」原则下被审查 —— 同職種 同年次 日本人社員 的报酬作为对照基准；东京 / 大阪 等大都市圈实务上月給 20 万円 以上是普遍下限，地方圈可能更低但仍须能说明合理性。

## 为什么重要
- 在留期間更新：收入与原认定标准对比，下降 → 续签可能短期化或不许可。
- 转职 case：新公司 給与 低于前职或低于行业平均 → 入管 在 在留資格変更 / 更新 时审查「同等以上」要件。
- 永住申請：法定「独立生計」要件，5 年内 課税証明 显示稳定收入是关键。

## 判断标准
- **绿**：q1 ≥ 中段 + q2 == yes + q3 != no（一致或更高 + 覆盖家族）
- **黄**：q1 偏低 OR q2/q3 unknown
- **红**：q1 = under_300 OR q2 == no OR q3 == no

## 处理建议
### 绿
档案归档直近 3 年 源泉徴収票 / 給与明細 / 課税証明書 / 雇用契約書。续签前 90 天复检。

### 黄
1. 对照档案 直近 3 年 源泉徴収票 + 課税証明書 与申请时声明。
2. 收入波动正常（如奖金 / 副業 切换 / 试用期低薪） → 申請理由書 简单说明即可。
3. 地方圈 + 短期试用期低薪需在 申請理由書 中说明业界惯例 / 同社日本人社員 報酬 对比。

### 红
1. 收入显著下降 → 准备客观证据（雇用契約書 / 給与变动通知）+ 申請理由書 说明原因（部门变动 / 经济周期 / 业务调整 等）。
2. 月給 不足 20 万円 + 大都市圏 case → 准备 同社日本人社員 報酬規定 / 業種 平均年収 数据补强「同等以上」论证。
3. 配偶担保 case 担保人收入不足 → 评估自己独立収入或本人转 在留資格。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 源泉徴収票（直近 3 年）
- 給与明細（直近 12 个月）
- 課税証明書（直近 3 年，含所得 + 控除项目）
- 雇用契約書 + 給与改定通知（如有变动）
- 同社日本人社員 報酬規定（必要时）

## 价值 expiry 说明
expiry_days = 90。月薪 / 年収变化按月度走，但每年 6 月 課税証明書 切換期需重新上传新年度数据。
