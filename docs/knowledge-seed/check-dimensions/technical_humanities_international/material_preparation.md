# 续签 / 永住 材料 准备进度

## 这是什么
在留期間更新申請 必备材料分两部分：① 会社侧（在職証明書 / 履歴事項全部証明書 / 雇用契約書（写） / 源泉徴収票 等）② 個人侧（在留カード / パスポート / 課税証明書 / 納税証明書 / 住民票 等）。技人国 case 还需 学歴 与岗位相关性 的说明（毕业证 / 成績証明書 写）。会社材料一般需 2-3 周，市役所材料 当天可拿但需对应年度。

## 为什么重要
- 入管 受理窗口 D-90 启动；最迹 D-Day 前递交可启动 特例期間。
- 材料不齐 → 補正通知（来回 2-4 周）→ 拖到原 在留期間 后 = 风险。
- 技人国 case 学歴・職務 整合性 是审查重点，学歴材料缺失或与岗位不匹配会触发追加资料要求。

## 判断标准
- **绿**：D-90 ~ D-120 + 会社材料 已请 + 个人材料齐全
- **黄**：D-30 ~ D-90 OR 会社 / 个人 材料其中一类不确定
- **红**：D-30 内 OR 会社未请 OR 个人材料缺

## 处理建议
### 绿
档案归档完整清单 + 各项材料取得日期。续签前 14 天复检（材料更新状况）。

### 黄
1. 对照档案 + 本签证 必备材料清单（在 TEBIQ 知识中心查看 技人国 续签材料文章）逐项核对。
2. 会社材料 一般 2-3 周，提前 1 个月请人事开始；规模大的公司可能 1 个月以上。
3. 市役所材料（住民票 / 課税証明書 / 納税証明書）当天可拿，但 6 月切換新年度时点要注意。
4. 学歴材料（毕业证 / 成績証明書 写）：海外大学需 翻译者 署名翻译。

### 红
1. D-30 内 → 即使材料未齐也优先到 入管 递交 申請受理票，启动 特例期間（最长 到期后 2 个月）。
2. 会社侧 缺 → 立刻联系人事 + 提供详细清单。建议按 在職証明書 → 履歴事項全部証明書 → 源泉徴収票 → 雇用契約書（写）顺序请。
3. 个人侧 缺（如 課税証明書 跨自治体）→ 邮寄申请，需提前 1-2 周。
4. 材料涉及外国语翻译（学歴証明 等）→ 须 翻译者 署名。复杂情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料

**会社侧**：
- 在職証明書（職務内容 + 給与 + 雇用形态）
- 履歴事項全部証明書（雇主公司，3 个月以内）
- 雇用契約書（写）/ 給与改定通知（如有）
- 源泉徴収票（直近 1 年）

**个人侧**：
- 在留カード（正反面写）
- パスポート（写 + 出入国印章页）
- 申請書（最新样式从 [出入国在留管理庁](https://www.moj.go.jp/isa/applications/procedures/) 下载）
- 課税証明書（直近 1 年；职位 / 收入 大幅变动 case 可能 3 年）
- 納税証明書（其 1 / 其 2 / 其 3）
- 住民票（直近 3 个月以内発行）
- 写真 4×3cm（3 个月以内）
- 学歴証明書 / 成績証明書 写（与岗位相关性说明 case）
- 手数料 4,000 円分 収入印紙（許可时購入）

## 价值 expiry 说明
expiry_days = 14。续签材料准备期高频变化（市役所材料新発行 / 会社材料处理进度），14 天内复检最稳。
