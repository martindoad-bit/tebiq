# 住民税 缴纳状况

## 这是什么
住民税 由 都道府県民税 + 市町村民税 构成（東京 23 区为「特別区民税 + 都民税」），地方税法 第 41 条以下规定。课税基准日为当年 1 月 1 日，按前年所得课税。来日次年 6 月起开始负担。

技人国 多为给与所得者，主流为 特別徴収（公司从月薪代扣），切换公司、入职 / 退職 当月或转 普通徴収 时段最易出现漏缴。

## 为什么重要
公租公课 缴纳履历是 在留期間更新 / 永住申請 / 帰化申請 的核心审查项。出入国在留管理庁 在 永住許可ガイドライン（令和 8 年 2 月 24 日改訂版）中明示「公租公課按期缴纳」为 国益適合 要件之一，逾期缴纳「原则上消极评价」。技人国 续签时同样会被纳入审查考量。

## 判断标准
- **绿**：直近 1-2 年全部按期缴纳，无 督促状
- **黄**：缴纳履历不确定，需到 市役所 取 納税証明書 比对
- **红**：有 滞納 / 督促状 / 延滞金 / 差押予告 历史

## 处理建议
### 绿
档案归档直近年度 課税証明書 + 納税証明書，下次续签前 90 天复检。

### 黄
1. 持 在留カード 到 市役所 / 区役所 申请「納税証明書」（覆盖直近 3 年），手数料约 300 円 / 通。
2. 对照档案 与官方记载差异。
3. 6 月切換新年度，前 1-5 月只能取去年度的；递交续签前注意时点。

### 红
1. 联系 市役所 確認滞納金額 + 状態（一般徴収 / 督促 / 強制徴収）。
2. 递交续签 / 永住 申請前完納；困难时申请 分納計画。
3. 督促状 已发 / 差押予告 → 续签风险显著上升。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 課税証明書（直近 3 年）
- 納税証明書（直近 3 年）
- 住民税 决定通知書（雇主代扣 / 普通徴収）
- 督促状 / 催告状（如有，作为风险记录）

## 价值 expiry 说明
expiry_days = 90。住民税 按月度走，状态稳定时 90 天内变化有限；超过 90 天若涉及年度切換（5-6 月）需重检。
