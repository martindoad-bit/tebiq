# 违规记录 / 罚金 / 入管警告

## 这是什么
日本签证审查中的「素行善良」要件，依据 入管法 / 国籍法。範围包括：刑事处罚（罚金以上）/ 行政处罚（道交法违反、风俗営業 等）/ 入管法 上的警告 / 出頭通知 / 補正 通知 / 不許可 通知 / 在留資格 取消手続。

## 为什么重要
出入国在留管理庁 / 法務省 在 在留期間更新 / 永住申請 / 帰化申請 时通过 法務省内 共有データベース 调取记录。永住許可ガイドライン 明示「素行善良」为法定要件之一。重大违反 → 直接 不許可 / 取消。轻微违反 → 时间 + 全部信息开示后可能通过。

技人国 持有人常见消极类型：
- 道交法 飲酒運転 / 当て逃げ / 二重制限速度 等被赤切符 → 罚金以上 → 自动进入刑事统计。
- 風俗営業 范围内劳动（含 资格外活動 在 風俗 店家上班）= 入管法 违反 + 在留資格 取消高风险。
- 雇主 不適正受入 被 労基署 / 入管 立件，本人作为关系人也可能被 出頭通知。

## 判断标准
- **绿**：q1=no + q2=no + q3=no（无任何记录）
- **黄**：q1=no + q2=no + q3=yes（仅有入管系警告 / 補正通知 / 不许可 历史）
- **红**：q1=yes 或 q2=yes（有刑事 / 行政处罚）

## 处理建议
### 绿
档案保留「无记录」状态。下次续签前 90 天复检。即使无记录，仍须填写 申請書 「過去の犯罪歴」栏诚实申告。

### 黄
1. 整理过去的 不許可通知書 / 補正通知 / 警告 文件原件 + 已采取的应对记录。
2. 续签 / 永住 申请书「申請理由書」中说明 + 申告 過去の経緯。
3. 不许可 / 補正 未回应 = 续签前必须先解决。

### 红
1. 罚金以上 → 不主动告知 = 不許可。判決書 / 略式命令 / 反則金 凭证档案归档。
2. 道交法重大违反（飲酒運転 / 当て逃げ / 二重制限速度 等）= 即使已完納，5 年内审查仍消极评价。
3. 風俗営業 違反 / 資格外活動 違反 等 = 在留資格 取消风险，技人国 续签可能直接 不許可。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 价值 expiry 说明
expiry_days = 90。违反记录是历史事实，状态稳定；但新发生的违反在 90 天内会出现在档案中需复检。
