# 工作变更 / 所属机构届出

## 这是什么
入管法第 19-16 条规定，中长期在留者所属机构发生消灭、离脱、移籍时，事由发生 14 日内须向 出入国在留管理庁 提交「契約機関に関する届出」。技人国 持有者的雇用主变更、公司被并购、公司更名、转籍至关联会社、被派遣至其他公司等情形均属届出对象。

## 为什么重要
技人国 在留資格 严格对应「自然科学 / 人文科学 / 国际业务」三领域。届出义务履行情况是续签审查的素行善良 + 公的義務履行 评价项。岗位偏离原 在留資格 认定的活动范围（例如技人国 持有者去做现场体力劳动或单纯店员），会被视为資格外活動 风险，续签时即便届出齐备也可能不许可。

## 判断标准
- **绿**：q1=no（无变更） 或 q1=yes + q2=yes + q3=yes（已届出 + 岗位匹配三领域）
- **黄**：q1=yes 但 q2 或 q3 unknown（届出记录或岗位匹配性不确定）
- **红**：q1=yes 且 q2=no（未届出）或 q3=no（岗位偏离自然科学 / 人文科学 / 国际业务范围）

## 处理建议
### 绿
档案归档「最近一次届出受理票」+ 雇用契約書 副本 + 新公司「業務内容説明書」。下次续签前 90 天复检即可。

### 黄
1. 登录 [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 查届出受理记录。
2. 对照原 在留資格 认定的活动范围（在留カード 上「就労制限」栏 + 当初许可的職務内容），比对当前 雇用契約書 与 業務内容。
3. 不确定新岗位是否仍属技人国 三领域时，可先取得 就労資格証明書 事前确认。

### 红
1. 14 日窗口已过 → 立即补办「契約機関に関する届出」（事業所所在地 / 名称变更 / 消灭 / 离脱 / 移籍 等分别填报）。
2. 岗位偏离 → 评估是否需要 在留資格変更 或 就労資格証明書 事前确认。当前活动若属於「現場作業」「販売業務」等非技人国 范围，不可继续从事。
3. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（含「就労制限」栏）
- 雇用契約書（前公司离职 + 新公司入职）
- 在職証明書 / 退職証明書
- 届出受理票（電子届出 PDF 或纸质回執）
- 就労資格証明書（如已申请）
- 新公司 登記事項証明書 + 直近年度 決算文書（续签时使用）

## 价值 expiry 说明
expiry_days = 90。续签材料准备期一般 3-5 个月前启动，90 天能覆盖整个准备窗口；超过 90 天若工作未变，状态仍可信。
