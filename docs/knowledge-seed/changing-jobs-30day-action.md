# 我刚签新公司 offer letter，从今天起 30 天行动表

## 这事跟我有关吗

如果你符合以下条件，请继续往下看：

- 现在持有 技人国 / 経営・管理 / 高度専門職 等 就労系 在留資格
- 已签下新公司 offer letter（包括同业跳槽 / 异业跳槽 / 创业转入新法人）
- 旧公司离职日已确定，或已经在交接期

不在这个范围内的（例：家族滞在、留学、永住者）请看其他条目。

## 这是什么

換職 涉及两件事：① 对入管的 届出 義務（不做会影响下次更新的信用），② 新岗位与你现有 在留資格 的 適合性 判定（不契合则要变更或被否定）。这两件事在时间上是并行的，所以下面按"距离 offer 签订日"切成 4 段。

## 你应该做什么

### Day 0–1（offer 签订当天 ～ 次日）

- 把旧公司的 退職予定日 写进日历
- 准备旧公司的 退職証明書 / 離職票 申请通道（後で必要）
- 不需要立即向入管报告——届出 義務 从离职日起算 14 日

### Day 1–7（旧公司离职后）

- **離脱届（旧 契約機関に関する届出）**：旧公司离职日起 14 日内，向入管提交。可以在线（電子届出）或 郵送 / 窓口
- 同步把 退職証明書 / 源泉徴収票 收齐——下次 更新 / 変更 都用得上

### Day 1–7（新公司入社后）

- **新 契約機関に関する届出**：新公司入社日起 14 日内提交
- 新公司的 雇用契約書 / 労働条件通知書 留底——岗位职务、月薪、雇用形態 必须与上次 在留資格 申请时申报的活动 一致 或在合理范围内变化

### Day 7–14：评估新岗位 vs 现有 在留資格

判断岗位是否仍属于 在留資格 范围内：

- 同职种、同业种 → 一般问题不大
- 跨职种（例：技术 → 销售管理）/ 跨资格（例：技人国 → 経営・管理）→ 强烈建议申请 **就労資格証明書**

就労資格証明書 是入管对"新岗位是否可以用现 在留資格 干"的事前确认书。审查 1–3 月，没有也能干，但有了之后下次 更新 几乎不会被驳。

### Day 14–30：把后台切干净

- **健康保険 / 厚生年金**：旧公司退会 → 新公司加入 之间如果有空白，要在 14 日内到市区町村 加入国民健康保険 / 国民年金；新公司入社后再切回 社保
- **住所変更**：如果换工作伴随搬家，14 日内 转出 / 转入届
- **収入証明**：明年 6 月 課税証明書 出来之前，新旧两份 源泉徴収票 都要留好——下次 更新 入管会看连续性
- **マイナンバー**：新公司提交时用

## 不做的后果

- 届出 過怠：14 日内不报，最高 20 万円 過料 + 下次 更新 信用扣分
- 新岗位不契合却没申请 就労資格証明書：下次 更新 时入管发现实际工作内容偏离，可能不予更新，等于让你 90 天内离境
- 健保 / 年金 空白期未补：再续签时 納付状況 不连续，経営・管理 / 高度専門職 受影响最大
- 住所变更不报：在留カード 上的住所失效，相关通知收不到

## 何时找专家

下面任一情况，请联系 行政書士 / 入管専門 弁護士：

- 新岗位与原 在留資格 跨范围（技人国 → 経営・管理 / 自主創業）
- 高度専門職 持有人换工作（要重新算 ポイント、可能要重新申请）
- 月薪从原职位明显下降（入管会问"为什么")
- 新公司规模过小 / 成立不满 1 年 / 没有 决算書（入管会要求 立証 強化）
- 就労資格証明書 申请被退回或追加资料

書士 在这种节点的价值不是替你跑流程，而是判断你下次 更新 是否会被卡，以及现在要不要顺手做 在留資格変更。
