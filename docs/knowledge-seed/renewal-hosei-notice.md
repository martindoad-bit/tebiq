# 我收到入管局补正通知，要补什么 / 怎么回应

## 这事跟我有关吗？

你递了 在留期間更新 / 変更 / 永住 申請后，过了几周收到一份信，标题写着「資料提出通知書」「補正通知」「追加資料の依頼」之类的——这就是 補正通知。

很多人第一反应是慌：是不是被拒了？

**不是。補正通知 ≠ 不許可。** 它意味着：审查官在认真看你的材料，但缺一块拼图，让你补上。审查仍在进行中。

但**不回应 = 视为撤回 / 直接不許可**。所以这事很急。

## 这是什么

補正通知 的常见形式：

- **邮寄信件**（最普遍）：白色信封，发件人「○○出入国在留管理局」
- **电话**：审查官直接打你登记的电话
- **邮件**（少数）：在线申請 的话，通过 在留申請オンラインシステム 通知
- **窗口口头**：递交时审查官当场指出（这种不算正式 補正通知，是受理前的事前确认）

通知里会有：

1. 你的申请受理号
2. 缺什么材料（具体到文档名）
3. 补的方式（邮寄 / 窗口持参 / 在线上传）
4. 期限（多数 2-4 周内，最常见是发信日起 14 日 / 1 个月）
5. 入管 担当者 的内线 / 邮件

最常被要求补的材料 Top 5：

| 材料 | 谁开 | 取得时间 |
|---|---|---|
| 課税・納税証明書（追加年度） | 市役所 | 当天 |
| 在職証明書（更新版 / 详细版） | 公司人事 | 3 天-1 周 |
| 履歴事項全部証明書（公司谋本） | 法务局 / オンライン | 当天 |
| 雇用契約書 写し | 公司 | 当天 |
| 申請理由書（更详细版） | 自己 | 当天 |
| 預金残高証明書 | 银行 | 当天-3 天 |

## 你应该做什么

**第 1 步（收到当天）：核对要补什么**

- 把通知书读 3 遍。日文写得很正式，但要点清楚
- 不确定 → 打通知里的内线电话，用日文问"○○の件で確認したいことがあります"，担当者会解释
- 不行就找会日文的朋友 / 行政書士 翻译

**第 2 步（48 小时内）：评估难易**

- 简单（市役所 一张证明）→ 直接去取
- 中等（公司开的证明）→ 联系人事，告诉他们紧急
- 复杂（要重写 申請理由書 / 财务说明）→ 当天就开始写

**第 3 步（期限的一半之内）：递交**

回应方式（按通知里要求）：

- **窗口持参**：拿原件 + 复印件 + 申請受理票 + 通知书原件去入管
- **邮寄**：挂号送，信封外注明「在留期間更新 補正資料 在中」+ 申請番号
- **在线上传**：在留申請オンラインシステム 中你的申請详情页有"資料追加"按钮

**重要**：每次补完材料，**审查重新计时**。所以补正越快，下结果越快。

**第 4 步（保留证据）**

- 邮寄寄出收据 / 窗口收的 受領書 / 在线上传截图，全部留底
- 万一寄丢，能证明你按期回应了

## 不做的后果

- 期限过了不回应 → 視為撤回（实务上）或 直接不許可
- 不許可的理由 = "材料不足"，再申請时这次的不利记录会被参考
- 期间到期日 + 補正期限重合时 → 处理顺序错乱可能转 不法滞在
- 一次申請被退后再续，往往年限缩短（5 年 → 1 年）

## 什么时候你应该找专家

以下情况强烈建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）：

- 通知里写的日文你看不懂确切要什么
- 要求补「申請理由書を更に詳細に」之类的开放问题
- 入管 怀疑你"虚偽申告"，要补反驳材料
- 公司不配合开 在職証明書 / 履歷事項全部証明書
- 你换过工作，要补新雇主的全套材料
- 期限只剩 1 周

専家能在 48 小时内帮你梳理回应策略，避免"补错方向再被补正一次"。
