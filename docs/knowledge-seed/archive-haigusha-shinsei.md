# 配偶者 签证审查：档案准备的角色

## 这事跟我有关吗？
你正在申请『日本人の配偶者等』或『永住者の配偶者等』在留資格——无论是配偶来日的 在留資格認定証明書（COE）、还是从其他在留資格 变更、或常规更新。这类申请的特殊之处在于：审查的不是「你的工作能力」或「你的纳税记录」，而是「这段婚姻是否真实」。判断真实性的材料分散在过去几年的生活痕迹里。档案的作用，是让这些痕迹在申请前就以可检索的形式存在，而不是临时翻手机相册凑数。

## 这是什么
入管 对配偶者签证的真实性审查重点：

- **年齢差**：差距大（如 15 岁以上）会增加审查问询
- **相识时间到结婚时间**：过短（如 3 个月内）触发更严格审查
- **同居证据**：住民票 上记载同一住所、共同住所的写真、賃貸借契約書 是否双方姓名
- **共同生活痕迹**：共同口座 的入出金履歴、生活費 送金記録、共同的水道光熱費契約
- **SNS 历史**：相识到结婚期间的聊天记录、双方互发的写真
- **質問書 内容一致性**：双方独立填写 質問書 的回答是否在事实层面一致

档案需保存的事件与材料：

- 双方往来照片（按时间顺序，标注地点 / 场合）
- 双方共同住所的 住民票（記載事項証明）
- 共同口座 通帳 副本 / 共同名義 的賃貸借契約書
- 生活費 送金記録（双方互转、或单方负担生活费的银行履历）
- 婚姻届 受理証明書 / 戸籍謄本（若日本人配偶）/ 結婚証明書（中国民政部）
- SNS / メッセージ 聊天记录（关键节点截图，如求婚、家族介绍）
- 結婚式 写真 / 招待状（若有）
- 双方家族介绍场景（餐厅照、家访照）
- 質問書 双方各自的草稿 + 最终提交版

入管 質問書 的核心问项：

- 相识経緯（何时、何地、通过什么方式）
- 交往期间的同居 / 异地情况
- 結婚式 是否举办、规模、出席者
- 双方家族介绍的具体场景与时间
- 婚后的生活分担（家务、家计）
- 本国家族的反応 / 同意

## 你应该做什么
1. **婚姻届 提交后立即**：在档案中开「配偶者 在留資格 准备」事件，把 受理証明書 / 戸籍謄本 / 結婚証明書 三件套入档。
2. **婚后 3 个月内**：把交往到结婚的所有关键节点照片按时间排序入档，每张标注日期 + 地点。共同口座 / 共同住所 / 共同水道光熱費 的契约一旦成立立刻归档。
3. **COE / 変更 申请前 30 天**：从档案调出材料清单与入管「立証資料の例」逐条比对——同居資料、交際 経緯、生活費 負担、家族交流 四类至少各有 3-5 件具体材料。
4. **質問書 填写时**：双方分别参照档案的事件时间线独立填写，重点核对相识経緯、結婚式、家族介绍三段事实层面是否一致。
5. **审查中收到 補正通知 / 追加資料要求**：从档案中调取已有但未提交的材料补充，避免临时拍照导致「材料前后矛盾」。

## 不做的后果
配偶者签证审查的不许可率高于其他在留資格，主因不是材料数量不够，而是材料无法呈现「真实婚姻」的连续轨迹。临时凑材料会出现：照片只有结婚后的、没有交往期的；質問書 双方对相识時期的描述差 3 个月；共同口座 是申请前 1 周才开的——这些会被入管标记为「偽装結婚 嫌疑」。

一旦被認定 偽装：

- 在留資格 取消（入管法第 22 条の 4）
- 5-10 年 上陸拒否
- 公正証書原本不実記載罪（刑法第 157 条）：5 年以下懲役 或 50 万円以下罰金
- 日本人配偶 同样可能被追究刑事责任

档案不能制造不存在的事实，但能确保已经存在的事实不被遗漏入材料。

## 什么时候你应该找专家
以下场景强烈建议咨询行政書士：（1）首次 COE 申请、配偶在海外、双方相识时间短于 1 年；（2）从其他 在留資格（特に 短期滞在 / 留学）変更 为 配偶者 签证；（3）有过去不许可履历、年齢差大、或相识渠道为 マッチングアプリ / 国際結婚仲介；（4）一方有 退去強制歴 或 在留資格 取消 履歴。配偶者签证的不许可后续修复成本远高于事前专业审查。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
