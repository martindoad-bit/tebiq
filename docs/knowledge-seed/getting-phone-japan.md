# 我刚来日本，怎么办手机号 / 哪些坑要避开

## 这事跟我有关吗？
如果你刚拿到 住民票、想办一个能长期用的日本手机号、看不懂 大手 vs 格安 SIM 的差异——这篇适合你。如果只是想短期用 1-2 个月，去机场柜台买 prepaid 或开 eSIM 就够了，不必读这篇。

## 这是什么
日本通信業 受 [総務省 電気通信事業](https://www.soumu.go.jp/main_sosiki/joho_tsusin/d_syohi/) 监管。所有提供电话号码（090 / 080 / 070）的事業者 必须做 本人確認。对外国人来说，关键是「你的 在留期間 是否覆盖契約期間」——这是被拒的最常见原因。

## 你应该做什么

### 第 1 步：理解市场分两层
- **大手三家（MNO）**：[NTT docomo](https://www.docomo.ne.jp/) / [au](https://www.au.com/) / [SoftBank](https://www.softbank.jp/mobile/) + 楽天モバイル。基地局自有，覆盖最稳，店头多，但月额最贵（5GB プラン 通常 3,000-5,000 円）。
- **格安 SIM（MVNO）**：IIJmio / mineo / 日本通信 / OCN モバイル ONE / LINEMO（SoftBank 系）/ ahamo（docomo 系）/ povo（au 系）。借大手网络，月額便宜（3GB 約 900-1,500 円），多数 オンライン 申込 only。

### 第 2 步：准备材料
- 在留カード（住所栏已填写）
- 住民票 写し 或 在留カード 上记载の住所が一致する公共料金請求書
- 銀行口座（口座振替）或 クレジットカード（指名一致）
- パスポート（部分要求）
- 印鑑（部分大手店头要求）

### 第 3 步：避开 5 个常见坑
1. **在留期間 不足契约期间被拒**：大手 通常要求 在留期間 残 90 日 以上。残期 < 半年 时优先选 [楽天モバイル](https://network.mobile.rakuten.co.jp/) 或 prepaid 系（b-mobile / 日本通信 SIM）。
2. **クレジットカード 名义不一致**：用配偶 / 朋友的卡支付会被系统拒。新来日没卡时，选支持 デビットカード / 口座振替 的事業者。
3. **2 年契約 / 縛り**：2022 年 電気通信事業法 改正 后大手 解約金 上限 1,100 円，但 端末 分割購入残債 仍然存在。来日不满 1 年的人不建议买 端末 分割。
4. **MNP 番号 持ち運び**：未来如果想换号但留住号码，需事先发行 MNP 予約番号。来日初次办号不涉及，但要知道这个机制。
5. **eSIM 限定 プラン**：部分プラン 只支持 eSIM（如 povo / LINEMO 部分）。確認 自己手机是否 eSIM 対応 + SIM ロック解除済。

### 第 4 步：临时方案
落地当天 → 第 1-2 周用机场 prepaid SIM 或国内开的 eSIM（如 Ubigi / Airalo）。这段期间不办长约，等 住民票 + 銀行口座 都齐了再办，被拒概率最低。

## 不做的后果
- 没有日本号码 → 銀行 / 役所 / 病院 / 二段階認証 全部受阻（多数日本服务以日本号码为本人确認手段）
- 用国外号码长期 ローミング → 月费可能数万円
- 强行办大手长约时被拒 → 留下 信用情報 拒否 履歴，6 ヶ月内 复申也会被拒

## 什么时候你应该找专家
通信契約 是消費者契約，不属行政書士业务范围。被拒时换事業者即可。下列情形才需要专家：
- 你被识别为 経営者 / 役員，公司番号同时申请，需理解 法人契約 名义
- 你被卷入悪質 サブスク / リース 詐欺 → 找 [国民生活センター](https://www.kokusen.go.jp/) 或 弁護士
- 涉及 在留資格 信息错位（地址不一致导致 NTT / SoftBank 拒否 = 你的住民票 / 在留カード 信息其实有问题）
