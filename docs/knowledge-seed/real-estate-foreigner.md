# 我是外国人，能在日本买房吗 / 房贷怎么过

## 这事跟我有关吗？

如果你持 技人国、経営・管理、高度専門職、永住者 在留資格，已经在日本住了几年，开始考虑买房自住或投资，这篇适合你。结论先给：法律上 完全可以；现实上 卡在 房贷审批——能否拿到 住宅ローン、利率多少、首付多少，取决于 在留資格 类别、在日年限、收入安定度。

## 这是什么

**法律层面**：日本《不動産登記法》《民法》对不动产取得没有国籍限制。外国人本人 / 外国法人 都可以以自己名义在日本购入土地、戸建、マンション。所有权 与日本人完全同等——所有権登記、転売、相続 全部一样的法律待遇。这点和韩国、新加坡、泰国、越南 等多数亚洲国家不同（那些国家对外国人有不同程度的购房限制）。

**现实层面**：买房本身不难，难在 住宅ローン。日本 银行的 住宅ローン 审查标准里，**「永住者 / 在留期間 安定性」是核心因素之一**，原因是 35 年 长期贷款时银行需要确保借款人长期在日履约的可能性。

非现金一次付清的话，绕不开下面三个现实约束。

## 你应该做什么

1. **第一步：判断 自己 属于哪一档**

   银行风险排序大致是：
   - **A 档（最优）**：永住者 / 日本人配偶者等 + 大手企业正社員 / 公務員 + 在日 5 年以上 → 与日本人几乎同等审查，主要 都市銀行・地方銀行 全部可申請。
   - **B 档**：高度専門職（1 号 / 2 号）+ 在日 3 年以上 + 安定 高収入 → 多数 都市銀行 / ネット銀 可申请，部分要求担保人。
   - **C 档**：技人国 / 経営・管理 + 在日 5 年以上 + 上場企業 や 自社決算良好 → 部分 都市銀 / 地銀 / 専門金融机构 可申请，首付 30–50% 起、利率较 永住者 高 0.3%–1%。
   - **D 档**：技人国 / 経営・管理 + 在日 3 年未満 / 中小企業 勤務 / 自営業赤字 → 都市銀 ほぼ 通らない。SBI 新生 / SMBC 信託（プレスティア）/ 東京スター / 中国系 在日金融机构 / オリックス銀行 等専門外国人房贷 が 現実的選択肢。首付 30–50%、利率高、審査厳しい。

2. **第二步：把材料准备齐**

   - **在留カード**（裏面 含む）写し、有效期 残半年 以上 が望ましい
   - **住民票**（マイナンバー 入 / 不入 各 1 通、世帯全員 続柄入）
   - **課税証明書**（直近 3 年）
   - **納税証明書**（直近 3 年、未納なし 確認）
   - **源泉徴収票**（直近 3 年、給与所得者 の場合）
   - **確定申告書 控え**（直近 3 年、自営業 / 役員 の場合）
   - **預金通帳 写し**（頭金 出所 確認）
   - **健康保険証** 写し
   - **印鑑証明書**

   永住申請 と 同样、3 年 一貫した 課税 / 納税 履歴 が 通过的核心。1 年でも 未納 / 遅納 があれば 銀行 審査 で 大幅減点。

3. **第三步：先做事前審査（仮審査）→ 本審査**

   - **仮審査**：年収 / 勤続 / 在留資格 / 既存債務 をベースに、銀行 が 借入可能額 を 仮判定。3 営業日 ~ 1 週間。
   - **本審査**：物件決まった後、物件 評価 + 個人 信用調査（CIC / JICC）+ 健康診断（団信 加入審査）+ 在留資格 詳細確認。2 ~ 4 週間。
   - 物件 引き渡し 同日 に 司法書士 立会いで 所有権移転登記 + 抵当権設定登記。

4. **必要 经费 / 不动产税 一覧**

   購入時の 諸費用 は 物件価格の **6%–10%** を見ておく：

   - **印紙税**：売買契約書 に 貼る 収入印紙（1,000 万 ~ 5,000 万円 の 売買契約 → 1 万円、2027 年 3 月 31 日まで 軽減税率）
   - **登録免許税**：所有権移転登記（土地 1.5%、建物 0.3%~2%、軽減税率 あり）+ 抵当権設定登記（借入額 × 0.4%、軽減 0.1%）
   - **不動産取得税**：物件取得後 数ヶ月で都道府県 から請求。土地 / 建物 の 固定資産税評価額 × 3%（軽減 あり、住宅 用 軽減 適用 で 大幅減）
   - **司法書士 報酬**：登記 関連 10–20 万円
   - **仲介手数料**（中古）：物件価格 × 3% + 6 万円 + 消費税
   - **火災保険料**（10 年一括 等）：10–30 万円
   - **団信 保険料**（多くは 利率 上乗せ で組込み）

   保有期間中 毎年：
   - **固定資産税**：固定資産税評価額 × 1.4%
   - **都市計画税**：市街化区域内 評価額 × 0.3% 上限（多数 0.2%–0.3%）

5. **有 5 件 の 落とし穴 を 避ける**

   - **頭金 来源 説明**：海外送金 で 一括 充当 する 場合、 国外送金等調書 で 税務署 に 把握 される。来源 説明 ができないと 贈与税 課税 リスク。
   - **団体信用生命保険（団信）の 健康告知**：嘘 / 漏れ は 後日 保険金 不払い + 契約解除。
   - **住宅ローン控除**：合計所得金額 2,000 万円 以下 等 要件 充足 で 年末ローン残高 × 0.7% を 13 年間 所得税 / 住民税 から 控除。確定申告 が 必要。
   - **永住申請 直前 の 大型借入**：ローン 自体は 不利 評価 では ない が、返済比率 が 高すぎる と 「安定 した 生活」要件 で 審査 が 厳しくなる。
   - **賃貸 / 投資 目的 で 居住用 ローン を 借りる**：用途違反 / ローン 一括 返済 請求 リスク。投資用 物件 は 別途 不動産投資ローン（金利 高め）。

## 不做的后果

- **永住者 / 在日 5 年未満 で 都市銀 ばかり 申し込む**：連続 否決 が CIC 信用情報 に 6 ヶ月 残り、後の ローン 審査 にも 不利。最初から 自分の 段階 に 合った 銀行 を 選ぶ。
- **頭金 来源 不明 で 海外大額送金**：贈与税 課税 + 国税庁 お尋ね（前述 国外送金等調書 制度）。
- **不動産取得税 / 固定資産税 を 把握せず キャッシュフロー 破綻**：購入後 半年 ~ 1 年で 数十万 ~ 百万 オーダー の 請求 が来る ことを 想定 して おく。
- **住宅ローン控除 を 知らず 確定申告 しない**：年 数十万円 の 節税機会 を毎年 失う。
- **団信 健康告知 で 嘘**：契約解除 + 保険金 不払い → 残債務 全額 即時 弁済 義務。

## 什么时候你应该找专家

**你可以自己处理**，如果同时满足：

- 永住者 / 高度専門職 で 大手企業 正社員 / 5 年以上 安定収入
- 課税 / 納税 履歴 完璧 3 年以上
- 頭金 が 日本国内口座 の 自己資金 で 説明可能
- 物件価格 8,000 万円程度まで 居住用

**建议找专家**（税理士 / 不動産仲介 / 宅建士 / 行政書士）如果出现以下任一：

- 技人国 / 経営・管理 で 在日 5 年未満
- 自営業 / 役員 で 直近 確定申告 が 不安定
- 頭金 全額 を 海外送金 で 充当（贈与税 / 国外送金等調書 対応）
- 法人名义 で 投資用 物件 を 取得（経営・管理 ビザ との 整合 / 法人税 対応）
- 永住申請 と 並行 して ローン 申込
- 配偶者 と 共有名義（贈与税 / 持分 設計）
- 相続 / 親 から 贈与 で 物件 取得

不动产税务咨询 税理士；契約 / 重要事項説明 は 宅建士；在留資格 と 連動する 法人設立・名義 設計 は 行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
