## 是什么

在留期間更新許可申請（俗称「签证更新」）的必备材料并非一份固定清单，而是分为两层：**共通基础材料**（所有 在留資格 申请人必须提交）和**资格差异化材料**（根据持有的 在留資格 类型追加）。入国管理局 对两层材料的完整性都会逐项核对，缺一即视为材料不齐。

共通基础材料包括以下四项：

- **在留期間更新許可申請書**（最新版本，可从 入国管理局 官网下载）
- **写真 1 枚**（4×3cm，3 个月以内拍摄，无背景，正面无遮挡）
- **パスポート 与 在留カード**（提示原件，部分窗口需提交复印件）
- **手数料 4,000 円**（许可时以 収入印紙 缴纳）

差异化材料则取决于目前持有的 在留資格——同样是更新申请，技人国、経営・管理、家族滞在、永住者 所需材料差异显著。

## 为什么重要

入国管理局 收到申请后会先做形式审查。材料不全会触发 **補正通知**，要求补充提交。后果链条是：

- 補正通知 → 审查中断 → 实际审查时间延长 1–2 个月
- 审查延长 → 现有 在留期間 逼近终止日 → 即便有 特例期間 覆盖，也会影响海外出入境、转职手续、贷款审批等
- 多次 補正 或材料质量低下 → 担当官对申请人「在留状況」的整体印象下降 → 不许可风险上升

材料齐全本身不是加分项，而是基础门槛。在材料齐全的前提下，担当官才会进入实质判断（在留状況、纳税状况、活动实绩等）。

## 怎么办

按 在留資格 类型对照清单：

**技人国（技術・人文知識・国際業務）更新**

- 雇主出具的 **在職証明書**（注明职位、职务内容、年收）
- 直近 1 年的 **源泉徴収票**
- 直近 1 年的 **課税証明書** 与 **納税証明書**（市役所 开具）
- 雇主公司的 **履歴事項全部証明書**（3 个月以内）
- 必要时追加 **雇用契約書 写し**

担当官关注点：实际从事的业务是否仍符合「技人国」范围、年收是否稳定、纳税是否完整。

**経営・管理 更新**

- 自身公司的 **登記事項証明書**（履歴事項全部証明書，3 个月以内）
- 直近期的 **決算報告書**（貸借対照表 + 損益計算書）
- 法人税・消費税・源泉所得税 的 **納税証明書**
- **事業計画書**（本期实绩 + 下期计划）
- 役員報酬 相关的 **源泉徴収票** 或 **確定申告書 写し**

担当官关注点：事业是否实际运营、是否持续盈利或有合理亏损说明、纳税是否完整。

**家族滞在 更新**

- 扶养者（持有就労系 在留資格 的家属）的 **在職証明書**
- 扶养者的 **課税証明書** 与 **納税証明書**
- 全员合并的 **住民票**（注明 世帯 关系，不省略）
- 関係証明（戸籍謄本 译本、結婚証明書、出生証明書 等）

担当官关注点：扶养者收入是否足以支撑家族、家族关系是否真实持续。

**永住者 更新（在留カード 7 年更新）**

永住者本身没有「在留期間」，但 在留カード 每 7 年需更新一次（高度専門職 2 号同样适用）。所需材料极其简化：

- **在留カード 更新申請書**
- **写真 1 枚**（4×3cm）
- **パスポート 与 现行 在留カード**
- 手数料原则上不要（仅卡片再发行手续）

注意：这是「卡片更新」，与「在留期間更新」是不同的手续，不要混淆。

## 不办的后果

实务中常见的失败模式：

- 写真规格错误（半年前拍的、有背景、尺寸不对）→ 当场要求重拍
- 課税証明書 / 納税証明書 拿错年度（应取最新年度）→ 補正
- 履歴事項全部証明書 超过 3 个月有效期 → 補正
- 経営・管理 提交了赤字決算 却没有合理说明 → 担当官追加资料请求或不许可
- 家族滞在 扶养者收入不足以支撑申请人和子女 → 不许可风险

材料准备阶段花的时间，远比 補正 后的等待时间便宜。建议在现在的 在留期間 满了前 3 个月开始准备，留出窗口处理任何意外。
