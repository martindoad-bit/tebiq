# 续签材料到底要交哪些，我的在留资格对应清单是什么

## 这事跟我有关吗？

如果你准备续签 在留期間 但还没搞清楚要交什么，或者听到「不同 在留資格 材料不一样」却不知道自己这一类对应哪份清单，这篇适合你。覆盖 技人国、経営・管理、家族滞在 三类常见就労 / 滞在系，以及 永住者 的 在留カード 7 年更新。

## 这是什么

在留期間更新許可申請 的必备材料分为两层：

- **共通基础材料**：所有申请人必须提交
- **资格差异化材料**：根据持有的 在留資格 类型追加

共通基础材料：

- 在留期間更新許可申請書（最新版本，可从 出入国在留管理庁 官网下载）
- 写真 1 枚（4×3cm，3 个月以内拍摄，无背景，正面无遮挡）
- パスポート 与 在留カード（提示原件，部分窗口需提交复印件）
- 手数料 4,000 円（许可时以 収入印紙 缴纳）

差异化材料因 在留資格 类型而异。出入国在留管理庁 收到申请后会先做形式审查，材料不全会触发 補正通知，要求补充提交。补正会让实际审查时间延长 1–2 个月。

## 你应该做什么

按 在留資格 类型对照清单准备：

1. **技人国（技術・人文知識・国際業務）更新**
   - 雇主出具的 在職証明書（注明职位、职务内容、年收）
   - 直近 1 年的 源泉徴収票
   - 直近 1 年的 課税証明書 与 納税証明書（市役所 开具）
   - 雇主公司的 履歴事項全部証明書（3 个月以内）
   - 必要时追加 雇用契約書 写し
   - 审查关注点：实际从事的业务是否仍符合「技人国」范围、年收是否稳定、纳税是否完整

2. **経営・管理 更新**
   - 自身公司的 登記事項証明書（履歴事項全部証明書，3 个月以内）
   - 直近期的 決算報告書（貸借対照表 + 損益計算書）
   - 法人税・消費税・源泉所得税 的 納税証明書
   - 事業計画書（本期实绩 + 下期计划）
   - 役員報酬 相关的 源泉徴収票 或 確定申告書 写し
   - 审查关注点：事业是否实际运营、是否持续盈利或有合理亏损说明、纳税是否完整

3. **家族滞在 更新**
   - 扶养者（持有就労系 在留資格 的家属）的 在職証明書
   - 扶养者的 課税証明書 与 納税証明書
   - 全员合并的 住民票（注明 世帯 关系，不省略）
   - 関係証明（戸籍謄本 译本、結婚証明書、出生証明書 等）
   - 审查关注点：扶养者收入是否足以支撑家族、家族关系是否真实持续

4. **永住者 在留カード 7 年更新**
   永住者 本身没有「在留期間」，但 在留カード 每 7 年需更新一次（高度専門職 2 号同样适用）。这是「卡片更新」，与「在留期間更新」是不同的手续。
   - 在留カード 更新申請書
   - 写真 1 枚（4×3cm）
   - パスポート 与 现行 在留カード
   - 手数料原则上不要（仅卡片再发行手续）

## 不做的后果

实务中常见的失败模式：

- 写真规格错误（半年前拍的、有背景、尺寸不对）→ 当场要求重拍
- 課税証明書 / 納税証明書 拿错年度（应取最新年度）→ 補正通知
- 履歴事項全部証明書 超过 3 个月有效期 → 補正通知
- 経営・管理 提交了赤字決算 却没有合理说明 → 担当官追加资料请求或不许可
- 家族滞在 扶养者收入不足以支撑申请人和子女 → 不许可

被 補正通知 后，审查时间通常延长 1–2 个月。多次 補正 或材料质量低下，会影响担当官对申请人「在留状況」的整体印象。

## 什么时候你应该找专家

简单 case 自办即可：

- 同一家公司持续就职、上次更新顺利、本次没有变化
- 家族滞在 配偶和子女信息无变更、扶养者收入稳定
- 永住者 卡片到期更新（手续极简）

下列情形建议咨询行政書士：

- 経営・管理 含赤字決算、股东变更、事業内容大幅调整
- 技人国 此次想从 1 年延长为 3 年或 5 年
- 家族滞在 扶养者收入接近基准线、家族关系证明涉及多国文件
- 收到过 補正通知 或 不許可通知 后再申请
- 中途有过转职、副业、起业等变化
