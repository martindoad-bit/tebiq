# 我的孩子要上日本公立学校，手续怎么办

## 这事跟我有关吗？

如果你的孩子已经办完 在留資格 / 住民票，准备进入日本公立 小学校 或 中学校，或在学年中途从国外迁入需要 編入（插班），这篇适合你。日本对外国籍儿童的 公立義務教育 是不收学费的，但流程要主动走——市役所 不会自动把你的孩子塞进学校。

## 这是什么

日本的 義務教育 是 小学校 6 年 + 中学校 3 年，针对 6–15 歳。法律上 義務教育 仅强制日本国籍儿童，但 文部科学省 政策对**希望就学的外国籍儿童与日本籍儿童同等待遇**——免学费、免教材费（教科書 無償）、午餐费 / 教材副材料费等部分自付。

入学的「学校」原则上由 住所地 教育委員会 按学区指定（指定校制度）。可以申请 学校選択制 或 指定校変更，但需正当理由（兄弟姉妹 同校、通学距离、特别支援）。

时点关键：
- 4 月 入学（新一年级）：前一年 11–12 月 起 教育委員会 发出 就学通知書，1–2 月 安排 就学時健康診断。
- 学年中途（編入 / 插班）：随时可办，材料齐当周或当月 即可入学。

高校（高等学校）非義務教育，需要 受験——但公立高校 对外国籍考生开放，部分自治体有 外国人特別選抜 枠。

## 你应该做什么

1. **完成 住民登録**

   进入公立学校的前提是孩子有 住民票。新到日本或迁入新住所，先到 市役所 / 区役所 完成 住民異動届（転入 / 転居 / 出生 等）。

2. **小学校 / 中学校 4 月 入学（新一年级）**

   - 前一年 10 月 1 日 时点 在该自治体 住民登録 的 日本籍 / 外国籍 児童，自治体会自动发 就学通知書。
   - 外国籍 児童 部分自治体不会自动发——需要 监护人 到 教育委員会 学校教育課 / 学事課 填写「就学申請書」「外国人児童生徒 就学届」明确表达入学意愿。
   - 11–12 月 收到 就学通知書 后，1–2 月 参加 就学時健康診断。

3. **学年中途 編入（插班）**

   - 持 住民票（异动后立即去拿）、母子健康手帳 或 健康診断書、過去 在校証明書 / 成績証明書（原籍国，附日译）、孩子 在留カード 写し，到 市役所 教育委員会 学事担当 窓口。
   - 教育委員会 决定指定校并发 就学通知書 / 入学許可書。
   - 持通知书到指定学校办理入学手续——上学日大致 1 周以内 即可。

4. **必备材料 一覧**

   - 住民票（世帯全員、続柄入）
   - 母子健康手帳 / 予防接種記録（部分接種未完成会被要求补打）
   - 在留カード 写し
   - 印鑑 / サイン
   - 原籍 在校証明書 / 成績証明書（编入時、附日译）
   - 健康診断書（部分自治体 编入時 要求）

5. **日本語支援 制度**

   - **日本語学級 / 日本語指導教室（取り出し）**：把日语弱的学生 部分時間 抽离正常课堂集中授业。
   - **日本語指導教員 配置**：班里有外国籍学生达一定数量时，自治体派遣 日本語指導 担当。
   - **JSL カリキュラム**：文部科学省 制定的「日本語を学習言語として学ぶ」体系。
   - 自治体差异极大——东京 23 区、横浜、川崎、大阪、名古屋 等外国人多的地方制度成熟；部分地方自治体 几乎沒有支援，需要家长主动询问 教育委員会。

6. **高校 受験**

   - 公立高校：基本对外国籍 考生开放，普通入試 即可参加。
   - **外国人特別選抜**：东京都、神奈川、大阪、京都、千葉、埼玉 等多个都府県有特別枠——降低 国语难度、增加 作文 / 面接 比重、要求 来日年数 在一定年限以内（多为 来日 3 年 / 6 年 以内）。
   - 私立高校：各校自行决定，多数不限国籍。
   - 详细情况向 都道府県 教育委員会 / 中学校 進路指導 窓口确认。

## 不做的后果

- **不办 就学手续**：孩子待在家里没人来叫——日本对外国籍儿童 義務 是「同等待遇」而非强制，不会有 児童相談所 上门。但孩子失学会影响今后日本社会适应、進学、最终连接到 父母 在留更新 / 永住申請 时家庭情況的整体评价。
- **预防接種 不全 / 健康診断書 缺失**：被学校要求补打或暫不入学。
- **不申请 編入 直接登校**：学校无法接收——必须先经过 教育委員会 指定。
- **错过 外国人特別選抜 申请期限**：高校 受験 时只能走普通入試，国语劣势放大。

## 什么时候你应该找专家

**你可以自己处理**，如果同时满足：

- 住所稳定、孩子住民登録 已完成
- 4 月 入学 或学年中途編入 普通流程
- 孩子日语沟通无重大障碍 或自治体 日本語支援 制度成熟

**建议找专家 / 第三方支援**（教育委員会 国際教育担当、NPO、留学 / 移民咨询）如果出现以下任一：

- 自治体 日本語支援 几乎沒有，孩子日语零基础
- 孩子有特别支援需求（発達障害、不登校 履历、身体障害）
- 高校 受験，需要选 外国人特別選抜 / 来日年数 卡线
- 计划让孩子走 国際学校（インターナショナルスクール，多为 私立、学費 1 年 100–300 万円）
- 重国籍、监护权、离婚后亲权 等家庭法律问题影响入学手续

教育相关咨询通常无需 行政書士。需要确认 在留資格 与就学衔接（如 家族滞在 → 留学 转换）时再咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
