# 在留期間更新申請書 怎么填，常见出错的地方

## 这事跟我有关吗？

只要你要续 在留資格，这张表就跑不掉。它本身不难，但**填错一处就被退回，少则多跑一次入管，多则审查时间被拉长**。

不同 在留資格 用不同版本（"申請書 別表"），下载时要选对。

## 这是什么

正式名称：**在留期間更新許可申請書**。两部分：

- **本人记入用**：所有签证类型共通的基础信息
- **所属机关 / 配偶 / 監督者 等记入用**：因签证类型而异（"技術・人文知識・国際業務"用别表，"家族滞在"用别表，"経営・管理"用别表）

下载地址：[出入国在留管理庁 申請書様式（在留期間更新）](https://www.moj.go.jp/isa/applications/procedures/16-2.html)。

2026/01/05 起，**在留申請オンラインシステム** 全面改版，本人也能在线填+提交，不需要去窗口排队。手机也能用。

## 你应该做什么

**主要栏位检查清单**（按顺序）：

1. **氏名（漢字）/ 氏名（ローマ字）**：和 在留カード、護照 严格一致。漢字有"曾"和"曽"、"高"和"髙"的细微差别，对了护照写哪个写哪个
2. **国籍・地域**：写"中国"，不写"中華人民共和国"
3. **生年月日**：西历、年/月/日
4. **性別**：男 / 女
5. **出生地**：写省 + 市（例：上海市、遼寧省瀋陽市）
6. **配偶者の有無**：有 → 写配偶 在留資格、护照号
7. **本邦居住地**：和住民票一致（番地、号、室号都不能漏）
8. **電話番号**：能联系到的真号码
9. **旅券番号 / 有效期限**：当前护照
10. **现有 在留資格 / 在留期間**：和 在留カード 上一致
11. **在留カード番号**
12. **希望在留期間**：通常写"5 年"或"3 年"。写多少不影响审查，写"1 年"反而被认为有问题
13. **更新の理由**：**最容易被人忽视的栏，又是最关键的栏**
14. **犯罪歴**：有就老老实实写（细节后面会问），漏写比写了更糟
15. **就労先 / 所属機関**（就労系签证）：公司全名、地址、电话、雇佣开始日
16. **職務内容**（就労系）：具体到日常工作内容，不能只写"システムエンジニア"或"会社員"

**最高发的 5 个错误**：

| 错误 | 后果 |
|---|---|
| 申請理由 写"工作关系"/"为了继续生活" | 退回要求重写，审查时间 +2-4 周 |
| 雇主 公司名 用日文写一半 / 罗马字一半 | 必须和 履歴事項全部証明書 完全一致 |
| 在留期間 起算日写成 在留カード 发行日，而非"更新日" | 起算日是上次许可日 |
| 漏写 副业 / 第二职业 | 永住申請时被翻出来"虚偽申告"，致命 |
| 配偶 在留資格 写错（"家族滞在" vs "技術・人文知識・国際業務"） | 影响家族滞在子女审查 |

**申請理由 怎么写**（一段就够，约 200-400 字）：

> 例（技人国）：「現在、株式会社○○においてシステムエンジニアとして 3 年間勤務しております。今後も同社にて、現在担当している××システムの開発業務を継続するため、在留期間の更新を希望いたします。年収は ¥××× 万円、住民税および所得税はすべて納付済みです。」

要点：现在做什么、未来做什么、收入、税缴清楚。**不要写"为了家庭"**——那是 家族滞在 的理由，不是 技人国 的。

## 不做的后果

- 写错被退回 → 多跑 1 次入管（东京 2-4 小时排队）
- 申請理由 太抽象 → 审查官心证不好，下年限缩短（5 年 → 3 年 → 1 年）
- 漏写 / 虚偽 → "在留資格取消" 风险（出入国管理法 第二十二条の四），永久记录

## 什么时候你应该找专家

以下情况强烈建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）：

- 你换过工作，新旧职务内容差异大

- 公司刚成立 1-2 年（経営・管理）
- 上一次续签拿了 1 年，这次想拿 3 年 / 5 年
