# 我失业了，签证多久会失效 / 我要做什么

## 这事跟我有关吗

如果你符合以下条件，请继续往下看：

- 持有 就労系 在留資格（技人国 / 経営・管理 / 高度専門職 等）
- 已经离职（被解雇 / 自主退職 / 公司倒闭 / 経営失敗 都算）
- 还没找到下一份与 在留資格 契合的工作

## 这是什么

入管法 第 22-4 条 第 1 項 第 5 号 规定：

> 在留資格 を有する 外国人 が、その 在留資格 に応じた 活動 を 継続して 3 月以上 行っていないとき、当該 在留資格 を 取り消すことができる。

口径解释：

- 起算日 = 离职当天（不是签证到期日）
- "3 个月" = 暦日 3 个月，不是 90 日（4 月 1 日离职 → 7 月 1 日触发）
- 例外：你能 立証 有"正当理由"——例如 産休 / 育休 / 病気療養 / 求職活動中 + 真摯に 求職している証拠 等
- "取消" = 入管发出 取消通知書 → 一般给 30 日内出国 期间，特殊情况可申 在留資格 変更

注意：3 个月没满之前，在留資格 仍然有效。但接近这条线时，下次 更新 / 変更 都会被严格审查。

## 你应该做什么

### 离职日起 14 日内

1. **離脱届（契約機関に関する届出）**：14 日内向入管提交。这是义务，不交会有 過料 + 信用扣分
2. **健康保険 / 厚生年金 切换**：去市区町村役所 加入 国民健康保険 + 国民年金（找到新工作再切回 社保）
3. **離職票 拿到**：公司一般离职后 10 日左右出。这是 雇用保険 失業給付 申请的必备件

### 离职后 1–4 周

4. **ハローワーク 求職登録 + 失業給付 申请**
   - 持 離職票 + 在留カード + マイナンバー + 通帳 去 居住地管轄ハローワーク
   - 受給 期间一般 90–360 日（年龄 / 雇用保険 加入年数 而定）
   - 求職登録 本身就是"我在积极找工作"的官方证据，将来入管要看

5. **記録留底**：保留 面接 通知 / メール / 求人応募 履歴——3 个月线触发时，这些是 正当理由 立証 的核心

### 离职后 1–3 个月

6. 找到新公司 → [30 天行动表](changing-jobs-30day-action.md) 流程走
7. 进入 60 日还没找到 → 开始考虑 在留資格 変更 路径：
   - 已婚 + 配偶为日本人 / 永住者 → 配偶者签证 変更
   - 配偶为 就労系 持有人 → 家族滞在 変更
   - 都不适用 → 短期滞在 変更（少见，且不能就劳）+ 国外重新申请

### 离职后 80 日 ～ 90 日

- 必须做出选择：要么签新合同，要么提前申请 在留資格 変更，要么主动出国
- 这个阶段不要等入管来通知——入管发取消通知书后再行动几乎没有空间

## 不做的后果

- **3 个月 不动**：取消通知書 → 30 日 退去命令 → 不出国 = 強制送還 + 5–10 年 入国禁止
- **離脱届 不交**：20 万円 以下 過料；下次 更新 / 変更 信用扣分
- **求職活動 不做记录**：3 个月线触发时，没有"正当理由"立証，入管直接取消
- **健保 / 年金 断保**：再续签时 納付状況 不连续，影响 定住者 / 永住 申请

## 何时找专家

下面任一情况，请联系 行政書士 / 入管専門 弁護士：

- 离职 60 日 以上仍未找到 新工作（开始研究 変更 路径）
- 你是 経営・管理 持有人，公司 廃業 / 倒産（事業継続 立証 几乎不可能时）
- 高度専門職 持有人，离职后 ポイント 重新计算可能不达 70 点
- 你已经接到入管 取消事由 的 意見聴取 通知書（这一步已经晚了，需要立即应対）
- 求職活動 状况 复杂（産休 / 病気療養 / 家庭事情）需要书面 立証 設計

書士 在 失業 这种节点最大的价值是设计"離脱日 → 新就労 / 変更 / 出国"的最佳时间线，把 3 个月底线之前能用的所有合法路径都摆上桌。等到入管寄出 取消通知書 再找書士，能做的就只剩 出国 准备 了。
