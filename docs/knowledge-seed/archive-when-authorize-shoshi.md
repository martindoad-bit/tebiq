# 何时该把档案授权给行政書士

## 这事跟我有关吗？
你在考虑 ¥9,800 咨询，但担心档案里包含 護照 / 在留カード / 課税証明書 / 給与明細 等敏感信息，授权给外部第三方会不会泄漏。这篇说明授权机制的设计、可控范围、以及不授权的实际影响。

## 这是什么
档案的默认状态是：**仅你本人可见，不分享给任何外部第三方**，包括 TEBIQ 内部任何非工程必须的人员。

¥9,800 咨询预约时，预约表单会出现一个**可选**勾选项：「授权 行政書士 在咨询前 24 小时内查看我的档案」。

授权的可控范围：

- **全部档案**：所有 doc_type、所有时间段
- **仅最近 1 年**：把档案过滤为最近 12 个月的内容
- **仅特定 doc_type**：例如只授权 在職証明書 / 課税証明書 / 在留カード，不授权 婚姻 / 医疗相关
- **仅特定时段**：例如只授权 2023-04 至今的内容

授权的撤回：

- 咨询前任何时点可在档案设置撤回，撤回后行政書士 端立即失去访问权
- 咨询结束后系统自动撤回（默认设置为咨询后 24 小时失效）
- 你也可在咨询结束后立即手动撤回

不授权也可以咨询。在这种情况下，行政書士 依靠：

- 你在咨询时口述的履历
- 你现场带来的纸质 / 电子材料

但效率显著降低——口述容易遗漏 / 时间错位（参见档案如何帮助行政書士判断），现场带的材料通常也不会包含 過去 5 年完整的纳税记录。30 分钟咨询里能讨论的具体方案因此减少。

授权的法律边界：

- 授权动作本身记录在 access log（你可在档案查看：什么时间、哪位行政書士、看了哪些 doc_type）
- 行政書士 受 行政書士法 第 12 条保密义务约束，违反将面临业务停止 / 登录抹消
- TEBIQ 受 個人情報保護法 约束，授权范围之外的数据不会传输

## 你应该做什么
1. 第一次 ¥9,800 咨询时，建议至少授权"最近 1 年 + 全 doc_type"。这是覆盖大多数咨询场景（续签 / 補正 / 入管来文）的最小必要范围。
2. 涉及 永住 / 帰化 申请的咨询，建议授权"全部档案"。这两类申请审查 過去 5-10 年履历，不授权全部档案会让 行政書士 无法判断你是否符合"引き続き 在留要件"。
3. 涉及健康 / 婚姻敏感事项的，可以排除该 doc_type，只授权与签证审查直接相关的部分。
4. 咨询结束 24 小时内手动检查 access log，确认行政書士 端访问已失效。

## 不做的后果
完全不授权也可以做咨询，但 30 分钟的产出密度会显著低于授权 case。如果你的目的只是"了解概念 / 政策方向"，不授权是合理选择。如果你的目的是"针对我 case 的具体步骤"，不授权会把咨询变成原则性讨论。

## 什么时候你应该找专家
你不确定授权范围如何取舍、或档案中有需要法律意见判断的敏感事件（例如刑事记录、行政处分、家庭暴力相关材料），建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
