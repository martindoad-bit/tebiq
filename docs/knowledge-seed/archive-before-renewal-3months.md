# 续签前 3 个月：档案能告诉我什么

## 这事跟我有关吗？
你的 在留カード 距到期日 90-120 天，意味着 在留期間更新 申请的官方受理窗口（在留期限到来前 3 个月）已经打开或即将打开。这 3 个月是档案产出最大价值的时期——它能在窗口开放当天就生成「公司侧待办 / 个人侧待办 / 届出义务复核」三张清单，避免你临窗口才发现某份证明需要 2 周才能开出。

## 这是什么
档案在续签前 3 个月的作用，是把分散在不同部门、不同期限的材料和义务，转化成按"取得时间"排序的待办列表。

档案根据你的 在留資格 自动列出的清单：

- **公司侧待办（雇主出具，常需 1-3 周）**：在職証明書、雇用契約書（写）、源泉徴収票（前年）、給与明細（直近数月）、法人侧需要时含 履歴事項全部証明書、決算書、事業計画書（経営・管理）
- **个人侧待办（市役所 / 税务署，当天可取）**：直近 1-3 年 課税証明書、納税証明書、住民票（必要时含 世帯全員）
- **本人侧（自备）**：在留カード、護照、申請書、写真，已婚的话含配偶相关材料

档案能识别的"未届出"风险事件：

- 过去 1 年内拍照入档的"新公司 雇用契約書 / 退職証明書"，对应的 契約機関 変更届出 是否在 14 日内完成（档案会标记届出完成事件，未标记即风险）
- 拍照入档的"新住居 賃貸契約書"，对应的 住居地届出（市区町村）是否在 14 日内完成
- 婚姻 / 离婚 / 配偶死亡 等身份变化的 14 日内届出

档案能跨年度对照的内容：

- 年收入趋势（源泉徴収票 / 確定申告 控え）：是否稳定或下降到不再符合 在留資格 期待水准
- 纳税缴纳节奏：是否每年度准时，有无延滞补缴

## 你应该做什么
1. **D-90（在留期限前 3 个月）**：打开档案 续签自查，生成三张清单。把公司侧清单立刻发给人事部门——开 在職証明書 / 源泉徴収票 一般需要 2-3 周，越早越好。
2. **D-60（前 2 个月）**：去 市役所 一次性开齐 課税証明書 / 納税証明書 / 住民票，拍照入档与档案对账。
3. **D-30（前 1 个月）**：复核届出义务清单。任何过去 1 年的"未届出"事件，立即向 出入国在留管理庁 提交补办（迟れた届出 仍可受理，但会在续签时被纳入审查）。
4. **D-7（前 1 周）**：用档案中的全部事件做最后一次自查。在线申请（在留申請オンライン）或临窗口前确认材料齐备。

## 不做的后果
不在 D-90 启动，只剩 D-30 时才行动会进入紧急状态：开 在職証明書 来不及（人事部门标准回复时间 2 周）、過去 3 年 課税証明書 如果跨市需要分别申请、未届出事项已无补救空档。最坏情况是只能在 在留期限 当天提交申请、依靠 申請受理票 的 特例期間 续命，但材料瑕疵此时已无法补救。

## 什么时候你应该找专家
档案识别出 過去 1 年内有 2 次以上转职、收入下降到 在留資格 临界水准、有滞納或督促记录、或当前岗位与 在留資格 实质契合度有疑问，建议在 D-60 之前咨询行政書士。経営・管理 续签涉及事业计划与决算文书的，专业判断对许可率影响显著。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
