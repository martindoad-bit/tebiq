## 这事跟我有关吗？

你刚到日本、刚换工作、刚退职、或者从公司辞职准备转 自営業——突然不知道自己的**健康保険証**要找谁办、空档期怎么办。日本实行「**国民皆保険**」体制，所有在日本拥有住所、在留期間 3 个月以上的人（含外国人）都必须加入某一种公的医疗保险。这篇适合你。

## 这是什么

日本的公的医疗保险分两大体系：

- **社会保険（健康保険）**：面向雇员的「被用者保険」，由企业加入的**健康保険組合**或全国统一的**協会けんぽ**承保。**公司直接办**。包含 健康保険 + 厚生年金 一套。**保険料**按**標準報酬月額**计算，公司与本人折半负担，从工资直接代扣。**被扶養者**（年収原则在 130 万円未満，60 岁以上或残障者 180 万円未満，且不超过被保険者收入一半）可挂在被保険者名下，无需另缴。
- **国民健康保険（国保）**：面向其他居民，由市区町村经营。**自己到市役所 / 区役所办**。覆盖：自営業者、自由职业、退職后未加入任意継続的人、留学生、家族滞在 中未被认定为被扶養者者、未满足社保加入要件的非常勤等。**保険料**按前年所得 + 加入人数计算，每个市町村税率不同，**世帯主**为缴纳义务者。**没有「扶養」概念**——家庭成员每人都计入保険料。

两者都覆盖一般医疗费 70%、本人**自己負担割合 30%**（学龄前 20%，70 岁以上 10–30%）。都领取**健康保険証**作为就诊凭证。加入后即可使用**高額療養費制度**：当一个月的自负医疗费超过所得对应的上限（例如 年収 370 万円以下区分约 5.7 万円 / 月），超过部分可申请退还。事前申请「**限度額適用認定証**」可在窗口直接按上限结算。

**社会保険强制对象**：日本国内的法人企业、五人以上的部分个人事业所原则上必须为常勤雇员加入；非常勤雇员若每周工时达常勤 3/4 以上，或满足「週 20 小时 + 月薪 8.8 万円以上 + 雇用見込 2 个月超 + 非学生」等要件也属强制对象。

## 你应该做什么

按场景对号入座：

1. **刚入境 14 天内**：到市役所 / 区役所办**転入届** + 国保加入。如果已就职公司则社保由雇主统一加入，**不用自己跑市役所**。
2. **刚入职公司**：把マイナンバー、必要时把家族信息提交给人事，公司会办「健康保険被保険者資格取得届」。家族要挂扶養的，提交「健康保険被扶養者異動届」。
3. **刚退职 / 转自営業（失去社保资格次日起 14 天内）**：二选一——
   - **任意継続**（最长 2 年，全额自负担：原本公司负担那部分也要自己出）：到原健康保険組合 / 協会けんぽ 申请。期限内未缴会立即丧失资格且不能恢复。
   - **国保**：到新住所地市役所 / 区役所办加入手续。带退職証明書 / 離職票、マイナンバー。
4. **转职（前后无空档）**：原公司办脱退，新公司办加入，本人无需到市役所。如有空档期，先办**任意継続**或临时**国保**覆盖。
5. **転出 / 离开日本**：办**転出届**并办理国保 / 社保脱退。
6. **扶養者新增 / 失去**（结婚、生子、配偶就职 / 失业）：通过雇主向**年金事務所**或健康保険組合申请变更。
7. **就医前**：高額医療予定者，事前到健康保険組合 / 国保窗口申请「**限度額適用認定証**」。

## 不做的后果

- **无保险期间生病就诊**：医疗费 100% 自费。普通门诊一次轻则数千日元，住院手术常达数十万到数百万円。
- **未加入期间被发现**：原则上**追溯加入**并补缴最长 2 年保険料，金额往往一次到账数十万円。
- **国保滞納路径**（机械推进）：滞納 → 督促状 → **短期保険証**（有效期 1–6 月，需反复领取）→ **資格証明書**（医疗费先全额自付，事后向市役所申请 7 成还付，欠保険料可能直接抵扣）→ 财产**差押**（給与、預金、生命保険解約金均可被扣押）。
- **任意継続滞納**：期限内未缴立即丧失资格，事后只能转国保。
- **永住・帰化审查**：保険料滞納记录是入管和法務局重点查看的项目，公的義務履行 不达标会显著增加不许可风险。

## 什么时候你应该找专家

**简单情形**（自己处理即可）：单纯转入 / 转出、家族扶養认定标准内、单一就业身份、按时缴纳。

**复杂情形**（建议咨询）：

- 退職后任意継続 vs 国保哪边保険料 更便宜（需逐项试算）
- 配偶年収临近 130 万円，扶養 認定 / 取消 的临界判定
- 自営業 + 副業 + 兼职多重身份的加入归属
- 长期滞納面临差押 / 即将申请永住・帰化
- 国保保険料 减免申请（経済困難）
- 跨国就业、海外医疗费 海外療養費 申请
