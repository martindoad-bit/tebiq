# 国保 / 社保 切換情景与加入空白防止

## 适用场景
- 入社 / 退職 / 転職 / 自営業 開始 / 配偶者 扶養 加入 or 除外 等で **健康保険 + 年金 加入区分が変わる人**。
- 退職と次の入社の間に **空白期間**（数日〜数ヶ月）がある人 — 加入空白防止が最重要論点。
- 不适用：海外転出（→ 回国清算条目）/ 75 歳以上（後期高齢者医療制度）/ 生活保護 受給者。

## 时间线

### 事由発生当日（入社日 / 退職日 / 開業日）
- 必办：状況区分の確認 — **入社 = 厚生年金 + 健康保険 加入義務発生（公司側）**、**退職 = 同日喪失**、**自営業 開始 = 国民年金 + 国民健康保険 加入義務発生**。
- 必办：旧 健康保険証 / 在職証明 / 退職証明書 を保管。

### 14 日以内（法定期限）
- 必办：退職後・自営業 開始後 14 日以内 = **国民健康保険 + 国民年金 加入届出**（区役所 / 市役所）。
- 必办：扶養 加入除外（収入基準超過 = 130 万円 / 106 万円）= 14 日以内 国保 + 国民年金 1 号 への切替届出。
- 可办：任意継続被保険者制度（旧 健康保険 を 最大 2 年間 継続）= **退職翌日から 20 日以内 申請**（こちらが厳しい）。

### 1 ヶ月以内
- 必办：保険料 納付方法 確定 — 普通徴収（納付書 / 口座振替）or 給与天引き（社保 加入時）。
- 可办：国民健康保険 高額療養費 限度額適用認定証 申請（医療費 高額発生予定時）。
- 可办：国民年金 免除 / 猶予 申請（前年所得 基準満たす時）。

## 必办事項（按重要性排序）

### 1. 退職時 健康保険 切替（社保 → 国保 or 任意継続 or 家族扶養）
- **在哪办**：国保 = 区役所 / 市役所 国保年金課。任意継続 = 旧加入 健康保険組合 / 協会けんぽ。家族扶養 = 配偶者 / 親族 加入の 健康保険組合。
- **要什么材料**：退職証明書 or 健康保険資格喪失証明書、在留カード、護照、印鑑、マイナンバー、収入見込証明（扶養 加入時）。
- **办不办的后果**：**無保険状態 = 医療費 100% 自己負担**。後から国保 加入 = **退職日翌日から遡及課金（最大 2 年）**。任意継続 申請期限 20 日 を過ぎると以後申請不可。

### 2. 退職時 年金 切替（厚生年金 2 号 → 国民年金 1 号 or 3 号）
- **在哪办**：1 号（自営業 / 無職）= 区役所 / 市役所 国保年金課。3 号（配偶者扶養）= 配偶者の勤務先 経由。
- **要什么材料**：年金手帳 or 基礎年金番号通知書、離職票 or 退職証明書、在留カード、印鑑、配偶者の健康保険証 写し（3 号 加入時）。
- **办不办的后果**：未届け = **国民年金 未加入期間として記録、後日 遡及納付通知**。**老齢基礎年金 受給資格期間（10 年）に算入されない**。永住 申請 / 在留期間更新 で 「年金 未加入期間あり」と判明 → 公的義務不履行 評価マイナス。

### 3. 入社時 健康保険 + 厚生年金 加入（公司手続）
- **在哪办**：会社 人事 / 経理（手続主体は会社、本人は書類提出）。
- **要什么材料**：年金手帳 or 基礎年金番号通知書、マイナンバー、扶養家族 関係書類（配偶者・子の在留カード / 戸籍 / 収入証明 等）。
- **办不办的后果**：会社が手続怠ると **入社日からの保険料 遡及徴収 + 会社が罰則対象**。**本人側 国民年金 / 国保 解約手続 を忘れると 二重加入 + 二重課金**。社保 加入後 14 日以内 に 区役所 で **国保 / 国民年金 1 号 喪失届** 必要。

### 4. 自営業 開始時 国保 + 国民年金 加入
- **在哪办**：区役所 / 市役所 国保年金課。
- **要什么材料**：開業届 写し（税務署提出済）、退職証明（社保 から切替時）、在留カード、護照、印鑑、マイナンバー。
- **办不办的后果**：**国保 保険料 = 前年所得 + 世帯人数 で算定**、自営初年度は 前年給与所得ベースで計算 → 高額になりがち。**未加入期間 は遡及徴収（最大 2 年）**。**国民年金 1 号 = 月額 16,980 円（2024 年度）固定**、未納分 2 年以内 なら追納可能。

### 5. 扶養 加入 / 除外（配偶者・親族）
- **在哪办**：被保険者（扶養者）の勤務先 健康保険組合 + 年金事務所。
- **要什么材料**：扶養 加入 = 配偶者 / 子の 在留カード 写し、収入証明（年収見込）、続柄 立証書類（戸籍謄本 / 婚姻届受理証明 等、外国人は本国 戸口簿 + 翻訳）。除外 = 収入超過 立証（給与明細 / 確定申告書）。
- **办不办的后果**：扶養 加入の収入基準 = **130 万円**（60 歳未満一般）or **106 万円**（一定要件下のパート）。超過後の届出遅延 = **被扶養者資格 遡及取消 + 過去医療費 7 割分 返還請求**。除外時に 国保 + 国民年金 1 号 への切替届 14 日以内 提出。

### 6. 任意継続被保険者制度 利用判断
- **在哪办**：退職時 加入していた 協会けんぽ / 健康保険組合（**退職日翌日から 20 日以内 必着**）。
- **要什么材料**：任意継続被保険者資格取得申出書、健康保険資格喪失証明書、本人確認書類、保険料初回納付（一括 or 月払）。
- **办不办的后果**：申請期限 20 日 を過ぎると **以後申請不可**。任意継続 = 最大 2 年、保険料は 退職時標準報酬月額 ベース（上限あり、協会けんぽ 2024 年は 上限 30 万円）。**国保 と比較 = 自治体・所得・扶養家族数 で どちらが安いか分岐**、要試算。

## 容易遗漏的事

### 坑 1：「退職と入社の間 1 週間」も加入空白対象
転職で「金曜退職 + 月曜入社」= 土日 2 日間 の空白がある場合でも、**厳密には その期間 国保 / 国民年金 1 号 加入義務**。実務では新雇用主の社保 加入日が退職翌日 = 空白なし、ただし **入社日が 退職日翌日 でない場合 = 空白期間分の届出 必要**。短期空白で届出忘れ → 後年 永住 / 更新審査で 加入記録 ギャップ発見 → 説明要求。**1 日でも空白あれば 即届出**。

### 坑 2：扶養家族 の収入基準「130 万円」の罠
「年収 130 万円未満」= 1 月-12 月 の暦年計算ではなく、**「今後 1 年間の見込収入」が 130 万円未満**。月収換算 = 約 108,333 円。**月収が一時的に超過 = 即扶養除外** ではないが、**3 ヶ月連続超過 や 賞与含めて年収見込 130 万円超過 = 健康保険組合 から 扶養除外通知**。気づかず継続 = 遡及取消 + 医療費返還。**年初予定が変わったら 即報告**。

### 坑 3：国保 保険料 の 前年所得 連動 の落とし穴
国保 保険料 = **前年（1-12 月）所得** ベースで翌年 6 月-翌々年 5 月 課税。**社保 から 国保 へ切替時 = 前年が高所得（社保 加入で給与高い時期）= 国保保険料 高額になる**。退職翌年の国保 は 高額予算が必要。**任意継続 の方が安い場合多し**（任意継続は 退職時標準報酬月額 ベース、上限あり）。**両方の保険料を試算してから選択**。
