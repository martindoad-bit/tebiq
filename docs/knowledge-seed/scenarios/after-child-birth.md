# 孩子出生后必办清单

## 适用场景
日本国内出生的子女家庭。海外出生 + 父母任一方在日 case 適用「3 ヶ月内 出生届 + 出生時 国籍留保」专用流程。仅有「妊娠中」尚未出生的家庭不适用本篇，参考母子健康手帳 / 出産予定 別篇。

## 时间线

### 第 1 日（出生当日）
- 病院 出生証明書 取得
- 母子健康手帳 出生記録

### 第 14 日内（出生届 期限）
- 出生届 提出（戸籍法 第 49 条 出生日含 14 日内）
- 健康保険 加入手続（健保組合 / 国保 5 日 - 14 日内目安）
- 出産育児一時金 直接支払制度 申請（病院経由）

### 第 30 日内（在留資格 取得期限）
- 在留資格 取得許可申請（入管法 第 22-2 条：出生日含 30 日内）
- 児童手当 認定請求（出生日翌日起 15 日内 → 遡及支給）
- 高額療養費 / 乳幼児医療証 申請

### 第 60 日内
- 保育園 / 認定こども園 入園申込（自治体ごとに申込締切）
- 子女戸籍 整理（国際結婚家庭 / 認知届 等）
- 銀行 子女口座 開設（学資保険 / 児童手当 受給口座）

## 必办事项（按重要性排序）

### 1. 出生届 提出
- **在哪办**：出生地 / 父母本籍地 / 父母所在地 市役所 / 区役所 戸籍課（24 时间）
- **要什么材料**：出生届書（医師 / 助産師 出生証明書 付）/ 母子健康手帳 / 父母戸籍謄本（日本人親）or 在留カード（外国人親）/ 印鑑（任意）
- **办不办的后果**：戸籍法 第 49 条 出生日含 14 日内（海外出生 3 ヶ月内）。怠慢则 5 万円 以下 過料（同法 第 137 条）。 14 日経過時は 戸籍課 に届出遅延理由書 提出。

### 2. 在留資格 取得許可申請
- **在哪办**：所在地 出入国在留管理庁
- **要什么材料**：在留資格取得許可申請書 / 出生届受理証明書 / 母子健康手帳 / 父母 在留カード / 父母 住民票 / 写真（4cm×3cm）/ 質問書（家族滞在 等）
- **办不办的后果**：入管法 第 22-2 条第 1 項：出生日含 30 日 を超えて在留する場合に必要。怠慢 + 30 日 以上滞在 = 不法滞在 状態。在留期限 は 父母の在留期限 と原則同じ。
- **特例**：日本人親 + 嫡出 / 認知済 子女 = 出生時 日本国籍 取得 → 在留資格 不要。

### 3. 国籍 確認 + 国籍留保届（出生時）
- **在哪办**：戸籍課（出生届 と同時）
- **要什么材料**：出生届 内「日本国籍を留保する」記載
- **办不办的后果**：日本人親 + 海外出生 子女が 出生時 重国籍 になる場合、国籍留保届 を 出生 3 ヶ月内 にしないと 日本国籍 喪失（国籍法 第 12 条）。日本国内出生 + 父母双方外国人 → 父母 国籍法 による（中国国籍 父母 → 中国籍取得 + 後日 中国大使館 へ届出）。
- **両方外国人 case**：日本国籍 取得不可。子女は 外国籍 + 在留資格 取得 が必要。

### 4. 健康保険 加入届
- **在哪办**：父母就業会社 健保組合（健保 加入者 case） / 市役所 / 区役所 国保課（国保 加入者 case）
- **要什么材料**：被扶養者異動届 / 出生届受理証明書 / 戸籍謄本 / マイナンバー
- **办不办的后果**：健保 加入は出生日 起算 5 日以内（被扶養者異動届 健保法）/ 国保 加入は 14 日以内（国保法 第 9 条）。怠慢则 出産時 / 新生児期 医療費 自己負担 → 後日遡及還付 手続煩雑。

### 5. 出産育児一時金 申請（¥500,000）
- **在哪办**：健保 / 国保 経由（直接支払制度 ある病院は病院窓口）
- **要什么材料**：出産育児一時金支給申請書 / 出生証明書 / 健康保険証 / 振込先口座
- **办不办的后果**：1 児当 ¥500,000（2023.04 から）。直接支払制度 利用時は出産費用との差額のみ請求。申請時効 2 年。

### 6. 児童手当 認定請求
- **在哪办**：住民登録地 市役所 / 区役所 こども家庭課
- **要什么材料**：児童手当認定請求書 / 受給者 マイナンバー / 受給者口座 / 健康保険証 / 子女マイナンバー（出生届 後 1-2 週で発行）
- **办不办的后果**：出生日翌日起 15 日内 申請なら出生月翌月 から支給。15 日 過ぎた申請月翌月 から支給（遡及不可）。2024.10 から所得制限撤廃 + 高校生 まで延長 + 第 3 子以降 ¥30,000/月。

### 7. 出産手当金 / 育児休業給付金（働いている母親）
- **在哪办**：会社経由 健保組合 / ハローワーク
- **要什么材料**：出産手当金支給申請書（産前 6 週 + 産後 8 週） / 育児休業給付金支給申請書（雇用保険）
- **办不办的后果**：出産手当金 = 標準報酬月額 ÷ 30 × 2/3 × 産休日数。育児休業給付金 = 6 ヶ月 まで 67% / 7 ヶ月以降 50%。雇用保険 加入 + 産前 2 年 内に被保険者期間 12 ヶ月以上 が要件。

### 8. 高額療養費 / 乳幼児医療証
- **在哪办**：市役所 / 区役所 健康課（自治体により名称異なる：マル乳 / 子ども医療 / 乳幼児医療証 等）
- **要什么材料**：健康保険証 / 出生届受理証明書 / 所得証明書 / 在留カード
- **办不办的后果**：自治体により 0 歳 - 中学生 まで 医療費自己負担 ¥0 - ¥500/回。新生児集中治療室 入院 = 高額医療費 + 乳幼児医療証 で自己負担大幅削減。

### 9. 保育園 / 認定こども園 入園申込
- **在哪办**：住民登録地 市役所 / 区役所 保育課
- **要什么材料**：保育所等利用申込書 / 就労証明書（父母双方） / 課税証明書 / 子女健康保険証 / 在留カード
- **办不办的后果**：4 月入園 申込締切は前年 11 月 - 12 月。途中入園は毎月締切。激戦区（東京 23 区 / 横浜 / 川崎 等）は 0 歳児 4 月入園 が最易、1-2 歳児 4 月入園 が最難。

### 10. 子女戸籍 整理（国際結婚 / 認知 case）
- **在哪办**：戸籍課
- **要什么材料**：出生届 / 認知届（父母 未婚 + 日本人父認知 case）/ 戸籍謄本
- **办不办的后果**：認知届 出生前 = 胎児認知 → 出生時 日本国籍。認知届 出生後 = 出生後認知 → 国籍法第 3 条による国籍取得 手続別途。

### 11. 配偶者 公司届出
- **在哪办**：自分 / 配偶就業会社 人事
- **要什么材料**：扶養家族異動届 / 出生届受理証明書 / 児童手当 申請（社内）
- **办不办的后果**：扶養手当 / 家族手当 / 育児休業 / 育児短時間勤務 / 配偶者出産休暇 等の社内制度 利用。年末調整 扶養控除 反映（16 歳未満は税額控除 0 だが住民税 非課税限度額 影響）。

## 容易遗漏的事

### 坑 1：在留資格 取得 30 日 期限 を「出生届 14 日 と一緒」と誤解
出生届 = 戸籍法（市役所）/ 在留資格 取得 = 入管法（入管）。別の役所、別の期限。30 日経過 + 在留資格 取得無し = 子女が 不法滞在 状態に。

### 坑 2：両親 外国籍 case で 日本国籍 取得できると誤解
日本は出生地主義ではなく血統主義。日本国内出生 でも 父母双方外国籍 = 子女 外国籍。父母の本国大使館 で出生届 + パスポート 取得 が必要。中国籍父母 case → 中国大使館 出生届 90 日以内（中国国籍法）。

### 坑 3：児童手当 認定請求 の 15 日 を逃すと 1 ヶ月分 損
出生日翌日起 15 日内 認定請求 = 出生月翌月 から支給。16 日目 以降 = 申請月翌月 から支給。月 ¥10,000 - ¥30,000 損。

> # ⚠️ 需创始人/書士复核：2024.10 児童手当 改正後の 第 3 子以降 ¥30,000 の算定における「子」の定義（22 歳までカウント）の正確な書き方
