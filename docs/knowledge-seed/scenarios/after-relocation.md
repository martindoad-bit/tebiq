# 搬家后必办 13 桩事

## 适用场景
日本国内 转居（含同市内 転居 / 市外 転出転入 / 県外 転出転入）的 在日外国人。海外 から 帰任 / 海外 への出国 是另一篇別扱い。本篇以新住所への入居日 = 第 0 日计时。

## 时间线

### 転居 14 日前
- 転出届（旧市役所 / 区役所 / 県外転居 时の事前届出）
- 不動産 解約通知（契約により 1-2 ヶ月前）
- 引越業者 予約

### 転居当日（第 0 日）
- 鍵渡し / 立会い
- 旧住所 退去 / 公共料金 メーター 検針

### 第 14 日内（法定期限）
- 転入届 / 転居届（新住所 市役所 / 区役所）
- 在留カード 住所変更（入管法 第 19-9 条 / 14 日内）
- 国民健康保険 切換
- 国民年金 住所変更
- 子女 学校転入届

### 第 30 日内
- 銀行口座 / クレカ 住所変更
- 携帯 / NHK / 公共料金 名義 + 住所変更
- 配偶 / 自分の会社 住所変更届
- 自動車 / 駐車場 住所変更
- 不動産 火災保険 / 賃貸保証会社 住所変更

## 必办事项（13 桩 / 按法定期限 + 重要性排序）

### 1. 転出届（市外 / 県外 転居 时）
- **在哪办**：旧住所 市役所 / 区役所 住民課
- **要什么材料**：転出届 / 在留カード / 印鑑（任意） / マイナンバーカード（あれば 特例 転出 = 紙の転出証明書 不要）
- **办不办的后果**：転出届 受理 → 転出証明書 交付（紙）or マイナ 特例。新住所自治体での転入手続に必須。市内同自治体内 転居 case は 転居届 のみで 転出届 不要。

### 2. 転入届 / 転居届（新住所到着 14 日内）
- **在哪办**：新住所 市役所 / 区役所 住民課
- **要什么材料**：転入届 / 転出証明書 / 在留カード（同居家族全員分） / マイナンバーカード（あれば）
- **办不办的后果**：住民基本台帳法 第 22 条：14 日 を超えて届出をしない場合 5 万円 以下の過料。住民票 が動かないと 国保 / 国民年金 / 児童手当 / 学校 等の手続全てが詰まる。

### 3. 在留カード 住所変更（入管 14 日内）
- **在哪办**：新住所 市役所 / 区役所（転入届 と同時 = 在留カード裏面に住所記載）
- **要什么材料**：在留カード（同居家族全員分） / 転入届
- **办不办的后果**：入管法 第 19-9 条：住居地変更日含 14 日 内届出義務。怠慢 = 20 万円 以下罰金（同法 第 71-2 条）。届出から 90 日 以上届出無しは 在留資格 取消事由（第 22-4 条第 1 項第 9 号）。
- **重要**：市役所 で住民票 + 在留カード裏面記載 が同時完了。入管 へ別途行く必要なし（住民登録 ある中長期 在留者 case）。

### 4. 国民健康保険 切換
- **在哪办**：旧 + 新 市役所 / 区役所 国保課
- **要什么材料**：旧住所 国保保険証 返還 / 新住所 国保 加入届 / 在留カード
- **办不办的后果**：国保法 第 9 条：転居日翌日起算 14 日以内 加入届 義務。怠慢期間中の医療費 = 自己負担 100% → 後日遡及還付 手続煩雑。健保 加入者は会社経由で住所変更のみで OK。

### 5. 国民年金 住所変更
- **在哪办**：原則 マイナンバー 連動で自動反映 / 反映漏れ時は 年金事務所 / 市役所 国民年金課
- **要什么材料**：年金手帳 / 基礎年金番号通知書 / 在留カード
- **办不办的后果**：マイナンバー 連動が機能しない外国人 case では 年金記録 漏失 → 老後年金 計算エラー。第 1 号 / 第 3 号 被保険者は要確認。第 2 号（厚生年金）は会社が手続。

### 6. 子女 学校転入届（公立 小中学校）
- **在哪办**：新住所 市役所 / 区役所 教育委員会 / 学務課
- **要什么材料**：転入学届 / 在学証明書（旧校発行） / 教科書給与証明書 / 在留カード / 健康保険証
- **办不办的后果**：転入届 提出時に教育委員会で同時手続可能 = 入学指定校通知 取得。 1-2 日で新校 通学開始。私立 / インター スクール は別個に学校窓口 + 通学変更届。

### 7. 銀行口座 住所変更
- **在哪办**：銀行窓口 / 網銀 / アプリ
- **要什么材料**：通帳 / キャッシュカード / 届出印 / 在留カード
- **办不办的后果**：住所変更しないと 重要書類（クレカ更新 / ローン審査 / 確定申告 還付通知）届かず。MUFG / SMBC / みずほ / ゆうちょ は網銀で完結可能 + 印鑑窓口必要 case あり。

### 8. クレジットカード 住所変更
- **在哪办**：各社 アプリ / 網銀 / 電話
- **要什么材料**：会員番号 / 本人確認書類
- **办不办的后果**：住所一致しない = カード更新時 配達不能 → カード失効。延滞でなくても クレヒス 上「住所不明」記載されることも。

### 9. 携帯 / インターネット 住所変更 + 移転手続
- **在哪办**：各社 アプリ / 店舗
- **要什么材料**：契約者本人確認書類 / 旧住所証明 / 新住所証明
- **办不办的后果**：固定回線（光 / ケーブル） は 移転工事 1-2 ヶ月前予約推奨。携帯番号は変更不要だが住所未更新 = 重要通知不達。

### 10. 配偶者会社 / 自社 住所変更届
- **在哪办**：人事 / 総務
- **要什么材料**：住所変更届（社内様式） / 通勤手当再申請
- **办不办的后果**：通勤手当 / 住宅手当 / 緊急連絡先 更新。年末調整 扶養 / 家族手当 反映。健保 加入者 の被扶養者住所も同時更新。

### 11. 不動産 旧契約 解約 + 新契約 + 火災保険
- **在哪办**：不動産会社 / 大家 / 損保会社
- **要什么材料**：解約通知書（1-2 ヶ月前必着） / 退去立会い / 鍵返却 / 新契約書 / 連帯保証人 / 火災保険新加入
- **办不办的后果**：解約通知遅延 = 翌月家賃発生。原状回復 立会いなし = 敷金返還トラブル。火災保険は 賃貸契約 ほぼ必須 = ¥15,000 - ¥25,000 / 2 年。

### 12. 電気 / ガス / 水道 / NHK 名義 + 住所変更
- **在哪办**：各社 アプリ / 電話 / Web
- **要什么材料**：お客様番号 / 旧住所最終検針 / 新住所開通予約
- **办不办的后果**：ガス は新住所開栓に立会い必須（午前 / 午後 指定）。電気 / 水道 は遠隔開通可能 case 多。NHK 受信契約 は住所変更でも継続。引越し後 2 - 3 日内 集金人訪問あり。

### 13. 自動車 / 駐車場 住所変更
- **在哪办**：警察署（車庫証明）/ 運輸支局（車検証）/ 運転免許センター
- **要什么材料**：車庫証明申請書 / 新住所 駐車場使用承諾書 / 車検証 / 運転免許証 / 在留カード / 住民票
- **办不办的后果**：道路運送車両法 第 12 条：住所変更日 から 15 日 内に車検証変更届義務。怠慢 = 50 万円 以下の罰金。運転免許証 住所変更も 道路交通法 第 94 条 違反 = 2 万円 以下罰金。

## 容易遗漏的事

### 坑 1：14 日 期限 を「実日」ではなく「営業日」と誤解
住民基本台帳法 / 入管法 の 14 日 は 暦日（ カレンダー上の日）。土日祝含む。引越が金曜日 → 翌々週金曜日 が期限。

### 坑 2：県内同自治体内転居 で「特に手続なし」と誤解
同自治体内でも 転居届（住民票内住所変更） が必要。在留カード 裏面記載も同様。引越し業者 / 不動産会社 の手続のみでは 行政手続完了せず。

### 坑 3：マイナンバー 通知 / マイナカード を旧住所宛で受取設定
マイナンバーカード 取得申請中 / 更新中 + 引越 = 通知書不達 で再申請。引越予定があれば申請を後ろ倒すか、転送届 を郵便局 に出す。

### 坑 4：自動車 住所変更 を放置 → 駐車違反 通知 / リコール 通知 不達
車検証 と運転免許証 の住所変更は別手続 + 別罰金。車庫証明 は新住所の警察署管轄。

> # ⚠️ 需创始人/書士复核：マイナンバーカード 特例 転出（紙の転出証明書 不要）の最新運用（2024 年 引越しワンストップ サービス）の正確な現状
