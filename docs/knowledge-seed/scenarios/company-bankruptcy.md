# 公司倒闭后必看清单

## 适用场景
对象：勤務先 が 破産 / 民事再生 / 会社更生 / 事実上倒産（事業活動停止 + 再開見込なし）した 在日就労者。
触发事件：破産管財人 通知書、賃金未払、事務所閉鎖、社長 連絡途絶。
不适用：単純 解雇 / 整理解雇（会社存続）→ 离职 30 日清单 を 参照。

## 时间线

### 第 1 日（倒産事実 認知日）
- 必办：書面 / メール / LINE / 給与明細 / タイムカード コピー 確保（賃金請求 証拠）。
- 必办：在留カード コピー、雇用契約書 原本 確保。

### 第 7 日内
- 必办：旧雇用主 / 破産管財人 から 離職票、源泉徴収票、健康保険資格喪失証明書 入手 試行。会社 連絡途絶 の 場合 は ハローワーク に 「事業所廃止」確認 依頼 → 仮 離職票 発行 可能。
- 必办：所在 市役所 へ 国民健康保険 / 国民年金 切替 申請。

### 第 14 日内（重要：入管 / 雇用保険 多在 14 日内）
- 必办：出入国在留管理庁「契約機関に関する届出」（消滅 を 選択）。
- 必办：ハローワーク へ 倒産特例 給付（特定受給資格者）申請 準備。
- 不届出 → 罰金 20 万円以下 + 永住申請 不利。

### 第 30 日内
- 必办：雇用保険 受給資格決定 → 待期 7 日 → 給付制限 なし → 即 給付開始（倒産 / 解雇 = 会社都合 = 特定受給資格者）。
- 必办：未払賃金立替払制度 申請準備（労働者健康安全機構 経由）。

### 之后（認定サイクル / 受給期間 等）
- 雇用保険 受給：90-330 日（年齢 + 加入年数 + 特定受給資格者 区分）。
- 未払賃金立替払：退職 6 ヶ月前 から 申請日 前日 までの 未払賃金（定期賃金 + 退職金）の 80% 立替。上限 88-296 万円（年齢別）。
- 求職活動 → 在留期間更新 時 求職実績 提示。

## 必办事项（按重要性排序）

### 1. 雇用保険 倒産特例給付（特定受給資格者）
- **在哪办**：住居地 管轄 [ハローワーク](https://www.hellowork.mhlw.go.jp/)。
- **要什么材料**：離職票（仮 でも 可）、在留カード、マイナンバー 確認、写真 2 枚、本人名義 通帳、印鑑。離職理由 欄 に「事業所 倒産」記載 確認。
- **办不办的后果**：給付制限 2 ヶ月 を 待たず 即給付。給付日数 も 一般離職者 より 30-120 日 多い。申請 しないと 受給期間 1 年 経過後 受給不可。

### 2. 未払賃金立替払制度（労働者健康安全機構）
- **在哪办**：管轄 労働基準監督署 → [独立行政法人 労働者健康安全機構](https://www.johas.go.jp/)。
- **要什么材料**：賃金台帳 / タイムカード コピー、雇用契約書、給与明細、破産手続開始決定書（裁判所 発行）、立替払請求書。
- **办不办的后果**：未払賃金 の 80%（上限 年齢別 88-296 万円）を 国 が 立替。申請期限 = 破産手続開始決定 等の 翌日 から 2 年。期限 過ぎ ると 立替 不可。

### 3. 入管「契約機関に関する届出」14 日内
- **在哪办**：[出入国在留管理庁 在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) または 入管 窓口。
- **要什么材料**：在留カード コピー、届出書（消滅 を 選択）。
- **办不办的后果**：未届 → 罰金 20 万円以下（入管法 71-3）。会社 倒産 でも 届出義務 は 在留者 本人 にある。

### 4. 国民健康保険 / 国民年金 切替
- **在哪办**：市役所 / 区役所 保険年金課。
- **要什么材料**：健康保険資格喪失証明書（会社 発行 不可 の 場合 は 離職票 / ハローワーク 受理票 で 代替可 の 自治体多）、在留カード、年金手帳。
- **办不办的后果**：無保険 → 医療費 10 割。年金 未納 → 老齢 / 障害 年金 受給 影响。

### 5. 賃金 / 退職金 債権 届出（破産管財人 経由）
- **在哪办**：破産管財人 から 送付 される「債権届出書」を 裁判所 / 管財人 へ 返送。
- **要什么材料**：賃金台帳、雇用契約書、給与明細、未払額 計算書。
- **办不办的后果**：届出 しないと 配当 受け取れない。立替払制度（80%）の 残 20% 部分 は この 配当 で 回収（実務 上 ほぼ ゼロ 配当 多 い）。

### 6. 在留期間更新 / 求職活動 計画
- **在哪办**：管轄 入管 / [出入国在留管理庁](https://www.moj.go.jp/isa/)。
- **要什么材料**：在留カード、パスポート、雇用保険 受給資格者証、求職活動 記録（ハローワーク 紹介状 等）、預金残高証明。
- **办不办的后果**：技人国 / 経営・管理 失業 3 ヶ月 超 → 在留期間更新 不利。6 ヶ月 超 求職実績 なし → 在留資格取消（入管法 22-4）対象。

### 7. 源泉徴収票 / 確定申告
- **在哪办**：破産管財人 経由 で 入手 試行 → 入手不可 の 場合 [税務署](https://www.nta.go.jp/) に「源泉徴収票不交付の届出書」提出 → 給与明細 ベース で 確定申告 可能。
- **要什么材料**：給与明細 全月分、雇用契約書、銀行振込 記録。
- **办不办的后果**：未申告 → 還付金 受け取れない。倒産 で 年末調整 されない 場合 ほぼ 確実 に 還付 発生。

## 容易遗漏的事

### 坑 1：会社 連絡途絶 で 離職票 出ない → 諦める
ハローワーク に 行けば「事業所廃止」確認 → 仮 離職票 発行 可能。離職票 なし でも 給付申請 開始 でき、後日 補正。会社 倒産 で 連絡 つか ない こと は ハローワーク も 想定済み。

### 坑 2：未払賃金立替払制度 を「倒産 した 会社 から しか 取れ ない」と 誤解 して 申請 しない
立替払 は 国（労働者健康安全機構）が 立替。会社 から 取る わけ で は ない。法的破産 + 事実上倒産（中小企業 は 労基署 認定 で OK）両方 対象。退職 6 ヶ月前 以降 の 未払賃金 + 退職金 が 80% カバー。

### 坑 3：在留 焦って「とりあえず アルバイト」→ 在留資格 違反
技人国 / 経営・管理 等 就労ビザ で「資格外活動許可」なし に アルバイト → 不法就労（入管法 73-2）。倒産 後 求職中 でも 雇用保険 給付 受け取り は OK。違う 業種 で 働く 場合 は 在留資格変更 / 資格外活動許可 が 必要。
