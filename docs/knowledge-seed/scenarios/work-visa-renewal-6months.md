# 工作签证到期前 6 个月准备

## 适用场景
- 持 技人国 / 経営・管理 / 高度専門職 / 特定技能 / 企業内転勤 等就労系 在留資格、在留期限 6 ヶ月以内に到来する人。
- 過去 1 年で転職・離職・公司情報変更・婚姻状態変化があった人 — 通常更新より準備項目多。
- 不适用：留学 / 家族滞在 / 永住者 — 别条目处理（在留資格 由更新審査基準が異なる）。

## 时间线

### 6 ヶ月前（準備開始）
- 必办：在留カード「在留期限」確認。**更新申請 受付は 期限満了 3 ヶ月前から**（高度専門職 6 ヶ月前から、一部例外あり）。
- 必办：過去 1 年の **所属機関 届出 履歴** を在留申請オンラインシステム で確認。届出漏れがあれば即補出。
- 可办：直近年 課税証明書 / 納税証明書 を 区役所 / 税務署 で取得（住民税 完納確認用）。
- 可办：過去 1 年 年金 / 健康保険 加入記録 を 年金事務所 / 健康保険組合 で確認。

### 3 ヶ月前（申請受付開始）
- 必办：申請書類 一式の準備開始 — **在留期間更新許可申請書**、**理由書**、**勤務先 関連書類**、**個人 関連書類**。
- 必办：所属機関カテゴリー（カテゴリー 1〜4）に応じた書類セット確定。**カテゴリー 1**（上場企業 / 国・地方公共団体 等）= 簡素化、**カテゴリー 4**（その他）= 全件提出。
- 必办：直近年 **源泉徴収票** を会社から取得。
- 可办：オンライン申請 アカウント 取得 / 認定 — 本人申請可、法人 / 行政書士 経由も可。

### 1 ヶ月前（申請提出）
- 必办：**在留期間更新許可申請** を 入管 窓口 or オンライン 提出。**特例期間**（在留期限経過後も 申請結果出るまで最大 2 ヶ月、就労継続可）が走る。
- 必办：在留カード 紛失 / 損傷 等 個別事情があれば 同時届出。
- 可办：申请受理票 を 雇用主 / 公司 経理 に共有（特例期間 の証明用）。

### 申请後（審査期間 通常 2 週間〜2 ヶ月）
- 可办：入管 から **追加書類提出依頼**（補正通知）が来た場合 = 期限内に対応。**期限超過 = 不許可リスク高**。
- 可办：審査状況 を 在留申請オンラインシステム で確認。

### 結果通知（許可 or 不許可）
- 必办：許可 = 入管 窓口で 新在留カード 受取（手数料 6,000 円）or オンライン 配送（4,000 円）。
- 必办：不許可 = **「出国準備期間」(30 日 / 31 日 / 60 日) の 在留資格「特定活動」付与**、不許可理由 確認 + 再申請 / 帰国 を選択。

## 必办事项（按重要性排序）

### 1. 在留期間更新許可申請 提出（受付期間 厳守）
- **在哪办**：所轄 出入国在留管理局（東京入管 / 大阪入管 等）窗口 or [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/)。
- **要什么材料**：在留期間更新許可申請書、写真（4cm × 3cm、3 ヶ月以内撮影）、護照、在留カード、所属機関カテゴリー別 書類セット（雇用契約書 / 在職証明書 / 給与明細 / 源泉徴収票 / 法人登記事項証明書 / 決算書 等）、手数料 6,000 円（許可時納付）。
- **办不办的后果**：**在留期限 経過 + 申請なし = 不法滞在**。在留期限 当日まで申請出せば **特例期間** で適法滞在継続。期限超過 = **退去強制 / 出国命令** 対象、再来日 5 年〜10 年禁止。

### 2. 所属機関 届出 履歴 補完
- **在哪办**：在留申請オンラインシステム or 入管 窓口（紙提出）。
- **要什么材料**：契約機関に関する届出 / 活動機関に関する届出（転職・離職時 14 日以内、入管法 第 19-16 条）。
- **办不办的后果**：届出漏れ = **公的義務不履行 として更新審査で「素行善良」評価マイナス**。複数年に渡って届出漏れがある場合、補正通知 or 直接 不許可 の事例あり。

### 3. 住民税 / 所得税 / 年金 / 健康保険 完納確認
- **在哪办**：住民税・所得税 = 区役所 / 税務署 で **納税証明書** 取得。年金 / 健康保険 = 年金事務所 / 健康保険組合 で 加入記録 + 納付記録 取得。
- **要什么材料**：本人確認書類、印鑑、手数料（300 円〜400 円 程度）。
- **办不办的后果**：**未納 / 滞納 = 公的義務不履行 として更新審査で重大マイナス評価**。永住 申請ほど厳しくないが、技人国・経営管理 でも 特に **住民税 滞納 + 年金 未加入** は不許可理由になり得る。**滞納分は申請前に完納 + 納税証明書 取得**。

### 4. 雇用契約 / 業務内容 と 在留資格 の整合性確認
- **在哪办**：自己 + 公司 経理 / 法務 + 必要に応じ 行政書士。
- **要什么材料**：現在の雇用契約書、職務記述書、給与明細、過去 1 年の業務実績資料。
- **办不办的后果**：技人国 = 大学卒業学部・専攻と業務内容の関連性が問われる。**転職後の業務内容が原 在留資格 認定範囲外 = 在留資格変更 必要**、誤って更新申請 → 不許可。経営・管理 = **2024 年改正で出資額 500 万円 → 3000 万円 + 常勤職員要件強化** の方向、要件再チェック必須。

### 5. 公司書類（カテゴリー別）整備
- **在哪办**：会社 経理・人事 部門。
- **要什么材料**：法人登記事項証明書（3 ヶ月以内）、決算書 写し（直近 1 期）、源泉徴収税額表 / 給与所得・退職所得等の所得税徴収高計算書、会社案内、雇用契約書 / 労働条件通知書、会社四季報 写し（カテゴリー判定用）。
- **办不办的后果**：会社書類 不備 = 申請受付不可 or 補正通知 → 審査長期化。**カテゴリー判定誤り** = 必要書類不足で 補正通知 → 期限超過リスク。

### 6. 補正通知 への対応（期限内）
- **在哪办**：補正通知 記載の窓口 / 担当部署 / オンライン。
- **要什么材料**：通知書 で要求された 追加資料、理由書（必要時）。
- **办不办的后果**：**補正期限（通常 2 週間〜1 ヶ月）超過 = 在留資格 不許可** 直行。**不許可後の再申請** は同年内 2 度目の申請 = 審査さらに厳格化。

### 7. 申請後の海外渡航 — 再入国許可 確認
- **在哪办**：入管 窓口 or 空港の再入国手続。
- **要什么材料**：在留カード（裏面 在留期限 + 再入国スタンプ）、護照、申請受理票（特例期間 中の場合）。
- **办不办的后果**：**特例期間 中の再入国 = みなし再入国（1 年以内）or 再入国許可（事前申請）必要**。**特例期間 中に再入国許可なしで出国 = 在留資格 失効**。海外滞在中に許可通知が出ても受取不可、最悪 再申請。

## 容易遗漏的事

### 坑 1：受付開始 3 ヶ月前 を「申請最早タイミング」と誤解
申請可能 = **在留期限満了の 3 ヶ月前から**（高度専門職 等は 6 ヶ月前から）。「6 ヶ月前から準備」= 書類収集・状況整理。「3 ヶ月前から提出可」= 実際の申請受付。**3 ヶ月前を過ぎてから準備開始すると、書類不備時に補正対応で 特例期間 まで食い込み、最悪 期限超過**。書類は 4 ヶ月前完成、3 ヶ月前最初の窓口で 提出が安全圏。

### 坑 2：転職後 14 日以内 届出 漏れの遡及補正
入管法 第 19-16 条 — 契約機関の消滅・離脱・移籍・新規契約 = 14 日以内 届出。**1 年以上前の漏れも申請可、ただし 「届出遅延 + 理由書」必要**。届出漏れを「無かったこと」にして更新申請 = 入管 が 雇用保険 / 厚生年金 加入記録 から発覚 → 信頼性 大幅低下。**漏れがあったら 必ず 申請前に補出 + 理由書添付**。

### 坑 3：給与額 が 在留資格 認定時より大幅減
**就労系 在留資格 は「日本人と同等以上の報酬」要件**（入管法施行規則 別表第 7-1）。認定時 月給 25 万円 → 更新時 月給 18 万円 = 業務内容変化 / 会社業績悪化 を疑われる。**理由書 で減額理由 + 業務内容継続性 を明示説明**。完全実績ベース給（インセンティブ主体）の場合、年収ベース で日本人同等性を立証。減額が 30% 超 = **在留資格 該当性 に疑義** で不許可リスク。
