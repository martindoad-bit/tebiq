# 刚来日本第 1-30 天必办清单

## 适用场景
- 持中长期 在留資格（90 日以上）首次入境日本的人。
- 在 成田 / 羽田 / 関西 / 中部 / 福岡 / 新千歳 / 那覇 主要空港已交付 在留カード 的人。
- 短期滞在（90 日以内观光签）不适用 — 不办住民登録、不能加入国保。

## 时间线

### 第 1 日（抵达当天）
- 必办：在主要空港交付 在留カード（裏面「住居地」栏空白，需 14 日内补登记）。
- 可办：购买 SIM 卡 / pocket WiFi 临时通信。日本主要空港均有柜台。
- 可办：入住临时住所 / Airbnb / 公司寮（住居地未确定时不能立即办住民登録）。

### 第 7 日（第 1 周内）
- 必办：确定长期居住地址、签订 賃貸借契約 或公司寮入居手续。
- 可办：在 区役所 / 市役所 周边踩点，确认 住民課 窗口受付时间（通常平日 8:30-17:00，部分自治体周日 / 夜间窗口可办）。

### 第 14 日（第 2 周内 — 法定截止日）
- 必办：到居住地的 区役所 / 市役所 提交 **転入届**，同时完成 在留カード 住居地登载。这两件事是同一柜台同时办，**14 日内未办违反入管法第 19-7 条**。
- 必办：申请 **マイナンバー通知**。住民登録完成后约 2-3 周后，「マイナンバー通知書」会寄到登记地址。
- 必办：加入 **国民健康保険**（不在公司社保覆盖时）。住民登録 当天可同窗口受理，自抵达日起算保険料。
- 必办：加入 **国民年金**（20-59 歳，不在 厚生年金 覆盖时）。1 号被保険者登记。
- 可办：申请 **印鑑登録**（不动产 / 车 / 公证场景需要，通常生活前 30 日不强制）。

### 第 30 日（第 1 个月内）
- 必办：到银行 / 信用金庫 / ゆうちょ 开户。无 マイナンバー 仍可开 ゆうちょ 普通口座 + 部分 ネット銀行（楽天 / 住信 SBI 等），但 都市銀行（三菱 UFJ / みずほ / SMBC）多数要求入国 6 個月以上 + マイナンバー。
- 必办：办理 携帯電話 主回线（要 在留カード + 银行口座 + 印鑑 / サイン）。MVNO（IIJmio / mineo / 楽天モバイル 等）门槛较低。
- 可办：マイナンバーカード 申请（通知書 收到后通过 申请）。

## 必办事项（按重要性排序）

### 1. 転入届 + 在留カード 住居地登载（**第 14 日截止**）
- **在哪办**：居住地所在 区役所 / 市役所 住民課 窗口（一窗同时办两件事）。
- **要什么材料**：在留カード（本人 + 同伴家族全员）、护照、賃貸借契約書 或居住证明（持家者出登記簿等本）、印鑑（无印鑑 用 サイン 也可）。
- **办不办的后果**：抵达日起 14 日内未办 → 违反 入管法 第 19-7 条，「在留資格取消」对象（同法第 22-4 条第 1 項第 8-2 号），且会被记录在 入管 系统。续签 / 永住 申请时素行善良评价直接受影响。

### 2. 国民健康保険 加入（住民登録 同日办）
- **在哪办**：区役所 / 市役所 国保年金課（同一日同一柜台流程办理）。
- **要什么材料**：在留カード、护照、住民票（住民登録完成后即可发行）、雇用契約書 或退職証明（如无社保）。
- **办不办的后果**：未加入 → 医疗费 100% 自付（不是 30%）。若后期补加入，**保険料 自抵达日 起 追溯征收**（最多 2 年），不会因「没办」而免除。

### 3. 国民年金 加入（20-59 歳）
- **在哪办**：区役所 / 市役所 国保年金課（与 国保 同窗）。
- **要什么材料**：在留カード、護照、雇用関係なし 时为 1 号被保険者。
- **办不办的后果**：保険料 自加入资格取得日 起追溯征收。免除 / 猶予 制度需主动申请（学生納付特例 / 免除申請）。永住 / 帰化 审查会查 年金 加入 + 缴纳记录。

### 4. 银行口座 开设
- **在哪办**：ゆうちょ銀行（最低门槛）/ ネット銀行（楽天・住信SBI・PayPay）/ 都市銀行（要求最严）。
- **要什么材料**：在留カード（裏面有住居地）、住民票、印鑑（部分行可サイン）、在职证明 / 学生証（部分行要求）。
- **办不办的后果**：无银行口座 → 公司无法发薪、无法签携帯契約、无法办水電瓦斯 自動引落、无法 メルカリ・PayPay 等收款。**部分都市銀行 要求入国 6 個月以上**，初期建议先开 ゆうちょ 或 ネット銀行。

### 5. 印鑑 制作 + 印鑑登録
- **在哪办**：印鑑屋 / ハンコ通販 制作（実印用）→ 区役所 / 市役所 住民課 印鑑登録。
- **要什么材料**：制作好的实印（外径 8mm-25mm 内方形）、在留カード、本人来店。
- **办不办的后果**：不动产购买、车辆登记、公正証書 等场景需要 印鑑登録証明書。日常生活初期不强制，但 賃貸 契約 / 入社手続 部分仍要求实印 + 印鑑証明。

### 6. 携帯電話 主回线契約
- **在哪办**：MNO（NTT docomo / au / SoftBank / 楽天モバイル）店舗 或 MVNO 网申。
- **要什么材料**：在留カード（在留期間 1 年以上一般无问题，1 年未満可能拒否或要求一括払）、银行口座 / クレジットカード、住民票（部分要求）。
- **办不办的后果**：无主回线 → 一切「本人確認」难以通过：銀行 ATM 限額・各種 SMS 認証 不能使用。空港 SIM 仅短期方案。

### 7. マイナンバー 通知書 接收 + 個人番号確認
- **在哪办**：登記住居地。住民登録后 2-3 週后由 J-LIS 寄送「個人番号通知書」到登記地址（不是 マイナンバーカード 本体）。
- **要什么材料**：登記時 確実接收 = 不需材料。本人 不在不可受取（書留）。
- **办不办的后果**：未收到 → 多数手续（入社、銀行、税務）替代材料是 住民票（マイナンバー記載あり）。但 マイナンバーカード 本身需另行申请，最长发行 1-2 个月。

## 容易遗漏的事

### 坑 1：在留カード「住居地」栏 14 日窗口的起算
14 日不是从「找到房」算，是从**抵达日本**起算。临时住宿 / Airbnb 算入这 14 日。如果第 14 日仍未确定永住住所，先以临时住所先做 転入届，搬家时再办 転居届（同一市内）/ 転入届（跨市）。空白超 14 日且无合理理由 → 在留資格取消対象。

### 坑 2：国保 / 国民年金 保険料 追溯征收
很多人以为「我没办，就没产生费用」— 错。保険料 自取得资格之日起算（即抵达日 + 住民登録日中较晚者），后期补办会一次性追溯。**国保 最多 2 年追溯**（地方税法 第 18 条 + 国民健康保険法 第 110 条）、**国民年金 2 年保険料免除可申请**。完全不办 = 帐上欠费。

### 坑 3：ゆうちょ 以外的银行 多要求 入国 6 個月以上
都市銀行（三菱 UFJ / みずほ / SMBC）多数对外国人 新规口座 要求「入国 6 個月以上」+ マイナンバー。来日初期会被劝返。**实战路径**：ゆうちょ 普通口座（在留カード ありで即日可）→ 半年后追加 楽天 / 住信 SBI ネット銀行 → 必要时再加 都市銀行。
