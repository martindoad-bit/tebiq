# 怀孕期间补助申请清单

## 适用场景
对象：在日妊娠中 / 出産予定者（在留資格 不限）。触发：医院确诊心音 / 取得 妊娠届出書。不适用：日本国外出産（出産育児一時金 海外分娩特例 別途）。

## 时间线

### 妊娠初期（〜妊娠 11 週）
- 必办：医院确诊后 → 市役所 / 区役所「妊娠届」提出 → 母子健康手帳 + 妊婦健診受診票（補助券）受領。
- 必办：会社へ 妊娠報告（産前産後休業 / 育児休業 計画）。

### 妊娠中期（妊娠 12-27 週）
- 必办：定期 妊婦健診（補助券利用 / 自己負担 ¥0-3,000 / 回）。
- 必办：出産予定 病院 / 助産院 確定 + 直接支払制度 / 受取代理制度 申込。

### 妊娠後期（妊娠 28 週〜）
- 必办：産休（出産予定日 6 週前）开始 → 健康保険 出産手当金 申請準備。
- 必办：育児用品 / 退院后 居住場所 確定。

### 出産後 1 月内
- 必办：出生届（出生 14 日内、市役所 戸籍課）+ 児童手当 認定請求 + 健康保険 加入（被扶養者）。
- 必办：出産育児一時金 申請（直接支払制度 利用時 病院手続；現金受給時 健康保険組合）。

### 育児期（出産 8 週〜 子 1 歳 / 1 歳半 / 2 歳）
- 必办：育児休業給付金 申請（ハローワーク 経由 / 会社）。
- 必办：保育園 申込（市役所 子ども家庭課）。

## 必办事项

### 1. 妊娠届 + 母子健康手帳
- **在哪办**：住民登録地 市役所 / 区役所 子ども家庭課。
- **要什么材料**：医院 妊娠診断書 / マイナンバー / 在留カード / 健康保険証。
- **办不办的后果**：母子手帳 不取得 = 妊婦健診 補助券 利用不可（自費 ¥5,000-10,000 × 14 回 = ¥70,000-140,000 損失）。

### 2. 妊婦健診費用助成（補助券 / 受診票）
- **在哪办**：母子手帳 同時交付。
- **要什么材料**：母子手帳 受診票 + 健康保険証。
- **办不办的后果**：14 回分（標準）助成、地域差有 — 東京都 1 回 ¥5,000-10,000 上限。

### 3. 出産育児一時金 ¥500,000
- **在哪办**：直接支払制度 = 病院 手続；現金受取 = 加入 健康保険組合 / 国民健康保険（市役所）。
- **要什么材料**：直接支払 合意文書 / 出生証明書 / 医療機関明細 / 健康保険証。
- **办不办的后果**：2023/4 改定 ¥420,000 → ¥500,000、双子 ¥1,000,000。出産費用 平均 ¥45-60 万、未申請 = 全額自己負担。産科医療補償制度 加入分娩 ¥500,000、未加入分娩 ¥488,000。

### 4. 出産手当金（健康保険 加入者）
- **在哪办**：協会けんぽ / 健保組合（会社経由）。
- **要什么材料**：申請書（医師記入欄 + 会社記入欄）+ 健康保険証 + 通帳。
- **办不办的后果**：産前 42 日 + 産後 56 日（双子産前 98 日）= 標準報酬日額 × 2/3。月 30 万円 = 約 67 万円 受取。国民健康保険 加入者は 対象外。

### 5. 育児休業給付金（雇用保険 加入者）
- **在哪办**：ハローワーク（会社経由）。
- **要什么材料**：受給資格確認票 / 育児休業給付金支給申請書 / 賃金台帳 / 出勤簿。
- **办不办的后果**：育休開始〜 180 日 = 賃金 67% / 181 日〜 1 歳 = 50%、最大 子 2 歳まで延長可。月収 30 万円 = 月 20 万円 × 12 ヶ月 = 約 240 万円。

### 6. 児童手当
- **在哪办**：市役所 子ども家庭課（公務員は職場）。
- **要什么材料**：認定請求書 + 健康保険証 + 通帳 + マイナンバー。
- **办不办的后果**：2024/10 改定 — 所得制限撤廃、3 歳未満 月 ¥15,000、3 歳〜高校生 月 ¥10,000、第 3 子以降 月 ¥30,000。請求月翌月から支給、遡及不可。

### 7. 出生届 + 健康保険 加入
- **在哪办**：出生届 = 市役所 戸籍課（14 日内）；健康保険 = 会社（被扶養者届）or 市役所（国保）。
- **要什么材料**：出生証明書 / 母子手帳 / 健康保険証 / 在留カード（両親）。
- **办不办的后果**：14 日超過 = 戸籍法上 過料（実務的には説明書添付で受理可）；健康保険 未加入 = 乳児医療費 全額自己負担。

### 8. 子の 在留資格（在留資格認定証明書 / 永住申請）
- **在哪办**：出入国在留管理庁。
- **要什么材料**：出生 30 日内届出。両親 永住者 = 子も 永住申請可（簡易手続）。
- **办不办的后果**：30 日超過 = 在留資格 取得不可リスク、出国 / 再入国 不可。

## 容易遗漏的事

### 坑 1：扶養手当金 vs 育児休業給付金
産休 = 出産手当金（健康保険）；育休 = 育児休業給付金（雇用保険）。前者は健康保険組合、後者はハローワーク経由 — 別制度別申請。会社の人事に「育休ください」だけ言うと出産手当金の手続が漏れる例多発。

### 坑 2：国民健康保険 加入者は 出産手当金 対象外
パート / アルバイト / 自営業（国民健康保険）= 出産手当金 受給不可。出産育児一時金 ¥500,000 は受取可。配偶者の被扶養者 = 出産手当金 不可（自身が被保険者でないため）。

### 坑 3：保育園 申込 タイミング
0 歳児 4 月入園 = 前年 10-11 月 申込（東京 23 区 平均 倍率 1.5-3 倍）。育休 1 歳 ピッタリ復職 = 0 歳 4 月 入園が最確率高。妊娠中期から保育園リサーチ + 4 月一斉申込 締切 確認必要。
