# 结婚后 30 日内必办清单

## 适用场景
刚提出 婚姻届 / 国際結婚 已成立的在日外国人。配偶为日本人或外国人均适用，但具体材料不同。仅有「結婚予定」尚未提出 婚姻届 的人不适用，本篇按照 婚姻届 受理日 = 第 0 日计时。

## 时间线

### 第 1 日（婚姻届 受理当天）
- 婚姻届 提出（市役所 / 区役所 戸籍課，24 小时收件）
- 取得 婚姻届受理証明書（当天可领，¥350 左右）

### 第 7 日内
- 戸籍謄本 反映婚姻事项后取得（日本人配偶方戸籍，反映需 1-2 周）
- 在留カード 住所変更（如同时搬家，14 日内）
- 配偶为日本人时，开始准备 在留資格変更（家族滞在 / 技人国 等 → 日本人配偶者等）

### 第 14 日内
- 厚生年金 / 健康保険 扶養 加入届（配偶会社人事，资格取得日翌日起 5 日内是公司侧期限，本人侧 14 日是稳健线）
- 国民健康保険 世帯主変更（双方均为外国人 + 国保 加入者时）
- 印鑑登録（如换姓 / 改名，旧印鑑登録 抹消 + 新规登録）

### 第 30 日内
- 在留資格変更許可申請 提出（拟变更为 日本人配偶者等 / 永住者配偶者等 时）
- 銀行口座 / クレカ 名义 + 住所変更
- 公司人事 通报 + 扶養手当 申请
- 確定申告 配偶者控除 / 配偶者特別控除 准备资料

## 必办事项（按重要性排序）

### 1. 婚姻届 提出
- **在哪办**：双方任一方 本籍地 / 所在地 的 市役所 / 区役所 戸籍課
- **要什么材料**：婚姻届書（証人 2 名签名）/ 双方身份证件 / 外国人方 婚姻要件具備証明書（本国大使馆开具，中国籍需「无配偶声明书」+ 翻译）/ 戸籍謄本（日本人方）
- **办不办的后果**：婚姻届 未受理则婚姻关系不成立。後续所有 在留資格 / 扶養 / 税务手续均无法启动。

### 2. 在留カード 住所変更（如搬家）
- **在哪办**：新住所 市役所 / 区役所 住民課
- **要什么材料**：在留カード 原件 / 転入届
- **办不办的后果**：入管法第 19-9 条规定 14 日内届出义务。怠慢可处 20 万円以下罚金，且影响下次 在留期間更新 / 永住申請 的素行善良评价。

### 3. 在留資格変更（拟变更为配偶者签证时）
- **在哪办**：所在地 出入国在留管理庁
- **要什么材料**：在留資格変更許可申請書 / 戸籍謄本（婚姻反映后）/ 双方住民票（世帯全員 + マイナンバー 省略）/ 配偶者方 課税証明書 + 納税証明書（直近 1 年）/ 質問書（婚姻经过详述）/ 婚姻 snapshot 与同居证据 / 在留カード
- **办不办的后果**：现有 在留資格（如 家族滞在 / 留学 / 技人国）继续有效则不强制变更，但 日本人配偶者等 在就労制限 / 永住申請要件（婚姻 3 年 + 在日 1 年）上更宽。

> # ⚠️ 需创始人/書士复核：日本人配偶者等 申請的「3 ヶ月以内 受理」实务期待与法定期限的差异表述

### 4. 厚生年金 / 健康保険 扶養 加入
- **在哪办**：配偶就業会社 人事 / 健保組合
- **要什么材料**：被扶養者異動届 / 戸籍謄本 或 婚姻届受理証明書 / 配偶（被扶養者）所得証明书（130 万円未满证明） / マイナンバー
- **办不办的后果**：被扶養者认定后保险料 0 円 + 国民年金 第 3 号被保険者（保険料 0 円但年金加算）。怠慢则被扶養者需自行加入 国保 + 国民年金 月约 ¥3 万。

### 5. 印鑑登録 更新（换姓时）
- **在哪办**：住民登録地 市役所 / 区役所
- **要什么材料**：旧 印鑑登録証 / 新印鑑（姓变更后） / 在留カード
- **办不办的后果**：不动产契约 / 自動車登録 / 銀行 重要手续 都需 印鑑登録証明書。旧姓印鑑 与 戸籍 / 住民票 上姓不一致时，証明書 不能使用。

### 6. 銀行口座 名義 + 住所変更
- **在哪办**：银行窗口 / 网银
- **要什么材料**：通帳 / キャッシュカード / 旧届出印 + 新届出印 / 戸籍謄本 或 婚姻届受理証明書 / 在留カード
- **办不办的后果**：振込 / 自動引落 仍可用旧姓但需 1 年内整理。新姓不更新则信用卡审查 / 住宅ローン 时姓名不一致被退回。

### 7. クレジットカード 名義変更
- **在哪办**：各发卡公司客服 / APP
- **要什么材料**：戸籍謄本 / 新印鑑 / 在留カード
- **办不办的后果**：旧姓カード 即使可用也影响 利用明細 与 確定申告 整理。

### 8. 国民健康保険 世帯主変更（双方均加入国保时）
- **在哪办**：住民登録地 市役所 / 区役所 国保課
- **要什么材料**：国民健康保険被保険者証（双方）/ 婚姻届受理証明書 / 在留カード
- **办不办的后果**：世帯单位计算 保険料，未合并则双方各自 世帯主 仍按单身保险料计算。

### 9. 国民年金 種別変更（被扶養者 → 第 3 号 时）
- **在哪办**：年金事務所 / 配偶会社 经由健保
- **要什么材料**：第 3 号被保険者該当届 / 戸籍謄本 / 年金手帳
- **办不办的后果**：未届出则继续按 第 1 号 自缴月 ¥17,000 左右。第 3 号 期间保険料 0 円但 年金 计入。

### 10. 公司人事届出
- **在哪办**：自己就業会社 人事
- **要什么材料**：婚姻届出書（社内）/ 扶養手当申請書 / 戸籍謄本
- **办不办的后果**：扶養手当 / 住宅手当 / 通勤手当 调整、年末調整 配偶者控除 反映、緊急連絡先 更新均依赖此届出。

### 11. 確定申告 / 年末調整 配偶者控除 准备
- **在哪办**：配偶所属公司 人事 经由 年末調整 / 自行 確定申告
- **要什么材料**：配偶 マイナンバー / 配偶 所得見積額 / 戸籍謄本
- **办不办的后果**：配偶者控除 最大 ¥38 万、配偶者特別控除 最大 ¥38 万。漏报则当年 所得税 / 住民税 多缴。

## 容易遗漏的事

### 坑 1：以为「婚姻届 受理 = 在留資格 自动变更」
婚姻届 是 戸籍 上的婚姻成立。在留資格（家族滞在 / 留学 / 技人国 → 日本人配偶者等）必须另行 在留資格変更許可申請。许多人持现有 在留資格 不做变更，日常无碍，但永住申請 要件（婚姻 3 年 + 实际同居 + 日本人配偶者等 在留 1 年）就要从 日本人配偶者等 取得日 重新计时。

### 坑 2：婚姻要件具備証明書 取得花费 1-2 个月
中国籍方需要 中国大使館 / 領事館 开具「无配偶声明书」+ 中文公证 + 日文翻译。手续 平均 1 个月。建议在拟提出 婚姻届 前 2 个月启动准备。

### 坑 3：忘记 国民年金 第 3 号 届出 → 未納期間 累积
日本人配偶 公司不主动办的 case 屡见。第 3 号 该当届 忘提则保險料 ¥0 期间被记为 第 1 号 未納，老後年金 减額。事後可遡及 2 年补正届出。
