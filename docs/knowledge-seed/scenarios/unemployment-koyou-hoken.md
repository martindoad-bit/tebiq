# 失业了：雇用保険 申請 全流程

## 适用场景
对象：雇用保険 被保険者 期間 を 持ち、離職 した 在日 就労者。
触发事件：離職票-1 / 離職票-2 受領、または「事業所廃止」状態 で ハローワーク へ。
不适用：雇用保険 被保険者期間 不足（自己都合 = 離職前 2 年内 通算 12 ヶ月 / 会社都合 = 1 年内 通算 6 ヶ月）。被保険者 ではない 役員 / 個人事業主。

## 时间线

### 第 1 日（離職票 入手日）
- 必办：離職票-1 / 離職票-2 内容 確認（離職理由、賃金支払状況、被保険者期間）。
- 必办：離職理由 欄 に 異議 あり の 場合 → ハローワーク で 異議申立 可能。

### 第 7 日内
- 必办：住居地 管轄 ハローワーク で 求職申込 + 雇用保険 受給資格決定 申請。
- 必办：受給資格決定日 から 待期 7 日 開始。

### 第 14 日内（重要：入管 / 雇用保険 多在 14 日内）
- 必办：入管「契約機関に関する届出」14 日内。
- 必办：雇用保険 受給説明会 出席（指定日、原則 1 回）。

### 第 30 日内
- 自己都合：給付制限 2 ヶ月 中（無給）。求職活動 実績 蓄積。
- 会社都合：待期 7 日 経過後 即 給付開始 → 第 1 回 認定日（受給資格決定 から 約 4 週後）。

### 之后（認定サイクル / 受給期間 等）
- 4 週（28 日）に 1 回 認定日。失業認定申告書 + 求職活動 実績 2 回 以上（自己都合 給付制限中 は 3 回 以上）。
- 受給期間：離職翌日 から 1 年（妊娠 / 出産 / 育児 / 介護 / 病気 等 で 30 日 以上 働けない 場合 最大 4 年 延長）。

## 必办事项（按重要性排序）

### 1. ハローワーク 求職申込 + 受給資格決定
- **在哪办**：住居地 管轄 [ハローワーク](https://www.hellowork.mhlw.go.jp/)。
- **要什么材料**：離職票-1 + 離職票-2、在留カード、マイナンバー 確認書類（マイナンバーカード or 通知カード + 身分証）、写真 2 枚（縦 3.0×横 2.4cm、3 ヶ月内）、本人名義 預金通帳、印鑑（認印 可）。
- **办不办的后果**：申請 しないと 給付 開始 されない。受給期間 1 年 内 に 申請 + 給付日数 受給 完了 が 原則。

### 2. 待期 7 日 経過 確認
- **在哪办**：ハローワーク（管轄）。
- **要什么材料**：受給資格者証（受給説明会 で 交付）。
- **办不办的后果**：待期 7 日 は 全員 一律。会社都合 / 自己都合 を 問わず 給付対象 にならない 期間。アルバイト / 内職 した 日 は 待期 完成 が 遅れる。

### 3. 給付制限 2 ヶ月（自己都合 の 場合）
- **在哪办**：給付制限 中 も 認定日 出頭 必要。
- **要什么材料**：失業認定申告書、求職活動 実績 3 回 以上。
- **办不办的后果**：給付制限 中 は 給付金 振込 なし。2025/4/1 改正：自己都合 給付制限 は 5 年内 2 回 まで 2 ヶ月、3 回目 以降 3 ヶ月。重大 な 自己責任 解雇 は 3 ヶ月。

### 4. 認定日 28 日 サイクル（求職活動 実績 報告）
- **在哪办**：ハローワーク（管轄）。
- **要什么材料**：失業認定申告書、受給資格者証、求職活動 実績（求人応募 / セミナー 受講 / 職業相談 等 2 回 以上 / 4 週）。
- **办不办的后果**：認定日 欠席 → 当該 認定対象期間 給付 受け取れない。次回 認定日 まで 給付 ストップ。

### 5. 給付金額 計算 + 受給期間 把握
- **在哪办**：[ハローワーク 受給資格者証 確認](https://www.hellowork.mhlw.go.jp/)。
- **要什么材料**：受給資格者証（基本手当日額 + 所定給付日数 記載）。
- **办不办的后果**：基本手当日額 = 離職前 6 ヶ月 平均賃金日額 × 50-80%（年齢 + 賃金 で 変動）。所定給付日数 = 90-360 日（年齢 + 加入年数 + 離職理由）。受給期間 1 年 過 ぎる と 残日数 受給不可。

### 6. 公共職業訓練 / 教育訓練給付
- **在哪办**：[ハローワーク](https://www.hellowork.mhlw.go.jp/)（公共職業訓練 紹介） / 教育訓練給付 指定講座 受講。
- **要什么材料**：受給資格者証、訓練 申込書、ハローワーク 受講指示。
- **办不办的后果**：公共職業訓練 受講中 は 給付制限 解除 + 受給期間 延長 + 訓練手当 受給可。教育訓練給付 は 雇用保険 加入期間 1-3 年 以上 で 受講料 20-70% 還付（資格 / 講座 で 異 なる）。

### 7. 再就職手当（早期就職 ボーナス）
- **在哪办**：ハローワーク。
- **要什么材料**：再就職手当支給申請書、新雇用主 の 採用証明書、給与明細（採用後 1 ヶ月内）。
- **办不办的后果**：給付残日数 が 所定給付日数 の 1/3 以上 残っている 段階 で 安定就職 → 残日数 × 60-70% × 基本手当日額 の 一時金。申請 しないと 受け取れない。

### 8. 入管「契約機関に関する届出」連動
- **在哪办**：[出入国在留管理庁 在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/)。
- **要什么材料**：在留カード コピー、届出書。
- **办不办的后果**：雇用保険 申請 と 入管届出 は 別 制度。両方 14 日内 / 早期 に 行 う。

## 容易遗漏的事

### 坑 1：給付制限 2 ヶ月 中 も「認定日 出頭」必要
給付制限 中 は 給付金 振込 なし、だから ハローワーク 行 か なく て 良い と 誤解 する 人 が 多 い。実際 は 認定日 出頭 + 求職活動 実績 報告 が 必要。欠席 → 制限 明け の 給付 開始 が 遅れる。

### 坑 2：受給期間「1 年」と 給付日数「90-360 日」を 混同
受給期間 = 離職翌日 から 給付 受け取れる 暦上 の 期間（原則 1 年）。所定給付日数 = 実際 に 給付金 が 振り込ま れる 日数。例：給付日数 150 日 でも 受給期間 1 年 経過 → 残日数 失効。給付制限 + 認定 ペース 遅 い と 失効 リスク。

### 坑 3：アルバイト / 内職 申告 漏れ → 不正受給 認定
給付期間 中 の アルバイト / 内職（4 時間 以上 / 日）は 失業認定申告書 で 全額 申告 必須。申告 しない → 不正受給 → 受給額 3 倍 返還（雇用保険法 10-4）+ 給付停止 + 在留 影响。「日雇 / 短期 だから 申告 不要」は 誤り。
