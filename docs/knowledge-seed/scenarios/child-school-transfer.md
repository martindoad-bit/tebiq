# 孩子日本学校转学手续清单

## 适用场景
学齢子女（小学校 6-12 歳 / 中学校 12-15 歳）が引越し / 私立受験合格 / 海外帰任 等で 日本校間 転校する家庭。高校（义务教育外）は学校 + 県教育委員会 別扱い、本篇 small touch のみ。幼稚園 / 保育園 転園は 自治体 子育て課 別篇参照。

## 時間線

### 転居 / 転校決定 直後
- 旧校 担任 / 教頭 へ 転校予定 通知
- 旧校 在学証明書 / 教科書給与証明書 / 学齢証明書 取得依頼
- 私立 / 国際校 转入 case = 編入試験 申込

### 転校 14 日前
- 旧住所 教育委員会 へ 転学届
- 新住所 教育委員会（同時に 転入届 と一括手続）
- 旧校 最終登校日 + 通知表 / 健康診断票 受領

### 転校 第 1 日（新校登校開始）
- 新校 へ 在学証明書 / 教科書給与証明書 持参
- 健康診断票 / 歯科検診票 移管
- 給食費 / PTA 費 等 引落口座 設定

### 転校後 1 ヶ月
- 学校保健 / アレルギー / 配慮事項 申告
- 旧校 卒業 PTA / 引継 行事 残務
- 通学路 / 緊急連絡網 確認

## 必办事項（按重要性排序）

### 1. 旧校 転校通知 + 書類取得
- **在哪办**：旧校 担任 / 教頭 / 学校事務室
- **要什么材料**：転校予定届（学校書式） / 転居先住所 / 転入予定校（決定済 case）
- **办不办的后果**：在学証明書 + 教科書給与証明書 が新校で必須。旧校で 学籍 整理 + 通知表 / 指導要録 / 健康診断票 抜粋 を準備。発行 1-2 週かかる case あり。

### 2. 旧住所 教育委員会 転学届
- **在哪办**：旧住所 市役所 / 区役所 教育委員会 学務課
- **要什么材料**：転出届（住民課同時） / 在学証明書（旧校発行） / 在留カード
- **办不办的后果**：旧自治体の学齢簿 から 抹消 + 旧校 退学処理。怠慢 → 旧校 出席簿 上 続いて記載 → 後日修正煩雑。

### 3. 新住所 教育委員会 転入届
- **在哪办**：新住所 市役所 / 区役所 教育委員会 学務課（住民課 転入届 と同日 一括手続推奨）
- **要什么材料**：転入届（住民課経由）/ 在学証明書（旧校）/ 教科書給与証明書 / 子女 在留カード / 健康保険証
- **办不办的后果**：教育委員会 が 入学指定校通知書 発行 → 新校 受入手続。指定校変更（区内別校希望 case）は別途申請 + 自治体審査 1-2 週。

### 4. 新校 編入手続 + 第 1 日登校
- **在哪办**：新校 事務室 / 担任
- **要什么材料**：入学指定校通知書 / 在学証明書 / 教科書給与証明書 / 健康診断票 / 学齢証明書 / 印鑑 / 上履き / 学用品
- **办不办的后果**：新校 = 旧校で交付された教科書 を引き継ぐか、新校 の使用教科書 と異なる場合 教科書給与証明書 で無償交付。学用品 / 制服 / 体操服 は新校指定。1-2 週間 で 揃え可能。

### 5. 私立 / 国際スクール 編入試験 + 入学手続
- **在哪办**：私立 / 国際校 入試事務室
- **要什么材料**：転入願 / 在学証明書 / 通知表写し / 健康診断票 / 推薦書（学校による） / 入学金 + 授業料前納
- **办不办的后果**：編入試験 = 国 / 算 / 英 / 面接 等。学校により欠員時のみ募集。1-2 ヶ月前申込締切。インター スクール は英語面接 + 親面接 + 高額入学金（¥1M-3M） + 授業料 ¥2M-4M / 年。

### 6. 高校 転入（義務教育外）
- **在哪办**：県 / 都 教育委員会 高校教育課 + 受入校 事務室
- **要什么材料**：転入学願書 / 在学証明書 / 単位修得証明書 / 成績証明書 / 健康診断票 / 写真
- **办不办的后果**：公立高校 転入は欠員校のみ + 転入試験（国 / 数 / 英 + 面接）。年 2 - 3 回 募集（学期始）。私立高校 は学校別。海外帰国生 編入は別枠。

### 7. 帰国生 / インター 出身者 → 公立校 編入
- **在哪办**：新住所 教育委員会
- **要什么材料**：在外学校 在籍証明書 + 日本語訳 / パスポート / 在留カード / 帰国前学習内容資料
- **办不办的后果**：帰国生 用の編入学級 / 日本語指導クラス を持つ自治体（東京 / 横浜 / 川崎 / 大阪 / 名古屋 等）あり。英語日常 + 日本語不十分の子女向け。教育委員会窓口で受入校 相談。

### 8. 健康保険 / 学校健康診断票 移管
- **在哪办**：新校 養護教諭 / 保健室
- **要什么材料**：健康診断票（旧校）/ 母子健康手帳 / 予防接種記録 / アレルギー診断書（あれば）
- **办不办的后果**：未移管 = 学校健診 / 給食 アレルギー対応 / 緊急連絡 で支障。歯科 / 内科 / 視力 等の継続記録が断絶。

### 9. 学校給食 / PTA 費 / 教材費 引落口座
- **在哪办**：新校 事務室
- **要什么材料**：銀行口座 / 印鑑 / 自動引落申込書
- **办不办的后果**：給食費 月 ¥4,000-5,000（小学校）/ ¥5,000-6,000（中学校）。PTA 費 年 ¥3,000-6,000。未引落 = 督促 + 担任から連絡。就学援助制度（経済困難 case）は教育委員会 別途申請。

### 10. 通学路 / 緊急連絡網 + 集団登校
- **在哪办**：新校 / 学童保育（あれば）
- **要什么材料**：緊急連絡先カード / 通学路 図 / 学童 申込書（共働 case）
- **办不办的后果**：小学校低学年は集団登校 班 編入 = 近隣保護者 とのコミュニケーション。学童保育 は 4 月入会申込が原則 + 途中入会は欠員待ち。

### 11. 在留資格 同伴者（家族滞在 子女）の整理
- **在哪办**：出入国在留管理庁
- **要什么材料**：在留資格変更申請（必要 case） / 在留期間更新 / 在留カード 住所変更
- **办不办的后果**：家族滞在 子女が 高校卒業 + 大学進学 = 留学 在留資格 への変更。中学卒業 + 就職 = 別の在留資格 への変更（高校進学なら家族滞在 継続 OK）。転校自体には在留資格影響なし。

## 容易遗漏的事

### 坑 1：「住民票 を移せば自動的に新校 へ」と誤解
住民票 = 市役所 住民課 / 学校編入 = 教育委員会 学務課 / 新校 受入 = 学校事務室。3 段階あり、住民票 移しただけで自動的に新校 通学はできない。1-2 日 内に教育委員会で 入学指定校通知書 取得が必要。

### 坑 2：教科書 を「旧校 で全部捨てる」
日本の小中学校 = 教科書無償給与制度（教科書無償措置法）。旧校 で受領した教科書 は学年末まで使う。新校 で同じ教科書を採択している場合 = そのまま継続使用。異なる場合 = 教科書給与証明書 で新教科書 無償交付。捨てると新校で再交付できない。

### 坑 3：私立 / インター 中途退学時の授業料 / 入学金 返還ルール 知らず
私立校 は 4 月入学前 退学 = 入学金 返還無し / 授業料 返還無し が一般的（消費者契約法 + 私立学校法 範囲内）。中途退学 = 月割 / 学期割 で計算 case あり、契約書要確認。

> # ⚠️ 需创始人/書士复核：帰国生 / 海外校から日本公立校 への 編入時の学年判定基準（年齢相当 vs 学習進度相当）の自治体ごとの差
