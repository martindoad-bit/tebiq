# 离婚后 30 日内必办清单

## 适用场景
協議 / 調停 / 裁判 离婚成立的在日外国人。本篇以 離婚届 受理日 = 第 0 日计时。協議離婚 中尚未签字提出 離婚届 的人不适用。子女親権 / 養育費 / 財産分与 是另一套议题，本篇只覆盖 30 日 + 6 ヶ月 内的法定届出与 在留資格 风险。

## 时间线

### 第 1 日（離婚届 受理当天）
- 離婚届 提出（市役所 / 区役所 戸籍課）
- 取得 離婚届受理証明書

### 第 14 日内
- 入管法 第 19-16 条 配偶者に関する届出（日本人配偶者等 / 永住者配偶者等 / 家族滞在 在留者）
- 在留カード 住所変更（搬家时）
- 戸籍謄本 离婚反映取得（1-2 周）

### 第 30 日内
- 健康保険 / 厚生年金 扶養 离脱届 → 国民健康保険 / 国民年金 加入
- 児童手当 受給者変更（子女親権 取得方）
- 銀行 / クレカ / 公共料金 名義 + 引落口座 整理
- 旧姓复帰 选择（離婚届 同时 或 3 ヶ月内 「離婚の際に称していた氏を称する届」）

### 6 ヶ月内（在留資格 风险窗口）
- 在留資格変更 検討（日本人配偶者等 / 永住者配偶者等 在留者必看）
- 入管法 第 22-4 条第 1 項第 7 号：配偶者活動を 6 ヶ月以上行わない → 在留資格 取消対象

## 必办事项（按重要性排序）

### 1. 入管 配偶者に関する届出（14 日内）
- **在哪办**：出入国在留管理庁（窓口 / 郵送 / オンライン）
- **要什么材料**：配偶者に関する届出書 / 在留カード
- **办不办的后果**：入管法 第 19-16 条规定 14 日 内届出义务。怠慢 / 虚伪届出 处 20 万円 以下罚金（同法 第 71-3 条）。下次 在留期間更新 / 永住申請 时素行善良评价直接受影响。

### 2. 在留資格変更 検討（日本人配偶者等 / 永住者配偶者等 在留者）
- **在哪办**：出入国在留管理庁
- **要什么材料**：取决于变更目的地（定住者 / 技人国 / 経営・管理 / 留学 等）
- **办不办的后果**：日本人配偶者等 在留者 离婚后 6 ヶ月内 必须做出方向决定。三个常见路径：
  1. **定住者 申請**：日本婚姻 3 年以上 + 实质生活基础 + 子女親権 等综合判断
  2. **就労系 在留資格 変更**：技人国 / 経営・管理 等，需对应学历 / 职务 / 雇用契約 / 出資
  3. **出国**：在留期限内出国 → 后续重新审查
  
  6 ヶ月 経過しても 配偶者活動なし且未变更，第 22-4 条第 1 項第 7 号 在留資格 取消事由 该当。

### 3. 離婚届 提出 + 戸籍 整理
- **在哪办**：双方任一方 本籍地 / 所在地 市役所 / 区役所 戸籍課
- **要什么材料**：離婚届書（証人 2 名 协議離婚 时）/ 戸籍謄本 / 調停調書 / 判決書（調停 / 裁判離婚 时）
- **办不办的后果**：戸籍 未反映 = 婚姻继续。在留資格 风险窗口不启动，但所有後续手续也无法进行。

### 4. 子女親権 / 監護権 决定 + 戸籍 入籍届
- **在哪办**：戸籍課 / 家庭裁判所
- **要什么材料**：協議書 / 調停調書 / 入籍届 / 子の氏の変更許可申立 等
- **办不办的后果**：子女親権者 未指定则 離婚届 不受理。子女戸籍 移籍 + 氏変更 是别的程序。子女在留資格（家族滞在 时）随之需要 在留資格 取得 / 変更。

### 5. 子女在留資格 整理
- **在哪办**：出入国在留管理庁
- **要什么材料**：在留資格変更 / 取得申請書 / 戸籍謄本 / 親権者 在留カード / 養育費 受給証明 等
- **办不办的后果**：子女が日本国籍取得（日本人親 + 認知 / 嫡出）则在留資格 不要。子女が外国籍 + 親権者が外国人方 → 家族滞在 → 親 在留資格 結束時 子女在留 也要重新整理。子女が外国籍 + 親権者が日本人方 → 定住者 申請。

### 6. 健康保険 / 厚生年金 扶養 离脱
- **在哪办**：配偶就業会社（事前通知）/ 国保 + 国民年金 加入是 市役所 / 区役所
- **要什么材料**：被扶養者異動届 / 健康保険証（旧）返还 / 戸籍謄本 / 在留カード
- **办不办的后果**：扶養 离脱遅延则 健康保険証 仍能使用，但是月 / 通院 後事後請求被扶養者外保険料。国民健康保険 加入は離脱日翌日起算 14 日以内（国保法 第 9 条）。

### 7. 国民年金 第 1 号 切替
- **在哪办**：住民登録地 市役所 / 区役所 国民年金課
- **要什么材料**：年金手帳 / 健康保険資格喪失証明書 / 在留カード
- **办不办的后果**：第 3 号 → 第 1 号 切替を忘れると 未納期間 累积。事後 2 年以内 遡及 訂正可能だが手続煩雑。

### 8. 児童手当 受給者変更
- **在哪办**：住民登録地 市役所 / 区役所 こども家庭課
- **要什么材料**：児童手当受給事由消滅届（旧受給者）/ 児童手当認定請求書（新受給者）/ 戸籍謄本 / 親権者 マイナンバー
- **办不办的后果**：親権者 と 受給者 が一致しないと支給停止。新規認定請求は離婚日翌日起算 15 日以内（こども家庭庁 規則）。

### 9. 旧姓 復氏 / 婚氏 続称 選択
- **在哪办**：戸籍課
- **要什么材料**：「離婚の際に称していた氏を称する届」（婚氏 続称 时）/ 期限：離婚日 から 3 ヶ月以内
- **办不办的后果**：3 ヶ月 経過後は 家庭裁判所 許可 が必要。子女戸籍 / 在留カード / 銀行 / 不動産 名義 が変動するので慎重に決定。

### 10. 不動産 賃貸 名義 + 退去 / 名義変更
- **在哪办**：不動産会社 / 大家
- **要什么材料**：賃貸借契約書 / 戸籍謄本 / 連帯保証人 再依頼書
- **办不办的后果**：契約者 が 出ていく方 で 残る方 が名義変更しないと、退去者 が引落・滞納責任 を引き続き負担。連帯保証人 が元配偶者 だった場合 も再設定が必要。

### 11. 銀行 / クレカ / 公共料金 整理
- **在哪办**：各社 窓口 / オンライン
- **要什么材料**：戸籍謄本 / 在留カード / 新印鑑（旧姓復氏 时）
- **办不办的后果**：自動引落 が元配偶者口座 のままだと延滞 → クレヒス 傷つき。3 ヶ月以内に整理。

## 容易遗漏的事

### 坑 1：14 日 配偶者届出 を「離婚届 で済んだ」と誤解
市役所 の 離婚届 と 入管 の 配偶者届出 は別の手続。市役所 が 入管 に通知することはない。離婚届 受理後、自分で 入管 に届け出る義務がある。

### 坑 2：6 ヶ月 ルールの過信 / 軽視
日本人配偶者等 在留者で「離婚直後に新しい交際 / 再婚予定があるから 6 ヶ月 過ぎても大丈夫」と考える人がいるが、入管 は 配偶者活動の実質 を見る。婚姻関係破綻 / 別居期間 が長い case で過去に在留資格 取消 された判例がある。再婚予定があっても 6 ヶ月 内に 出入国在留管理庁 へ相談 + 変更申請 が安全。

### 坑 3：子女親権 を「とりあえず母親」で決めると 在留資格 で詰む
子女が外国籍 + 母親（離婚した側）が 在留資格変更 困難 → 子女 家族滞在 在留 も同時に消失。親権 決定前に 子女在留資格 の見通し を立てるべき。

> # ⚠️ 需创始人/書士复核：日本人配偶者等 → 定住者 申請的「実務上の婚姻 3 年以上 + 同居 1 年以上」という不文律の正確性
