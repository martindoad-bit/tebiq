# 房屋租赁合同到期更新清单

## 适用场景
对象：日本 で 賃貸住宅（アパート / マンション / 戸建）に 居住、契約期間 通常 2 年 満了 を 迎 える 在日 居住者。
触发事件：管理会社 / 大家 / 保証会社 から「契約更新 案内」「更新料 請求書」到着、契約満了 6 ヶ月以内。
不适用：定期借家契約（再契約 のみ、更新 不可、別 シナリオ）。社宅 / 社員寮（会社 経由 手続）。

## 时间线

### 6 ヶ月前
- 必办：賃貸借契約書 確認 → 更新料 / 更新事務手数料 / 火災保険 / 保証会社 更新料 の 金額 把握。
- 必办：継続 vs 引っ越し 判断（家賃相場 比較、転職 / 通勤 変化、家族構成 変化）。

### 3 ヶ月前
- 必办：管理会社 / 大家 から 更新案内 受領 → 更新通知書 / 新賃貸借契約書 内容 確認（家賃改定 / 条件変更 注意）。
- 引っ越し の 場合：解約予告（通常 1-2 ヶ月前、契約書 規定 確認）+ 新居 探し 開始。

### 1 ヶ月前
- 継続 の 場合：更新料 / 火災保険 / 保証会社 更新料 振込 + 更新契約書 署名 / 押印 / 返送。
- 引っ越し の 場合：退去届 提出、引っ越し 業者 手配、ライフライン 解約 / 新規契約。

### 退去日 前後
- 引っ越し の 場合：退去立会 → 原状回復 確認 → 敷金 精算（清算書 受領）→ 鍵返却。
- 引っ越し の 場合：旧住所 市役所 で 転出届 → 新住所 市役所 で 転入届（14 日内）+ 入管 在留カード 住居地変更届（14 日内）。

## 必办事项（按重要性排序）

### 1. 賃貸借契約書 + 更新条件 確認
- **在哪办**：自分で 契約書 確認 / 不明点 は 管理会社 へ 照会。
- **要什么材料**：賃貸借契約書 原本、更新案内 通知書。
- **办不办的后果**：更新料（家賃 1 ヶ月分 多）、更新事務手数料（家賃 0.5 ヶ月分 多）、火災保険（2 年 で 1-2 万円）、保証会社 更新料（家賃 10-30% / 年 1 万円 等）。一括 で 家賃 1.5-2.5 ヶ月分 必要。

### 2. 更新料 / 更新事務手数料 振込
- **在哪办**：管理会社 / 大家 指定 口座 へ 振込。
- **要什么材料**：更新料 請求書、振込 控え。
- **办不办的后果**：期日 まで に 振込 + 更新契約書 返送 → 借地借家法 26 条 で 法定更新（自動更新）も 可能 だが、契約書 上 の 更新料 支払義務 は 残 る ケース が 多 い（最高裁 2011 年 判例 で 一定金額 までは 有効）。未払 → 信頼関係破壊 → 解約 / 強制退去 リスク。

### 3. 火災保険 更新
- **在哪办**：管理会社 指定 保険会社 / 自分 で 選んだ 保険会社。
- **要什么材料**：保険証券、火災保険 申込書。
- **办不办的后果**：契約書 上 加入義務 多。未加入 → 契約違反 → 解約 リスク + 火災 / 水漏 / 盗難 で 損害 自己負担。管理会社 指定 保険 は 多 く の 場合 自分 で 安 い 保険 に 切替 OK（契約書 確認）。

### 4. 保証会社 更新料
- **在哪办**：保証会社 へ 振込（管理会社 経由 多）。
- **要什么材料**：保証委託契約 更新通知。
- **办不办的后果**：保証会社 更新料 = 年 1 万円 / 家賃 10-30% / 月 1-2% 等 多様。未払 → 保証 失効 → 大家 から 連帯保証人 / 別 保証会社 要求。連帯保証人 立てられ ない 在日 外国人 多 く、保証会社 必須。

### 5. 引っ越し 場合：解約予告
- **在哪办**：管理会社 / 大家 へ 解約届 / 退去通知書 提出。
- **要什么材料**：解約届（管理会社 様式 / 自由様式）、契約書。
- **办不办的后果**：契約書 規定（通常 1-2 ヶ月前）守 ら ない → 解約予告期間分 家賃 請求。例：契約書 が「2 ヶ月前」で 1 ヶ月前 に 退去届 → 1 ヶ月分 余分 に 家賃 支払。

### 6. 引っ越し 場合：原状回復 + 敷金 精算
- **在哪办**：退去立会（管理会社 / 大家 + 入居者）→ 後日 敷金 精算書 受領。
- **要什么材料**：入居時 写真、契約書、敷金 預り証、特約 確認。
- **办不办的后果**：[国土交通省「原状回復をめぐるトラブルとガイドライン」](https://www.mlit.go.jp/jutakukentiku/house/torikumi/honbun2.pdf)：通常損耗 / 経年劣化 は 大家 負担、故意過失 / 善管注意義務違反 は 入居者 負担。過大 請求 → 消費生活センター / 少額訴訟 で 争え る。

### 7. 引っ越し 場合：転出 / 転入届 + 在留カード 住居地変更
- **在哪办**：旧住所 市役所（転出届 14 日前 から 14 日内）→ 新住所 市役所（転入届 14 日内）→ 同時 に 在留カード 裏面 住居地 書換。
- **要什么材料**：転出証明書、在留カード、マイナンバーカード（あれば）、印鑑。
- **办不办的后果**：14 日 過ぎ ると 住民基本台帳法 違反 → 5 万円以下 過料。在留カード 住居地変更 未届出 → 入管法 19-9 条 違反 → 在留資格取消 対象 + 罰金 20 万円以下。

### 8. ライフライン 解約 / 新規契約
- **在哪办**：電力会社 / ガス会社 / 水道局 / インターネット プロバイダ。
- **要什么材料**：契約者番号、引っ越し 日。
- **办不办的后果**：解約 し 忘れ → 旧住所 基本料金 継続 請求。新住所 ガス 開栓 立会 必要（東京ガス 等）→ 入居日 前 に 予約 必須。

## 容易遗漏的事

### 坑 1：更新料 を「絶対 払 わ ない と いけ ない」と 思 い 込 む
更新料 は 関東 / 関西 / 京都 等 で 慣行 が 異 な り、契約書 に 明記 さ れ て い る 場合 のみ 法的 に 有効（最高裁 2011/7/15 判決：高額 すぎ なけ れば 有効）。家賃 1 ヶ月分 程度 が 相場。家賃 数 ヶ月分 等 不当 に 高額 な 場合 は 消費契約法 10 条 で 無効 を 争 え る ケース あり。

### 坑 2：保証会社 滞納 1 回 で 一気 に「強制解約」リスク
保証会社（特 に 全国賃貸保証業協会 系）は 家賃 滞納 に 厳格。1 回 滞納 → 立替払 → 督促 → 信用情報 登録 → 次回 賃貸契約 審査 通 り にく く なる。家賃 引落 残高 不足 / 自動引落 設定 ミス で 滞納 する 在日 外国人 多 い。引落 口座 残高 を 給料日 前 に 確認 する 習慣。

### 坑 3：原状回復 で 大家 言 い 値 で 敷金 全額 没収
[国土交通省 原状回復ガイドライン](https://www.mlit.go.jp/totikensangyo/const/sosei_const_tk2_000113.html)：壁紙 黄ばみ / 床 日焼け / 家具設置 跡 等 は 通常損耗 で 大家 負担。タバコ ヤニ / ペット 傷 / 故意 破損 のみ 入居者 負担。入居時 / 退去時 部屋 写真 多 数 撮影 → 後日 トラブル 時 証拠。請求書 内訳 に「ハウスクリーニング」「鍵交換」が 全額 入居者 負担 で 計上 さ れ て い る ケース 多 い が、特約 が なけ れば 大家 負担 が 原則。
