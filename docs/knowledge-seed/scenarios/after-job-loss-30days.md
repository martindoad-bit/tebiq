# 离职 / 被裁后 30 日内必办清单

## 适用场景
对象：在日就労 / 配偶者 / 永住者 在留資格 持有者，刚收到 離職票 / 解雇通知 / 雇用契約終了 通知者。
触发事件：最終出社日 已确定，或 離職票-1 / 離職票-2 已到手。
不适用：派遣切替、出向、社内異動（雇用主未变更）。这些走「契約機関に関する届出」轻量流程。

## 时间线

### 第 1 日（离职当天 / 翌日）
- 必办：向 旧雇用主 索取 離職票-1（雇用保険被保険者離職票-1）+ 離職票-2、源泉徴収票、健康保険資格喪失証明書、年金手帳（保管中の場合）。
- 必办：领取 健康保険証 → 旧雇用主 返却（退職翌日 失効）。

### 第 7 日内
- 必办：所在 市役所 / 区役所 国民健康保険 加入届（健保 任意継続 选 旧雇用主 経由 20 日内 申請も可）。
- 必办：所在 市役所 国民年金 第 1 号被保険者 種別変更 届出（厚生年金 → 国民年金）。
- 必办：ハローワーク 求職申込 + 雇用保険 受給資格決定 申請（離職票 到着 後 早めに）。

### 第 14 日内（重要：入管 / 雇用保険 多在 14 日内）
- 必办：出入国在留管理庁 へ「契約機関に関する届出」14 日内 提出（入管法 19-16）。
- 必办：国民年金 / 国保 加入届 厳格には 退職日 翌日から 14 日内（市町村窓口）。
- 不届出 → 罰金 20 万円以下（入管法 71-3）+ 永住申請 / 在留期間更新 不利。

### 第 30 日内
- 必办：雇用保険 待期 7 日 経過 → 受給資格者証 交付（会社都合 即給付開始 / 自己都合 給付制限期間 開始）。
- 必办：住民税 普通徴収 切替 通知 受領 / 一括徴収 を 退職時 給与 から 控除済 か 確認。

### 之后（認定サイクル / 受給期間 等）
- 4 週（28 日）に 1 回 ハローワーク 認定日 出頭 → 失業認定申告書 提出 → 給付金 振込。
- 受給期間 原則 離職翌日から 1 年。期間内に 給付日数（90-360 日）受給 完了。

## 必办事项（按重要性排序）

### 1. 入管「契約機関に関する届出」14 日内
- **在哪办**：[出入国在留管理庁 在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 或 最寄 入管 窓口、郵送。
- **要什么材料**：在留カード コピー、届出書（離脱 / 移籍 / 消滅 选离脱）、署名。
- **办不办的后果**：未届 → 20 万円以下 罰金（入管法 71-3）。永住申請 時「公的義務履行」項目 マイナス 評価。

### 2. 雇用保険 受給資格 申請（ハローワーク）
- **在哪办**：住居地 管轄 [ハローワーク](https://www.hellowork.mhlw.go.jp/)。
- **要什么材料**：離職票-1 / 離職票-2、在留カード、マイナンバー 確認書類、写真 2 枚（縦 3.0×横 2.4cm）、本人名義 預金通帳、印鑑。
- **办不办的后果**：給付 受け取れない。受給期間 1 年 経過後 申請不可（離職翌日 起算）。

### 3. 国民健康保険 加入 OR 健康保険 任意継続
- **在哪办**：国保 → 市役所 / 区役所 保険年金課。任意継続 → 旧 健保組合 / 協会けんぽ。
- **要什么材料**：健康保険資格喪失証明書、在留カード、マイナンバー 通知カード、印鑑。任意継続 は 20 日内 申請。
- **办不办的后果**：無保険期間 発生 → 医療費 全額自己負担（10 割）。後日 加入 でも 遡及 保険料 徴収。

### 4. 国民年金 種別変更 届出
- **在哪办**：市役所 / 区役所 国民年金課、または [日本年金機構](https://www.nenkin.go.jp/)。
- **要什么材料**：年金手帳 / 基礎年金番号通知書、健康保険資格喪失証明書、在留カード。
- **办不办的后果**：未納期間 → 老齢年金 / 障害年金 受給資格 影响。配偶者 第 3 号 該当 時 別 手続。

### 5. 住民税 支払 確認
- **在哪办**：市役所 / 区役所 税務課。
- **要什么材料**：退職時 受領 給与明細 + 住民税 普通徴収 通知書。
- **办不办的后果**：1-5 月 退職 → 残額 一括 給与控除（多）。6-12 月 退職 → 普通徴収 切替、年 4 期 自分で 納付。滞納 → 督促 / 在留期間更新 申請時「公的義務」項目 不利。

### 6. 失業 中 在留期間更新 / 変更 検討
- **在哪办**：管轄 入管 / [出入国在留管理庁](https://www.moj.go.jp/isa/)。
- **要什么材料**：在留カード、パスポート、雇用保険 受給資格者証（求職活動中 証明）、預金残高証明、家族 扶養関係 書類。
- **办不办的后果**：技人国 失業 3 ヶ月 超 → 在留期間更新 不利。6 ヶ月 超 求職実績 ない → 在留資格取消（入管法 22-4）対象 になり 得る。

### 7. 源泉徴収票 / 確定申告 準備
- **在哪办**：旧雇用主 から 退職後 1 ヶ月内 交付。翌年 2/16-3/15 [国税庁 e-Tax](https://www.nta.go.jp/) で 確定申告。
- **要什么材料**：源泉徴収票、雇用保険 受給金額 明細（非課税）、医療費 / 社会保険料 控除証明。
- **办不办的后果**：年内 再就職 ない → 年末調整 されない → 確定申告 で 還付 受ける。未申告 → 還付金 受け取れない。

## 容易遗漏的事

### 坑 1：入管 14 日届出 を「雇用保険 受給開始 してから」と 誤解
入管 届出 は「離職事実」発生 14 日内。給付 申請 / 受給 とは 別 制度。給付 待期中 でも 14 日 過 ぎたら 届出 違反。離職票 到着 を 待たず、最終出社日 翌日 から カウント 開始。

### 坑 2：健保 任意継続 申請期限「退職翌日 から 20 日内」を 過ぎる
20 日 過ぎると 任意継続 不可、国保 一択。任意継続 / 国保 どちらが 安いかは 標準報酬月額 と 前年所得 で 個別判定。国保 保険料 は 前年所得 ベース、退職直後 は 高くなり がち。

### 坑 3：自己都合 退職 の 給付制限「2 ヶ月」を「3 ヶ月」と 誤認
2025/4/1 改正で 給付制限 は 自己都合 でも 原則 2 ヶ月（5 年内 2 回まで。3 回目 以降 は 3 ヶ月）。会社都合（特定受給資格者 / 特定理由離職者）は 給付制限 なし、待期 7 日 後 即給付。離職票-2 の 離職理由欄 に 異議 ある 場合 ハローワーク で 異議申立 可能。
