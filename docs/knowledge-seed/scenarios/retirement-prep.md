# 退休前后清单（60 歳 前後）

## 适用场景
对象：日本 で 厚生年金 加入歴 を 持ち、60-65 歳 で 退職 / 再雇用 を 検討 する 在日就労者。永住者 / 配偶者 / 就労ビザ 全 種 対象。
触发事件：60 歳 / 65 歳 誕生日、退職予定日 確定、年金事務所 から「ねんきん定期便」「年金請求書」到着。
不适用：日本 年金 加入歴 6 ヶ月 未満（脱退一時金 検討、別 シナリオ）。

## 时间线

### 6 ヶ月前
- 必办：[ねんきんネット](https://www.nenkin.go.jp/n_net/) で 加入歴 + 見込額 確認。漏れ あれ ば 年金事務所 へ 照会。
- 必办：勤務先 退職金規程 確認。退職届 提出 タイミング 検討。

### 3 ヶ月前
- 必办：年金請求書（事前送付）受領 → 必要書類 準備（戸籍謄本、住民票、銀行通帳 コピー、雇用保険 被保険者証、配偶者 年金手帳 等）。
- 必办：健康保険 退職後 選択（任意継続 / 国保 / 家族 扶養）比較。

### 1 ヶ月前
- 必办：退職届 提出（就業規則 規定 確認、原則 1 ヶ月前）。
- 必办：再雇用 / 継続雇用 制度 の 利用 確認（高年齢者雇用安定法 で 65 歳 まで 雇用機会 確保 義務）。

### 退職後 14 日内
- 必办：健康保険 任意継続（20 日内）or 国保 加入（14 日内）or 家族 扶養 加入（5 日内）。
- 必办：国民年金 切替（60 歳 未満 の 場合）/ 受給開始 手続（60-65 歳 で 受給開始 選択 する 場合）。
- 必办：入管「契約機関に関する届出」14 日内。

### 退職後（受給開始 後）
- 老齢年金 受給：原則 65 歳 開始、繰上 60 歳 / 繰下 75 歳 まで 選択可。在職老齢年金 制度（働きながら 受給）に 注意。
- 介護保険：65 歳 で 第 1 号被保険者 自動切替 → 介護保険料 別途 徴収。

## 必办事项（按重要性排序）

### 1. 老齢厚生年金 / 老齢基礎年金 受給開始 申請
- **在哪办**：[年金事務所](https://www.nenkin.go.jp/)（年金請求書 提出）。
- **要什么材料**：年金請求書、戸籍謄本（受給権発生後 6 ヶ月内）、住民票、本人名義 預金通帳、年金手帳 / 基礎年金番号通知書、配偶者 年金手帳（加給年金 該当 時）、雇用保険 被保険者証。
- **办不办的后果**：申請 しないと 受給 開始 されない。受給権 発生 から 5 年 経過 → 時効 で 過去分 受け取れない（5 年 超 過去分 の みつ 部分 失効）。

### 2. 繰上 / 繰下 受給 選択
- **在哪办**：年金事務所（請求時 選択）。
- **要什么材料**：年金請求書（繰上 / 繰下 希望 欄）。
- **办不办的后果**：60 歳 繰上 → 月 0.4% 減額（最大 24%、生涯）/ 75 歳 繰下 → 月 0.7% 増額（最大 84%、生涯）。一度 選択 で 取消不可。健康 / 在職予定 / 配偶者年金 と 総合判断。

### 3. 健康保険 退職後 選択（任意継続 vs 国保 vs 家族扶養）
- **在哪办**：任意継続 → 旧 健保組合 / 協会けんぽ（20 日内）/ 国保 → 市役所 / 区役所（14 日内）/ 家族扶養 → 配偶者 / 子 の 健保組合（5 日内）。
- **要什么材料**：健康保険資格喪失証明書、在留カード、マイナンバー、印鑑、（家族扶養 の 場合）所得証明（年収 130 万円 未満、60 歳以上 は 180 万円 未満）。
- **办不办的后果**：任意継続 = 退職時 標準報酬 の 全額自己負担、最長 2 年。国保 = 前年所得 ベース、初年度 高 め。家族扶養 = 保険料 ゼロ、所得制限 あり。無保険 → 医療費 10 割。

### 4. 国民年金 種別変更（60 歳 未満 退職 の 場合）
- **在哪办**：市役所 / 区役所 国民年金課。
- **要什么材料**：年金手帳、健康保険資格喪失証明書、在留カード。
- **办不办的后果**：60 歳 未満 退職 → 厚生年金 → 国民年金 第 1 号 切替。未納 → 老齢基礎年金 減額。任意加入（60-65 歳）で 受給額 増 加 可能。

### 5. 介護保険 第 1 号被保険者 切替（65 歳）
- **在哪办**：市役所 / 区役所（自動切替、通知 来る）。
- **要什么材料**：介護保険被保険者証 受領 確認。
- **办不办的后果**：65 歳 誕生月 から 自動 で 第 1 号 切替。保険料 は 年金 から 特別徴収（年金 月額 1.5 万円 以上）または 普通徴収。要介護認定 申請 で サービス 利用可。

### 6. 在留期間更新 影响（60 歳 超 退職）
- **在哪办**：管轄 入管 / [出入国在留管理庁](https://www.moj.go.jp/isa/)。
- **要什么材料**：在留カード、パスポート、年金受給証明、預金残高証明、扶養関係 書類。
- **办不办的后果**：永住者 → 影响 なし。技人国 / 経営・管理 等 就労ビザ → 退職 + 仕事 なし → 在留期間更新 不利、6 ヶ月 超 求職実績 なし → 在留資格取消 対象（入管法 22-4）。配偶者 / 定住者 → 配偶者 / 親族 関係 維持 されれば 影响 弱。

### 7. 在職老齢年金（働きながら 受給）
- **在哪办**：年金事務所（自動 計算）。
- **要什么材料**：給与明細、年金支払通知書。
- **办不办的后果**：2024 年 制度：65 歳 以上 + 老齢厚生年金 月額 + 給与（標準報酬月額）合計 が 月 50 万円 超 → 超過分 の 1/2 支給停止。給与 設定 で 受給額 変動。

### 8. 入管「契約機関に関する届出」14 日内
- **在哪办**：[出入国在留管理庁](https://www.ras-immi.moj.go.jp/)。
- **要什么材料**：在留カード コピー、届出書（離脱 / 消滅）。
- **办不办的后果**：未届 → 罰金 20 万円以下（入管法 71-3）。永住者 でも 届出義務 あり。

## 容易遗漏的事

### 坑 1：「年金 は 65 歳 から 自動 で 振り込まれる」と 誤解
日本 の 年金 は「請求主義」。65 歳 誕生日 が 来 て も、年金請求書 を 提出 しない 限り 振込 されない。日本年金機構 から 事前送付 される 緑色 の 封筒 を 受け取 っ たら 必ず 提出。請求権 5 年 で 時効 → 過去分 失効。

### 坑 2：脱退一時金 と 老齢年金 の 二者択一 を 知らない
日本 国籍 ない 人 で 厚生年金 / 国民年金 加入 6 ヶ月 以上 → 出国後 2 年内 に 脱退一時金 申請可（最大 60 ヶ月分）。受給 すると その 期間分 の 老齢年金 受給権 消滅。日本 で 老後 を 過 ごす 予定 なら 脱退一時金 申請 しない 方 が 得 な ケース 多 い。社会保障協定 国（中国 / 韓国 / 米国 等）の 場合 通算 規定 もあり、要確認。

### 坑 3：再雇用 で 給与 大幅減 → 高年齢雇用継続給付 を 申請 し 忘れ
60-65 歳 で 再雇用 / 継続雇用 + 給与 が 60 歳 時点 の 75% 未満 → 高年齢雇用継続基本給付金（雇用保険）申請可。最大 給与 の 15%。会社 の 人事 から 自動的 に 案内 されない こと も ある。ハローワーク で 申請。
