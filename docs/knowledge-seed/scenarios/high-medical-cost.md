# 病了 / 受伤了 高額療養費 申请清单

## 适用场景
对象：健康保険（協会けんぽ / 健保組合 / 国民健康保険）加入者で 月 ¥80,000 以上 医療費発生見込者。触发：入院決定 / 手術 / 高額外来治療。不适用：労災適用（労災保険）/ 自由診療 / 美容整形 / 海外受診。

## 时间线

### 入院前（理想 1 週間以上前）
- 必办：加入 健康保険組合 へ「限度額適用認定証」申請（マイナ保険証 利用時 不要）。
- 必办：医院 入院案内 + 概算費用 確認。

### 入院中
- 必办：限度額認定証 / マイナ保険証 を 病院 受付提出。
- 必办：医療費領収書 全保留（差額ベッド代 / 食事代 / 病衣代 等）。

### 退院当日
- 必办：医療費明細書 + 領収書 受領 + 内容確認（保険適用部分 vs 自費部分 区別）。

### 退院後 1 月
- 必办：認定証 利用しなかった場合 = 高額療養費 還付請求（健康保険組合）。
- 必办：医療費明細 を 月単位 / 病院単位 で 整理。

### 退院後 2-3 年
- 必办：医療費控除（確定申告）= 翌年 2-3 月 / 5 年遡及可。
- 必办：多数該当（直近 12 月内 3 回以上）= 4 回目以降 限度額減額確認。

## 必办事项

### 1. 限度額適用認定証（事前申請 推奨）
- **在哪办**：協会けんぽ（年金事務所 / 郵送）/ 健保組合（会社人事）/ 国民健康保険（市役所 国保課）。
- **要什么材料**：認定申請書 + 健康保険証 + マイナンバー。
- **办不办的后果**：未申請 = 病院 窓口 一旦 全額（3 割）支払 → 後日 還付請求。月末跨月 = 月単位計算で 二度限度額 発生リスク。マイナ保険証 利用 = 認定証 不要。

### 2. 高額療養費制度 限度額（70 歳未満 一般所得 / 標準報酬月額 ¥280,000-500,000）
- **在哪办**：制度適用 自動。
- **要什么材料**：医療費 ¥267,000 超過分 1% 上限。
- **办不办的后果**：限度額 = ¥80,100 + (医療費 - ¥267,000) × 1%。
  - 例：医療費 100 万円 → 自己負担 ¥87,430（30 万円 - 21 万円超過分還付）
  - 例：医療費 50 万円 → 自己負担 ¥82,430
  - 区分 ア（標報 ¥830,000+）= ¥252,600 + (医療費 - ¥842,000) × 1%
  - 区分 オ（住民税非課税）= ¥35,400

### 3. 多数該当（直近 12 月内 3 回以上）
- **在哪办**：健康保険組合 / 市役所 国保課。
- **要什么材料**：過去 12 月分 高額療養費 受給記録。
- **办不办的后果**：4 回目以降 限度額 = 区分ウ（一般）¥44,400 / 区分エ ¥44,400。慢性疾患 / 透析 患者 大幅減額。

### 4. 世帯合算（同 健康保険 加入家族 月 ¥21,000 以上 医療費）
- **在哪办**：健康保険組合 / 市役所 国保課。
- **要什么材料**：家族 全員 医療費明細。
- **办不办的后果**：同月内 同世帯 ¥21,000+ 自己負担分 合算 → 限度額 適用。夫婦 別の健保 = 合算不可。

### 5. 高額療養費 還付請求（事後）
- **在哪办**：健康保険組合 / 市役所 国保課。
- **要什么材料**：請求書 + 領収書 + 健康保険証 + 通帳。
- **办不办的后果**：時効 2 年（診療月翌月初日 起算）= 2 年経過後 = 還付不可。協会けんぽ は 受診 3 ヶ月後 自動的に「高額療養費支給申請書」郵送。

### 6. 高額医療費貸付制度
- **在哪办**：健康保険組合 / 市役所。
- **要什么材料**：貸付申込書 + 医療費明細書（病院記入）。
- **办不办的后果**：高額療養費 還付 まで 数ヶ月かかる場合 = 限度額分の 8 割相当 無利子貸付。退院費用 一括支払困難時 利用。

### 7. 医療費控除（確定申告）
- **在哪办**：税務署 / e-Tax。
- **要什么材料**：医療費明細書 + 源泉徴収票 + 領収書（5 年保存）。
- **办不办的后果**：年間自己負担 ¥100,000 超 / 所得 200 万円未満は 所得 5% 超 = 超過分が 所得控除。10 万円 控除 → 所得税率 20% 場合 = ¥20,000 還付。

### 8. 差額ベッド代 / 食事代 / 病衣代（保険外）
- **在哪办**：—（高額療養費 対象外）。
- **要什么材料**：—。
- **办不办的后果**：差額ベッド ¥6,000-30,000/日（個室）= 自己選択 場合 自己負担。「治療上必要」/「病院都合」の場合 請求拒否可（厚労省 通達）。食事代 ¥460/食 × 3食 × 入院日数。

## 容易遗漏的事

### 坑 1：月跨ぎ 入院 二重限度額
入院 7/25-8/5 = 7 月分（25-31 日）+ 8 月分（1-5 日）= それぞれの月で限度額計算。1 ヶ月内に収まるよう 入院日 調整可能なら、月初め入院が有利。

### 坑 2：国民健康保険 + 在留資格
国民健康保険 加入者でも 高額療養費 制度 全額適用。在留資格 / 国籍 不問。ただし 帰国予定者は 退院後 還付前に出国すると 受取困難（日本国内口座必須）。

### 坑 3：マイナ保険証 + 限度額認定証
2024/12 以降 マイナ保険証 利用 = 限度額認定証 申請不要（自動連携）。ただし 健保組合 によっては 連携未対応の場合あり、入院前に病院 + 健保 確認推奨。
