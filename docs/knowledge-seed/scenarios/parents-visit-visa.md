# 父母来日探亲签证前后准备

## 适用场景
- 中国の両親（または祖父母 / 兄弟姉妹）を 日本へ 短期滞在（90 日以内）で招聘する人。
- 招聘者（在日本 在住者）= 中長期 在留資格 保持（技人国 / 経営・管理 / 永住 / 日本人配偶者等 等）。
- 不适用：観光単独申請（家族訪問 ではなく 観光査証 で申請、書類が異なる）/ 90 日超滞在希望（→ 家族滞在 や 特定活動 等別カテゴリ）。

## 时间线

### 来日 60 日前
- 必办：招聘者 = 自分の在日資料準備（在職証明書、課税証明書、住民票 等）。
- 必办：両親 = 中国側 申請窓口（在中国日本国大使館 or 領事館 or 査証申請センター）の場所と必要書類確認。
- 可办：航空券・宿泊先 仮押さえ（**未確定 でも 滞在予定表 で代替可、未購入が一般的**）。

### 来日 30 日前
- 必办：**招聘理由書 / 滞在予定表 / 身元保証書** を招聘者が作成・署名。
- 必办：日本側 書類 一式 を 国際郵便（EMS 等）で両親へ郵送。
- 必办：両親 = 中国側 立証書類 取得（戸口簿 / 出生公証書 / 親族関係公証書 + 翻訳公証）。

### 来日 14 日前
- 必办：両親 = 査証申請（在外公館 or 査証申請センター 経由）。**現在 中国 = 北京・上海・広州・瀋陽・重慶・青島 + 香港 等 + 査証申請センター（JVAC）経由 が主流**。
- 必办：申請後 通常 5 営業日〜2 週間 で 査証発給。受領後 護照に貼付確認。
- 可办：日本国内の旅行 / 受診 計画詳細化、両親の海外旅行保険 加入確認。

### 来日当日
- 必办：成田 / 羽田 / 関西 等の入国審査で **「短期滞在」上陸許可スタンプ + 在留期間 90 日（or 30 日 / 15 日）付与確認**。
- 可办：空港 ピックアップ + 自宅 移動。
- 可办：両親が 携帯 / 銀行 等 利用予定なら 短期 SIM 等準備。

### 滞在中
- 可办：観光・通院・親族交流。**短期滞在 は 報酬を受ける活動 不可（資格外活動）**。
- 可办：滞在期間延長 = 原則 不可（特別事情 = 病気・事故 等 のみ 1 回 90 日まで延長可）。

## 必办事項（按重要性排序）

### 1. 招聘理由書（招聘者作成）
- **在哪办**：招聘者本人 作成。様式 = [外務省指定様式](https://www.mofa.go.jp/mofaj/toko/visa/tanki/short_stay.html)。
- **要什么材料**：招聘者氏名・住所・連絡先、招聘目的（親族訪問 / 知人訪問 等）、招聘経緯、訪問者との関係、過去の訪問歴 等を記載。
- **办不办的后果**：招聘理由書 不備 = 査証審査で **招聘の真実性に疑義** → 査証拒否。書式 + 内容の整合性が必要。

### 2. 滞在予定表（招聘者作成）
- **在哪办**：招聘者作成。**様式自由 だが [外務省様式](https://www.mofa.go.jp/mofaj/toko/visa/tanki/short_stay.html) 推奨**。
- **要什么材料**：日付ごとの滞在場所・活動・宿泊地（招聘者宅 等）。航空券情報、出入国予定日。
- **办不办的后果**：滞在予定表 不備 / 内容矛盾（航空券日付と滞在予定日付不一致 等）= **査証拒否要因**。**未確定でも 仮計画 + 注記** で提出可。

### 3. 身元保証書（招聘者作成 — 必要に応じ）
- **在哪办**：招聘者作成。様式 = 外務省指定。
- **要什么材料**：招聘者氏名・住所、保証内容（**滞在費 / 帰国旅費 / 法令遵守の保証**）、署名捺印。
- **办不办的后果**：身元保証 = 法的拘束力なし、ただし 招聘者 の保証履行不能歴 = 今後の 招聘査証 申請で「招聘者信用評価」に影響。**招聘者収入が低い + 両親の経済力立証もない場合 = 査証拒否リスク高**。

### 4. 招聘者側 立証書類（在日本 招聘者）
- **在哪办**：在職証明 = 勤務先。課税証明書 / 住民票 = 区役所 / 市役所。
- **要什么材料**：
  - 在職証明書（3 ヶ月以内）or **法人登記事項証明書**（経営者の場合）
  - 直近年 課税証明書 + 納税証明書（市役所）
  - 住民票（マイナンバー記載なし、世帯全員）
  - 在留カード 写し（表裏両面）
  - 招聘者 護照 写し
- **办不办的后果**：書類不備 / 期限切れ（多くは 3 ヶ月以内）= 査証拒否要因。**課税証明書 = 招聘者の経済力立証 で重要**。年収が極端に低い + 両親側資金力立証もない = 拒否リスク。

### 5. 両親側（中国側）立証書類
- **在哪办**：中国国内戸籍所在地 公安局 / 公証処。査証申請 = 在中国日本国大使館 / 領事館 or [査証申請センター（JVAC）](https://www.visaforchina.org/)。
- **要什么材料**：
  - 護照（残存有効期間 6 ヶ月以上推奨）
  - 査証申請書（外務省様式、写真 4.5cm × 4.5cm）
  - **戸口簿 全頁 写し**（家族構成立証）
  - **親族関係公証書 + 翻訳公証**（招聘者との続柄立証 — 父母と子 / 祖父母と孫 等）
  - **出生公証書 + 翻訳公証**（招聘者の出生証明、親子関係立証）
  - 在職証明書（在職時）or 退職証明（退職者）
  - 銀行残高証明書（経済力立証、3 ヶ月以内、推奨残高 = 滞在期間相応 e.g. 30 日 = 数万元〜十万元 程度の目安）
  - 申請手数料（短期滞在シングル = 約 200 元）
- **办不办的后果**：**親族関係立証書類 不備 = 査証拒否 直行**。中国の戸口簿 だけでは 親子関係立証 不足、**親族関係公証書（中国側 公証処発行）+ 日本領事館認証 or 単独 公証処 翻訳公証** 必須。

### 6. 在外公館 / 査証申請センター（JVAC）申請
- **在哪办**：[在中国日本国大使館](https://www.cn.emb-japan.go.jp/itpr_ja/visa.html) / 各領事館 / 査証申請センター（北京・上海・広州 等 主要都市）。**個人申請ではなく 指定旅行代理店 経由 が原則**（観光査証 と同じ運用）。
- **要什么材料**：上記 5 + 6 の全書類一式、護照、写真、申請書。
- **办不办的后果**：**処理期間 = 通常 5 営業日**、繁忙期は 2 週間〜1 ヶ月。**直前申請 = 来日予定日に間に合わないリスク**。航空券日付調整可能性も含めて 4 週間前提出が安全圏。

## 容易遗漏的事

### 坑 1：航空券「購入済」要件は不要、但仮押さえ情報は必要
よく誤解 = 「査証申請には航空券購入が必要」→ **不要**。**未購入 でも 仮予約番号 / 旅行会社見積 / 予定便名 等の情報 で代替可**。実務では **査証発給後に正式購入** が一般的（拒否時のキャンセル料回避）。**ただし 滞在予定表 は具体的日付・便名 記載必須** — 「日付未定」は不可。

### 坑 2：マルチエントリー（数次） vs シングル（一次）
中国国民の短期滞在査証 = **シングル（1 回限り、3 ヶ月有効、1 回 90 日以内）が標準**。マルチ（数次、3 年〜5 年有効）= **過去訪問歴 + 経済力立証 + 査証発給歴 等の条件** クリアで取得可（2017 年〜中国向け緩和実施）。**初回招聘 = ほぼシングル**、複数回訪問予定でも初回はシングル受け入れ前提。

### 坑 3：両親の年齢 + 健康状態 と 査証審査
**70 歳以上 + 既往症あり = 海外旅行保険 加入推奨**（査証審査で必須要件ではないが、招聘者の身元保証範囲に「医療費」が含まれるため 実費負担リスク）。**85 歳以上 = 査証審査で「健康診断書」を求められる場合あり**。日本側 医療費 = 短期滞在者 は 国保 加入不可、**全額自己負担 + 海外旅行保険 のみ**。緊急入院 1 週間 = 数十万円〜100 万円超 の事例あり。

> # ⚠️ 需创始人/書士复核：マルチエントリー（数次）査証 の最新発給条件 + 中国査証申請センター（JVAC）の指定旅行代理店経由 義務範囲
