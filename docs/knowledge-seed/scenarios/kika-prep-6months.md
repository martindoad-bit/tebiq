# 帰化申请前 6 个月准备清单

## 适用场景
对象：居住満 5 年以上、20 歳以上、素行善良、独立生计能力、日本国籍取得意愿明确者。触发：永住与帰化两项里选择了 帰化（或日本人配偶者特例 / 简易帰化）。不适用：未満足 5 年居住条件 / 中国国籍放弃意向不明者。

## 时间线

### 6 月前
- 必办：管轄 法務局（住所地）窗口面谈预约 — 事前面谈 + 必要书类清单获取。
- 必办：列举 直近 5 年 居住履历 / 在职履历 / 出国履历 自查。
- 必办：日本語能力自查（小学 3 年生程度 読み書き）。

### 3 月前
- 必办：取得 中国 国籍证明 / 出生公证 / 親族関係公証書（中国驻日大使馆 / 领事馆）。
- 必办：直近 5 年 課税・納税証明書 / 国民年金 / 健康保険 缴纳记录。
- 必办：在职証明 + 直近 3 年 源泉徴収票。

### 1 月前
- 必办：帰化動機書（自筆 A4 / 自由形式）定稿。
- 必办：履歷書（自筆）/ 親族概要 / 生计概要 / 事業概要（経営者）记入。
- 必办：写真（5cm × 5cm，6 个月内）。

### 申請当日
- 必办：管轄 法務局 窗口提交，本人出頭。
- 必办：受付印 取得，記録 申請受付日。

### 面接当日（申請后 3-6 个月）
- 必办：法務局 担当官 面接（日語）— 動機 / 家族 / 在职 / 反対者有無 等。

## 必办事项

### 1. 居住要件（5 年以上 / 1 年同一公司 推奨）
- **在哪办**：管轄 法務局 国籍課。
- **要什么材料**：在留カード / 旅券（出入国記録）/ 住民票 5 年分。
- **办不办的后果**：未満足 5 年 = 不受理。简易帰化（日本人配偶者 / 元日本国民 等）有缩短特例。

### 2. 公的義務（税金 / 年金 / 健康保険）
- **在哪办**：市役所（住民税）/ 税務署（所得税）/ 年金事務所（年金）/ 市役所 国保課。
- **要什么材料**：直近 5 年 課税・納税証明書 + 年金保険料納付確認書 + 健康保険料 納付証明書。
- **办不办的后果**：1 件未納 / 延滞 = 素行要件 不満，多被劝告補完后再申請。

### 3. 中国国籍 関連書類
- **在哪办**：在日中国大使馆 / 总领事馆 + 中国户籍所在地（亲属代办）。
- **要什么材料**：出生公証書 / 親族関係公証書 / 国籍公証書（公证处 + 駐日大使馆 認証）/ 戸籍記載事項（中国户口本 翻訳）。
- **办不办的后果**：缺漏 = 法務局 不受理；旧字体姓名 / 簡体字 / 繁体字 表記不一致 = 全件 補正。

### 4. 帰化動機書（自筆）
- **在哪办**：自宅作成、本人自笔记入（A4 / 鉛筆不可）。
- **要什么材料**：来日経緯 / 在留中体験 / 帰化決意 理由 / 日本社会貢献意欲。
- **办不办的后果**：他人代筆 / 印刷 = 即返戻。日本語表現が稚拙すぎる = 言語能力要件 不安要素。

### 5. 履歷書 / 親族概要 / 生计概要
- **在哪办**：法務局 提供様式记入。
- **要什么材料**：履歷書（出生 → 现在）/ 親族（含中国本国 父母 兄弟姉妹）/ 月収 月支 資産負債 表。
- **办不办的后果**：親族欄 漏記 中国本籍家族 = 反共・反日活動関係者 否定の审查段階で疑义。

### 6. 在职・事業 関連
- **在哪办**：会社 / 個人事業（自営）。
- **要什么材料**：在職証明書 + 源泉徴収票 + 雇用契約書（会社員）；確定申告書 + 営業許可証 + 取引先 一覧（自営）。
- **办不办的后果**：生計の独立要件 不満 = 不许可。同居家族（配偶者）的収入合算可。

### 7. 言語能力（小学校 3 年生程度）
- **在哪办**：面接当日 担当官 直接 評価（簡単な漢字読み書き）。
- **要什么材料**：JLPT 不要、面接で 自然対話。
- **办不办的后果**：判读 / 記入不可 = 学习半年再申请劝告。

### 8. 申請料
- **在哪办**：無料（永住の ¥8,000 と異なる）。
- **要什么材料**：—
- **办不办的后果**：—

## 容易遗漏的事

### 坑 1：中国国籍喪失届 タイミング
帰化許可后，中国国籍法 第 9 条「自动喪失」是日本侧解釈，中国侧需要本人到驻日大使馆办理「退出中国国籍申请」並取得「退出国籍许可」。不办理 = 中国侧仍然认定为中国公民 = 中国出入国时风险。許可后 1-3 ヶ月内办理。

### 坑 2：親族欄「反対者」自查
申請書「親族の同意」记入欄 — 中国本国 父母 / 兄弟姉妹 反対者有無 必记入。即使无法获得书面同意，「反対」记入是诚实义务。隐瞒后被审查中识破 = 不许可的强力理由。

### 坑 3：交通違反 / 駐車違反 5 年内
帰化是「素行善良」要件最严格的审查（永住より厳しい）。5 年内反則金 1 件以上 = 多被劝告 1 年以上无违反后再申请。罰金以上 1 件 = 5 年経過する前に申請不可（事実上）。
