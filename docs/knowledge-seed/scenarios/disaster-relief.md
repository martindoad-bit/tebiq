# 灾害发生后救助申请清单

## 适用场景
对象：日本国内で 地震 / 台風 / 水害 / 土砂災害 等 自然災害により 住居 / 家財 / 生計 に被害を受けた者（在留資格 不問、観光ビザ 等 短期滞在は対象外の制度あり）。触发：罹災 = 自宅 損壊 / 避難所 入所 / 地域 災害救助法 適用。不适用：火災（自己原因）/ 海外 災害 / 工事 起因。

## 时间线

### 罹災当日
- 必办：身体安全 + 避難所 / 安全な場所 確保。
- 必办：被害状況 写真撮影（外観 / 室内 / 家財 / マイナス側面 + 全方位）。
- 必办：在留カード / 旅券 / 健康保険証 / 通帳 携帯（避難時必携）。

### 罹災 1-7 日
- 必办：市役所 / 区役所 災害対策本部 へ「罹災届」提出 → 罹災証明書 申請。
- 必办：避難所 入所手続（食料 / 水 / 毛布 等 配給対象）。
- 必办：会社 / 学校 連絡 + 在留資格 関係（特例期間 / 出国困難 等）相談。

### 罹災 1 月内
- 必办：罹災証明書 受領（市役所 調査員 現地確認後 発行）。
- 必办：災害弔慰金 / 災害障害見舞金 申請（家族 死亡 / 障害時）。
- 必办：被災者生活再建支援金（基礎支援金）申請開始。

### 罹災 3 月内
- 必办：義援金 配分申請（日赤 / 共同募金会 / 自治体）。
- 必办：火災保険 / 地震保険 請求（保険会社）。
- 必办：住宅応急修理制度 / 公営住宅 一時入居 申請。

### 罹災 1 年内
- 必办：被災者生活再建支援金（加算支援金）住宅再建後 申請。
- 必办：税金 / 社会保険料 減免申請（市役所）。

## 必办事项

### 1. 罹災証明書
- **在哪办**：市役所 / 区役所 税務課 / 災害対策本部。
- **要什么材料**：罹災届 + 被害状況写真 + 本人確認書類（在留カード）。
- **办不办的后果**：全壊 / 大規模半壊 / 中規模半壊 / 半壊 / 準半壊 / 一部損壊 6 区分。証明書 = 各種支援制度の前提条件。未取得 = 義援金 / 弔慰金 / 支援金 全部 申請不可。発行 1-4 週間（被害規模による）。

### 2. 災害弔慰金（家族死亡時）
- **在哪办**：市役所 / 区役所 福祉課。
- **要什么材料**：死亡診断書 + 罹災証明書 + 申請書 + 戸籍 / 親族関係証明。
- **办不办的后果**：生計維持者 死亡 = ¥5,000,000 / その他家族 = ¥2,500,000。災害弔慰金法 適用、自治体 負担。

### 3. 災害障害見舞金（重度障害時）
- **在哪办**：市役所 福祉課。
- **要什么材料**：医師診断書 + 罹災証明書。
- **办不办的后果**：両眼失明 / 両上肢失う 等 重度障害 = 生計維持者 ¥2,500,000 / その他 ¥1,250,000。

### 4. 被災者生活再建支援金
- **在哪办**：市役所 → 都道府県 → 被災者生活再建支援法人。
- **要什么材料**：罹災証明書（全壊 / 大規模半壊 / 中規模半壊）+ 申請書 + 通帳。
- **办不办的后果**：基礎支援金（被害程度）+ 加算支援金（再建方法）= 最大 ¥3,000,000。
  - 全壊 = 基礎 ¥1,000,000 + 加算（建設・購入 ¥2,000,000 / 補修 ¥1,000,000 / 賃貸 ¥500,000）
  - 大規模半壊 = 基礎 ¥500,000 + 加算同上
  - 中規模半壊 = 基礎 ¥250,000 + 加算同上（2020/12 追加）
  - 期限：基礎 13 ヶ月内 / 加算 37 ヶ月内。

### 5. 義援金
- **在哪办**：日本赤十字社 / 中央共同募金会 / 自治体。
- **要什么材料**：罹災証明書 + 申請書 + 通帳。
- **办不办的后果**：被害規模 / 被害区分 / 募金状況 によって 配分額決定（一律ではない）。災害規模大なら 全壊 数十万円-100 万円規模。

### 6. 住宅応急修理制度（半壊以上）
- **在哪办**：市役所 災害対策本部。
- **要什么材料**：罹災証明書 + 修理見積書 + 申請書。
- **办不办的后果**：屋根 / 壁 / 床 / 水道 等 日常生活 必要箇所 修理 = 上限 ¥706,000（半壊 / 大規模半壊 / 全壊）/ ¥343,000（準半壊）。市町村 が 業者 直接支払（本人立替不要）。

### 7. 火災保険 / 地震保険
- **在哪办**：契約 損害保険会社。
- **要什么材料**：保険証券 + 罹災証明書 + 被害状況写真 + 修理見積書。
- **办不办的后果**：火災保険 = 風水害 / 雪害 対象（地震・噴火・津波 不対象）。地震保険 = 地震・津波・噴火 起因 = 火災保険の 30-50% 範囲、上限 5,000 万円（建物）+ 1,000 万円（家財）。請求 3 年時効。

### 8. 在留資格 / 出入国 関係
- **在哪办**：出入国在留管理庁（地方入管 / 各種ホットライン）。
- **要什么材料**：罹災証明書 + 在留カード + 旅券。
- **办不办的后果**：災害により 出国困難 / 在留期間更新 期限内不可 = 特例措置（在留期間 1-3 ヶ月延長 / 申請手数料 減免 等）。在留カード 紛失 = 14 日内再交付申請（災害特例 期間延長可）。

## 容易遗漏的事

### 坑 1：被害状況写真 = 罹災証明書 査定の鍵
市役所 調査員の現地調査までに 数日〜数週間。その間 片付け / 修理 で原状失われると 被害区分 過小評価のリスク。罹災当日 / 翌日に 全方位（外観 4 面 / 室内 各部屋 / 屋根 / 床下 / 家財 個別）写真。撮影日時 EXIF 残しで 日付証拠化。

### 坑 2：火災保険 vs 地震保険 起因区分
津波被害 = 地震保険のみ（火災保険 不対象）。台風で家屋倒壊 → 二次的火災 = 火災保険対象。被害原因の区分が 保険金支払 可否を分ける。請求書 記入時「地震 / 津波 / 火災 / 風 / 水」起因明示。

### 坑 3：在留資格 期限切れ + 災害
災害で 在留期間更新 / 在留資格変更 の申請ができない場合 = 特例措置 利用可。事前連絡（電話 / メール）+ 後日 申請が原則。何もしないで期限切れ = 不法滞在扱い、災害特例 適用も困難化。
