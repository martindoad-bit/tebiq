# 我的档案如何帮助行政書士判断

## 这事跟我有关吗？
你预约了 30 分钟 ¥9,800 行政書士 咨询，希望尽量在这 30 分钟里拿到可执行的方案，而不是把一半时间花在重复讲述自己的来日履历、转职次数、收入区间。档案能否在咨询前提供一份完整事实底盘，直接决定这次咨询的产出密度。

## 这是什么
行政書士 在 在留審査 / 永住申請 / 帰化申請 / 補正回应 等场景下做出判断，依赖的核心是三层信息：

- **客观事实**：履历、出入国记录、收入、纳税、届出 是否齐备
- **个体情况**：家庭结构、转职原因、滞納原由、健康状况
- **当前规则**：moj.go.jp 公开的最新审查要点 / 各 地方入管 实务倾向

其中"客观事实"这一层，靠用户口述容易出现：

- **遗漏**：忘记 2024 年短暂滞納过 1 期 国民年金、忘记中途有过 14 日的离日记录
- **时间错位**：把"提出 退職届"的日期记成实际离职日、把"届出 完成"的时间记成"决定 离职"的时间
- **混淆文书**：把 課税証明書 当成 納税証明書、把 雇用契約書 当成 在職証明書

档案则可以在咨询前直接提供：

- **完整时间线**：在留期間 / 转职 / 婚姻 / 住所 / 届出 全部按日期排列
- **已识别文书**：每张拍照入档的纸由 doc_type 自动归类，行政書士 不需要逐张辨认
- **公租公課 缴纳履历**：所得税、住民税、国民年金、国民健康保険 跨年度记录
- **转职履历**：每段雇用 起止日期 / 岗位 / 是否完成 契約機関 変更届出
- **出入国记录**：再入国 / 短期出国 / 在外日数

30 分钟 ¥9,800 咨询的结构因此从"先听 15 分钟用户口述、再讨论 15 分钟方案"，变成"行政書士 3 分钟看完档案、27 分钟讨论方案"。同样的 30 分钟，方案讨论时间几乎翻倍。

## 你应该做什么
1. 在咨询预约前 1 周，把档案补齐到当前时点：最近 1 次的 課税証明書 / 納税証明書、最近 1 次的 在職証明書 / 給与明細、护照最新页（含最近的出入国印章）。
2. 检查档案的"未识别文书"列表。任何标记为"待识别"的纸，要么补充信息让档案归类、要么从档案删除。
3. 在预约表单的备注栏简要写明咨询主题（续签 / 永住 / 帰化 / 補正 / 不許可后再申请），让 行政書士 提前定位档案中要重点查看的部分。

## 不做的后果
档案不齐 → 咨询时仍要花时间做事实复核 → 30 分钟实际方案讨论时间被压缩到 10-15 分钟。¥9,800 的边际产出降低不是行政書士 的判断质量问题，而是输入信息密度问题。极端情况下，事实层不清晰，行政書士 只能给出"原则性建议"，而不是"针对你 case 的具体步骤"。

## 什么时候你应该找专家
你计划做 永住申請 / 帰化申請 / 在留資格変更、或刚收到 補正通知 / 不許可通知、或有 2 年以上未审视的 在留歴史，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
