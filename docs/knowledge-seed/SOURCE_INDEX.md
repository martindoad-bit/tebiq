## 引用模式（合规说明）

每篇内容文件的「## 信息来源」段落严格遵循：

```markdown

```

**取得日 2026-04-29** 全部统一 — 上线后由行政書士复核时可对照取得日检查 url 是否仍可达 / 内容是否仍最新。
