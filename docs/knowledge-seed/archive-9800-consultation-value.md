# TEBIQ ¥9,800 咨询的真实价值是什么

## 这事跟我有关吗？
你正在判断 ¥9,800 是否值得花。这篇说明定价逻辑、与市场标准的对比、以及哪些 case 适合 / 不适合用这次咨询。

## 这是什么
**市场参考价：**

行政書士 30 分钟咨询的市场标准价格区间是 **¥9,000-¥15,000**，随事务所规模与咨询主题复杂度而变。日本行政書士会連合会 没有强制价目，各事务所自定。

- 単独事务所、一般咨询：¥9,000-¥11,000 / 30 分钟
- 都心中等事务所、入管 / 永住 / 帰化 主题：¥10,000-¥15,000 / 30 分钟
- 大型事务所、英语 / 中文对应、企业 case：¥15,000-¥30,000 / 30 分钟

TEBIQ ¥9,800 属于该区间的中等定位。

**与单独预约行政書士 的区别：**

普通预约行政書士 的咨询流程：

- 你到事务所 / 接通电话
- 行政書士 先听 10-15 分钟你口述履历 / 现状
- 剩下 15-20 分钟做方案讨论
- 期间需要的客观事实（你某年的课税额、某次转职的具体日期）你回答不出来，只能"回去查了再说"

TEBIQ ¥9,800 咨询的流程：

- 你预约时授权档案给指定行政書士
- 行政書士 在咨询前用 3-5 分钟通览档案
- 30 分钟全部用于方案讨论
- 期间需要的客观事实直接在档案中查证，不依赖你现场回忆

差别不在 行政書士 的判断质量（同一位行政書士 在两个场景下给的法律意见相同），而在 30 分钟的产出密度。

**适合 ¥9,800 咨询的 case：**

- 你收到入管 補正通知，自己读不懂哪些段落是关键、哪些是格式要求
- 你考虑 永住申請，但要件中"継続して 10 年以上 在留"或"独立生計"判断不确定
- 你刚转职 / 计划转职，担心新岗位与 在留資格 实质契合度
- 你 不許可 一次后想再申请，需要判断"以同一理由再申请"还是"等 6 个月 / 改变 在留資格"
- 你 帰化申請 前需要判断 国籍要件 / 住居要件 / 生計要件 是否齐备
- 你收到 質問書，不知道如何如实回答又不被解读为不利证据

**不适合 ¥9,800 咨询的 case：**

- 单纯材料填表（在留期間更新申請書 自己能填，不需要付费咨询）
- 政策概念了解（"永住要件大致是什么" → 看 TEBIQ 知识库 / 入管 公式即可，免费）
- 政策追踪（"2025 年 永住 制度修订了什么" → 看 TEBIQ 政策追踪即可，免费）
- 对方案已有判断、只想"找人确认一下"（你已经有判断时，咨询的边际价值低；除非判断错了的成本极高）

## 你应该做什么
1. 在预约前，先列出 3 个具体问题。30 分钟通常够回答 3-5 个具体问题，但只够回答 1-2 个抽象问题。
2. 把这 3 个问题写进预约备注，让行政書士 提前定位档案中相关部分。
3. 咨询时优先讨论"决策卡点"——不确定怎么选 / 不确定怎么写的事项，而非"概念理解"。

## 不做的后果
不区分"概念理解"和"决策卡点"，把 ¥9,800 咨询用于了解概念，会得到一份你在 TEBIQ 知识库或 入管 公式页面也能读到的答案。¥9,800 的价值在"针对你 case 的具体步骤"，不在"通用知识"。

## 什么时候你应该找专家
你 case 涉及 永住 / 帰化 / 不許可后再申请 / 補正回应 / 質問書 回应 / 在留資格変更 等具体决策卡点，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
