# 老婆要来日本陪我，我能选哪些路径 / 哪条最快

## 这事跟我有关吗

如果你符合以下条件，请继续往下看：

- 你已经在日本，持有 就労系 / 永住者 / 日本国籍 任一身份
- 配偶（妻 / 夫）目前在中国大陆 或第三国，准备来日生活
- 已经合法结婚 或 即将结婚（中日两边登记完成）

未婚同居 / 事実婚 不适用本条——日本入管不承认事実婚关系作为 配偶者签证 基础。

## 这是什么

配偶来日的路径不是单一答案，要看你和配偶各自的身份组合，主要分 3 条：

### 路径 1：配偶者签证（最快，要求最严）

- 你是 **日本人** → 配偶申请 「日本人の配偶者等」
- 你是 **永住者** → 配偶申请 「永住者の配偶者等」
- 配偶在日 **可以全职就劳，无职种限制**
- 配偶将来申请 永住 时，**婚姻 3 年 + 在日 1 年** 即可（一般要在日 10 年）
- 入管审查重点：婚姻是否真实（写真 / チャット履歴 / 同居計画 / 交際経緯書）

### 路径 2：家族滞在（次快，限制多）

- 你是 **就労系 在留資格 持有人**（技人国 / 経営・管理 / 教授 / 技能 等）→ 配偶申请 「家族滞在」
- 配偶在日 **本身禁止就劳**，申请 資格外活動許可 后 28 小時 / 週 上限（详见 [副业条目](side-job-legal-scope.md)）
- 你需要证明 **抚养能力**：年収 一般 200–300 万円 以上 + 安定的 雇用 + 居住可能 住居
- 高度専門職 持有人有特别优待：配偶可以全职就劳（家族滞在 是被覆盖在 高度専門職 特定活動 框架下的）

### 路径 3：配偶以独立资格来

- 配偶有自己的職歴 / 学歴 → 直接申请 技人国 / 高度専門職 / 経営・管理
- 配偶要继续学业 → 留学
- 双方都是独立身份 → 不依赖配偶关系，将来離婚 也不影响在留

### 速度比较（一般情况）

| 路径 | 在留資格認定証明書 审查 | 国外签证 + 入境 | 合计 |
|---|---|---|---|
| 1. 配偶者签证（日本人 / 永住者 配偶） | 1–3 月 | 2–4 周 | 1.5–4 月 |
| 2. 家族滞在 | 1–3 月 | 2–4 周 | 1.5–4 月 |
| 3. 独立资格（技人国 等） | 1–4 月 | 2–4 周 | 1.5–5 月 |

差异主要不是审查时间，而是**通过率**——路径 1 婚姻真实性审查严，路径 2 抚养能力审查严，路径 3 看个人条件。

## 你应该做什么

### Step 1：先在中国大陆完成 結婚登記（已结婚的跳过）

- 民政局 結婚登記 → 拿 結婚証
- 結婚証 公証 + 中国外交部 認証 + 在中国日本大使館 認証（或新规以来的アポスティーユ流程）

### Step 2：你（在日方）准备资料并提交 在留資格認定証明書交付申請

提交人 = 你（在日代理）；提交地点 = 你居住地管轄入管局；申请书 = 在留資格認定証明書交付申請書。

**配偶者签证（路径 1）必备**：

- 申請書 + 写真
- 中国 結婚証 + 公証 + 認証 / 婚姻関係証明書
- 戸籍謄本（日本人配偶）or 在留カード + 住民票（永住者配偶）
- 質問書（婚姻经过、双方家庭、交際開始日 / 同居開始日）
- 写真（交際〜結婚 経過、双方家族、結婚式 等 5–10 张）
- 课税証明書 + 納税証明書（直近 1 年）
- 在職証明書 + 雇用契約書

**家族滞在（路径 2）必备**：

- 申請書 + 写真
- 中国 結婚証 + 公証 + 認証
- 你的 在留カード + 住民票
- 你的 在職証明書 + 課税証明書 + 納税証明書
- 你的 預金残高証明書（资产佐证）
- 配偶的 護照 复印件 + 写真

### Step 3：入管发出 認定証明書

- 1–3 月，邮寄到你日本住址
- 你把 原件 邮寄给国外的配偶

### Step 4：配偶在国外日本大使馆 / 領事館 申请签证

- 持 認定証明書 + 護照 + 戸口簿 等到管轄領事館（中国境内 5 个：北京 / 上海 / 広州 / 重慶 / 瀋陽，按户籍地分管）
- 5 工作日 ～ 2 周左右

### Step 5：入境后 14 日内

- 在 空港 領取 在留カード（成田 / 羽田 / 関空 / 中部 / 福岡 / 新千歳 等可即时发放）
- 14 日内到市区町村役所 住所届 + マイナンバー 取得 + 国民健康保険 加入

## 不做的后果

- 婚姻真实性 立証 不足 → 不交付認定証明書（"虚偽結婚" 嫌疑会留底，将来再申请极难）
- 抚养能力 立証 不足（年収 200 万 以下 + 没有 預金 佐证） → 家族滞在 不许可
- 中国 結婚証 公証 / 認証 缺失 → 申请直接退回
- 入境 14 日内 不办住所届 → 5 万円 以下 過料 + 信用扣分

## 何时找专家

下面任一情况，请联系 行政書士 / 入管専門 弁護士：

- 你和配偶 **年龄差 15 岁以上** / 交往时间短 / 结婚 1 年内（入管会着重审查 真実性）
- 你年収 < 300 万円 申请 家族滞在（需要 預金 / 親族支援 等補強）
- 配偶有过 **日本入境拒否歴** / 不法滞在歴
- 国際結婚仲介 介绍认识（不一定不许可，但 質問書 难写）
- 婚姻经历复杂（再婚 / 双方都有前段婚姻 / 跨国前段婚姻離婚証明 不齐）
- 你是 高度専門職 想让配偶利用 70 点优待全职就劳（需要正确指定 特定活動 类別）

書士 在 配偶来日 节点的最大价值是 質問書 / 交際経緯書 的写法——这两份是入管判断婚姻真实性的核心，写得好 1–2 月通过，写得不到位 半年以上 反复要求追加资料，最后还可能驳回。
