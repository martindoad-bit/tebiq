# 1 号 通算 5 年 上限 + 2 号 移行 / 家族帯同 戦略

## 这事跟我有关吗？
你持 特定技能 1 号、累计在留 1 ~ 5 年 之间、或正在考虑 2 号 移行 + 家族帯同。本篇是给 1 号 後半 ~ 2 号 移行 検討 期 的人看的長期戦略。

## 这是什么
特定技能 制度 の 構造的特性：

**1 号 特定技能**
- **通算 5 年 上限**（入管法 + 制度告示）
- 在留期間：4 ヶ月 / 6 ヶ月 / 1 年 を 更新（最長 5 年 まで）
- 一時帰国 期間 含めて 通算（再入国 期間 も 算入 が 原則）⚠️ 算定 細則 は 業種別 ガイドライン 確認
- **家族帯同 不可**（配偶者 + 子 の「家族滞在」付与 されない）
- 業種別 試験 + 日本語試験 合格 必須

**2 号 特定技能**
- **通算 上限 なし**（在留期間 更新 を 続ける限り 滞在可）
- 在留期間：6 ヶ月 / 1 年 / 3 年 を 更新
- **家族帯同 可**（配偶者 + 子 が「家族滞在」で 帯同可）
- 業種別 2 号 試験 + 一定 実務経験 + 監督経験 が 要件
- 2024 年 以降 受入 業種 拡大（介護 を 除く 多 業種 で 2 号 開設）⚠️ 介護 は 別途「介護」在留資格 経由

**永住 ルート へ の 接続**
- 1 号 在留期間 は 永住 申請 の 居住要件 に 原則 算入されない
- 2 号 期間 は 算入 対象 になる 取扱い + 永住 申請 可能性 あり ⚠️ 入管 ガイドライン 最新版 確認

**5 年 到達 時 の 選択肢**
1. 2 号 移行（試験 + 実務経験 要件 充足）
2. 別 在留資格 へ 変更（技人国 / 技能 等、要件 充足 する 場合）
3. 帰国（再来日 は 別 在留資格 ベース）

5 年 上限 計算 ミス + 駆け込み 申請 = 否認 → 在留期限 経過 = 不法残留 リスク。

## 判断标准
- **绿**：q1=2号 或 q1=1号 + q2<3 年（余裕 あり）
- **黄**：q1=1号 + q2=3-5 年（移行 準備 急務）
- **红**：q1=1号 + q2=5 年 到達（続签 不可 状態）

## 处理建议

### 绿
档案归档：
- 在留カード（在留期限 + 区分）
- 全 在留期間 履歴（過去 在留カード コピー / 旅券 査証 + スタンプ）
- 試験合格証（1 号 / 2 号 該当分）
- 雇用契約書 + 直近 給与 履歴
- 家族帯同 場合：配偶者 / 子 の 家族滞在 関連書類（2 号 のみ 該当）

2 号 場合 6 ヶ月前 復検。

### 黄
1. 過去 全 在留カード + 旅券 で 通算 在留 月数 を 計算（一時帰国 期間 算入 / 不算入 を 業種別 ガイドライン 確認）。
2. 業種別 2 号 試験 開催 状況 + 受入機関 の 2 号 受入 体制 確認。
3. 2 号 移行 申請 = 在留資格変更 申請。試験合格 + 実務経験 + 雇用契約 が 揃ってから 提出。
4. 家族帯同 計画 ある 場合、2 号 取得後 家族滞在 認定証明書 申請 を 並行。

### 红
1. **5 年 到達 + 2 号 試験 未合格** → 直近 試験 受験 + 業種別 受入機関 と 連携。試験 合格前 に 在留期限 到達 する 場合 帰国 + 試験合格後 再入国 ルート。
2. **5 年 到達 + 業種 で 2 号 未開設** → 別 在留資格 への 変更（要件 充足 場合）または 帰国。
3. **算入 期間 に 認識 ズレ** → 入管 へ 在留歴 照会 + 証明書 発行 申請。
4. **不法残留 リスク**（在留期限 経過 後 滞在）→ 即時 出頭 + 出国命令 / 在留資格変更 検討。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 在留カード（現行 + 過去分）
- 旅券（査証 + 上陸スタンプ 履歴）
- 在留期間 算定 メモ（一時帰国 期間 含む）
- 試験合格証（1 号 / 2 号）
- 雇用契約書 + 給与振込 履歴
- 1 号 特定技能外国人 支援 計画書
- 家族滞在 関連書類（2 号 + 帯同 場合）
- 在留資格変更許可申請書（2 号 移行 時）
