# 事務所 賃貸 vs 自宅兼用 / 自有物件的边界

## 这事跟我有关吗？
你是 経営・管理 持有人 + 設立 1-3 期 + 事務所コスト 圧縮 中。或你检查续签材料发现 事務所 项不清楚。这篇适合此场景。

## 这是什么
出入国在留管理庁 経営・管理ガイドライン 对「事業所」要件的核心：
- 物件 が 独立 + 継続的に 確保されている
- 事業 内容 に 適合する 規模・設備
- 居住空間 と 事業空間 が 区分されている

**バーチャルオフィスのみ**（住所貸し + 郵便受け + 共有会議室）= 経営・管理 では 原則 NG。実体無し と判断。

**自宅兼用**は条件付き OK：
- 大家 / オーナー の 事業使用 承諾（賃貸契約書 上「事務所兼住居」記載 or 別途 承諾書）
- 居住部分と物理分離（玄関 / 部屋 / 看板 等）
- 法人名義での 光熱費 / 通信費 一部負担 or 案分

**法人 自有物件**（法人名義で購入）= 最強。登記簿 + 固定資産税 納税通知書 で証明。

**法人 賃貸**（法人名義で賃貸契約）= 一般推奨。賃貸契約書 + 重要事項説明書 + 賃料 領収書 で証明。

2025/10/16 改正後 経営実態 審査強化。バーチャルオフィスのみ + 自宅一室で看板無し ≒ 不許可 直結。

## 判断标准
- **绿**：q1=法人賃貸 or 法人自有 + q3=yes（使用目的 明確「事務所」or「事務所兼住居」）
- **黄**：q1=自宅兼用 + q2=yes（物理隔離有り） + q3=yes（用途 明記）
- **红**：q1=バーチャルオフィスのみ / q1=自宅兼用 + q2=no（無隔離） / q3=no（賃貸契約書「居住専用」のみ）

## 处理建议

### 绿
档案归档：
- 法人賃貸：賃貸借契約書（法人名義 / 使用目的「事務所」明記）+ 重要事項説明書 + 直近 3 ヶ月 賃料 振込 履歴
- 法人自有：登記事項証明書（建物）+ 固定資産税 納税通知書 + 取得時 売買契約書
- 写真：外观 + 看板 + 入口 + 内部 + 独立 郵便受
- 光熱費 領収書（直近 3 ヶ月）

下次续签前 180 天复检。

### 黄
1. **自宅兼用 + 物理隔離あり**：
   - 平面図（居住部分 vs 事業部分 色分け、面積 案分）
   - 写真：独立入口 / 看板 / 専用部屋 / 専用 郵便受
   - 大家 事業使用 承諾書（書面、署名 + 印）
   - 賃貸契約書「使用目的」欄に「事務所兼住居」明記 or 覚書 で 追記
   - 光熱費 / 通信費 法人 案分（按比例 法人計上）
2. 写真は 外観 + 看板 + 入口 + 内部 + 独立 郵便受 + 平面図 セット で続签 補足資料 として提出。

### 红
1. **バーチャルオフィスのみ**：実体 事務所 確保 必要。法人賃貸 契約 + 引越し → 本店所在地 変更 登記（2 週間以内、法務局）+ 入管「契約機関に関する届出」（14 日以内）。
2. **自宅一室 + 隔離無し + 看板無し**：物理隔離 + 看板 設置、独立 郵便受 設置。it 仍可能不足 → 法人賃貸 への 切り替え 検討。
3. **賃貸契約書「居住専用」**：大家 と 用途変更 合意 + 契約書 巻き直し or 覚書。大家 拒否 = 移転 必要。
4. **本店所在地 と 実態事務所 不一致**：本店移転 登記 必要。商業登記 + 入管届出 セット。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 賃貸借契約書（法人名義 / 使用目的「事務所」or「事務所兼住居」）
- 重要事項説明書
- 大家 事業使用 承諾書（自宅兼用 / 個人名義 賃貸 場合）
- 登記事項証明書（建物）+ 固定資産税 納税通知書（自有 場合）
- 事務所 写真（外観 + 看板 + 入口 + 内部 + 郵便受）
- 平面図（自宅兼用 場合、案分 比率 明記）
- 光熱費 領収書（電気 / ガス / 水道 / 通信 直近 3 ヶ月）
- 法人 履歴事項全部証明書（本店所在地 確認）
