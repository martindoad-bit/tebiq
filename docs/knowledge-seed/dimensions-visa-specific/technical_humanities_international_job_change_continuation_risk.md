# 转职后续签风险 / 就労資格証明書 申请判断

## 这是什么
入管法 第 19 条の 2 规定，外国人 可向 出入国在留管理庁 申请 就労資格証明書 — 由 法務大臣 出具的「現在 在留資格 で 行うことができる 就労活動」確認文書。技人国 持有者 転職 时无强制申请义务，但实务上是規避「在留期間更新 不許可」的核心防御手段。

转职后 法定的 必办手续：

1. **「契約機関に関する届出」**：入管法 19-16 条，事由发生 14 日 内提交（前公司離脱 + 新公司加入 各 1 通）
2. **国民年金 / 厚生年金 切替**：日本年金機構（最寄り 年金事務所 / マイナポータル）
3. **健康保険 切替**：協会けんぽ → 国保 / 別会社協会けんぽ
4. **住民税 特別徴収 → 普通徴収 切替**（如适用）

「就労資格証明書」是任意手续，但若不申请且新職務 偏离三领域 → 続签时被认定 資格外活動 → 不許可 + 過去就労期间被視為違法。

## 为什么重要
転職後 続签 的 3 大风险点：

- **業種ミスマッチ**：原 在留資格 認定 是「IT エンジニア」，新公司让做「現場 SE 派遣 + 顧客先 営業」→ 営業 部分超出原认定範围 → 資格外活動
- **空白期過長**：3 ヶ月 以上 無職 + 無 求職活動証拠 → 「指定された活動を継続して 3 ヶ月以上行っていない」→ 在留資格 取消事由（入管法 22-4 条 1 項 6 号）
- **届出怠慢**：14 日 ルール 不履行 → 20 万円 以下 罰金（入管法 71-3 条）+ 续签 素行善良 评价 消极

永住申請 时「公的義務履行」连续 5 年 审查 — 任何一次 届出 怠慢 / 資格外活動 嫌疑都会回溯影响永住可否。帰化申請 同様。

## 判断标准
- **绿**：q1=no（无転職）或 q1=yes 且 q2=yes（業種一致）+ q3=no（無空白期）+ q4=yes（已取得 就労資格証明書）
- **黄**：q1=yes 但 q4=no / unknown（未申请 証明書）或 q3=yes（空白期 3 ヶ月超）
- **红**：q1=yes 且 q2=no（新職務 明确偏离 自然科学 / 人文科学 / 国際業務 三领域）

## 处理建议
### 绿
档案归档：退職証明書 / 在職証明書 / 新雇用契約書 / 業務内容説明書 / 「契約機関に関する届出」受理票（前公司・新公司 各 1 通）/ 就労資格証明書（写）。続签申请 时 提交 上记一套即可，原則 在留期間 3 年 / 5 年 維持。

### 黄
1. 「就労資格証明書」未申请 → 立即准备申请书 + 雇用契約書 + 業務内容説明書 + 法人登記事項証明書 + 直近年度 決算文書 + 申請人 在留カード。手数料 1,200 円（収入印紙）。审查期间 1〜3 ヶ月。
2. 空白期 3 ヶ月超 → 整理 退職理由書 + 求職活動記録（ハローワーク 求職票 / 转职エージェント 履歴 / 面接記録）+ 雇用保険 受給資格者証 / 受給状況。育休 / 産休 / 病療養 期间不算入空白期，但需 診断書 / 育児休業給付金 受給記録 等举证。
3. 「契約機関に関する届出」14 日 期限 已过 → 立即補出（電子届出 推奨：在留申請オンラインシステム）。

### 红
1. 新職務 = 単純労働 / 現場作業 / 接客販売 等 → 资格外活動 当下成立，立即停止該当業務。
2. 在留資格変更 路线评估：特定技能 1 号 / 経営・管理 / 技能 / 高度専門職 等。变更不可时 → 帰国 + 新規 在留資格認定 申请。
3. 続签前已 1 ヶ月以上 該当不適合 業務 → 申請理由書 中 経過説明 + 是正措置 説明 必备。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
