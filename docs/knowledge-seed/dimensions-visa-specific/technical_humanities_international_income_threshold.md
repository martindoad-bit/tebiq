# 收入是否达标 / 年収下限・産休育休・副業 实操

## 这是什么
入管法 別表第一の二「技術・人文知識・国際業務」基準省令 第 1 号 ニ 规定：

> 日本人が従事する場合に受ける報酬と同等額以上の報酬を受けること

法令上 没有「年収 300 万円」的明文下限，但实务运用上：

- 出入国在留管理庁 内部审查参考：年収 300 万円 前後 是 1 つの目安
- 同職種 同年齢 同経験 の 日本人 給与水準 比較（厚労省「賃金構造基本統計調査」参考）
- 月額 報酬 18〜20 万円 は 高度人材 業種 で 不適合 とされる傾向

審査 重点：

1. **雇用契約書 記載額** が 日本人同等性 を 満たすか
2. **課税証明書** 上の 実際 給与収入 が 契約額 と 乖離していないか
3. **収入 持続性** — 過去 1 年 大幅 減少 / 不安定 はないか
4. **副業・業務委託** の 資格外活動許可 取得 状況

## 为什么重要
収入 不充足 = 「報酬 同等性」否定 = 在留資格 該当性 否定 → 不許可。即便許可、収入 不安定 → 在留期間 1 年 短縮（永住・帰化 起算 リセットのリスク）。

具体的 続签 影响:

- 年収 200 万円 未満 + 産休育休 等正当理由 無 → 不許可 リスク 高
- 副業 = 雇主以外 で 報酬 受領 = 資格外活動許可（個別許可 / 包括許可）必要、無 → 入管法 19-3 条 違反 → 罰金 200 万円 以下 / 退去強制 事由
- 海外送金 受領 = 税務署 海外財産調書 / 国外送金等調書（金融機関 自動報告）→ 確定申告 漏れ は 公的義務 不履行 → 永住 不許可

永住許可ガイドライン（2019 年改正以降）「直近 5 年 各年 年収 300 万円 以上 + 過去全期間 公的年金・健康保険・住民税 滞納 0」是 实质的运用基準。

## 判断标准
- **绿**：q1=yes（雇用契約 同等性 充足）+ q2=300 万円以上、或 q3=yes（産休育休等正当事由）+ q2=200〜300 万円
- **黄**：q2=200〜300 万円（正当事由なし）/ q1=unknown / q4=yes（副業あり、要 資格外活動許可 確認）/ q2=不明
- **红**：q2=200 万円未満 / q1=no（雇用契約 額 < 日本人同等水準）

## 处理建议
### 绿
档案归档：雇用契約書 / 給与明細 12 ヶ月分 / 源泉徴収票 / 課税証明書 / 納税証明書 その 1（納税額）/ 確定申告書 控（如适用）。続签申请 時 提交 直近 1 年分 即可，3 年 / 5 年 期間 取得 確率 高。

### 黄
1. 年収 200〜300 万円 → 「給与説明書」: 業種・地域・経験年数・职责と給与の関連 を 雇主 に 出してもらう。厚労省「賃金構造基本統計調査」(https://www.mhlw.go.jp/toukei/list/chinginkouzou.html) で 同職種 平均 と 比較。
2. 副業 / 業務委託 / 海外送金 あり → 申请「資格外活動許可」:
   - 包括的許可（週 28 時間以内・風俗関係除く）→ 在留カード 裏 押印
   - 個別許可（特定 雇用主 / 業務 限定）→ 資格外活動許可申請書 + 就労理由書 + 副業先 雇用契約書
   - 海外送金 → 確定申告書 + 国外送金等調書（金融機関）+ 海外財産調書（5,000 万円超）整合
3. 産休育休 → 出産手当金 受給記録（協会けんぽ 等）+ 育児休業給付金 受給記録（ハローワーク）+ 雇主の 産休育休 取得証明書 を 添付。「収入減 = 育児休業 によるもの」を 申請理由書 に 明記。

### 红
1. 年収 200 万円 未満 + 正当事由 なし → 雇主と 給与 改定 交渉、または 転職 検討。
2. 雇用契約書 額 と 課税証明書 給与所得 に 乖離 → 雇主 に 源泉徴収 / 給与支払 整合性 を 確認、修正申告 必要 か 評估。
3. 海外送金 を 確定申告 漏れ → 修正申告 + 延滞税 完納 → 公的義務履行 を 続签 までに 是正。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
