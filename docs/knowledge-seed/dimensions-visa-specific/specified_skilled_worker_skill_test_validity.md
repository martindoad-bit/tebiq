# 試験合格証 の 有効性 + 紛失 再発行

## 这事跟我有关吗？
你持 特定技能 1 号 或正在准备 1 号 → 2 号 移行、続签提出前 60 ~ 180 日 范围。本篇是给 試験合格証 的 保管 / 有効性 / 業務区分 対応 需要核查的人看的。

## 这是什么
特定技能 制度 で 必要な 試験 は 大きく 2 系統：

**技能評価試験（業種別）**
- 介護：「介護技能評価試験」（NRCD）
- 建設：「建設分野 特定技能 1 号 評価試験」（業務区分 別、JAC）
- 自動車整備・飲食料品製造業・外食業・宿泊・農業・漁業 等：各 所管団体 が 運営
- **原則 期限 なし**（一度 合格 すれば 失効 しない 取扱い が 一般的）⚠️ 業種別 ガイドライン で 例外 確認

**日本語試験**
- JFT-Basic（A2 相当 / 国際交流基金）
- JLPT N4 以上（日本語能力試験）
- 介護 のみ 加えて「介護日本語評価試験」必要
- **原則 期限 なし**（合格 履歴 が 残る）

**1 号 → 2 号 移行 の 場合**
- 2 号 用 技能 試験（別物）受験 必須
- 業種により 一定 実務経験 + 監督経験 を 要件 化
- 日本語 要件 は 業種により 異なる（建設 = N3 相当 等）

**業種・業務区分 の 一致 が 大前提**：合格 した 業種・業務区分 と 雇用 業務 が 一致 しないと 続签 / 変更 否認。技能実習 2 号 良好修了 で 試験 免除 になっても、業種・職種 対応 は 入管 + OTIT 公式表 で 確認 必要。

合格証 紛失 時：各 試験運営団体 公式 マイページ / 問合せ窓口 へ 再発行 申請。電子合格証 を 発行 する 団体 も 増加。

## 判断标准
- **绿**：q1=yes + q2=yes（合格証 保管 + 業務区分 確認済）
- **黄**：q1 或 q2 unknown
- **红**：q1=no（合格証 不所持・紛失 状態）

## 处理建议

### 绿
档案归档：
- 技能評価試験 合格証（PDF + 紙）
- 日本語試験 合格証（JFT-Basic / JLPT N4 / 介護日本語 該当分）
- 試験 受験記録（受験日 + 業種 + 業務区分）
- 業種別 運用 ガイドライン（直近版）
- 雇用契約書（業務区分 記載）

続签 / 2 号 移行 計画 が ある なら 180 日前 から 復検。

### 黄
1. 試験運営団体 公式 マイページ で 合格 履歴 + 合格証 ダウンロード。
2. 業種別 ガイドライン（所管省庁 公式 PDF）で 業務区分 + 試験 対応 確認。
3. 雇用 業務 が 合格 業務区分 と 一致 してるか 受入機関 と 突合せ。
4. 1 号 → 2 号 移行 計画 ある 場合、2 号 試験 直近 開催 日程 を 確認。

### 红
1. **合格証 紛失** → 各 試験運営団体（介護 = NRCD、外食 = 日本フードサービス協会、建設 = JAC 等）へ 再発行 申請。電子合格証 の 場合 マイページ 再ログイン。
2. **業務区分 不一致**（合格 業務区分 と 雇用 業務 が 別物）→ 雇用契約 修正 / 別 業務区分 試験 追加 受験 / 在留資格変更（業種跨ぎ）。
3. **2 号 移行 試験 未受験 + 5 年 接近** → 早期 受験 計画 + 受入機関 と 連携。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
