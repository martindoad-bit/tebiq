# 公的義務 滞納清算 / 年金・健保・介護保険

## 这事跟我有关吗？

如果你在准备永住申请，或想知道"年金没交满会不会影响永住"，这篇适合你。2026/2/24 永住法改正后，**直近 2 年 公的義務 履行状況** 由 出入国在留管理庁 直接审查 —— 即便 住民税 / 所得税 全部完納，只要 年金 / 健保 / 介護保険 任一项滞納，也会被认定 公的義務 履行欠缺。

## 这是什么

公的義務 履行 在永住审查里覆盖三大类：

1. **年金**：会社員 → 厚生年金（給与天引）/ 自営業 + 経営・管理 1 人法人 + 無職 → 国民年金（市役所 経由 年金事務所）。免除 / 猶予 制度 利用期间不算滞納，但需事前申请受理。
2. **健康保険**：会社加入者 → 健康保険（給与天引）/ 退職後 / 自営業 / 経営・管理 → 国民健康保険（市役所）。
3. **介護保険**：40 岁以上全员加入。会社員 → 健保 給与天引；国保 加入者 → 国保 保険料 合算徴収。

2026/2/24 改正后，年金 / 健保 / 介護保険 滞納 与 税滞納 同等扱い，且查证范围 直近 2 年。会社員 給与天引 一般不会有问题，但以下场景容易出现空白：

- 跳槽时 退職 → 入職 间隔 1 个月以上未办理 国保 / 国民年金 加入
- 経営・管理 法人立ち上げ初期 役員報酬 未确定 → 厚生年金 加入手続迟延
- 一时帰国 期间未办 转出 → 国保 / 国民年金 仍按在留中扱い 累積滞納
- 家族滞在 → 就労ビザ 切替时配偶第三号被保険者 手続漏れ

## 你应该做什么

1. **取得 年金履歴**：[ねんきんネット](https://www.nenkin.go.jp/n_net/) 注册 → 月別納付状況 全期間 PDF 出力。空白月找出。
2. **取得 健保 缴纳记录**：会社員 → 経理 / 健保組合 调取 直近 2 年 標準報酬月額 + 保険料納付通知。国保 加入期間 → 市役所 「保険料納付確認書」（手数料 300 円前後）。
3. **取得 介護保険 履歴**：40 岁以上者，市役所 介護保険課 「保険料額決定通知書」+ 「納付確認書」。給与天引者 通常合并入 健保。
4. **空白发现时处理**：
   - 国民年金 → 過去 2 年分 追納可能（年金事務所 窓口）；超过 2 年原则不可追納 但可以申请「免除 / 猶予 遡及承認」。
   - 国保 → 滞納分 一括 / 分割 完納；督促状 阶段 仍可补救。
   - 介護保険 → 同 国保 流程。
5. **完納証明 取得**：滞納分 完納後 1-2 周再次申请 納付確認書，确认状态已变更为「完納」。
6. **归档**：直近 2 年 月別 納付記録 + 完納証明 + 領収書（如有現金納付）。

## 不做的后果

- 年金 滞納 1 期（1 个月）→ 受理後 補正通知；補正期内 完納 + 説明書 可继续审查。
- 年金 滞納 累積 6 个月以上 → 多数 case 不许可。即便永住法改正前已完納，2 年内有滞納履歴 仍消極評価。
- 国保 滞納 → 督促状 → 短期保険証 / 資格証明書 切替 → 差押。任一阶段都是 公的義務 不履行 強い証拠。
- 介護保険 滞納（40 岁以上）→ 与年金 同等扱い，2026 改正后 明確に審査対象 化。
- 「給与天引 だから 大丈夫」误解 → 跳槽空白期 / 役員報酬 0 円期间 实際は 国保・国民年金 加入義務 発生 中，未手続 = 滞納累積。

## 什么时候你应该找专家

**简单 case 你自己能办**：
- 5 年同一公司 + 給与天引 全部完納 → ねんきんネット + 健保組合 取证即可
- 国民年金 期間も 自身で 全納 + 領収書あり → 自行整理

**复杂 case，建议咨询行政書士**：
- 過去に 6 个月以上の 滞納 + 完納済み → 説明書 必要
- 複数回の 跳槽空白期 + 国保 / 国民年金 手続漏れ
- 一時帰国 中の 転出届 漏れ + 滞納累積
- 経営・管理 法人 厚生年金 適用事業所 未届出 期間あり
- 免除 / 猶予 申請 が 認められず 滞納扱い になっている
