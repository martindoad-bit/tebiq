# 离婚后签证路径 / 離婚届 6 ヶ月届出 + 在留資格変更

## 这是什么

日本人配偶者等 / 永住者の配偶者等 在留資格 持有人 離婚 / 配偶死別 后，原 在留資格 的「配偶者活動」基础消失。入管法第 22-4 条规定，6 月以上不行「配偶者活動」可被取消 在留資格。同时 入管 要求 14 日内提交「配偶者に関する届出」（届出義務）。在 6 ヶ月窗口内须完成 在留資格変更，否则进入 不法滞在 / 在留資格 取消 风险区。

主要变更目的地：
- **定住者**：婚姻 ≥ 3 年 + 育児実績（未成年子の親権 + 養育実績）等综合判断。无明确法定年限，是「告示外定住」裁量。
- **技人国 / 経営・管理 / 高度専門職**：本人有独立就労実績 + 学历 / 业务经验匹配。
- **特定活動**：少数过渡期裁量。

## 为什么重要

离婚后续签时间线非常严：

- **離婚届 提出当日** → 婚姻消滅。
- **+ 14 日**：「配偶者に関する届出」期限。不届出 = 入管法 19-16 条 違反 + 罚则。
- **+ 6 ヶ月**：「配偶者活動継続不行」 = 在留資格 取消事由（22-4 条）。
- **窗口内未变更** = 在留資格 取消手続 開始 → 退去強制 / 出国命令 风险。

別居 / 協議 阶段虽未 離婚届，6 ヶ月以上不同居 同样属於「配偶者活動継続不行」。 入管 审查时通过 住民票 共住状況 + 共同生活実態 综合判断。婚姻実質的破綻 + 別居 6 ヶ月 ≈ 取消事由 同等扱い。

定住者 路径 不是自动获得，是裁量。审查重点：婚姻持续 ≥ 3 年 + 育児実績（未成年子と同居 + 親権） + 独立生計能力 + 素行善良 + 日本語能力 综合判定。无 育児 + 婚姻 < 3 年 = 定住者 难度高 → 多走 在留資格変更（就労系）路径。

## 判断标准

- **绿**：q1=无（婚姻継続）
- **黄**：q1=协议中 / 别居 6 ヶ月以上（窗口未关闭，但需提前规划）
- **红**：q1=已离婚（届出 + 変更 时间表 已启动）

## 处理建议

### 绿
档案归档 婚姻実態 证据。如出现 別居 / 協議 苗头 → 立即进入「黄」流程。

### 黄
1. 別居 / 協議中 也属 入管 关注対象。整理 婚姻持续证据（共同生活写真、共同账户、共同育児）+ 別居事由（出張、看護、調停中 等）。
2. 协议方向预判：
   - 育児 + 婚姻 ≥ 3 年 → 准备 定住者 路径材料。
   - 本人有独立工作 / 公司 → 准备 技人国 / 経営・管理 路径材料。
3. 在留期限 < 180 天 + 离婚 几乎确定 → 提前向 入管 咨询 在留資格変更 可能性。

### 红
1. **離婚届 提出后 14 日内**：本人或代理人向 出入国在留管理庁 提交「配偶者に関する届出」（書面 / 電子）。可在 [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 提交。不届出 = 罚则。
2. **+ 6 ヶ月内**：根据状况选 在留資格変更 路径：
   - **定住者**：婚姻 ≥ 3 年 + 与未成年日本人子 / 永住者子 同居 + 親権 + 養育実績 + 独立生計。提交 在留資格変更許可申請書 + 戸籍謄本（離婚记载）+ 親権者 + 養育費 决定 + 課税証明書 + 雇用契約書 + 申請理由書。
   - **技人国 等就労系**：本人独立工作 + 学历对应 + 工作内容契合。提交 在留資格変更許可申請書 + 雇用契約書 + 学位証 + 課税証明書。
3. 婚姻 < 3 年 + 无 育児 + 无独立就労 → 在留資格変更 困难。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
