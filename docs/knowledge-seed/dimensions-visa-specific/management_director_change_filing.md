# 役員変更 / 任期満了 → 商業登記 + 入管届出

## 这事跟我有关吗？
你是 経営・管理 持有人 + 法人代表者 / 役員。設立後 2 年以上経過、または 役員 任期 満了 が 近い。本篇适用。

## 这是什么
役員 任期 は 会社法 上 原則：
- 取締役：2 年以内（非公開会社 は 定款で 最長 10 年まで延長可）
- 監査役：4 年以内（非公開会社 は 定款で 最長 10 年まで延長可）

任期 満了 で **自動退任**。**重任**（再任）も「役員 変更」として **法務局 登記 必要**（会社法 第 911 条 / 商業登記法）。

**変更日 から 2 週間以内 に 法務局 で 役員変更 登記**（会社法 第 915 条）。

懈怠 = **100 万円 以下 過料**（会社法 第 976 条）。実務上 数万 - 十数万円 の 過料 通知が 裁判所から届くケース多い。

経営・管理 在留資格 持有人 が 役員 場合：
- 自身 の 役員 異動（就任 / 退任）= 「契約機関に関する届出」14 日以内（入管法 第 19-16 条）
- 公司 役員 構成 変化（他役員 加入 / 退任）= 公司 自体 の 届出義務 対象

「12 年放置 = 自動 解散 みなし」（休眠会社 整理 / 会社法 第 472 条）= 役員 変更 登記 を 12 年間 全くしない 公司 は **強制 みなし解散** 対象。

## 判断标准
- **绿**：q1=no（変更なし） or q1=yes + q2=yes（登記 OK） + q3=yes（届出 OK）
- **黄**：q1=yes 但 q2 or q3 unknown（手続 状況 不明）
- **红**：q1=yes + q2=no（登記 懈怠） or q3=no（入管届出 懈怠）

## 处理建议

### 绿
档案归档：
- 登記事項証明書（履歴事項全部証明書、直近 取得）
- 株主総会 議事録 + 取締役会 議事録（役員 選任 関連）
- 就任 承諾書 + 印鑑証明書（役員 本人）
- 入管 届出 受理票（自身 役員 場合）

下次役員 任期 満了 90 日前 リマインド。

### 黄
1. 法務局 で 履歴事項全部証明書 取得（窓口 600 円 / オンライン 500 円）。
2. 役員 履歴 + 任期 満了日 突合。**重任** 登記 含めて 漏れなく 確認。
3. [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) ログイン → 過去 届出 受理記録 確認。
4. 任期 満了 → 自動退任 → **重任** 登記 期限切れ 場合：早めに 株主総会 + 重任 登記。

### 红
1. **役員変更 登記 懈怠**：
   - 早めに 株主総会 議事録 + 重任 / 変更 登記 申請。
   - 過料 通知（数万 - 十数万円）が 裁判所 から届く可能性。納付 + 異議申立 期間 2 週間。
   - 12 年 放置 = 強制 みなし解散 対象（会社法 第 472 条）→ 解散 登記 後 3 年以内 継続 可能だが 経営・管理 続签 致命的。
2. **入管 届出（本人 役員 異動）懈怠**：
   - 14 日 過ぎ → 早めに 補正届出。
   - 「届出 義務 違反」= 続签 審査 で 素行 善良 / 公的 義務 履行 評価 マイナス。
3. **代表取締役 退任 + 後任 不在**：
   - 仮取締役 選任 / 株主総会 で 後任 選任。
   - 経営・管理 本人 が 退任 = 在留資格 該当性 喪失 → 在留資格 変更 必要（技人国 等）。
4. **任期 自動延長 と 思い込み**：
   - 定款 に 「最長 10 年」記載 あれば 10 年。記載 無し は 原則 2 年。要 定款 確認。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 履歴事項全部証明書（最新 + 変更履歴）
- 定款（任期 規定 確認用）
- 株主総会 議事録 + 取締役会 議事録
- 就任 承諾書 + 印鑑証明書（役員 本人）
- 役員 変更 登記 申請書 控
- 過料 通知書（届いた 場合）
- 入管「契約機関に関する届出」受理票
- 在留カード（役員 本人）
