# 雇用契約書 关键条款 / 職務・給与・期間・転勤・副業

## 这是什么
労働基準法 第 15 条 + 労働基準法施行規則 第 5 条 规定，雇主 は 労働者 採用時 に 以下 14 項目 を 書面（労働条件通知書 / 雇用契約書）で 明示する 義務 を 負う:

**絶対的明示事項（書面交付必須）**:
1. 労働契約 の 期間
2. 有期労働契約 の 更新 基準
3. 就業の場所・従事する業務
4. 始業 / 終業時刻、所定労働時間 を 超える労働 の 有無、休憩時間、休日、休暇、交替制勤務 の 就業時転換
5. 賃金（退職手当・臨時賃金 を 除く）の 決定・計算・支払方法、賃金 の 締切・支払 の 時期、昇給 に 関する事項
6. 退職に関する事項（解雇事由 含む）

**相対的明示事項（定めある場合）**: 退職手当 / 臨時賃金 / 食費・作業用品 負担 / 安全衛生 / 職業訓練 / 災害補償 / 表彰制裁 / 休職 関連

技人国 続签 审查中，雇用契約書 是 「職務内容 = 在留資格 該当性」「報酬 = 日本人同等性」「雇用 持続性」3 大要件 的同时举证文件。

## 为什么重要
雇用契約書 不備 / 不適 が 引き起こす 続签 リスク:

- **「職務内容」抽象化**:「総合職」「営業企画」だけ 記載 → 入管 が 自然科学 / 人文科学 / 国際業務 三領域 該当性 を 判断不能 → 不許可 又は 在留期間 1 年 短縮
- **有期契約 1 年未満**: 在留期間 が 契約期間 と 連動 する 傾向、3 年 / 5 年 取得 不可 / 続签頻度 上昇
- **転勤・出向 条項 + 業務内容 大幅変更**: 関連会社 移籍 で 自然科学 / 人文科学 / 国際業務 範围 から 外れる → 資格外活動 リスク
- **副業禁止 違反**: 副業禁止 条項あり + 副業 = 解雇事由 + 資格外活動 = 二重 リスク
- **試用期間中 退職**: 退職 → 3 ヶ月以内 転職 を 強制 + 「契約機関に関する届出」14 日

労基法 違反 雇主 = 労基署 是正勧告 + 入管 「適正な雇用慣行」評価 否定 = 雇主 カテゴリー 評価 低下。永住・帰化 申請 の 過去 5 年 雇用 履歴 全期間 を 個別 審査 する。

## 判断标准
- **绿**：q1=yes（職務内容 具体記載）+ q2=無期 or 有期 1 年以上 + q4=yes（賃金 明記）
- **黄**：q2=有期 1 年未満 / 試用期間中、q1 / q3 / q4 unknown
- **红**：q1=no（職務内容 不記載）/ q2=不明 / q4=no（賃金 不明確）

## 处理建议
### 绿
档案归档：雇用契約書 / 労働条件通知書 / 就業規則 写 / 賃金規程 / 退職金規程（あれば）/ 直近 改定履歴。続签申請 時 雇用契約書 写 + 業務内容説明書（雇主出具）+ 法人登記事項証明書 + 直近年度 決算文書 を セット で 提出。期間 3 年 / 5 年 取得 可能性 高。

### 黄
1. **有期契約 1 年未満** → 雇主 と「無期転換ルール」(労契法 18 条 / 通算 5 年超 で 無期転換権 発生) 交渉、または 1 年以上 更新 を 確認。
2. **試用期間中** → 試用期間 終了後 本採用 通知書 を 取得、続签 直前 の 試用期間中 申請 は 在留期間 1 年 リスク。
3. **副業条項 不明確** → 雇主 に 副業可否 + 届出制 / 許可制 を 確認、副業 する 場合 「資格外活動許可」(包括許可 か 個別許可) 申請。
4. **転勤・出向 条項 あり** → 出向先 / 転勤先 業務 が 自然科学 / 人文科学 / 国際業務 範囲内 か 事前確認、必要 なら 就労資格証明書 申請。

### 红
1. **職務内容 不記載 / 抽象的** → 雇主 に 労働基準法 15 条 違反 を 指摘、労働条件通知書 再交付 を 請求。「担当業務 = ◯◯ システム 開発・要件定義・テスト」レベル の 具体性 が 必要。
2. **賃金 計算方法 不明** → 月給 / 年俸 / 時給 区分、所定労働時間、残業代 単価、固定残業代（みなし残業）の 時間数・金額 内訳 を 書面 で 取得。
3. **契約期間 不明** → 雇用形態（正社員 / 契約社員 / アルバイト）+ 期間 を 明記 した 契約書 を 再締結。技人国 で アルバイト 雇用 は 通常 認定 困難。
4. **副業禁止 + 既に 副業中** → 即時 副業停止 + 雇主 に 開示 / 解雇 リスク 評估 + 資格外活動許可 整合性 確認。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
