# 経営実態 / 办公室、雇佣、営業実績 自查

## 这事跟我有关吗？
你持 経営・管理 在留資格，续签前 3-6 个月范围。不论你是 設立 1 期目还是 第 5 次更新，本篇都适用。2025/10/16 改正后，経営実態 是续签审查最先看的一组事实。

## 这是什么
出入国在留管理庁 在 在留期間更新 审查中，对 経営・管理 持有人核查三块「実態」：
- 事務所：独立物理空间、法人名义 賃貸契約書、看板 / 郵便受 / 光熱費 等存在
- 雇佣：常勤職員 数、社会保険 加入状况、雇用契約書
- 営業実績：直近 決算書、売上台帳、取引先 等

2025/10/16 改正方向：
- 資本金 3000 万円（旧 500 万円）
- 常勤職員 1 名以上（日本人 / 永住者 / 日本人配偶者等 / 永住者配偶者等 / 定住者 任一）
- 経営者本人 経営学修士 或 3 年以上 経営経験 ⚠️ 实施细则以入管告示为准
- 日本語対応能力 + 外国语対応能力

旧期更新者一般有経過措置，但續签时 有可能 被要求按新基准 適合性 説明。

## 判断标准
- **绿**：q1=yes（独立事務所） + q2=yes（常勤職員 1 名以上） + q3=yes（実売上）
- **黄**：q2=yes 但 q1 或 q3 unknown（事實层面尚未确认）
- **红**：q1=no（仅自宅、无独立空间） 或 q2=no（无常勤職員） 或 q3=no（売上 0 円、空壳）

## 处理建议

### 绿
档案归档：
- 賃貸契約書（最新 + 更新履歴）
- 事務所 写真（外观看板 + 内部办公区 + 入口）
- 光熱費 領収書（直近 3 ヶ月）
- 雇用契約書 + 給与支払 明細（直近 3 ヶ月）
- 直近 決算書 + 売上台帳

下次续签前 90 天复检。

### 黄
1. 事務所 拍照取证（外观、看板、内部、独立入口、独立 郵便受）。
2. 整理 売上台帳 + 主要取引先一覧 + 請求書 / 領収書 サンプル。
3. 雇佣证明：雇用契約書 + 社会保険 加入届 受理票 + 直近 給与振込 履歴。
4. 自宅兼用情况见 `office_lease_vs_owned` 维度。

### 红
1. **无独立事務所**：仅 バーチャルオフィス / 仅自宅一室且无物理隔离 → 续签前重整事務所，签订法人名义 賃貸契約書。
2. **常勤職員 0 人**：新规要求 1 名以上。直近 雇用 + 社会保険 加入 → 但雇佣临时性会被 入管 询问可持续性。
3. **売上 0 円 or 仅 役員報酬 流出**：事業 継続性 説明書 + 事業計画書 + 取引先 名簿 准备。修正申告 / 翌期 売上 立て直し 必要。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 賃貸借契約書（法人名義 / 个人 → 法人 転貸 場合 別途証明）
- 事務所 写真（外观 + 看板 + 内部 + 独立入口 / 郵便受）
- 光熱費 領収書（電気 / ガス / 水道 / 通信 直近 3 ヶ月）
- 雇用契約書 + 社会保険 加入届 受理票
- 給与振込 履歴 / 源泉徴収簿
- 直近 2 期 決算書 + 法人税申告書 控
- 売上台帳 + 主要 請求書 / 領収書
- 取引先 一覧
- 事業計画書（売上 0 円 or 赤字 場合）
