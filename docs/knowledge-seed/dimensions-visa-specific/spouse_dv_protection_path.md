# DV 受害者保护路径 / DV防止法 + 入管特例措施

## 这是什么

「DV」=「配偶者からの暴力」（Domestic Violence）。在日本由「配偶者からの暴力の防止及び被害者の保護等に関する法律」（DV防止法）规定。身体的暴力 + 精神的暴力 + 経済的暴力 + 性的暴力 均属対象。法律対象 包括 法律婚 + 事実婚 + 离婚后 同居経験 者。

入管 对 DV 受害者有特例運用：在 配偶者ビザ 持有期间因 DV 而 別居 / 離婚 时，「配偶者活動継続不行 6 月」（入管法 22-4 条）的取消事由 在 DV 認定 ケース 中一般不適用，并可申请 在留資格変更（多走 定住者 路径）。

不是「自动豁免」。需要 DV 認定 的客观証拠（保護命令 / 警察 110 番通報記録 / 配偶者暴力相談支援センター 相談記録 / 医師 診断書）+ 整体审查。

## 为什么重要

DV ケース 的特别考虑：

- **物理安全 优先**：在留資格 议题 在 身体安全 之后。配偶者暴力相談支援センター（全国 47 都道府県设置）+ 警察 110 + 一時保護施設（シェルター）= 安全网。
- **保護命令**：弁護士 / 本人 可向 地方裁判所 申请 接近禁止命令 / 退去命令（DV防止法 第 10 条）。决定書 是 入管 / 离婚 / 親権 审查 的强证据。
- **入管特例**：DV 認定 + 別居 6 ヶ月以上 时，「配偶者活動継続不行」不直接構成 在留資格 取消事由。可申请 在留資格変更 → 定住者（DV を 理由 とする 告示外定住）等路径。
- **届出義務**：別居 / 離婚 后 14 日内「配偶者に関する届出」仍需办，以避免 19-16 条 罚则。但 DV ケース 可在 申請理由書 中说明事由。

入管 / 警察 / 弁護士 / 行政書士 / DV 相談所 是 5 个独立窗口。各窗口的角色：

- **配偶者暴力相談支援センター**：相談 + 情报 + 一時保護 + 安全计划 + 関係機関連携。全国共通 #8008（はれれば）。
- **警察 110**：紧急 介入 + 加害者 排除。
- **弁護士**：保護命令 申请 + 离婚 + 親権 + 損害賠償。法テラス 可低费 / 免費 援助。
- **行政書士**：在留資格変更 申请 + 入管 対応。
- **入管**：在留資格変更 受理 + DV 特例 適用 判断。

## 判断标准

- **绿**：q1=no（无 DV）
- **黄**：q1=yes + q4=yes（身体安全） + q2=yes（已对接相談 / 警察 / 弁護士 等支援网络）
- **红**：q1=yes + q4=no（身体危险） OR q2 不是 yes（未对接支援网络）

## 处理建议

### 绿
无需行动。

### 黄
1. 整理 DV 認定 客观証拠：警察 110 番通報記録、配偶者暴力相談支援センター 相談記録、医師 診断書（如有）、保護命令 決定書（如有）、メール / LINE 履历（精神的・経済的 暴力 証拠）。
2. 在留資格変更（定住者 / 就労系）申请准备：
   - 戸籍謄本 / 婚姻関係 説明資料
   - 課税証明書 / 雇用契約書（独立生計能力）
   - 子の親権 書類（如有）
   - 申請理由書（DV 経緯 + 別居 / 離婚 状況 + 今後生計）
3. 弁護士 + 行政書士 协作 → 离婚 + 親権 + 在留資格 三轨同步。

### 红
1. **身体危险**：先打 110（警察 紧急通报）/ #8008（配偶者暴力相談支援センター 全国共通）/ 一時保護施設（シェルター）。其他事务在身体安全之后。
2. 連絡支援机构（不是劝说，是制度路径）：
   - **配偶者暴力相談支援センター**：[内閣府 男女共同参画局 配偶者からの暴力被害者 支援](https://www.gender.go.jp/policy/no_violence/)。全国共通 #8008。
   - **警察相談 #9110**：非紧急 暴力相談。
   - **法テラス**：弁護士 援助。
3. 入管 対応：
   - 14 日内「配偶者に関する届出」（DV 事由 説明）。
   - 6 ヶ月内 在留資格変更 申请（定住者 / 就労系）。 DV 認定 が ある 場合，入管法 22-4 条 取消事由 一般不適用。
4. 弁護士 / DV 相談所 / 行政書士 三方协作。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
