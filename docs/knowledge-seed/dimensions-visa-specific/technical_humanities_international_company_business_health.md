# 公司经营状态 / 决算赤字 / 解雇风险

## 这是什么
出入国在留管理庁「在留期間更新許可申請」审查中，雇用主 的「事業 持続性 + 安定性」是 技人国 続签的審査要素。雇主按年度源泉徴収税額 + 法人形态 划分为 4 个 カテゴリー：

- **カテゴリー1**：日本 証券取引所 上場企業 / 保険業相互会社 / 国・地方公共団体 / 独立行政法人 / 公益財団・社団法人 等
- **カテゴリー2**：前年分 給与所得・退職所得等 の 所得税徴収高計算書（納期特例分は除く）の 源泉徴収税額 1,500 万円 以上 的 団体・個人
- **カテゴリー3**：前年分 法定調書合計表 提出ありで カテゴリー2 不該当
- **カテゴリー4**：1〜3 不該当（新設 / 小規模 / 源泉徴収無等）

カテゴリー 数字越小，提出书类越简化；カテゴリー 4 需提交 雇主 的 法人登記事項証明書 + 直近年度 決算文書 + 事業内容説明書 等，决算赤字 直接进入审查視野。

## 为什么重要
雇主 経営 不振 → 「雇用 継続可能性」否定 → 即便申请人 個人 资格全合，仍可能 不許可 或 在留期間 1 年（非 3 年 / 5 年）。具体审查信号：

- 直近 2 期連続 経常損失 + 債務超過 = 高消极评价
- 解雇予告（労基法 20 条 30 日前通知）/ 整理解雇 4 要件 适用 = 雇用 終了 即将発生 → 续签 基础消失
- 民事再生・会社更生・破産手続開始 = 雇主法人格 消滅 / 縮小 → 雇用 維持 不可能
- 大規模リストラ 报道 / 厚労省 大量雇用変動届 提出 = 同行业整体審查 警戒度 上升

技人国 申请人若在续签直前換工作，新雇主 也按上述 カテゴリー 标准重审。永住申請 时，「収入 安定性」连续 3〜5 年 也会回溯审查所属公司经营状况。

## 判断标准
- **绿**：q1=no（决算 黒字）+ q2=カテゴリー 1〜3 + q3=no（無解雇信号）
- **黄**：q1=unknown 或 q2=カテゴリー 4 / 不明（雇主小規模・新設・决算情况未确认）
- **红**：q1=yes（决算 赤字）或 q3=yes（解雇 / 倒産信号 已发生）

## 处理建议
### 绿
档案归档 雇主 法定調書合計表（写）+ 源泉徴収税額証明 + 直近年度 決算文書（黒字）+ 雇用契約書 + 給与振込履歴 12 ヶ月分。续签前 90 天复检即可。カテゴリー1 上場企業 通常仅需归档「上場 確認資料」（有価証券報告書 写 等）。

### 黄
1. カテゴリー 4 新設公司 → 请雇主出具：法人登記事項証明書 / 履歴事項全部証明書 / 定款 写 / 直近年度 決算文書 / 事業内容説明書 / 損益計算書 / 貸借対照表。
2. 决算情况 unknown → 通过 国税庁「法人番号公表サイト」(https://www.houjin-bangou.nta.go.jp/) 查 法人実在性 + 雇主 IR 資料 / 決算公告 等。
3. 雇主 拒绝提供 決算文書 → 申请人侧准备 給与振込履歴 / 源泉徴収票 / 厚生年金 加入記録（年金事務所 取得）作为 在職証拠 補強。

### 红
1. 决算 連続 赤字 + 債務超過 → 评估 雇用 継続 可能性，提前规划 転職 + 就労資格証明書 申请（手数料 1,200 円）。
2. 解雇予告 已收到 → 30 日 内确认是否争议解雇 / 整理解雇 4 要件 充足；雇用保険 受給資格 確認；3 ヶ月 内 転職 不到 → 资格外活動 风险，评估 在留資格変更 或 帰国 路线。
3. 雇主 民事再生 / 破産 開始 → 立即评估 転職 时间窗，「契約機関に関する届出」14 日 内提交（離脱事由）。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 雇主 法人登記事項証明書 / 履歴事項全部証明書
- 直近 1〜3 年 決算文書（損益計算書 / 貸借対照表）
- 法定調書合計表（写）
- 源泉徴収税額証明 / 納期特例承認書（写）
