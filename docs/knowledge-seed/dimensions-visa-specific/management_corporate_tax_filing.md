# 法人税務申告 / 青色 vs 白色、決算期、法人税申告書

## 这事跟我有关吗？
你是 経営・管理 持有人 + 法人代表者。続签前 3-6 个月窗口内，本篇适用。法人税務申告 是续签材料中「事業の継続性 + 安定性」最直接的客观证据。

## 这是什么
法人税申告 在 決算期末 翌日起 2 ヶ月以内 须向 所轄税務署 提出（法人税法 第 74 条）。主要書類：
- 法人税申告書（別表一、四、五、十一 等）
- 決算書（貸借対照表 + 損益計算書 + 株主資本等変動計算書）
- 勘定科目内訳明細書
- 法人事業概況説明書

**青色申告**：事前に「青色申告承認申請書」提出 + 帳簿（複式簿記）+ 7 年保存。欠損金 10 年繰越控除、各種特別償却 等優遇。
**白色申告**：簡易帳簿、優遇なし。設立 1 期目で青色申請忘れ常見。

経営・管理 続签时，入管 通过 直近 2 期 法人税申告書 控 + 納税証明書（その 1、その 2、その 3）确认：
- 法人税 / 消費税 / 法人住民税 / 事業税 完納状况
- 売上規模 + 当期純利益 + 役員報酬 妥当性
- 欠損 連続 期数（赤字 3 期連続 = 重点審査対象）

## 判断标准
- **绿**：q1=yes（直近申告期内提交） + q3=no（无 申告漏れ / 期限後）
- **黄**：q1=unknown 或 q2=未確認（青色 / 白色 不明）
- **红**：q1=no（期限後 or 未提出） 或 q3=yes（直近 3 期内 申告漏れ / 修正申告）

## 处理建议

### 绿
档案归档：
- 直近 2 期 法人税申告書 控（税務署 受付印 or e-Tax 受信通知）
- 直近 2 期 決算書
- 納税証明書（その 1 = 納税額 / その 2 = 所得金額 / その 3 = 未納無し）
- 青色申告 承認 通知書（or 控）

下次续签前 90 天复检。

### 黄
1. e-Tax 登录 → 「申告のお知らせ」/「メッセージボックス」查直近 申告 受信通知。
2. 設立時 提出済み 書類 確認：
   - 法人設立届出書（税務署 / 都道府県 / 市町村 各 1 通）
   - 青色申告承認申請書（原則 設立日から 3 ヶ月以内）
   - 給与支払事務所等の開設届出書
   - 源泉所得税 納期の特例 承認申請書
3. 青色 / 白色 不明 → 顧問税理士 or 税務署 で 確認。

### 红
1. **期限後申告**：早めに 期限後申告書 提出。**無申告加算税** 課税対象（原則 15 %、自主申告で 5 %）+ **延滞税**（年 7.3 % or 14.6 %、期間で異なる）⚠️ 具体税率以国税庁公式为准。
2. **申告漏れ / 修正申告**：修正申告書 + 完納証明 准备。**過少申告加算税** 10–15 %、**重加算税** 35–40 %。
3. **2 期連続 期限後 申告**：青色申告 承認 取消（法人税法 第 127 条）→ 白色 強制移行 + 入管側 信用度 大幅低下。
4. **3 期 連続 赤字**：事業 継続性 説明書 + 翌期 黒字化 計画 必要。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
