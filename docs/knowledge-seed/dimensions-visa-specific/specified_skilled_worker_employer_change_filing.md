# 受入機関 変更 / 14 日 届出 + 登録支援機関

## 这事跟我有关吗？
你持 特定技能 1 号 或 2 号、最近换了 雇用主、或正在 退職 + 入職 之间。本篇是给「離職 ~ 入職 + 14 日 届出 期限」之间这段窗口的人看的。届出 は 受入機関 と 在留者 双方の 義務。

## 这是什么
入管法 第 19-16 条 + 特定技能 制度 の 規定：

**受入機関 = 直接 雇用 する 機関**。一定数 以上 の 1 号 特定技能外国人 を 受入 する 場合、自社 で 1 号 支援計画 を 履行 する か、登録支援機関 へ 全部 委託 する 必要。

**登録支援機関 = 出入国在留管理庁 に 登録 された、支援計画 履行 を 受託 する 第三者**。中小 受入機関 は 通常 委託。

**届出 義務（14 日 以内）**
- 在留者 側：「所属機関に関する届出（特定技能 用）」入管 へ
- 受入機関 側：「契約機関に関する届出」入管 へ + 業種別 所管省庁 へ 別途 届出 ある 業種 あり
- 退職 / 転職 / 解雇 / 機関 移転 / 名称変更 / 消滅 すべて 対象

特定技能 1 号 は 同 業種 内 転籍 のみ 原則 可。離職 期間 が 3 ヶ月 超 = 在留資格 取消 リスク（入管法 第 22-4 条）。

新 受入機関 側の 要件：
- 特定技能所属機関 登録（業種別）
- 受入 体制（雇用契約 + 支援計画 + 報酬 同 等以上 + 一時帰国 配慮 等）
- 過去 5 年 違反歴 なし

支援計画 必須 項目（10 項目）：事前ガイダンス / 出迎え / 住居 確保 / 生活オリエンテーション / 公的手続 同行 / 日本語学習 機会 / 相談 苦情 対応 / 日本人 との 交流促進 / 転職支援 / 定期面談 + 報告。

## 判断标准
- **绿**：q1=no（無 変更） 或 q1=yes + q2=yes + q3=yes（届出 + 登録機関 + 委託 全 充足）
- **黄**：q1=yes 但 q2 或 q3 unknown
- **红**：q1=yes 且 q2=no（14 日 届出 未提出） 或 q3=no（非登録機関 / 委託 不備）

## 处理建议

### 绿
档案归档：
- 「所属機関に関する届出」受理票（入管 オンライン or 紙）
- 雇用契約書（新 受入機関、業種別 様式）
- 支援計画書（新版）
- 登録支援機関 委託契約書（委託 場合）
- 新 受入機関 の「特定技能所属機関」登録 証明

下次 続签前 60 日 复检。

### 黄
1. [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) で 届出 受理 履歴 確認。
2. 出入国在留管理庁 公開 の「特定技能所属機関」一覧 + 登録支援機関 一覧 で 新 機関 を 確認。
3. 受入機関 担当者 + 登録支援機関 へ 委託契約 + 支援計画 状況 を ヒアリング。
4. 離職 期間 が 1 ヶ月 超 → 求職活動 状況 を 入管 へ 任意 説明。

### 红
1. **14 日 届出 未提出** → 即日 補完。電子届出 が 最速。状況 説明書 添付。
2. **3 ヶ月 超 離職 + 同業種 転職 不能** → 在留資格 取消 手続 リスク。試験 受け直し + 在留資格変更 検討。
3. **新 受入機関 が 非登録**（特定技能所属機関 でない） → 即時 雇用 停止。合規 機関 へ 転職。
4. **登録支援機関 委託 不備**（中小 受入機関 で 自社支援 体制 不足） → 委託契約 締結 + 支援計画 再作成。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
