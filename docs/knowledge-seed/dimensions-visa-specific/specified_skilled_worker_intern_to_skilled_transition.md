# 技能実習 → 特定技能 移行 / 在籍証明 + 試験免除

## 这事跟我有关吗？
你处在 技能実習 2 号 / 3 号 修了 前後（一般在 第 3 年 / 第 5 年），考虑 在留資格変更 到 特定技能 1 号。这篇是给 修了见込 期 ~ 変更申請 提出 之间这一段窗口的人看的。

## 这是什么
入管法上「技能実習 2 号 良好修了」者移行至 特定技能 1 号 时：
- 同 業種・同 職種・同 作業 範囲内 → 技能評価試験 + 日本語試験（JFT-Basic / JLPT N4 以上）双重 免除
- 異 業種 → 試験 必須（技能 + 日本語）
- 同 業種 但 職種・作業 不一致 → 試験 部分免除 / 部分 必須（業種別 ガイドライン による）

「良好修了」要件：技能実習 計画 通り 2 号 / 3 号 を 修了、且つ 出勤率・態度・技能習得 状況 が 監理団体 / 実習実施者 評価 で 一定基準 を 満たす。

申請時 提出する 標準的な 在籍系 証拠：
- 技能実習 2 号 / 3 号 修了 証明書（監理団体 発行）
- 在籍証明書（実習実施者 発行、技能実習 期間 + 出勤状況 記載）
- 技能検定 3 級 合格証 / 専門級 修了証（保有時）
- 評価調書（OTIT 経由 或 実習実施者 作成）

2027 施行 予定 的「育成就労」制度 が 技能実習 を 段階的に 置き換える 場合、移行路径も 改正される。⚠️ 詳細 以 厚労省 / 入管 公式 通知 为准。

## 判断标准
- **绿**：q1=yes（修了者 / 修了见込） + q2=yes（業種・職種 対応） + q3=yes（証明書 + 在籍証明 取得済）
- **黄**：q1=yes 但 q2 或 q3 unknown（対応関係 / 証拠書類 未確認）
- **红**：q1=yes 且 q2=no（業種・職種 不対応） 或 q3=no（証明書 取得不能）

## 处理建议

### 绿
档案归档：
- 技能実習 2 号 / 3 号 修了 証明書
- 在籍証明書（期間 + 業種・職種 + 出勤率 記載）
- 技能検定 3 級 合格証 / 専門級 修了証
- 雇用契約書（特定技能 用、移行先 受入機関 名義）
- 1 号 特定技能外国人 支援 計画書（受入機関 / 登録支援機関 作成）

変更申請（在留資格変更許可申請）准备 90 日前 開始。

### 黄
1. 出入国在留管理庁 + OTIT 公式 业种・職種 対応表 で 移行可否 を 確認。
2. 旧 監理団体 / 実習実施者 へ 修了 証明書 / 在籍証明書 の 再発行 を 依頼。
3. 移行先 受入機関 と 雇用契約 + 支援計画 を 詰める（登録支援機関 委託 場合 別途契約）。
4. 評価調書 不備 場合 → 監理団体 経由 で 補完。

### 红
1. **業種・職種 不対応**（例：技能実習 で 縫製、移行先 が 介護）→ 技能評価試験 + 日本語試験 を 受験。試験合格後 改めて 変更申請。
2. **在籍証明 不取得**（旧 監理団体 / 実習実施者 が 倒産・連絡不能）→ OTIT へ 在籍 証明 の 代替発行 相談。
3. **「良好修了」要件 不満たし**（出勤率 不足 / 計画 中断）→ 通常の 技能評価試験 ルート に 切替。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
