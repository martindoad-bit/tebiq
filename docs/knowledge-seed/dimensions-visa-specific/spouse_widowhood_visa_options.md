# 配偶死亡后签证选项 / 死別 + 在留資格変更

## 这是什么

日本人配偶者等 / 永住者の配偶者等 在留資格 持有人，在 配偶 死亡后，「配偶者活動」基础消失。处置时间表与离婚等同：14 日内届出 +  6 ヶ月内 在留資格変更，否则 入管法 22-4 条 在留資格 取消事由 適用。

死別 比 離婚 多一个事实：婚姻関係 不是「破綻」而是「終了」。这在 定住者 申请审查中是有利情节——「日本人 / 永住者 と 婚姻関係 が 配偶者の死亡 によって 終了」是 告示外定住者 認定 的常见考虑情形之一。审查仍需婚姻持续期 + 在日期间 + 育児実績 + 独立生計能力 综合判断，不是自动获批。

## 为什么重要

时间表与 離婚 一致：

- **配偶死亡当日** → 婚姻 終了。
- **+ 14 日**：「配偶者に関する届出」期限（入管法 19-16 条）。
- **+ 6 ヶ月**：「配偶者活動継続不行」 → 在留資格 取消事由（22-4 条）。

死亡 場合 的特别考虑：
- 戸籍謄本 死亡記載 + 死亡診断書 / 死体検案書 是核心証拠。
- 遺族年金 受給資格 / 相続 関係 与 在留資格変更 审查互相印证「家族関係 + 生計 自立」。
- 未成年日本人子 + 親権 + 養育実績 = 定住者 認定 的强力情节。

定住者 申请审查 vs. 离婚情形：死別 一般审查门槛略低（无「破綻 帰責性」议题），但仍非自动 → 综合判断「婚姻実態 真実性 + 在日 期間 + 育児 / 看護 实绩 + 独立生計 + 素行善良」。

## 判断标准

- **绿**：q1=no（配偶健在）
- **黄**：q1=yes + q2=yes（已届出，准备 在留資格変更）
- **红**：q1=yes + q2 不是 yes（届出未做 / 不确定 → 罚则风险 + 取消窗口正在缩小）

## 处理建议

### 绿
档案归档 婚姻実態 证据。无需进一步行动。

### 黄
1. 已届出 → 評価 在留資格変更 路径：
   - **定住者**：婚姻 ≥ 3 年 OR 与未成年日本人子 / 永住者子 同居 + 親権 + 養育実績。死別 是有利情节，但 婚姻 < 3 年 + 无子 仍是高门槛。
   - **就労系（技人国 / 経営・管理 等）**：本人有独立工作 + 学历 / 业务匹配 + 雇用契約。
   - **特定活動**：少数过渡期裁量。
2. 准备 戸籍謄本（死亡記載）+ 死亡診断書 + 婚姻持续期间的同居 / 共同生活 证据。
3. 在留期限 < 180 天 → 立即启动 在留資格変更 申请。

### 红
1. **未届出**：今天补提交「配偶者に関する届出」。書面 / 電子（[在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/)）均可。届出 14 日 已过 = 罚则対象 但不补办风险更大。
2. **6 ヶ月窗口**：根据 q3 选路径：
   - 婚姻 ≥ 3 年 + 有未成年子 + 親権 → 定住者 路径。
   - 婚姻 ≥ 3 年 + 无子 → 定住者 + 就労系 双轨准备。
   - 婚姻 < 3 年 + 无子 → 就労系 路径为主，定住者 难度高。
3. 相続 / 遺族年金 等家族法事务与签证审查并行 → 整理 戸籍謄本 + 死亡診断書 + 相続関係説明図 + 不動産登記。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
