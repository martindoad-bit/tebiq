# 婚姻真实性 / 同居実態 + 入管面谈

## 这是什么

入管 审查 日本人配偶者等 / 永住者の配偶者等 在留資格 时，核心要件是「真实婚姻 + 共同生活実態」。审查不仅看 戸籍謄本 与 婚姻届 受理証明書，更看双方是否实际同居、共同支付生活费、有共同社交圈。入管法第 22-4 条规定：「日本人の配偶者等」「永住者の配偶者等」の在留資格 を持つ者が、配偶者の身分を有する者としての活動を継続して 6 月以上行わないで在留している場合，可取消 在留資格。

「配偶者活動」= 同居 + 共同生活。仅有 婚姻届 不构成「配偶者活動」。

## 为什么重要

入管 在 在留期間更新 / 在留資格変更 时通过 4 个层次核查 婚姻実態：

1. **書面层**：双方 住民票 是否同一住所、課税証明書 上 配偶者控除 有无申告、健康保険 扶養 関係。
2. **写真层**：结婚式 / 披露宴 / 共同旅行 / 家庭日常 / 双方家族集合照。
3. **送金 / 共同账户层**：水道光熱費 / 家賃 / クレジット支付分担、夫妻共同账户、相互送金记录。
4. **面谈层**：入管 可召唤双方分别面谈，问相识经过、初次见面地点、双方家族姓名、平时家中语言、共同朋友等。两人答案如出现重大矛盾 → 偽装結婚 怀疑 → 不許可 / 在留資格 取消。

国际婚姻（日中、日韩等）+ 年龄差大 + 相识时间短 + 介绍人介绍 + 双方语言不通 → 入管 审查更严。

## 判断标准

- **绿**：q1=yes（同居）+ q2=no（无 6 ヶ月以上分居）+ q3=yes（有同居证据 3 件以上）+ q4=no（无 入管 面谈历史）
- **黄**：q1=yes 但 q2 / q3 unknown（同居证据不全 / 分居期间不确定）
- **红**：q1=no（不同居）/ q2=yes（连续 6 ヶ月以上分居）/ q4=yes（有 入管 面谈历史）

## 处理建议

### 绿
档案归档 4 类同居证据：① 双方 住民票（同住所） ② 共同照片 6 张以上（含家族成员） ③ 共同账户 / 共同光熱費 / 家賃 領収書 ④ 結婚式 / 披露宴 写真 + 招待客リスト。下次续签前 90 天复检。

### 黄
1. 同居证据 3 件不足 → 1 个月内补：双方共同 旅行 e ticket / 双方家族 LINE 履历 截图 / 共同就医记录 / 育儿记录（如有子）。
2. 出张 / 长期回国 6 ヶ月以上 → 准备 出張証明書（公司发行）/ 帰国期间 機票登机証 + 説明書，事由要可证（亲属看护、产育、单位派遣等）。
3. 续签申请书 申請理由書 中诚实说明分居事由 + 同居恢复时间。

### 红
1. 现状不同居 / 分居 6 ヶ月以上 → 入管法 22-4 条 在留資格 取消高风险。需在续签前先恢复同居 + 准备完整 申請理由書。
2. 已收到 入管 面谈通知 / 補正通知 → 双方分头准备答问要点（相识経過、結婚決意、現在生活、家族関係）+ 一致性核对。
3. 婚姻関係 已实质破裂但未离婚 → 进入 spouse_divorce_visa_path 路径，6 ヶ月内 在留資格変更（定住者 / 技人国 等）。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
