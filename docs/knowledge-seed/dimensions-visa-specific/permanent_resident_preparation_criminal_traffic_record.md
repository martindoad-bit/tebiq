# 犯罪歴 / 交通違反 / 反則金 vs 罰金 区别

## 这事跟我有关吗？

如果你在准备永住申请、过去 5 年内开过罚单、或曾收到 略式命令 / 罰金 通知，这篇适合你。永住審查的「素行善良」要件中，**交通違反は最も誤解が多い項目**。轻视 反則金 件数、混淆 反則金 与 罰金 区别，是不许可常见原因。

## 这是什么

日本的 交通法規違反 处理体系分两层，永住审查中扱い 不同：

**反則金（青切符 / 赤切符 のうち反則金扱い）**
- 軽微な違反（駐車違反 / 速度 30km/h 未満超過 / 一時不停止 等）
- 行政処分性格 → 違反点数 + 反則金（行政罰）
- 不要 裁判，反則金 完納で 終了
- 永住審査での扱い：単独件数 重要。一般 5 年以内 5 件以上で 不許可可能。3-4 件は 黄信号。

**罰金（赤切符 / 略式命令）**
- 重大な違反（飲酒運転 / 30km/h 以上超過 / 無免許 / 救護義務違反 等）→ 起訴 → 略式命令 / 公判 → 罰金
- 刑事処分性格 → 前科 として 残る
- 永住審査での扱い：通常 5-10 年以内 不許可。仮に 完納済み でも 素行善良要件 否定的評価。

両者の決定的な違い：**「反則金」は前科にならない、「罰金」は前科になる**。両者を混同して申告書に「無し」と記入すると、申告内容不実 + 素行善良 否定 の二重否定要素 になる。

刑事 罰金以上 の処分（傷害 / 窃盗 / 業務上過失致死傷 等の交通以外も含む）は 5-10 年以内 永住申請 で 通常不許可。10 年以上経過しても 申請理由書 で 詳細申告 が 必要。

## 你应该做什么

1. **過去 5 年違反履歴 取得**：[警察庁「運転記録証明書」](https://www.jsdc.or.jp/kirokushomei/tabid/107/Default.aspx)（自動車安全運転センター 経由）申請。670 円。過去 5 年 全違反履歴 + 累積点数 が 出る。
2. **罰金 経験者 → 判決書 / 略式命令書 取得**：管轄裁判所 で 写し申請。当時の弁護人 経由でも可。
3. **件数判定**：
   - 0-2 件 反則金 → 緑信号、申請書「過去 5 年以内」欄正直に記載すれば OK
   - 3-4 件 → 黄信号、最後の違反 から 1-2 年待って 申請が無難
   - 5 件以上 → 赤信号、5 年待って 0 件 状態を作るのが基本
4. **赤切符（罰金扱い）取得した場合**：罰金完納証 + 不起訴処分告知書（不起訴の場合）+ 判決書（罰金の場合）必須。略式命令 でも 罰金 = 前科。
5. **申請書「過去の犯罪歴」欄**：反則金 1 件でも 正直に記載（多数の不許可 例の半数は虚偽記載 由来）。罰金以上 は 全件 + 経緯 + 反省 を 申請理由書 に 書く。
6. **歸化と永住の違い意識**：歸化（国籍法）は 罰金以上 + 反則金 累積 ともに さらに 厳しく審査。永住で 黄信号 なら 歸化は 赤信号扱い。

## 不做的后果

- 反則金 5 件以上（5 年以内）+ 申告 → 多数 case 不許可。再申請まで 0 件期間 5 年積み直し。
- 反則金 1-2 件 でも 申告書「無し」記載 → 申告内容不実 → 素行善良 + 信用 双方否定 → 不許可。
- 飲酒運転 1 回 → 罰金以上 + 違反点数 一発取消相当。永住は 5-10 年事実上不可能。再申請には 反省 + 経過 + 改善 を 多面的に 立証 必要。
- 罰金 払い済み だから「終わったこと」と申告漏れ → 法務省内データベース で 即発覚 → 不許可。
- 略式命令 を「罰金 じゃない」と誤解 → 略式命令 = 罰金 の一種 = 前科。

## 什么时候你应该找专家

**简单 case 你自己能办**：
- 5 年内 反則金 0 件 + 罰金歴 無 → 申請書 正直記載 + 自分で永住申請
- 反則金 1-2 件 + 軽微 + 違反点数残 0 → 同上、補足説明書 不要

**复杂 case，建议咨询行政書士**：
- 反則金 3 件以上 + 申請時期 を 戦略的に 判断したい
- 過去に 罰金以上（含 略式命令）の経験
- 飲酒運転 / 危険運転致死傷 等 重大違反歴
- 不起訴処分 だが 取調履歴あり
- 5 年以前の 犯罪歴 + 経過の長さで 攻める 申請理由書 を 作りたい
