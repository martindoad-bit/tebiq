# 学历与职务相关性 / 理由書 必备项

## 这是什么
入管法 別表第一の二「技術・人文知識・国際業務」基準省令 规定，技人国 在留資格 申请人须满足以下任一条件：

1. 大学（含 短大）卒业且专攻 与 職務内容 相关
2. 日本 専門学校 卒业并取得「専門士」「高度専門士」称号且专攻 与 職務 相关
3. 自然科学 / 人文科学 領域 10 年以上 実務経験（含大学・高専・高校・専修学校での科目専攻 期间）
4. 国際業務 領域 3 年以上 実務経験（大卒者 翻译・通訳・语学指导 等職務 不要求経験）

「専攻 と 職務 の関連性」是 在留資格認定 与 在留期間更新 共同审查项。卒業証明書 / 成績証明書 / 学位記 / 履修科目一覧 是核心举证材料。

## 为什么重要
2024 年以降 出入国在留管理庁 对 関連性 审查趋严，特别针对：

- 文系大卒 申请 IT エンジニア 職务（专攻 経済学 / 経営学 但担当 開発業務）
- 理系大卒 申请 営業職 / 事務職（关联性需 理由書 桥接）
- 中国国内 専科 / 高职 学历（不被视为日本 短大 同等）担当 専門職

関連性 否定 = 在留資格 該当性 否定 = 不許可。续签时若 職務内容 已偏离当初許可 範围，即便公司未变更，也按 新規認定 标准重审。永住申請 时，過去 在留歴 中任何一次「資格外活動」嫌疑会作为 素行善良 否定要素。

## 判断标准
- **绿**：q1=yes 且 q2=本科以上 / 短大・専門士（专攻与三领域職務直接相关，学历达标）
- **黄**：q1=unknown 或（q1=no 但 q3=yes，即学历不直接相关但有 10 年 / 3 年 実務経験 可替代）
- **红**：q1=no 且 q3=no（学历不相关 + 経験不足），或 q2=高中 / 中专（学历不达技人国 基準）

## 处理建议
### 绿
档案归档 卒業証明書 + 学位記 + 成績証明書（履修科目）+ 当前 雇用契約書 + 業務内容説明書。续签时 履歴書 与 理由書 中明示「専攻 → 担当業務」对应关系即可。専門学校 卒业者必须同时归档「専門士・高度専門士 称号証明書」。

### 黄
1. 専攻 与 職務 间接相关 → 起草 理由書，论证学习内容与 担当業務 的具体关联点（履修科目 → 業務スキル 对应表）。
2. 経験替代型 → 归档過去 全部 在職証明書（公印 + 期間 + 職務内容 明记），从大学・専修学校 起 持续算入「専攻に関する科目を専攻した期間」。
3. 不确定时申请 就労資格証明書（手数料 1,200 円）事前确认現職 与 在留資格 の 適合性。

### 红
1. 学历未达本科 / 短大・専門士 + 無 10 年経験 → 当前許可 可能基于过去 入管 审查放宽期，续签时按现行严格基準 重审，不許可 概率高。
2. 専攻完全不相关（如:汉语言文学 → IT 開発）+ 無 10 年経験 → 評估 高度専門職 / 経営・管理 / 特定技能 等他資格 转换可能性。
3. 中国 専科 学历主张「短大 同等」→ 不被认可，需补足経験 或转 在留資格。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 相关材料
- 卒業証明書 / 学位記（中国学历需 中国学位与研究生教育发展中心 CDGDC 认证 + 公証 + 翻訳）
- 成績証明書 / 履修科目一覧
- 専門士・高度専門士 称号証明書（仅専門学校卒业者）
- 履歴書（職歴 不间断记载）
- 在職証明書（過去全部公司，含 期間 / 職務内容 / 給与）
- 業務内容説明書（雇主出具，记 担当業務 + 使用知识 + 専攻関連性）
- 雇用契約書 / 労働条件通知書
- 申請理由書（専攻 → 職務 桥接论证）
