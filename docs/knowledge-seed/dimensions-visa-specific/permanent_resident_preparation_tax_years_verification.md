# 纳税年限要求与实操查证 / 直近 5 年 課税・納税証明書

## 这事跟我有关吗？

如果你在准备 永住申请，或在预备期内想盘点档案完整度，这篇适合你。永住审查的「公的義務履行」栏，**直近 5 年纳税完整性**是核心证据。即便每年都正常缴纳，只要文件链断掉一环，结果与「未缴」近似。

## 这是什么

永住申请须证明 直近 5 年 住民税 + 所得税 完整缴纳。证据链由两类文件组成：

- **課税証明書**（市役所 / 区役所 发行）：证明每年 住民税 課税額，按年发行，每份 300 円左右。
- **納税証明書**（住民税 → 市役所；所得税 → 税務署）：证明税款已完納（不滞纳）。

2026/2/24 永住法改正 后，公的義務履行 履歴 的审查范围明确为 直近 5 年。会社員 通常通过 給与天引（特別徴収）缴纳 住民税，但凭证仍需自行向 市役所 申请。経営・管理 / 自営業 / 高度専門職 持有人若有 確定申告 義務，所得税 納税証明書（その1：納税額 / その2：所得金額 / その3：未納無し証明）三种通常都被要求。

跨自治体搬家时，**每个居住过的 市町村 都要分别申请**——这是漏掉率最高的环节。

## 你应该做什么

1. **盘点过去 5 年居住地**：对照 在留カード 裏面 / 住民票 履歴，列出所有居住过的 市町村。
2. **逐个申请 課税証明書**：每年 1 份 × 居住自治体数。当前居住地 市役所 / 区役所 直接窗口，旧居住地通过郵送請求（手数料 + 返信用切手 + 本人確認書類 写し）。
3. **取得 納税証明書（住民税）**：与 課税証明書 同窗口申请。
4. **取得 所得税 納税証明書**：管轄 税務署 窗口、または [e-Tax 納税証明書 オンライン請求](https://www.nta.go.jp/taxes/nozei/nozei-shomei/01.htm)。其1 + その3 是永住申请最常见组合。
5. **核对税额一致性**：源泉徴収票 上的「住民税控除額」与 課税証明書 数字应一致。差额超 1,000 円时，去 市役所 询问原因（多为 ふるさと納税 / 医療費控除 处理差异）。
6. **归档**：5 年 × 自治体数的 課税証明書 + 全套 納税証明書 + 源泉徴収票 / 確定申告書控え 一并保存。

## 不做的后果

- 缺 1 年 課税証明書 → 永住申请受理时 補正 通知。补正期内补齐可继续审查；若無法取得（自治体保存期限切，通常 5-7 年）= 受理后撤回 / 不许可。
- 5 年内有 1 笔 住民税 滞納记录（即便已完納）→ 公的義務履行 不充分，多数 case 需再积累 1-2 年清白记录后重申。
- 旧住所地 課税証明書 漏掉 → 受理時是否被指摘取决于審査官；漏报后被发现 = 申告内容不実 嫌疑，影响素行善良 评价。
- 高度専門職 70 点 / 80 点 路径下，課税証明書 仍按 5 年要求；点数 不能减免 公的義務 履歴 范围。

## 什么时候你应该找专家

**简单 case 你自己能办**：
- 5 年同一市町村 + 給与天引 で完納 + 源泉徴収票 全套 → 自行去 市役所 一次性取齐
- 確定申告 自己每年提交 + 無滞納 → e-Tax 调取 納税証明書

**复杂 case，建议咨询行政書士**：
- 5 年内跨多个 都道府県 搬家 + 部分自治体保存期限临近
- 期间有過 滞納 → 督促状 → 完納 的循环
- 副業 / 海外汇款 / 不動産所得 等多源 所得 但未做 確定申告
- 経営・管理 法人 申告 与 個人 申告 之间数字不一致
