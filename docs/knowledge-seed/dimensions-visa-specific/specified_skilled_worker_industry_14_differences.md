# 業種別 試験 + 日本語 基準 差异自查

## 这事跟我有关吗？
你持 特定技能 1 号 或 2 号、所属 業種 已确定，正在续签前 3-6 个月 或 考虑跨业种。本篇适用于 業種別 微差（試験 / 日本語 / 業務範囲）需要核对的场景。

## 这是什么
特定技能 在留資格 制度的核心特点：「業種ごとに 所管省庁 + 試験運営団体 + 業務範囲 が 別々」。统一的「特定技能」框架下，业种之间存在以下差异：

**所管省庁（代表）**
- 介護：厚生労働省
- 建設・造船・舶用工業：国土交通省
- 自動車整備・宿泊・航空：国土交通省
- 飲食料品製造業・外食業・農業・漁業：農林水産省
- ビルクリーニング・素形材・産業機械・電気電子：経済産業省 + 厚労省

**技能 試験**
- 介護：「介護技能評価試験」+「介護日本語評価試験」（NRCD 運営）
- 外食業：「外食業特定技能 1 号 技能測定試験」
- 飲食料品製造業：別 試験
- 建設：建設分野 特定技能 1 号 評価試験（業務区分 別）
- 各業種ごと 試験 内容・合格基準 異なる

**日本語 試験（共通要件）**
- JFT-Basic（A2 相当） 或 JLPT N4 以上 が 原則
- 介護 のみ 加えて「介護日本語評価試験」必要
- 技能実習 2 号 良好修了 → 日本語試験 免除

**業務範囲**
- 各 業種 ごとに「対象 業務」+「関連 業務」が 業種別 ガイドライン で 定義
- 「関連 業務」は 主たる 業務 と 一体 で 行われる 範囲 のみ。範囲外 = 業務逸脱 = 続签 否認 リスク

特定技能 2 号 への 移行 時 も 業種別 試験（2 号用）が 必要。家族帯同 可否 も 1 号 / 2 号 で 異なる。

## 判断标准
- **绿**：q2=yes + q3=yes（公式案内 確認済 + 業務範囲 内）
- **黄**：q2 或 q3 unknown（情報 古い / 業務範囲 不明確）
- **红**：q3=no（業務逸脱 確定）

## 处理建议

### 绿
档案归档：
- 業種別 ガイドライン（直近版 PDF / URL + 取得日）
- 雇用契約書（業務内容 詳細）
- 試験合格証（技能 + 日本語）
- 受入機関 業種登録 証明
- 業務 日報 / 月報 サンプル（業務内容 立証用）

続签前 90 日 复检。

### 黄
1. 業種 所管省庁 + 試験 運営団体 公式 サイト で 直近 ガイドライン 確認。
2. 雇用契約書 + 実際 業務内容 を ガイドライン と 対照、関連業務 比率 が 過半 超えてないか 確認。
3. 受入機関 担当者 + 登録支援機関 と 業務範囲 認識 を 揃える。

### 红
1. **業務逸脱 確定**（例：飲食料品製造業 採用 だが 実際 は 配送 専従）→ 続签前 雇用契約 修正 + 業務 是正、或 異 業種 への 在留資格変更（試験 受け直し）。
2. **業種 廃止 / 再編**（2024 改正 で 14 → 16 業種 化、業務区分 統合 等）→ 入管 + 業種別 所管省庁 公式 通知 確認。
3. **試験 合格証 紛失** → 試験運営団体 へ 再発行 申請。
4. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
