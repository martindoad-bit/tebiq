# 中日跨国婚姻材料 / 結婚証 + 認証 + 翻訳

## 这是什么

中日跨国婚姻 在 配偶者ビザ 申请 / 续签 / 在留資格認定証明書 申请时，常见 3 类材料失误：

1. **「結婚証」（红本）≠「結婚証明書」**：中国 民政局 在 結婚登記 后发的 红色 結婚証（双方各持一本）是 結婚事実 的本人证明，但日本 入管 / 区役所 通常要求是 中国民政局 开具的「結婚証明書（夫婦関係証明書 / 婚姻状況証明書）」。两者的形式不同。
2. **未経 中国駐日大使館 / 領事館 認証**：中国发行的公文書，要被日本机关接受，原则上需经 中国駐日大使館 / 領事館 認証（领事认证）。仅原本+翻訳 不足。
3. **日本語訳 不规范**：訳者氏名 + 連絡先 + 訳者署名 + 訳文奥付 不完整，或仅自译无任何身份信息。入管 可能因「訳責任不明」补正。

正确流程：中国民政局「結婚証明書」原件取得 → 中国公証処 公証（如需）→ 中国駐日大使館 / 領事館 認証 → 日本国内 日本語訳。出生公証 / 親族関係公証 / 未婚 / 離婚歴 等中国側書類 同流程。

## 为什么重要

入管 在 在留資格認定証明書 / 在留資格変更 / 在留期間更新 审查中，对 跨国婚姻 关注 3 个维度：

- **婚姻成立 真実性**：中国 結婚登記 + 日本 婚姻届 双重登记，戸籍謄本（日本人配偶）+ 結婚証明書（中国側）+ 双方原籍証明 缺一可能补正。
- **国籍 + 親族関係**：中国側 親族関係（父母氏名、出生地、戸籍所在地）通过 公証書 + 認証 提供，与 日本側 戸籍謄本 印证。
- **形式 適格性**：書類は 原本 + 認証 + 訳 三件套。入管 不接受未認証 中国公文書 や 自译無責任表示文書。

中国 結婚登記 制度提示：

- **結婚証（红本）**：双方各 1 本，结婚登記时发。属本人持有資料。
- **結婚証明書 / 夫婦関係証明書 / 婚姻状況証明書**：中国民政局婚姻登記処 / 婚姻登記機関 后续应申请人请求发行的 公文書。日本 入管 / 区役所 婚姻届 提出 / 配偶者ビザ 申请时多以此为基础資料。
- **公証書**：中国 公証処（公証役場 相当）发行。出生公証、親族関係公証、未婚 / 離婚 公証 等。

中国駐日大使館 / 領事館 認証：在中国国内取得 結婚証明書 + 公証書 后，可在日本国内 中国駐日大使館 / 領事館 申请 認証。也可在中国国内向 中国外交部 / 外事弁公室 認証。日本入管 一般要求 駐日認証 或外交部認証。

## 判断标准

- **绿**：q1=no（双方皆日本籍 / 跨国但已完成）OR q1=yes + q2/q3/q4 全 yes（中国側材料三件套齐）
- **黄**：q1=yes + 任一 unknown（材料履歴不確）
- **红**：q1=yes + 任一 no（材料缺失或形式不適格）

## 处理建议

### 绿
档案归档：① 中国民政局 結婚証明書 原件 + 認証 + 訳 ② 中国側 公証書（親族関係 / 出生 等）+ 認証 + 訳 ③ 日本側 戸籍謄本（婚姻記載）④ 婚姻届 受理証明書。下次 在留期間更新 / 永住 申請时直接复用。

### 黄
1. 区分手中文書：是「結婚証」（红本）还是「結婚証明書」（公文書）？仅红本 → 需另申请 結婚証明書。
2. 認証 履歴 不確 → 中国駐日大使館 / 領事館 取得 認証。各地领事馆办理范围不同（例 大使館 / 大阪 / 福岡 / 名古屋 / 札幌 / 新潟）。
3. 訳 是否有 訳者氏名 + 連絡先 + 署名 + 日付？无 → 整理规范訳文，签名负责。

### 红
1. 中国民政局 結婚証明書 未取得：通过中国国内親族 / 委任手続 申请。原則 結婚登記地 民政局 受理。
2. 認証 未做：中国駐日大使館 / 領事館 申请 認証。原本提交 + 訳 提交（中国語可），1-2 周取得。
3. 訳 不规范：可由本人 / 親族 / 翻訳会社 翻訳，但必须含 訳者氏名 + 連絡先 + 署名 + 日付。重要文書 推荐 公証 訳文。
4. 中国側 親族関係公証 / 出生公証 / 未婚 / 離婚歴公証 缺：通过中国 公証処 申请 → 中国側 認証 / 駐日 認証 → 訳。
5. 形式 適格性 + 婚姻実態 同时存在问题 → 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
