# 出資金 / 資本金 来源说明

## 这事跟我有关吗？
你是 経営・管理 申請者 / 続签者，特に 設立 1 期目 or 増資 直後。2025/10/16 改正で 資本金 3000 万円 への引き上げ方向。資金 出処 説明 は 続签 + 増資 審査 の核心论点。

## 这是什么
資本金 / 出資金 来源説明 は 出入国在留管理庁 が 経営・管理 審査で重点的に確認する項目。論点：
- 資金 が **本人 の 正当 蓄積** か（海外 給与 / 海外 事業所得 / 親族 贈与 / 不動産 売却 等）
- 資金 が **借入金 / 名義借し** ではないこと
- 資金 が **法人口座 に 実際 入金** されていること
- 「形だけ 増資」（一時 借入 → 設立直後 引出し）でないこと

2025/10/16 改正 方向：
- 資本金 3000 万円 ⚠️ 既存事業者 経過措置 詳細 入管告示 待ち
- 「実態 を 伴わない 形式 出資」への 審査 強化

審査 で 求められる 主要 書類：
- 本人 銀行 通帳 写し（直近 6-12 ヶ月）
- 銀行 残高 証明書（設立直前）
- 海外送金 場合：SWIFT 控、送金 申込書、受取 銀行 入金通知
- 借入 場合：借入契約書 + 貸主 銀行 通帳 写し（資金 出処 明示）
- 親族 贈与 場合：贈与契約書 + 贈与税 申告書 + 贈与者 銀行 通帳

「資金 出処 説明書」自体 が 別紙 として 求められることが 多い。

## 判断标准
- **绿**：q1=本人自己資金 or 海外送金（本人名義） + q2=yes（連続追溯可） + q3=yes（法人口座 入金 証明）
- **黄**：q2 unknown 或 q3 unknown（資料 部分欠落）
- **红**：q1=借入金（特に 設立直後 返済） or 第三者 名義借し / q2=no（出処 説明 不可） / q3=no（法人口座 入金 不明）

## 处理建议

### 绿
档案归档：
- 本人 銀行 通帳 写し（直近 12 ヶ月）
- 銀行 残高 証明書（設立直前 + 直近）
- 海外送金 SWIFT 控 + 受取 銀行 入金通知
- 法人 設立 時 払込証明書（金融機関 発行）
- 法人 銀行口座 通帳 写し（設立直後 入金 確認）
- 資金 出処 説明書

下次续签前 180 天复检。

### 黄
1. 本人 銀行 通帳 直近 6-12 ヶ月 写し 取得 + 大口入金 各件 出処 メモ。
2. 海外送金：
   - SWIFT 控 + 送金 申込書 + 受取 銀行 入金 通知
   - 海外側 通帳 写し（出金 確認）
   - 海外 給与 / 事業所得 場合：源泉徴収票 / 給与明細 / 海外 確定申告書
3. 不動産 売却 / 退職金 場合：売買契約書 / 退職金 明細書。
4. 設立 払込証明書 + 法人口座 入金 履歴 突合。

### 红
1. **借入金 で 資本金**：
   - 設立直後 返済 = 「形だけ 出資」と判断。
   - 借入契約書 + 貸主 通帳 + 返済計画 必要。事業 純利益 で 返済可能 立証 必要。
2. **第三者 名義借し**：
   - 出資 法律行為 上 第三者 のもの → 経営者 と 出資者 不一致 = 経営・管理 適格性 揺らぐ。
   - 名義借し 発覚 = 在留資格 取消 リスク（入管法 第 22-4 条）。
3. **出資金 出処 不明**：
   - 銀行 通帳 突発入金 で 出処 説明 不能 = 資金 洗浄 / 不正 入金 疑い。
   - 直近 12 ヶ月 通帳 + 出処 各件 説明書 必要。
4. **海外送金 受取 不可 / 中国 海外送金 規制**：
   - 中国 個人 年間 5 万 USD 上限。複数人 名義 分割送金 = 違法。代替 方法（増資 タイミング 分散、海外 法人 → 日本法人 出資）検討。
5. 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
