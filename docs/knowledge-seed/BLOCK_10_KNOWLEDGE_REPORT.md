
### 正文 6 段结构（取代旧 4 段法）

1. **## 这事跟我有关吗？** — 直接判断「如果你是 X / 处在 Y / 收到 Z，这篇适合你」
2. **## 这是什么** — 客观描述，不评价不建议
3. **## 你应该做什么** — 编号步骤，每步可执行
4. **## 不做的后果** — 客观时间线，不戏剧化不猜测
5. **## 什么时候你应该找专家** — 简单/复杂分流。复杂 case 引导到 `/consultation`
6. **## 信息来源** — markdown 链接 + 取得日 + 法律条文

### 写作准则执行情况

- ✓ "书士" / "行政書士" 仅在第 5 段引流句出现（每篇一次）
- ✓ 不出现"待书士审核"标签
- ✓ 去戏剧化（删除"立即/必须/重要/严重/杀手"等）
- ✓ 去猜测（"可能影响 X" → 客观陈述或删除）
- ✓ 中日混合规则保留（在留期間更新 / 住民税 / 確定申告 / 在留カード / 入国管理局 / 出入国在留管理庁 等术语日文原文）
