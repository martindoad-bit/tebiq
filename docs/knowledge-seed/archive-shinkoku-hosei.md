# 入管 補正通知：档案如何快速回应

## 这事跟我有关吗？
你已经把 在留期間更新 / 在留資格変更 / 永住 申请书递了出去，在审查中收到入管寄来的一份通知，要求补充材料、解释材料矛盾、或就某个具体问题做書面回答。这就是 補正通知（資料追加 / 説明書 提出依頼通知）。它不是「不許可」，但它有期限——通常 2-4 周——而期限内回应不到位 = 视为撤回 或 直接 不許可。档案在这一刻的作用是：让你能在 30 分钟内列出原递交清单与入管要求的差集，而不是花 3 天回忆当时提交了什么。

## 这是什么
補正通知（一般包括「資料提出通知書」「説明書提出依頼」「来庁依頼」等多种形式）的核心信息：

- **要求项目**：缺什么材料 / 需补充说明的具体事项
- **回应期限**：通常 2-4 周（信件上明记）
- **回应方式**：邮寄返送 / 入管窓口 持参 / オンライン申請 系统上传
- **担当部門 / 担当者**：信件抬头与署名
- **申請受付番号**：必须在回应文件上同时记载

档案需保存与回应直接相关的材料：

- 当初申請時 提交的全部材料清单（申請書 / 添付書類 / 写真 / 立証資料）
- 各材料的版本与发行日期（住民票 / 課税証明書 / 在職証明書 等都有有効期）
- 对应每项要件的立証資料（如 永住 的「公的義務 完納」对应 納税証明書 / 年金記録 / 健保 記録）
- 之前与入管的所有書面 / 来庁 履歴

档案能完成的对账动作：

- 原递交材料清单 vs 入管 補正通知 要求 → 显示「缺什么」
- 已有但未递交的材料 → 显示「能从档案直接补充什么」
- 需要新申请的材料（如最新 課税証明書）→ 显示「市役所 / 税务署 要去开什么」

補正通知 与「不許可」「撤回」的关系：

- 補正通知 = 审查中、入管认为材料尚有不足，给一次补充机会
- 期限内不回应 = 申請取下 みなし（视为撤回）→ 等同重新申请
- 内容不充分 = 不許可（决定通知書）

## 你应该做什么
按照客观顺序处理：

1. **核对要补什么**：打开 補正通知 原文，把「要求事项」逐条写下。每一条对应档案中哪一类事件 / 材料，做交叉对账。
2. **准备材料**：能从档案直接抽取的（已有的 住民票 副本、源泉徴収票、賃貸借契約書 等），直接打印；需要新申请的（如最新一年 課税証明書 / 納税証明書），当周内向 市役所 / 税务署 申请。
3. **回应方式**：按 補正通知 指定方式回应。邮寄返送时挂号 / レターパック プラス，保留 追跡番号；窓口持参时档案中记录受付日 / 担当者。回应文件首页注记「申請受付番号 / 氏名 / 在留カード番号 / 補正通知書 受信日」四项。
4. **回应后**：把 補正回応書 全套 + 邮寄追跡 / 窓口受领票 入档，作为「与入管交互履歴」事件保留。后续审查或不許可时这一份履歴是申诉与再申请的基础材料。

## 不做的后果
補正通知 期限内不回应：

- 入管 视为「申請取下 みなし」→ 当前申请失效
- 在留期限 已过 = 不法滞在 风险（特例期間 在审查终了同时結束）
- 必须重新申请，前期已支付的手数料無法返还
- 短期再申请会被入管视为「同一案件再申請」，审查会更严

回应不充分（材料不齐 / 説明不到位）：

- 不許可 决定 → 在留期限 結束后 出国準備期間 通常 30-60 日
- 永住 申请不許可 = 当前 在留資格 不变但永住计画推迟数年
- 在留資格変更 不許可 = 必须出国 或回到原资格状態（若仍有效）

档案如果不能在收到通知的当天列出「我已经交了什么」，光是回忆与翻邮件就要花掉期限的 1/3。

## 什么时候你应该找专家
収到 補正通知 后，以下情形强烈建议咨询行政書士：（1）通知要求的「説明書」涉及偽装 / 真実性 質問（婚姻、雇用、活动实绩）；（2）通知列举具体的「不許可事由 候補」(如 公的義務 不履行、納税不足额、虚偽申告 嫌疑)；（3）期限剩 1 周以内材料仍未齐；（4）通知是「来庁依頼」而非「資料追加」（来庁面接通常意味着审查官有较深的疑问）；（5）有不許可 履歴 的再申請过程中收到 補正通知。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
