# 我和日本人结婚要申请配偶者签证，入管会查什么真实性证据

## 这事跟我有关吗？

如果你和日本人或 永住者 完成婚姻登记，准备申请「日本人の配偶者等」或「永住者の配偶者等」在留資格，这篇适合你。如果你在日本国外通过短期相识结婚、和配偶 年龄差较大、或者对方有过多次国际婚姻经历，更需要在递件前看完——入管 现在对偽装結婚 的审查力度一直在加严，材料不够 的真实婚姻也可能被判 立証不足 而不许可。

## 这是什么

「日本人の配偶者等」「永住者の配偶者等」属于 身分系 在留資格，没有学历、收入门槛，但有一个核心条件：婚姻必须真实持续——双方有共同生活意思、实际共同生活，且不是为了取得 在留資格 而办理的形式婚姻。

入管 在审查这一要件时，依据的是申请人提交的 質問書 与各类客观证据，结合数据库中扶養者侧的资料、过往出入境记录、过往婚姻履历等综合判断。审查官没有义务相信「我们是真感情」的口头主张——立証責任 在申请人一方。

**高频审查触发点**（出现任一项，材料密度都要拉高）：

- 双方 年齢差 较大（一般 15 歳 以上 会被重点关注，差距越大越严）
- 相识到结婚 时间短（数月以内的快速结婚）
- 来自被入管列为 偽装結婚 高发风险国 的申请人
- 双方在结婚后或申请前有较长 不同居 期间
- 配偶任何一方有 多次国際結婚歴（特别是间隔短、对象国家集中）
- 双方语言不通，且无翻译辅助的共通沟通工具
- 配偶有 風俗営業 相关从业经历或 在留資格 不许可 履历

## 你应该做什么

1. **质問書 由本人亲笔填写**

   质問書 是入管核心审查材料。两人对相识経緯、求婚时点、结婚仪式、共同生活细节的描述会被交叉比对——配偶日方一份、外国人本人一份。建议两人各自独立填写，不互相对答案，但事先把客观事实（日期、地点、人名）确认一致。代写、内容雷同、明显由第三方一手包办，反而会被怀疑。

2. **準備 相识経緯 説明書 + 时间线证据**

   按时间顺序写明从认识、交往、求婚到结婚登记的过程，每个关键节点附 客观证据：
   - 第一次见面：当时的机票、酒店预订、相亲所提供的紹介状
   - 交往期间：往返机票存根、出入境记录、酒店住宿记录、礼物快递单
   - 求婚 / 婚约：餐厅订单、戒指购买记录、纪念品照片
   - 结婚登记：婚姻届受理証明、結婚式 照片与场地费用记录

3. **共同生活照片（含双方家族）**

   不是几张自拍，而是覆盖时间跨度的照片集：恋爱期、订婚、结婚式、婚后日常、与对方父母兄弟姐妹的合影、与共同朋友的聚会、回中国 / 第三国旅行。建议 20–40 张，每张备注时间地点人物。和配偶的家族 合影 是高权重证据。

4. **SNS / 通讯履历**

   LINE、微信、WhatsApp、Skype 等聊天记录截图，覆盖从相识到现在的代表性对话。建议每个月挑 2–3 段，自然内容（生活琐事、关心、争吵和好），不是刻意拍给入管看的。视频通话频率截图、Instagram / 朋友圈 互动也算。

5. **共同生活的 客观文件**

   - **共同住所証明**：双方一致的 住民票（世帯全員）、电气 / 水道 / ガス 契約者名义、賃貸借契約書 上有双方名字、配送地址记录
   - **共同口座 / 家計 共有**：可以是夫妻共同口座 的通帳写し，或一方负担家计而另一方有 生活費送金記録 的银行流水
   - **保险 / 緊急連絡先**：互相把对方设为 健康保険 被扶養者、生命保险 受益人、紧急联系人

6. **走完正式流程**

   登录 [出入国在留管理庁「日本人の配偶者等」申請页面](https://www.moj.go.jp/isa/applications/status/spouse.html)，按 在留資格認定証明書交付申請（境外召请）/ 在留資格変更許可申請（境内身份变更）/ 在留期間更新許可申請（既有签证更新）三个流程之一提交。受理後 1–3 个月出结果，部分案件会有追加資料 通知或本人 / 配偶 出頭 要求。

## 不做的后果

- **真実関係 但材料不够 → 立証不足 不许可**

  这是最常见的情形。婚姻是真的，但入管 看完材料无法形成「双方有持续共同生活意思」的心证，结果不许可。本身不是黑历史，可以补强材料后再申请，但当事人会损失数月到一年时间，期间外国人方可能要先转去 短期滞在 或回国。

- **偽装結婚 認定 → 在留資格 取消 + 退去強制 + 上陸拒否**

  如果入管 认定为 偽装結婚，后果是连环的：
  - 在留資格 取消（入管法 第 22 条の 4）
  - 退去強制手続 启动，被强制遣返
  - 5 年 或 10 年 上陸拒否（前科有无、是否自主出国 影响年数）
  - 配偶（日本人 / 永住者）侧也可能被以 公正証書原本不実記載罪（刑法 第 157 条）追究刑责
  - 介绍人 / 中介 涉及金钱报酬时按 入管法違反幇助 / 偽装結婚あっせん 立件

- **真実関係 但隐瞒不利信息**

  比如不主动申报过往婚姻、过往 在留資格 不许可 履历、配偶 風俗営業 從業歴。入管 数据库会查到，被认定隐瞒后即使关系真实也很难翻盘。

## 什么时候你应该找专家

**你可以自己处理**，如果同时满足：

- 双方年龄差在常识范围内
- 相识到结婚有 1 年以上 自然交往
- 双方语言可正常沟通
- 没有过往 在留資格 不许可 / 退去強制 履历
- 配偶 收入安定、过往無前科
- 共同住所、共同口座、共同生活照片 等基础材料齐全

**建议找专家**，如果出现以下任一：

- 双方 年齢差 15 歳 以上
- 相识到结婚 不足 1 年
- 配偶 / 自己有多次国际婚姻经历
- 申请人来自入管 重点审查国
- 不同居 期间较长（双方分居两国超过半年以上）
- 双方语言不通
- 任一方有 風俗営業 從業歴、过往 在留資格 不许可、退去強制、上陸拒否 履历
- 已经被入管 要求追加資料 或本人出頭
