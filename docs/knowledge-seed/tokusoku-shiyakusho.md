# 我收到市役所「督促状」，要紧吗 / 几天内交

## 这事跟我有关吗？

你最近收到了一封寄件人为 **市役所 / 区役所 / 都税事務所 / 県税事務所 / 年金事務所 / 国民健康保険組合** 等行政机关、封面 / 内文标题为「**督促状**（とくそくじょう）」的信件——这篇适合你。无论你是会社员、自営業者、留学生、家族滞在，只要你是 **公租公课的纳税 / 缴费义务人**，就都可能收到。

「督促状」**不是普通的提醒**，而是法律意义上的 **滞纳确定通知**——已经触发了一系列后续强制手续。**要紧。建议在 7 日内回应。**

## 这是什么

「**督促状**」是行政机关依《地方税法》第 329 条 / 《国税通則法》第 37 条 / 《国民年金法》第 96 条 等法律，对 **过了納期限仍未缴纳的公租公课** 发出的正式催促通知。

常见对象：

- **住民税**（市町村民税 + 都道府県民税）
- **固定資産税**
- **軽自動車税**
- **国民健康保険料 / 後期高齢者医療保険料**
- **国民年金保険料**（由日本年金機構发出）
- **介護保険料**
- **保育料 / 給食費**（部分自治体）
- **国税**（所得税 / 消費税、由 税務署 发出）

### 时间线

```
納期限     → 督促状     → 催告状        → 差押予告      → 差押執行
            (約 20 日後)  (督促後 1～3 ヶ月)  (最終警告)      (強制処分)
```

- **納期限後约 20 日**：自治体依法发出 **督促状**（地方税法 第 329 条所定）
- **督促後 10 日経過後**：自治体取得 **差押実施権**（不需法院判决）
- **督促状以降**：每日按法定利率（特例基準割合）计算 **延滞金 / 滞納金**
- **督促後 1～3 ヶ月**：发出 **催告状 / 最終催告状**（更严厉的督促）
- **催告无效**：发出 **差押予告書**
- **差押執行**：給与 / 預金 / 不動産 / 動産 / 生命保険 等被强制变现

### 督促状内容

- **滞纳的税目 / 保険料种类**（住民税 第○期 / 国保料 ○年度 等）
- **本税 / 本料金额**
- **延滞金 / 督促手数料**（自治体大多 100 円程度）
- **新的纳付期限**（通常督促状发出后 10～14 日内）
- **缴纳方式**（コンビニ払 / 銀行振込 / Pay-easy / 窓口）
- **联系窗口**（市役所税務課 / 收納課 / 滞納整理課）

## 你应该做什么

**几天内交：建议在督促状指定的新納付期限内（通常 7～14 日）缴清。即便无法立刻全额缴清，也应在 7 日内联系窗口商谈分纳。**

1. **立刻看清欠的是什么**。督促状第一行 / 表格里写明税目 / 保険料种类 + 期数 + 金额。同时收到多张督促状的话，一一确认。
2. **能交立刻交**。现金充足者 → 当天 / 次日 通过 コンビニ / 銀行 / Pay-easy / 自治体公式アプリ 全额缴清。**全额缴清后，差押手续中止。**
3. **不能一次交清者**：**绝不要装不知道**。立即电话或前往 **市役所税務課 / 収納課 / 滞納整理課**（年金的话是 **年金事務所**）说明情况、申请：
   - **分割納付（分納）**：一般可分 6～12 期、每月支付一部分
   - **納税の猶予**（地方税法 第 15 条）：因災害 / 病気 / 事業休廃 / 著しい损失 等理由，**最长 1 年（特定情况下可再延 1 年）猶予徵收**
   - **換価の猶予**（地方税法 第 15 条之 5 / 6）：财产仍在但一次缴清困难时
   - 申请通过后 **延滞金免除 / 軽減** 可能适用
4. **去窗口前准备**：在留卡 / マイナンバーカード、督促状原件、最近 3 个月家计 / 收入流水（分纳申请时被要求）。
5. **缴清后保留収納証明 / 領収書**。日後永住 / 帰化申请 / 在留更新时可能需要证明 **過去の納税 / 納付履歴に問題がない**。

## 不做的后果

- **第 1 日（督促状）後**：每日开始累积 **延滞金 / 滞納金**（年率约 8.7%、特例基準割合 + 7.3%、納期限後 1 个月以内的较低、其后较高、具体每年由総務省告示）
- **督促後 10 日経過**：自治体取得 **差押実施権**——理论上随时可启动強制手続
- **1～3 ヶ月後**：**催告状 / 最終催告状** 寄达
- **持续无视**：**財産調査**（銀行口座 / 勤務先 / 取引先 等被照会）
- **差押執行**：
  - **給与差押**：手取り部分から差押可能（《国税徴収法》第 76 条 等：原则上 **給与の 1/4** 上限、但生活費 控除後）
  - **預金差押**：銀行口座残高一括押え
  - **動産 / 不動産差押**：自宅 / 自動車 等
  - **生命保険解約金 差押**
- **永住 / 帰化 / 在留更新审查**：纳税 / 公的保険納付 是入管 **核心审查项目**——「滞纳记录 + 督促 + 差押」会直接构成 **不利推定**、严重时直接不许可（参照 [moj.go.jp 永住許可ガイドライン](https://www.moj.go.jp/isa/applications/procedures/16-4.html)）

## 什么时候你应该找专家

- **金额过大、本人完全无法分纳承担** → 司法書士 / 弁護士（必要时考虑 **個人再生 / 任意整理 / 自己破産** 的可能性、但公租公课 **原则上不能因破産而免除**、需慎重）
- **已经收到 差押予告書 / 差押通知書** → 立即弁護士 / 司法書士（残期可能仅有数日）
- **永住 / 帰化申請 间近** + **滞纳 / 督促 / 差押 履历** → 行政書士（评估 影响 + 补救方案）
- **申请 納税の猶予 / 換価の猶予 但不知如何说明事由** → 税理士 / 行政書士
- **督促对象金额你认为有错** → 先到市役所税務課申请「**課税内容の確認**」、必要时 **不服申立て（審査請求）**
