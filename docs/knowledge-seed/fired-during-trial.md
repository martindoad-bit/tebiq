# 我试用期被辞退了，签证还有效吗 / 多久必须找下家

## 这事跟我有关吗

如果你符合以下条件，请继续往下看：

- 持有 就労系 在留資格（技人国 / 経営・管理 / 高度専門職 等）
- 入社后处于 試用期間（一般 1–6 个月）被公司单方面解除雇用
- 解雇通知书 / 退職勧奨 已经收到，或离职日已确定

被自己辞掉的（自主退職）也大体适用，但 雇用保険 受給 的等待期不同。

## 这是什么

很多人误以为"试用期解雇"在法律上比较随意——这不对。日本 労働契約法 第 16 条：解雇 必须有 客観的合理理由 + 社会通念上相当。試用期間 解雇 也是解雇，公司必须给出理由（能力不足 / 出勤状况 / 業務適性 等），并按规定的 30 日前通知 或支付 解雇予告手当。

入管这边的逻辑是分开的：

- **在留資格 不会因为被解雇而立即失效。** 你在 在留期間 内，签证本身仍然有效。
- 入管法 第 22-4 条 規定：在留資格 持有人持续 3 个月 不从事相关活动 = 在留資格 取消事由（详见 [失业 90 天规则](unemployed-90-day-rule.md)）。

短期内（数周到 1–2 个月）找到新工作 = 几乎无影响。

## 你应该做什么

### 当天 ～ 离职后 14 日

1. 拿到 解雇通知書 / 退職証明書 原件——写明 解雇日 / 解雇理由 / 試用期間 結了 还是 试用期间内 解雇
2. 拿 離職票（公司离职后 10 日左右出，用于 雇用保険 申请）
3. 旧公司 契約機関 離脱届：14 日内向入管提交（线上 / 邮寄 / 窓口）
4. 健康保険 / 厚生年金：旧公司退会 → 立刻办 国民健康保険 + 国民年金 加入（市区町村役所）；找到新工作再切回

### 离职后 1–4 周：积极找下一家

- 入管对"积极找工作"的判断主要看 ハローワーク 求職登録 / 面接記録 / メール 履歴 等可视化的努力
- 即使没立刻找到，雇用保険 失業給付 可以申请——会计入下次 更新 时的"在日活动状况"评估
- 求職活動的時候，理论上 在留資格 範圍 还在保护你，但你不能拿 就労系 在留資格 去做超出范围的工作

### 找到新公司后

- 入社日起 14 日内 提交 新 契約機関に関する届出
- 新岗位与现 在留資格 跨范围 → 申请 就労資格証明書 事前确认（详见 [30 天行动表](changing-jobs-30day-action.md)）
- 下次 更新 时入管会问 解雇事由——准备 解雇通知書 + 新公司的雇用合同 一并提交

## 不做的后果

- 14 日内不报 離脱届：最高 20 万円 過料，下次 更新 信用扣分
- 不申请 失業給付 / 没有求職記録：3 个月 转眼到了，入管 取消事由 触发
- 解雇通知書 / 離職票 没拿：下次 更新 时入管要问"为什么离职"，没有书面证据，对方默认你 状况不稳定
- 健保 / 年金 空白：経営・管理 / 高度専門職 在 更新 时纳付状况 直接被看，断保会留痕

## 何时找专家

下面任一情况，请联系 行政書士 / 入管専門 弁護士 / 労働問題 弁護士：

- 解雇 理由 是"能力不足"但你怀疑实际是 不当解雇 / 内定取消（労働法 上可能可以争解雇無効 + 賃金請求）
- 已经离职 60 日以上仍未找到新岗位（90 天前界点逼近）
- 解雇 同时伴随 賃金未払 / 残業代未払（労働基準監督署 / 弁護士）
- 你是 経営・管理 持有人 但实质上是被合伙人挤出的，需要判断公司事業継続 / 自分で再起 路径
- 高度専門職 持有人，新岗位的 ポイント 计算可能不达标

書士 在试用期解雇这种节点的价值是判断你"距离 90 天底线还有多少安全垫" + "下次更新如何把 解雇 这一段写得对入管交代得过去"。
