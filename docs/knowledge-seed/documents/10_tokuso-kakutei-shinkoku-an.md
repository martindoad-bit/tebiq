> # ⚠️ 需创始人/書士复核：ハガキ 様式 ＋ 利用者識別番号（U で 始まる 16 桁） 通知方法 毎年運用改定；e-Tax 移行後 紙 ハガキ 廃止予定 自治体 / 税目 あり。

# 確定申告 案内ハガキ

## 信封长什么样
- 寄件人：「○○税務署」「国税庁」
- 封面印字：圧着 ハガキ（はがきの三つ折り型）/ 圧着 シール 開封式 / 「親展」/ 「所得税及び復興特別所得税の確定申告について（お知らせ）」
- 信封颜色 / 大小：白色 / 浅藍色 圧着ハガキ、定形 + 圧着部分 開封式
- 同梱物（ハガキ 内部）：申告期限 + 利用者識別番号（U で 始まる 16 桁、e-Tax ログイン用）+ 振替納税 引落予定日 + マイナポータル 連携 案内 + 決算書 / 収支内訳書 提出案内（事業所得者）

## 这是什么
確定申告 案内ハガキ = 税務署 が 申告対象者 に 「今年も 確定申告 期間 です + あなたの 利用者識別番号 + 振替納税 予定日 + 提出方法」を まとめて 通知する 圧着ハガキ。
2020 年代 から 紙 申告書 用紙 一式 送付 → ハガキ 通知 + 国税庁ホームページ で 自分で 作成 ＝「申告書 用紙 不送付」運用 に 順次 切替。
ハガキ ＝ 「申告対象者」と 税務署 が 認識 している サイン。但し ハガキ 来ない = 申告義務 なし では ない（自営 / 副業 / 不動産 / 株式譲渡 等 で 申告義務 ある 人 は 自主申告 必須）。

## 为什么收到
- 前年 確定申告 提出履歴 あり
- 給与支払報告書 / 支払調書 / 不動産 / 株式譲渡 通知 等 で 税務署 が 申告対象者 と 把握
- 還付申告 該当（医療費控除 / 寄付金控除 / 住宅ローン控除 1 年目 等）の 過去申請者
- 振替納税 登録済 → 振替日 確認用 通知 含まれる

不会収到の人：給与 1 か所 + 年末調整 完了 + 副収入 ¥20 万円以下 + 控除追加申請 不要 + 申告履歴 ない 新規在留者。

## 必看的关键栏
- **申告期限**：原則 翌年 3 月 15 日（休日翌営業日 へ 順延）
- **利用者識別番号**：U で 始まる 16 桁。e-Tax ログイン に 必須。紛失時 再発行 可能
- **振替納税 予定日**：通常 4 月下旬（毎年改定）
- **マイナポータル 連携**：給与所得 / 公的年金 / 医療費通知 / 生命保険料控除 等 を 自動取得 → 申告書 自動入力
- **問合せ先**：管轄 税務署 + 国税庁 確定申告電話相談センター（毎年 1 月下旬 - 3 月中旬 開設）

## 不处理的后果（时间梯度）
ハガキ 単独 で 「やらないと 罰則」ではないが、確定申告 義務 = 別途 法定。
- **期限後 1 ヶ月内**：期限後申告 自主提出 → 無申告加算税 軽減（5%）
- **税務署指摘後**：無申告加算税 15-20%
- **申告 完了 + 納付 遅延**：延滞税 年 2.4% / 1 ヶ月後 年 8.7% 前後
- **過少申告 / 隠蔽**：過少申告加算税 10-15% / 重加算税 35-40%
- **未申告 持続**：税務調査 → 7 年遡及 / 差押
- **更长期**：在留期間更新 / 永住 で 公的義務不履行 評価

逆 に 還付申告 該当者 が ハガキ 受領後 何も しない → 還付金 受領機会 喪失（5 年以内 なら 後日 還付申告 可能）。

## 该怎么处理（3 种典型路径）

### 路径 A：申告対象者 → 期限内 e-Tax 申告（推奨）
1. ハガキ 上 利用者識別番号 確認
2. [国税庁 確定申告書等作成コーナー](https://www.keisan.nta.go.jp/) ログイン
3. マイナポータル 連携：マイナンバーカード + スマホ / IC カードリーダー で 給与 / 年金 / 医療費 / 生命保険 控除 自動取得
4. 申告書 作成 → e-Tax 直接送信（24 時間 受付、3 月 15 日 23:59 まで）
5. 納付：振替納税（4 月下旬 自動引落 = 実質 1 ヶ月延長）/ クレカ / コンビニ / 銀行 / ダイレクト納付
6. 申告書 控 + 受信通知 を 7 年保存

### 路径 B：還付申告 該当（医療費 / 寄付金 / 住宅ローン控除 等）
触发条件：給与所得者 + 控除追加 → 還付。
1. 期限：原則 申告期限 と 同じ だが、還付申告 は 翌年 1 月 1 日 から 5 年間 受付（過去 5 年分 まで 遡及可）
2. 必要書類：医療費通知 / 寄付金受領証明書 / 住宅ローン残高証明書 / マイナンバー 等
3. 還付金 振込先 口座 申告書 に 記入 → 申告後 1-2 ヶ月 で 振込

### 路径 C：申告義務 不明 / 複雑 ケース
触发条件：複数収入 / 海外所得 / 不動産売却 / 株式譲渡 / 暗号資産 譲渡 等。
1. 国税庁 タックスアンサー / チャットボット / 電話相談センター で 一次確認
2. 税務署 窓口 個別相談（要予約、確定申告期間 中 混雑）
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士 / 税理士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 在留期間更新：直近 1-3 年 「納税証明書（その 1：所得税）」「課税証明書」が 標準提出書類。本ハガキ + 申告書 控 = 申告履歴 の 直接証拠。
- 永住申請：直近 5 年 納税状況 + 期限内申告 が 重要評価項目。「期限後申告 + 加算税」は マイナス。
- 経営・管理：役員 個人 確定申告 = 給与所得 + 役員報酬 が 1 か所 + 年末調整 完了 + 副業 ¥20 万円以下 → 申告 不要 ケース あり。但し 不動産 / 株式譲渡 / 海外所得 等 で 別途 申告義務。
- 「ハガキ 来ない = 申告義務なし」誤解 多発。給与 ¥2,000 万円超 / 副業所得 ¥20 万円超 / 不動産家賃 / 株式譲渡 / 海外所得 / 暗号資産 譲渡 / 個人事業 開始 = ハガキ 関係なく 申告義務。
- 海外（中国 等）所得 ある 在留者：在留 5 年 経過 → 「永住者」課税区分 = 全世界所得 課税 → 中国 国内 不動産家賃 / 株式譲渡 / 利息 等 全部 申告対象。租税条約 + 外国税額控除 で 二重課税 調整。
