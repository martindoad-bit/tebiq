# 入管 不許可通知書（理由付き）

## 信封长什么样
- 寄件人：申請先の地方入管局
- 形式：茶色 葉書 / 定形封筒。封面に「出頭通知書」「不許可通知」「在留審査結果通知」
- 同梱物：受付番号、出頭指定日 / 場所、担当審査官 名、内線、持参物（旅券、現 在留カード、申請受理票）
- **不許可理由の詳細は通常 書面化されず、出頭時の口頭説明** が原則（メモ取り必須）

## 这是什么
在留期間更新 / 在留資格変更 / 永住許可 / 在留資格認定証明書（COE） いずれかの申請に対する **不許可** を伝える書面。出頭通知書 形式が多く、出頭日 に 担当審査官 から 理由概要 を口頭で受ける構造。

## 为什么收到
申請内容が 入管法 / ガイドライン の許可基準を満たさないと判断されたため。代表的な理由：
- **学歴・職歴 と 職務内容 不一致**（技人国）
- **出資要件・事業計画 不十分 / 事業継続性 否定**（経営・管理）
- **素行善良性 否定**（交通違反、税滞納、年金未納、刑事処分）
- **独立生計能力 否定**（永住、収入が要件未満）
- **資料不備 / 虚偽記載**

## 必看的关键栏
- **出頭日時 / 場所 / 担当審査官名 / 内線**
- **持参物**：旅券、現 在留カード、申請受理票、本通知書
- **書面記載の不許可理由概要**：抽象的な 1 行（「在留資格該当性 を満たさない」「素行善良 が認められない」等）
- **元 在留資格 の残期間**：これにより次の動きが決まる

## 不处理的后果（時間梯度）
- **出頭日 不出頭**：再度 通知 / 場合により 退去強制手続 着手の前段
- **元 在留資格 残期間 内 何もしない**：在留期限 までに 別申請（再申請 / 在留資格変更）必要、超過すれば オーバーステイ
- **元 在留資格 期限切れ + 不許可**：「特定活動（出国準備）30 日 / 31 日」自動付与 ケース、それも経過で 退去強制手続
- **退去強制手続 着手後**：上陸拒否期間 5 年（通常）/ 10 年（再犯）

> 入管法 第 24 条（退去強制事由）、第 22 条の 4（在留資格 取消）

## 该怎么处理（3 种典型路径）

### 路径 A：出頭 → 不許可理由 詳細聴取
1. 指定日時 に 指定入管局 へ持参物揃えて出頭。
2. 担当審査官 と面談、不許可理由 を **聞き取りメモ**（書面交付なし、録音可否は要確認）。
3. **元 在留資格 残期間** を確認、次の選択肢（再申請 / 別資格変更 / 出国）を聴取。
4. 旅券 / 在留カード を返却受領（持ち込まれていた場合）。
5. 档案归档：通知書本体、面談メモ、申請理由書 控え（前回分）、不許可理由整理。

### 路径 B：再申請 / 別資格変更
1. **同一理由での即時再申請 = 再不許可 ほぼ確実**。理由補強 / 書類差替 / 雇用主変更 / 事業計画刷新 が前提。
2. **再申請までの最低期間**：明文規定なし、ただし不許可理由 の改善に通常 6 ヶ月〜2 年かかる（特に 永住・素行善良 系）。
3. **別在留資格 への変更**：技人国 不許可 → 特定活動 / 高度専門職 検討、永住 不許可 → 在留期間更新 で繋ぐ等。

### 路径 C：弁護士 / 行政書士 緊急対応
- 元 在留資格 期限切れ間近 + 不許可（タイムプレッシャー高）
- 不許可理由 が「資格該当性」「素行不良」「虚偽申請疑い」等 構造的・重い
- 過去 2 回以上 不許可履歴あり（再申請 戦略要再構築）
- 退去強制手続 着手 / 出国命令 寸前
- 入管法 上 異議申立（行政不服審査法）は **適用除外**、救済は **取消訴訟** か **再申請** に絞られる
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **不許可履歴 は 入管 内部記録に永続保存**。次回更新・変更・永住・帰化 すべての審査で参照される
- **複数回 不許可** = 素行善良性 否定的評価。永住 / 帰化 のハードル大幅上昇
- **再申請理由書** で「前回不許可理由 + 改善状況」を明記する戦略が標準
- **2024 年 6 月 法改正**：永住者 でも 取消事由 拡大 → 不許可履歴 + 違反 で 永住資格 取消もあり得る

> # ⚠️ 需创始人/書士复核：入管法上の異議申立 適用除外 の正確な根拠条文（行政不服審査法 第 7 条 1 項 10 号 / 入管法 特例の整理）と、取消訴訟 の出訴期限
