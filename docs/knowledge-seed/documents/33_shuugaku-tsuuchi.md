# 就学通知書 / 小学・中学 入学通知

## 信封长什么样
- 寄件人：「○○市 教育委員会」「○○区 教育委員会 学務課」
- 封面印字：「就学通知書 在中」「重要書類」「親展」
- 颜色：白色 / 浅黄色 角 2 号 信封 + 自治体ロゴ
- 同梱物：就学通知書 本体 / 入学する学校 案内 / 健康診断（就学時健康診断）案内 / 学用品リスト / 入学式 案内 / 就学援助 制度 案内（参见文档 #34）

## 这是什么
学校教育法 第 17 条 に基づき、当年度 4 月 1 日 時点 で 6 歳（小学校）/ 12 歳（中学校）に 達する 子女 の 保護者 に対し、市町村 教育委員会 が 指定校 を通知 する 文書。学齢簿（住民基本台帳 ベース）に 子女が 登録 されている すべての 世帯 に 送付。外国籍 子女 も 公立学校 入学希望 申出 を出していれば 同様 に通知 が届く（学校教育法 上 義務教育 対象は 日本国籍 子女、但し 外国籍 子女も 受入 される）。

## 为什么收到
- 子女 が 学齢（6 歳 / 12 歳）に達する → 翌年 1 月頃 一斉 発送
- 転入 → 学齢簿 登録 → 1-2 ヶ月以内 に 改めて発送
- 外国籍 子女 で 「就学希望 申出書」を 教育委員会 に 提出 → 通知書 発送
- 学齢期 中盤（小 2-小 6 / 中 2-中 3）の 転入 → 「転学通知書」（本書類 と 別形式）

## 必看的关键栏
- **入学 学校 名称・所在地**（住所 で 自動指定、原則 学区制）
- **入学日・入学式 日時**（多数 4 月 6-9 日）
- **就学時健康診断 日時**（小学校 入学前 10-11 月、中学校 は 通常 なし）
- **持参物 / 提出書類**（誓約書 / 連絡カード / 緊急連絡先 / 通学路 確認図 等）
- **学校選択制 利用可否**（自治体 により あり / なし）
- **問合せ先**（教育委員会 学務課 直通番号）

## 不处理的后果（时间梯度）
- **就学時健康診断 欠席**：個別 受診案内 → 入学日 までに 受診 必要（学校保健安全法 第 11 条）
- **入学式 欠席 + 連絡なし**：学校 から 個別連絡 → 不登校 リスク 早期介入
- **長期不就学（30 日以上）**：教育委員会 → 児童相談所 連携 → 就学督励 → 児童虐待 / ネグレクト 疑義 case では 児童福祉法 介入
- **保護者 の 就学義務 不履行（日本国籍）**：学校教育法 第 144 条 で 10 万円以下 罰金（実務上 ほぼ 適用例 なし、行政指導 が中心）

外国籍 子女 は 義務教育 対象外 のため 罰則適用 なし、ただし 教育委員会 は 就学機会 確保 の 行政指導 を継続。

## 该怎么处理（3 种典型路径）

### 路径 A：通知通り 入学
1. 就学時健康診断 を 指定日 に 受診（多数 自治体 無料）
2. 入学説明会（2-3 月）出席 → 学用品 / 制服 注文 / 通学路 確認
3. 入学式 出席 → 提出書類 一式 持参（誓約書 / 連絡カード 等）
4. 通知書 / 健康診断結果 / 入学関連書類 を 5 年保管（転校 / 就学援助 申請 / 永住申請 補強）

### 路径 B：学校選択制 利用 / 区域外就学 申請
- **学校選択制 自治体**：通学区域 外 の 学校 を 第 1-3 希望 で 申込 → 抽選 / 選考
- **区域外就学 許可申請**：兄弟姉妹 同校 / 通院 / 部活動 等 特別事情 → 「区域外就学許可 申請書」+ 理由書 提出 → 教育委員会 審査
- **私立 / 国立 / インターナショナルスクール 進学**：「就学校 変更届」+ 入学許可書（写） を 教育委員会 に 提出 → 公立校 学齢簿 から 削除
- 提出期限：入学日 1 ヶ月前 まで（自治体 差あり）

### 路径 C：転入 / 転出 に伴う 学校変更
1. 転入：転入先 自治体 で 「転入届」+ 学齢簿 登録 → 教育委員会 が 「転入学通知書」発行 → 指定校 に 在学証明書 / 教科書給与証明書 持参
2. 転出：転出先 自治体 へ 在学証明書 + 教科書給与証明書 を持参 → 編入手続
3. 帰国 / 海外転居：「就学免除 / 猶予 願出書」を 教育委員会 に 提出（学校教育法 第 18 条）
4. 子女 在留資格 / 国籍 / 入管 関連争議（在留資格更新時 子女滞在資格 変更 等）を伴う case、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 教育系 文书 は 在留資格 審査 で 直接審査項目 ではない
- 但 子女 の 公立校 在学 = 「在日生活 定着 + 家族同居 実績」 → 永住申請（独立生計 要件 + 国益適合 要件）/ 帰化申請（素行善良 + 居住要件）の 補強資料
- 経営・管理 / 高度専門職 持有者 の 子女 公立校 入学 = 「事業継続性 + 日本社会 統合」 評価 上 プラス
- 子女 の 在留資格 が「家族滞在」「定住者」「永住者の配偶者等」の 場合、公立校 入学 自体 に 在留資格 制限 なし
- 通知書 自体 を 入管 申請 に 直接添付 する 機会 は 少 ないが、住民票 / 課税証明書 等 と 併せて 保管 推奨
