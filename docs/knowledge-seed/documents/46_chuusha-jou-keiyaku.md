# 駐車場 契約 / 契約更新通知

## 信封长什么样
- 寄件人：不動産会社（「○○不動産」「○○管理」）/ 駐車場運営会社（「タイムズ 24」「三井のリパーク」「NPC24H」月極部門）/ 大家 個人名
- 封面印字：「駐車場使用契約 更新のご案内」「月極駐車場 契約書類」「保管場所使用承諾証明書 在中」
- 同梱物：
  - 駐車場使用契約書（更新合意書）
  - 保管場所使用承諾証明書（**車庫証明 用**、有効期間注意）
  - 振込先案内 / 払込取扱票
  - 駐車区画 図面 / 車種制限案内

## 这是什么
月極駐車場 の使用契約 締結 or 更新の案内書。住宅賃貸契約とは別契約 で、借地借家法 の保護対象外（純然たる場所貸しのため）。**保管場所使用承諾証明書** は警察署で **自動車保管場所証明書（車庫証明）** を取得するための必須書類。

## 为什么收到
- **新規契約**：物件の駐車場利用申込後
- **契約更新**：契約期間満了（通常 1-2 年）の 1-2 ヶ月前
- **車庫証明 再取得時**：自動車買替・名義変更で新しい承諾証明書 が必要

## 必看的关键栏
- **駐車場 所在地 / 区画番号**：車庫証明 申請時に警察署提出
- **使用料**：月額（地域差大、首都圏 ¥15,000-¥50,000、地方 ¥3,000-¥10,000）
- **契約期間**：通常 1-2 年、自動更新条項の有無
- **更新料**：駐車場は更新料なし or ¥3,000-¥10,000 程度が多い
- **保管場所使用承諾証明書**：有効期間（多くは発行から 3 ヶ月）、車庫証明 申請には 1 ヶ月以内の発行が望ましい
- **車種制限**：全長 / 全幅 / 全高 / 重量 / 車種（軽 / 普通車 / SUV）
- **解約予告期間**：通常 1 ヶ月前
- **支払方法**：口座振替 / 振込 / 自動引落

## 不处理的后果（時間梯度）
- **未返送 / 未契約**：契約期間満了で **使用契約終了** → 退去命令 → 強制移動（民事執行）
- **更新料・月額未払**：1-2 ヶ月分滞納で督促、3 ヶ月程度で契約解除 + 民事訴訟
- **車庫証明 失効リスク**：保管場所変更時に警察署届出（自動車保管場所法 第 7 条）+ 道路上保管の場合は虚偽届出 → **罰金 20 万円以下**（自動車保管場所法 第 17 条）
- **無断駐車 / 期限切れ放置**：管理会社による民事執行（陸送費・保管料が後日請求、平均 ¥30,000-¥100,000）
- **賃貸住宅付帯駐車場**：住宅契約と連動の場合、住宅契約解除リスクあり

> ⚠️ 罰金額・民事執行費用は事案毎で変動。

## 该怎么处理（3 种典型路径）

### 路径 A：継続使用 → 更新書類記入・返送
1. 駐車場使用契約書 / 更新合意書に署名・捺印、振込明細を返送。
2. 新たな **保管場所使用承諾証明書** が同封 → 車庫証明 再取得が必要かを確認。**車庫証明 は車検証 記載住所と一致が必要**。
3. 月額自動引落の場合は引落口座を確認。残高不足での戻入は督促対象。

### 路径 B：契約条件変更 / 値上げ拒否
1. 周辺相場を確認（[全国の月極駐車場相場](https://www.tokyutimes.co.jp/) 等の比較サイト）。
2. 借地借家法 適用外のため、家賃減額請求権 は法的根拠なし → 純然たる契約交渉。
3. 値上げ拒否で合意なければ、契約期間満了で **当然終了**、別駐車場へ移動 + 車庫証明 再取得。

### 路径 C：解約 / 引っ越し
1. 解約予告期間（通常 1 ヶ月前）を契約書で確認。
2. 「解約通知書」を書面で送付、引落停止依頼。
3. 引っ越し先で新駐車場 契約 → 保管場所使用承諾証明書 取得 → 警察署で **保管場所変更届出**（自動車保管場所法 第 7 条、引っ越し後 15 日以内）。
4. 滞納金回収・退去命令を伴う紛争に至れば、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 駐車場 契約は 在留期間更新 / 永住 / 経営・管理 の **直接審査項目ではない**
- ただし **車庫証明 関連の虚偽届出 / 道路上保管 違反** は 道路交通法・自動車保管場所法違反 → 罰金 / 反則金 → 永住「素行善良」評価で間接影響
- 経営・管理：会社車両の駐車場契約は 事業継続性 + 事務所運営の物的基盤 として、続签時に 賃貸借契約書 と並んで参考資料

> ⚠️ 永住「素行善良」運用は事案毎、罰金・前科 を伴う場合のみ実務的に問われる。
