# 児童手当 現況届 / 儿童补贴 现状申报

## 信封长什么样
- 寄件人：市役所 子ども家庭支援課（こども政策課 / 児童福祉課 等 自治体ごと 名称異なる）
- 封面常见印字：「児童手当 現況届 在中」「6 月中 に 必ずご返送ください」「重要書類」
- 颜色：白底 + 自治体 LOGO（多くは 6 月 上旬 一斉郵送）
- 同梱物：現況届 用紙 1 枚 / 記入見本 / 返信用封筒（料金受取人払）/ 健康保険証 写し 添付台紙

## 这是什么
児童手当 受給中 の保護者 が 毎年 6 月 に 提出 する **養育状況 継続確認** 書類。2022 年 6 月 制度改正 で 「マイナンバー 連携 + 所得情報 照会 が可能 な世帯」 は 提出不要 化されたが、**離婚・別居・公務員・海外渡航 児童 等 一部世帯** は引続き提出義務あり。提出した世帯 のみ 6 月以降分 の 児童手当 が継続支給される。

> # ⚠️ 需创始人/書士复核：2024 年 10 月 児童手当 拡充（高校生まで延長 + 所得制限撤廃 + 第 3 子 月額 3 万円）後 の 現況届 提出範囲 規定 が自治体差大。具体提出要否 は 各市役所 案内書面 に従う。

## 为什么收到
- 児童手当 受給認定中 の世帯
- 自治体 が マイナンバー 情報連携 で 所得・世帯状況 を確認できない事由（DV 避難・別居・海外居住児 等）あり
- 公務員 受給者（職場経由 受給 のため）
- 監護 / 生計同一 / 居住状況 の 個別確認 が必要 と判断された世帯

## 必看的关键栏
- **受給者氏名 + 基礎年金番号 / マイナンバー**
- **対象児童**（氏名・生年月日・住所・続柄）
- **養育状況**（同居 / 別居 / 海外 / 施設 入所 等）
- **配偶者 の有無 + 生計関係**
- **提出期限**（原則 6 月末日。自治体 で多少前後）
- **添付書類**（健康保険証 写し / 別居監護申立書 / 海外居住児 在学証明 等）

## 不处理的后果（時間梯度）
- **6 月末 提出期限 経過**：7 月以降分 の 児童手当 支給 一時停止
- **2 ヶ月 経過**：催告書 送付
- **6 ヶ月 経過 までに 未提出**：受給資格 消滅。8 月 / 10 月 / 12 月 / 翌 2 月 / 4 月 の 振込 全停止
- **受給資格 消滅 後 の 再認定申請**：消滅日 まで 遡及不可。**翌月分 から 再支給 開始**（過去分 喪失）
- 信用 / 在留 への 直接影響 なし（公的給付 の受給停止 のみ）

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 期日内 返送（提出義務世帯）
1. 同梱 用紙 に 受給者・配偶者・対象児童 の現況 を記入。
2. **健康保険証 写し**（被扶養者欄 に児童名 が記載されているもの）を添付。被用者保険 加入者 のみ。
3. 返信用封筒（料金受取人払）で 6 月末 までに 子ども家庭支援課 へ送付。窓口持参 / 電子申請 も自治体 により可。
4. 8 月 振込（4-7 月分）以降 が継続。

### 路径 B：別居 / 海外 / 特殊事情 申告
- **児童 と別居（単身赴任 等）** → 「別居監護申立書」+ 別居児童 の 住民票 添付。
- **児童 が海外 在住**（留学 / 母国帰省 中）→ 原則 「日本国内 に住所 を有する 児童」 が対象。3 年以内 + 教育目的 + 継続して 国内 居住 であれば 一部例外あり（在学証明書 + 渡航証明）。
- **離婚協議中 / DV 避難中** → 監護実態 を有する 親 が受給者。住民票 上 単身世帯化 + 申立書 + 必要に応じ 弁護士 / 民生委員 の 副署。

### 路径 C：認定取消 / 額算定 不服申立
1. 現況届 提出後、自治体 が 「監護要件 不充足」「生計同一 不充足」 等 で 受給資格 取消 / 減額 を通知 した場合、**60 日以内** に 都道府県知事 へ 審査請求（行政不服審査法）。
2. 児童 の海外居住 取扱、別居監護 認定、外国籍 児童 の住民登録 状況 等 が争点 になりやすい。
3. 在留資格 / 住民票 / 監護 の関係 が複雑 な場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 児童手当 受給 自体 は 永住・帰化 の **直接 不利益 要因 ではない**。法定 給付 で 公的負担 依存 とは別カテゴリ。
- 受給対象 は 「日本国内 に住所 を有する 監護者 + 児童」。在留カード 所持者 + 住民登録 + 在留期間 3 ヶ月超（短期滞在 除く）が対象。
- **不正受給**（児童 が長期 海外居住 にもかかわらず 申告 せず受給継続）は 全額返還 + 加算金 + **素行不良** 評価 → 永住・更新 不許可 リスク。
- 高度専門職・経営・管理 の世帯 で 子女 が 母国 と日本 を往復 している ケース は、現況届 + 海外居住児 規定 を慎重 に確認。
