# 傷病手当金 支給決定通知 / 伤病津贴 支给决定通知

## 信封长什么样
- 寄件人：健康保険組合（〇〇健康保険組合）/ 全国健康保険協会（協会けんぽ）支部
- 封面常见印字：「傷病手当金 支給決定通知書 在中」「親展」
- 颜色：白底 + 健保組合 ロゴ
- 同梱物：支給決定通知書 1 枚 / 振込明細 / 次回申請 案内（継続受給 用 申請書）

## 这是什么
業務外 の 傷病 により 連続 3 日（待期）+ 4 日目以降 労務不能 で 給与支給 されない期間 に対する 健康保険 法定給付 の 支給決定 通知。標準報酬日額 の 2/3 が 支給日数 分 振込 され、**通算 1 年 6 ヶ月** が上限（2022 年 1 月 改正：暦日 1 年 6 ヶ月 から 通算 へ変更）。業務上 / 通勤災害 は 労災保険 の 休業補償給付 が対象 で、本給付 ではない。

## 为什么收到
- 業務外 の 傷病 で 連続 4 日以上 休業 + 給与 不支給
- 主治医 の 「労務不能」 証明 + 事業主 の 給与不支給 証明 を添付して申請
- 健康保険組合 / 協会けんぽ 審査完了

## 必看的关键栏
- **被保険者番号 / 記号番号**
- **支給対象期間**（自 〇月〇日 至 〇月〇日）
- **支給日数 + 標準報酬日額 + 1 日あたり 支給額**（標準報酬日額 × 2/3）
- **支給金額 合計**
- **通算 支給済日数** + **残日数**（通算 1 年 6 ヶ月 = 約 540 日 上限）
- **振込予定日 + 振込口座**
- **次回申請可能 期間**（継続受給 の場合）

## 不处理的后果（時間梯度）

**支給決定 の場合**：
- 受領 のみで 行動義務 なし
- ただし **継続受給** には 1 ヶ月ごと（又は 給与締日ごと）の 追加申請 が必要 — 主治医 / 事業主 証明 を毎回取得

**不支給 / 一部支給 / 計算誤り の場合**：
- 審査請求 期限：**3 ヶ月以内**（健康保険法 第 189 条）
- 期限経過 で確定、以後 訴訟 でも 覆すのは 困難

**消滅時効**：
- 傷病手当金 請求権 の時効 は **2 年**（健康保険法 第 193 条）。労務不能日 ごと に時効 進行 — 2 年 経過分 は失権

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 継続申請（多数人）
1. 通知書 + 振込明細 を保管。指定口座 に通知額 が振込（通常 通知 から 1-2 週間）。
2. 労務不能 が継続 する場合 → **次回申請書** を 主治医 + 事業主 に依頼。給与締日 / 月末 等 区切ごと に申請する のが一般的。
3. 通算 支給済日数 を毎回 確認。**1 年 6 ヶ月 = 約 540 日** 上限到達後 は 障害厚生年金 / 障害基礎年金 への 切替 を検討。
4. **退職 後 の継続給付**：退職日 まで 連続 1 年以上 の 被保険者期間 + 退職日 当日 が労務不能（出勤不可）状態 → 退職後 も 残日数 受給可能（協会けんぽ / 一部健保組合 のみ）。

### 路径 B：労災 該当性 / 制度 切替
- **業務起因 / 通勤起因** が判明 → 労働基準監督署 へ **労災申請**。労災 認定 されれば 休業補償給付（給付基礎日額 × 60% + 休業特別支給金 20% = 80%）に 切替。傷病手当金 と労災 給付 は 同時 受給不可（労災優先、健保 が支給済 なら 返還）。
- **1 年 6 ヶ月 経過 + 治癒 せず**：障害厚生年金 / 障害基礎年金 へ申請（年金事務所）。同年同 期間 の 並行受給 は 一部調整 あり。
- **退職 + 継続給付 不可** で長期療養 → 国民健康保険 へ切替 + 障害年金申請 + 自立支援医療（精神疾患 等）併用。

### 路径 C：不支給 / 計算 異議
1. 通知 受領 から **3 ヶ月以内** に 社会保険審査官 へ 審査請求（健康保険法 第 189 条）。
2. 主な争点：労務不能 認定範囲 / 待期完成 認定 / 標準報酬日額 計算 / 給与一部支給 との調整 / 業務上 / 業務外 区分。
3. 主治医意見書 と健保 審査医 意見書 が乖離する ケース は 立証 が複雑、社労士 / 行政書士 / 弁護士 の関与 を検討。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 傷病手当金 受給 自体 は 永住・更新 への **直接 不利益 要因 ではない**。健康保険料 拠出 から の 法定給付。
- ただし **療養 期間中 の 在留資格 該当性** は別問題：
  - **技人国 / 経営・管理 / 高度専門職** 等 就労系：3 ヶ月以上 該当活動 不実施 → 在留資格取消 リスク（入管法 第 22-4 条）。傷病 が事由 でも 「正当な理由」 立証 が必要 — 主治医診断書 + 傷病手当金 支給決定通知書 が 重要証拠。
  - **家族滞在**：扶養者 が 傷病療養中 でも 扶養関係 維持されれば 在留資格 影響なし。
- **永住申請** の 直近 5 年間 安定収入 要件 — 傷病手当金 を 「安定収入」 として 算入 するか は ケースバイケース。直近 1 年 の 給与所得 復帰 で 補強 可能。
- 経営・管理 で 役員 が長期療養 → 会社 の 「事業継続性」 評価 に影響。代表者 の 業務代行体制 が 続签材料 で問われる。
