> # ⚠️ 需创始人/書士复核：復興特別所得税 適用期間（2013-2037 年度）+ 防衛特別税 創設で 2027 年度 以降 構成変更可能性；源泉徴収 税率 7.6% / 10.21% / 20.42% 等 区分。

# 復興特別所得税 通知

## 信封长什么样
- 寄件人：「○○税務署」「国税庁」
- 封面印字：単独通知書 として 送付 されることは 稀。所得税 確定申告書 / 納付書（参见文档 #06）/ 源泉徴収票 / 予定納税通知 の 一部 として 「所得税及び復興特別所得税」と 記載
- 信封颜色 / 大小：所得税 関連 信封 と 共通
- 同梱物：所得税 申告書 一式 + 復興特別所得税 = 所得税額 × 2.1% の 加算 計算欄

## 这是什么
復興特別所得税 = 東日本大震災 復興財源確保法（平成 23 年法律第 117 号）に 基づき 2013-2037 年度（25 年間）の 所得税額 に 一律 2.1% を 上乗せ する 国税。
所得税 と 一体 運用 = 別個 通知書 / 別個 納付書 ない。給与天引き / 確定申告 / 源泉徴収 全て に 自動 加算。
源泉徴収 税率 例：給与 ＝ 通常 5-45% に 復興分 含む / 報酬・原稿料 = 10.21%（10% × 1.021）/ 非居住者 = 20.42%（20% × 1.021）/ 退職金 = 通常 + 復興分。
2027 年度 以降 「防衛特別税」（防衛力強化財源 1%）創設予定（取得日时点 法案成立 状況 確認要）。

## 为什么收到
- 給与所得者：源泉徴収票 「源泉徴収税額」欄 = 所得税 + 復興特別所得税 合算済
- 確定申告：申告書 上 「所得税 + 復興特別所得税」一体 計算
- 報酬 / 講演料 / 原稿料 / 配当 / 利子 受領者：支払調書 + 源泉徴収 10.21% / 20.42% に 含 まれる
- 海外居住者（非居住者）への 日本源泉所得 支払 = 20.42% で 源泉徴収

不会収到の人：所得税 課税対象 が 一切 ない 人（年金 + 各種控除で 所得税 ¥0 等）。

## 必看的关键栏
- **税率**：所得税額 × 2.1%（独立税率 ではなく 加算 方式）
- **記載位置**：
  - 源泉徴収票：「源泉徴収税額」欄 に 含 まれる（内訳 別記 されない）
  - 確定申告書：第一表 「再差引所得税額」次行 「復興特別所得税額」欄
  - 納付書：「本税」欄 に 一体記入
- **適用期間**：2013 年 1 月 1 日 - 2037 年 12 月 31 日（25 年間）
- **問合せ先**：管轄 税務署 + 国税庁 タックスアンサー

## 不处理的后果（时间梯度）
復興特別所得税 単独 で 滞納 する ケース は 発生しない（所得税 と 一体）。所得税 滞納 = 復興分 も 自動的 に 滞納 状態。
- **期限後 1 ヶ月内**：期限後申告 自主提出 → 無申告加算税 軽減（5%）
- **税務署指摘後**：無申告加算税 15-20%
- **延滞税**：年 2.4% / 1 ヶ月後 年 8.7% 前後（取得日时点）
- **過少申告 / 隠蔽**：過少申告加算税 10-15% / 重加算税 35-40%
- **未申告 持続**：税務調査 → 7 年遡及 / 差押
- **更长期**：在留期間更新 / 永住 で 公的義務不履行 評価

## 该怎么处理（3 种典型路径）

### 路径 A：給与所得者（多数人）
1. 会社 が 源泉徴収 + 年末調整 で 完了 → 個人 で 何も しない
2. 源泉徴収票 「源泉徴収税額」欄 = 所得税 + 復興特別所得税 合算 = 確認 のみ
3. 副業 / 医療費控除 / 寄付金控除 等で 確定申告 → 申告書 上 自動計算

### 路径 B：自営業 / フリーランス / 確定申告者
1. [国税庁 確定申告書等作成コーナー](https://www.keisan.nta.go.jp/) 利用 → 自動計算（手書きでも 復興分 計算欄 あり）
2. 源泉徴収 された 報酬：支払調書 で 「源泉徴収税額」確認 → 確定申告で 既納付分 として 控除
3. 納付：所得税 と 同時、振替納税 / クレカ / コンビニ / 銀行
4. 過少申告 防止：報酬 受領時 「源泉徴収 ¥X 含」記録 → 確定申告 で 一致確認

### 路径 C：源泉徴収 誤り / 還付申告
触发条件：源泉徴収 過大 / 二重源泉 / 海外居住者 認定誤り。
1. 確定申告 / 修正申告 / 更正の請求（5 年以内）
2. 非居住者 → 居住者 切替 + 復興分 過大徴収 → 還付請求
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士 / 税理士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 在留期間更新 / 永住申請：所得税 と 一体運用 = 「納税証明書（その 1）」に 復興分 含めて 記載。単独 で 評価対象 にはならない。
- 経営・管理 役員：役員報酬 源泉徴収 + 確定申告 で 自動計算。法人税 とは 別建。
- 海外（中国等）への 配当 / 利子 / 使用料 支払 = 非居住者 源泉 20.42%。租税条約 軽減税率 適用 で「租税条約 に 関する 届出書」事前提出 → 軽減税率 直接適用 + 復興分 も 軽減対象。
- 中国 → 日本 講演料 / 原稿料 受領者：日本国内 源泉 場合 20.42% 源泉徴収 → 中国 で 外国税額控除 申告。
- 「2037 年 12 月 31 日 で 廃止」予定 だが、新設 防衛特別税 が 重複 / 入替 / 並走 か は 取得日时点 法令 動向 確認要。
