# 保育園 入園決定通知書 / 保育园 录取通知

## 信封长什么样
- 寄件人：「○○市役所 保育課」「○○区役所 子ども家庭支援課」「○○市 こども未来部」
- 封面印字：「保育所 入所承諾通知書 在中」「教育・保育給付認定 結果通知書 同封」「重要書類」
- 颜色：白底 + 自治体ロゴ 印章。多数自治体 4 月入园时期（1-2 月）使用专用封筒
- 同梱物：入園決定通知書 / 支給認定証（マイナンバーカード 不所持者向け）/ 入園説明会 案内 / 持参物リスト / 保育料 概算 案内 / 慣らし保育 スケジュール 案内

## 这是什么
市役所 / 区役所 保育課 在「利用調整」（自治体 内部审查 各家庭 保育必要度 点数）后，向 入园 决定家庭 发送的承諾通知。子ども・子育て支援法 第 20 条 / 第 24 条 が根拠。同时附带 支給認定証（1 号 / 2 号 / 3 号、保育標準時間 / 保育短時間 区分）— 本通知書 = 该子女 来年度（或 直近）确实可入园 的 唯一 公式确认。

## 为什么收到
- 前年 10-11 月 提出 「保育所 入所申込書」+ 「教育・保育給付認定 申請書」+ 就労証明書 / 診断書 等
- 年度途中 入園希望 提出 → 空き発生 → 利用調整 通过
- 転入後 申込 → 新住所 自治体 で 利用調整 通过
- 育児休業 復帰 連動 申込 → 復職予定月 から 入園決定

不通过的家庭 收到「保留通知書」/「不承諾通知書」— 此为不同文书，本条目 不适用。

## 必看的关键栏
- **入園日（利用開始日）**：通常 4 月 1 日 / 月初。慣らし保育 期間（通常 1-2 週間）期间 短時間 利用
- **入園先 保育園 名称・所在地・連絡先**：第 1 希望 と 異なる場合あり
- **支給認定 区分**：1 号（教育のみ）/ 2 号（3 歳以上 保育）/ 3 号（0-2 歳 保育）+ 標準時間（最大 11 時間）/ 短時間（最大 8 時間）
- **入園説明会 日時・場所**：欠席 即 重大欠手续
- **持参物 提出期限**：児童票 / 健康診断書 / 食物アレルギー 申告書 / 緊急連絡先 / 口座振替 依頼書 等
- **保育料 概算**：本通知書 に 概算記載、確定額 は 別通知（参见文档 #32）
- **問合せ先**：保育課 直通番号 + 保育園 直通番号

## 不处理的后果（时间梯度）
- **入園説明会 欠席**：園 / 自治体 から 個別連絡 → 説明資料 後送、但 持参物 提出期限 短縮 リスク
- **辞退 連絡 が 入園日 直前**：補欠繰上 機会 が 他家庭 に渡らず、自治体内 待機児童 増加 → 翌年度 申込 时 「利用調整 加点」見送り 等 事実上不利益（条例 / 運用 で 明文化 自治体 あり）
- **入園後 連続 欠席（保育必要性 喪失）**：就労実態 喪失 / 育休 復帰せず → 「保育の必要性 認定 取消」(子ども・子育て支援法 第 23 条) → 退園 命令
- **支給認定 内容 と 実態 が 異なる**（短時間 認定 で 標準時間 利用 等）→ 認定変更 申請 が 必要

> # ⚠️ 需创始人/書士复核：辞退 连络 给来年度 申込 加点 影响 程度 — 自治体 条例 / 運用 大きく 異なる、明文化 自治体 と 内部運用 自治体 混在。

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 入園説明会 → 入園
1. 入園説明会 出席（夫婦 どちらか 1 名 でも 可、多数 自治体）
2. 持参物 リスト 確認 → 期限内 提出（児童票 / 健康診断書 等）
3. 保育料 口座振替 依頼書 を 金融機関 経由 で 提出 → 翌月 から 自動引落
4. 慣らし保育 スケジュール（1 日 → 半日 → 終日 を 1-2 週間 で段階的）に従い 通園開始
5. 通知書 / 支給認定証 を 5 年保管（保育料 軽減 / 児童手当 / 永住申請 時 補強資料）

### 路径 B：辞退 / 内定保留
触发条件：転居 / 仕事キャンセル / 認可外 保育園 確保 等。
1. 速やかに 保育課 に 電話連絡 → 「入園辞退届」を 提出（多数 自治体 入園日 1 週間前 まで）
2. 翌年度 再申込 时、辞退理由 を 申込書 に記載（「やむを得ない事情」 認定 されれば 加点 維持）
3. 認可外 / 企業主導型 への 移行 時 は 保育料 軽減（幼児教育・保育の無償化）対象 確認

### 路径 C：認定変更 / 転園 申請
触发条件：就労時間 変更（短時間 → 標準時間）/ 転居 / 兄弟同園希望 等。
1. 「教育・保育給付認定 変更申請書」+ 就労証明書（変更後）を 保育課 に 提出
2. 転園希望 は 利用調整 を 再度受ける（待機児童 多 自治体 では 数ヶ月-1 年 待ち）
3. 入管法 関連争議（在留資格 喪失 / 子女 在留資格 変更 等）を伴う場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 教育系 文书直接影响 在留資格 审查 弱
- 但 保育園 入園 = 子女 監護・養育 体制 の証拠 → 永住申請（独立生計要件）/ 家族滞在 子女養育 実績 の補強資料 として 提出可能
- 経営・管理 役員 の子女 が 認可保育園 入園 → 「事業の継続性 + 家族の定住意思」評価 上 プラス
- 保育料 滞納 は 在留審査 に 通常 直接影響 なし（市町村 内部 督促 で 完結）— ただし 市町村 が「公的義務不履行」 として 課税証明書 / 納税証明書 取得 時 言及 する ケース 自治体 あり
- 子女 の 在留資格 が 「家族滞在」 の 場合、保育園 利用 自体 は 在留資格 制限なし（資格外活動 該当 せず）
