# 追加資料提出依頼 / 補正通知

## 信封长什么样
- 寄件人：審査担当の地方入管局
- 形式：茶色 葉書 / 定形封筒、封面「資料提出通知書」「追加資料提出のお願い」
- 同梱物：受付番号、提出期限（通常 14 日 ～ 1 ヶ月）、提出先窗口 / 郵送先、担当審査官、内線、必要書類リスト
- 提出方法：窗口持参 / 郵送（書留推奨）/ オンライン申請 の場合は マイページ アップロード

## 这是什么
在留期間更新 / 在留資格変更 / 永住 / COE 申請 の審査過程で、入管 が **追加資料 / 補正書類** を要求する書面。提出期限内に出さないと **審査打切 = 不許可** になる構造。

## 为什么收到
申請書類の不備、内容説明不足、最新情報の未反映、入管 側の追加確認事項が発生したため。典型例：
- **課税証明書 / 納税証明書** の追加（永住・経営・管理 で複数年分）
- **雇用契約書 の最新版 / 給与明細 直近 3 ヶ月分**
- **登記事項証明書 / 決算書 / 事業計画書**（経営・管理）
- **学歴 詳細（卒業証明書 / 成績証明書 / 学位記）**
- **配偶者関係 の追加（戸籍 / 婚姻届出受理証明書 / 同居実態 写真）**
- **理由書 補正 / 質問への回答書**

## 必看的关键栏
- **提出期限**：通常 14 日 ～ 1 ヶ月、明記される
- **必要書類リスト**：番号 / 種類 / 部数（写し可 / 原本要 区分）
- **提出方法 / 提出先**：窗口 / 郵送 / オンライン
- **担当審査官 / 内線**：書類選定で疑問あれば事前電話確認可
- **「期限までに提出なき場合 不許可」** の文言（標準書式）

## 不处理的后果（時間梯度）
- **提出期限 超過**：審査打切 → **不許可** 確定（入管 標準運用）
- **不許可後 元 在留資格 期限内**：再申請 可能だが、不許可履歴 残る
- **不許可後 元 在留資格 期限切れ**：「特定活動（出国準備）30 日 / 31 日」/ オーバーステイ リスク
- **資料一部未提出 / 不完全提出**：追加催促なしで 不許可 へ進むケースあり

> 入管法 第 7 条 / 第 20 条 / 第 21 条 / 第 22 条 各申請の審査原則、行政手続法 第 7 条（標準処理期間）

## 该怎么处理（3 种典型路径）

### 路径 A：期限内 提出（標準）
1. 必要書類リストを 1 件ずつ確認。原本 / 写し / 部数 / 取得日 の制限に注意（住民票・課税証明書 は通常 3 ヶ月以内）。
2. 取得元：
   - 課税証明書 / 納税証明書：市役所 / 区役所
   - 登記事項証明書：法務局
   - 戸籍謄本：本籍地 市役所
   - 雇用契約書 / 給与明細 / 在勤証明書：雇用主
3. 提出方法を遵守（窗口 / 郵送 / オンライン）、書留 + 配達記録 で送付推奨。
4. 提出後、申請受理票 + 提出記録 を档案归档。
5. 担当審査官 内線 へ「提出済み」を電話確認（任意だが推奨）。

### 路径 B：期限内取得困難 → 期限延長 / 部分提出
1. 担当審査官 へ早期に電話 → 取得困難な事情 + 取得見込日 を伝達。
2. 取得困難の合理的理由（海外取り寄せ・原本待ち）があれば **期限延長** 認められるケースあり、ただし書面回答は通常なし。
3. 取れる書類は **期限内に部分提出**、残りは「追って提出」と添え状で明記。

### 路径 C：弁護士 / 行政書士 緊急対応
- 補正内容 が **構造的問題**（学歴・職歴 と 職務 不一致 / 事業計画 弱 / 婚姻実態 疑義）を指している
- 期限まで 1 週間未満 + 取得困難書類 多数
- 過去 不許可履歴 + 今回も補正要求（審査心証悪化サイン）
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **補正対応 履歴** = 入管 心証形成の重要素材。期限内 + 完全提出 = 「協力的・適格」評価
- **補正後 不許可** = 構造的問題の典型サイン、再申請 戦略を抜本見直し必要
- **永住申請** では「課税証明書 5 年分」等の補正要求が頻発、すべて市役所 で取得（過去年度 は本籍 / 旧住所地）
- **経営・管理** では「直近 3 期 決算書 / 月次試算表 / 取引先 リスト / 事業計画書 修正」等、書類量が多い → 期限管理が肝
- **オンライン申請** の場合 マイページ メッセージ で補正依頼が来る → 紙通知より見落としリスクあり、メール通知設定 必須

> # ⚠️ 需创始人/書士复核：補正期限 経過後の「審査打切」運用と、行政手続法 第 7 条 標準処理期間 との整合（補正期間中の処理期間 取扱）
