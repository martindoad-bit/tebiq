# 粗大ゴミ 回収申込確認 / 領収書

## 信封长什么样
- 寄件人：市役所 環境課 / 清掃事務所 / 粗大ごみ受付センター（自治体名 + 部署名）
- 封面印字：「粗大ごみ収集 申込受付確認書」「粗大ごみ処理手数料納付済証」「処理券」
- 同梱物：
  - 受付確認書（受付番号 / 収集日 / 排出場所）
  - 粗大ごみ処理手数料納付券（シール / 券）or その購入案内
  - 粗大ごみ排出ルール案内（出し方 / 朝何時まで / 屋外排出場所）
- メール / Web 申込の場合は紙書類なし（電子受付確認のみ）

## 这是什么
自治体が **廃棄物処理法（一般廃棄物）** + 自治体条例 に基づき提供する 粗大ごみ 戸別収集 サービスの **申込受付確認** + **処理手数料納付の領収書（納付済証）**。督促書ではなく、本人申込後の確認書。

## 为什么收到
- 自分（or 家族）が 粗大ごみ受付センター に **電話 / Web / LINE** で申込み、収集日・手数料 が確定した後の確認書
- **コンビニ等で粗大ごみ処理券（シール）購入**後の領収書

## 必看的关键栏
- **受付番号**：問合せ・キャンセル時必須
- **収集日**：指定日 当日朝（多くは 8:00-9:00）までに屋外排出
- **排出場所**：玄関前 / 集合住宅指定場所 / 道路境界
- **品目別 処理手数料**：自治体で異なる
  - 例（東京 23 区一例）：椅子 ¥400 / ソファ ¥1,200-¥2,000 / 冷蔵庫 ¥3,000 程度（2026 年現行料金は自治体公式 HP で要確認）
- **処理券（シール）貼付方法**：品目に氏名 or 受付番号 を記入し貼付
- **回収不可品**：家電リサイクル法 4 品目（エアコン / TV / 冷蔵庫 / 洗濯機）+ パソコン + バイク + 危険物 等は別ルート

> ⚠️ 自治体毎に料金・運用が大きく異なる。本書面の自治体公式 HP を確認。

## 不处理的后果（時間梯度）
- **収集日 排出忘れ**：再申込（手数料再払）必要、または 自治体 指定の持込処分施設 に自己搬入
- **処理券貼付忘れ**：未収集 → 玄関前に「貼付不備」紙が貼られ持ち戻り
- **無断不法投棄**（道路 / 公園 / 山林）：**廃棄物処理法 第 25 条 違反 → 5 年以下の懲役 or 1,000 万円以下の罰金（個人）/ 3 億円以下の罰金（法人）**
- **家電リサイクル法 4 品目 を粗大ごみとして出す**：違法、不法投棄扱いで指導 / 罰則対象
- **集合住宅 ルール違反**：管理会社 → 大家 経由で警告書、繰返しで賃貸借契約 信頼関係破壊評価

## 该怎么处理（3 种典型路径）

### 路径 A：通常通り排出（受付確認書通り）
1. 受付確認書記載の品目に **粗大ごみ処理券（シール）** を貼付（氏名 or 受付番号 記入）。
2. 収集日 当日朝の指定時刻までに、指定排出場所に出す。雨天決行が一般的。
3. 受付確認書 / 領収書を保管（年度内）。

### 路径 B：キャンセル / 変更 / 追加
1. 粗大ごみ受付センター に電話 or Web マイページで **キャンセル / 日程変更 / 品目追加**。
2. 多くの自治体では 収集日 2-3 日前まで変更可。
3. キャンセル時の手数料返金条件は自治体毎（処理券は概ね返金不可、未使用なら他品目に流用可）。

### 路径 C：家電リサイクル / 大量廃棄 / 引っ越し
- **家電 4 品目**：[家電リサイクル券センター](https://www.rkc.aeha.or.jp/) で家電リサイクル券購入 + 指定取引場所へ自己搬入 or 購入店回収依頼。リサイクル料金（¥1,000-¥6,000 程度）+ 収集運搬料金。
- **大量廃棄（引っ越し時）**：粗大ごみ受付の点数制限超過 → 自治体 持込処分施設 自己搬入（事前予約必要、料金は kg 単位）or 一般廃棄物収集運搬業 許可業者へ依頼。
- **不用品買取・無料回収業者**：「無料回収」を謳う無許可業者は 廃棄物処理法 違反、後日不法投棄被害で住民が責任を問われる例あり、必ず 自治体許可業者 を選ぶ。
- 大量廃棄・トラブル化の場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 粗大ごみ 申込・処理手数料は 出入国在留管理庁 の **直接審査項目ではない**
- 不法投棄等の **罰金 / 前科** に発展した場合のみ、永住「素行善良」評価に影響
- 経営・管理：事業所からの産業廃棄物 / 一般廃棄物処理は別法令（廃棄物処理法 産業廃棄物 + 委託契約・マニフェスト制度）→ 違反は事業継続性に影響

> ⚠️ 永住「素行善良」運用は事案毎、罰金・前科 を伴う場合のみ実務的に問われる。
