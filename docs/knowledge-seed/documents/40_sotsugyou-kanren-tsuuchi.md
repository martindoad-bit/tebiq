# 卒業関連 通知 / 毕业相关 通知

## 信封长什么样
- 寄件人：「○○小学校 校長」「○○中学校 学年主任」「○○高等学校 進路指導部」
- 封面印字：「卒業式 のご案内 在中」「卒業関係書類」「進学先連絡」
- 颜色：白色 / 浅黄色 + 学校ロゴ、卒業期 専用 封筒（金箔 / 校章 印 etc）
- 同梱物：卒業式 案内 / 卒業証書 受領手続 / 卒業アルバム 案内 / 進路調査票 / 指導要録 写し（進学先要請時）/ 教科書 / 学用品 持帰 案内 / 通学定期券 解約 案内

## 这是什么
学校教育法 施行規則 第 58 条（小学校 卒業）/ 第 79 条（中学校 卒業）/ 第 96 条（高校 卒業）に基づく 課程修了 認定 + 卒業証書 授与 の 一連手続。同時並行 で：(1) 進路調査（進学先 / 就職先 連絡）、(2) 指導要録 抄本 / 写し の 進学先 送付（学校教育法 施行規則 第 24 条 = 5 年保存、入学者 名簿 = 20 年保存）、(3) 学用品 / 教科書 持帰、(4) 通学定期 解約 / 学割証 失効 案内 が 集約。

## 为什么收到
- 卒業 学年（小 6 / 中 3 / 高 3）の 12 月-3 月 → 卒業式 / 進路調査 案内
- 進路 確定後（中 3 高校 受験 合否 / 高 3 大学 進学 / 就職 内定）→ 進学先 連絡 + 指導要録 送付 案内
- 卒業式 後：卒業証書 / 修了証書 / 卒業アルバム 受領 / 同窓会 案内
- 転出 / 中退 / 留学 等 課程修了 以外 の 終了 → 「転学・退学 通知」（本書類 と 別形式）

## 必看的关键栏
- **卒業式 日時 / 場所**（多数 3 月 中下旬）
- **卒業証書 受領方法**（卒業式 当日 校長 直接交付 が 標準、欠席 case は 後日 学校事務 受領）
- **進路調査票 提出期限**（中 3 = 高校受験 合否 / 高 3 = 進学・就職先 確定後 1-2 週間以内）
- **指導要録 写し 送付 同意**（進学先 高校・大学 から 要請 された場合）
- **卒業アルバム 受領 / 配送方法**
- **通学定期 解約 / 学割 失効 期限**
- **問合せ先**：学年主任 / 進路指導部 / 学校事務室

## 不处理的后果（时间梯度）
- **卒業式 欠席 + 連絡なし**：卒業証書 後日 受領可、但 学校事務 と 個別調整 必要
- **進路調査票 未提出**：学校 進路統計 / 卒業後追跡調査 から 漏れ → 学校 が 進学先 把握不能 → 後年 在学証明書 / 卒業証明書 取寄 時 学校 と 進学先 連携 困難
- **指導要録 写し 送付 同意 漏れ**：進学先 高校 / 大学 が 評定資料 を 受領できず → 進学先 で 別途 提出書類 要求
- **通学定期 解約 遅延**：4 月以降 も 学生扱い で 不正使用 → 鉄道会社 から 是正請求 リスク
- **卒業証書 紛失**：再発行 不可（原本 1 通限り）→ 「卒業証明書」（再発行可、有料 ¥300-¥500）で代替

## 该怎么处理（3 种典型路径）

### 路径 A：卒業式 出席 → 進路調査 → 卒業証書 受領
1. 卒業式 出席（保護者 通常 出席可）→ 卒業証書 / 修了証書 受領
2. 進路調査票 を 期限内 提出（受験結果 / 内定先 確定後）
3. 通学定期 解約 / 学割証 返却（中 3 → 高校 進学 で 通学定期 切替 必要 case 多）
4. 教科書 / 学用品 持帰 → 自宅 / 進学先 で 継続使用 / 廃棄
5. 卒業証書 / 卒業証明書 / 指導要録 写し を **永久保管**（進学 / 就職 / 海外留学 / 帰化 / 永住申請 / 学歴照会 で 必要）

### 路径 B：進学先 / 就職先 への 書類送付
- **国内 高校 / 大学 進学**：指導要録 抄本 / 調査書 を 学校 が 直接 進学先 に 送付（保護者 同意 必要）
- **海外 大学 / 高校 進学**：英文 卒業証明書 / 成績証明書（英訳）を 申請 → 学校 / 教育委員会 経由 発行（手数料 ¥500-¥2,000、発行 1-2 週間）
- **就職先 への 提出**：卒業見込み 証明書（高 3 = 9-10 月）+ 卒業証明書（卒業後）
- **進学先 在留資格 関連**：留学 → 在留資格 「留学」 取得 / 子女 「家族滞在」継続 → 子女 在留資格 関連書類 で 卒業証明書 が 必要

### 路径 C：海外渡航 / 帰国 / 中退 / 留学 等 特殊事情
1. 卒業前 海外帰国 → 「除籍 / 中退」扱い、課程未修了 で 卒業証書 不交付。但 「在学期間 証明書」「修了科目 単位証明書」発行 可能
2. 留学（高校 / 大学 編入留学）→ 「休学届」「在学届」維持 → 復学 → 卒業 ルート、または 「転学届」→ 留学先 に 学籍 移動
3. 海外 大学 進学 で 日本国内 高校 卒業証明書 が 留学先 提出書類 になる場合 多 → 早期 申請 推奨
4. 子女 国籍 / 在留資格 / 入管 関連 紛争（在留資格 喪失 / 帰化申請 / 留学先 ビザ 不許可 等）伴う case、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 卒業証書 / 卒業証明書 = 学歴 客観証拠、子女 の 後年 在留資格変更（留学 → 技人国 → 経営・管理 等）の 必須書類
- 子女 中学卒業 → 高校進学（在留資格 「家族滞在」継続）/ 子女 高校卒業 → 大学進学（在留資格 「留学」変更 / 「家族滞在」継続）の どちら ルート で 進む か 慎重 計画
- 永住申請 / 帰化申請 で 子女 学歴 自体 は 直接審査項目 ではない、但 「家族同居 + 在日生活 定着 + 子女 教育 完遂」 = 評価上 プラス
- 経営・管理 役員 子女 が 海外大学 進学 → 海外居住期間中 子女 在留資格 維持 困難（家族滞在 / 永住者の配偶者等 は 1 年以上 海外滞在 で 失効リスク）→ 渡航前 入管 確認推奨
- 帰化申請 子女 同伴 case で 在学中 / 卒業後 タイミング を 戦略的 計画（学校 / 在留資格 + 帰化 同期）

教育系 文書 だが 在留資格 / 帰化 連動 影響 が 集中 する のが 卒業期。卒業証書 / 卒業証明書 / 指導要録 写し は 永久保管 が 鉄則。
