# 給食費 納付書 / 学校午餐费 缴纳通知

## 信封长什么样
- 寄件人：「○○小学校」「○○中学校」「○○市 学校給食センター」「○○市 教育委員会 給食係」
- 封面印字：「給食費 納付書 在中」「学校給食費 払込のお願い」「重要」
- 颜色：白色 長 3 号 / クラフト封筒、ロゴ や 印章 比較的少
- 同梱物：納付書 / 払込書 / 口座振替 依頼書 / 月別 給食予定表 / 食物アレルギー 申告書（初回 のみ）

## 这是什么
学校給食法 第 11 条 に基づき 学校給食 の 食材費（食材実費）を 保護者 が 負担 する 制度。給食費 = 食材費 のみ（人件費 / 施設費 は 自治体 / 国庫 負担）。徴収主体 は 自治体 に より 異なる：(A) 学校徴収（口座振替 / 納付書）、(B) 公会計化（市町村 直接徴収、税金 と 同様 取扱）。多数 自治体 で 公会計化 が 進行中（文科省 推進方針 2019-）。

## 为什么收到
- 入学時：年間 給食費 案内 + 口座振替 依頼書 同送（毎月 自動引落 設定 する 流れ）
- 口座振替 設定 なし / 残高不足 / 引落不能 → 翌月 納付書 郵送
- 公会計化 自治体：年 4-12 期 分割 で 納付書 郵送（住民税 と 同様 形式）
- 食材費 改定 時（年度替 / 物価変動 反映 時）→ 改定通知 + 新 納付書

## 必看的关键栏
- **金額**（月額 / 期分 / 年額）
- **納期限**（月次 自動引落 = 月末 / 納付書 = 翌月 末日 多数）
- **納付方法**：口座振替（推奨）/ コンビニ（バーコード 付）/ 銀行 / 郵便局 窓口 / Pay-easy / 自治体 指定電子決済
- **食物アレルギー対応**（除去食 / 代替食 / 弁当持参）— 別書類 だが 同梱 多
- **欠席時 返金**：連続 5 日 / 1 ヶ月以上 欠席 で 食材費 返金（学校 / 自治体 規程）
- **問合せ先**：学校 事務室 / 給食センター / 教育委員会 給食係

## 不处理的后果（时间梯度）
- **口座振替 引落不能**：翌月 納付書 郵送 + 督促連絡（学校事務 / 担任 経由）
- **2-3 ヶ月 滞納**：学校 / 給食センター から 督促状 + 個別電話連絡
- **6 ヶ月以上 滞納**：(A) 学校徴収 自治体 → 学校長 / PTA 経由 督促（実務上 法的強制力 弱）/ (B) 公会計化 自治体 → 地方自治法 第 231 条の 3 に基づき 強制徴収（給与 / 預金 差押 可能）
- **長期 滞納 + 連絡無視**：児童虐待 / ネグレクト 疑義 case では 児童相談所 連携（極端 case のみ）

実務上 給食 が 提供されない / 退学 等 の 直接ペナルティ なし — 子女 に 影響 を及ぼさない 運用。

## 该怎么处理（3 种典型路径）

### 路径 A：口座振替 設定 → 月次 自動引落
1. 入学時 同梱 口座振替 依頼書 → 指定 金融機関 に 提出 → 翌月 / 翌々月 から 自動引落 開始
2. 残高不足 防止：給食費 専用 普通預金 口座 推奨、児童手当 振込口座 と 同一 にすると 残高不足 リスク 低
3. 引落明細 を 5 年保管（年末調整 / 確定申告 で 控除対象 にはならない、就学援助 申請 時 補強資料）

### 路径 B：就学援助 認定 → 給食費 免除 / 給付
触发条件：世帯所得 が 就学援助 認定基準 以下（参见文档 #34）。
1. 就学援助 申請 → 認定 → 給食費 給付項目 として 自治体 が 直接 学校 / 給食センター に 支払 → 保護者 負担 ¥0
2. 一部 自治体 は 第 3 子以降 / 多子世帯 給食費 無償化 制度 あり（独自施策、年々 拡大傾向）
3. 給食費 完全無償化 自治体 増加中（2024-2025 年度、多数 市町村 で 段階的 導入）

### 路径 C：分納 / 一時 支払猶予
触发条件：失業 / 病気 / 災害 等 一時的 困窮。
1. 学校事務 / 給食係 に 電話連絡 → 「分納誓約書」/ 「支払計画書」を 提出
2. 月割 / 隔月 / 学期末 一括 等 個別計画 締結
3. 同時に 就学援助 申請 を 検討（基準該当 すれば 翌月 から 給食費 ¥0）

教育系 文書 で 行政書士 / 弁護士 関与 は 通常 不要。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 給食費 公会計化 自治体 では 「公的料金」位置づけ → 滞納 = 公的義務不履行 評価 リスク（住民税 / 国保 / 年金 と 同様 扱い 自治体 あり）
- 学校徴収 自治体 では 「私的契約 類似」位置づけ → 滞納 が 直接 在留審査 に 影響 する 可能性 低
- 永住申請 / 帰化申請 で 給食費 滞納 が 直接審査項目 になった 公開事例 は ない（2026 年時点）
- 但 自治体 の 課税証明書 / 納税証明書 取得 時 「給食費 未納 あり」 と 記載 される 公会計化 自治体 では 入管 が 認識 する 可能性 あり
- 経営・管理 / 高度専門職 高所得層 は そもそも 滞納 リスク 低、無償化 自治体 では 全員 ¥0 負担

> # ⚠️ 需创始人/書士复核：給食費 公会計化 自治体 における 滞納 が 永住 「公的義務履行」 評価 で 住民税・国保 と 同等扱い か — 自治体 運用 / 入管 認識 の 両方 で 揺れ。
