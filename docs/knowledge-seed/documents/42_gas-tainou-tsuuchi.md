# ガス料金 滞納督促 / ガス供給停止予告

## 信封长什么样
- 寄件人：「東京ガス株式会社」「大阪ガス株式会社」「東邦ガス」「西部ガス」「ニチガス」/ プロパン（LP ガス）の場合は地域販売店名
- 封面印字：「ガス料金お支払いのお願い」「ガス供給停止予告通知書」「重要」赤帯 / 黄帯
- 同梱物：払込取扱票（コンビニ用バーコード）、未払明細、供給停止予定日案内
- LP ガス：販売店訪問時の手書き「最終通告」紙が入る場合あり

## 这是什么
都市ガス（東京ガス / 大阪ガス 等）または LP ガス（プロパン）販売事業者が、契約者に ガス料金 未払の支払を求める督促。最終段階は ガス供給停止 + 契約解除予告。ガス事業法 + 各社託送供給等約款 / ガス供給約款 に基づく。

## 为什么收到
ガス料金 の支払期日（通常 検針日から 30 日）経過後、入金未確認のため。標準的督促タイムライン：
- 期日経過 約 20 日 → 督促状（最初）
- 約 30-50 日 → 供給停止予告（赤帯）
- 約 50-60 日 → 供給停止 + 閉栓
- LP ガスは販売店判断で短縮されるケースあり（10-30 日で停止予告）

## 必看的关键栏
- **未払金額**：当月 + 過去未払分（合計欄）
- **支払期限**：太字記載
- **延滞利息**：年 10% 程度（約款記載）が日割加算
- **供給停止予定日**：赤字
- **払込方法**：コンビニ払込取扱票 / 口座振替再振替 / クレジット / Web
- **問合せ先**：東京ガス お客さまセンター 0570-002211 / 大阪ガス 0120-000-555 等
- **お客さま番号 / 供給地点特定番号 / 検針番号**

## 不处理的后果（時間梯度）
- **期日経過 20 日**：督促状到着、延滞利息加算
- **30-50 日**：「供給停止予告通知」発送
- **50-60 日**：閉栓・ガスメーター封印、開栓には全額支払 + 再開手数料（¥3,000-¥5,000 程度、各社で異なる）
- **契約解除後**：未払額一括請求、民事訴訟 / 簡易裁判所 支払督促へ
- **賃貸物件 退去時**：未払額が 敷金 から控除、または保証会社請求
- **LP ガス**：閉栓 + 容器（ボンベ）撤去、新規開栓には保証金 / 工事費 必要

> ⚠️ 供給停止までの猶予期間・延滞利息率は会社・約款で差異あり。請求書記載の約款条文を要確認。

## 该怎么处理（3 种典型路径）

### 路径 A：全額支払で供給停止回避
1. 督促状同封の 払込取扱票 で コンビニ / 銀行 / Web 支払。
2. 支払 → 反映には 1-2 営業日。停止予告日 直前ならお客さまセンターに支払完了を電話で伝達。
3. 領収書保管、口座振替戻入の再振替日確認。

### 路径 B：分割払 / 支払猶予 申請
1. 請求書記載のお客さまセンターに連絡し「支払困難」を申告。
2. 分割払 / 期日延長 を相談、失業 / 病気等の事情があれば申し出る。
3. 合意内容を書面 or メールで保存。
4. 自治体 生活困窮者自立支援相談窓口（厚生労働省）併用可。

### 路径 C：閉栓後 / 名義違い / 訴訟予告
- 既に閉栓 → 全額支払 + 再開申込 + 開栓立会（在宅必要）。再開作業は申込から 1-3 日。
- 入居前の旧入居者宛督促 → 賃貸借契約書 写しで「契約者違い」回答。
- 簡易裁判所 支払督促 到着 → 2 週間以内に 異議申立 しなければ仮執行宣言 → 強制執行。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
ガス料金 滞納自体は 出入国在留管理庁 の直接審査項目ではない。**ただし**：
- 永住申請「素行善良」要件 → 公租公課（税・年金・健康保険）が中心、ガス料金 単独で不利益となる事例は通常なし
- **訴訟・強制執行 履歴に発展**した場合のみ、信用情報・賃貸更新審査に間接影響
- 経営・管理 では事業所宛 ガス契約 滞納 → 事業の継続性 説明資料に影響する可能性

> ⚠️ 永住「素行善良」運用は事案毎、訴訟記録 を伴う場合のみ実務的影響。
