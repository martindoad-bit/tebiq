# 国民年金 免除申請 結果通知 / 国民年金 免除申请 结果通知

## 信封长什么样
- 寄件人：日本年金機構（〇〇年金事務所 / 中央年金センター）
- 封面常见印字：「国民年金 関係 重要書類」「日本年金機構 在中」「親展」
- 颜色：白底 + 日本年金機構 オレンジ系 LOGO
- 同梱物：結果通知書 1 枚 / 追納制度 案内 / 過去分 納付書（一部 / 半額 等 部分免除 の場合）

## 这是什么
国民年金 第 1 号被保険者（自営業 / 無職 / 学生 / 経営・管理 で 厚生年金 非加入 等）が 提出した「国民年金保険料 免除・納付猶予 申請書」に対する 日本年金機構 の審査結果。結果は 「全額免除」「4 分の 3 免除」「半額免除」「4 分の 1 免除」「納付猶予」「学生納付特例」「却下」 のいずれか で通知される。

## 为什么收到
- 過去 に 免除申請書 / 納付猶予申請書 / 学生納付特例申請書 を提出
- 年度更新（毎年 7 月 切替）で 継続申請 が再審査
- 申請対象期間：申請月 の 2 年 1 ヶ月前 まで 遡及可能
- 失業 / 廃業 / 災害 等 特例事由 を申請

## 必看的关键栏
- **基礎年金番号**（10 桁）
- **承認区分**（全額免除 / 4 分の 3 免除 / 半額免除 / 4 分の 1 免除 / 納付猶予 / 学生納付特例 / 却下）
- **承認期間**（〇年 7 月 〜 〇年 6 月、原則 1 年 単位）
- **却下理由**（却下 の場合：本人所得 / 配偶者所得 / 世帯主所得 が基準超過 等）
- **追納可能 期限**（承認月 から 10 年以内）
- **問合せ先**（管轄 年金事務所）

## 不处理的后果（時間梯度）
- **却下 の場合 + 放置**：未納期間 として記録、督促 → 強制徴収（差押）リスク
- **部分免除（4 分の 3 / 半額 / 4 分の 1）+ 残額 未納**：免除期間 全体 が「未納」 扱い に降格 → 老齢基礎年金 算定上 反映 されない
- **追納 しない場合**（10 年以内 追納可能）：将来の 老齢基礎年金 額 が 減額（全額免除 期間 は 国庫負担分 1/2 のみ反映、納付期間 と比較して 半額）
- **障害基礎年金 / 遺族基礎年金 の保険料納付要件**：免除・猶予期間 は **「保険料納付済期間 と同等」 に算入**（未納 とは違う）
- **永住・帰化 評価**：免除承認期間 は「公的義務 不履行」 にならない。却下 + 未納 のみ問題

## 该怎么处理（3 种典型路径）

### 路径 A：承認 → 維持（多数人）
1. 結果通知書 を保管（5 年）。年金記録 ねんきんネット で「免除月」反映 を確認。
2. 経済状況 改善 後、追納 を検討。**承認 から 10 年以内** なら追納可能、過去 3 年度 を超える 部分 は **加算金** が上乗せ。
3. 翌年度 も継続希望 の場合 → 全額免除 / 納付猶予 承認者 は **継続申請 ✓ 欄** チェック で自動更新可。部分免除 は 毎年 申請必要。

### 路径 B：却下 → 再申請 / 納付計画
- **本人所得 基準超過** で却下 → 翌年度 所得状況 が変われば 再申請可。
- **配偶者所得 / 世帯主所得 で却下**：世帯分離 を検討。住民票 上 別世帯化 で再申請可能（実態判断 あり）。
- **失業 特例**：離職票 / 雇用保険 受給資格者証 添付 で 本人所得 を 0 とみなす特例 ─ 却下後 でも 失業特例 で再申請可（離職日 翌月 から 翌々年度 末まで）。
- 一括納付 困難 → 年金事務所 で **分割納付 相談**（口座振替 月払 / 半年払 / 1 年前納割引）。

### 路径 C：審査請求（不服申立）
1. 結果通知 受領 から **3 ヶ月以内** に **社会保険審査官** へ 審査請求（国民年金法 第 101 条）。
2. 主な争点：所得計算 誤り / 失業特例 適用漏れ / 災害特例 適用漏れ / 配偶者所得 認定誤り。
3. 審査請求書 の書式 と 立証資料 が複雑な場合、社労士 又は 行政書士 へ相談 を推奨。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 国民年金 加入 + 保険料納付（免除承認期間 含む）は 永住申請 ガイドライン の **「公的義務 履行」評価** の 中核項目。
- 直近 2 年（永住）/ 5 年（高度専門職 経由 永住）の年金 加入歴・納付歴 が **「ねんきんネット」スクリーンショット** + **被保険者記録照会回答票**（年金事務所 発行）で 立証 される。
- **免除承認期間** は 「未納」 ではない。審査上 「経済的合理性 ある 公的義務 履行 形態」 として 容認 される。
- **却下 + 未納** が 直近 2 年 に 1 ヶ月 でもあれば 不許可 リスク高。
- 経営・管理 で会社が 厚生年金 未加入 の状態 が長く続くと「事業継続性」評価 にも影響 — 会社設立後 速やかに 社会保険 加入 が原則。
