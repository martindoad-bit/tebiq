# 生活保護 申請結果通知 / 生活保护 申请结果通知

## 信封长什么样
- 寄件人：福祉事務所（市役所 福祉課 / 生活福祉課 / 区役所 保健福祉センター）
- 封面常见印字：「決定通知書」「親展」「重要書類」
- 颜色：白底 + 自治体 LOGO
- 同梱物：決定通知書 1 枚 / 保護費 計算書 / 受給中 の権利義務 案内 / 訪問調査 案内 / ケースワーカー 連絡先

## 这是什么
生活保護 申請 に対する 福祉事務所長 の **支給開始 / 却下 / 一部支給** 決定通知。生活保護法 第 24 条 に基づき、申請日 から **原則 14 日以内**（最大 30 日）に書面 通知される。通知書 には 保護種類（生活扶助・住宅扶助・教育扶助・医療扶助・介護扶助・出産扶助・生業扶助・葬祭扶助 8 種類）と 月額 が記載される。

## 为什么收到
- 福祉事務所 で 生活保護 申請書 を提出済
- 預貯金・収入・親族 扶養可否・稼働能力 等 の調査完了
- 申請日 から 14-30 日経過

> # ⚠️ 需创始人/書士复核：在留外国人 の生活保護 受給資格 は 法律上 「権利」 ではなく 「準用措置」（昭和 29 年 厚生省通知）に基づく。永住者 / 定住者 / 日本人配偶者等 / 永住者の配偶者等 / 認定難民 が対象。技人国 / 経営・管理 / 留学 等 就労・活動系 在留資格 は対象外。最新 取扱 は管轄 福祉事務所 で確認推奨。

## 必看的关键栏
- **決定区分**（保護開始 / 却下 / 廃止 / 変更）
- **保護種類 + 月額**（生活扶助 / 住宅扶助 / 医療扶助 等）
- **支給開始日**（申請日 に遡及 が原則）
- **却下理由**（資産超過 / 稼働能力 あり / 扶養 可能 親族 あり / 在留資格 対象外 等）
- **受給中 の義務**（収入申告 / 資産変動届 / ケースワーカー 訪問受入）
- **不服申立 期限**（3 ヶ月以内、都道府県知事 宛）

## 不处理的后果（時間梯度）

**保護開始 決定 の場合**：
- 受給開始 後 **収入 / 資産変動 の 報告義務**（生活保護法 第 61 条）→ 違反 で返還請求 + 不正受給認定 リスク
- 不正受給 → 全額返還 + 100% 加算（同法 第 78 条）+ 詐欺罪 告発リスク

**却下 決定 の場合**：
- **3 ヶ月以内** に審査請求 しなければ 確定
- 別事由 で再申請 は 即日 可能（前回 却下後 状況変化 ある場合）

## 该怎么处理（3 种典型路径）

### 路径 A：保護開始 → 受給中 義務履行
1. 決定通知書 を保管。月初 に保護費 振込（自治体 により 銀行振込 / 福祉事務所 窓口受領）。
2. **毎月 収入申告書** を ケースワーカー に提出。アルバイト 収入 / 仕送り / 一時金 全て申告対象。基礎控除（基礎額 + 必要経費）後 の 額 が保護費 から 減額 される。
3. **資産変動届** — 預貯金 増加 / 自動車 取得 / 保険加入 / 不動産取得 等 は速やかに届出。
4. 月 1-2 回 の ケースワーカー 訪問調査 を受入。

### 路径 B：要件未達 で却下 → 別制度 / 再申請
- **稼働能力 あり と判定** → 求職活動 + ハローワーク 受給資格者証 + 再申請。
- **資産（預貯金 / 自動車 / 不動産）超過** → 当面 の生活費 として消費 + 売却 後 再申請 可能。
- **扶養可能 親族 あり と判定** → 民法上 の扶養義務 と 生活保護 要件 は別判断。実態 として 扶養 されていない事 を立証 し再申請。
- **在留資格 対象外**（技人国 / 経営・管理 / 留学 等）→ 生活保護 不受給。代替制度：**生活福祉資金貸付** / **住居確保給付金** / **緊急小口資金** 等 を福祉事務所 / 社会福祉協議会 で確認。

### 路径 C：却下 / 減額 / 廃止 への 審査請求
1. 決定通知 受領 から **3 ヶ月以内** に 都道府県知事 へ 審査請求（生活保護法 第 64 条）。
2. 60 日経過 で裁決 されない場合、再審査請求 を 厚生労働大臣 へ。
3. 在留資格 と 受給可否 の判断 が複雑 / 福祉事務所 の 申請拒否（「水際作戦」）に 直面 した場合、法テラス（[https://www.houterasu.or.jp/](https://www.houterasu.or.jp/)） / 弁護士会 法律相談 / 行政書士 へ相談。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **生活保護 受給 中 / 受給歴あり** は 永住申請 ガイドライン の 「独立生計要件」 を 充足しないと 判断 される ケース が大多数。受給期間中 の 永住申請 は 不許可 が原則。
- **過去 の 受給歴**：脱却後 + 自立収入 の継続 が証明 できれば 申請可能 だが、審査 で 不利材料 として 考慮 される。
- 在留期間 更新 では 生活保護 受給 のみ で 直接 不許可 にはならない（永住者・定住者 は 原則 在留資格更新 不要）が、**就労系（技人国・経営・管理 等）は そもそも 対象外** — これらの在留資格 で生活保護 受給 自体 が 在留資格該当性 を 失う 可能性 あり。
- 帰化申請 の「生計要件」（国籍法 第 5 条）も同様 に 受給中 は 充足 しないと判断。
- **判断 と 価値観 はTEBIQ の領域外** — 制度上 の 受給可否 / 受給後 の 在留・永住 影響 のみ 客観 提示。受給 すべきか の 判断 は 福祉事務所 / 弁護士（法テラス） と 個別 相談。
