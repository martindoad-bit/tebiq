# 児童扶養手当 認定通知 / 儿童扶养补贴 认定通知

## 信封长什么样
- 寄件人：市役所 子ども家庭支援課（児童家庭課 / こども育成課 等）
- 封面常见印字：「児童扶養手当 認定通知書」「親展」「重要書類」
- 颜色：白底 + 自治体 LOGO
- 同梱物：認定通知書 1 枚 / 児童扶養手当証書 / 現況届 案内（毎年 8 月）/ 受給者のしおり

## 这是什么
父母 の 離婚・死別・未婚 等 により ひとり親 状態 にある 18 歳到達後 最初の 3 月末 までの 児童（障害児 は 20 歳未満）を養育する 親 等 に対する 国の 給付制度 の **認定結果**。児童手当 とは別制度で、所得制限 が厳格 + 毎年 現況届 提出義務 がある。認定 されると 翌月分 から 支給開始（年 6 回 振込：1月・3月・5月・7月・9月・11月 各月 11 日）。

## 为什么收到
- 児童扶養手当 認定請求書 を 市役所 へ提出
- 戸籍謄本 / 離婚届 受理証明 / 死亡診断書 / 住民票（世帯全員）/ 課税証明書 等 の審査完了
- 申請月 の翌月分 から 支給対象（遡及不可、申請月 が決定的）

## 必看的关键栏
- **受給資格者氏名 + 児童氏名**
- **手当月額**（全部支給 / 一部支給 / 児童数 加算）
- **支給開始月**（申請受理月 の翌月 から）
- **次回 現況届 提出期限**（毎年 8 月、未提出 で 11 月以降 支給停止）
- **公的年金等 受給状況**（遺族年金 / 障害年金 と 全額 又は 差額調整 あり）
- **問合せ先**（子ども家庭支援課 ひとり親 担当）

## 不处理的后果（時間梯度）
- **現況届 未提出**（毎年 8 月）→ 11 月分 以降 支給停止
- **2 年連続 現況届 未提出**：受給資格 消滅
- **生計同一 男性（事実婚）発覚 等 で 不正受給 認定**：全額返還命令 + 加算金（最大 40%）+ 児童扶養手当法 第 35 条 詐欺罪 告発（懲役 3 年以下 / 罰金 30 万円以下）
- **公的年金 受給開始 で差額調整 漏れ**：過払金 返還命令
- 信用 / 在留 影響 は 不正受給 + 詐欺罪 告発 ケース のみ。法令通り受給 は 影響なし

## 该怎么处理（3 种典型路径）

### 路径 A：認定受領 → 通常受給（多数人）
1. 認定通知 + 児童扶養手当証書 を保管。
2. 振込口座 を確認 — 1月・3月・5月・7月・9月・11月 の各 11 日（休日 の場合 直前 平日）に 2 ヶ月分 まとめて 振込。
3. **毎年 8 月 1-31 日** の 現況届 を必ず提出。健康保険証 写し / 養育費 受取状況 申告書 / 就業状況 を更新。
4. 児童 18 歳到達後 最初 の 3 月 31 日 で資格喪失 → 自動的 に支給終了。

### 路径 B：所得増加 / 状況変更 申告
- **就職 / 昇給 で所得増** → 「所得 額 改定」 申請。一部支給 → 全部停止 への 自主申告 が義務。怠ると 後日 過払 返還命令。
- **公的年金 受給開始**（遺族年金 / 障害年金 等）→ 差額調整 の改定届 提出。
- **再婚 / 事実婚 関係 開始** → 受給資格 喪失。住民票 + 戸籍 変更後 速やかに 喪失届 提出（届出義務、児童扶養手当法 第 28 条）。
- **児童 が父母 と同居 開始**（離婚 後 復縁 / 同居 等）→ 受給資格 喪失。

### 路径 C：認定 / 額 不服申立
1. 認定通知 受領 から **3 ヶ月以内** に 都道府県知事 へ 審査請求（児童扶養手当法 第 19 条）。
2. 主な争点：所得計算 誤り / 公的年金 額 認定誤り / ひとり親要件（事実婚 推定 等）認定誤り。
3. 「事実婚 とみなされた」「生計同一 男性 がいる」 と判定 された ケース は 立証 が困難 で実例 が複雑、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 児童扶養手当 は 拠出制 ではなく 国費負担 給付（こども家庭庁 + 自治体 1/3 + 国 1/3）。永住申請 ガイドライン上 「公的負担」 に分類 される 可能性 あり。
- 法定給付 として 受給 自体 は 即時 不許可事由 ではない が、**生活保護 + 児童扶養手当 + 住宅扶助 等 を併給 し 自立収入 がほぼ無い 状態** は 「独立生計要件」評価 に影響。
- 不正受給 → 詐欺罪 告発 → **素行善良要件** 不充足 → 永住・更新 不許可 リスク高。
- 在留資格別影響：日本人配偶者等 / 永住者 配偶者等 / 定住者 で 離婚後 受給 する場合、**離婚 → 在留資格変更（定住者 等）** の手続 と 同時並行 になる ケース 多。
