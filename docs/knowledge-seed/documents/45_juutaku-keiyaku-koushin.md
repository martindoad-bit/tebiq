# 住宅 賃貸契約 更新通知

## 信封长什么样
- 寄件人：管理会社（「○○不動産」「○○ハウジング」「○○管理サービス」）または 大家 個人名
- 封面印字：「賃貸借契約 更新のご案内」「契約更新書類在中」「重要書類」
- 同梱物：
  - 賃貸借契約 更新通知書 / 更新合意書
  - 重要事項説明書（再契約型の場合）
  - 火災保険 / 家財保険 更新案内 + 申込書
  - 保証会社 更新通知（ジェイリース / 日本セーフティー / Casa 等）
  - 連帯保証人 同意書（保証人立て型の契約）
  - 振込先案内 / 払込取扱票

## 这是什么
賃貸借契約 の契約期間（通常 2 年）満了に伴う更新案内。日本の住宅賃貸は **普通借家契約** と **定期借家契約** で運用が異なる。
- **普通借家契約**：借地借家法 第 26 条 → 借主に法的保護、貸主が更新拒絶するには「正当事由」必要、特約に従い更新料発生
- **定期借家契約**：借地借家法 第 38 条 → 期間満了で当然終了、再契約は別途合意必要、更新料概念なし

## 为什么收到
契約期間満了の **2-3 ヶ月前** に管理会社が更新書類を送付。継続居住するか退去するかの意思確認 + 更新条件の提示。

## 必看的关键栏
- **契約種別**：普通借家 / 定期借家（書類冒頭）
- **新契約期間**：通常 2 年
- **更新料**：地域慣行で異なる（首都圏：家賃 1 ヶ月分が一般的、関西：更新料なし or 保証金償却が主流）
- **新家賃 / 共益費・管理費**：値上げ提示の場合あり、合意なければ法的に拒否可能（借地借家法 第 32 条 借賃増減請求権）
- **火災保険 更新料**：¥10,000-¥25,000/2 年が一般的
- **保証会社 更新料**：年額 ¥10,000 or 月額家賃の 1-3% / 1-2 年毎
- **更新事務手数料**：管理会社 ¥10,000-¥30,000 程度
- **連帯保証人 / 緊急連絡先**：再記入欄
- **回答期限 / 返送期限**

## 不处理的后果（時間梯度）
- **回答期限経過**：管理会社から催促電話 / 書面
- **契約期間満了 + 何も返送せず**：
  - **普通借家**：法定更新（借地借家法 第 26 条 第 1 項）→ 期間の定めなき契約として継続。**ただし更新料 特約は有効** とする最高裁 2011/7/15 判決があり、未払の場合は別途請求 / 訴訟リスク
  - **定期借家**：期間満了で **当然終了** → 退去命令 / 不法占拠扱い → 明渡訴訟
- **更新料 / 火災保険 / 保証料 未払**：滞納督促 → 連帯保証人請求 → 強制退去訴訟（賃料滞納 3 ヶ月程度から信頼関係破壊）
- **連帯保証人 不可 / 保証会社不可**：契約更新拒否 → 退去要求

> ⚠️ 更新料 慣行は地域差大。首都圏は家賃 1 ヶ月、関西・北海道は無し or 礼金 / 償却で代替が多い。

## 该怎么处理（3 种典型路径）

### 路径 A：継続居住 → 更新書類記入・返送
1. 同封書類を確認、特に **契約種別**（普通 / 定期）+ **新家賃** + **更新料合計**（更新料 + 事務手数料 + 火災保険 + 保証料）を計算。
2. 連帯保証人 / 緊急連絡先 記入、必要に応じ印鑑証明書 / 住民票 添付。
3. 更新料・火災保険料・保証料 振込 → 振込明細書を返送 or 受領確認。
4. 新契約書 / 重要事項説明書 + 火災保険証券 + 保証会社契約書 を保管。**在留期間更新・永住申請 で住居安定性証明として必要**。

### 路径 B：家賃減額交渉 / 条件変更
1. 周辺相場（[国土交通省 不動産取引価格情報](https://www.land.mlit.go.jp/webland/) / SUUMO 等）を調査。
2. 借地借家法 第 32 条（借賃増減請求権）に基づき、書面で減額希望を管理会社へ提示。値上げ提示への拒否 + 据置 / 減額交渉も可。
3. 合意至らなければ 簡易裁判所 民事調停。
4. 更新料の減額 / 不払 交渉は最高裁 2011/7/15 判決（更新料特約有効）以降ハードル高い。

### 路径 C：退去 / 引っ越し
1. **解約予告期間**（多くは 1-2 ヶ月前）を契約書で確認。
2. 管理会社へ「解約通知書」送付（書面 / 内容証明 推奨）。
3. 退去立会日決定、原状回復精算（敷金 返還・追加請求）。
4. 引っ越し後 14 日以内に 市役所 / 区役所 で 転入届 / 転出届、入管 へ「住居地届出」（在留カード 裏面記載）。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 賃貸借契約書（更新後）は **在留期間更新申請** の住居証明として必要となる場合あり（特に家族滞在 / 配偶者ビザ / 永住）
- 永住申請：「独立生計」+「素行善良」評価で安定した住居 + 公共料金・家賃滞納なし が事実上の前提
- 経営・管理：事務所と住居が別の場合、事務所賃貸借契約書 が更新審査の柱。住居契約も生活基盤証明として補強材料
- **家賃滞納 + 訴訟・強制退去 履歴**は永住「素行善良」の説明資料に間接影響

> ⚠️ 続签時に求められる住居証明書類は審査官・在留資格 で異なる。
