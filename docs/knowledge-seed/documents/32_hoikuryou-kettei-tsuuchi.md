# 保育料 決定通知書 / 保育费 决定通知

## 信封长什么样
- 寄件人：「○○市役所 保育課」「○○区役所 子ども家庭支援課」
- 封面印字：「保育料 決定通知書 在中」「利用者負担額 決定通知書」「重要書類 親展」
- 颜色：白底 + 自治体ロゴ。年度替 4 月 / 9 月（住民税 確定後 切替）に集中発送
- 同梱物：保育料 決定通知書 / 利用者負担額 算定根拠（前年度 住民税 額）/ 口座振替 依頼書 / 軽減制度 案内 / 副食費（給食費）案内

## 这是什么
保育園 / 認定こども園 / 地域型保育 利用児童 の 保育料（正式名称 = 利用者負担額）を 市町村 が 確定 して 通知 する 文書。子ども・子育て支援法 施行令 第 4 条 が根拠。算定 = 前年度（4-8 月分）/ 当年度（9 月-翌 3 月分） 父母 合算 住民税 所得割額 → 自治体 階層表 に 当てはめ → 月額決定。3-5 歳児（保育標準時間 / 短時間）は 幼児教育・保育の無償化 で 保育料 ¥0（副食費 ¥4,500/月 程度 別途）。

## 为什么收到
- 4 月：新年度 保育料 決定（前年度 住民税 ベース）
- 9 月：当年度 住民税 確定 反映 → 9 月 - 翌 3 月分 改定
- 入園決定 直後：初回 保育料 通知
- 世帯構成 / 所得 変更 時：随時改定
- 転入後：新自治体 で 改めて 決定通知

## 必看的关键栏
- **月額 保育料**（利用者負担額）— 年齢階級 / 認定区分 / 所得階層 別
- **適用期間**（4-8 月 / 9 月-翌 3 月、または 月額単位）
- **算定根拠 住民税 額**（父母合算 市町村民税 所得割額）
- **第 2 子 半額 / 第 3 子 無料**（多子軽減）
- **減免該当**（ひとり親世帯 / 在宅障害児 / 災害被災 等）
- **納付方法**（口座振替 / 納付書 / 子ども・子育て支援システム 経由）
- **副食費**（3-5 歳児 月 ¥4,500 程度、年収 360 万円未満 世帯 等 免除）
- **問合せ先**（保育課 料金担当 直通番号）

## 不处理的后果（时间梯度）
- **納期 翌日**：延滞金 加算（市町村 条例 で 通常 年 14.6%、最初 1 ヶ月 年 7.3%）
- **20-60 日**：督促状 + 督促手数料（¥100 程度）
- **3 ヶ月以上 滞納**：園 から 直接連絡、保育課 と 分納相談 案内
- **6 ヶ月-1 年 以上 滞納**：強制徴収（地方自治法 第 231 条の 3、地方税 滞納処分 例 と 同様 給与 / 預金 差押 可能）
- **継続滞納 + 連絡無視**：退園 措置（自治体 条例 / 運用 で 異なる、必ずしも 即退園 ではない）

延滞金 / 給付制限 / 永住申請 影响 については 介護保険料・国民健康保険料 と 同様 「公的料金」位置づけ。

> # ⚠️ 需创始人/書士复核：保育料 滞納 が 永住申請 「公的義務履行」評価 で 国保・年金 と 同等扱い か、それとも 軽度扱い か — 入管 公開ガイドライン に 明示 なく、運用判断 領域。

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 口座振替 設定 → 月次 自動引落
1. 同梱 口座振替 依頼書 → 金融機関 窓口 / Web 提出 → 翌月-翌々月 から 自動引落
2. 引落不能時 → 納付書 別途 郵送 → コンビニ / 銀行 / 自治体窓口 で 納付
3. 領収書 / 振替済通知 を 5 年 保管（年末調整 控除 / 永住申請 補強資料）

### 路径 B：軽減 / 免除 申請
- **多子世帯**：第 2 子 半額 / 第 3 子 無料（年齢制限 自治体差あり、年収 360 万円未満 世帯 は 年齢制限なし）
- **ひとり親世帯 / 障害児 在宅 世帯**：所得階層 第 2 階層以下 等 で 全額 / 半額 軽減
- **災害・失業・廃業**：保育料 減免 申請（罹災証明 / 離職票 / 確定申告書 等 添付）
- **副食費 免除**：年収 360 万円未満 相当 世帯 + 第 3 子以降
- 申請窓口：市役所 保育課（保育園経由 申請可 自治体 あり）

### 路径 C：算定 異議 / 不服申立
触发条件：住民税 額 反映誤り / 世帯員 範囲 誤り / 多子軽減 漏れ。
1. 通知書 受領 から **3 ヶ月以内** に 「審査請求書」を 市町村長 に 提出（行政不服審査法 第 18 条）
2. 同時に 保育課 窓口 で 算定根拠 説明 受領 → 修正可能なら 行政手続 で 解決
3. 住民税 自体 の 算定誤り であれば 課税課 に 課税更正 請求（参见文档 #01）
4. 在留資格 / 国籍 紛争 を伴う 複雑 case では、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 保育料 = 公的料金、永住申請 / 帰化申請 「公的義務履行」評価 対象（介護保険料 / 国民健康保険料 と 並列）
- 滞納 1 期 で 即不許可 ではないが 直近 5 年 全期 完納 + 領収書 控え保管 が 安全
- 経営・管理 役員 で 役員報酬 高 → 保育料 上限階層（月 ¥80,000-¥104,000）該当 多。法人税 と 個人住民税 が 別管理 のため 個人 保育料 認識漏れ ケース あり
- 技人国 / 家族滞在 世帯 の 配偶者 は 「資格外活動 28 時間以内」 内 就労 でも 保育の必要性 認定 可能（自治体 運用 確認）
- 保育料 軽減 申請 履歴 自体 は 在留審査 に マイナス 影響 なし（軽減 = 制度利用、滞納 とは 異なる）
