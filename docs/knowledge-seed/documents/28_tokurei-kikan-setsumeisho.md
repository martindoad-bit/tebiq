# 特例期間 説明書 / 通知

## 信封长什么样
- 寄件人：審査担当の地方入管局
- 形式：茶色 葉書 / 案内文書 / 申請受理票 と一体になっているケースが多い
- 関連物：申請受理票（裏面 or 別紙に「特例期間に関する説明」）、在留カード 裏面 押印（「在留期間更新中」「在留資格変更中」）
- 単独通知ではなく、**申請受付時に交付される説明書 が原型**

## 这是什么
在留期間更新 / 在留資格変更 を **在留期限まで に申請** したが、審査中に在留期限が来てしまった場合、**最長 2 ヶ月** は 元の 在留資格 のまま在留が認められる制度（入管法 第 20 条 5 項 / 第 21 条 4 項）。これが **特例期間**。

## 为什么收到
- 在留期間更新 / 在留資格変更 を 在留期限前 に出した → 審査長期化 → 在留期限 到来時、特例期間 自動適用
- 申請受付時、受理票 + 説明書 で「特例期間 制度」を案内される
- 在留カード 裏面 に「在留期間更新中 / 在留資格変更中」のスタンプ + 期間記載

## 必看的关键栏
- **特例期間の起算日 / 満了日**：在留期限の翌日 から **最大 2 ヶ月**、または 審査結果 通知時点 のいずれか早い方
- **元の 在留資格 / 在留期間 が継続される旨**
- **就労制限**：元の 在留資格 の制限がそのまま継続（資格外活動 不可）
- **再入国**：可能だが、**特例期間 中の出国 → 再入国** には注意（再入国許可 / みなし再入国 + 在留カード 裏面スタンプ 必要、空港で要確認）

## 不处理的后果（時間梯度）
- **特例期間 中に審査結果 出れば**：そのまま 許可 / 不許可 通知 へ移行
- **特例期間 2 ヶ月 経過 + 結果未通知**：実務上は 入管 が結果通知を出すのが標準、ただし 法定上 は不許可 とみなされる扱い
- **特例期間 中に何もせず 2 ヶ月 + 不許可**：「特定活動（出国準備）30 日 / 31 日」へ自動変更、それも経過で オーバーステイ
- **特例期間 中に他の活動（資格外就労 等）**：資格外活動違反、不許可 + 退去強制 リスク

> 入管法 第 20 条 5 項（在留資格 変更申請中の特例期間）、第 21 条 4 項（在留期間 更新申請中の特例期間）

## 该怎么处理（3 种典型路径）

### 路径 A：特例期間 → 結果待ち（標準）
1. 在留カード 裏面 「在留期間更新中 / 在留資格変更中」スタンプを確認。
2. 元の 在留資格 / 就労制限 / 活動範囲 をそのまま継続。
3. 雇用主 / 学校 へ 在留カード 写し（裏面含む）+ 申請受理票 写し を提出、特例期間 中であることを共有。
4. 結果通知（許可 / 不許可）が来るまで、在留カード 持参を維持。
5. 档案归档：申請受理票、在留カード 裏面スタンプ 写真、雇用主への通知メール記録。

### 路径 B：特例期間 中の出国
1. **再入国許可** / **みなし再入国** + 在留カード（裏面 特例期間スタンプ）+ 旅券 を持参。
2. 空港 出国審査で「審査中」「特例期間」を伝達、係官の指示に従う。
3. 帰国時、特例期間 内に戻ること。**特例期間 満了後 / 結果通知後** の入国は別ロジック。
4. 出国 / 帰国 タイミングが微妙な場合、**事前に 入管 へ電話確認** 推奨。

### 路径 C：弁護士 / 行政書士 緊急対応
- 特例期間 2 ヶ月 経過 + 結果未通知（審査長期化 異常）
- 特例期間 中の海外渡航 + 審査結果 通知が出国中に到達するリスク
- 特例期間 中に 雇用主 変更 / 退職 した（届出義務 + 申請内容変動）
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **特例期間 自体は 在留資格 履歴に「ネガ」評価ではない**。期限前申請 + 審査長期化 は本人の責任ではない
- ただし **特例期間 中の不適切活動（資格外就労 / 雇用主変更未届出）** は素行善良性 否定 → 続签・永住 で重大マイナス
- **永住申請** で「在留期間中の継続居住」を計算する際、特例期間 も継続居住期間 に算入される（運用上）
- **経営・管理** で 特例期間 中に事業計画変更 / 役員変動 があった場合、結果通知前に 補正資料 提出 / 申請取下げ + 再申請 を検討

> # ⚠️ 需创始人/書士复核：特例期間 中の出国 / 再入国 の最新運用と、特例期間 を含めた永住要件「引き続き居住」期間の算入実務
