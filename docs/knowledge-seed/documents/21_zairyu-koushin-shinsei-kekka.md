# 在留期間更新 申請 結果通知（許可 / 不許可）

## 信封长什么样
- 寄件人：「東京出入国在留管理局」「大阪出入国在留管理局」等地方入管局名（信封左上 / 法務省 紋章）
- 形式：常见为茶色官製ハガキ（葉書）或定形封筒，封面印「在留期間更新許可申請に係る通知」
- 同梱物（若邮寄申請时）：申請受理票（受付番号）、本人宛 返信用封筒
- 在窗口领取在留カード 时，会同时返还旅券（パスポート）

## 这是什么
出入国在留管理庁 对你提交的「在留期間更新許可申請」做出的最终判断通知。两种结果：**許可**（含期限・在留資格・条件）或 **不許可**（含不許可理由 + 出頭日时）。許可时，通知書同时是 在留カード 受领凭证。

## 为什么收到
你或申請取次（行政書士 / 雇用主）在 在留期限 到期前提交了「在留期間更新許可申請書」。审查完毕后，入管按申請书记载地址寄出结果。审查期一般 2 週間〜2 ヶ月（在留資格・繁忙期不同）。

## 必看的关键栏
- **申請番号 / 受付番号**：与受理票 一致，问合せ 必备
- **結果区分**：「許可」「不許可」「在留資格認定変更」等
- **新しい在留期間 / 在留資格**：5 年 / 3 年 / 1 年 / 6 月 / 4 月 ／ 3 月
- **出頭日時 / 場所**：通常指定 14 日内，到指定窗口领取 在留カード
- **持参物**：旅券、現在の 在留カード、手数料（収入印紙 ¥4,000）、申請受理票、通知書本体
- **不許可时**：理由概要欄 + 担当審査官 + 内示出頭指定日

## 不处理的后果（時間梯度）
- **指定出頭日経過**：許可は出てるが 在留カード 未交付 → 旧在留カード 期限後は 不法滞在 リスク
- **在留期限 当日まで未出頭**：旧 在留カード 失効 → **特例期間**（最大 2 ヶ月）に入る場合あり、ただし在留資格 取得手続未完
- **特例期間も超過**：オーバーステイ（不法残留） → 退去強制手続 / 上陸拒否期間 5 年
- **不許可 + 在留期限 切れ**：出国準備期間（30 日 / 31 日）の特定活動 が付与される場合あり、未対応で 退去強制 へ

> 入管法 第 22 条の 4、第 24 条（退去強制事由）に基づく。

## 该怎么处理（3 种典型路径）

### 路径 A：許可 → 在留カード 受領
1. 持参物 揃えて 指定出頭日 までに 指定入管局 窗口へ。
2. 旧 在留カード 回収 → 新 在留カード 交付（収入印紙 ¥4,000）。
3. 旅券 へ 証印（or 在留カード 自体に新期限記載）。
4. 受領後 14 日内、市役所 / 区役所 で 住民票 更新（住所 変更ある場合）。
5. 档案归档：新 在留カード 両面、許可通知書、領収印紙 控え。

### 路径 B：不許可 → 再申請 / 出国準備
1. 通知書記載の **担当審査官 / 出頭日** 確認 → 不許可理由 詳細聴取（口頭）。
2. **30 日以内**、再度 在留期間更新 を申請可能（ただし 同一理由なら 再不許可 確実）。
3. 在留資格 該当性 を 満たさない場合、**特定活動（出国準備）30 日 / 31 日** に変更され、その間に出国。
4. 異議申立：入管法 上 行政不服審査法 の 適用除外（入管法 第 61 条の 9 の 3 等）→ 一般的な 異議申立 不可、再申請 か 司法救済（取消訴訟）が選択肢。

> # ⚠️ 需创始人/書士复核：在留期間更新 不許可 後の 異議申立 / 再申請 期限の正確な根拠条文と、特定活動「出国準備」付与の標準パターン

### 路径 C：弁護士 / 行政書士 緊急対応
- 不許可理由 が「在留資格 該当性 否定」「素行不良」「経済的安定性 欠如」など重い場合、再申請 戦略の組立てが必要。
- 在留期限 切れ寸前 / 出国準備期間 中 / 退去強制手続 着手前 → 緊急対応窗口。
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **不許可履歴** は 入管 内部記録に永続保存。次回更新・在留資格変更・永住許可申請 で 過去申請履歴 として参照される。
- **更新期間 が 1 年 → 3 年 → 5 年** と伸びることは「素行善良 + 公的義務履行」の累積評価サイン。永住要件「就労系で最長期間（現行運用は 3 年以上）」を満たす必要あり。
- **不許可 後の 再申請 で 許可** が出た場合でも、不許可履歴 は 残る。永住申請 時に 申請理由書 で説明を要する場合がある。
- 経営・管理 の場合、不許可 = 事業継続性 / 安定性 否定 のシグナル → 翌期 決算書 / 事業計画 で挽回必要。
