# 罰金通知 / 略式命令書（在留関連）

## 信封长什么样
- 寄件人：
  - **略式命令書**：簡易裁判所（「東京簡易裁判所」「○○簡易裁判所」）
  - **罰金等納付告知書 / 納付書**：地方検察庁 徴収事務担当
- 形式：定形封筒、白地、封面に「親展」「裁判所」「検察庁」、内部に略式命令書 + 罰金額 + 納付期限 + 納付書（金融機関 払込用）
- 同梱物：略式命令書 写し、罰金等納付告知書、納付書（コンビニ / 銀行 / 郵便局 払込可）、正式裁判申立 案内

## 这是什么
**略式命令書**：100 万円以下の罰金 / 科料 を 簡易裁判所 が略式手続（公開法廷なし、書面審理）で言い渡す決定。本人の **同意** を前提に検察官が請求。
**罰金等納付告知書**：略式命令確定後、検察庁 から罰金 の納付を求める書面。

在留関連で頻出する原因：
- 道路交通法違反（飲酒運転 / 無免許 / 人身事故 / ひき逃げ / 速度超過）
- 入管法違反（資格外活動 / 不法就労 / オーバーステイ）
- 軽微な刑法犯（万引き / 暴行 / 業務妨害 等）

## 为什么收到
警察 / 検察 の取調べ → 検察官 が略式起訴を請求 → 簡易裁判所 が略式命令 → 本人へ 略式命令書 + 納付書 が郵送される流れ。**略式手続には事前に本人の同意（略式手続意思確認）が必要**（刑事訴訟法 第 461 条の 2）。

## 必看的关键栏
- **罰金額** / **科料額**（罰金は ¥10,000 以上、科料は ¥1,000 以上 ¥10,000 未満）
- **納付期限**：通常 略式命令書 受領後 30 日内 + 検察庁 の納付告知日から定められた期限
- **正式裁判申立期間**：略式命令書 告知日 から **14 日以内**（刑事訴訟法 第 465 条）。これを過ぎると確定
- **罪名 / 適用法条**：道交法第○条、入管法第 70 条 等
- **納付方法**：納付書 で 金融機関 / コンビニ / 検察庁 窗口

## 不处理的后果（時間梯度）
- **正式裁判申立期間（14 日）経過**：略式命令 確定、判決と同等の効力
- **納付期限 経過**：督促 → **労役場留置**（罰金額を日額（通常 ¥5,000）で割った日数 拘禁、刑法 第 18 条）
- **罰金 確定 = 前科**：在留資格 への重大影響
- **2024 年 6 月 入管法改正**：永住者 でも「1 年以下の懲役 / 禁錮 / 罰金」で 在留資格 取消対象（入管法 第 22 条の 4 改正）
- **罰金 + 続签**：素行善良性 否定 → 在留期間更新 / 在留資格変更 / 永住 / 帰化 すべてに不利益
- **長期未納 + 国外滞在**：再入国時に 入管 で確認、国外退去 / 上陸拒否 リスク

> 刑法 第 18 条（労役場留置）、刑事訴訟法 第 461 条〜第 470 条（略式手続）、入管法 第 22 条の 4

## 该怎么处理（3 种典型路径）

### 路径 A：罰金 納付（標準対応）
1. 略式命令書 + 納付書 を写真撮影 + 写し保管。
2. **正式裁判申立 期間（14 日）** 内に異議なければ、納付期限内 に納付。
3. 納付方法：納付書 で 金融機関 / コンビニ / 郵便局 / 検察庁 徴収事務窗口。
4. 領収書 / 納付済証 を **永久保管**（後の 続签・永住・帰化 で証明資料）。
5. 档案归档：略式命令書、納付書、領収書、罪名 / 適用法条 / 罰金額 / 納付日 一覧。

### 路径 B：正式裁判 申立（争う場合）
1. 略式命令書 告知日 から **14 日以内**、書面で 略式命令を出した 簡易裁判所 へ「正式裁判請求書」提出（刑事訴訟法 第 465 条）。
2. 通常の刑事公判手続 へ移行、公開法廷で審理。
3. 期間徒過 = 略式命令 確定、争う機会 喪失。
4. **争う前提**：罪を否認 / 量刑が重すぎ / 証拠不十分 / 手続瑕疵 等の主張がある場合のみ。

### 路径 C：弁護士 / 行政書士 緊急対応
- **正式裁判 申立 検討**：14 日 期限あり、刑事弁護人 早期依頼必須（当番弁護士 / 法テラス）
- **罰金確定 + 在留資格 取消 リスク**：永住者 / 高度専門職 / 経営・管理 で重大影響
- **複数件 罰金履歴**（前科累積）= 続签 / 永住 / 帰化 ほぼ不可能水準
- **入管法違反 系 罰金**：資格外活動 / 不法就労 認定 + 在留資格 取消 + 退去強制 一体リスク
- **再入国 / 出国 タイミング** に罰金確定 が重なる
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **罰金 = 前科**。**在留資格 申請書「過去の処分歴」欄** に正直記載必須（虚偽 = 在留資格 取消事由 + 退去強制）
- **永住申請 ガイドライン**：「素行善良 = 法律を遵守し、日常生活においても住民として社会的に非難される行為がない」→ 罰金前科 は直接マイナス。罰金確定後 **少なくとも 5 年** は永住申請 困難（実務目安、明文規定なし）
- **帰化申請**：素行要件 / 法律遵守 で 罰金前科 はさらに重い影響
- **2024 年 6 月 入管法改正**：永住者 で「1 年以下の懲役・禁錮・罰金」確定 → 永住資格 取消 + 元在留資格 へ降格 / 退去強制 対象
- **道交法 違反系 罰金**：人身事故・飲酒運転 は永住・帰化 でほぼ確実に否認材料
- **入管法 違反系 罰金**：資格外活動 ¥30 万円以下、不法就労 助長 ¥300 万円以下、不法残留 ¥30 万円以下（入管法 第 70 条 各号）

> # ⚠️ 需创始人/書士复核：罰金確定後の永住・帰化申請 待機期間（5 年 / 10 年）の最新運用、および 2024 年 6 月 改正後の 永住資格 取消「故意」要件の認定基準
