# 就労資格証明書（交付 / 不交付）

## 信封长什么样
- 寄件人：地方入管局（「東京出入国在留管理局」等）
- 形式：定形封筒 + 葉書、または窗口での **当日交付** が多い
- 交付物：A4 白紙 1 枚、上部「就労資格証明書」、本人氏名 / 在留資格 / 就労可能な活動範囲（具体雇用主・職務記載あり / なし版）+ 入管局長 印
- 不交付通知：理由付き、別書面

## 这是什么
入管法 第 19 条の 2 に基づき、外国人本人の申請で 入管 が発行する「現在の 在留資格 で この就労活動が可能」と証明する書面。**転職時 や 内定時** に新雇用先の活動が現在の 在留資格 でカバーされるか **事前確認** するのが主用途。法的には申請任意だが、転職時 強く推奨。

## 为什么收到
本人が「就労資格証明書交付申請書 + 在留カード 写し + 旅券 写し + 雇用契約書 / 内定通知書 + 雇用主の登記事項証明書 / 決算書」等を 入管 に提出。標準処理期間 約 1 ～ 3 ヶ月。

## 必看的关键栏
- **証明する活動内容**：職務内容 / 雇用主名 / 雇用形態（具体記載 ありの方が後の更新審査で有利）
- **在留資格 / 在留期間**：申請時点の情報
- **発行日 / 有効性**：明示の有効期限はないが、**雇用契約 や 在留期間 の変動で実質失効**
- **不交付 の場合**：理由欄（「該当 在留資格 では対象活動を行えない」「資料不足」等）

## 不处理的后果（時間梯度）
- **転職時 未取得 + 続签時 否認**：新職務が在留資格 と不一致 → 在留期間更新 不許可
- **就労資格証明書 不交付 を放置 → 同雇用主で就労継続**：実態が 在留資格 範囲外 = 資格外活動 / 不法就労（入管法 第 70 条 1 項 4 号）
- **不交付 通知後 14 日内に何もしない**：再申請 / 在留資格変更 / 退職 のいずれかが必要、放置すれば次回更新で問題発覚
- **転職事実 + 所属機関に関する届出 14 日内未提出**：素行善良性 否定材料

> 入管法 第 19 条の 2、第 19 条の 16、第 70 条

## 该怎么处理（3 种典型路径）

### 路径 A：交付 → 雇用主と共有 / 档案保管
1. 窗口受領 or 郵送受領 → A4 原本 を **雇用主 人事部 に写し提出**（採用審査 / 入社時 ID 確認）。
2. 原本 は本人保管。次回 在留期間更新 申請時に「就労資格証明書」として添付。
3. 「所属機関に関する届出」（入管法 第 19 条の 16）を 14 日内 に出していること再確認。
4. 档案归档：就労資格証明書 原本写真、申請書 控え、印紙 控え（¥1,200）。

### 路径 B：不交付 → 在留資格変更 / 転職撤回
1. 不交付理由 を確認：
   - 「該当 在留資格 では対象活動を行えない」→ **在留資格変更** が必要（例：技人国 → 経営・管理）
   - 「学歴・職歴 と 職務内容 不一致」→ 職務記述書 の補強 / 別職種 検討
2. **在留資格変更許可申請** を出すか、内定辞退 / 別ポジション へ移動。
3. 不交付 そのものは 在留資格 を直接変えない（元の 在留資格 は残る）が、**実際に該当業務を行えば 資格外活動 違反**。

### 路径 C：弁護士 / 行政書士 緊急対応
- 不交付理由 が複雑（学歴 / 職歴 / 専攻 と 新職務 のミスマッチ判定）
- 高度専門職 ポイント計算 / 特定技能 → 技人国 への変更タイミング
- すでに新雇用主で就労開始してしまった + 不交付通知 受領
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **転職時の 就労資格証明書 取得** は法的義務ではないが、**続签時の リスクヘッジ** として強く推奨。証明書 ありで 入管 心証が大幅改善
- **所属機関に関する届出**（入管法 第 19 条の 16）は別途 14 日内 義務、これは強制
- **経営・管理 でこの証明書 を取るケース**：他社へ役員兼務する場合、活動範囲が現在の 経営・管理 でカバーされるか事前確認
- **永住申請** で 直近 3 年〜10 年の就労状況 が問われるとき、就労資格証明書 履歴が**証拠として強い**

> # ⚠️ 需创始人/書士复核：就労資格証明書 不交付後 同雇用主で就労継続した場合の「不法就労」「資格外活動」具体線引き
