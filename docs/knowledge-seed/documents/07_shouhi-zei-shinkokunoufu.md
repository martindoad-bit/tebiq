# 消費税 申告納付書

## 信封长什么样
- 寄件人：「○○税務署」「国税庁」
- 封面印字：「消費税及び地方消費税 申告書 在中」「適格請求書発行事業者」案内 同梱 場合あり
- 信封颜色 / 大小：白色 / 浅藍色 角 2 号、所得税 申告書 と 別便 で 届く ケース 多
- 同梱物：消費税 確定申告書（一般用 / 簡易課税用 / 2 割特例用）+ 付表 + 納付書 + 中間申告書（前年 納税額 一定額以上時）+ e-Tax 案内

## 这是什么
消費税 = 国内 取引 / 輸入 取引 に 課される 国税（消費税法 第 4 条）+ 地方消費税（地方税法）。標準税率 10%（うち 国 7.8% + 地方 2.2%）/ 軽減税率 8%（飲食料品 / 新聞）。
納税義務者 = 課税事業者（基準期間 課税売上高 ¥1,000 万円超 / または インボイス 登録事業者 / 特定期間 ¥1,000 万円超 等）。
2023 年 10 月 1 日 から インボイス制度（適格請求書等保存方式）開始 → 売上 ¥1,000 万円 以下 でも 「適格請求書発行事業者」登録 すれば 課税事業者 に 自動切替（取引先 が 仕入税額控除 を 受ける ため 登録要請 多発）。
申告期限：法人 = 課税期間 末日翌日 から 2 ヶ月以内 / 個人事業主 = 翌年 3 月 31 日（所得税 と 別期限、注意）。

## 为什么收到
- 前々年（基準期間）課税売上高 が ¥1,000 万円超
- インボイス 登録事業者（売上関係なく 登録 = 課税事業者）
- 前年 中間申告 該当 + 中間申告書 が 自動送付
- 新設法人 で 資本金 ¥1,000 万円以上 → 設立 1 期目 から 課税事業者
- 「課税事業者選択届出書」自主提出（輸出業 / 設備投資多 で 還付 狙い）

不会収到的人：免税事業者（基準期間 売上 ¥1,000 万円以下 + インボイス 未登録）/ 給与所得者 のみ。

## 必看的关键栏
- **課税期間**：法人 = 事業年度（4-3 月 等）/ 個人 = 暦年（1-12 月）
- **申告期限**：法人 = 期末日 + 2 ヶ月（延長特例 + 1 ヶ月 申請可）/ 個人 = 翌年 3 月 31 日
- **納付期限**：申告期限 と 同日
- **中間申告**：前年 確定 消費税額（地方分除く）¥48 万円超 = 年 1 回 / ¥400 万円超 = 年 3 回 / ¥4,800 万円超 = 年 11 回
- **付款码**：納付書 OCR + バーコード、コンビニ ¥30 万円 以下
- **計算方式**：一般課税 / 簡易課税（基準期間 売上 ¥5,000 万円以下 + 「簡易課税制度選択届出書」事前提出 + みなし仕入率 業種別）/ 2 割特例（インボイス 移行 経過措置、令和 5-8 年度）

## 不处理的后果（时间梯度）
- **期限後 1 ヶ月内**：期限後申告 自主提出 → 無申告加算税 軽減（5%）
- **税務署指摘後**：無申告加算税 15-20%
- **申告したが 納付遅延**：延滞税 年 2.4% / 1 ヶ月後 年 8.7% 前後
- **過少申告 / 隠蔽**：過少申告加算税 10-15% / 重加算税 35-40%
- **インボイス 登録 後 申告 怠る**：取引先 信用毀損 + 適格請求書発行事業者 登録取消 リスク
- **未申告 持続**：税務調査 → 7 年遡及 / 差押 / 公売
- **更长期**：在留期間更新 / 永住 で 公的義務不履行 評価 / 経営・管理 では 事業継続性 否認 リスク

## 该怎么处理（3 种典型路径）

### 路径 A：期限内 申告 + 納付（一般 / 簡易 / 2 割特例 を 選択）
1. 課税方式 確認：
   - 一般課税：仕入 / 経費 の 消費税 を 全額控除（書類保存 + 帳簿要件 厳格）
   - 簡易課税：「簡易課税制度選択届出書」事前提出 + 売上 ¥5,000 万円以下 + みなし仕入率（卸 90% / 小売 80% / 製造 70% / その他 60% / サービス 50% / 不動産 40%、⚠️ 取得日时点 確認）
   - 2 割特例：インボイス 移行 経過措置、令和 5 年 10 月 - 令和 8 年 9 月 課税期間 → 売上税額 × 2 割
2. 申告書 作成：[国税庁 確定申告書等作成コーナー](https://www.keisan.nta.go.jp/)（消費税 対応）
3. 提出：e-Tax / 郵送 / 税務署 窓口
4. 納付：振替納税 / クレカ / コンビニ / 銀行 / ダイレクト納付
5. 帳簿 / 請求書 / 領収書 を 7 年保存（インボイス は 適格請求書 保存 必須）

### 路径 B：期限後申告 / 修正申告 / 還付申告
触发条件：期限超過、誤り、輸出 / 設備投資 で 仕入税額 > 売上税額 → 還付。
1. 還付申告：早めの 提出 で 還付加算金 早期受領
2. 修正申告：自主提出で 加算税 5%
3. 期限後申告：早期 = 加算税 軽減

### 路径 C：異議申立 / 不服申立
触发条件：税務調査 後 更正処分 / 決定処分 / 課税事業者 認定 に 不服。
1. 処分通知 受領翌日から **3 ヶ月以内** に 「再調査の請求」/ 「審査請求」
2. インボイス 登録取消 処分 → 同様 不服申立可
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士 / 税理士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 経営・管理：法人 / 個人事業 で 消費税 課税事業者 = ほぼ 全例 該当。申告 納付 履歴 = 事業継続性 + 公的義務履行 の 直接証拠。
- 在留期間更新：法人 / 個人事業 の 場合 「納税証明書（その 1：消費税）」が 標準提出書類。滞納 / 未申告 = 即マイナス。
- 永住申請：自営業者 / 経営者 は 直近 5 年 の 消費税 申告 + 納付 状況 が 審査対象。
- インボイス 登録 = 課税事業者化 → 急に 消費税 申告義務 発生 した 個人事業主 の 場合、初年度 「2 割特例」適用 で 負担軽減 可能だが、要件 + 期間 取得日时点 で 確認要。
- 「免税事業者 のまま」選択 → 取引先 仕入税額控除 不可 → 値引要請 / 取引停止 リスク。但し 業態（BtoC / 給与所得者 取引メイン）次第。
