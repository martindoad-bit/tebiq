# 在留資格変更 申請 結果通知（許可 / 不許可）

## 信封长什么样
- 寄件人：申請先の地方入管局（「東京出入国在留管理局」「名古屋出入国在留管理局」等）
- 形式：茶色 葉書 或 定形封筒、封面印「在留資格変更許可申請に係る通知」
- 同梱物：申請受理票（受付番号）、返信用封筒（窓口提出時には無し）
- オンライン申請の場合は メール + マイページ 通知が先行、紙は出頭時 / 後日

## 这是什么
留学 → 技人国 / 家族滞在 → 経営・管理 / 技人国 → 高度専門職 など、現在の 在留資格 を別の 在留資格 に変更する申請への結果通知。許可 / 不許可 + 理由 + 新在留資格・期間 + 出頭指示 が記載される。

## 为什么收到
本人または取次が「在留資格変更許可申請書」を提出済み。審査期間は通常 2 週間〜3 ヶ月、変更先（特に 経営・管理 / 高度専門職 / 永住者）と繁忙期で大きく変動する。

## 必看的关键栏
- **申請番号 / 受付番号**
- **結果**：「許可」「不許可」
- **新在留資格 / 在留期間 / 就労制限の有無**
- **出頭期限**（通常通知発行から 14 日 ～ 1 ヶ月）
- **持参物**：旅券、現 在留カード、収入印紙 ¥4,000、通知書、申請受理票、写真追加要求がある場合あり
- **不許可理由 / 担当審査官 / 出頭指定日**

## 不处理的后果（時間梯度）
- **出頭期限 経過**：許可済みでも 新 在留カード 未交付。旧 在留資格 期限後は 不法滞在 リスク
- **不許可 + 旧 在留資格 期限切れ寸前**：「特定活動（出国準備）30 日 / 31 日」へ自動変更されるケースあり、それも経過すれば オーバーステイ
- **オーバーステイ**：退去強制手続、上陸拒否期間 5 年（通常）/ 10 年（再犯）
- **不許可履歴**：永住 / 帰化 / 次回 在留資格変更 すべての審査で 過去申請履歴 として参照

> 入管法 第 20 条（在留資格の変更）、第 22 条の 4、第 24 条

## 该怎么处理（3 种典型路径）

### 路径 A：許可 → 在留カード 受領
1. 出頭指定日 までに 指定入管局 へ持参物揃えて出頭。
2. 収入印紙 ¥4,000 → 新 在留カード 交付。旅券に 証印（or カード 一体）。
3. 在留資格 変更により 就労制限 が変わった場合、雇用主へ 在留カード 写し提出。
4. 14 日以内、市役所 / 区役所 で 住所 / 所属機関 の届出が必要なケースあり。
5. 档案归档：新 在留カード 両面、変更許可通知書、領収印紙 控え、申請理由書 控え。

### 路径 B：不許可 → 再申請 / 出国準備 / 別資格 検討
1. 出頭日 に 担当審査官 へ 不許可理由 詳細聴取。理由概要のみ書面記載、詳細は口頭。
2. 不許可後も **元の 在留資格 が残期間ある** 場合、その範囲で滞在継続可能（例：留学のまま 残期間内に再就活）。
3. 元の 在留資格 期限切れの場合：「特定活動（出国準備）」に変更、30 日 / 31 日 で出国 or 別資格 申請。
4. 不許可 直後の **再申請** は 同一理由 なら 再不許可 がほぼ確定。理由補強・書類差替が前提。

> # ⚠️ 需创始人/書士复核：在留資格変更 不許可 後の「特定活動（出国準備）」付与の運用と、元在留資格 残期間 中の他資格 再申請 可否の整理

### 路径 C：弁護士 / 行政書士 緊急対応
- 留学 → 技人国 不許可（職務内容と専攻 不一致 が理由）/ 経営・管理 への 変更不許可（出資要件・事業計画 弱）/ 高度専門職 ポイント計算誤りなど、構造的な問題がある場合
- 在留期限 切れ間近 / 出国準備 中 / 退去強制 着手前
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **不許可履歴** は永住・帰化審査で「過去申請履歴」として参照。複数回 不許可 は 素行善良性 を疑われる材料。
- **変更直後の 在留期間 は通常 1 年**：以降の更新で 3 年 / 5 年 へ伸長 → 永住要件 の「最長在留期間（現行運用 3 年以上）」を満たす必要。
- **経営・管理 への変更**：500 万円出資（2024 年改正後 段階的に 3000 万円方向）、事務所要件、常勤職員 要件 の充足を変更時点で証明 → 続く 1 年更新時に 決算書・事業継続性 が問われる。
- **高度専門職 1 号 → 2 号** への変更通知も この書式。1 号 3 年継続 + 70 点維持 が前提。
