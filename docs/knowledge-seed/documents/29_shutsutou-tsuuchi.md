# 出頭通知書 / 召喚状（在留・刑事 関連）

## 信封长什么样
- 寄件人：
  - 入管 系：地方入管局（「東京出入国在留管理局」等）
  - 警察 系：所轄警察署 / 警視庁 / 道府県警
  - 検察 系：地方検察庁 / 区検察庁
- 形式：定形封筒、白地 / 茶色、封面に「親展」/「重要」/「出頭通知書」/「召喚状」
- 同梱物：日時 / 場所 / 担当部署 / 担当者名 / 内線 / 持参物 / 出頭しない場合の措置

## 这是什么
本人 を **公的機関の指定窗口** へ出頭させる命令文書。代表的な 3 系統：
1. **入管 系**：在留資格 審査結果 説明 / 不許可理由 説明 / 違反調査 / 退去強制手続
2. **警察 系**：参考人 / 被疑者 として事情聴取 / 取調べ
3. **検察 系**：被疑者 取調べ / 略式手続 案内 / 起訴前面談

## 为什么收到
- **入管**：不許可通知 + 出頭日 / 違反疑いの調査（資格外活動 / 不法就労 / オーバーステイ 等）
- **警察**：交通違反（人身事故 / ひき逃げ / 飲酒運転）/ 万引き / 暴力 / 詐欺 等の被疑者・参考人
- **検察**：警察 から送致された事件の被疑者 取調べ、略式命令 検討段階

## 必看的关键栏
- **出頭日時 / 場所 / 部署 / 担当者 / 内線**
- **出頭の趣旨**：「事情聴取」「取調べ」「説明」「弁解録取」等
- **持参物**：旅券、在留カード、本通知書、運転免許証、印鑑、本人確認書類
- **出頭しない場合の措置**：再通知 / 強制捜査 / 逮捕状 / 退去強制手続 等
- **任意 / 強制 区分**：「任意」記載でも実質義務、不出頭は 不利益

## 不处理的后果（時間梯度）
- **入管 出頭通知 不出頭**：再通知 → 退去強制 容疑者 として 違反調査 着手、収容も
- **警察 出頭要請 不出頭（任意）**：再要請 → **逮捕状** 請求 → 強制捜査 / 逮捕 へ進むケース
- **検察 召喚 不出頭**：勾引状（刑事訴訟法 第 152 条）/ 逮捕状
- **入管法 違反 + 不出頭**：退去強制手続 + 上陸拒否期間 5 年（通常）/ 10 年（再犯）/ 永久（重大事由）
- **刑事処分 + 在留資格**：1 年以下の懲役 / 禁錮 / 罰金 で 在留資格 取消（2024 年 6 月 改正後 永住者 も対象）

> 入管法 第 24 条（退去強制事由）、第 22 条の 4（在留資格 取消）、刑事訴訟法 第 198 条（任意出頭）、第 152 条（勾引）

## 该怎么处理（3 种典型路径）

### 路径 A：出頭（標準対応）
1. 通知書 を 写真撮影 + 写し保管。
2. 持参物 を出頭前日までに準備（旅券、在留カード、運転免許証、本通知書、印鑑、本人確認書類）。
3. 指定日時 厳守、5-10 分早く到着。
4. 担当者 と面談、聴取内容 / 質問 をメモ（録音可否は要確認）。
5. **黙秘権 / 弁護人選任権 / 通訳要請権** を理解。日本語に不安あれば **通訳 を必ず要請**（無料）。
6. 出頭後、面談記録 を 档案归档。

### 路径 B：日程調整（やむを得ない事情）
1. 通知書 記載の **担当者 内線** へ即電話。
2. 出張 / 入院 / 試験 等 やむを得ない事情を伝達 → 別日 指定 / 延期。
3. 書面回答は通常なし、電話メモ + 担当者名 + 通話日時 を記録。
4. 無断不出頭 は絶対に避ける。

### 路径 C：弁護士 / 行政書士 緊急対応
- **警察 / 検察 系**：被疑者 として呼ばれた場合、**刑事弁護人 を出頭前に依頼** 推奨（当番弁護士制度 / 法テラス）
- **入管 系 違反調査**：退去強制 容疑者 / 収容リスクある段階、入管 専門 行政書士 / 弁護士 同行
- 在留資格 取消 / 退去強制 手続 着手前 → 緊急対応窗口
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **出頭履歴 自体は記録に残る**。聴取結果 / 処分内容（処分なし / 不起訴 / 罰金 / 起訴）が 続签・永住・帰化 で参照
- **罰金以上の刑事処分** = 素行善良性 否定の重大材料、永住・帰化 に直接影響
- **2024 年 6 月 入管法改正**：永住者 でも「1 年以下の懲役 / 禁錮 / 罰金」で 在留資格 取消対象
- **入管 違反調査** で「資格外活動」「不法就労」認定 = 在留資格 取消 + 退去強制 + 上陸拒否期間
- **不起訴処分** でも履歴 は残る。続签 / 永住申請 で「過去の犯罪歴」欄に記載必要（虚偽記載 = 在留資格 取消事由）

> # ⚠️ 需创始人/書士复核：出頭通知（任意 / 強制）の刑事訴訟法 上の整理と、在留資格 への影響範囲（不起訴 vs 起訴猶予 vs 略式罰金 の区別）
