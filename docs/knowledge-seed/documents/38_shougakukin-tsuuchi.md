# 奨学金 採用 / 振込通知書 / 奖学金 录用·汇款通知

## 信封长什么样
- 寄件人：「独立行政法人 日本学生支援機構（JASSO）」「○○都道府県 育英会」「公益財団法人 ○○奨学会」
- 封面印字：「奨学金 採用候補者決定通知 在中」「奨学金 振込通知書」「重要書類 親展」
- 颜色：白色 + JASSO ロゴ（青）/ 育英会 ロゴ
- 同梱物：採用通知 / 貸与・給付 区分 案内 / 月額 / 振込口座 / 返還誓約書（貸与 のみ）/ 連帯保証人 / 機関保証 案内 / 在学届・休学届・異動届 様式

## 这是什么
学生 / 留学生 / 子女 進学者 に対する 経済的支援。日本学生支援機構（JASSO）= 国の制度（貸与型 第一種・第二種、給付型）。都道府県 育英会 / 公益財団法人 = 地方・民間 制度。区分：(1) **給付型**（返還不要、住民税 非課税世帯 / 準ずる世帯 + 学業成績）、(2) **貸与型 第一種**（無利子）、(3) **貸与型 第二種**（有利子、年 3% 上限）。本通知 = 採用 / 振込開始 = 法的に「貸与契約 / 給付決定」が成立。

## 为什么收到
- 高校 3 年（予約採用）/ 大学入学後（在学採用）の 申込 → 採用候補者決定 通知
- 採用 → 進学届 提出 → 月次 振込開始
- 給付額 / 貸与額 改定（収入区分 変更 / 学年進行）→ 改定通知
- 留学奨学金（JASSO 海外留学支援 / 民間財団）採用 → 採用通知 + 渡航前 説明会 案内

## 必看的关键栏
- **奨学金 種類**：給付型 / 貸与第一種（無利子）/ 貸与第二種（有利子）
- **月額**：給付（¥29,200-¥75,800、自宅通学 / 自宅外 + 国公立 / 私立 + 区分 I/II/III）/ 貸与第一種（¥20,000-¥64,000）/ 貸与第二種（¥20,000-¥120,000）
- **採用期間**（在学期間中、最大 標準修業年限）
- **保証**：人的保証（連帯保証人 + 保証人 各 1 名）/ 機関保証（保証料 月額 控除）
- **振込口座**：本人名義 + 金融機関 / 支店 / 口座番号 — **誤入力 = 振込延滞 主因**
- **返還開始**（貸与 のみ）：卒業 後 7 ヶ月目 から 月賦 / 半年賦 + 月賦 併用
- **返還免除制度**：教員 / 研究者 採用 / 障害 / 死亡 等

## 不处理的后果（时间梯度）
- **進学届 / 在学届 未提出**：採用 内定 取消 → 振込 開始 されない
- **振込口座 届出 不正 / 残高凍結**：振込延滞 → 個別連絡 + 口座変更届
- **異動届（休学 / 退学 / 転学 / 留学）未提出**：給付 / 貸与 継続 → 過払い分 後日 一括返還請求
- **返還 滞納 3 ヶ月**：個人信用情報機関（KSC）に 「延滞 情報」登録 → クレジットカード / 住宅ローン 審査 影響
- **返還 滞納 9 ヶ月**：法的措置（支払督促 / 訴訟）→ 給与 / 預金 差押 可能
- **継続滞納**：連帯保証人 / 保証人 への 一括請求 → 機関保証 の場合 保証会社 が代位弁済 → 保証会社 から 求償

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 進学届 → 月次受領 → 卒業後 返還（貸与 の場合）
1. 進学先 大学 経由 で 進学届 をスカラネット 入力（採用通知 後 30 日以内 等）
2. 連帯保証人 / 保証人 印鑑証明書 + 収入証明 等 提出 / または 機関保証 申込（保証料 月額 控除）
3. 月次 振込確認 → 学費 / 生活費 充当
4. 卒業後 7 ヶ月目 から リレー口座（本人 引落口座）から 自動引落 で 返還開始
5. 返還完了 通知 を 5 年保管（住宅ローン 審査 等 で 必要）

### 路径 B：返還 困難 → 減額返還 / 返還期限猶予 / 返還免除
- **減額返還**：月額 を 1/2 / 1/3 に減額、返還期間 延長（最長 15 年）
- **返還期限猶予**：1 年単位 で 最長 通算 10 年（経済困難 / 災害 / 傷病 / 失業 等）
- **返還免除**：本人 死亡 / 重度障害 → 全額免除。研究者 / 教員 特定職 採用 → 一部免除（旧制度 / 一部 給付型）
- 申請窓口：JASSO スカラネット・パーソナル / 育英会 / 財団 個別

### 路径 C：留学 / 異動 時 届出
1. 留学 / 休学：「異動届」スカラネット 提出 → 給付・貸与 一時停止 → 復学届 で 再開
2. 退学 / 転学：「退学届」「転学届」→ 給付 / 貸与 終了
3. 子女 の 在留資格 が 「留学」「家族滞在」 で 留学先 帰国 等 在留資格 関連 紛争 を 伴う case、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- JASSO 奨学金 受給 は 在留資格 審査 で **マイナス 評価 ではない** — 教育投資 + 学業継続 の 客観証拠
- 永住申請 「独立生計要件」: 奨学金 = 一時的 借入 / 給付、収入 / 資産 とは 異なる 性質。返還 履歴 は 「公的義務履行」評価 対象
- 返還 滞納 → 個人信用情報 延滞 登録 → 永住申請 / 帰化申請 「素行善良 + 公的義務履行」 評価 マイナス（直接条文 ない が 運用上 確認事項）
- 子女 の 在留資格 が 「留学」 で JASSO 給付奨学金 受給中 → 卒業後 「技人国」「特定技能」「高度専門職」 等 在留資格変更 申請 時 経歴 として 申告
- 経営・管理 / 高度専門職 子女 が 大学進学 時 給付奨学金 区分 I（住民税非課税）非該当 多 — 親 高所得 のため 第二種（有利子）が 主流

返還 滞納 = 個人信用情報 機関（KSC）登録 = 在留資格 / 永住 審査 で 「公的義務不履行」評価 直結 リスク。返還義務 を 軽視 しない。
