# 児童 入学準備 案内 / 儿童 入学准备 通知

## 信封长什么样
- 寄件人：「○○市 教育委員会」「○○区 教育委員会 学務課」「○○小学校」「○○中学校」
- 封面印字：「新入学 準備のご案内 在中」「入学準備 関係書類」「就学時健康診断 のお知らせ」
- 颜色：白色 / 浅黄色 角 2 号 + 自治体・学校 ロゴ
- 同梱物：入学説明会 案内 / 学用品 / 制服 / 通学用品 リスト / 就学時健康診断 案内 / 入学準備金（就学援助 該当者）案内 / 通学路 図 / 給食費 / PTA 入会 案内

## 这是什么
学齢期 子女 が 4 月 入学 を 控える 前年 11 月 - 当年 3 月 に 教育委員会 / 学校 から 送付 される 入学準備 一式 案内。学校教育法 + 学校保健安全法 + 自治体 条例 に基づく 行政手続 と、学校 / PTA 主導 の 任意手続 が 混在。本通知書 自体 = 「行政処分」ではなく 「事前案内 + 持参物 / 提出書類 リスト」が中心。就学通知書（参见文档 #33）と 別物 / 並行 で送付 される ケース 多。

## 为什么收到
- 当年 4 月 1 日 時点 6 歳 / 12 歳 子女 の 保護者 全員 → 11 月頃 一斉発送
- 転入後 学齢簿 登録 → 入学日 まで 1-2 ヶ月以内 に 個別送付
- 学校選択制 利用申込 後：選考結果 と 併せて 入学準備 案内 送付
- 私立 / 国立 / インターナショナルスクール 進学 → 公立校 案内 は 「就学校変更届」提出 後 停止

## 必看的关键栏
- **入学日・入学式 日時**（4 月 6-9 日 多数）
- **就学時健康診断 日時**（小学校 入学前 10-11 月、無料、自治体 指定医療機関）
- **入学説明会**（2-3 月、保護者 出席 必須）
- **学用品 / 制服 / ランドセル 注文 期限**
- **入学準備金（就学援助 該当者）**：申請 → 入学前 3 月 / 入学後 7 月 振込（自治体 で 異なる）
- **通学路 / 集団登校 案内**
- **アレルギー / 既往歴 申告 / 健康調査票 提出 期限**
- **問合せ先**：教育委員会 学務課 / 入学予定校 事務室

## 不处理的后果（时间梯度）
- **就学時健康診断 欠席**：個別 受診案内 → 入学日 までに 受診（学校保健安全法 第 11 条 / 12 条）
- **入学説明会 欠席**：説明資料 後送 + 個別連絡、但 持参物 リスト 共有 遅れ → 入学式 当日 不備 リスク
- **入学準備金 申請 期限超過**：当年度 給付 不可、翌年度 通常 給付 を 受領
- **アレルギー 健康調査 未提出**：入学後 給食 対応（除去食 / 代替食）遅延 → 食物アレルギー 子女 に 直接 リスク
- **学用品 / 制服 注文 遅延**：入学式 / 入学後 1 週間 で 一部 不備 → 学校 で 個別対応

## 该怎么处理（3 种典型路径）

### 路径 A：通常 入学準備
1. 就学時健康診断（小学校 入学前 10-11 月）受診
2. 入学説明会（2-3 月）出席 → 持参物 / 提出書類 リスト 受領
3. 学用品 / 制服 / ランドセル / 通学用品 注文（量販店 / 学校指定店）
4. 提出書類（誓約書 / 緊急連絡先 / 健康調査票 / 食物アレルギー 申告書 / 通学路 確認書 等）を 期限内 提出
5. 入学式 出席 → 第 1 週 に 持参物 確認 → 翌週 から 通常授業

### 路径 B：入学準備金（就学援助）申請
触发条件：世帯所得 が 就学援助 認定基準 以下（参见文档 #34）。
1. 「入学準備金 申請書」+ 課税証明書 / 給与所得 源泉徴収票 / 児童扶養手当 受給証明 等 を 教育委員会 / 学校 経由 で 提出（11-12 月）
2. 認定 → 入学前 3 月（前倒し支給 自治体）/ 入学後 7 月（後払い支給 自治体）に 保護者 口座 振込
3. 給付額：小学校 ¥54,060 / 中学校 ¥63,000 程度（2024 年度 文科省 補助単価、自治体 で 増額あり）
4. 領収書 / 振込明細 を 5 年保管

### 路径 C：転入 / 私立 / 国立 / インター 進学 / 入学猶予
- **転入**：転入先 自治体 で 「転入届」+ 学齢簿 登録 → 教育委員会 が 改めて 案内 発送 → 指定校 と 連絡 → 持参物 / 提出書類 個別調整
- **私立 / 国立 / インター**：「就学校変更届」+ 入学許可書（写）を 教育委員会 に 提出 → 公立校 学齢簿 削除 / 案内 停止
- **入学猶予 / 免除**：海外居住 / 重度障害 等 → 「就学猶予 / 免除願出」（学校教育法 第 18 条）→ 教育委員会 認定
- 子女 国籍 / 在留資格 / 入管 関連 紛争（在留資格 喪失 / 子女 在留資格 変更 等）伴う case、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 入学準備 案内 自体 は 在留資格 審査 と 無関係 の 行政・学校 案内
- 但 子女 の 公立校 入学準備 = 「家族同居 + 在日生活 定着」 の 客観証拠 → 永住申請 / 帰化申請 「居住要件 + 国益適合」 の 補強資料
- 入学準備金（就学援助）受給歴 = 世帯所得 が 認定基準 以下 を 客観証明 → 永住 「独立生計要件」評価 で マイナス 影響 リスク（参见文档 #34）
- 経営・管理 / 高度専門職 高所得層 は 入学準備金 認定基準 非該当 多 — 通常 路径 A
- 子女 が 「家族滞在」「定住者」「永住者の配偶者等」在留資格 で 公立小・中 入学 → 在留資格 制限 なし、無償（外国籍 子女 への 公立校 受入 は 国際人権規約 + 文科省 通知 で 保障）

> # ⚠️ 需创始人/書士复核：入学準備金（就学援助 入学準備金 給付）受給歴 が 永住申請 「独立生計要件」 で どの程度 評価対象 になる か — 入管 公開ガイドライン に 明示 なく、世帯収入 全体 評価 領域。
