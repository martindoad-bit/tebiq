# 水道料金 滞納督促 / 給水停止予告

## 信封长什么样
- 寄件人：「東京都水道局」「大阪市水道局」「○○市上下水道局」「○○市水道部」等自治体名
- 封面印字：「水道料金等納入のお願い」「督促状」「給水停止予告書」赤帯
- 同梱物：払込取扱票（コンビニ用バーコード付）、未納明細、給水停止予定日案内
- 自治体直営のため、市役所 名義の催告書として届く場合あり

## 这是什么
自治体（市町村）水道局 / 上下水道局 が、水道料金 + 下水道使用料 の滞納者に発する督促書。最終段階は「給水停止予告書」で、指定日までに支払なき場合 給水停止 を実施。地方自治法 + 水道法 + 各自治体 水道事業給水条例 に基づく。

## 为什么收到
水道料金 + 下水道使用料 の納期限（請求書発行から約 20-30 日）を経過後、入金未確認のため。標準的タイムライン：
- 納期限経過 約 20 日 → 督促状（最初）+ 督促手数料 ¥100 程度加算（自治体条例）
- 約 30-50 日 → 催告書 / 給水停止予告書
- 約 50-90 日 → 給水停止 + 滞納処分（差押）

## 必看的关键栏
- **未納金額**：水道料金 + 下水道使用料 + 督促手数料 + 延滞金
- **督促手数料 / 延滞金**：自治体条例で定額（多くは ¥100-¥200）+ 年率（多くは年 14.5% 等、地方税法準用）
- **納期限 / 給水停止予定日**：太字 / 赤字
- **払込方法**：コンビニ / 銀行 / 水道局窓口 / 口座振替再振替 / Web 支払（自治体により ペイジー / クレジット / PayPay 対応）
- **問合せ先**：水道局営業所 / 上下水道局 料金センター
- **お客さま番号 / 水栓番号**

## 不处理的后果（時間梯度）
- **納期限経過 20 日**：督促状送付、督促手数料 + 延滞金 加算開始
- **30-50 日**：催告書 / 給水停止予告書（「○月○日までに納付なきとき給水停止」）
- **50-90 日**：給水停止（メーター栓閉鎖）。再開には全額納付 + 開栓立会必要
- **長期滞納**：水道料金は **公金（地方自治体の歳入）** のため、地方税同様 **滞納処分（差押）** の対象。給与・預貯金・財産差押の可能性
- **賃貸物件 退去時**：未払額は契約者本人請求、保証会社経由で 強制執行 リスク

> ⚠️ 延滞金率・督促手数料は自治体条例で異なる。請求書 / 督促状記載を確認。

## 该怎么处理（3 种典型路径）

### 路径 A：全額納付で給水停止回避
1. 督促状同封 払込取扱票 で コンビニ / 銀行 / 水道局窓口 で納付。
2. 自治体により Web 支払（ペイジー / クレジット / モバイル決済）対応。
3. 支払反映 1-2 営業日。停止予告日直前なら水道局営業所に納付完了を電話連絡。
4. 領収書 / 支払完了画面 保管。

### 路径 B：分割納付 / 納付相談 申請
1. 水道局 料金センター（東京都水道局：03-5326-1101 等）に電話、「納付相談」希望と伝達。
2. 失業 / 病気 / 生活困窮 等の事情を申告 → 分割納付 / 納期限延長 / 減免（自治体により制度あり）を相談。
3. 合意内容を書面で受領、納付計画通り履行。
4. 自治体 生活困窮者自立支援窓口 併用可。

### 路径 C：給水停止後 / 名義違い / 滞納処分
- 既に 給水停止 → 全額納付 + 開栓申込（立会必要）。当日 or 翌日再開。
- 入居前の旧入居者宛督促 → 賃貸借契約書 写し送付で名義違い対応。
- 差押予告 / 滞納処分通知 到着 → 給与・預貯金 差押直前。即時納付 or 分納協議。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
水道料金 は **公金** だが、 出入国在留管理庁 の続签 / 永住 審査では **公租公課（税・年金・健康保険）** が直接対象。水道料金 滞納単独での不利益は通常なし。**ただし**：
- **滞納処分（差押）に発展**した場合は公租公課類似の履歴扱いとなる事例あり、永住「素行善良」「独立生計」評価に間接影響
- 経営・管理 では事務所水道滞納 → 事業の継続性 説明資料に影響する可能性

> ⚠️ 永住「素行善良」運用は事案毎、差押履歴 を伴う場合のみ実務的に問われる。
