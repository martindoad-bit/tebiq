# 雇用保険 受給資格者証 / 失业保险 受给资格证

## 信封长什么样
- 寄件人：公共職業安定所（ハローワーク）/ 厚生労働省 都道府県労働局
- 封面常见印字：「雇用保険 関係書類」「親展」
- 通常不是 邮寄。多数 在ハローワーク 窗口 当場 受取（離職票 提出 + 受給資格決定 の日 から 約 7 日後 説明会 で交付）
- 同梱物：失業認定申告書（雛形）/ 説明会案内 / 求職活動 実績記入用紙 / 雇用保険 受給者のしおり

## 这是什么
雇用保険 の 基本手当（一般所謂「失業給付」）の受給資格 が認定された証明書。表面 に 離職理由 / 所定給付日数 / 基本手当日額 / 受給期間 が記載され、以後 4 週間ごと の 失業認定日 に 持参・提示 する 義務文書。受給期間 中 の 求職活動 履歴 もこの証 の裏面 に スタンプ で記録される。

## 为什么收到
- 退職 後、前職 の 離職票-1 / 離職票-2 を ハローワーク へ提出 + 求職申込 を行った
- 離職前 2 年間 に 雇用保険 加入期間 通算 12 ヶ月以上（特定理由離職者 / 特定受給資格者 は 6 ヶ月以上）
- 受給資格決定 + 7 日間 の 待期期間 経過後 + 雇用保険 説明会 出席 で交付

## 必看的关键栏
- **離職理由コード**（11-55）— 自己都合 / 会社都合 / 特定理由 を区分。自己都合 は 2 ヶ月 給付制限 あり
- **基本手当日額**（年齢区分 で上限 ¥7,065-¥8,490 / 2024 年 8 月以降）
- **所定給付日数**（被保険者期間 + 離職理由 + 年齢 で 90-360 日）
- **受給期間 満了日**（原則 離職日 翌日 から 1 年）
- **次回 認定日**（4 週間 ごと、日時 + 担当窓口）
- **失業認定 必要 求職活動 実績数**（原則 2 回以上 / 認定対象期間）

## 不处理的后果（時間梯度）
- **認定日 欠席**：当該認定対象期間 の 基本手当 不支給。次回 認定日 まで 持越 不可。
- **求職活動 実績 不足**：当該期間 の 給付 不支給。
- **不正受給**（虚偽申告 / 就労 隠蔽）：返還命令 + 不正受給額 の 2 倍 の納付命令（合計 3 倍）+ 詐欺罪 告発 リスク。
- **受給期間 満了日 経過**：未支給 残日数 は **失効**（病気 / 出産 等 やむを得ない 事由 は最大 4 年 延長申請 可）。
- 離職日 翌日 から 1 年 を 1 日 でも 過ぎる と、所定給付日数 を消化 し切れずに 終了。

## 该怎么处理（3 种典型路径）

### 路径 A：通常受給（多数人）
1. 受給資格者証 + 失業認定申告書 + 印鑑 を持参 して 4 週間ごと の 認定日 にハローワーク 出頭。
2. 認定対象期間 中 に 求職活動 実績 2 回以上（原則）— 求人応募 / 職業相談 / セミナー受講 等。
3. 認定後 約 5 営業日 で指定口座 に振込。記録 は 受給資格者証 裏面 に スタンプ で押印。
4. **就職決定 時** は 直ちに ハローワーク へ報告。条件次第 で **再就職手当**（残日数 × 基本手当日額 × 60-70%）受給可。

### 路径 B：給付延長 / 受給期間 延長申請
- **病気・ケガ・出産・育児・介護** で 30 日以上 求職活動 不能 → 受給期間 延長申請（最大 4 年 まで）。働けない 状態 が 30 日経過 した翌日 から **1 ヶ月以内** が原則申請期限（2025 年 4 月以降 規定変更 確認推奨）。
- **公共職業訓練** 受講 → 訓練期間中 + 終了後 まで 給付延長（訓練延長給付）。
- 病気 で 働けない 場合 は 雇用保険 ではなく **傷病手当**（雇用保険）/ 健康保険 の 傷病手当金 と 切替 を検討。

### 路径 C：受給資格 / 給付制限 不服申立
1. 受給資格 不認定 / 離職理由 判定 不服 / 給付制限 期間 不服 等 → **雇用保険審査官** へ 審査請求（処分 を知った日 から **3 ヶ月以内**）。
2. 自己都合 退職 と判定されたが 実態 は 会社都合（パワハラ / 賃金未払 / 労働条件 著しく異なる）の場合 — 在職中 の メール / タイムカード / 給与明細 等 客観証拠 を揃える。
3. 在留資格 と退職 の関係 が複雑 な場合（技人国 で 退職後 3 ヶ月以上 無職 → 在留資格 取消 リスク）、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 雇用保険 受給 自体 は 永住・帰化申請 の 直接 不利益要因 ではない。「公的負担 への 依存」 とは別カテゴリ（雇用保険 は 拠出制、自己が 払った 保険料 から の給付）。
- ただし 技人国 / 高度専門職 等 就労系 在留資格 で 退職後 3 ヶ月以上 無職 → **「契約機関に関する届出」未提出** + **在留資格該当 活動 不実施** で 在留資格取消（入管法 第 22-4 条）リスク。
- 失業中 でも 求職活動 中 と認められれば 即時取消 はないが、ハローワーク の **求職証明** + **活動実績** が 入管 説明 の根拠 になる。
- 永住申請 での「独立生計要件」評価上、雇用保険 受給歴 単体 は問題視 されない。連続性 ある 就労歴 と 直近 安定収入 が重要。
