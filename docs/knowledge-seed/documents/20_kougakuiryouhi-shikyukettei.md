# 高額療養費 支給決定通知 / 高额疗养费 支给决定通知

## 信封长什么样
- 寄件人：健康保険組合（〇〇健康保険組合）/ 協会けんぽ 支部 / 国民健康保険連合会（市役所 国保年金課 経由）/ 後期高齢者医療広域連合
- 封面常见印字：「高額療養費 支給決定通知書 在中」「親展」「重要書類」
- 颜色：白底 + 保険者 ロゴ
- 同梱物：支給決定通知書 1 枚 / 計算明細（受診月 / 医療機関 / 自己負担額 / 限度額 / 支給額）/ 振込口座 確認

## 这是什么
1 ヶ月（暦月：1 日 〜末日）の 医療費 自己負担額（窓口 3 割 等）が 所得区分別 自己負担限度額 を超えた場合、超過分 を払戻 する 法定給付 の 支給決定通知。70 歳未満 / 70 歳以上 / 後期高齢者 で 区分 + 計算式 が異なる。事前 に「限度額適用認定証」 を取得 し医療機関 窓口 で提示 すれば 立替不要、事後請求 はこの場合 発生しない。

## 为什么收到
- 単月 で 窓口自己負担 が 自己負担限度額 を超過 + 限度額適用認定証 不提示 + 全額立替払
- 健保組合 / 協会けんぽ：レセプト 受領 後 自動的 に審査 → 支給決定（多くは 受診月 から 3-4 ヶ月後）
- 国民健康保険：被保険者 本人 が 市役所 で申請 → 審査 → 支給決定
- 高額医療・高額介護合算療養費（年度 単位）も同様 通知

## 必看的关键栏
- **被保険者番号 / 記号番号**
- **対象 受診月**（〇年〇月分）
- **医療機関名 + 自己負担額**
- **適用区分**（ア / イ / ウ / エ / オ — 70 歳未満 / 標準報酬月額 別）
- **自己負担限度額**（区分別 計算式）
- **支給金額**（自己負担額 - 限度額）
- **振込予定日 + 振込口座**

## 不处理的后果（時間梯度）

**支給決定 の場合**：
- 受領 のみで 行動義務 なし
- ただし **不支給 / 計算誤り** の場合 審査請求 期限：3 ヶ月以内

**消滅時効**：
- 高額療養費 請求権 の時効 は **2 年**（健康保険法 / 国民健康保険法 各規定）。診療月 翌月初日 から進行 — 2 年経過分 は失権

**未申請（国保 / 後期高齢者）**：
- 健保組合 / 協会けんぽ は 自動払戻 が多い が、**国保 / 後期高齢者 は 申請主義** が原則。市役所 から「該当 のお知らせ」 が届く 自治体 もある が、届かない自治体 もあり 申請忘れ で 失権 リスク

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 振込確認（多数人）
1. 通知書 + 計算明細 を保管。指定口座 に 通知額 が振込（通常 通知 から 1-2 週間）。
2. 確定申告 で **医療費控除** を併用する場合、自己負担額 から 高額療養費 還付額 を差引 した 純自己負担 を 控除対象 として申告。二重控除 は不可。
3. 翌月以降 も 高額医療 が継続 する見込み なら、**限度額適用認定証** を保険者 に申請（協会けんぽ：[https://www.kyoukaikenpo.or.jp/](https://www.kyoukaikenpo.or.jp/) / 健保組合：勤務先 経由 / 国保：市役所）。次回以降 立替不要。
4. **マイナ保険証** 利用 + オンライン資格確認 対応 医療機関 では 認定証 不要 で 限度額適用 自動化 されつつある（2023 年 以降 普及中）。

### 路径 B：合算 / 多数該当 / 世帯合算 申請
- **多数該当**：直近 12 ヶ月 で 3 回 以上 高額療養費 該当 → 4 回目以降 限度額 が さらに引下（第 4 月以降 軽減）。
- **世帯合算**：同一健保 の 同一世帯 で 同月内 に 21,000 円 以上 の 自己負担 が複数 あれば 合算 して 限度額 計算可。
- **高額医療・高額介護合算療養費**：年度（8 月-翌 7 月）単位 で 医療 + 介護 合算限度額 超過分 を別途 払戻。健保 / 国保 / 介護保険 の連携申請。
- **特定疾病療養受療証**（人工透析 / 血友病 / HIV）：自己負担月額 ¥10,000（上位所得 ¥20,000）に固定。

### 路径 C：不支給 / 計算 異議
1. 通知 受領 から **3 ヶ月以内** に 社会保険審査官（健保）/ 都道府県知事（国保）へ 審査請求。
2. 主な争点：適用区分（標準報酬月額 / 所得）誤り / 食事療養費 / 差額ベッド代 / 保険外併用療養費 の控除誤り / 多数該当 認定誤り / 世帯合算 範囲誤り。
3. 審査請求 と 確定申告 医療費控除 が 同時並行 で複雑 になる場合、税理士 又は 行政書士 へ相談。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 高額療養費 受給 は 健康保険料 / 国民健康保険料 拠出 から の 法定給付。永住・更新 への **直接 不利益 要因 ではない**。
- ただし **健康保険 / 国民健康保険 加入歴 + 保険料 納付歴** は 永住申請 の 「公的義務 履行」 評価項目。高額療養費 受給 の前提 として 保険料 完納 が確認される ことになり、結果的 に 加入・納付歴 の補強材料 にもなる。
- **入院・治療 期間中 の 在留資格** 該当性：
  - 技人国 / 経営・管理 等 就労系：3 ヶ月以上 該当活動 不実施 → 在留資格取消リスク。診断書 + 高額療養費 通知 で 「正当事由」 立証 補強。
  - 短期滞在 → 治療目的 で来日 + 長期療養 が必要 → 「医療滞在」 在留資格（特定活動）への変更 を検討。
- 高額療養費 + 限度額適用認定証 + マイナ保険証 連携 で 立替不要 化 が進行中 — 在留外国人 も 同制度 利用可（住民票 + 在留カード + マイナンバー 取得 が前提）。
