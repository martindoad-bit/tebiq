# 再入国許可書（みなし再入国 含む）

## 信封长什么样
- 寄件人 / 発行：地方入管局 窗口 で **その場交付**（郵送なし）
- 形式：旅券 への 貼付シール / 押印（再入国許可印）+ 印紙貼付
- **みなし再入国** の場合：信封なし、出国時に 空港カウンター で「再入国出国記録（EDカード）」にチェック → 旅券 に「みなし再入国」スタンプ
- 関連書類：申請書 控え（再入国許可申請書）、収入印紙 控え

## 这是什么
中長期在留者 が出国後 再入国 する権利を保証する制度の証票。2 種類：
1. **再入国許可（通常）**：入管局 で事前申請、有効期間 最長 5 年（特別永住者 6 年）、在留期限 を超えない範囲
2. **みなし再入国許可**：1 年以内（特別永住者 2 年以内）の出国は 申請不要、再入国出国記録 にチェック するだけで OK

## 为什么收到
- 再入国許可申請：本人が 入管 窗口で「再入国許可申請書」+ 旅券 + 在留カード + 収入印紙（¥3,000 / ¥6,000）提出 → 即日交付
- みなし再入国：出国当日、空港 / 港 で 再入国出国記録 提出 → 旅券 にスタンプ（事前申請不要）

## 必看的关键栏
- **有効期間**：再入国許可 は最長 5 年、ただし 在留期限 を超えない（短い方が優先）
- **回数区分**：「一回限り」¥3,000 / 「数次」¥6,000
- **みなし再入国** チェック欄：「みなし再入国許可による出国を希望する」へチェック必須（チェック漏れ = 在留資格 失効）
- **出国期限 / 再入国期限**：みなし は 出国日 から 1 年 以内（特別永住者 2 年）、かつ 在留期限 を超えない
- **在留カード**：再入国時 必ず 持参（パスポートだけでは戻れない）

## 不处理的后果（時間梯度）
- **みなし再入国チェック漏れ**：出国時点で 在留資格 失効。日本へ戻るには 在留資格認定証明書（COE）から再申請が必要、数ヶ月〜半年 のロス
- **みなし期限（1 年 / 出国日から）超過**：在留カード 失効、在留資格 喪失、再入国不可
- **再入国許可期間 超過**：同上、在留資格 喪失
- **在留期限 を超える出国**：再入国許可 / みなし いずれも 在留期限 までしか有効。在留期限 切れて帰国 = 不法残留 ではなく 在留資格 喪失（帰国不可）

> 入管法 第 26 条（再入国の許可）、第 26 条の 2（みなし再入国許可）

## 该怎么处理（3 种典型路径）

### 路径 A：短期出国（1 年以内）→ みなし再入国
1. 出国前：旅券 + 在留カード + 再入国出国記録（EDカード）準備。
2. 空港 出国審査：「みなし再入国許可」チェック欄に **必ずチェック**。
3. 帰国時：自動化ゲート / 有人カウンター で 在留カード + 旅券 提示。
4. 1 年以内（特別永住者 2 年）に必ず帰国。在留期限 が 1 年以内に切れる場合は 在留期限 が優先。
5. 档案归档：旅券 出入国スタンプ 写真、再入国出国記録 控え。

### 路径 B：長期出国（1 年超〜5 年）→ 再入国許可（数次）
1. 出国前、地方入管局 窗口で「再入国許可申請書」+ 旅券 + 在留カード + 収入印紙 ¥6,000（数次）提出。
2. 即日、旅券 に 再入国許可シール 貼付（有効期間 最長 5 年、ただし 在留期限 まで）。
3. 在留期限 が 5 年未満なら 在留期限 まで → **延長手続（在外公館での 再入国許可期間延長申請）** が可能（例外あり）。
4. 出国 → 期間内に帰国 → 在留資格 維持。
5. 档案归档：再入国許可シール、申請書 控え、収入印紙 控え。

### 路径 C：弁護士 / 行政書士 緊急対応
- 在外滞在中に **みなし期限 / 再入国許可期限** が迫り帰国困難（病気・災害・家族事情）→ 在外公館 経由の延長 / 例外措置 検討
- 在留カード 紛失 / 旅券 失効 で再入国できない
- 永住者 / 高度専門職 で 1 年以上 の長期出国計画
- 建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- **永住申請の「引き続き 10 年居住」要件**：再入国許可 / みなし再入国 で出国していても 連続性 は維持されるが、**1 回 90 日以上 / 年間合計 100 日以上 の出国** は「引き続き」中断と判断されやすい
- **高度専門職 1 号 → 永住 短縮（1 年 / 3 年）** でも、出国日数 は 居住要件 計算で控除される
- **経営・管理**：長期出国は「事業の継続性」否定材料。3 ヶ月以上連続出国は 続签時 質問対象
- **技人国 / 家族滞在**：1 年以内の短期出国 が 雇用主・配偶者 と整合していれば問題なし

> # ⚠️ 需创始人/書士复核：永住要件「引き続き」の出国日数 具体閾値（90 日 / 100 日 ルールの最新運用）
