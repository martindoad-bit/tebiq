# 就学援助 認定通知書 / 就学援助 认定通知

## 信封长什么样
- 寄件人：「○○市 教育委員会」「○○区 教育委員会 学務課 就学援助係」
- 封面印字：「就学援助 認定通知書 在中」「重要書類」「親展」
- 颜色：白色 角 2 号 / 長 3 号 + 自治体ロゴ
- 同梱物：認定通知書 本体 / 給付項目 一覧 / 給付スケジュール / 口座振替 申込書（未提出 の場合）/ 学校給食費 取扱 案内

## 这是什么
学校教育法 第 19 条 が根拠（経済的理由 で 就学困難 と 認められる 学齢児童・生徒 の 保護者 に対し、市町村 が 必要 な 援助 を与える）。生活保護 受給世帯 = 要保護、それに準ずる 程度 困窮 = 準要保護 として 認定。給付項目：学用品費 / 通学用品費 / 校外活動費 / 修学旅行費 / 給食費 / 医療費 / 体育実技用具費 / 入学準備金 等。給付額 自治体差大、年額 ¥30,000-¥150,000 程度。

## 为什么收到
- 4-6 月：当年度 申請 → 6-7 月 一斉 認定通知
- 年度途中 申請 → 1-2 ヶ月後 個別認定通知
- 再申請（毎年度 申請必要）→ 認定継続 / 再認定 通知
- 所得激変（失業 / 廃業 / 災害）→ 随時申請 → 認定通知

申請しても 不認定 の場合 「不認定通知書」が届く（基準所得 超過、世帯員 範囲 認定誤り 等）。

## 必看的关键栏
- **認定区分**：要保護（生活保護 受給）/ 準要保護（市町村 基準）
- **給付項目 と 金額**：学用品費 / 給食費 / 修学旅行費 / 入学準備金 等
- **給付方法**：保護者 口座 振込 / 学校経由 現物給付（給食費 直接相殺 等）/ 振込時期（多数 7 月 / 12 月 / 3 月 の 年 3 回）
- **適用期間**：当該年度（4 月 - 翌 3 月）/ 月途中 認定時 認定月 から
- **問合せ先**：教育委員会 就学援助係 直通番号

## 不处理的后果（时间梯度）
- **口座振替 申込書 未提出**：給付金 振込先 不明 → 学校経由 現金支給 / 給付遅延
- **再申請 未提出**：翌年度 自動継続 されない 自治体 多 → 援助打切 → 学用品費 / 修学旅行費 等 自己負担
- **所得変動 未申告**：所得増加 で 認定基準 超過 → 過払い分 返還請求（過払い 認識日 から 5 年 時効）
- **転居 未届出**：転入先 自治体 で 改めて申請 必要、援助 一時中断

## 该怎么处理（3 种典型路径）

### 路径 A：認定 → 給付受領 → 翌年度 再申請
1. 同梱 口座振替 申込書 提出（学校経由 / 直接 教育委員会）
2. 給付スケジュール（7 月 / 12 月 / 3 月 等）に従い 自動振込
3. 翌年度 4-6 月 「就学援助 申請書」+ 前年度 課税証明書（マイナンバー 提出 で 省略可 自治体 多）/ 給与所得 源泉徴収票 を 学校経由 / 直接 提出
4. 認定通知書 / 給付明細 を 5 年保管

### 路径 B：所得変動・世帯変動 随時申請
触发条件：失業 / 病気休職 / 配偶者 離別 / 災害 等。
1. 「就学援助費 申請書」+ 「家計状況 申告書」+ 離職票 / 罹災証明 / 確定申告書 等 提出
2. 認定基準 = 生活保護 基準額 × 1.0-1.3 倍（自治体差大、東京 23 区 で 約 1.3 倍 多）
3. 認定 されれば 申請月 から 給付対象、遡及給付 不可 自治体 多

### 路径 C：不認定 異議 / 再申請
触发条件：基準額 微妙超過 / 世帯員 範囲 誤認 / 一時所得 過大評価 等。
1. 不認定通知 受領 から **3 ヶ月以内** に 「審査請求書」を 市町村長 / 教育委員会 に 提出（行政不服審査法 第 18 条）
2. 教育委員会 窓口 で 算定根拠 説明 受領 → 世帯員 範囲 / 控除 漏れ 修正可能なら 再申請 が 速い
3. 算定 が 複雑 / 在留資格 / 戸籍関連 紛争 を伴う場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 就学援助 受給 自体 は 在留資格 審査 で マイナス 評価 ではない（生活保護 とは 異なり 「公的扶助」ではなく 「行政給付」位置づけ）
- 但 永住申請 「独立生計要件」では 世帯 全体 の 安定収入 が 審査対象 → 就学援助 受給 = 世帯所得 が 一定基準 以下 を 客観証明 する 結果 → 独立生計要件 充足 に マイナス 影響 リスク
- 高度専門職 / 経営・管理 の 高所得層 は そもそも 認定基準 を超過 → 該当しない
- 技人国 / 家族滞在 の 中所得層 で 子女 が 多 い場合 認定 ケース あり、永住申請 タイミング を 慎重 に 計画
- 援助 受給中 でも 在留期間更新 自体 は 通常 問題 なし（収入 安定性 + 公的義務履行 が 別途 確認 される）

> # ⚠️ 需创始人/書士复核：就学援助 受給歴 が 永住 「独立生計要件」評価 で どの程度 マイナス か — 入管 公開ガイドライン に 明示 なく、世帯収入 全体 で 判断 される 領域。
