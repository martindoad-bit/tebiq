# 健康保険 給付決定通知書 / 健康保险 给付决定通知

## 信封长什么样
- 寄件人：健康保険組合（〇〇健康保険組合）/ 全国健康保険協会（協会けんぽ）各支部
- 封面常见印字：「給付決定通知書 在中」「健康保険組合」「親展」
- 颜色：白底 + 健保組合 ロゴ
- 同梱物：給付決定通知書 1-2 枚（窓口立替払 / 海外療養費 / 装具療養費 等 の場合）/ 振込口座 確認案内

## 这是什么
健康保険 加入者 が 申請した 各種給付（療養費 / 海外療養費 / 移送費 / 埋葬料 / 出産育児一時金 / 高額療養費 等）に対する 健康保険組合 又は 協会けんぽ の **支給決定** 通知。立替払 した 医療費 の 払戻、又は 法定給付 の支払 を行う旨 を通知 する性格 の文書 で、受領 のみで 行動義務 は基本ない。

## 为什么收到
- 海外 で 受診 し 立替払 → 海外療養費 申請 → 審査完了
- 緊急時 に 保険証 不携帯 で 全額自費 払 → 療養費 申請（10 割払 → 7 割 還付）
- 治療用 装具（コルセット / 義眼 / 弾性ストッキング 等）作成 → 療養費（補装具）申請
- 入院移送 費 → 移送費 申請
- 出産育児一時金 / 家族出産育児一時金（直接支払制度 不利用 の場合）
- 埋葬料 / 家族埋葬料（被保険者 又は 被扶養者 死亡）

## 必看的关键栏
- **被保険者番号 / 記号番号**
- **給付項目**（療養費 / 海外療養費 / 移送費 / 埋葬料 / 出産育児一時金 等）
- **決定金額**（支給額 / 不支給 / 一部支給）
- **振込予定日 + 振込先口座**
- **不支給 / 一部支給 の場合 の理由**（保険適用外 治療 / 標準額 超過分 等）
- **問合せ先**（健保組合 給付課 / 協会けんぽ 支部）

## 不处理的后果（時間梯度）
- 給付決定通知 自体 に **行動義務 はない**（給付受領通知）
- ただし **不支給 / 一部支給 の場合**：審査請求 期限 経過 で確定
- 審査請求 期限：通知 受領 から **3 ヶ月以内**（健康保険法 第 189 条 / 社会保険審査官）
- 振込口座 不正 / 解約 → 健保 から再連絡 入れば 速やかに 新口座 を届け出
- 信用 / 在留 への 影響 は通常なし

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 振込確認（多数人）
1. 通知書 + 振込予定日 を確認。指定口座 に通知書 記載額 が振込まれる（通常 通知 から 1-3 週間以内）。
2. 通知書 + 領収書 写し + 健康保険 給付申請書 控え を **5 年間** 保管（消滅時効 2 年 + 確定申告 医療費控除 連動 のため）。
3. 同年中 に 高額療養費 / 医療費控除（確定申告）も併用 する場合、給付決定通知書 の支給額 を 控除対象医療費 から 差引 する。

### 路径 B：不足金額 / 追加申請
- **海外療養費 の 厚労省 標準額 部分 のみ支給** → 自費負担 が残る。原則 制度上 これ以上 の支給 不可。海外旅行 保険 / クレジットカード 付帯保険 へ別途請求 が現実的。
- **療養費 計算誤り** → 領収書原本 + 診療明細書 を再提出 し 再審査依頼。
- **附加給付**（健保組合 独自）の併給 がある場合、自動 反映 されない 健保 もあり別途申請。

### 路径 C：不支給 / 一部支給 への 審査請求
1. 通知 受領 から **3 ヶ月以内** に 社会保険審査官 へ 審査請求（健康保険法 第 189 条）。
2. 主な争点：保険適用範囲 認定誤り / 必要医療性 否認 / 海外療養費 標準額 算定誤り / 装具療養費 必要性 否認。
3. 不服申立 を 専門的 に行う場合、社労士 / 弁護士 / 行政書士 の 関与 を検討。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 健康保険 給付 受領 自体 は 永住・更新 に **影響なし**。拠出制 保険給付 で「公的負担 への依存」とは別カテゴリ。
- 健康保険 加入 + 保険料納付 状況 こそが 永住申請 「公的義務 履行」 評価項目 — 給付 受領 の有無 ではなく **加入歴 と納付歴**。
- 海外療養費 を頻繁 に 申請 すると、「事実上 海外居住」 と 入管 が判断 する リスク（日本国内 居住要件 に影響）。年間 出国日数 + 海外療養費 申請 の 整合性 を在留資格 更新時 説明できる体制 で。
- 経営・管理 で 法人 加入 健保組合 の場合、被保険者資格 取得・喪失日 が 在留資格 と整合 している必要あり。
