# 電気料金 滞納督促 / 送電停止予告

## 信封长什么样
- 寄件人：「東京電力エナジーパートナー株式会社」「関西電力株式会社」「中部電力ミライズ」「九州電力」等具体小売事業者名
- 封面印字：「電気料金お支払いのお願い」「送電停止予告通知書」「最終のお願い」「重要」赤字スタンプ
- 颜色：白封筒 + 赤帯 / 黄色ハガキ（最終予告）
- 同梱物：払込取扱票（コンビニ用バーコード）、未払明細書、送電停止予定日記載書

## 这是什么
電気事業者（小売）が、契約者に対して 電気料金 未払の支払を求める段階的督促。最終段階は「送電停止予告通知書」で、指定日までに支払なき場合 送電停止 を実施する旨の予告。電気事業法 第 18 条 + 各社の託送供給等約款・電気需給約款 に基づく。

## 为什么收到
電気料金 の支払期日（通常 検針日から 30 日）を経過し、入金確認がとれていないため。一般的な督促タイムライン：
- 支払期日経過 約 20 日 → 督促状（最初）
- 約 30-50 日 → 送電停止予告（赤帯）
- 約 50-60 日 → 送電停止 + 契約解除予告

## 必看的关键栏
- **未払金額**：当月分 + 過去未払分（合計欄）
- **支払期限**：「○月○日までに」記載
- **延滞利息**：年 10%（一般電気事業者の託送供給等約款）が日割で加算される場合あり
- **送電停止予定日**：赤字 / 太字
- **払込方法**：コンビニ払込取扱票 / 口座振替 / クレジットカード / Web 支払
- **問合せ先 / お客さま番号**：請求書記載のお客さま番号 / 供給地点特定番号
- **契約名義**：本人 / 旧入居者の可能性

## 不处理的后果（時間梯度）
- **支払期日経過 20 日**：督促状到着、延滞利息加算開始
- **30-50 日**：「送電停止予告通知」発送、指定日 7-10 日後に停止予定
- **50-60 日**：実際に 送電停止 → 契約解除 → 退去時 原状回復費用 + 未払額が一括請求
- **契約解除後**：信用情報機関への登録は基本なし（公共料金は CIC / JICC 共有対象外）。ただし 民事訴訟 → 簡易裁判所 → 強制執行 履歴は別途残る
- **集合住宅 高圧一括契約物件**：管理会社経由で大家に通知 → 退去督促リスク

> ⚠️ 各社の停止までの日数・延滞利息率は約款で異なる。請求書記載の約款条文を確認。

## 该怎么处理（3 种典型路径）

### 路径 A：全額支払で送電停止回避
1. 督促状同封の 払込取扱票 を コンビニ / 銀行 / 郵便局 で支払、または Web 支払 / クレジットカード即時決済。
2. 領収書 / 支払完了画面を保管。停止予告日前 1-2 営業日には反映。
3. 口座振替 戻入の場合は再振替日を確認。

### 路径 B：分割払 / 支払猶予 申請
1. 請求書記載の お客さまセンター（東京電力：0120-995-001 / 関西電力：0800-777-8810 等）に連絡。
2. 「支払が困難」旨を伝え、分割払 / 支払期日延長 を相談。失業 / 病気等の事情があれば申し出る。
3. 合意内容（分割回数 / 期日）を書面 or メールで残す。
4. 生活困窮時は 自治体 生活困窮者自立支援制度（厚生労働省）併用可。

### 路径 C：契約名義違い / 送電停止後 / 訴訟予告
- 入居前の旧入居者宛督促 → 同封返信用封筒 + 賃貸借契約書 写しで「契約者違い」回答。
- 既に 送電停止後 → 全額支払 + 再開手数料（数千円）で復旧。再開は通常翌日。
- 訴訟予告 / 簡易裁判所 支払督促 が届いた場合：2 週間以内に 異議申立 しないと仮執行宣言 → 強制執行（給与差押等）。建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
公共料金 滞納（電気）自体は 出入国在留管理庁 の直接審査項目ではない。**ただし**：
- 永住申請 のガイドライン上「素行善良」要件 → 公租公課（税・年金・健康保険）が中心評価項目で、電気料金 滞納は直接対象外
- **訴訟・強制執行 履歴に発展した場合**は別途、信用情報や住居確保（賃貸更新時の保証会社審査）に間接影響
- 経営・管理 では会社契約の電気滞納 → 事業の継続性・安定性 に疑義 → 続签時 決算書 + 公共料金支払状況 で説明求められる可能性あり

> ⚠️ 永住「素行善良」運用は事案毎、訴訟記録 を伴う場合のみ実務的影響。
