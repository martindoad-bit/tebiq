# NHK 受信料 督促 / 訴訟予告

## 信封长什么样
- 寄件人：「日本放送協会」「NHK ふれあいセンター」「NHK 営業センター」「NHK 受信料関係」等
- 封面印字：「受信料お支払いのお願い」「重要」「最終のお願い」「民事訴訟予告通知」「支払督促申立予告」赤帯
- 同梱物：払込取扱票（コンビニ用バーコード）、契約変更届ハガキ、口座振替・クレジット申込書
- 訪問員（受信料委託会社）が手渡しで「契約のお願い」紙を投函する場合もあり

## 这是什么
NHK が、放送法 第 64 条 に基づく受信契約義務者 / 既存契約者に対し、受信料 の未払・未契約状態の解消を求める書面。最終段階は「民事訴訟予告」「簡易裁判所 支払督促 申立予告」。**最高裁 2017 年 12 月 6 日判決**で受信契約締結義務 + 過去分一括請求 + 受信契約 の合憲性が確定。

## 为什么收到
- **未契約**：テレビ等受信設備（ワンセグ / カーナビ TV / チューナー付 PC モニター 含む）設置を NHK が把握、契約勧奨段階で未締結
- **既契約 + 未払**：口座振替不能 / 払込未対応 / 引っ越し後手続未了
- **衛星契約 未対応**：地上契約のみで衛星アンテナ / 視聴環境を NHK が把握

## 必看的关键栏
- **契約区分**：地上契約 / 衛星契約 / 特別契約
- **受信料額**（2023 年 10 月改定後の標準額）：
  - 地上契約：¥1,100/月（口座・クレジット 12 ヶ月前払 ¥12,276）
  - 衛星契約：¥1,950/月（口座・クレジット 12 ヶ月前払 ¥21,765）
- **未払期間 + 合計額**：契約成立日に遡り全額が請求対象（最高裁判決による）
- **支払期限 / 訴訟予告日**
- **問合せ先**：NHK ふれあいセンター 0120-151515（営業）/ 0570-077-077
- **既存契約者番号 / お客様番号**

> ⚠️ 受信料額は改定履歴あり。請求書記載額を確認。

## 不处理的后果（時間梯度）
- **未払 / 未契約**：継続的督促書 + 訪問員来訪
- **6 ヶ月-1 年経過**：「民事訴訟予告通知」「支払督促 申立予告」送付
- **訴訟予告無視**：簡易裁判所 から **支払督促** 送達 → 2 週間以内に **異議申立** しなければ **仮執行宣言** → 給与・預貯金 差押（強制執行）
- **正式訴訟移行**：最高裁 2017/12/6 判決（最高裁大法廷）により、受信契約締結義務 + テレビ設置時点に遡る受信料支払義務が確定 → 敗訴前提
- **時効**：受信料債権の消滅時効は 5 年（民法 / 最高裁 2014/9/5 判決の整理）。ただし契約成立後の請求は時効起算が個別

> 入管法・在留資格 とは別領域、ただし **訴訟記録 / 強制執行 履歴は別途残る**。

## 该怎么处理（3 种典型路径）

### 路径 A：契約 + 支払（通常パス）
1. NHK ふれあいセンター（0120-151515）または同封ハガキ / Web で受信契約 締結。
2. 地上 or 衛星 区分確認、支払方法（口座振替 / クレジット / 払込）選択。継続割引（口座 / クレジット）で月額が下がる。
3. **家族割引 / 学生割引（家族割引 / 親元等別住居 学生免除）/ 事業所割引** が該当する場合は同時申請。
4. 過去分は契約日に遡り請求、分割払 相談可（次の路径 B）。

### 路径 B：分割払 / 一部減額 交渉
1. 受信料窓口で「一括支払困難」を申告 → 分割払 案内。
2. **全額免除 / 半額免除** 該当性確認：
   - 全額免除：生活保護受給世帯、市町村民税非課税の障害者世帯 等
   - 半額免除：視覚 / 聴覚障害者世帯 等
   - 詳細：[NHK 放送受信料の免除基準](https://pid.nhk.or.jp/jushinryo/menjo/index.html)
3. 申請書 + 証明書類（生活保護受給証明 / 障害者手帳 等）提出。

### 路径 C：解約 / 異議申立 / 訴訟対応
- **解約条件**（厳格）：受信設備の **廃止** が必要。「受信機廃止届」+ 廃棄証明（リサイクル券 / 譲渡先住所）or 引っ越しで日本国外居住となる場合の「住所変更（海外転出）届」。テレビを所持したままの解約は不可（放送法 第 64 条）。
- **支払督促 異議申立**：簡易裁判所から支払督促が届いたら 2 週間以内に **督促異議申立書** を提出 → 通常訴訟に移行。
- 訴訟段階に至った場合は、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
NHK 受信料 滞納は 出入国在留管理庁 の **直接審査項目ではない**。
- 永住申請「素行善良」要件は 公租公課（税・年金・健康保険）が中心、NHK 受信料 単独で不利益となる事例は通常なし
- **訴訟記録 / 強制執行 履歴に発展**した場合は、賃貸更新審査・信用情報・永住「素行善良」の説明材料となる事例あり
- 在留 中の海外転出（みなし再入国 含む）時は「住所変更 + 受信機廃止届」を出さないと請求が継続する

> ⚠️ 永住「素行善良」運用は事案毎、訴訟・差押履歴 を伴う場合のみ実務的に問われる。
