# 出産手当金 支給決定通知 / 产假津贴 支给决定通知

## 信封长什么样
- 寄件人：健康保険組合（〇〇健康保険組合）/ 全国健康保険協会（協会けんぽ）支部
- 封面常见印字：「出産手当金 支給決定通知書 在中」「親展」
- 颜色：白底 + 健保組合 ロゴ
- 同梱物：支給決定通知書 1 枚 / 振込明細 / 次回申請 案内（産後 分 / 育児休業給付金 案内）

## 这是什么
健康保険 被保険者 本人 が 出産 のため 労務 に服さず 給与 支給されない 場合 の 法定給付 の 支給決定通知。産前 42 日（多胎妊娠 98 日）+ 産後 56 日 の範囲内 で、標準報酬日額 × 2/3 が 1 日あたり 支給される。被扶養者（配偶者 等）の出産 は対象外 — 別途 出産育児一時金 ¥500,000（2023 年 4 月以降）が支給される。

## 为什么收到
- 健康保険 被保険者 本人 が 出産（妊娠 4 ヶ月 = 85 日 以降 の 早産 / 死産 / 流産 / 人工妊娠中絶 含む）
- 産前産後休業 中 + 給与 不支給（又は 給与 < 出産手当金 額 の 一部不支給）
- 出産手当金 申請書（医師 / 助産師 証明 + 事業主 証明）提出済
- 健康保険組合 / 協会けんぽ の審査完了

## 必看的关键栏
- **被保険者番号 / 記号番号**
- **出産日 / 予定日**（実出産日 が予定日 と異なる場合 計算が変動）
- **支給対象期間**（産前 〇日 + 産後 〇日）
- **標準報酬日額**（直近 12 ヶ月 平均 / 各月 標準報酬月額 ÷ 30）
- **1 日あたり 支給額**（標準報酬日額 × 2/3）
- **支給金額 合計 + 振込予定日 + 振込口座**
- **問合せ先**

## 不处理的后果（時間梯度）

**支給決定 の場合**：
- 受領 のみで 行動義務 なし
- 産前分 → 産後分 を 分けて申請する 場合、産後 56 日経過後 に 産後分 を申請

**不支給 / 一部支給 / 計算誤り**：
- 審査請求 期限：**3 ヶ月以内**（健康保険法 第 189 条）

**消滅時効**：
- **2 年**（健康保険法 第 193 条）。出産日（産前分）/ 産後 56 日経過日（産後分）の 翌日 から進行

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 育児休業 給付金 へ移行（多数人）
1. 通知書 + 振込明細 を保管。指定口座 に通知額 が振込（通常 通知 から 1-2 週間）。
2. **産後 57 日目 以降** は 出産手当金 終了 → **育児休業給付金**（雇用保険 / ハローワーク 経由 / 事業主 が代行申請 が一般的）に切替。
3. 産休・育休中 の 健康保険料 / 厚生年金保険料 は **免除**（事業主 が「産前産後休業取得者申出書」/「育児休業等取得者申出書」を年金事務所 へ提出）— 通知 とは別 に 自動処理 されているか 確認。
4. 出産育児一時金 ¥500,000（直接支払制度 利用 で 病院窓口 と健保 直接精算 が一般的）も別途確認。

### 路径 B：給与 一部支給 / 退職 ケース
- **産休中 給与 が標準報酬日額 × 2/3 未満 の 一部支給**：差額分 のみ 出産手当金 支給。事業主証明 で 正確 な 給与額 を申告。
- **退職 + 継続給付**：退職日 まで 連続 1 年以上 の 被保険者期間 + 退職日 が産休期間中 + 退職日 当日 出勤 していない（労務不能）→ 退職後 も 残日数 受給可。
- **被扶養者 配偶者 の出産** → 出産手当金 対象外。**家族出産育児一時金 ¥500,000** の対象。

### 路径 C：不支給 / 計算 異議
1. 通知 受領 から **3 ヶ月以内** に 社会保険審査官 へ 審査請求（健康保険法 第 189 条）。
2. 主な争点：標準報酬日額 算定誤り / 産前産後 期間 計算誤り / 給与調整誤り / 退職後 継続給付 要件 認定誤り。
3. 産休・育休 + 退職 + 在留資格更新 が同時並行 する複雑 ケース は 社労士 / 行政書士 へ相談。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 出産手当金 受給 自体 は 永住・更新 への **直接 不利益 要因 ではない**。健康保険料 拠出 から の 法定給付。
- 産休・育休 期間中 の 在留資格 該当性：
  - **技人国 / 高度専門職** 等：産休育休 は「正当な事由 による 一時的 休業」 として 在留資格該当性 を維持。事業主 と の 雇用関係 継続 が前提。
  - **家族滞在**：扶養者 が 産休中 でも 扶養関係 維持 で 影響なし。
  - **経営・管理**：自身 の出産 + 産休 中 の 事業継続 を 副代表 / 取締役 / 業務委託 等 で確保 する設計 が必要。
- 永住申請 の 直近 5 年 安定収入 要件 — 出産手当金 + 育児休業給付金 期間 は 「正当な理由 ある 一時的減収」 として 説明可能。納税証明書 + 課税証明書 + 給与復帰後 の連続性 で補強。
- 出産育児一時金 / 児童手当 受給 → 子女 の在留資格認定 / 帯同 / 国籍 取得 と連動 して整理。
