# 修学旅行費 積立 / 納付書 / 修学旅行费 积立通知

## 信封长什么样
- 寄件人：「○○小学校」「○○中学校」「○○高等学校」学年主任 / 事務室
- 封面印字：「修学旅行 積立金 案内」「学年費 納付書 在中」「重要」
- 颜色：白色 長 3 号 + 学校ロゴ
- 同梱物：積立計画 案内 / 月額 / 期分 金額表 / 口座振替 依頼書 / 修学旅行 概要（行先 / 日程 / 内容）/ 旅行傷害保険 案内

## 这是什么
修学旅行 は 学校教育法 施行規則 第 50 条（教育課程）に基づく 学校行事 の 一部。費用 は 食材費 / 交通費 / 宿泊費 / 入場料 等 の 実費 を 保護者 が 負担。多数 学校 で 出発 の 1-2 年前 から 月額 / 期分 で 積立 集金 → 出発前 に 残額 一括精算。徴収主体 = 学校 学年費 会計（私会計）or 自治体 公会計（最近 公会計化 進行中）。

## 为什么收到
- 修学旅行 実施 学年（小 6 / 中 3 / 高 2 多数）の 1-2 年前 → 積立 開始 案内
- 積立 開始後：月次 / 期次 引落不能 / 残高不足 → 納付書 郵送
- 出発 直前：残額 精算 通知（積立 不足分 の 追加納付 案内）
- 修学旅行 不参加 申出 後：返金 通知（既積立分）

## 必看的关键栏
- **総額・行先・日程**（小学校 1-2 泊 / 中学校 2-3 泊 / 高校 3-4 泊 多数）
- **積立 期間 と 月額**（例：月 ¥3,000 × 24 ヶ月 = ¥72,000）
- **納付方法**（口座振替 推奨 / 納付書 / 現金集金）
- **不参加 / 中途退学 時 返金規程**（既積立分 - 既支出 経費 = 返金額）
- **キャンセル料**（出発直前 不参加 → 旅行会社 規程 に従い 返金額減）
- **問合せ先**（学年主任 / 事務室）

## 不处理的后果（时间梯度）
- **積立 引落不能**：翌月 納付書 + 個別連絡
- **継続 滞納**：出発前 残額 一括精算 求められる → 一括 困難 case で 不参加 / 別行動 検討
- **出発前 残額 未精算**：(A) 学校 / 学年費 会計 が 一時 立替 → 後日 督促 / (B) 不参加 措置（学校 / 自治体 / 旅行会社 規程 で 異なる）
- **公会計化 自治体**：地方自治法 第 231 条の 3 で 強制徴収 可能（給食費 と 同様）

不参加 の 場合 でも 学校 で 別 教育活動（自習 / 学校待機）が 用意 される — 子女 が 教育機会 を 失う ことは ない。

## 该怎么处理（3 种典型路径）

### 路径 A：積立 → 期日通り 出発
1. 口座振替 依頼書 提出 → 月次 / 期次 自動引落
2. 残高不足 防止 のため 児童手当 / 給与振込 口座 と 別 専用口座 推奨
3. 出発前 残額精算 → 出発
4. 帰着後 余剰金 / 返金 ある場合 → 学校経由 振込 / 現金返却
5. 領収書 / 振替明細 を 5 年保管（就学援助 申請 / 確定申告 寄附金控除 等 補強）

### 路径 B：就学援助 認定 → 修学旅行費 給付
触发条件：就学援助 認定（参见文档 #34）。
1. 就学援助 給付項目 「修学旅行費」が 上限額 内 で 給付（小学校 ¥22,690 / 中学校 ¥60,910 程度、年度・自治体差あり）
2. 給付方法：(A) 保護者 口座 振込 → 学校 へ 納付 / (B) 学校 へ 直接 振込 → 自己負担 軽減
3. 給付額 が 実費 を 下回る 場合 差額 自己負担

### 路径 C：不参加 / 別行動 / 中途退学 時 返金
触发条件：病気 / 家庭事情 / 転校 / 海外旅行 / 子女 の 修学旅行 拒否 等。
1. 不参加 届 を 学校 に 提出（出発 1 ヶ月前 まで 多数）
2. 既積立金 - 既発生 経費（旅行会社 キャンセル料 / 事前 入場券 等）= 返金額
3. キャンセル料 規程：出発 21 日前 以降 で 段階的 増額（旅行業約款）
4. 行政書士 / 弁護士 関与 は 通常 不要 — 学校 と 旅行会社 内部 で 解決

## 这封信和续签 / 永住 / 経営・管理 的关系
- 修学旅行費 = 学校 私会計 / 公会計 の 旅行実費。**公的義務 とは 言い難い**
- 滞納 / 不参加 は 在留資格 審査 / 永住申請 / 帰化申請 に **通常 直接影響 なし**
- 公会計化 自治体 で 滞納 した場合 は 給食費 と 同様 「公的料金 滞納」扱い される 可能性 あり（参见文档 #35）
- 経営・管理 / 高度専門職 高所得層 は 通常 滞納 リスク 低
- 修学旅行 行先 が 海外 / 韓国・台湾 等 の 場合 子女 の 在留資格 と 再入国許可 関連 確認 必要 — みなし再入国許可（出国 1 年以内 戻り）で 通常 対応可、但 在留資格 「家族滞在」「定住者」等 で 個別事情 ある case では 入管 確認 推奨

> # ⚠️ 需创始人/書士复核：海外修学旅行 で みなし再入国許可 を 使う 場合、子女 の 在留カード / パスポート 状態 と 学校提出書類（旅行許可申請書 等）の 整合性 — 自治体・学校 ガイドライン で 揺れ。
