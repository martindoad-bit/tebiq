# 介護保険料 納付通知書 / 介護保险费 缴纳通知

## 信封长什么样
- 寄件人：市役所 介護保険課（区役所 福祉保健部 介護保険課）
- 封面常见印字：「介護保険料 納付通知書 在中」「重要書類」「親展」
- 颜色：白底 + 蓝色或绿色印章（市町村 LOGO）
- 同梱物：納付書（コンビニ払 用）4-6 枚 / 口座振替 案内 / 所得段階 一覧表 / 減免制度 案内チラシ

## 这是什么
介護保険 第 1 号被保険者（65 歳以上）每年向 市役所 缴纳的介護保険料 年度通知。同时附本年度所得段階判定结果与 4-12 期分割納付書。40-64 岁的 第 2 号被保険者 不会单独收到这封信 — 介護保険料 已合并在 国民健康保険料 或 健康保険料 里直接扣除。

## 为什么收到
- 65 歳到达 → 自动从 第 2 号被保険者 切换为 第 1 号被保険者
- 上一年度 所得 / 世帯課税状況 已确定，4-6 月发出年度賦課通知
- 转入新市町村 → 转入后 1-2 个月内 重新発行
- 年金天引（年金から 自動引落）対象外（年金額 年 18 万円未満等）→ 普通徴収（自分で 納付）

## 必看的关键栏
- **被保険者番号**（10 桁）— 問合せ时必报
- **年額 介護保険料**（所得段階 1-13 段階）
- **所得段階**（第何段階 / 本人 課税 or 非課税 / 世帯 課税 状況）
- **納期**（普通徴収：通常 4-6 月起 4-9 期 / 特別徴収：年金支給月 6 回）
- **納付方法**（普通徴収 / 特別徴収 / 口座振替）
- **問合せ先**（介護保険課 直通電話）

## 不处理的后果（時間梯度）
- **納期 翌日**：延滞金 加算開始（年 14.6%、最初 1 ヶ月 年 7.3%）
- **20 日経過**：督促状 送付 + 督促手数料（市町村 条例 で ¥100 程度）
- **3 ヶ月以上 滞納**：給付制限 開始（介護サービス 利用時 1 割→3 割 自己負担 へ引上）
- **1 年以上 滞納**：保険給付 一時 差止（要介護認定 を受けても サービス費 全額立替）
- **2 年経過**：時効により納付不可 → 給付額 引下 確定

> # ⚠️ 需创始人/書士复核：介護保険料 滞納 是否计入 永住申請 「公的義務履行」評価。介護保険料 属公的保険料、与 健康保険・年金 同列、滞納記録 在 ガイドライン 上属审查対象、但实务 影响程度 与 国保・年金 比较 是否相同 需確認。

## 该怎么处理（3 种典型路径）

### 路径 A：受領 → 期日通り 納付（多数人）
1. 同梱 納付書 をコンビニ / 銀行 / 郵便局 で 各期 期日までに納付。
2. 口座振替 を希望する場合、同梱 申込書 を 金融機関 へ提出（次回以降 自動引落）。
3. 納付書 や 領収書 は 5 年間 保管（永住申請 时 公的義務履行 証明として使用可能）。

### 路径 B：減免 / 分納 申請
- **災害・失業・廃業 等で 大幅 所得減少**：介護保険料 減免制度 あり（市町村 条例）。介護保険課 で 減免申請書 + 罹災証明 / 離職票 / 確定申告書 提出。
- **一括 納付困難**：分納相談（介護保険課 収納係）で 月割 / 隔月 等 個別計画 締結 可能。延滞金 完全 免除 にはならないが 給付制限 は回避できる。
- 高額医療・高額介護合算療養費 制度 と 連動 する 軽減制度 もあるため 同時 確認。

### 路径 C：所得段階 異議 / 審査請求（不服申立）
1. 通知書 受領 から **3 ヶ月以内** に 介護保険審査会 へ 審査請求書 提出（介護保険法 第 184 条）。
2. 主な争点：所得段階 判定誤り（住民税 課税 / 非課税 区分、世帯員 範囲、合計所得金額 計算）。
3. 審査請求 の手続が 複雑 / 所得段階 算定 根拠 が不透明な場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 介護保険料 は **公的保険料 の一種**。永住申請 / 帰化申請 では 「公的義務履行」評価項目 に含まれる。
- 65 歳以上 の在留外国人（永住者 / 定住者 / 配偶者 等）が 主な 対象。技人国 / 経営・管理 の現役世代 は通常 第 2 号被保険者 として 健康保険料 経由 で納付済み、本通知 は届かない。
- 滞納 がある状態 で 永住許可申請 すると 不許可 リスク高。直近 2-3 年 全期 完納 + 領収書 控え保管 が安全。
- 延滞金 のみ未納 でも 「未納」 扱い になる場合あり。納付書 の 各期 金額 を完全一致 で納付 することを推奨。
