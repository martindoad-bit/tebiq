> # ⚠️ 需创始人/書士复核：給与天引き 月額 12 ヶ月均等 ではなく 6 月 + 7-翌 5 月 11 ヶ月均等運用 の 自治体あり；普通徴収 切替 要件 各市町村で 異なる。

# 市県民税 特別徴収 通知書

## 信封长什么样
- 寄件人：「会社」「人事部」「経理部」（市役所 から 直接 個人 に は 来ない、会社経由 配布）
- 封面印字：「給与所得等に係る 市民税・県民税 特別徴収税額の決定通知書」「親展」「○○区長 印」
- 信封颜色 / 大小：会社 内便 + 個人配布、原本 = 市役所 が 会社宛 一括送付 → 会社 が 従業員 に 配布
- 同梱物：通知書 本人控 1 枚（事業主 控 + 本人控 の 2 枚 構成）。納付書 なし（会社 一括納付 のため）

## 这是什么
給与所得等に係る 市民税・県民税 特別徴収税額 通知書 = 市役所 が 地方税法 第 321 条 の 4 に 基づき 1 月 1 日 住所地 の 自治体 から 会社（特別徴収義務者）+ 従業員本人 双方 に 送付する 給与天引き 課税通知。
特別徴収 = 会社 が 6 月 - 翌年 5 月 の 12 ヶ月 で 給与から 天引き → 会社 が 翌月 10 日 まで に 自治体 に 一括納付。
個人 で 納付書 持って コンビニ / 銀行 行く 必要なし（普通徴収 と 異なる）。
通知書 = 「あなたの 1 年間 住民税 は こう天引き されます」の 内訳開示。

## 为什么收到
- 給与所得者（正社員 / 契約社員 / アルバイト で 一定要件）
- 前年（1-12 月）給与所得 が 非課税限度額 超
- 1 月 1 日 時点 日本 居住 + 住民票 あり
- 会社 が 給与支払報告書 を 市役所 に 提出 → 市役所 が 特別徴収 決定

不会収到的人：給与 1 月 1 日 - 12 月 31 日 で 退職 + 転職先 で 特別徴収 引継ぎ なし → 普通徴収 切替（自宅宛 納税通知書 #01 が 来る）/ 海外転出 / 非課税限度額 以下 / 自営業 のみ。

## 必看的关键栏
- **年税額**：「年税額」欄、市民税 + 県民税 合計
- **月割額**：6 月 = 端数調整月（少し多め）/ 7 月 - 翌 5 月 = 11 ヶ月均等
- **所得 / 控除 内訳**：給与所得 / 給与所得控除後 / 各種控除（基礎 / 配偶者 / 扶養 / 社会保険料 / 生命保険料 / 医療費 等）
- **適用税率**：市民税 6% + 県民税 4%（標準）+ 均等割 ¥3,500 + ¥1,500 + 森林環境税 ¥1,000（2024 年度から、復興特別税 と 入替）
- **問合せ先**：通知書末尾 市役所 課税課 直通 + 受付時間。会社 経理 へ も 並行 確認可。

## 不处理的后果（时间梯度）
本通知書 は 「会社 が 天引き する 内容 の 開示」で、本人 が 直接 何か する 必要 = 通常 ない。但し 内容 誤り 放置 = 過大 / 過少 課税 が 1 年間 続く。
- **誤り 発見 → 放置**：1 年 過大 課税 / 還付請求 不能化（5 年時効 で 是正可能 だが 早期対応 必須）
- **退職 + 普通徴収 切替 失念**：退職翌月 から 自宅宛 納税通知書 が 来る が、転居 + 転送届 なし → 受領せず → 滞納 → 督促 → 差押
- **転職先 で 特別徴収 継続手続 怠る**：旧会社 が 残月分 を 一括徴収 / 普通徴収 切替

直接 罰則 ない が、誤り の 連鎖で 滞納 → 在留期間更新 / 永住 評価 に 影響。

## 该怎么处理（3 种典型路径）

### 路径 A：内容確認 + 保管（多数人）
1. 通知書 受領 → 「所得 / 控除 / 税額」を 源泉徴収票（前年分）と 照合
2. 配偶者控除 / 扶養控除 / 生命保険料控除 等 が 反映 されている か 確認
3. 通知書 控 を 5-7 年保存（在留期間更新 / 永住申請 / 確定申告 修正 で 必要）
4. 月給与明細 で 「住民税 ¥XXX」天引き が 通知書 月割額 と 一致 確認

### 路径 B：誤り発見 → 是正手続
触发条件：扶養家族 反映漏れ / 控除 反映漏れ / 所得 二重計上。
1. 「修正申告 / 更正の請求」を 税務署 + 市役所 双方 に 提出（所得税 / 住民税 連動）
2. 過去 5 年分 まで 還付請求 可能（地方税法 第 17 条 の 5）
3. 還付発生 → 数ヶ月後 振込 / 翌年度 税額 から 減算

### 路径 C：転職 / 退職時 切替手続
触发条件：年度途中 退職 / 転職。
1. 退職時 「給与所得者異動届」を 旧会社 が 市役所 に 提出（法定義務）
2. 退職月 + 5 月 まで の 残月分：
   - 新会社 で 特別徴収 引継ぎ：「特別徴収継続申出」
   - 一括徴収：最終月 給与 から 残月分 全額 天引き
   - 普通徴収 切替：自宅宛 納税通知書（参见文档 #01）→ 自分 で 納付
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 在留期間更新：本通知書 + 源泉徴収票 + 住民税 課税証明書 の 3 点 = 入管 提出 標準セット。給与天引き = 滞納リスク 最小、評価 安定。
- 永住申請：5 年 給与所得者 + 特別徴収 完遂 = 「公的義務履行」評価で プラス材料。
- 経営・管理：自社 役員 = 役員報酬 から 自社 で 特別徴収（法人 が 義務）。役員 1 名 / 妻 役員 等 小規模法人 で 特別徴収 実務 怠る ケース あり、市役所 から 督促 + 「滞納処分」予告 リスク。
- 退職時 普通徴収 切替 → 自宅 で 納付書 受領 → 海外転出 / 引越し で 受取らず = 滞納 連鎖。退職前 最終月 給与 で 一括徴収 推奨。
- 「住民税 通知書 で 副業 が 会社 に バレる」言説 → 副業所得 の 住民税 を 「普通徴収 選択」で 自宅 別納付（確定申告書 第 2 表 で 選択）→ 会社 通知書 に 反映 されない 運用 が 一般的、但し 自治体 で 取扱 異なる。
