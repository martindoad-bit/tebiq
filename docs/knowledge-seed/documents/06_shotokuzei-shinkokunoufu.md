# 所得税 確定申告書 / 納付書

## 信封长什么样
- 寄件人：「○○税務署」「国税庁」「○○国税局」
- 封面印字：「所得税及び復興特別所得税 確定申告書 在中」「国税庁」ロゴ、税務署 公印
- 信封颜色 / 大小：白色 / 浅藍色 角 2 号（申告書 一式 入り 厚め）
- 同梱物：申告書 用紙（A 様式 / B 様式 / 第三表 分離課税用 等、対象者で 異なる）+ 手引き + 納付書 + e-Tax 案内 + 振替納税 申込書

## 这是什么
所得税 確定申告 = 1 月 1 日 - 12 月 31 日 の 1 年間 所得 を 翌年 2 月 16 日 - 3 月 15 日 に 自己申告 + 納付する 国税 手続（所得税法 第 120 条）。
復興特別所得税 = 所得税額 × 2.1%、所得税 と 一体納付（2013-2037 年、25 年間）。
税務署 から 「申告書 用紙」が 届く ＝ 過去 申告履歴 / 給与支払報告書 等 から 「申告対象者」と 把握 されている サイン。
逆に 用紙 来ない 場合 でも 申告義務 ある ケース 多数（給与 ¥2,000 万円超 / 副業 ¥20 万円超 / 自営 / 不動産所得 / 年金収入 / 株式譲渡 等）。

## 为什么收到
- 前年 確定申告 提出履歴 あり
- 給与支払報告書 / 支払調書 で 税務署 が 申告対象 と 認識（自営業 / フリーランス / 不動産家賃収入 等）
- 還付申告 該当（医療費控除 / 寄付金控除 / 住宅ローン控除 1 年目 等）の 過去申請者
- 「お知らせ通知」と 別に「納付書」が 届く 場合 = 既 確定済 / 振替納税 案内 / 予定納税 通知

不会収到的人：給与 1 か所のみ + 年末調整 完了 + 副収入 ¥20 万円 以下 + 控除 追加 申請 不要 / 申告履歴 ない 新規 在留者。

## 必看的关键栏
- **申告期限**：原則 翌年 3 月 15 日 23:59（休日翌営業日 へ 順延）。e-Tax 24 時間 受付。
- **納付期限**：申告期限 と 同日（原則 3 月 15 日）
- **振替納税 引落日**：通常 4 月下旬（毎年改定。取得日时点 確認要）
- **予定納税**：前年 納税額 ¥15 万円以上 → 第 1 期 7 月 / 第 2 期 11 月 で 予定 納税通知書 別途
- **付款码**：納付書（白色 OCR）に 自分で 金額記入 → 銀行 / 税務署 窓口 / コンビニ ¥30 万円 以下
- **問合せ先**：管轄 税務署 + 国税庁 確定申告 電話相談センター。

## 不处理的后果（时间梯度）
- **期限後 1 ヶ月内**：「期限後申告」自主提出 → 無申告加算税 軽減（5%）
- **税務署 から 指摘 後**：無申告加算税 15-20%（⚠️ 取得日时点 改正数字 確認）
- **申告したが 納付遅延**：延滞税（年 2.4% / 1 ヶ月後 年 8.7% 前後）
- **過少申告 / 隠蔽**：過少申告加算税 10-15% / 重加算税 35-40%
- **未申告 持続**：税務調査 → 7 年遡及 課税（仮装隠蔽 認定時）
- **更长期**：差押 / 在留期間更新 で 「公的義務不履行」評価 / 永住 不許可 リスク

## 该怎么处理（3 种典型路径）

### 路径 A：期限内 申告 + 納付
1. 申告書 作成：[国税庁 確定申告書等作成コーナー](https://www.keisan.nta.go.jp/)（無料、e-Tax 直接送信 可能）
2. 提出方法：
   - e-Tax（マイナンバーカード + ICカードリーダー / スマホ）：24 時間 提出可、控除 + ¥10,000（青色申告 65 万円控除条件）
   - 郵送：消印 = 提出日（3 月 15 日 消印有効）
   - 税務署 窓口：閉庁時間外 ポスト 投函
3. 納付方法（複数選択可）：
   - 振替納税：口座登録 → 4 月下旬 自動引落（実質 1 ヶ月 期限延長）
   - クレジットカード：[国税クレジットカードお支払サイト](https://kokuzei.noufu.jp/)（決済手数料 ¥83〜/¥10,000）
   - コンビニ（QR コード / バーコード、¥30 万円 以下）
   - 銀行 / 郵便局 窓口（納付書 持参）
   - ダイレクト納付（e-Tax 連動 口座引落）
4. 申告書 控 / 納付書 領収印 を 7 年 保存（在留期間更新 / 永住申請 で 必要）

### 路径 B：期限後申告 / 修正申告
触发条件：期限 過ぎた / 申告内容 に 漏れ・誤り。
1. 期限後申告：早ければ 早い ほど 加算税 軽減
2. 修正申告：「更正の請求」（5 年以内、税過大時 還付）/ 「修正申告」（税過少時 自主訂正）
3. 加算税：自主提出 5% / 税務署指摘後 15-20%

### 路径 C：異議申立 / 不服申立
触发条件：税務署 更正処分 / 決定処分 に 不服。
1. 処分通知 受領翌日から **3 ヶ月以内** に 「再調査の請求」を 税務署長 宛 提出
2. または 「審査請求」を 国税不服審判所 に 直接提出
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士 / 税理士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 在留期間更新：直近 1-3 年 「納税証明書（その 1：所得税）」「課税証明書」が ほぼ全カテゴリ 必要。本申告書 = 源データ。
- 永住申請（一般 5 年路径）：直近 5 年 納税状況 + 公的年金 + 公的医療保険 加入実績 すべて 審査対象。1 年でも 期限後申告 + 加算税 = マイナス材料。
- 経営・管理：法人税 + 役員 個人所得税 が 別建。役員報酬 を 給与所得 として 自社 で 年末調整 完了する 場合 個人 確定申告 不要 だが、役員報酬 ¥2,000 万円超 / 副業所得 / 不動産所得 等 で 申告義務 発生。
- 「源泉徴収 されているから 申告不要」は 給与 1 か所 + 年末調整 完了時 のみ。複数収入 / 家賃収入 / 株式譲渡 / 海外所得 ある 場合 別途 申告義務。
- 海外所得（中国 国内 不動産 / 株式 / 利息 等）→ 在留 5 年超 = 「永住者」課税区分 → 全世界所得 課税対象。租税条約 + 外国税額控除 で 二重課税 調整。
