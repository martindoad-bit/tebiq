# 国民健康保険料 納税通知書

## 信封长什么样
- 寄件人：「○○区長」「○○市長」「国民健康保険課」「国保年金課」
- 封面印字：「国民健康保険料 納入通知書 在中」「親展」「国保 通知書」
- 信封颜色 / 大小：白 / 緑 / 黄 系长 3 号信封、多数自治体在 6 月中旬発送
- 同梱物：納入通知書 + 算出根拠（所得情報 + 世帯員）+ 納付書 8-10 期分 + 口座振替 申込書 + 高額療養費 / 限度額認定証 案内

## 这是什么
国民健康保険料 納入通知書 是 市役所 / 区役所 依据 国民健康保険法 第 76 条 + 各自治体 国民健康保険条例 向 国保 加入世帯主 发送的保険料 賦課通知。世帯主課税 ＝ 即使加入者本人不是世帯主、納付書名義人 = 世帯主。一封信涵盖 1 世帯 全員分（家族 全員 国保 加入時）。
保険料构成 = 医療分 + 後期高齢者支援金分 + 介護分（40-64 歳のみ）。各分 = 所得割 + 均等割 + 平等割（自治体不同，東京 23 区无平等割）。

## 为什么收到
- 自营业 / 自由职业 / 無職 / 退職後で 健康保険（社保）に 加入していない
- 会社退職後 14 日以内に 国保加入届 提出済
- 前年度 確定申告 / 住民税申告 結果が 反映され、保険料 確定
- 4 月以降 転入 → 転入月から日割計算

不会收到的人：会社の 健康保険（協会けんぽ / 健保組合）加入中、後期高齢者医療制度（75 歳以上）対象者、生活保護 受給中。

## 必看的关键栏
- **保険料 年額**：「年間保険料額」表頭、医療分 / 支援金分 / 介護分 で 合計。
- **期限**：通常 第 1 期 6 月末、以降 月 1 期 で 翌年 3 月まで（自治体で 8 期 / 10 期 / 12 期 異なる）。
- **付款码 / バーコード**：納付書 下部 OCR 区 + バーコード。コンビニ 上限 ¥30 万円。
- **办理窗口**：銀行 / 郵便局 / コンビニ / Pay-easy / クレカ / スマホ決済 / 口座振替（推奨、申込で 翌期から）
- **問合せ先**：通知書末尾「国民健康保険課」「国保年金課」直通 + 受付時間 + マイナンバー 案内。

保険料 上限額（年間、自治体共通の枠）：医療分 上限、後期高齢者支援金分 上限、介護分 上限 が 法令で 設定。⚠️ 毎年 政令改正 で 改定（取得日时点 数字 確認要）。

## 不处理的后果（时间梯度）
- **納期限 翌日 - 20 日内**：督促状 発送（国民健康保険法 第 79 条）、手数料 ¥100 前後
- **滞納 1 年 経過**：保険証 を 「短期保険証」（有効期限 1-3 ヶ月）に 切替
- **滞納 1 年 6 ヶ月 経過**：保険給付 一時差止（医療費 全額自己負担、後で 償還払）
- **滞納 2 年 経過**：「資格証明書」交付（医療機関 で 全額 10 割 自己負担、後日請求）
- **より長期**：給与 / 預金 / 不動産 差押（国民健康保険法 第 79 条 の 2）；在留期間更新 / 永住 で 公的義務不履行 评价

延滞金：納期限 翌日 から 年 2.4% / 1 ヶ月後 から 年 8.7% 前後（住民税 と 同基準）。⚠️ 取得日时点で 公示数字 確認。

## 该怎么处理（3 种典型路径）

### 路径 A：按期缴纳（多数人）
1. 8-10 期 ある うち 各期 缴纳 / 一括前納（割引なし、住民税 と 異なる）
2. 缴款方式：
   - コンビニ：1 件 ¥30 万円 以下、現金 のみ
   - 銀行 / 郵便局 窓口：金额无上限
   - Pay-easy / ネットバンキング：24 時間、上限なし
   - クレジットカード：[地方税お支払サイト](https://www.zeikin.metro.tokyo.lg.jp/)（決済手数料あり）
   - スマホ決済（PayPay / au PAY 等）：バーコード 読み取り、1 件 ¥30 万円 以下
   - 口座振替：申込書 → 翌期 自動引落（推奨、滞納防止）
3. 領収証書 を 5 年保存（確定申告 「社会保険料控除」/ 在留期間更新 / 永住申請 時 必要）

### 路径 B：減免 / 分割納付 申請
触发条件：失業 / 災害 / 病気 / 事業休廃止 / 所得 大幅減少。
1. 「国民健康保険料 減免申請書」+ 所得証明 / 雇用保険 受給資格者証 等
2. 倒産・解雇 等 非自発的失業 → 軽減特例（前年所得を 30/100 換算で 算定）対象
3. 災害 / コロナ等 特例 → 自治体 公表 制度に従う
4. 分納 計画 → 国保課 窓口 で 個別相談、原則 12 ヶ月 以内

### 路径 C：异议申立 / 不服申立
触发条件：所得 認定 誤り、世帯員 数 違い、加入 期間 認定 誤り。
1. 通知書 受領翌日から **3 ヶ月以内** に 「審査請求書」を 国民健康保険審査会（都道府県）に 提出
2. 同時に 国保課 窓口 で 課税情報 訂正依頼（住民税 修正申告 → 国保 自動再計算 の ケース 多）
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 在留期間更新 / 在留資格変更 时 入管 要求「健康保険 加入状況 確認」。本通知書 + 領収証書 = 加入実績 証明。
- 永住申請 时 直近 1-2 年 公的医療保険 + 公的年金 加入履歴 + 納付状況 を 必須提出（2019 年改正、2024 年運用更厳格化）。1 期でも 滞納 = 即マイナス。
- 経営・管理 自営業者 が 法人化 前 / 役員報酬 ¥0 期間 → 国保 加入 が 通常。法人設立 + 健保加入 後 国保 脱退届 を 14 日以内 に 提出。
- 「保険料 を 払わない 在日外国人」言説 が SNS で 流布 されるが、滞納 = 在留資格 リスク + 医療費 全額自己負担 リスク の 二重リスク。
