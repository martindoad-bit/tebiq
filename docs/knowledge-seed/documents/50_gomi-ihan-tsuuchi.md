# ゴミ違反通知 / 警告

## 信封长什么样
- 寄件人：市役所 環境課 / 清掃事務所 / 廃棄物指導課（自治体名 + 部署名）/ 自治会・町内会（連名 or 単独）
- 形式：
  - **啓発シール**：ゴミ袋に直接貼付（黄色 / 赤色）「分別違反」「収集できません」「○月○日 持戻り」
  - **警告書（書面）**：玄関ポスト投函 or 郵送、自治体ロゴ + 違反項目チェック式
  - **集合住宅**：管理会社経由で「ごみ出しルール厳守のお願い」書面
- 同梱物：分別ルールパンフレット / カレンダー / 排出時間案内

## 这是什么
廃棄物処理法 + 各自治体 廃棄物の処理及び清掃に関する条例 に基づき、収集職員 / 自治体指導員 が排出ルール違反を確認した際の **指導 + 警告**。第 1 段階は啓発（啓発シール / 持戻り）、繰返し違反 + 大量不適正排出で **行政指導 → 罰則** へ進む。

## 为什么收到
よくある違反事由：
- **分別違反**：燃やすごみ / 燃やさないごみ / 資源 / 粗大ごみ の混在
- **収集日違反**：曜日違い、収集日前夜から放置
- **指定袋未使用**（自治体指定袋の地域で透明 / 黒袋使用）
- **氏名未記入**（一部自治体ルール）
- **粗大ごみ シール 未貼付**
- **時間外排出**（朝の収集時刻後の追加排出）
- **不法投棄**（道路・公園・他地区集積所への投棄）

## 必看的关键栏
- **違反内容**：チェック式で具体項目（分別違反 / 曜日違反 / 袋違反 等）
- **発見場所 / 発見日時**
- **対応指示**：「次回収集日まで持戻り」「指定排出方法に変更後再排出」
- **問合せ先**：環境課 / 清掃事務所 担当者
- **再発時の措置記載**（行政指導書 / 命令書 / 罰則 等）
- **集合住宅の場合**：管理会社・大家 名

## 不处理的后果（時間梯度）
- **第 1 回 啓発シール / 警告**：法的罰則なし、ただし 持戻り → 次回収集日に再排出
- **繰返し違反（複数回 同一者）**：自治体による **戸別訪問指導** / 書面警告 / 自治会経由通知
- **悪質 / 大量 / 故意違反**：廃棄物処理法 第 16 条（投棄禁止）違反 → 第 25 条 罰則
  - 不法投棄：**5 年以下の懲役 or 1,000 万円以下の罰金**（個人）
  - 自治体条例違反：**5 万円以下の過料 / 罰金**（多くの自治体条例）が一般的
- **集合住宅**：管理会社 → 大家 経由で警告 → 賃貸借契約 信頼関係破壊評価（極端な繰返しで退去要求事例あり）
- **ゴミ集積所撤去**：自治会管理の場合、住民全体への迷惑として集積所が移設・閉鎖される事例

> ⚠️ 自治体条例の罰則額は自治体毎で異なる。本書面記載の条例条文を要確認。

## 该怎么处理（3 种典型路径）

### 路径 A：違反内容修正 + 再排出
1. 警告書 / 啓発シール の **違反項目** を確認、自治体配布の **分別ルール冊子 / カレンダー** で正しい区分を再確認。
2. 持戻り後、次回該当収集日に正しい区分・指定袋・収集時間で再排出。
3. 自治体 HP / アプリ（多くの自治体が「ごみ分別アプリ」配信）で品目検索可。

### 路径 B：自治体への問合せ / 不服申立
1. 自治体ルールに不明点がある場合、環境課 / 清掃事務所 に電話 or 窓口で確認。
2. 自分の排出ではない誤指摘の場合、書面で異議連絡。
3. 言語面で困難な場合は 国際交流協会 / 多文化共生センター 等で日本語通訳補助を依頼。

### 路径 C：罰則・行政処分・訴訟
- **行政指導書 / 行政命令書** が届いた場合：行政手続法 第 13 条 弁明機会あり、無視すると過料 / 罰金。
- **不法投棄 容疑で警察捜査**：弁護士相談（法テラス：0570-078374 / 利用条件あり）。
- **賃貸退去要求 / 訴訟**：建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- ゴミ違反の **啓発シール / 第 1 回警告** は 出入国在留管理庁 の **直接審査項目ではない**
- **罰金 / 過料 / 前科** に発展した場合のみ、永住「素行善良」評価で間接影響
- **不法投棄罰金（廃棄物処理法 第 25 条）** は重大な前科、永住申請 + 在留更新で説明資料となる事例あり
- 経営・管理：事業所からの産業廃棄物・一般廃棄物 不適正処理は別途、廃棄物処理法 + マニフェスト制度違反 → 事業継続性 + 役員 善管注意義務 に直接影響

> ⚠️ 永住「素行善良」運用は事案毎、罰金・前科 を伴う場合のみ実務的に問われる。
