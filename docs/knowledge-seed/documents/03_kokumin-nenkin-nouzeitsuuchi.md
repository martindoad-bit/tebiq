# 国民年金 納付書

## 信封长什么样
- 寄件人：「日本年金機構 ○○広域事務センター」「日本年金機構 ○○年金事務所」
- 封面印字：「国民年金保険料 納付書 在中」「日本年金機構」(青ロゴ)
- 信封颜色 / 大小：白色長 3 号 / 角 形、青色「日本年金機構」ロゴ
- 同梱物：納付書（月別 / 半年分 / 1 年分 / 2 年分）+ 口座振替 / クレカ 申込書 + 前納割引案内 + 免除・納付猶予 案内

## 这是什么
国民年金 納付書 是 日本年金機構（厚生労働省 所管）依据 国民年金法 第 87 条 向 第 1 号被保険者（自営業 / 無職 / 学生 / 配偶者扶養外）发送的保険料 納付通知。一封信 = 通常 1 年度分（4 月 - 翌年 3 月）の 納付書 12 枚 + 前納用納付書。
保険料 月額 ＝ 全国一律（市町村関係なし）。⚠️ 取得日时点 月額 約 ¥17,000 前後（毎年 4 月 改定）。

## 为什么收到
- 20 歳到達 / 第 2 号（厚生年金）脱退（退職、自営業 移行）/ 第 3 号（被扶養配偶者）資格喪失（離婚 / 配偶者退職）
- 在留資格 取得 / 住民登録 後 自動加入（外国人も 20-59 歳 居住者は 強制加入、国籍不問）
- 厚生年金 加入 中の 会社 退職 → 翌月 から 第 1 号被保険者 自動切替

不会收到的人：第 2 号（厚生年金、給与天引き）/ 第 3 号（年収 130 万円 未満 の 配偶者扶養）/ 60 歳以上（任意加入除く）/ 海外 転出（国民年金 任意加入 申出 なし の 場合）。

## 必看的关键栏
- **月額**：「保険料額」欄。同一年度内 一律。
- **期限**：各納付書「使用期限」欄。原則 翌月末。
- **前納割引**：1 ヶ月前納（口座振替 のみ、月割引 約 ¥50）/ 6 ヶ月前納 / 1 年前納 / 2 年前納（最大割引、約 ¥16,000 / 2 年）
- **付款码 / バーコード**：納付書 下部、コンビニ ¥30 万円 以下
- **問合せ先**：「ねんきんダイヤル」0570-05-1165（ナビダイヤル）+ 同梱 年金事務所 直通 + 受付時間。

## 不处理的后果（时间梯度）
- **納期限 翌日 - 1 ヶ月内**：「特別催告状」発送（黄色封筒）
- **2-3 ヶ月**：「特別催告状」（赤色 / ピンク色）— 強制徴収 予告
- **6 ヶ月 - 1 年**：「最終催告状」+ 「督促状」発送
- **督促状 後 10 日経過**：差押 法律上可能（国民年金法 第 96 条）
- **未納 持続**：年金事務所 → 国税庁 委任 → 強制徴収（給与 / 預金 / 不動産 差押）。年所得 ¥300 万円以上 + 7 ヶ月以上滞納 が 強制徴収 基準（2019 年度以降）。
- **より長期**：受給資格期間（10 年）満たさず → 老齢基礎年金 受給不可 / 障害年金 / 遺族年金 受給制限；在留期間更新 / 永住 で 公的義務不履行 评价

延滞金：納期限 翌日 から 法定利率（取得日时点 約 年 2.4%-8.7%、税金 と 同基準）。

## 该怎么处理（3 种典型路径）

### 路径 A：按期缴纳（多数人）
1. 月別 納付 / 前納 を 自由選択（前納 ＝ 割引 + 1 回 で 完結 で 滞納防止）
2. 缴款方式：
   - コンビニ：1 件 ¥30 万円 以下、現金 のみ
   - 銀行 / 郵便局 窓口：上限なし
   - Pay-easy / ネットバンキング：24 時間
   - クレジットカード：申込書 提出 → 翌月 から 自動決済
   - 口座振替：最も推奨。前納割引 適用 + 滞納防止
   - スマホ決済（PayPay / au PAY 等）：バーコード 読み取り、¥30 万円 以下
3. 領収証書 / 控除証明書（毎年 11 月 発送）保存 → 確定申告「社会保険料控除」/ 永住申請

### 路径 B：免除 / 納付猶予 申請
触发条件：所得 低い / 失業 / 学生 / 育児休業 等。
1. 「国民年金保険料 免除・納付猶予 申請書」を 市役所 国保年金課 / 年金事務所 に 提出
2. 種類：全額免除 / 4 分の 3 免除 / 半額免除 / 4 分の 1 免除 / 納付猶予（50 歳未満）/ 学生納付特例
3. 認定 = 老齢基礎年金 受給資格期間 に カウント（年金額 は 一部減額）
4. 免除期間 → 10 年以内 に 「追納」可能（老齢年金額 を 満額 復元）

### 路径 C：异议申立 / 不服申立
触发条件：所属期間 認定 誤り、二重加入（厚生年金 同時期 重複） 等。
1. 通知書 受領翌日から **3 ヶ月以内** に 「審査請求書」を 社会保険審査官 に 提出（国民年金法 第 101 条）
2. 同時に 年金事務所 で 加入記録 訂正依頼
3. 涉及 異議申立 / 法定争訟 时，建议咨询行政書士 / 弁護士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 永住申請（一般 5 年路径）：2019 年改正以降 直近 2 年（高度専門職 1 年路径）/ 直近 1 年（高度専門職 3 年路径）の 公的年金 + 公的医療保険 加入実績 + 納付状況 が 必須。1 件 でも 未納 = 不許可リスク 高。
- 在留期間更新：法令上 直接要件 ではないが、入管 審査で「公的義務履行」が 評価項目。
- 経営・管理 役員：法人 設立 + 厚生年金 加入 → 第 2 号被保険者 切替（給与天引き）。役員報酬 ¥0 / 月額 が 標準報酬月額 下限 未満 の 期間 → 第 1 号 のまま。
- 帰国時 「脱退一時金」（最大 5 年分 / 2021 年改正後）→ 永住 申請予定 が ない 場合 のみ 検討。脱退一時金 受給 = 加入期間 リセット。
