# 共益費 / 管理費 滞納督促

## 信封长什么样
- 寄件人：「○○マンション 管理組合」「○○管理組合 理事長」「○○管理株式会社（管理委託会社）」/ 賃貸の場合は「○○不動産（管理会社）」
- 封面印字：「管理費等滞納に関するお知らせ」「督促状」「催告書」「重要」
- 同梱物：
  - 滞納明細（月別 / 項目別 内訳）
  - 払込取扱票（コンビニ / 銀行用）
  - 管理規約 抜粋（遅延損害金条項）
  - **区分所有の場合**：理事会議事録 抜粋、訴訟予告通知

## 这是什么
- **区分所有マンション**：管理組合 が 区分所有法 + 管理規約 に基づき、所有者に **管理費 + 修繕積立金 + 駐車場使用料 等** の支払を求める書面
- **賃貸物件**：管理会社が賃借人に **共益費（共用部光熱費・清掃費）** の未払を求める書面（家賃滞納と同列）

両者は法的根拠が異なる：
- 区分所有：建物の区分所有等に関する法律（区分所有法）第 7 条 → 管理費債権は **先取特権** で優先弁済対象
- 賃貸 共益費：借地借家法 + 賃貸借契約 → 賃料滞納と同様の信頼関係破壊評価

## 为什么收到
- **口座振替 戻入** / **振込忘れ**
- **長期滞納**（3 ヶ月以上）→ 段階的督促
- **大規模修繕一時金 未払**（修繕積立金 不足分の臨時徴収）
- **委託管理会社の催告ルール**（多くは滞納 1 ヶ月で電話、3 ヶ月で書面、6 ヶ月で訴訟予告）

## 必看的关键栏
- **滞納区分**：管理費 / 修繕積立金 / 駐車場使用料 / 自治会費 / 遅延損害金 / 専有部水道料金 等
- **滞納期間 + 月別内訳**
- **遅延損害金率**：管理規約で定額（標準管理規約 第 60 条 第 2 項：年 14.6% 程度が一般的）
- **支払期限**
- **支払方法**：管理組合口座振込 / コンビニ払込取扱票 / 直接持参
- **問合せ先**：管理組合 理事長 or 委託管理会社 担当者
- **訴訟予告 / 法的措置 予告日**

## 不处理的后果（時間梯度）
- **滞納 1-3 ヶ月**：電話督促 + 督促状（書面）
- **3-6 ヶ月**：催告書 + 理事会で「法的措置 検討」議題化
- **6 ヶ月-1 年**：訴訟予告通知 → 簡易裁判所 **支払督促** or 通常訴訟
- **支払督促 異議申立せず（2 週間）**：仮執行宣言 → 給与・預貯金 差押
- **訴訟敗訴**：強制執行（差押）+ 訴訟費用 + 弁護士費用 が追加請求
- **区分所有：先取特権 行使**：競売手続（区分所有法 第 7 条）→ **マンション競売** リスク（実例：滞納 100 万円程度から競売申立例あり）
- **賃貸：信頼関係破壊**：3 ヶ月以上の滞納で 賃貸借契約解除 + 明渡訴訟

> ⚠️ 競売 / 訴訟移行の閾値は管理組合方針・滞納規模で異なる。標準管理規約準拠が多い。

## 该怎么处理（3 种典型路径）

### 路径 A：全額支払 / 早期解消
1. 督促状同封 払込取扱票 で支払 or 管理組合口座振込。
2. 振込明細を委託管理会社に通知（FAX / メール）。
3. 口座振替戻入の場合、振替口座の残高補填 + 次月以降の自動引落確認。
4. **遅延損害金** が加算されている場合、元本 + 遅延損害金 一括支払。

### 路径 B：分割払 / 支払猶予 申請
1. 委託管理会社 / 理事長 に電話、「支払困難」事情（失業 / 病気 / 一時金不足）を申告。
2. **理事会承認の分割合意書** を作成（口頭合意のみは後日争点化）。
3. 区分所有：先取特権 行使前に分割合意できれば競売回避。
4. 賃貸：3 ヶ月超滞納の場合、保証会社経由で解約催告される前に交渉。

### 路径 C：訴訟予告後 / 競売予告 / 異議
- **支払督促 / 訴訟予告 到着**：2 週間以内に **督促異議申立** → 通常訴訟移行で和解協議余地あり
- **区分所有：競売予告**：先取特権 + 区分所有法 第 59 条（競売請求）→ 専有部分の競売申立。
- **時効主張**：管理費債権の消滅時効は **5 年**（民法 第 166 条 一般債権）。ただし管理組合からの請求・承認で時効中断。
- 訴訟・競売段階に至った場合、建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。

## 这封信和续签 / 永住 / 経営・管理 的关系
- 共益費 / 管理費 滞納自体は 出入国在留管理庁 の **直接審査項目ではない**
- **訴訟・差押・競売 履歴に発展**した場合は永住「素行善良」「独立生計」で間接影響
- 区分所有マンション 所有者で **競売申立** を受けた履歴は信用情報・住居安定性の説明材料となる
- 経営・管理：法人所有事務所の管理費滞納 → 事業継続性 説明資料に影響する可能性

> ⚠️ 永住「素行善良」運用は事案毎、訴訟・競売 履歴 を伴う場合のみ実務的に問われる。
