# 我收到一个蓝色信封写「年金保険料納付書」，是什么

## 这事跟我有关吗？

你最近收到了一个 **青色（蓝色）信封**，封面上写着「**国民年金保険料納付書**」「日本年金機構」字样——这篇适合你。出现这种情况的常见身份是：

- **自営業 / 個人事業主**（経営・管理 自社代表，未加入厚生年金的）
- **留学生**（20 歳以上）
- **家族滞在 / 配偶者ビザ** 中本人有所得且未走厚生年金的
- 刚 **退职、自由职业、转换工作** 的过渡期被自动归为第 1 号被保険者
- **無職 / 求職中** 的 20 歳～59 歳居住者

如果你是 **正社員 / 契約社員，公司加入了「厚生年金」**（即第 2 号被保険者），通常 **不会** 收到这种纳付書（保险费已从工资扣除）。如果你已经开了 **口座振替** 或 **クレジットカード払い**，也通常不会再收到纸质纳付書，只会收到「**控除証明書**」。

## 这是什么

蓝色信封内装的「国民年金保険料納付書」是 **日本年金機構** 依据《国民年金法》第 88 条向 **第 1 号被保険者** 发出的保险费缴纳通知。

国民年金被保険者分三种：

- **第 1 号**：20 歳～59 歳、日本国内居住、不属于第 2 号 / 第 3 号 的所有人（自営、留学、無職、自由职业 等）—— **本人自行缴纳**
- **第 2 号**：厚生年金加入者（公司员工 / 公務員）—— 工资自动扣除
- **第 3 号**：第 2 号被保険者的被扶养配偶（20～59 歳）—— 不需自交，由配偶者所属制度代缴

蓝色信封通常一年发一次，里面装的是 **当年度全 12 期** 或剩余期数的 **コンビニ・銀行用納付書**（バーコード付）。

**保险费金额**（2026 年度）：

- 月额定额 **17,510 円 程度**（每年由厚生労働大臣告示微调，参照 [nenkin.go.jp 公示数据](https://www.nenkin.go.jp/service/kokunen/hokenryo/hokenryo.html)）
- 全年合计约 **210,120 円 程度**
- **付加保険料**（任意加入、月 400 円）选择者另算
- **前納割引**：1 年前納 / 2 年前納 / 半年前納 可享受较 月払 更优的折扣

## 你应该做什么

1. **核对你的被保険者资格**。如果你是公司员工但收到了第 1 号納付書，可能是 **厚生年金加入手续未完成** 或 **离职后切换信息延误**。立即联系公司人事部 / **年金事務所**（电话 0570-05-1165 / ねんきんダイヤル）。
2. **选择缴纳方式**：
   - **コンビニ**（持纳付書 + 现金，期限内有效）
   - **銀行 / ゆうちょ**窗口
   - **Pay-easy**（ATM / 网银）
   - **口座振替**（事前申请、有割引）—— 推荐长期居住者使用
   - **クレジットカード払い**（事前申请）
   - **スマホ決済**（au PAY / d払い / PayPay 等、納付書バーコード対応）
3. **困难时申请减免**。收入低 / 失业 / 学生 时可申请：
   - **保険料免除制度**（全額 / 4分の3 / 半額 / 4分の1 免除）
   - **納付猶予制度**（50 歳未満对象）
   - **学生納付特例制度**（学生专用）
   - 申请窗口：市役所国民年金係 或 年金事務所
4. **保留控除証明書**。每年 11 月～翌年 2 月会另寄「**社会保険料控除証明書**」，作为 **確定申告 / 年末調整** 的社会保険料控除资料，全额可扣除所得。
5. **永住 / 帰化申请预备者**：保管好缴纳记录，**直近 2 年（永住）/ 5 年（帰化）的纳付状况会被审查**。

## 不做的后果

- **納期限後约 2 个月内**：日本年金機構发出 **督促状**（青色信封 + 红字）
- **督促後仍未缴**：发出 **催告状**（更强烈的督促）
- **持续未缴**：发出 **最終催告状** → **督促状（強制執行予告）** → **財産差押（給与・預金・不動産）**
- **永住 / 帰化申请**：年金未納记录是 **极不利因素**，[moj.go.jp 永住許可ガイドライン](https://www.moj.go.jp/isa/applications/procedures/16-4.html) 明确将「公的義務（納税・公的年金・公的医療保険の保険料納付）」列为审查重点
- **将来年金受給額减少**：国民年金 40 年（480 月）满额者月额约 6.9 万円程度（2026 年度），未納期间会按比例扣减
- **障害年金 / 遺族年金 受給権**：保険料納付要件未达成（一般要求加入期間的 2/3 以上有效缴纳 或 直近 1 年無滞納）将丧失受給资格

## 什么时候你应该找专家

- 保険料金额过高、家计困难，但不知如何申请免除 / 猶予 → 先咨询 **市役所国民年金係 / 年金事務所（无料）**
- 公司应加入厚生年金但实际未加入，导致你被列为第 1 号 → 行政書士 或 社会保険労務士
- 永住 / 帰化申请前需要补缴大量未納期間，且涉及 **追納制度**（10 年内可追納） → 行政書士
- 离日 / 帰国时申请 **脱退一時金** 或 **納税管理人** 手续
- 收到 **差押予告書 / 差押通知書** → 立即联系 年金事務所 並 視情況咨询律师 / 司法書士
