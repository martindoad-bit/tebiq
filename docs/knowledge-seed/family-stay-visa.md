# 我用家族滞在签证，能打工吗、怎么打才合规

## 这事跟我有关吗？

如果你以「家族滞在」身份在日本和配偶/父母同住，想打工补贴生活，这篇适合你。如果你的扶養者准备从 技能実習 切换到其他 在留資格，或考虑邀请父母来日，也建议读完——里面写清了哪些组合做不到。

## 这是什么

家族滞在 是日本入管法下专为支持中长期在日就劳者及留学者的家庭团聚而设立的 在留資格。其核心特征是依附性——被扶養者本人不享有独立 就労 资格，其在日资格完全依附于扶養者的 在留資格。

**适格扶養者的范围**：教授、芸術、宗教、報道、高度専門職、経営・管理、法律・会計業務、医療、研究、教育、技術・人文知識・国際業務、企業内転勤、介護、興行、技能、特定技能 2 号、文化活動、留学。

技能実習、特定技能 1 号、特定活動 等 在留資格 持有者**不可**作为 家族滞在 的扶養者招请家属。

**适格被扶養者的范围**：仅限配偶者与子女。父母、兄弟姐妹、祖父母不属于 家族滞在 的对象。

**就労**：家族滞在 持有者本人禁止从事一切收入性活动，包括打工。如希望就劳，必须事前向地方 出入国在留管理局 申请「資格外活動許可」，获批后方可在每周 28 小时以内从事兼职。

「28 小时」按每一周（任意起算的 7 日）计算，不是月平均。多份兼职合计也不得超过此限。学校长期假期内的 留学 生有别枠特例，但 家族滞在 没有类似宽限。

風俗営業（含風俗関連営業）相关工作，无论时间长短、是否取得 資格外活動許可，一概禁止。資格外活動許可書 上有明文记载。

## 你应该做什么

1. **境外召请家属**
   登录 [出入国在留管理庁 在留資格認定証明書交付申請页面](https://www.moj.go.jp/isa/applications/procedures/nyuukokukanri07_00251.html) 准备：
   - 在留資格認定証明書 交付申請書
   - 申请人证明照（3 个月以内）
   - 与扶養者关系证明（结婚证、出生证、户口本等公证认证文件）
   - 扶養者 在留カード 写し
   - 扶養者 在職証明書 或在学证明书
   - 扶養者直近年度的 住民票、課税証明書、納税証明書
   - 扶養者送金能力证明（存款余额证明、給与明細 等）

2. **境内变更或更新**
   另需 家族滞在 持有者本人的 住民票、在留カード 写し。

3. **打工前申请「資格外活動許可」**
   登录 [出入国在留管理庁 資格外活動許可申請页面](https://www.moj.go.jp/isa/applications/procedures/16-8.html)。可在新规入境时机场同时申请（包括式），也可入境后在地方入管局申请。包括式 資格外活動許可 适用于 家族滞在 与 留学，许可范围为「每周 28 小时以内、風俗営業除外」的兼职。
   申請書 一份、护照、在留カード 即可。

4. **打工时记录工时**
   按每一周（任意起算的 7 日）计算，包括所有兼职合计。建议自留 出勤記録 或 給与明細，以备更新审查时核对。

5. **查询自己的扶養者资格是否仍适格**
   扶養者从 技人国 转 経営・管理 仍可继续；从 技能実習 转 特定技能 1 号则家属不能继续以 家族滞在 同居。

## 不做的后果

- **超 28 小时打工**：在留期間更新 时入管比对扣税额与申报工时会发现。最常见的发现路径是雇主提交的 給与支払報告書、市役所 课税资料。一旦认定，更新申请被不许可；严重时被认定为「資格外活動違反」启动 退去強制手続。扶養者本人在更新审查中也会被减分。
- **未取得 資格外活動許可 即打工**：即使时间在 28 小时以内，仍属违反。
- **从事 風俗営業 相关工作**：记录留存于入管档案。日后申请 永住者、帰化、配偶者ビザ 变更时被作为不利事由审查。
- **扶養者切换到 技能実習 / 特定技能 1 号 / 特定活動**：家属不能继续以 家族滞在 同居，需另寻路径或暂时分离。
- **试图招请父母**：家族滞在 不适用于父母，须考虑 特定活動（老亲扶養）等其他路径。

## 什么时候你应该找专家

**你可以自己处理**，如果同时满足：
- 扶養者在适格 在留資格 列表内
- 被扶養者为配偶或子女
- 打工合计每周 28 小时以内
- 已取得 資格外活動許可
- 普通更新申请

**建议找专家**，如果出现以下任一：
- 扶養者将切换 在留資格（特别是切换到非适格类别）
- 想招请父母在日同住
- 已超过 28 小时打工或被入管问询
- 曾从事 風俗営業 相关工作
- 子女即将成年准备转 在留資格
- 被扶養者本人考虑申请独立的就労系 在留資格
