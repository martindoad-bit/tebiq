# 我刚落地，前 7 天每天做什么

## 这事跟我有关吗？
如果你刚拿着中长期在留资格（就労 / 留学 / 家族滞在 / 経営管理 等）落地日本、还没拿到正式住所、行李还在拖着——这篇适合你。如果你已经办完 転入届 并拿到 住民票，请直接跳到入职 / 报税 / 银行类文章。

## 这是什么
来日本第一周的核心是两件事：「住」（找到能签約的住所）和「身份激活」（让 在留カード 上的住所栏被填上、让 マイナンバー 进入系统）。这一周还不需要急着办手机长约或银行账户——这些都依赖住民票，而住民票要先有住所。

## 你应该做什么

### Day 1-2：入境后立刻确认的 3 件事
1. **检查 在留カード**：成田 / 羽田 / 関西 / 中部 / 福岡 / 新千歳 等机场入境时会现场发放 在留カード，住所栏写着「未定」是正常的。其他口岸入境时会贴 上陸許可印 在护照上，写着「在留カードを後日交付」，卡片会按你之后申报的住所邮寄。
2. **解决 7 天临时上网**：机场 SIM 柜台买 7-30 日 prepaid SIM（不用住所，护照即可），或提前在国内开 eSIM。这一步只是为了过渡，不签长约。
3. **查目标住所所属的 市役所 / 区役所**：在 [総務省 住民基本台帳ネットワーク](https://www.soumu.go.jp/main_sosiki/jichi_gyousei/c-gyousei/zairyu.html) 或直接搜「○○市 区役所 転入届」，确认窗口位置和办公时间（多数平日 8:30-17:15，部分周六上午开窗）。

### Day 3-5：找住所并签约
1. **理解房租结构**：除月租外，初期费用通常包含 敷金（押金，1-2 ヶ月）、礼金（不退还，0-2 ヶ月）、仲介手数料（0.5-1 ヶ月）、保証会社費用（0.5-1 ヶ月）。整体初期费用约房租的 4-6 倍。
2. **保証人 / 保証会社**：多数物件要求 保証会社加入（连帯保証人 多由公司或学校充当）。无 保証人 时可走纯 保証会社 路线，部分外国人友好物件直接公示。
3. **签約时一次拿齐**：賃貸借契約書 副本、入居日记录、鍵 引き渡し书面、保証会社 契約書。

### Day 6-7：身份激活
1. **拿到住所当天 / 第二天去 市役所 / 区役所** 办 [住居地届出（転入届）](https://www.moj.go.jp/isa/applications/procedures/16-3.html)。带：在留カード、パスポート、賃貸借契約書 或 住所が確認できる書類。窗口会把住所写到 在留カード 背面。
2. **同窗口（或邻窗口）触发** 住民票 登録 + マイナンバー 通知書 邮寄（按现行制度，マイナンバーカード 通知书在登録后 1-2 周邮寄到登録住所，不是当场拿）。
3. **国民健康保険 加入**：如果你不是公司社保对象（社員 / 自営業 / 学生 / 家族滞在 等），同日在国保窗口加入。带：在留カード、住民票（当日可同时取）。资格自 来日 当日 起算（不是登録 当日）。

## 不做的后果
- 住居地届出 超过 14 日 不报：入管法 第 19-7 条规定 14 日内报告住所，超期可能影响下次更新审查记录，最坏情况是 在留資格 取消 的法律根据成立（不等于一定取消）。
- 没办住民登録 → 无法开银行户、无法签手机长约、无法加入国保。
- 临时 SIM 用完未续 → 最多损失通信，不影响身份。

## 什么时候你应该找专家
独自落地、住所、転入届、国保 流程标准化，绝大多数人靠 市役所 窗口的多语种对应 + 翻译 App 即可自办。以下情形再考虑专家协助：
- 在留カード 信息错误（姓名、生年月日、在留期間 错印）需在 7 日内向地方入管申请订正
- 家族滞在 配偶 / 子女随后入境，需要同步申请 在留資格認定証明書
- 経営管理 在留 来日，事业所选址直接影响 在留審査
