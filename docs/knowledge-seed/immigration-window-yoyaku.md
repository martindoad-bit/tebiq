# 入管局窗口预约怎么做 / 排队多久

## 这事跟我有关吗？

如果你需要去 出入国在留管理庁（俗称"入管"）窗口办 续签 / 资格变更 / 永住申請 / 在留カード 更新 / 再入国，先看这篇。

最大的两个误区：

1. "我直接去窗口就行" → 东京入管 在留審査 部门高峰期排队 2-4 小时
2. "我必须去窗口" → 实际上多数手续可以**在线申請** 或 **邮寄**，省队

## 这是什么

全国 出入国在留管理局 主要分局：

| 局 | 所在地 | 主要管辖 |
|---|---|---|
| 東京出入国在留管理局 | 東京都港区港南 5-5-30 | 東京・千葉・茨城等 |
| 横浜支局 | 横浜市金沢区 | 神奈川 |
| 大阪出入国在留管理局 | 大阪市住之江区 | 大阪・京都・奈良・滋賀・和歌山 |
| 名古屋出入国在留管理局 | 名古屋市港区 | 愛知・三重・岐阜・福井・富山・石川・静岡 |
| 福岡出入国在留管理局 | 福岡市博多区 | 九州・沖縄 |
| 札幌出入国在留管理局 | 札幌市中央区 | 北海道 |
| 仙台出入国在留管理局 | 仙台市宮城野区 | 東北 |

每个局还有更多支局・出張所。具体管辖以你住民票登记的地址为准。

**受理时间（共通）**：

- 月-金 9:00-12:00 / 13:00-16:00（午休 12:00-13:00 不办理）
- 周末 / 节日 / 12/29-1/3 不办理

## 你应该做什么

**优先级排序**：

**1. 在线申請（最优）**

- [在留申請オンラインシステム](https://www.ras-immi.moj.go.jp/) 2026/01/05 改版后**本人也可使用**（之前必须通过律师 / 行政書士 / 雇主登录）
- 24 小时受付、不用排队、审查进度可查
- 可申請：在留期間更新、在留資格変更、永住、再入国 等几乎所有手续
- 必备：マイナンバーカード + 读卡器（手机 NFC 也可）

**2. 窗口直接受理**

- 早上 8:30 前到，排队约 30 分钟-1 小时
- 9:30 后到（东京・新宿支局），排队 2-4 小时
- 取整理券：进场后取号，号码顺序受理
- 拿 申請受理票 后即可离开，不用等审查结果

**3. 邮寄申請**

- 部分手续可邮寄（在留期間更新、再入国 等）
- 信封写"在留期間更新申請書 在中"，挂号
- 入管局 拿到后会邮寄回 ハガキ 通知"已受理"
- 不是所有局都受理邮寄，事前电话确认

**避开高峰**：

- 周一・连休后：最拥挤
- 月末：拥挤
- 周三・周四 上午 10:00-11:30：相对空
- 4 月、9 月：留学生续签高峰
- 12 月末・3 月末：避开

**带的东西**：

- 申請書 + 全部材料（一式 1 套，部分手续 2 套）
- 在留カード 原件
- 護照 原件
- 写真 1 张
- 收入印紙（4,000 円 续签用，部分窗口隔壁有卖）
- 笔（窗口有，但自带稳）

## 不做的后果

- 没有事先准备 → 现场被告知"缺这张那张" → 重排队
- 不知道在线申請 → 浪费半天工作时间
- 邮寄不挂号 → 寄丢无证据
- 走错局（按户籍而非住所） → 当场被退

## 什么时候你应该找专家

以下情况建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）：

- 你的案件复杂（換職、収入下降、过去拿过短期），不确定窗口能否当天受理
- 在留申請オンラインシステム 你不会用 / 没有 マイナンバーカード
- 经营・管理签证 / 高度専門職 等材料厚度大
- 你时间紧，希望全程委托代行
