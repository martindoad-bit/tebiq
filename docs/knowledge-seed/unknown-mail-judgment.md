# 我收到一封看不懂的日文信，怎么判断是不是要紧

## 这事跟我有关吗？

你刚到日本不久 / 日语阅读还不熟练，收到了一封不知道寄件人是谁、写了什么、要不要做什么的日文信件——这篇适合你。在日生活前 1～2 年里，几乎每个人都会经历这种「**这封信看起来很官方、但我看不懂**」的瞬间。**90% 的紧急通知都不会通过电子邮件发——只走纸质邮政**，所以收到不明日文信不能直接扔。

## 这是什么

日本的行政机关 / 准公的机关 / 公共服务机关 都习惯使用 **纸质郵便** 发出正式通知。即便你已注册了 マイナポータル / 各机关的网络账号，**重要通知仍会以纸质 + 親展（しんてん）封筒** 寄到登录住所。看不懂不是因为你日语差——是因为日本行政文体本身就刻意正式 / 繁琐。

## 你应该做什么

### 三步快速判断法

#### 第 1 步：看寄件人（封面上方 / 印章 / 差出人欄）

记住下面这几个关键 **机关名汉字**——见到任何一个都要 **当天拆开**：

| 机关名（汉字） | 业务领域 |
|---|---|
| **市役所 / 区役所 / 町役場 / 村役場** | 住民票、住民税、国保、介護、保育、児童手当 |
| **税務署 / 国税庁** | 所得税、消費税、確定申告、源泉徴収 |
| **都税事務所 / 県税事務所** | 都道府県民税、自動車税 |
| **年金事務所 / 日本年金機構** | 国民年金、厚生年金 |
| **健康保険組合 / 協会けんぽ** | 健康保険 |
| **出入国在留管理庁 / 入国管理局 / 地方出入国在留管理局** | 在留资格、在留卡、永住、再入国 |
| **法務局 / 法務省** | 戸籍、登記、帰化、相続 |
| **警察署 / 警視庁 / 県警** | 各种警察事務、紛失届、违法相关 |
| **裁判所** | 民事 / 刑事 / 家事 司法事務 |
| **労働基準監督署 / ハローワーク** | 雇用保険、労災、職業紹介 |
| **検察庁** | 刑事関連——非常严肃 |
| **公証役場** | 公正証書関連 |

**反过来：私营企业、广告 / DM、不明业者** → 不需立刻回应、但要分清楚不是诈骗。

#### 第 2 步：看封面 / 第一页关键词

| 关键词（汉字） | 意思 / 紧急程度 |
|---|---|
| **通知書 / お知らせ** | 一般通知、看清内容即可 |
| **決定通知書** | 已经做出 行政决定、要看具体决定内容 |
| **督促状** | **滞纳催缴、要紧、见 [tokusoku-shiyakusho.md](./tokusoku-shiyakusho.md)** |
| **催告状 / 最終催告状** | 比督促状更急 |
| **差押予告 / 差押通知** | **极其紧急、立即专家** |
| **出頭通知書 / 呼出状** | 要求本人出頭，**不可缺席** |
| **不許可通知書** | 申请被拒、注意 3 ヶ月内审查请求期限 |
| **補正通知 / 資料提出のお願い** | 要求补充材料、通常 14 日期限 |
| **還付 / 還付金 / 還付通知** | 退钱给你、好消息（注意诈骗、官方还付不会要求你点 URL） |
| **入金 / 振込通知** | 钱进账通知、好消息 |
| **検診 / 健診のご案内** | 健康检查通知、非紧急 |
| **選挙のお知らせ / 投票所入場券** | 選挙関連（有投票权者）|
| **照会 / 問い合わせ** | 询问 / 调查、需要回应 |

#### 第 3 步：看封口标识

- **赤字「速達」「書留」「親展」「重要」「至急」** → 几乎都是急事
- **本人限定受取 / 配達証明** → 邮差要求出示在留卡 / 当面签收 → **法律意义重大的文书**（不利处分通知 / 訴状 等）
- **裁判所 から の 特別送達** → **绝对不能无视**——可能是支払督促 / 訴状 / 仮処分 决定

### 用工具辅助翻译

1. **Google Lens / Google 翻訳 拍照翻译**：第一步快速读懂大意。
2. **DeepL アプリ**：长段落翻译质量较高。
3. **TEBIQ 拍照即懂功能**：拍封面 + 关键页、即时识别 寄件机关 + 紧急程度 + 推荐应对。
4. **不能完全依赖机器翻译**——关键的 **期限 / 金额 / 法律术语** 容易翻错。重要信件最终请人确认。

### 判断后的行动决策树

- **不知道是谁寄的、看不出关键词** → 拿到附近 **市役所多文化共生窗口 / 国際交流協会** 求助（多自治体提供 **中国語 / 英語 通訳** 无料服务）
- **认出是入管局** → 见 [immigration-letter-judgment.md](./immigration-letter-judgment.md)
- **认出是市役所税務 / 督促状** → 见 [tokusoku-shiyakusho.md](./tokusoku-shiyakusho.md)
- **认出是年金事務所** → 见 [blue-envelope-nenkin.md](./blue-envelope-nenkin.md)
- **认出是裁判所** → **当天联系日本人朋友 / 律师 / 司法書士**——不是 营销广告
- **看起来像詐欺 / 不审 業者**（自称银行 / 警察 / 国税庁 但要求点击 URL / 转账 / 提供卡号）→ 不要回应、向 **消費者ホットライン 188** 通报

## 不做的后果

- **错过 入管 補正期限** → 申请直接 不許可
- **错过 督促状 应对期限** → 触发 差押 / 強制徴収
- **错过 不許可后的 3 ヶ月 审查请求期** → 行政救済彻底关闭
- **无视 裁判所 支払督促 / 訴状** → 缺席判決、债务确定、強制執行
- **无视 健康診断 案内**（仅生活影响）→ 错过免费 / 补助检查、但不是法律问题
- **无视 マイナンバーカード 关连通知** → 部分公的サービス 受限
- **永住 / 帰化 申請履历**：常年无视行政通知 = 「**素行善良性 不適合**」推定的间接证据

## 什么时候你应该找专家

- **任何裁判所文书** → 弁護士 / 司法書士（依事案种类）
- **入管 不利处分通知 / 出頭通知** → 行政書士（申請取次） / 入管専門弁護士
- **督促 / 差押 関連** → 司法書士 / 弁護士 / 税理士（依税目）
- **看不懂但金额超过 10 万円 / 涉及 法律行為**（契約解除 / 訴訟 / 強制執行 など）→ 弁護士相談（多自治体提供 **無料法律相談**）
- **怀疑是詐欺**（特殊詐欺 / オレオレ詐欺 / 還付金詐欺）→ **警察 #9110**、**消費者ホットライン 188**
