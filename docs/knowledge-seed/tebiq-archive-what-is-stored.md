# 我的 TEBIQ 档案保存了什么

## 这事跟我有关吗？
你在 TEBIQ 拍照识别过文书、做过续签自查、用过文字即懂、或写过任何手写备注，这些都进入了你的档案。档案是 TEBIQ 的产品基座，决定了续签自查、政策匹配、咨询准备能否跨年度连贯。看一遍档案到底存了什么，可以判断是否值得长期保留。

## 这是什么
档案是 TEBIQ 给每位注册用户的中心化数据集合。所有工具的调用结果都归档到时间线，按时间倒序排列，可按文书类型、在留資格、年度过滤。

档案保存的内容：

- **拍照识别结果**：doc_type（在留カード / 課税証明書 / 源泉徴収票 / 納税証明書 / 確定申告 控え / 雇用契約書 等）、issuer（市役所 / 税务署 / 雇主名）、amount（年收入、纳税额）、deadline（有效期、缴纳期限）、完整识别 payload（原始结构化字段）
- **自查结果**：当前在留資格、续签风险打分、历史每次自查的得分变化
- **文字即懂查询历史**：你查过的术语、对应的解释、相关文书
- **用户手写备注**：你在某条文书或时间线节点上写的自由文字
- **政策匹配标签**：基于你的档案匹配到的政策提醒（例如 2026 年永住制度变更、2025 年経営・管理签证调整）

档案不保存的内容：

- **原始照片**：识别后立即丢弃，仅保留结构化结果
- **マイナンバー 个人番号原文**：识别时屏蔽
- **银行账户具体号码**：除非你在手写备注里主动写入

## 你应该做什么
1. 打开 TEBIQ 档案首页，查看时间线视图，按文书类型过滤一次，确认识别结果和你预期一致。
2. 检查"待补"区域，看是否有续签所需文书还没拍照入档。
3. 在续签前 6 个月再回看一次，确认 1-3 年课税 / 纳税 / 在职履历齐全。

## 不做的后果
不查看档案不会损失什么，但续签前临时翻找文书的时间成本会显著高于持续维护的档案。30 天免费保留期到期后，档案进入 archived 状态，时间线视图不再显示。

## 什么时候你应该找专家
档案识别结果与原件不一致、或 doc_type 识别错误：先在 TEBIQ 内点击"修正"重新识别。涉及在留資格变更、归化、永住准备等需要把档案数据转成正式申请材料的场景，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
