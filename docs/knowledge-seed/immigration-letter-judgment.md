# 我收到入管局来信，怎么判断是好消息还是坏消息

## 这事跟我有关吗？

你最近收到了一封寄件人为「**出入国在留管理庁**」「**○○出入国在留管理局**」「**○○地方出入国在留管理局**」「**法務省入国管理局**」（旧称）、或是你的 **担当行政書士 / 受入機関 / 所属企業** 转交给你的入管来信——这篇适合你。无论你是申请者本人、雇主代理、还是家族滞在的家属，都需要快速判断这封信的紧急程度。

入管来信几乎都是 **时限敏感** 的——错过期限轻则申请白做，重则在留资格直接失效。

## 这是什么

入管局发出的信件 / 通知，按 **结果好坏 + 紧急程度** 分为以下几大类：

### 好消息（许可类）

- **「在留資格認定証明書（COE）交付通知」**：海外申请人通过雇主 / 代理人收到，凭此证明书赴日大使馆办签证。封面写「**在留資格認定証明書交付**」、内附 COE 原件
- **「在留期間更新許可通知書 / はがき」**：续签获批通知，要求你 **持つ在留卡 + 通知書 + 手数料 4,000 円（収入印紙）+ パスポート** 赴入管局换发新卡
- **「在留資格変更許可通知書 / はがき」**：变更签证种类（如 留学 → 技人国）获批
- **「永住許可通知書」**：永住申请获批，需赴入管换发永住者在留卡
- **「再入国許可 / みなし再入国 関連通知」**

判断标志：含「**許可**」「**認定**」「**交付**」字样

### 需立即回应（中性 / 风险类）

- **「資料提出通知書」/「追加資料提出のお願い」**：审查中需要补充材料。**通常給予期限 14 日 程度（1～2 周）**，超期可能直接不许可
- **「補正通知 / お問い合わせ」**：申请书 / 添付資料に不備があり、補正を求められる
- **「出頭通知書 / 呼出状」**：要求本人到入管局面谈、说明情况。日期固定、不能随意更改
- **「届出受理通知 / 届出済証明書」**：所属機関 / 契約機関 変更届的受理回执，本身是好消息但要保管好

判断标志：含「**提出**」「**補正**」「**出頭**」「**呼出**」「**お問い合わせ**」字样

### 坏消息（不利处分类）

- **「不許可通知書」**：申请被拒。通常会附 **理由要旨**、并记载「**不服がある場合は処分の通知を受けた日から 3 ヶ月以内に審査請求が可能**」
- **「在留資格取消通知書」**：依《入管法》第 22 条之 4 取消在留资格。会附 **意見聴取通知**、有 **30 日内出国 / 申辩** 的猶予期間
- **「退去強制令書」/「退去強制手続関連通知」**：最严重一类，涉及强制驱逐
- **「警告書 / 違反調査関連通知」**：怀疑有不法滞在 / 不法就労 等违反，要求出頭

判断标志：含「**不許可**」「**取消**」「**退去強制**」「**違反**」字样

### 信封外观判断思路（仅供初步判断、最终以内容为准）

- **公文常见信封**：白色，左上角 **法務省 / 出入国在留管理庁** 字样，**親展（しんてん）** 印
- **重要 / 急ぎ 含み**：封面有「**速達**」「**書留**」赤字
- **配達証明 / 本人限定受取郵便**：邮差要求当面签收 / 出示 在留卡 → **几乎都是重要内容**（许可通知书 / 不利处分通知）
- **薄信封（はがき / 1 枚）**：常常是 **更新 / 変更許可的简易通知（はがき形式）**，要求带原信赴入管换卡
- **厚信封（多枚）**：可能是 **不許可通知 + 理由書** 或 **資料提出依頼 + 補正案内**

## 你应该做什么

1. **当天拆开**。**任何入管来信都不要拖过 24 小时**，更不要拖过 7 天。
2. **看封面 + 第一页大标题**：找前述关键词（許可 / 不許可 / 提出 / 出頭 / 取消 / 補正）。
3. **找期限**：
   - 補正 / 資料提出 → 通常 **14 日内**
   - 出頭 → **指定日不可缺席**（如何确实无法出席，需立即电话联系入管局担当部门请求改期）
   - 不許可 後の審査請求 → **3 ヶ月内**
   - 取消手続 → **意見聴取通知から 30 日程度**
4. **不要私自处理不利通知**。收到 **不許可 / 取消 / 退去強制 / 違反調査** 类通知时：
   - 不要立即出国（部分情况下出国会丧失审查请求权）
   - 不要在出頭时不带任何资料 / 不带翻译就独自前往
   - 立即联系 **行政書士（申請取次資格付き）** 或 **入管法専門の弁護士**
5. **保管好原件**。无论是好消息还是坏消息，原件都是 **法律文书**——不要扔、不要折叠、扫描归档。
6. **不识日文 / 看不懂术语**：用 Google Lens 拍照翻译做第一步判断，但 **关键期限和回应方式必须找懂日语 + 入管法的人确认**。

## 不做的后果

- **不在指定期限内提交資料 / 補正** → 申请按现有材料审查、通常直接 **不許可**
- **不出頭** → 视为放弃陈述权 / 申辩权、不利处分自动确定
- **不在 3 ヶ月内提出審査請求** → 不許可处分确定、丧失行政上的救済手段
- **不应不許可而继续滞留至在留期間満了** → **不法残留**（オーバーステイ）、最高 3 年以下懲役 或 300 万円以下罰金（《入管法》第 70 条第 1 項第 5 号）
- **取消手続中无视通知出国** → 5 年间不能再入国（一定情况下）

## 什么时候你应该找专家

- **任何含有「不許可」「取消」「退去強制」「違反」字样** → 立即联系 行政書士（申請取次） / 入管専門弁護士
- **資料提出通知，但你不知道要补什么 / 怎么补** → 行政書士
- **出頭通知，且你日语 听 / 说 不流利** → 委托行政書士陪同（部分行政書士提供同行サービス）
- **永住申请不許可、考虑再申请** → 行政書士分析 不許可理由 后规划再申请策略
- **在留期間 即将届满（残 1 个月内）但仍在審査中** → 联系行政書士 + 直接前往入管确认 **特例期間** 适用情况
