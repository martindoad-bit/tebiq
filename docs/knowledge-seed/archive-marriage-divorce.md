# 我刚结婚 / 离婚，档案要更新什么

## 这事跟我有关吗？
你刚提交 婚姻届 或 離婚届，或正在准备办理。无论配偶是日本人、永住者还是同为外国人，婚姻关系变动会触发在留資格、税务、社保、家族构成等多条线索的同步更新。档案里这些事件不同步，下一次续签、永住申请、或配偶来日的家族签证申请会直接对不上数。

## 这是什么
婚姻届 / 離婚届 提交后，档案需要更新的核心事件：

**结婚情况**
- 婚姻届 必须在结婚 14 日内向 市役所 / 区役所 提交，受理后档案标记「已结婚 + 配偶资料」（配偶姓名、国籍、出生年月、配偶在留資格）。
- 配偶若是日本人或永住者：档案自动开始计算永住缩短判断。日本人 / 永住者の配偶者等 在留資格持有者，永住申请的居住要求是婚姻 3 年以上 + 在日 1 年以上（普通要件 10 年的特例）。
- 配偶若在海外，需申请来日：先做 在留資格認定証明書 申请（COE），档案新增「家族构成」事件，配偶取得『日本人の配偶者等』或『家族滞在』在留資格。
- 婚后住所变更：同步提交 転入届 / 住所変更，档案更新住民票履历。

**离婚情况**
- 離婚届 14 日内向 市役所 提交，受理后档案标记「配偶者关系終了」+ 离婚日期。
- 持有『日本人の配偶者等』或『永住者の配偶者等』者：婚姻关系終了起 6 个月内未做 在留資格変更 = 取消事由。原配偶者签证原则失效，需变更为 定住者（有抚养未成年子女、3 年以上婚姻实绩等条件）或 短期滞在 后再考虑下一步。
- 财产分与、年金分割、子女親権 等记录单独留存在档案的「事件备注」区，不参与续签判断但供 確定申告 与 永住申请说明使用。

## 你应该做什么
1. 提交 婚姻届 / 離婚届 的当日，在 TEBIQ 拍照上传 受理証明書，档案自动归档为婚姻状态变更事件。
2. 配偶来日的 COE 申请，先在档案里调出「家族构成」事件，把配偶资料、收入证明（直近 1 年源泉徴収票）、住居证明（賃貸借契約書）一次列齐。
3. 离婚后 30 日内，把当前 在留資格 与离婚事件交叉核对一次。如果是配偶者签证，档案会在 D-30 / D-90 / D-180 三个时点提示「6 个月期限将至」。
4. 婚后或离婚后住所变更，同步提交 転入届 / 転出届 / 転居届，档案更新住民票事件。

## 不做的后果
婚姻届 / 離婚届 14 日内不提交属 戸籍法 违反，可能受过料处分。配偶者签证持有者婚姻终了 6 个月内不做 在留資格変更，档案里就留下一段「身份与活动不一致」的空窗，下一次续签或永住申请会被入管追问，部分情况会被认定取消事由直接撤销在留資格。

档案里婚姻事件缺失，永住缩短判断（婚姻 3 年 + 在日 1 年）的起算日就无凭据，需要事后补提交 戸籍謄本 / 婚姻届受理証明書，时间成本远高于当下入档。

## 什么时候你应该找专家
婚姻届 / 離婚届 本身不需要书士，市役所窗口直接办理。但以下场景建议咨询行政書士：（1）配偶在海外，准备做 COE 申请；（2）正在持有配偶者签证、刚离婚或正在办理离婚，需要判断变更为 定住者 还是其他在留資格；（3）婚姻 3 年准备走配偶者特例申请永住。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
