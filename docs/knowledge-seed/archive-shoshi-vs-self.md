# 哪些 case 自己解决 / 哪些必找行政書士

## 这事跟我有关吗？
你不想为本可以自己办的事多花咨询费，但也不想为低估了复杂度的 case 在没有专业意见的情况下 強行 提交而吃 不許可。这篇按 case 类型给出客观判断标准。

## 这是什么
**自己能办的 case（标准化、要件清晰、风险低）：**

- **常规 在留期間更新（续签）**：5 年内同公司同岗位、收入稳定、无 滞納、无未届出事件、无家庭结构变化。在留申請オンライン 自己提交即可，档案的续签自查已能覆盖材料清单。
- **住所変更 后的 住居地届出**：14 日内携 在留カード 至新住所所属 市区町村役场，背面盖章即完成。
- **転入届 / 転出届**：搬家时市役所 标准窗口业务，免费。
- **課税証明書 / 納税証明書 取得**：所在市区町村 税务窗口或 マイナンバーカード 网络申请，¥300 / 通。
- **マイナンバーカード 申请**：在线申请 + 市役所 取卡，免费。
- **普通税务申告（給与所得 単一来源）**：使用国税庁「確定申告書等作成コーナー」自己完成。
- **国民年金 加入 / 喪失**：年金事务所 / 市役所 標準窗口，自己办即可。

**建议咨询 行政書士 的 case（要件复杂、判断主观、风险高）：**

- **永住申請**：要件审查涵盖 引き続き 10 年在留 / 独立生計 / 公租公課 完納 / 素行善良 等多条主观标准；2026 年制度修订后 公租公課 不履行的影响显著加重，自己判断要件是否齐备的失误成本约为 4 个月审查时间 + 不許可记录。
- **帰化申請**：要件涵盖 住居 5 年 / 能力 / 生計 / 不喪失 / 思想 / 日本語 等 6 条；提出前的 法務局 事前面谈环节复杂，行政書士 同行可显著降低反复跑访次数。
- **不許可通知 后再申请**：以同一理由再申请通常被认定为缺乏新证据，需重新构造 case；策略性极强。
- **補正通知 回应**：補正 范围理解错误会被视为不补正而 不許可，且补正期限通常 2 周。
- **配偶者签证 真实性审查严格**：偽装結婚 审查近年加强，質問書 回应稍有不慎被解读为矛盾陈述，许可率显著下降。
- **経営・管理 5 项硬条件审视**：500 万円以上 出資 / 事務所要件 / 2 名以上常勤雇用 或同等规模 / 事業計画 实质性 / 决算 持续性 等，每项都有判例边界。
- **高度人材 80 分边界**：得点表上 75-80 分附近的 case，调整某一项加点（学历 / 年齢 / 研究実績 / 收入）可能从 75 跳到 80 直接获 1 号在留，决策价值高。
- **転職後 岗位偏离原 在留資格**：例如 技人国 中"人文知識・国際業務"持有者转入纯翻译外的岗位、或"技術"持有者转入纯营业岗位，岗位与在留資格 的契合度判断需要专业意见。

**判断的客观标准：**

- 这件事的失败成本是否超过 ¥9,800？（不許可 = 4 个月审查白等 + 记录在卷 + 再申请难度上升 → 远超）
- 公式信源是否有清晰的判断标准？（「収入安定性」「素行善良」「独立生計」 是主观判断，没有数值标准）
- 同类 case 是否有近 1 年的判例 / 实务变化？（永住 / 経営・管理 / 高度人材 近年实务变化频繁）

任何一项答"是"，倾向于咨询；三项都答"否"，自己办即可。

## 你应该做什么
1. 把当前要处理的事项对照上面两类清单。
2. 不确定属于哪一类时，用判断标准的三个问题再过一遍。
3. 仍不确定的事项，至少先用 TEBIQ 知识库做一次扫描——若知识库已有标准化指引，按指引办即可；若知识库提示"建议咨询"，再考虑 ¥9,800 咨询。

## 不做的后果
属于"建议咨询"清单的 case 自己办，最常见的两种结果：申请被 不許可（成本：4 个月 + 再申请难度上升 + 记录在卷）；或 補正通知 回应不充分被 不許可（成本：再申请需重新付申请费 + 重新等 2-4 个月）。两种 case 的累计成本均显著超过 ¥9,800。

## 什么时候你应该找专家
你的 case 落在"建议咨询"清单内、或近 1 年内政策有变更（特别是 永住 / 経営・管理 / 高度人材 / 配偶者）、或档案中已识别的风险事件需要专业判断，建议咨询行政書士。TEBIQ 提供专业咨询服务（[预约](/consultation)）。
