# 我想做副业，能赚多少 / 哪些不能做

## 这事跟我有关吗

如果你符合以下条件，请继续往下看：

- 现在持有 就労系（技人国 / 経営・管理 / 高度専門職）/ 家族滞在 / 留学 任一在留資格
- 主业之外想做：兼职、顾问、freelance、内容创作、电商、投资、不动产收租 等
- 想知道哪些不报会出事 / 哪些不能做

身分系 在留資格（永住者 / 日本人の配偶者等 / 永住者の配偶者等 / 定住者）几乎没有副业限制，本条不针对你们。

## 这是什么

日本对外国人 副业 的限制不是按"金额"限制，而是按 **在留資格 範囲** 限制。规则分两类：

### 就労系（技人国 / 経営・管理 / 高度専門職）

- 副业内容 **属于现 在留資格 範囲内** → 不需要追加许可，可以做
  - 例：技人国（ITエンジニア）副业接 IT 開発外包 → 在范围内
- 副业内容 **超出现 在留資格 範囲** → 必须申请 **資格外活動許可**
  - 例：技人国（ITエンジニア）周末去做コンビニ店員 → 超出，需要許可
- 経営・管理 持有人想 兼任 他社 役員 → 一般在范围内，但要看是否有実質的経営参与

### 家族滞在 / 留学

- 这两个本来都是"非就労系"，**本身禁止任何 収入活動**
- 申请 資格外活動許可 后：每周 28 小時 上限（学校 长期假期 留学生 1 日 8 時間）
- 即使有 許可 也不能做 **風俗営業 関連業務**（パチンコ / スナック / キャバクラ 等，不分職種）

### 投资性收入（不分签证）

- 株式 / 投資信託 / 暗号資産 配当・売却益 → 不算"就労"，不需要 許可
- 不動産 賃貸 1–2 件 個人レベル → 一般不算就労（被认定为"財産管理"）
- 不动产 5 件以上 / 持续大规模运营 → 可能被认定为 事業 → 需要 経営・管理 路线

## 你应该做什么

### 副业开始前

1. 判断副业内容是否属于 在留資格 範囲：
   - 不确定 → 入管 [外国人在留総合インフォメーションセンター](https://www.moj.go.jp/isa/consultation/center/index.html) 0570-013904 可以匿名问
2. 超出范围 → **資格外活動許可 申請**（在留カード 持参 + 申请书 + 雇用契約書 / 業務内容 説明書，地方入管，无料，1–2 月）
3. 与本職 公司确认是否有 副業禁止規定（労働契約 / 就業規則）

### 副业进行中

4. 留底 副業 収入证明（支払調書 / 報酬明細 / 振込履歴）
5. **副業所得 20 万円 / 年 超过 → 確定申告 必要**（給与所得者の場合，即"主业是 給与 + 副业 20 万 超" → 第二年 2/16–3/15 自主申报）
6. 主业是個人事業主 / 経営・管理 一律 確定申告，无 20 万 免征线
7. 住民税：申告时选"普通徴収"可以避免 副业 被本职公司发现（仅适用于副业不是 給与 的情况）

### 留学 / 家族滞在 特别注意

8. 必须 28 小時 / 週（複数のバイトの合計）。超过 1 小时也可能 取消事由
9. 風俗営業 場所 内的工作（即使做收银 / 清扫）一律禁止——是按"場所"判定不是"職務"

## 不做的后果

- **未申请 資格外活動許可 做了範囲外副业**：违法就労 → 取消 在留資格 + 退去強制 + 5–10 年 上陸拒否；雇主 不法就労助長罪 3 年以下懲役 / 300 万円 罰金
- **留学 / 家族滞在 超过 28 時間**：在留期間 更新 直接拒否案例多发；情节重 = 取消
- **副业 確定申告 漏報**：本税 + 無申告加算税 15–20% + 延滞税 + 重加算税（恶意时 40%）
- **風俗営業 地点 工作**：即使有 資格外活動許可 也不行——直接 取消事由

## 何时找专家

下面任一情况，请联系 行政書士 / 入管専門 弁護士 / 税理士：

- 副业接近"独立事業"规模（年商 数百万円，雇人，租办公室）→ 你应该考虑 経営・管理 在留資格 変更
- 副业是 海外公司 远程工作 / 海外报酬（要判断"日本国内活动"还是"海外所得"）
- 高度専門職 持有人 兼任 / 兼业（規定 比一般 就労系 更严，要重新算ポイント）
- 留学 / 家族滞在 想长期增加收入 → 是不是该走 在留資格 変更 路径
- 副业被本职公司发现 + 公司有 副業禁止 規定（劳动法 + 入管 双轨）

書士 在 副业 节点的价值是判断你这个副业是 該当範囲 / 申请就行 / 必须変更 哪一类——很多人以为申请 資格外活動許可 就万事大吉，实际上 就労系 的人范围内做副业根本不需要申请，而范围外申请也未必能批。
