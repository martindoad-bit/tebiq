## GOLD-01 — 公司休眠了要不要交国民年金？

```yaml
question: 公司休眠了要不要交国民年金？

aliases:
- 公司休眠了要不要交国民年金
- 公司不营业了社保怎么办
- 公司停了我的年金怎么办
- 公司决议休眠 厚生年金 怎么办
- 转职中间没有社保怎么办
- 老板说公司要关 我自己要交什么
- 厚生年金到国民年金切换
- 健保资格丧失证明书 拿到后怎么办
- 公司停业 个人年金义务

intent_keywords:
- 公司休眠
- 公司停业 / 公司不营业
- 厚生年金 → 国民年金
- 资格喪失
- 个人年金义务
- 国保切换

must_not_match:
- 经营管理签证还能续吗
- 经管 改正 既存
- 在留资格 取消程序
- 经营管理 续签

answer_type: workflow
risk_level: medium

applies_when:
- 公司决议休眠 / 解散 / 停业，你的厚生年金 / 健康保険 即将失效
- 已收到「健康保険資格喪失証明書」
- 离职后还没进入新公司（空白期）

does_not_apply_when:
- 公司还在正常运营，本人未离职
- 已经直接进入下家公司 + 新公司当日完成厚生年金加入
- 想问的是公司休眠对経営・管理 在留资格 的影响（属于另一条线，需另查）

最紧的两件:
- 取得「健康保険資格喪失証明書」（公司应在喪失日起 5 日内交付）
- 喪失日起 14 日内 切换 国民年金 第 1 号 + 国民健康保険

步骤:
- 1. 联系公司或元雇主，索取「健康保険資格喪失証明書」+ 確認 喪失日
- 2. 携 喪失証明書 + 在留カード + マイナンバー 通知書 → 居住地 区役所 / 市役所 国保年金課
- 3. 同时办理 国民健康保険 加入届 + 国民年金 第 1 号被保険者 加入届
- 4. 收入低 / 失业 / 倒産特例 → 同窗口申请 国保保険料 軽減 + 国民年金 免除申請
- 5. 收 新健康保険証 + 年金加入記録更新；如短期会进入新公司，新公司入职日改回 厚生年金

去哪办:
- 居住地 区役所 / 市役所 国保年金課（一站式办理 国民健康保険 + 国民年金）
- 必要时 居住地管轄 年金事務所（单独办理国民年金部分）
- 倒産特例軽減 / 失业認定 → 同时去 ハローワーク

要带什么:
- 健康保険資格喪失証明書（公司发行）
- 在留カード
- マイナンバー 通知書 / カード
- 印鑑（部分自治体要求）
- 离職票（如同时申请失业 / 倒産特例軽減）
- 銀行通帳（口座振替申请用）

期限:
- 喪失日起 14 日内（国民年金 + 国民健康保険 加入义务）
- 喪失証明書：公司应在喪失日起 5 日内发给本人
- 国保保険料 軽減 / 国民年金 免除：同窗口办理或后续补申

不做会怎样:
- 14 日内未办：保険料 遡及徴収（最大 2 年分）+ 后日督促状
- 病院就诊：100% 自費（後日 国保加入後 7 割 還付申請可，但流程繁琐）
- 永住申請 直近 2 年「公的義務履行」评价负分
- 国民年金 长期未缴：将来年金额减少 + 老龄给付资格不足

要找专家的情况:
- 公司过去 2 年以上 未缴厚生年金（雇主未届出 → 本人加入記録缺失）→ 社労士
- 倒産特例給付（雇用保険）+ 国保軽減 同时申请 → 社労士
- 配偶扶養加入审查 被拒 / 异议 → 社労士
- 公司休眠后你的 経営・管理 在留資格 是否影响 → 行政書士（这是另一条线）

复制给客户: |
  公司休眠不等于个人年金义务消失。喪失日确认后 14 日内必须切到 国民年金 第 1 号 + 国民健康保険，否则保险料会被 遡及徴収 最大 2 年分，且影响后续永住审查。今天先去公司要「健康保険資格喪失証明書」，然后带 在留カード + マイナンバー 去居住地 区役所 国保年金課 一站办理。如果同时收入低或申请倒産特例軽減，可以先和社労士聊一下。

source_hint:
- 日本年金機構（国民年金 加入手続）
- 厚生労働省（国民健康保険 加入・脱退）
- 居住地 区役所 / 市役所 国保年金課
- 年金事務所
- 倒産・解雇特例軽減 → 厚生労働省

requires_review: false
```

## GOLD-02 — 办公室搬迁要做哪些手续？

```yaml
question: 経営・管理 持有人办公室搬迁要做哪些手续？

aliases:
- 公司搬家 几天内办登记
- 经营管理 事務所変更 手续
- 法人 本店所在地変更 手续
- 公司搬迁后 入管 要不要报告
- 跨都道府県 本店移転 多少钱
- 自宅兼办公室 搬到独立办公室
- 经管 事務所搬迁 完整 checklist
- 我是经管签证 公司换新地方 要做啥

intent_keywords:
- 办公室搬迁
- 公司搬家
- 事務所変更
- 本店移転
- 経営・管理 搬迁
- 法人 住所変更

must_not_match:
- 个人搬家
- 在留カード 住所変更
- 住民票 転入届

answer_type: workflow
risk_level: medium

applies_when:
- 持 経営・管理 在留資格
- 公司本店所在地 / 事務所 即将搬迁或已搬迁
- 想知道完整手续 + 期限 + 不办的后果

does_not_apply_when:
- 个人住所搬家（→ 看 GOLD-08）
- 仅改邮寄地址（无登记变更需要）
- 公司本店地址不变 + 仅业务部门外迁（不构成本店移転）

最紧的两件:
- 法務局 本店移転登記：决议日起 2 週間内（不办 → 過料 100 万円以下）
- 出入国在留管理庁 所属機関等届出：14 日内（不办 → 続签时材料不一致）

步骤（按顺序 + 表格化）:
- |
  | 顺序 | 做什么 | 去哪办 | 期限 | 要带什么 |
  |---|---|---|---|---|
  | 1 | 取締役会 / 株主総会 决议本店所在地変更 + 作成議事録 | 公司内部 | 决议日即日 | 議事録 |
  | 2 | 申請本店移転登記 | 法務局（同管轄）/ 双方法務局（跨管轄）| 决议日起 **2 週間内** | 議事録 + 登録免許税（同 3 万円 / 跨 6 万円）|
  | 3 | 提交「所属機関等 に関する届出書」 | 出入国在留管理庁 | **14 日内** | 届出書 + 在留カード写し |
  | 4 | 提交 異動届出書 | 税務署 + 都道府県税事務所 + 市町村役場 | 1 ヶ月内 | 異動届 + 新登記事項証明書 |
  | 5 | 提交 適用事業所 所在地・名称 変更届 | 年金事務所 | 5 日内 | 変更届 + 新登記事項証明書 |
  | 6 | 提交 事業主事業所 各種変更届 | ハローワーク | 翌日から 10 日以内 | 変更届 |
  | 7 | 同步更新 銀行 / 各種許認可 / 取引先 地址 | 各机关 + 取引先 | 各自规定 | 各自要求 |

去哪办:
- 法務局（管轄 / 跨管辖时双方）— 本店所在地変更 商業登記
- 出入国在留管理庁 — 所属機関等 届出（オンライン / 郵送 / 持参）
- 税務署 + 都道府県税事務所 + 市町村役場 — 法人 異動届
- 年金事務所 — 適用事業所 変更届
- ハローワーク — 事業主事業所 各種変更届
- 銀行 / 各種許認可機関 / 取引先 — 同步更新

要带什么:
- 株主総会 / 取締役会 議事録（本店移転决议）
- 新办公室 賃貸契約書（法人名义 + 事業使用可明记）
- 新办公室 看板 / 入口 / 内装 / 平面图 照片（入管立证用）
- 旧办公室 解約証明 + 新办公室 入金 / 賃料振込記録
- 新登記事項証明書 + 印鑑証明書
- 入管 所属機関等 届出 控え
- 各机关 異動届 控え

期限:
- 法務局 本店移転登記：决议日起 2 週間以内
- 入管 所属機関等届出：14 日以内
- 年金事務所 適用事業所変更：5 日以内
- ハローワーク 事業主届：翌日から 10 日以内
- 税務 / 都道府県 / 市町村 異動届：1 ヶ月以内

不做会怎样:
- 14 日入管届出 懈怠 →「届出義務違反」记录 + 下次更新审查不利
- 2 週間 商業登記 懈怠 → 過料（100 万円以下）
- 只改邮寄地址、不办登记 → 登記簿 vs 实际住所 vs 申告住所 不一致 → 入管现地調査 / 续签立证不一致
- 跨法務局 + 期限超过 + 経営場所 立证薄弱 三件套 → 续签风险显著上升

要找专家的情况:
- 跨法務局搬迁（旧 + 新双方手续）→ 司法書士 + 行政書士
- 兼办公兼自宅 / 共享办公室 → 経営場所 立证 边界 case → 行政書士
- 経営・管理 在留期限临近（< 6 ヶ月）+ 同时搬迁 → 行政書士
- 搬迁同时涉及 役員変更 / 商号変更 / 業種拡大 → 司法書士 + 行政書士

复制给客户: |
  经营管理签的办公室搬迁不是「改个收件地址」就完事。涉及法务局 商業登記（2 週間内）+ 入管届出（14 日内）+ 税务 / 社保 異動届（1 ヶ月内）+ 银行 / 许可同步。今天先确认 3 件事：新办公室是否可用于法人办公 / 合同是否公司名义 / 是否本店所在地変更。其中法务局和入管是硬期限。如果跨法務局或经管在留快到期，可以先和 司法書士 + 行政書士 聊一下，避免登记 / 实际 / 申告地址不一致影响下次续签。

source_hint:
- 法務局（本店移転登記 商業登記）
- 出入国在留管理庁（所属機関等届出）
- 国税庁（法人異動届）
- 日本年金機構（適用事業所変更届）
- ハローワーク（事業主事業所届）

requires_review: false
```

## GOLD-03 — 永住者能不能带父母来日本养老？

```yaml
question: 永住者 / 工作签 持有人能不能把父母接来日本长居？

aliases:
- 永住者带父母来日本养老
- 永住者父母想长期住日本
- 工作签 父母 长期签证
- 家族滞在 包括父母吗
- 日本有没有像探亲签那样的长居证
- 老親扶養 ビザ 真的能办下来吗
- 我永住父母想退休来日本

intent_keywords:
- 父母 / 親
- 帯同 / 长居 / 探亲 / 养老
- 親族
- 老親扶養

must_not_match:
- 子女
- 配偶
- 短期来玩

answer_type: misconception
risk_level: high

applies_when:
- 你是永住者 / 高度専門職 / 工作签 持有者，想把父母接来长居
- 你以为「家族滞在」包括父母
- 你以为永住自动可以让父母来

does_not_apply_when:
- 父母只想短期访问 90 日以内（→ 短期滞在ビザ 中国侧申请）
- 配偶 + 子（属于「家族滞在」对象）
- 父母为日本国籍（不需要在留资格）

最紧的两件:
- 日本入管法「家族滞在」对象不含父母 — 永住者也无「自动带父母长居」资格
- 父母可来日的真实路径只有 3 条 + 都受严格条件限制

步骤:
- 1. 确认本人是否为高度専門職 1 号（仅高度人材有 親族帯同 路径）
- 2. 评估父母年龄是否 70 岁以上 + 母国是否还有其他子女 / 兄弟姐妹可扶养（决定老親扶養 可能性）
- 3. 优先走 短期滞在 90 日 + 多次入境策略
- 4. 必须长居 + 70 岁以上 + 母国无亲属 → 找有「老親扶養」实绩的行政書士面谈

去哪办:
- 短期滞在ビザ：父母在中国侧申请 → 在中国日本国大使館 / 領事館
- 高度専門職 親族帯同：日本侧 出入国在留管理庁
- 老親扶養 特定活動：日本侧 出入国在留管理庁（个別審查）

要带什么:
- 短期滞在 90 日：招聘理由書 + 滞在予定表 + 在職証明書 + 課税証明書 + 戸口簿関係立証
- 特定活動 老親扶養：医療診断書 + 介護必要性立证 + 母国家族構成証明 + 本人扶養能力立证
- 高度専門職 親族帯同：本人ポイント計算表 + 家族関係立证 + 高家庭年収立证

期限:
- 短期滞在：每次申请前 1-3 週査証発給；多次入境签可申「数次査証」最长 5 年有効
- 特定活動：個別审查，3-12 个月不等
- 高度専門職 親族帯同：与本人在留期限同步

不做会怎样:
- 硬以「家族滞在」名义申请 → 100% 不许可
- 父母短期滞在 超 90 日 → 不法滞在 + 退去強制 + 上陸拒否 5 年
- 假装结婚 / 寄宿 等绕路 → 重大违法

要找专家的情况:
- 父母年龄 70 岁以上 + 健康问题需要长期照护 → 行政書士（有老親扶養 实绩者）
- 母国无其他子女 / 兄弟姐妹可扶养 → 行政書士
- 高度専門職 + 親族帯同 申请被拒后异议 → 行政書士
- 父母持中国 因公护照 / 公務护照 涉及外交査証 → 行政書士

复制给客户: |
  日本「家族滞在」明确不包括父母 — 永住者也不能让父母自动长居。日本无中国「探亲签长居版」对应制度。父母可来日的真实路径只有 3 条：① 短期滞在 90 日（多次入境，最常见）；② 特定活動 老親扶養（极严，事实上极少下签）；③ 高度専門職 親族帯同（仅高度人材）。今天先确认本人是否高度専門職 + 父母是否真有看护需要。如果想走老親扶養路径，可以先和行政書士聊一下个别可能性。

source_hint:
- 出入国在留管理庁（在留資格一覧表）
- 出入国在留管理庁（高度専門職 親族帯同）
- 外務省（短期滞在 査証案内）
- 在中国日本国大使館（中国侧短期滞在申请）

requires_review: true
```

## GOLD-04 — 老板雇了签证不符的人我会不会受影响？

```yaml
question: 老板雇了签证不符的人（资格外活動 / 不法就労）我会不会受影响？

aliases:
- 老板雇错签证 我会被牵连吗
- 同事是黑工 我会不会受影响
- 公司雇了签证不符的人 我有问题吗
- 不法就労助長 会牵连普通员工吗
- 我不知道同事是黑工 算违法吗
- 同事 资格外活動 我会被处分吗
- 公司被入管查 我会怎样
- 雇主用错签证 我会不会被退去强制

intent_keywords:
- 老板雇错签证
- 不法就労
- 不法就労助長
- 资格外活動
- 同事黑工
- 雇主违法

must_not_match:
- 我自己资格外活動
- 我自己不法滞在
- 我自己签证错配

answer_type: misconception
risk_level: high

applies_when:
- 你持工作签 / 留学 / 家族滞在 等合法在留資格
- 你的雇主或同事 涉嫌雇用签证不符的人
- 你担心被牵连 / 被入管调查 / 续签受影响

does_not_apply_when:
- 你自己签证不符（→ 看 GOLD-99 不法滞在自首类问题）
- 你是雇主（→ 看雇主刑责类问题）
- 仅一般用工纠纷无入管成分

最紧的两件:
- 雇主刑责（不法就労助長罪）vs 雇员处分（資格外活動 → 退去强制）是 两条独立的法律线
- 你本人持合法在留資格 + 自己业务符合在留资格 → 一般不受牵连，但公司被调查时需要配合 + 续签时会被重点核查

步骤:
- 1. 确认你自己的在留資格 + 实际从事业务是否符合（持在留カード「就労制限」栏比对）
- 2. 确认你与涉事同事 / 老板的关系（是否为业务相关 / 是否经手相关合同 / 是否参与雇用决策）
```

```yaml
- 我搬到新地方在留卡要去哪改

intent_keywords:
- 搬家
- 在留卡 / 在留カード
- 住居地届出
- 14 日

must_not_match:
- 公司搬迁
- 事務所搬迁
- 経営・管理 本店移転

answer_type: workflow
risk_level: low

applies_when:
- 持有 在留カード（中长期在留者）
- 实际搬家 + 新住所在日本国内
- 想确认 14 日窗口

does_not_apply_when:
- 公司 / 事務所 搬迁（→ 看 GOLD-02）
- 海外搬家（出国 / 入国 是另一套流程）
- 短期滞在持有者（无在留カード）

最紧的两件:
- 要。搬家后 14 天内必须到新住址 区役所 / 市役所 办理在留カード 住居地届出
- 同窗口可一站办理 転入届 + 国保 / 年金 / 児童手当 / 介護保険 联动

步骤:
- 1. 旧住所 区役所 取「転出証明書」（マイナンバーカード 持有者可在マイナポータル 在线提交転出届）
- 2. 新住所 区役所 → 持转出证明 + 在留カード → 14 日内办理 転入届 + 在留カード 住居地届出
- 3. 同时联动 国保 / 年金 / 児童手当 / 介護保険 等住所更新
- 4. 銀行 / 携帯 / 電気 / 水道 / ガス / 郵便 / 各 ID 住所变更（自行）
- 5. 公司雇主 → 入管「所属機関等届出」如住所变更同时涉及就业形態变更（一般搬家不涉及）

去哪办:
- 新住所 区役所 / 市役所 戸籍住民課 + 国保年金課（同窗口或邻近窗口）
- 跨自治体搬家 → 旧住所 区役所 取「転出証明書」（持マイナンバーカード可走特例转出免书面）

要带什么:
- 在留カード
- 本人確認資料（旅券 + マイナンバー）
- 転出証明書（跨自治体搬家时；マイナ特例可豁免）
- 印鑑（部分自治体要求）
- 国保 / 年金手帳 / 児童手当 関連資料（联动办理时）

期限:
- 搬家后 14 日内（入管法 19 条の 9）
- 国保 / 年金 / 児童手当 / 介護保険 14 日内 同步更新
- マイナンバーカード 90 日内更新（不更新会失効）

不做会怎样:
- 14 日内未届出 → 入管法 19 条の 9 違反 → 罰金 ¥200,000 以下 + 续签时评价不利
- 住民票 転入未届 → 過料 ¥50,000 以下（住民基本台帳法 53 条 2 項）
- 国保未届 → 保険料 遡及徴収（最大 2 年分）
- マイナンバーカード 90 日未更新 → 失効

要找专家的情况:
- 通常自己可办，不需要专家
- 搬家 + 同时离婚 / 婚姻状态变更 → 行政書士
- 搬家 + 在留期限 < 30 日 → 行政書士
- 长期未届出（> 90 日）+ 想补办 → 行政書士

复制给客户: |
  要，搬家后 14 天内必须到新住址 区役所 / 市役所 办在留卡 住居地届出。带在留卡 + 本人確認資料（旅券 + マイナンバー），跨自治体的话同时带「転出証明書」（持マイナンバーカード可走特例転出免书面）。同窗口可顺便办 国保 / 年金 / 介護保険 住所更新。不办会被罚 ¥200,000 以下 + 续签评价不利，マイナンバーカード 90 日未更新会失效。

source_hint:
- 出入国在留管理庁（在留カード住居地届出）
- 総務省（住民基本台帳）
- マイナポータル（引越しワンストップサービス）
- 居住地 区役所 / 市役所 戸籍住民課

requires_review: false
```

## GOLD-09 — 経営・管理 资本金不够怎么办？

```yaml
question: 経営・管理 资本金不够怎么办？

aliases:
- 资本金不够 经管
- 经管 资本金 不足
- 资本金少 经管签证 还能办吗
- 资本金 没有 500 万 怎么办
- 我只有 300 万 经管 申请会不会拒
- 资本金不到 3000 万 经管 续签
- 资本金来源 怎么补
- 增资 经管签证

intent_keywords:
- 资本金 不够 / 不足
- 增资 / 増資
- 资金来源
- 借款
- 事业计划

must_not_match:
- 资本金 多少合适
- 公司休眠
- 办公室搬迁

answer_type: decision_card
risk_level: high

applies_when:
- 计划新申请経営・管理 但資本金 < 500 万円 或 < 3000 万円
- 续签 / 永住申请前 发现公司資本金不达新规
- 决算后 想増資 但资金来源不清

does_not_apply_when:
- 想问的是「資本金多少合适」（→ 看 GOLD-07 / 经管标准类问题）
- 公司休眠期 増資 问题（→ 先解决在留资格联动）
- 个人事业主（无資本金概念）

最紧的两件:
- 資本金不够时，先看你处于哪个阶段（新申请 / 续签 / 增资前 / 决算后）— 不同阶段路径不同
- 500 万円 / 3000 万円 是 不同制度门槛（旧规 500 万 / 2025/10/16 改正后 3000 万 + 经過措置至 2028/10/16）

步骤:
- 1. 确认当前阶段：新申请 / 续签 / 増資前 / 决算后
- 2. 盘点公司現状 资本金 + 純資産 + 直近 3 期决算
- 3. 评估資金来源（自己 / 亲属 / 投资人 / 借款）+ 跨境时 ODI 备案
- 4. 选择路径 A 增资（自己出资）/ B 増資（亲属·投资人）/ C 借款 + 资本金调整 / D 在留資格変更
- 5. 续签前 6-12 个月开始補足；新申请前完成増資 + 商業登記

去哪办:
- 行政書士事务所（在留 + 経営・管理 战略）
- 司法書士事务所（増資 + 商業登記）
- 税理士事务所（資金来源说明 + 法人税务联动）
- 銀行（借款 / 融資 / 法人口座）
- 法務局（商業登記 変更）
- 出入国在留管理庁（在留資格変更時）

要带什么:
- 銀行残高証明書（资本金来源）
- 事業計画書（業界 + 売上見込 + 雇用計画 + 資金繰り）
- 賃貸契約書（事務所）
- 株主名簿 + 出資証明書
- 增资時：株主総会議事録 + 募集株式発行決議書
- 跨境时：ODI 备案文件（中国侧）+ 海外送金記録 + SWIFT 単
- 现行 商業登記簿謄本

期限:
- 新申请：递申请前完成 増資 + 商業登記
- 续签：续签前 6-12 个月开始補足
- 経過措置 截止：令和 10/10/16（2028/10/16）
- 増資 商業登記：决议日起 2 週間内

不做会怎样:
- 资本金不达標硬递新申请 → 不许可 + 影响后续在留
- 借款冒充资本金 → 入管现地調査 / 资金来源问题 → 重大不利
- 跨境汇入未备案 → 中国侧外汇违规 + 日本反洗钱审查
- 拖到経過措置 最后 1 ヶ月才动手 → 増資 + 招聘 时间不足

要找专家的情况:
- 跨境资金（中国 ODI 备案 + 反洗钱 + 出資証明）→ 行政書士 + 跨境律师
- 親族出資 / 第三者出資 + 経営権 影响 → 司法書士 + 行政書士
- 已收到 補正通知 / 不許可通知 想增资补救 → 行政書士（紧急）
- 直近 3 期 連続赤字 + 净资产为负 → 税理士 + 行政書士

复制给客户: |
  経営・管理 资本金不够时，先看你在哪个阶段（新申请 / 续签 / 増資前 / 决算后）— 不同阶段对应不同路径。一般 4 条路径：① 自己增资 + 商業登記；② 亲属 / 投资人増資；③ 借款（不算資本金）；④ 在留資格変更。新规 3000 万円 + 経過措置 至 2028/10/16。今天先盘点：資金来源 + 当前資本金 + 在留期限。可以先和行政書士聊一下战略，跨境资金还要确认中国侧 ODI 备案。

source_hint:
- 出入国在留管理庁（経営・管理 ガイドライン + 改正告示）
- 法務局（商業登記 + 増資 登記）
- 国税庁（法人税务 + 資金来源说明）
- 中国国家发改委 / 商務部（中国侧 ODI 备案）
- 行政書士会 + 司法書士会 + 税理士会

requires_review: true
```

## GOLD-10 — 留学生能不能转人文签？

```yaml

aliases:
- 留学转工作签

must_not_match:

- 留学转経営・管理（起業）

answer_type: workflow
risk_level: medium

- 留学 → 技人国 不是「自动」— 必须有 雇用契約 + 业务内容符合三领域 + 大学卒以上 或 10 年実务経験

- 3. 准备「在留資格変更許可申請」+ 立证书类
- 4. 通常 1-3 个月审查；4 月入职新人最稳定的時序：3 月初递申 → 3 月底 / 4 月初 下签
- 5. 学历不够 + 拿不到符合三领域内定 → 评估 特定技能 / 経営・管理起業 / 帰国 等其他路径

去哪办:
- 居住地管轄 出入国在留管理局（在留資格変更申請）
- 拟雇主公司（雇用契約 + 業務内容説明書）
- 大学（卒業証明書 + 成績証明書）
- 行政書士事务所（路径策略 + 立证）

要带什么:
- 在留カード + 旅券
- 在留資格変更許可申請書
- 大学 卒業証明書 + 成績証明書（含専攻内容）
- 雇用契約書 / 採用通知書 / 業務内容説明書
- 拟雇主 公司決算書 + 商業登記簿謄本 + 法人税申告書
- 直近 3 个月給与明細（如已开始就业）
- 申請手数料 4,000 円

期限:
- 在留資格変更許可申請：建议毕业 / 在留期限到期前 6 个月开始
- 4 月入职的应届：3 月初递申最稳
- 在留期限失效后申请 → 不法滞在风险（不可）

不做会怎样:
- 内定后未及时变更 + 在留期限失效 → 不法滞在 + 退去强制
- 业务内容不符三领域硬递（如留学生想转「販売員」「サービス員」）→ 不许可
- 学历不达标（高专卒 / 専門卒 短期）+ 业务複雑 → 不许可率高
- 等内定确认才动手 + 时间紧迫 → 当年无法入职 + 在留期限失效风险

要找专家的情况:
- 学历是 専門学校卒（非大学卒）+ 想转技人国 → 行政書士（実务経験路径）
- 内定公司决算赤字 / 規模小 → 行政書士（公司側立证）
- 业务内容边界（看似 販売 但有 国際業務 成分）→ 行政書士

- 出入国在留管理庁（技術・人文知識・国際業務 ガイドライン）
```
