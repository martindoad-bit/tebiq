# founder_46 片山航｜健康/健身 app 订阅负责人，行为改变产品

## 1. 付费模式判断

**选择：选项 3。** 选 3。行为改变产品告诉我，用户付费给“记录 + 反馈 + 提醒”。TEBIQ 的健康形态是手续健康管理。

**¥980 的消费心理：** ¥980/月类似健康 app premium，但手续频次低，所以要强调长期档案而非每日使用。

**我做过的订阅里，触发付费的最强信号：** 最强信号是用户设置目标日期。没有目标日期就没有行为闭环。

## 2. 搜索作为主入口的判断

**判断：不同意把裸搜索框替代首页 CTA。** 健康 app 搜索弱，计划强。行政手续同理。

**如果不是搜索，应该是什么：** 首页应该问“你想处理哪件事”并要求输入日期。

## 3. 多语言节奏判断

**判断：需要修改节奏。** 中文后先不要扩。把一个语言的行为闭环做出来，再复制。日语 UI 不是替代。

**“日语 UI 给非日语用户用”是否可行：** 日语 UI 对非日语者不可行。

## 4. 拍照定位降级判断

**判断：** 拍照降级可以，但作为记录工具保留。

**“图片识别 + OCR”作为单独产品的商业价值：** OCR 没有行为闭环。

## 5. 场景驱动 vs 知识库驱动判断

**哪种更容易撑订阅：** 场景驱动更能撑订阅，因为它能产生下一步。

**我担心场景驱动的最大风险：** 风险是提醒带来压力，用户逃避。文案要冷静，不制造焦虑。

## 如果我今天接手 TEBIQ，第一个月最重要的 1 件事

第一个月我会把每个场景收敛成一个 next action，并在首页只显示 1-3 个当前动作。

## 创始人最该警惕的 1 件事

不要把复杂度展示给用户。产品内部复杂，用户只要下一步。
