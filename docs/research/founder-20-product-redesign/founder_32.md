# founder_32 陈梅｜越南留学/就职教育创业者，东京日本语学校合作

## 1. 付费模式判断

**选择：选项 2。** 选 2。学生会频繁问免费问题，付费能力低。搜索每日 3 次可接受，学校/中介批量付费更现实。

**¥980 的消费心理：** ¥980 对学生高，但对学校每名学生每月 ¥100-200 可接受。个人月付只有临近签证时可能。

**我做过的订阅里，触发付费的最强信号：** 最强信号是毕业/就职日期。TEBIQ 应捕捉 graduation_date 和 naitei_date，然后生成转签时间线。

## 2. 搜索作为主入口的判断

**判断：同意，但必须被场景化约束。** 学生喜欢搜索自然问题，但搜索词非常口语化。需要预制问法。

**如果不是搜索，应该是什么：** 支持搜索，但结果必须是学校能认可的 checklist，不是长答案。

## 3. 多语言节奏判断

**判断：拒绝这个节奏。** 越南语应该作为真正的第二 C 端语言，如果有合作学校。日语 UI 不是桥。英语对越南学生帮助小。

**“日语 UI 给非日语用户用”是否可行：** 不可行。学校通知是日语已经够难了，app 还日语会增加负担。

## 4. 拍照定位降级判断

**判断：** 拍照仍重要，学校通知、入管明信片、住民税单都靠拍。

**“图片识别 + OCR”作为单独产品的商业价值：** OCR 免费获客，学校批量管理才有收入。

## 5. 场景驱动 vs 知识库驱动判断

**哪种更容易撑订阅：** 场景驱动更适合学校，因为老师要看学生进度。

**我担心场景驱动的最大风险：** 风险是学生把结果当绝对保证。必须保守。

## 如果我今天接手 TEBIQ，第一个月最重要的 1 件事

第一个月我会做“毕业到就职签证变更”越南语试点，给 1 所日本语学校 50 个学生用。

## 创始人最该警惕的 1 件事

学生个人订阅不是好生意，学校/机构才是。
