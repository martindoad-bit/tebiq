# founder_45 高山奈绪｜前 note/付费内容产品经理，创作者订阅与资料包

## 1. 付费模式判断

**选择：选项 3。** 选 3。内容本身很难月订阅，除非用户持续生产自己的资料。TEBIQ 的“我的场景”比 165 篇知识库更像订阅资产。

**¥980 的消费心理：** ¥980/月对内容偏高，对个人档案工具可接受。若只是看文章，用户预期 ¥300-500 一次性。

**我做过的订阅里，触发付费的最强信号：** 最强信号是用户从阅读转为填写自己的信息。TEBIQ 应在每篇场景内容中嵌入 1 个字段，例如日期/在留资格/机构。

## 2. 搜索作为主入口的判断

**判断：不同意把裸搜索框替代首页 CTA。** 内容平台搜索使用率中等，但付费来自作者/主题信任，不来自搜索次数。

**如果不是搜索，应该是什么：** 搜索可以找内容，但首页应该卖“把这事保存成自己的清单”。

## 3. 多语言节奏判断

**判断：需要修改节奏。** 中文先。日语 UI 可用于 source citation；英语/越南语要有本地内容编辑，不是翻译。

**“日语 UI 给非日语用户用”是否可行：** 日语 UI 不能服务中文内容信任。

## 4. 拍照定位降级判断

**判断：** 拍照降级没问题。拍照是把内容变成我的资料的一种方式。

**“图片识别 + OCR”作为单独产品的商业价值：** OCR 单独免费化。

## 5. 场景驱动 vs 知识库驱动判断

**哪种更容易撑订阅：** 场景驱动更能订阅。知识库适合 SEO 和免费获客。

**我担心场景驱动的最大风险：** 风险是场景内容过度模板化，失去可信深度。

## 如果我今天接手 TEBIQ，第一个月最重要的 1 件事

第一个月我会把 20 个场景页写成“内容 + 个人字段 + 保存状态”的混合体，文章段落不超过 6 屏。

## 创始人最该警惕的 1 件事

不要把知识库完全隐藏。信任来自看得到的来源和解释。
