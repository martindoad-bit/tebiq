# TEBIQ 1.0 上线就绪诊断：客户50打分

评估日期：2026-04-29  
线上路径：`https://tebiq.jp` → `/photo` → `/check` → `技術・人文知識・国際業務` → 单维度 → `/timeline` → `/pricing`  
画像来源：`docs/research/round-3/customer-50.md`。备注：任务提到的 `docs/research/customer-50-perspectives.md` 在当前仓库不存在，本轮以 round-3 的 50 画像表复用。  
实测截图：`docs/research/round-3-launch/screenshots/`

## 线上实测共同事实

1. 首页首屏同时出现两个大 CTA（`拍一份文书试试` / `做一次续签自查`）和常用工具列表里的同名入口（`拍照即懂` / `续签自查` / `我的提醒`）。新用户会把它看成重复菜单，而不是主次结构。
2. `/photo` 能说明每天剩 1 次、支持图片/PDF/截图，但没有可点开的样例结果。用户还没上传前看不到“识别后到底长什么样”。
3. `/check` 已直接进入 `技術・人文知識・国際業務` 维度清单，不再是“先选签证类型”。横向签证切换存在，但用户未必知道自己已经被默认选中。
4. 从维度清单点击 `在留期限` 等单项，线上实际进入 404：`这一页迷路了`。这使“单维度展开”路径断掉。
5. `/timeline` 未登录空态显示 0 件、0 个月、0 类机构、暂无匹配记录；它像空数据库，不像能带来安心的提醒工具。
6. `/pricing` 展示 ¥980/月、¥8,800/年、免费每天 1 次和 7 天试用，但没有自动续费、取消、退款、数据保存/删除等付费前关键说明。

## 50人共同提到的 Top 5 问题

1. **单维度展开 404，是上线前硬断点**  
   路径：`/check` → 选择/默认签证 → 点击任一维度。  
   用户反应：续签自查失信，尤其是技人国跳槽、配偶者、永住准备用户会直接退出。

2. **首页入口重复，主任务不够锋利**  
   路径：首页首屏。  
   用户反应：不知道该点大按钮还是列表行；“拍照”和“自查”不像主流程，像重复菜单。

3. **拍照页没有真实识别样例，价值落不到“金额/期限/下一步”**  
   路径：`/photo`。  
   用户反应：愿意免费试一次，但付费前看不到结果质量，不敢上传敏感文件。

4. **自查语言仍像“风险判断”，不是“材料准备检查”**  
   路径：首页 `续签自查`、`/check` 维度清单。  
   用户反应：复杂身份/家庭/经管用户担心被误导；蓝领和学生又找不到自己的签证类型。

5. **¥980 的信任前提没出现：隐私、取消、退款、数据保存**  
   路径：`/pricing`。  
   用户反应：收入低者嫌贵，高收入者嫌不够可信；双方都不因当前会员页立刻付费。

## 客户侧上线判断

当前客户视角不是“没人需要”，而是“需要非常真，但不敢交给当前版本”。  
上线前最该先修的客户路径是：

1. `/check` 单维度点击必须可用，或先移除单维度可点击状态。
2. 首页只保留一个主入口层级：主 CTA 负责行动，常用工具改成最近/示例/状态，不再重复同名入口。
3. `/photo` 增加一个可查看的脱敏样例结果，直接展示“文书类型、金额、期限、下一步、提醒如何进入时间线”。
