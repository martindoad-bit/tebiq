## 用户場景

「帰化申請したいのですが、どのような書類が必要ですか？」  
「帰化申請の審査期間はどのくらいかかりますか？」  
「帰化申請が不許可になりました。再申請できますか？」

---

## 核心事実

**帰化申請は法務局（都道府県法務局・地方法務局）への申請。書類準備が複雑で在日期間によって必要書類が異なる。審査期間は約1〜1.5年（面接・調査含む）。不許可理由は通知されないが、再申請は可能。**

**帰化申請の基本書類（標準的なケース）:**

| 書類 | 内容 |
|------|------|
| 帰化許可申請書 | 法務局所定の書式 |
| 親族の概要を記載した書面 | 家族構成・本国の親族情報 |
| 履歴書 | 生年月日から現在までの職歴・住所歴 |
| 宣誓書 | 日本の法律を守る旨の宣誓 |
| 在留カード・パスポート（写し） | 身分証明 |
| 住民票（本人+世帯全員） | 住所確認 |
| 戸籍謄本（または本国の出生証明等） | 本国の身分関係証明 |
| 課税証明・納税証明（3〜5年分） | 納税状況の確認 |
| 給与明細・源泉徴収票 | 生活能力の確認 |
| 在勤・在職証明書 | 勤務先の確認 |
| 日本語能力の疎明資料 | 作文・面接で確認 |
| 帰化動機書 | なぜ帰化したいか |

**国籍別で追加書類が必要なケース:**

- 中国・韓国・台湾出身: 本国の戸籍関係書類（公証書・翻訳含む）
- ブラジル・ペルー等: 出生証明書・家族証明書（Cartão do CPF等）
- 本国書類には公証・アポスティーユ・日本語翻訳が必要

**帰化申請の審査フロー:**

1. 法務局への事前相談（書類の確認・必要書類リストを受取）
2. 書類収集・作成（数か月かかる場合あり）
3. 申請書提出（法務局窓口）
4. 担当官による調査・面接（1〜3回）
5. 法務大臣による審査・許可
6. 官報告示・在留カード返納・市区町村での日本人としての手続き

**審査期間の現実:**

- 一般的な審査期間: 申請から約12〜18か月。
- 複雑なケース（犯罪歴・複数国籍・長期海外在住等）: 2年以上かかることもある。

**不許可の場合:**

- 法務省から不許可の通知が来るが、不許可の「理由」は通知されない（法令上の規定なし）。
- 再申請: 期間制限なし（通常は問題となった事項を改善してから再申請）。
- 不許可理由の推測: 素行問題・生活能力の不足・書類の不備・日本語能力が基準未達等。

## 官方依据

| 法令・出所 | 条文 | 要旨 |
|-----------|------|------|
| 国籍法第4条〜第10条 | 帰化の要件 | 居住・能力・素行・生計等 |
| 国籍法第11条 | 帰化による国籍喪失 | 外国国籍の喪失 |
| 法務省「帰化申請」 | https://www.moj.go.jp/MINJI/minji78.html | 帰化申請手続き概要 |

---

## 実務来源

- 行政書士実務: 帰化申請の書類準備は標準で3〜6か月かかる。法務局によって要求する書類の量・形式が異なることがある（担当官の裁量が大きい）。事前相談を丁寧に行い、必要書類リストを完全に把握してから収集を開始するのが最善。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 不許可理由の通知 | 法令上の義務なし | 理由が分からず対処できないケースが多い |
| 審査期間 | 法令上の定めなし | 実態として12〜18か月が標準 |
| 書類の要件 | 法務省の標準書類リスト | 担当法務局によって追加書類を要求されることがある |

---

## 可注入答案的安全表述

> 「帰化申請は都道府県の法務局・地方法務局が窓口です。主な必要書類は申請書・戸籍関係書類・納税証明・職歴証明・帰化動機書等で、国籍によって追加書類が必要になります。
> 審査期間は通常**約1〜1.5年**かかります。
> 不許可になった場合でも、**再申請に期間制限はありません**。不許可理由は通知されませんが、問題点を改善してから再申請できます。
> 書類準備が複雑なため、行政書士への相談をお勧めします。法務局での事前相談も無料で行えます。」

---

## 材料連接

- 帰化許可申請書（法務局配布）
- 本国書類一式（出生証明・戸籍・公証+翻訳）
- 課税証明・納税証明（直近3〜5年）
- 勤務先証明・給与関係書類

---

## L5/深水信号

- 「帰化申請中に外国国籍の更新をしたいのですが問題ありますか？」→ 申請中に外国籍の更新・再取得は帰化意思の問題として確認が必要。法務局に事前相談。
- 「帰化後に子どもが生まれました。子どもは日本人になりますか？」→ 帰化後の子は日本人（血統主義）。子どもの国籍は親が日本人なら日本国籍取得。
