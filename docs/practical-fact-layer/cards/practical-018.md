## 用户場景

「在留期間更新の申請書類を準備したいのですが、何が必要ですか？」  
「技人国の更新に必要な書類を教えてください。」  
「経営管理の更新に必要な書類のリストをください。」

---

## 核心事実

**就労系在留資格の更新申請書類は「申請者共通書類」+「カテゴリ別書類（上場企業・中小・個人）」で構成される。ISAの書類チェックリストが公式に存在する。**

**書類準備の基本構造（ISAのカテゴリ制度）:**

ISAは審査簡略化のため、雇用企業を4つのカテゴリに分類している:

| カテゴリ | 対象 | 書類の簡略度 |
|---------|------|------------|
| カテゴリ1 | 上場企業・国・地方公共団体・独立行政法人等 | 最も少ない |
| カテゴリ2 | 前年分の給与所得源泉徴収票合計が1,500万円以上の企業 | 少ない |
| カテゴリ3 | 中小企業（カテゴリ1・2以外） | 標準 |
| カテゴリ4 | 設立1年未満・カテゴリ1〜3に該当しない | 最も多い |

## A. 技術・人文知識・国際業務（技人国）更新申請書類

### 申請者共通書類

| 書類 | 備考 |
|------|------|
| 在留期間更新許可申請書 | ISA所定様式（書式ダウンロード可） |
| 写真（縦4cm×横3cm） | 3か月以内に撮影、白背景 |
| パスポート | 原本（提示） |
| 在留カード | 原本（提示） |

### カテゴリ1・2の追加書類

| 書類 | 備考 |
|------|------|
| カテゴリ確認書類 | 会社の四季報・上場証明・源泉徴収票合計額証明等 |

### カテゴリ3・4の追加書類（最も多い）

| 書類 | 備考 |
|------|------|
| 源泉徴収票（前年分） | または課税証明書 |
| 雇用契約書（写し） | または労働条件通知書 |
| 会社の登記事項証明書 | 直近3か月以内に取得 |
| 会社の決算書（直近年度） | または確定申告書（法人） |
| 会社の概要書 | パンフレット・事業内容説明書 |
| 業務内容説明書 | 申請者が担当する業務の詳細 |
| 在職証明書 | 会社が発行（雇用継続中の場合） |

### 新規採用（転職直後）の場合（追加）

| 書類 | 備考 |
|------|------|
| 採用理由書 | 学歴・職歴と業務内容の関連性 |
| 採用先の財務書類 | 決算書・法人税申告書 |

## B. 経営管理 更新申請書類

### 共通書類（申請者）

| 書類 | 備考 |
|------|------|
| 在留期間更新許可申請書 | ISA所定様式 |
| 写真 | 縦4cm×横3cm |
| パスポート・在留カード | 原本提示 |

### 法人関係書類

| 書類 | 備考 |
|------|------|
| 登記事項証明書 | 直近3か月以内 |
| 定款 | 写し |
| 決算書（直近2〜3期） | 損益計算書・貸借対照表 |
| 法人税申告書（直近） | 法人税の確定申告書 |
| 役員報酬が確認できる書類 | 役員報酬月額・源泉徴収票等 |

### 事業所関係書類

| 書類 | 備考 |
|------|------|
| 事務所の賃貸契約書 | 写し |
| 事務所内部の写真 | デスク・設備等 |
| 事業実態証明書類 | 請求書・領収書・契約書等 |

### 赤字・業績不振の場合（追加）

| 書類 | 備考 |
|------|------|
| 理由書 | 赤字の理由・今後の事業見通し |
| 事業計画書 | 具体的な数値入り |
| 受注見込みを示す書面 | LOI・見積書・顧客との往来メール |

## 実務来源

- 行政書士実務: ISA公式の書類リストは最低限。実際にはカテゴリ・業種・個別事情によって追加書類が求められる。「書類が少ない=審査が簡単」ではなく、追加書類要求（補正）が来ることもある。
- ISA窓口実態: 書類が不完全でも「補正通知」で対応可能な場合と、窓口で受け付けてもらえない場合がある。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 書類リスト | ISA公式リストが「標準」 | 実務では個別事情・審査官の判断で追加書類が求められる |
| カテゴリ制度 | 上場企業は書類が少ない | 実務でも上場企業は審査が比較的スムーズ |

---

## 可注入答案的安全表述

> 「技人国・経営管理の更新申請に必要な書類は、雇用先の規模（ISAのカテゴリ1〜4）によって異なります。
> 中小企業（カテゴリ3）の場合は、雇用契約書・源泉徴収票・会社の登記事項証明書・決算書・業務内容説明書等が標準的に必要です。
> ISAの公式サイトに書類リストが掲載されていますが、個別の状況（転職直後・赤字等）によって追加書類が必要になることがあります。
> 具体的な書類準備は行政書士への相談をお勧めします。」
