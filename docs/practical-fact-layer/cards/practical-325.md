## 用者場景

「特定技能1号で働いています。妻と子どもを日本に呼び寄せられますか？」  
「特定技能2号になれば家族を呼べると聞きました。本当ですか？どんな条件ですか？」  
「特定技能2号の対象分野に自分が働く分野が含まれているかどうか確認したいです。」

---

## 核心事実

**特定技能1号（5年）は原則として家族帯同不可。特定技能2号（無期限・更新制）は家族帯同が認められる（配偶者・子どもを「家族滞在」として帯同可能）。2024年の制度改正で特定技能2号の対象分野が拡大された（従来の建設・造船舶用工業機械器具製造業の2分野から11分野に拡大）。ただし特定技能2号への移行には「特定技能2号評価試験」（分野別）または「高度熟練者認定」が必要。**

**特定技能1号・2号の家族帯同比較:**

| 項目 | 特定技能1号 | 特定技能2号 |
|------|-----------|-----------|
| 家族帯同 | **不可**（原則）| **可**（配偶者・子→「家族滞在」）|
| 在留期間 | 最大5年（更新不可・一回限り）| 無期限（更新制）|
| 在留カード | 特定技能1号 | 特定技能2号 |
| 移行要件 | 技能試験・日本語試験 | 特定技能2号評価試験（分野別）|

**特定技能2号の対象分野（2024年改正後）:**

| 分野 |
|------|
| 建設 |
| 造船・舶用工業 |
| 自動車整備 |
| 航空 |
| 宿泊 |
| 介護 |
| ビルクリーニング |
| 素形材・産業機械・電気電子情報関連製造業 |
| 農業 |
| 漁業 |
| 飲食料品製造業 |
| 外食業 |

※ 2号の対象分野・要件は制度変更があるため出入国在留管理庁の最新情報を確認。

**特定技能2号への移行の実務:**

- **分野別技能試験（特定技能2号評価試験）**に合格すること。
- 一定期間の実務経験（3年以上が目安）と高度な技能水準の証明。
- 監理団体・登録支援機関のサポートが有効。

**特定技能1号保持者の家族（配偶者・子）の日本滞在:**

- 特定技能1号では家族帯同は認められていない。
- 家族が「短期滞在（観光・90日以内）」で来日して面会することは可能。
- 家族が別の在留資格（技人国・留学等）で在留することは可能（「特定技能1号の家族として」ではなく、独自の在留資格で）。

**特定技能2号での家族帯同手続き:**

- 配偶者・子どもが「家族滞在」で在留資格認定証明書交付申請を行う（特定技能2号保持者が扶養者として認定される）。
- 配偶者の就労: 家族滞在では週28時間の資格外活動許可が必要（就労制限あり）。

| 法令・出所 | 条文 | 要旨 |
|-----------|------|------|
| 入管法別表第1「特定技能」| 活動範囲 | 特定技能1号・2号の活動・家族帯同 |
| 出入国在留管理庁「特定技能2号」ガイド | 制度詳細 | 2号対象分野・移行要件・家族帯同 |

---

## 実務来源

- 行政書士実務: 「特定技能1号で妻を呼びたい」という相談が非常に多い。1号では不可と明確に説明し、2号への移行が家族帯同の唯一の公式ルートであることを伝える。2号移行要件（試験・実務経験）はハードルが高く、現実的には5年の1号期間中は単身での就労が続くケースが多い。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 特定技能1号の家族帯同不可 | 制度上不可 | 単身就労が長期間続くことへの外国人労働者の不満・精神的負担が大きく、制度の改善を求める声がある |

---

## 可注入答案的安全表述

> 「**特定技能1号**（最大5年）は原則**家族帯同不可**です。配偶者・子どもを呼び寄せることはできません。
> **特定技能2号**（無期限更新制）は**家族帯同可**（配偶者・子どもを「家族滞在」として帯同）。
> 2号への移行には**分野別技能試験の合格**と一定の実務経験が必要です。
> 2号対象分野は2024年改正で**12分野に拡大**されました。
> 最新の対象分野・要件は出入国在留管理庁の公式情報を確認してください。」

## 材料連接

- 特定技能2号評価試験合格証明書（2号移行申請時）
- 家族滞在申請書類（配偶者・子どもの在留資格認定証明書交付申請）
- 扶養能力証明（収入証明・住民税課税証明等）

## L5/深水信号

- 「インドネシア国籍・介護分野の特定技能1号で2年目。妻と5歳の子どもをインドネシアに残しており、日本に呼び寄せたい。特定技能2号への移行はいつできますか？」→ 特定技能1号期間中は家族帯同不可。介護分野は2024年改正で特定技能2号の対象分野に含まれた（確認要）。2号移行には介護分野の特定技能2号評価試験合格・実務経験（目安3年以上）が必要。現時点（2年目）での2号移行は難しいが、2〜3年後に評価試験の受験準備を始めることを推奨。監理団体・登録支援機関に2号移行のサポートを相談。2号取得後に妻と子を「家族滞在」として呼び寄せる（妻の就労は資格外活動許可で週28時間まで）。行政書士に移行計画を相談。
