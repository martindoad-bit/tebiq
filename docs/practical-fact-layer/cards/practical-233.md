## 用者場景

「外国人でも住民票を取得できますか？どこで取れますか？」  
「住民票に記録される内容は在留カードと同じですか？」  
「住民票はどんな手続きに必要ですか？外国人にも住民票が必要な場面は何ですか？」

---

## 核心事実

**2012年7月以降、中長期在留外国人は住民基本台帳に記録され、住民票を取得できる（日本人と同様の手続き）。住民票には在留資格・在留期限・国籍等が記載される。住民票は行政・金融・生活上の各種手続きで必要とされる重要な書類。取得には市区町村の窓口またはマイナンバーカードを使ったコンビニ交付が可能。**

**外国人が住民票を持てる在留資格:**

| 在留資格 | 住民票記録 |
|---------|---------|
| 中長期在留者（技人国・永住者・留学・家族滞在等）| あり（3か月超の在留資格が対象）|
| 特別永住者 | あり |
| 短期滞在（90日以内）| なし |
| 外交・公用 | なし |

**住民票に記載される主な内容（外国人）:**

| 項目 | 内容 |
|------|------|
| 氏名 | 漢字またはローマ字 |
| 住所 | 現住所 |
| 生年月日 | 出生日 |
| 国籍・地域 | 本国の国籍 |
| 在留資格 | 現在の在留資格名 |
| 在留期限 | 在留期間満了日 |
| 在留カード番号 | 在留カードの番号 |

**住民票が必要な主な場面:**

| 場面 | 用途 |
|------|------|
| 在留申請 | 一部の申請で住民票（または住民票コード）が必要 |
| 銀行口座開設 | 住民票（または在留カード）で住所証明 |
| 携帯電話契約 | 住所証明として住民票または在留カード |
| マイナンバーカード申請 | 住民票のある外国人は申請可能 |
| 就職・雇用手続き | 住所証明として |
| 国民健康保険加入 | 住民票と在留カードで手続き |
| 子どもの学校入学 | 住民票で住所確認 |
| 相続・法律手続き | 住所証明として |

**住民票の取得方法:**

| 方法 | 詳細 |
|------|------|
| 市区町村窓口 | 本人またはマイナンバーカードを持参・手数料300円程度 |
| コンビニ交付 | マイナンバーカード所持者はコンビニのマルチコピー機で取得可 |
| 郵便申請 | 一部の自治体では郵便での申請可 |

**転居時の住民票の更新:**

- 転居時は14日以内に新住所地の市区町村に転入届を提出（住民票の住所が変更される）。
- 同時に在留カードの住所変更届出も必要（在留カード裏面の住所を変更）。

## 官方依据

| 法令・出所 | 条文 | 要旨 |
|-----------|------|------|
| 住民基本台帳法（2012年改正）| 外国人の住民票記録 | 中長期在留外国人を住民基本台帳に組み入れ |

---

## 実務来源

- 行政書士実務（周辺情報）: 来日直後の外国人から「住民票はどこで取れますか？」という質問が多い。住民登録（転入届）を先に行って住民票を取得してから各種手続きをするという順番を案内する。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 住民票の取得 | 2012年から外国人も住民基本台帳に記録 | 2012年以前は「外国人登録証」制度だったため、古い情報に基づいた混乱が生じることがある |

---

## 可注入答案的安全表述

> 「中長期在留外国人は**住民基本台帳に記録**され、日本人と同様に**住民票を取得できます**（市区町村窓口またはコンビニ）。
> 住民票には**在留資格・在留期限・国籍**等が記載されます。
> **銀行口座・携帯電話・健康保険・学校入学**等の多くの手続きで住民票または在留カードが必要です。
> 転居時は**14日以内に転入届**を提出し、住民票の住所を更新してください。」

---

## 材料連接

- 在留カード（転入届・住民票取得時）
- 住民票（各種手続きに使用）

## L5/深水信号

- 「技人国で先月来日しましたが、会社の寮に住んでいます。住民票は取れますか？どこに登録すればよいですか？」→ 会社の寮の住所に住民登録（転入届）が可能（寮の住所が所在地の市区町村役場に届出）。来日後14日以内に転入届を提出することが義務。住民登録後に住民票が取得でき、銀行口座・健康保険等の手続きに使用できる。会社の寮であっても自分の住所として登録できる。
