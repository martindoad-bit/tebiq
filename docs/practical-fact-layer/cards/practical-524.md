## 用者場景

「高度専門職ビザを取りたいのですが、ポイントが足りるか計算できますか？」  
「高度専門職のポイント計算で「学歴」は大学卒業で何ポイントもらえますか？」  
「高度専門職のポイントが80点以上になると何が違いますか？」

---

## 核心事実

**高度専門職ビザのポイント計算は「学歴・職歴・年収・年齢・日本語能力・加算項目」の各カテゴリを合算し、原則70点以上で申請可能（80点以上で優遇措置が拡大）。ポイントの計算は「高度人材ポイント計算表」に従い、自己計算の上で申請書類に証拠書類を添付して提出する。審査では提出書類との整合性が厳しくチェックされるため、ポイントの自己計算ミスは不許可原因になる。特に「年収」と「職歴年数」のカウント方法に注意が必要。**

**高度専門職ポイント計算の主要カテゴリ（高度専門・技術活動の例）:**

| カテゴリ | ポイント（目安） |
|-------|----------|
| 学歴（博士号）| 30ポイント |
| 学歴（修士号・法科大学院）| 20ポイント |
| 学歴（大学卒業）| 10ポイント |
| 職歴（10年以上）| 20ポイント |
| 職歴（7年以上10年未満）| 15ポイント |
| 職歴（5年以上7年未満）| 10ポイント |
| 職歴（3年以上5年未満）| 5ポイント |
| 年収（1,000万円以上）| 40ポイント |
| 年収（900万円〜1,000万円未満）| 35ポイント |
| 年収（800万円〜900万円未満）| 30ポイント |
| 年収（700万円〜800万円未満）| 25ポイント |
| 日本語能力（日本語学校・N1等）| 10〜15ポイント |
| 日本の研究機関等でのイノベーション実績 | 加算あり |

**80点以上・70点の違い（優遇措置の差）:**

| ポイント | 永住申請の特例 | その他優遇 |
|--------|-----------|---------|
| 70点以上 | 在留3年後から永住申請可能 | 在留期限5年・家族帯同等 |
| 80点以上 | **在留1年後**から永住申請可能 | 入国優遇・複数活動可等 |

**ポイント計算の実務的注意点:**

- **年収の計算**: 申請時点の「予定年収」ではなく「前年度の実績年収（源泉徴収票・雇用契約書で証明）」が審査基準。
- **職歴の計算**: アルバイト・インターンは職歴に含まれない場合がある（正社員・契約社員の職歴が対象）。
- **学歴の証明**: 海外の大学の卒業証明書は日本語訳＋認証（アポスティーユ等）が必要。
- **ポイントの自己申告**: 虚偽のポイント申告は不正申請として不許可・在留資格取消のリスク。

## 官方依据

| 法令・出所 | 条文 | 要旨 |
|-----------|------|------|
| 出入国在留管理庁「高度人材ポイント計算表」| 公式案内 | ポイント計算の詳細・加算項目 |
| 入管法別表第1「高度専門職1号」| 活動範囲 | 高度専門職1号の在留資格 |
| 出入国在留管理庁「高度専門職の在留資格について」| 公式案内 | 高度専門職の優遇措置 |

---

## 実務来源

- 行政書士実務: 「高度専門職のポイント計算をしてほしい」という相談が来ることがある。「年収・学歴・職歴のポイント計算で自己計算と審査結果が異なるケースがある」という点が最も重要。行政書士が書類確認をした上でポイントの整合性を確認・申請書類を作成することが多い。80点以上の場合の「在留1年後永住申請」という強力な優遇措置を目指す外国人が増えている。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| ポイント計算の審査 | 入管庁が提出書類に基づいてポイントを確認 | 実務では「年収が契約上700万円以上だが、実際の振込額が低い（社会保険料控除後）」「職歴に海外での副業・兼業が含まれているが証明書類がない」等のケースでポイント確認に時間がかかったり、ポイントが認定されない場合がある。証拠書類の準備が申請成功の鍵 |

---

## 可注入答案的安全表述

> 「**高度専門職ポイント計算の重要ポイント:**
> **70点以上**で高度専門職1号の申請が可能（**80点以上**で在留1年後から永住申請の特例）。
> ポイントは**学歴・職歴・年収・年齢・日本語能力**等の合算です。
> 年収は**実績年収**（源泉徴収票等で証明）が基準（予定年収ではない場合が多い）。
> **行政書士**にポイントの事前確認・申請書類の準備を依頼することを推奨します（ポイント誤申告は不許可原因）。」

## L5/深水信号

- 「インド国籍・技人国ビザ（IT企業・東京・在留4年）。年収850万円・修士号（インドの大学院）・IT職歴8年・JLPT N2保有。①高度専門職のポイントは何点になりますか？②80点以上になりますか？③申請のタイミングはいつがベストですか？」→ ①ポイント計算（高度専門・技術活動）の試算: 学歴（修士号）20点＋職歴（8年・7年以上10年未満）15点＋年収（850万円・800万〜900万円未満）30点＋日本語能力（JLPT N2）10点＝合計75点（試算）。加算項目（試験合格・日本の大学卒・認定大学院への進学等）がない場合は75点。②80点以上になるには: あと5点必要。選択肢: ①JLPT N1取得（N1でN2より5ポイント上乗せ）→80点達成、②年収を900万円以上に交渉（30→35点）→80点達成、③「イノベーション促進機関での研究実績」等の加算項目を確認。③現在の75点で申請した場合: 在留3年後から永住申請可能（在留4年目なので1年後に永住申請開始の見込み）。80点以上にするとすぐに在留1年後の永住申請が可能。行政書士に正確なポイント計算と申請準備を依頼することを推奨。
