## 用户場景

「行政書士に頼まないと在留資格の申請はできませんか？」  
「行政書士に依頼する費用はどのくらいかかりますか？」  
「どんな場合に行政書士に相談すべきですか？」

---

## 核心事実

**在留資格の申請は本人でも行えるが、複雑なケースや高リスク案件では行政書士への依頼が推奨される。費用相場は申請の種類・複雑さで異なる。**

**在留資格申請は「本人申請」が可能:**

- 在留資格の申請はISA窓口（または在留申請オンラインシステム）で本人が申請できる。
- 行政書士への依頼は法律上義務ではない。
- 単純なケース（同じ会社での更新・在留歴が安定等）は本人申請でも問題ない。

**行政書士に依頼すべきケース（推奨）:**

| ケース | 理由 |
|--------|------|
| 不許可になった過去がある | 申請戦略・書類の再構築が必要 |
| 転職直後・複雑な業務経歴 | 業務適合性の説明書の質が重要 |
| 経営管理の新規申請 | 事務所・資本金要件の準備が複雑 |
| 配偶者ビザで偽装婚を疑われている | 婚姻実態の証明戦略が重要 |
| 永住申請 | 要件確認・書類量が多い |
| 在留資格取消通知を受けた | 緊急対応（弁護士と連携して） |
| COEから始める複雑な新規申請 | 全体プロセスの管理が必要 |
| 素行上の問題がある（逮捕歴・交通違反等） | 審査官の心証対策が必要 |

**費用相場（一般的な目安・市場価格）:**

| 申請の種類 | 相場（税込み目安） |
|-----------|-----------------|
| 在留期間更新（単純ケース） | 3〜5万円 |
| 在留期間更新（複雑・転職等） | 5〜8万円 |
| 在留資格変更（技人国等） | 5〜10万円 |
| 経営管理の新規申請 | 15〜30万円 |
| 永住申請 | 10〜20万円 |
| 帰化申請 | 10〜20万円 |
| 配偶者ビザ（COE含む） | 5〜15万円 |
| 不許可後の再申請 | 15〜30万円（難易度による） |

> ⚠️ 上記は目安であり、事務所・地域・申請の複雑さによって大きく異なる。

**行政書士の探し方:**

- 「入国管理業務」「在留資格申請」を専門とする行政書士事務所を選ぶ。
- 日本行政書士会連合会の「全国行政書士会員検索」で検索可能。
- 「申請取次行政書士」の資格（ISAが認定する申請代行資格）を持つ者を選ぶ。

**「申請取次行政書士」の特徴:**

- ISAが認定する資格。
- 申請者本人の代わりにISA窓口で申請書類を提出できる（本人の窓口出頭不要）。
- 在留カードの受取代行も可能。

**格安・無認可業者への注意:**

- 行政書士の資格がない者が報酬を受けて申請代行を行うことは違法（行政書士法違反）。
- 「格安ビザ申請」「フリーランス申請代行」等の広告に注意。無資格者への依頼は書類の不備・偽造のリスクがある。

## 実務来源

- 行政書士実務: 費用相場は地域・事務所・複雑さで大きく変動。一律の「相場」はないが、上記は市場で多く見られる範囲。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 本人申請の可否 | 法令上可能 | 複雑なケースでは実質的に行政書士なしでの対応が困難 |
| 費用 | 法令上の規定なし | 市場によって大きく異なる |

---

## 可注入答案的安全表述

> 「在留資格の申請は本人で行うことができます。ただし、複雑なケース（転職直後・経営管理・永住・不許可後の再申請等）は、**入管業務を専門とする行政書士（申請取次行政書士）**への相談・依頼をお勧めします。
> 費用の目安は申請の種類によって異なります（更新: 3〜8万円、経営管理新規: 15〜30万円、永住: 10〜20万円程度が一般的）。
> 行政書士を選ぶ際は、「入管業務専門」「申請取次行政書士」の資格を確認してください。
> 資格のない業者による申請代行は違法ですのでご注意ください。」
