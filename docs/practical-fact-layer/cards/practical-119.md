## 用户場景

「海外にいる配偶者を日本に呼び寄せるにはどうすればよいですか？」  
「「日本人の配偶者等」のCOE（在留資格認定証明書）申請の書類を教えてください。」  
「COE申請から配偶者が来日するまでにどのくらい時間がかかりますか？」

---

## 核心事実

**海外配偶者の呼び寄せには「在留資格認定証明書（COE）」をまず取得し、その後配偶者が現地の日本大使館・領事館でビザ申請を行う。COEの審査は婚姻の真正性・呼び寄せ側の生活力・渡航歴が重視される。審査期間は標準1〜3か月。**

**呼び寄せの全体フロー:**

1. 在留申請者（日本在住のスポンサー）がISAにCOE申請。
2. ISAによる審査（標準：1〜3か月）。
3. COE発行 → 海外の配偶者に郵送。
4. 配偶者が現地日本大使館・領事館でビザ申請（COE提出）。
5. ビザ発給 → 来日 → 在留カード取得。

**COE申請に必要な書類（「日本人の配偶者等」の場合）:**

| 書類 | 提出者/内容 |
|------|-----------|
| 在留資格認定証明書交付申請書 | ISA書式 |
| 写真（4cm×3cm）| 申請者（海外配偶者）の顔写真 |
| 返信用封筒（簡易書留）| COE受取用 |
| 配偶者（日本人）の戸籍謄本 | 婚姻関係の証明 |
| 配偶者（日本人）の住民票 | 日本での居住確認 |
| 婚姻証明書（外国の婚姻届等）| 現地の婚姻が証明された書類+翻訳 |
| 夫婦の交際・婚姻を証明する写真 | 一緒に撮った写真（複数枚） |
| 配偶者（日本人）の在職証明書+源泉徴収票 | 生活力の証明 |
| 質問書（結婚の経緯）| ISA書式（出会い・交際・決婚の経緯）|

**審査でよく問われるポイント（真正性の確認）:**

- 出会いの経緯: マッチングアプリ・SNS等は詳しい説明が必要。
- 交際期間: 短期間の結婚は追加の説明書類が求められる。
- 言語の共通性: コミュニケーション手段（共通言語・翻訳ツールの使用）。
- 年齢差: 大きい年齢差がある場合は説明書類が必要。
- 会面歴: 実際に会ったことがある写真・渡航記録（パスポートスタンプ等）。

**COE審査から来日までの標準期間:**

- COE審査: 1〜3か月（混雑時は3〜5か月）。
- ビザ発給: 1〜2週間（各国の日本大使館による）。
- 合計: 申請から来日まで2〜6か月が目安。

---

## 実務来源

- 行政書士実務: COE申請で最も問われるのは「婚姻の真正性」。短期間・SNS出会い・大きな年齢差・渡航歴なしの組合せが重なると審査が厳しくなる。写真・ビデオ通話記録・渡航記録の証拠収集が事前準備として最重要。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| COE申請の書類要件 | ISAの書式通り | 婚姻の真正性証明が実質的な審査の核心（追加資料の充実度が合否を左右する）|

---

## 可注入答案的安全表述

> 「海外配偶者の呼び寄せは、まず日本でISAに**在留資格認定証明書（COE）を申請**します。
> COE取得後、配偶者が現地の日本大使館でビザ申請を行います。申請から来日まで通常**2〜6か月**かかります。
> COE審査では**婚姻の真正性**（出会い・交際の経緯・一緒の写真・渡航記録）が重視されます。
> 必要書類には戸籍謄本・住民票・収入証明・質問書・二人の写真等が含まれます。」

## 材料連接

- 在留資格認定証明書交付申請書（ISA書式）
- 戸籍謄本（日本人側）
- 婚姻証明書（外国）+翻訳文
- 夫婦の写真（交際・婚姻証明）
- 質問書（結婚の経緯）
- 在職証明書・源泉徴収票

## L5/深水信号

- 「COE申請が不許可になりました。再申請できますか？また何が問題でしたか？」→ 不許可理由はISAから書面で通知されないことが多い。行政書士に相談し、不許可の考えられる原因（真正性証明の不足・スポンサーの収入不足等）を分析し、補強して再申請することが基本戦略。
