## 用者場景

「高度専門職のポイント計算で、2024年に改正があったと聞きました。変更点は何ですか？」  
「高度専門職のポイントを計算したいのですが、年収・学歴・経験年数の加算の仕組みを教えてください。」  
「高度専門職のポイントが70点に達するかどうか確認したいです。加点になるものは何ですか？」

---

## 核心事実

**高度専門職のポイント計算は「学術研究活動（ロ）」「高度専門・技術活動（ロ）」「高度経営・管理活動（ハ）」の3区分で異なる計算表がある。ポイントの基本要素は①学歴②職歴③年収④研究実績⑤特別加算（日本語・博士号等）。70点以上で高度専門職1号、80点以上（かつ最終学歴から一定期間内等）で特別高度人材。2024年に特別高度人材（J-Skipの高度人材カテゴリ）の要件が整備された。**

**ポイント計算の区分:**

| 区分 | 活動内容 | 対象者の例 |
|------|---------|---------|
| 高度専門職1号イ（学術研究）| 研究・研究の指導・教育 | 大学教員・研究者 |
| 高度専門職1号ロ（高度専門・技術）| 自然科学・人文科学の専門的業務 | ITエンジニア・専門職 |
| 高度専門職1号ハ（高度経営・管理）| 事業の経営・管理 | 経営者・管理職 |

**主要ポイント項目（ロ区分の例・概要）:**

| 項目 | 加算ポイント |
|------|------------|
| 博士号取得（研究分野以外でも）| +30 |
| 修士号取得 | +20 |
| 学士号取得 | +10 |
| 職歴10年以上 | +20 |
| 職歴7〜9年 | +15 |
| 職歴3〜4年 | +5 |
| 年収1,000万円以上 | +40 |
| 年収700〜999万円 | +25 |
| 年収400〜499万円 | +10 |
| 日本語能力試験N1相当 | +15 |
| 日本の大学卒業 | +10 |
| グロービス等認定MBAプログラム卒業 | +5 |
| イノベーションを促進する機関勤務 | +10 |
| 成長分野への参入 | +10 |
| 中小企業勤務 | +10 |

※ 上記は概要。実際の計算は出入国在留管理庁の公式ポイント計算表（最新版）を参照。

**特別高度人材（J-Skipの高度人材カテゴリ）の要件（2023〜2024年整備）:**

- ポイント計算なしで高度専門職に認められる特例（年収要件等）。
- **年収2,000万円以上**: ポイント計算なしで申請可能。
- **世界トップ大学（QS・THE・ARWU上位300校）卒業**: 特定の在留要件で優遇。
- 詳細はJ-Skipの制度説明（practical-300参照）と組み合わせて確認。

**2024年以降の主な変更点（確認推奨）:**

- 特別高度人材・J-Skipの制度整備（年収2,000万円以上の特例等）。
- 一部の加点要素の追加・修正（最新の告示を確認）。
- スタートアップ・イノベーション支援機関での就労加点の追加。

**ポイント計算後の特典:**

- **70点以上**: 高度専門職1号（5年の在留・家族帯同・永住申請の短縮等）。
- **80点以上・就労3年以上**: 永住申請が最短3年（通常10年）に短縮。
- **70点以上・就労1年以上**: 永住申請が最短1年（特別高度人材）。

## 実務来源

- 行政書士実務: 「高度専門職70点に達するかどうかの計算」が申請相談の定番。年収・学歴・職歴の組み合わせで計算して、加点項目を最大化するアドバイスをする。2024年以降は最新の計算表で確認することが重要。

## 可注入答案的安全表述

> 「高度専門職のポイント計算は**学歴・職歴・年収・日本語能力・特別加算**等の組み合わせで70点以上が基準です。
> **年収1,000万円超は+40点**・**博士号は+30点**・**日本語N1は+15点**が主要加点です。
> **70点以上**で高度専門職1号・**80点以上かつ就労3年**で永住申請が**最短3年に短縮**されます。
> **年収2,000万円以上**は「**特別高度人材（J-Skip）**」としてポイント計算なしで申請可能です。
> 最新のポイント計算表は出入国在留管理庁の公式サイトで確認してください。」

## L5/深水信号

- 「インド国籍・技人国でITエンジニアとして5年就労中。年収650万円。修士号（インドの大学）あり。職歴は日本での5年のみ。高度専門職のポイント計算をしてください。」→ 概算計算（ロ区分）: 修士号+20点・年収650万円（700万円未満は+20点）・職歴5年（5〜6年は+10点）=合計50点。70点には20点不足。追加加点の可能性: ①日本語N1取得→+15点（合計65点）。②インドの大学をQS・THE・ARWU上位300校でリストアップして「認定大学卒業加点」を確認（+10点なら75点）。③中小企業勤務の場合→+10点。④転職して年収700万円以上に上げる→+25点（合計70点以上）。単純に現状では70点に達しないが、複数の加点を組み合わせることで達成可能。行政書士にポイント計算の詳細確認と最適化戦略を相談することを推奨。
