## 用户場景

「日本人と結婚しました。在留資格（配偶者ビザ）を申請するには何が必要ですか？」  
「配偶者ビザの更新申請に必要な書類を教えてください。」

---

## 核心事実

**「日本人の配偶者等」の申請（海外から・国内変更・更新）で必要な書類は、ISAが公表している。審査では婚姻の実態確認が最重視される。**

**申請の種類と申請先:**

| 申請の種類 | 対象 | 申請先 |
|-----------|------|--------|
| 在留資格認定証明書（COE） | 海外在住の配偶者が日本に来る場合 | ISA（スポンサーの在日者が申請） |
| 在留資格変更 | 日本に在留中の別の在留資格から変更 | 居住地管轄ISA |
| 在留期間更新 | 既に配偶者等で在留中の更新 | 居住地管轄ISA |

**新規（海外から来日）または変更申請の主要書類:**

| 書類 | 目的 |
|------|------|
| 在留資格認定証明書交付申請書/変更申請書 | 申請書類 |
| 婚姻を証明する書類 | 婚姻届受理証明書（日本）または婚姻公正証書（外国） |
| 双方の戸籍謄本 | 婚姻の実態・前婚の有無確認 |
| 外国人配偶者のパスポート・写真 | 身分確認 |
| 日本人配偶者の住民票 | 在日確認 |
| 日本人配偶者の職業・収入証明 | 生計能力確認 |
| 申請人の日本語能力・交際経緯（任意） | 婚姻実態の補強 |
| 交際証明書類（写真・SNS・通話記録等） | 偽装婚審査対策 |

**更新申請の主要書類:**

| 書類 | 目的 |
|------|------|
| 在留期間更新許可申請書 | 申請書類 |
| 戸籍謄本（最新） | 婚姻継続の確認 |
| 住民票（世帯全員） | 同居確認 |
| 日本人配偶者の課税証明・納税証明 | 生計能力確認 |
| 写真（交際・生活の写真等） | 婚姻実態の確認 |

**ISAが審査で重視するポイント:**

1. **婚姻の実態**: 同居・共同生活・交際の証拠（写真・通信記録）
2. **交際経緯**: どこで知り合ったか・どのくらいの期間交際したか
3. **同居の有無**: 住民票が同一住所か、実際に同居しているか
4. **年齢差・文化差**: 年齢差が大きい・文化背景が異なる場合は追加確認される傾向
5. **日本人配偶者の経済力**: 外国人配偶者の生計を維持できるか

**偽装婚審査のポイント（実務）:**

- 出会いのきっかけが婚活サービス・マッチングアプリ→ 追加証拠を準備
- 出会いから婚姻まで期間が短い→ 交際期間の証拠を詳細に
- 日本語が全く話せない外国人と日本語ができない日本人→ コミュニケーション手段の説明が必要

## 実務来源

- 行政書士実務: 配偶者ビザは「偽装婚疑い」での追加書類要求が増加傾向。申請時に婚姻実態の証拠（写真・通話記録・旅行記録等）を十分準備することが重要。
- ISA窓口実態: 年齢差15歳以上・短期間での婚姻は追加書類の要求確率が高い。

---

## 法条層 vs 実務層差異

| 観点 | 法条の建前 | 実務の現実 |
|------|----------|-----------|
| 書類一覧 | ISA公式リストが基準 | 実務では婚姻実態の補強書類が追加で求められることが多い |
| 偽装婚の審査 | 法令上の明示基準なし | ISA審査官の裁量で追加書類要求・面接が実施される |

---

## 可注入答案的安全表述

> 「日本人の配偶者等（配偶者ビザ）の申請・更新には、戸籍謄本・住民票・収入証明書に加えて、**婚姻の実態を示す書類**（交際・同居の写真・SNS記録等）が重要です。
> ISAは婚姻の実態（偽装婚かどうか）を審査するため、出会いから婚姻までの経緯・同居状況・コミュニケーション手段を具体的に説明できる書類を準備することをお勧めします。
> 申請書類の詳細はISAの公式サイトでご確認いただくか、行政書士にご相談ください。」

## L5/深水信号

- 「追加書類を求められました」→ ISAが婚姻実態に疑念を持っているサイン。行政書士相談推奨。
- 「外国の婚姻証書の翻訳・認証が必要です」→ アポスティーユ・大使館証明等の手続きが必要。複雑案件。
