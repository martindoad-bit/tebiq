# Matter v0 Copy Map v0.1

本文件定义"我的事项"（Matter）功能 v0 的所有状态文案。  
Matter = 用户在留风险相关事项的结构化记录单元，包含风险评估 + 行动路径。

面向：最终用户（经 DOMAIN + 项目负责人验收后方可用于 production）。

---

## 使用规则

- `state`：Matter 状态标识符
- `label`：界面中显示的状态名称（短）
- `user_copy`：主文案
- `action_copy`：行动按钮或行动提示文字
- `tone_note`：语气说明（不进 UI）
- `forbidden`：禁止出现的表达（不进 UI）

## 专业参照 Block（matter_professional_reference）

**Patch 5 + Patch 3 修正：**

专业人士类型不硬编码，由 DOMAIN 在具体场景中指定。

默认参照：
> 建议咨询行政書士或入管手续专业人士。

DOMAIN 指定覆盖规则：

| 场景类型 | 对应专业类型 | 示例文案方向 |
|---------|------------|-------------|
| 在留资格变更 / 更新 / 特定活动 | 行政書士 / 入管専門弁護士 | 建议咨询行政書士确认手续适用性。 |
| 税务居住地 / 扶养认定 | 税理士 | 建议咨询税理士确认具体情况。 |
| 公司登记 / 法律文件 | 司法書士 | 建议咨询司法書士确认文件要求。 |
| 劳动合同 / 社会保险 | 社労士 | 建议咨询社労士确认权益适用。 |
| 复合情形（多类型交叉）| DOMAIN 明确指定 | 不默认推荐单一类型 |

禁止：
- 在无 DOMAIN 确认的情况下推荐具体机构或服务商
- 将专业参照写成销售转化语言（"立即联系""预约专家""我来帮你联系"）
- 在不必要的低风险场景中插入专业参照
