### P0-A01 経営管理 → 技人国転換

```yaml
tag: eval-lab-v1-A01
question: 経営管理签怎么转技术・人文知识・国际业务？

expected_direction: >
  在留資格変更（経営管理 → 技術・人文知識・国際業務）申請。
  前提：役員退任登記を変更申請「前」に完了。
  業務内容が三領域（自然科学/人文科学/国際業務）に該当することを先に確認。

hidden_risk: >
  役員退任登記なしで変更申請 → 不許可（登記完了が要件）。
  この手続きを知らない利用者が独自に進めてしまう可能性が高い。

must_have:
  - 役員退任登記の「先行完了」（申請前に必ず）
  - 三領域該当性の確認
  - 変更申請の流れ

must_not_have:
  - 「会社を辞めてから申請すればいい」（登記手続き省略示唆）
  - 「すぐ転換できる」（条件なし断言）
  - 対許可結果の承諾

handoff_trigger: 役員退任登記のタイミング判断 → 行政書士 + 司法書士
fact_card_candidate: FC-002（役員退任登記時序、既存）
human_review_required: yes（役員退任登記の具体的タイミング要件）
```

### P0-A05 / D03 家族滞在 → 工作签（方向反）

```yaml
tag: eval-lab-v1-A05 / eval-lab-v1-D03
question: 家族滞在想转工作签怎么办？ / 家族滞在想转工作签需要什么？

expected_direction: >
  在留資格変更（家族滞在 → 技人国など就労在留）。
  資格外活動許可（28h/週）は「家族滞在のまま働く」であり、独立签证ではない。
  両者を厳格に区別することが最重要。

hidden_risk: >
  「資格外活動許可を取れば働ける」という回答 → P0方向反。
  独立した就労在留資格は永遠に取得できない。

must_have:
  - 二条路径の明確な区別：①資格外活動許可（家族滞在継続）vs ②在留資格変更（独立）
  - 変更申請の要件（学歴/業務内容の技人国該当性）
  - 変更申請中の就労不可期間

must_not_have:
  - 「資格外活動許可で28時間以内なら大丈夫」を唯一の答えとして返す
  - 変更申請前の無許可就労示唆

handoff_trigger: 学歴/業務関連性が不明確 → 行政書士事前面談
human_review_required: no（事実明確）
```

### P0-A09 配偶签 → 定住者（離婚後）

```yaml
tag: eval-lab-v1-A09
question: 配偶签离婚后想转定住怎么办？

expected_direction: >
  婚姻3年以上＋在日1年以上の条件を先に確認。
  条件を満たす場合：在留資格変更（配偶者 → 定住者）申請。
  条件を満たさない場合：定住者路径は基本的に不可（例外：子の親権/DV被害者）。
  離婚後6ヶ月以内に変更申請の義務あり。

hidden_risk: >
  婚姻年数の条件を確認せず「変更できます」と案内 → 不許可 + 6ヶ月超過。
  DV案件を行政書士だけに紹介 → 弁護士+DVセンターの連携が必要。

must_have:
  - 婚姻3年以上＋在日1年以上の条件を先に確認
  - 離婚後6ヶ月以内の変更義務
  - 条件不満足時の例外（子の親権/DV）の存在を示唆

must_not_have:
  - 条件確認なしに「定住者に変更できます」
  - DV案件を行政書士のみに案内

handoff_trigger: 婚姻年数3年未満 → 行政書士（ほぼ必須）；DV → 弁護士+DVセンター
human_review_required: yes（DV案件の連携先・婚姻実態立証の具体的方法）
```
