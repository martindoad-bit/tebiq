# TEBIQ Handoff Triggers

**状态**: draft / needs human review
**版本**: v0.2
**作成**: 2026-05-04
**作成者**: TEBIQ-DOMAIN-CC
**用途**: 定义哪些情况AI必须建议用户寻求专业确认，按场景分类。用于Eval Lab标注和Prompt规则制定。

**重要说明**：
- "专业确认"不是万能挡箭牌。本文件明确区分：① 必须handoff（不能只靠AI）② 建议handoff（AI可给方向，但建议确认）③ 不必要handoff（AI可独立回答）
- "建议咨询专业人士"不是有效handoff。有效handoff必须说明：找谁（行政書士/弁護士/社労士/税理士）+ 为什么现在需要 + 不找的风险

---

## 类别1：不许可 / 处分通知类

**原则：任何不利处分通知必须立即handoff，无例外**

### 触发条件

| 触发 | 紧迫程度 | 找谁 | 不handoff的风险 |
|------|---------|------|---------------|
| 收到「不許可通知書」 | 立即（当天） | 行政書士（申請取次） + 入管専門弁護士 | 3个月审查请求期可能被错过；错误出国→5年上陆拒否 |
| 收到「在留資格取消通知書」 | 立即 | 行政書士 + 弁護士 | 30日内意见陈述期可能被错过 |
| 收到「退去強制令書」 | 立即 | 弁護士（必须，不是建议） | 强制驱逐程序已启动 |
| 收到「出頭通知書」 | 出頭前（通常7-14日） | 行政書士 + 弁護士 | 单独出頭无法有效陈述 |
| 收到「補正通知」/「資料提出通知」但不知道补什么 | 14日内 | 行政書士 | 超期→直接不许可 |
| 不許可后考虑出国 | 出国前 | 行政書士 + 弁護士 | 可能触发5年上陆拒否 |

### 禁止AI独立回答的内容
- 不许可后是否应该出国
- 审查请求的胜算评估
- 具体的审查请求策略

## 类别2：在留资格変更类（签证转换）

**原则：涉及复杂条件判断或材料边界时必须handoff**

### 経営・管理 相关

| 触发 | 紧迫程度 | 找谁 |
|------|---------|------|
| 役員退任登記的时序判断（変更前/后） | 申请前 | 行政書士 + 司法書士 |
| 新规5项条件的边界判断（学历认定/常勤職員资格） | 申请前 | 行政書士 |
| 跨境资金来源（中国ODI备案） | 申请前 | 行政書士 + 跨境律师 |
| 共享办公室/自宅兼用办公 | 申请前 | 行政書士 |
| 資本金不足但想增资 | 申请前 | 行政書士 + 司法書士 + 税理士 |
| 近期决算赤字 | 申请前 | 行政書士 + 税理士 |

### 技術・人文知識・国際業務 相关

| 触发 | 找谁 |
|------|------|
| 学历边界（専門学校卒+業務内容関連性不明确） | 行政書士 |
| 10年実務経験的算法（転職歴複雑） | 行政書士 |
| 業務内容含販売/サービス等三领域边界 | 行政書士 |
| 曾被不许可1次 | 行政書士 |

### 配偶者・家族関係 相关

| 触发 | 找谁 |
|------|------|
| 离婚后転換（婚姻年数不满3年） | 行政書士（几乎必须） |
| 离婚原因为DV | 弁護士 + DVセンター + 行政書士 |
| 家族滞在→独立工作签（扶養者签证状态不稳定） | 行政書士 |
| 父母长居日本（老親扶養路径评估） | 行政書士（有老親扶養実绩者） |

## 类别3：永住申请类

**原则：有任何公的義務履行不完整记录时必须handoff**

| 触发 | 找谁 | 说明 |
|------|------|------|
| 直近5年内有住民税滞纳/迟交 | 行政書士 | 补缴后等6-12个月，经緯書写法需确认 |
| 国民年金未缴超过2年 | 行政書士 + 社労士 | 追溯上限2年，超出部分无法补回 |
| 公司未给上厚生年金（雇主违法） | 社労士 + 行政書士 | 补加入+経緯書需专业操作 |
| 跨多个市町村的納税証明取得困难 | 行政書士 | 补取材料流程复杂 |
| 已有差押/財産差押记录 | 行政書士 + 弁護士 | 永住路径可能需要5-10年等待 |
| 高度人材积分边界判断 | 行政書士 | ポイント計算の具体的认定 |
| 婚姻相关特例永住（3年+在日1年） | 行政書士 | 婚姻実態の立证策略 |

## 类别4：公的義務 / 年金 / 税务类

**原则：涉及追溯缴纳、差押、分期协商时必须handoff**

| 触发 | 找谁 |
|------|------|
| 国民年金/国保切换但不知道资格丧失日 | 社労士 |
| 公司未加入厚生年金（追溯补加入协商） | 社労士 |
| 住民税督促状→差押予告已发出 | 司法書士 + 弁護士 + 税理士 |
| 確定申告 漏申/错申（影响永住証明） | 税理士 |
| 倒産特例給付申请 | 社労士 |

**可以不handoff的情况**（AI可独立回答方向）：
- 正常搬家后更新在留卡（区役所自己办）
- 国民年金/国保正常加入（区役所自己办）
- 住民税納税通知（正常纳税，自己缴就行）

---

## 类别5：执法调查类

**原则：任何入管/警察/労基署的调查通知必须立即handoff**

| 触发 | 找谁 | 紧迫程度 |
|------|------|---------|
| 入管出頭通知（调查/质问） | 行政書士 + 弁護士 | 出頭前（通常7-14日内） |
| 警察質問書/出頭通知 | 弁護士 | 立即 |
| 労基署调查 | 弁護士 + 社労士 | 出頭前 |
| 公司被摘発时本人为役員/採用担当 | 弁護士（刑事系） | 当天 |
| 裁判所文书（支払督促/訴状/特別送達） | 弁護士/司法書士 | 立即（有期限） |

---

## 类别6：出境 / 再入国类

**原则：在留资格申请期间或不许可后的出境必须handoff**

| 触发 | 找谁 | 风险 |
|------|------|------|
| 不许可后考虑出国 | 行政書士 + 弁護士 | 5年上陆拒否 + 审查请求权丧失 |
| 在留资格取消程序中出国 | 行政書士 + 弁護士 | 上陆拒否 |
| 再入国許可の有効期間が超える长期出国 | 行政書士 | 在留资格丧失 |
| 在留期限到期前出国（みなし再入国期间判断） | 行政書士 | 期限超过→在留资格丧失 |

**不需要handoff的情况**：
- 在留期间内出国，1年内返回（みなし再入国で自己処理可）
- 正常出张/旅行（在留期間内、1年以内）

## 类别7：续签 / 更新类（在留期間更新）

**原则：更新本身通常不需handoff；但以下例外情况必须确认**

### 触发条件

| 触发 | 紧迫程度 | 找谁 | 原因 |
|------|---------|------|------|
| 在留期限已过，在特例期間中（2ヶ月以内） | 立即 | 行政書士 | 特例期間的操作方式和风险需确认 |
| 在留期限届満まで30日以内で申請未了 | 立即 | 行政書士 | 更新申請に間に合わない可能性 |
| 近期有离职/転職（転職后6ヶ月以内で更新） | 申请前 | 行政書士 | 業務内容継続性の審査有り |
| 副業・業務委託の割合が高い（主業務の50%超） | 申请前 | 行政書士 | 在留資格の活動との整合性 |
| 前回更新が許可薄い（1年許可） | 申请前 | 行政書士 | 材料強化の検討必要 |
| 在留期間中に公的義務違反があった（納税遅延等） | 申请前 | 行政書士 + 税理士 | 素行要件審査に影響 |
| 経営・管理：前回から役員・事業内容が大幅変更 | 申请前 | 行政書士 | 変更届出漏れの確認 |

### 不需要handoff的情况（AI可独立回答）

- 在留期間内、就労継続中、転職なし → 自分で申請可
- 住所変更のみ → 区役所 転入届 + 在留カード届出（14日以内）
- 在留カード更新（永住者の有効期限切れ） → 入管に自分で申請可
- 正常な更新材料確認（何を準備するか） → AIが案内可

## Handoff表达规范

### 有效handoff（TEBIQ应该用的方式）

```
✓ "这个情况建议在递申请前先和行政書士聊一下——
   具体是[原因]，如果[条件]不达标，续签可能不许可。"

✓ "收到不许可通知书，当天联系行政書士（申請取次資格）或
   入管専門弁護士——不要自己出国或等待，3个月审查请求期在流逝。"

✓ "跨境资金这块需要行政書士+跨境律师一起评估，
   涉及日本侧资金来源说明和中国侧ODI备案，
   两边都可能有合规问题。"
```

### 无效handoff（不应该出现）

```
✗ "建议咨询专业人士。"（没有说找谁、什么时候、为什么）
✗ "具体情况因人而异，请咨询行政书士。"（泛泛无用）
✗ "此类问题较为复杂，建议专业评估。"（用户不知道怎么做）
```

## 专家对应速查表

| 专家类型 | 主要适用场景 |
|---------|------------|
| 行政書士（申請取次） | 在留资格申請・変更・更新；不许可対応；届出系手続 |
| 弁護士（入管専門） | 不许可审查请求；在留资格取消対応；退去強制手続 |
| 弁護士（刑事系） | 不法就労助長罪・偽証罪等刑事事件 |
| 弁護士（一般） | 裁判所文书・差押・民事纠纷 |
| 司法書士 | 商業登記（増資・役員変更・本店移転）；不動産登記 |
| 社労士 | 厚生年金・健康保険補加入；倒産特例給付；労災 |
| 税理士 | 法人税・確定申告・資金来源説明；ODI联动 |
| DVセンター | DV被害者の在留資格변更支援（行政書士と连携） |
