## 2. 覆盖不足的风险类型

以下の領域はカバレッジが薄いまたは存在しない：

### GAP-01 親の長期居住（家族滞在対象外の親族）

- **現状**：100Q に「親を呼びたい」「両親の長期居住」の題目がない
- **リスク**：家族滞在の対象範囲（配偶者・子のみ）を誤解するユーザーが多い。「永住者なら親を呼べる」という誤解が特に高リスク
- **重要度**：MEDIUM〜HIGH（P0 候補：「親も家族滞在で来れる」の誤断言）
- **关联 DGC**：DGC-10

### GAP-02 高度専門職の具体リスク

- **現状**：A07（経管→高度専門職の転換）は 1 題のみ。高度専門職のポイント計算・維持条件・更新の具体リスクがない
- **リスク**：高度専門職資格者の維持条件（ポイント維持義務）や特典（永住優遇・家族呼び寄せ）に関する誤解
- **重要度**：MEDIUM（対象者は限定的だが誤解は高い）

### GAP-03 留学から就労ビザへの転換（卒業前・就職決定前）

- **現状**：A06（留学→人文転換）は 1 題。卒業直前の焦り・内定なし状態での転換はカバーなし
- **リスク**：「卒業後もしばらくいられる」という誤解。就職先決定前の在留申請タイミング
- **重要度**：MEDIUM（留学生数が多いため頻度は高い）

### GAP-04 DV（家庭内暴力）による在留問題

- **現状**：D04（配偶者分居）はあるが、DV 被害による在留保護・配偶者签の維持ルートが未カバー
- **リスク**：DV 被害者が「離婚したら帰国しなければならない」と誤解する可能性
- **重要度**：HIGH（人命・安全に関わる可能性）
- **注意**：専門性が高い。DOMAIN 草稿での言及に留める。DV センターへの handoff が最優先

### GAP-05 みなし再入国中の在留期限切れ

- **現状**：I03（みなし再入国の仕組み）はあるが、「出国中に在留期限が切れた」シナリオが未カバー
- **リスク**：出国中に在留期限が来ると、みなし再入国が無効になり帰国できない事態
- **重要度**：HIGH（I04/I05/I06 で部分カバーだが、このシナリオは明示的にない）

### GAP-06 複数の在留資格を同時に持つ場合（高度専門職 + 経管等）

- **現状**：未カバー
- **重要度**：LOW（対象者が限定的）

## 3. 应补充的问题类型

以下を優先的に補充することを DOMAIN は推奨する（産品負責人裁決が必要）：

| 優先度 | 補充すべき分野 | 理由 |
|---|---|---|
| P0 | GAP-04 DV 被害と在留 | 人命・安全への直接影響 |
| P1 | GAP-01 親の長期居住 | 誤解頻度が高く、P0 誤断言のリスク |
| P1 | GAP-05 出国中の期限切れ | 帰国不可の最悪シナリオ |
| P2 | GAP-03 留学→就労転換のエッジケース | 留学生数の多さ |
| P2 | GAP-02 高度専門職リスク | 対象者は限定的だが影響は大 |

## 8. 是否需要新增 Seed Questions

**DOMAIN 推奨**：暫定 3 題のみ追加を検討（#1 親の長期居住 / #2 DV / #3 出国中期限切れ）。

大量の新題追加は：
- Eval の焦点を分散させる
- 正式標注の workload を増やす
- 現行 100Q の検証が完了する前に優先度が不明確になる

**現行 100Q の Eval を先に完了させることを DOMAIN は推奨する。**
