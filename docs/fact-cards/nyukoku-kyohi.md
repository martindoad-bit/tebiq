## current_effective_fact

日本への入国（上陸）が拒否される主な事由は入管法第5条に定められている。退去強制を受けた者は原則5年間（または10年間）、自発的な出国命令で出国した者は1年間、日本への入国が禁止される。

**主な上陸拒否事由（{{TODAY_ISO}} 現在）：**

| 事由 | 上陸禁止期間 |
|------|------------|
| 退去強制（強制送還）された | **5年**（重大事案は**10年**または**永久**） |
| 出国命令による出国 | **1年** |
| 禁錮以上の刑罰（日本・外国の法律） | 事案に応じて |
| 麻薬・覚醒剤等の規制薬物関連犯罪 | 事案に応じて |
| テロリスト・公安上危険な者 | 永久禁止の可能性 |

## exceptions_or_transition

- **10年または永久禁止**：複数回の退去強制・薬物犯罪等、より重大なケースでは10年以上または永久禁止になる場合あり（`10nen_jijo_detail`確認要）
- **特別な上陸許可**：法務大臣の特別裁量による上陸許可（上陸拒否事由があっても例外的に許可される場合）が設けられているが、非常に限定的
- **日本国内への影響**：入国できなければ、当然ながら日本での就労・家族帯同等は不可能

## qa_cases

**Q1: 過去にオーバーステイで帰国しました。日本に戻れますか？**
A: オーバーステイ（不法残留）で退去強制令書の発付を受けて退去した場合、原則として5年間（重大事案は10年または永久）日本への入国が禁止されます。禁止期間が経過した後は入国申請可能ですが、過去の不法残留が審査に影響する場合があります。

**Q2: 出国命令で出国しました。何年後に戻れますか？**
A: 出国命令による出国の場合は、1年間の上陸禁止となります（退去強制による強制送還よりも短い）。1年経過後は入国申請が可能です。
