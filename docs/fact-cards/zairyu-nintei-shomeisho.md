## current_effective_fact

日本の就労・留学・家族ビザ（短期滞在・永住を除く）で来日する外国人は、事前に在留資格認定証明書（COE）の取得が必要。日本の雇用主・学校等が申請代理人となり、地方出入国在留管理官署（窓口またはオンライン）に申請。標準処理期間1〜3か月、手数料無料。2023年3月から電子メール交付が可能。

**COEから来日までの3ステップ（{{TODAY_ISO}} 現在）：**

| ステップ | 内容 | 場所 |
|---------|------|------|
| **STEP 1** | **在留資格認定証明書交付申請** 雇用主等が代理申請 | 地方入管（窓口/オンライン） |
| **STEP 2** | **査証（ビザ）申請** COEを提示して申請 | 在外公館（日本大使館・領事館） |
| **STEP 3** | **上陸申請** 空港でCOEを提示→在留カード交付 | 日本の空港 |

**COE申請概要（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 申請代理人 | 雇用主・学校等の職員・弁護士・行政書士等 |
| 申請場所 | 地方出入国在留管理官署（窓口/オンライン）— **郵送不可** |
| 標準処理期間 | **１か月〜３か月** |
| 手数料 | **無料** |
| COE有効期間 | **3か月**（入管法施行規則 — ai推定） |
| 電子メール交付 | **可**（2023年3月17日〜）— 海外に転送可能 |

**電子COEの利用フロー（{{TODAY_ISO}} 現在）：**

1. ISAから申請代理人に電子メールでCOEが送付される
2. 申請代理人がメール内のURLから**受領登録**を行う
3. 電子メールを**海外の外国人本人に転送**
4. 外国人本人が在外公館でメールを提示→査証申請
5. 来日時に空港でメールを提示→上陸申請

## exceptions_or_transition

- **COEが不要な場合**：短期滞在・永住者はCOEの対象外（ISA公式）。ワーキングホリデービザ等も通常COEは不要（査証のみ）
- **電子COEの注意**：電子メール受領後、URLから受領登録が必要。登録しないと有効なCOEとして扱われない可能性（ai推定）
- **COEの有効期限切れ**：3か月以内に来日・査証申請しなかった場合は再申請が必要（ai推定 — 更新制度なし）
- **2026年4月15日以降の追加要件**：技人国等のカテゴリー3・4は代表者申告書と（対人活動職種の場合）語学力証明の追加が必要。CEFR B2相当（JLPT N2等で免除）（zairyu-shinsei-category参照）

## qa_cases

**Q1: 海外にいる外国人社員を採用しました。COEはどこに申請しますか？処理期間は？**
A: 会社が申請代理人として、地方出入国在留管理官署の窓口またはオンラインで申請します（郵送は不可）（ISA公式）。標準処理期間は**1か月〜3か月**です（ISA公式）。手数料は無料です。

**Q2: COEは電子メールで受け取れますか？**
A: はい。2023年3月17日から電子メールでの交付が可能です。受領した電子メールを海外の外国人本人に転送でき、COEを海外に郵送する必要がなくなりました（ISA公式）。ただし、メール内のURLから「受領登録」を行う必要があります。

**Q3: COEを受け取った後、何をすればいいですか？**
A: ① COEを海外の外国人本人に送付（電子メール転送または紙を郵送）、② 本人が在外公館（日本大使館等）でCOEを提示して**査証（ビザ）申請**、③ 入国時に空港でCOEを提示して**上陸申請**、という流れです（ISA公式）。
