## current_effective_fact

技術・人文知識・国際業務（技人国）の在留期間更新申請は、在留期限の概ね**3か月前**から申請できる。申請は住所地管轄の地方出入国在留管理局または出張所で受け付けている（オンライン申請も可）。

**技人国更新申請の基本情報（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 申請開始時期 | 在留期限の概ね**3か月前**から（6ヶ月以上の在留期間保有者） |
| 標準処理期間 | **2週間〜1か月** |
| 申請場所 | 住所地管轄の地方入管局・出張所（またはオンライン） |

### 基本書類（共通部分）

1. 在留期間更新許可申請書
2. 写真 1葉（縦4cm×横3cm、16歳未満は不要）
3. パスポート・在留カード（提示）
4. カテゴリー該当証明書類（雇用先の規模・属性による）

所属機関代表者申告書、住民税の課税（非課税）証明書および納税証明書等は、所属機関カテゴリーや更新時の状況により扱いが変わるため、ISA公式ページのカテゴリー別資料を確認する。

### カテゴリー3・4 — 転職後の追加書類（{{TODAY_ISO}} 現在）

転職後の申請者（特にカテゴリー3・4対象の雇用先）には以下の追加書類が必要：

- 労働条件通知書または雇用契約書（写）
- 登記事項証明書（法人の場合）
- 事業内容を明らかにする資料（会社案内・パンフレット等）
- 直近年度の決算書の写し

## exceptions_or_transition

- **在留期限内に更新申請済みで審査中の場合**：在留期間の満了日から2か月を経過する日または申請への処分日のいずれか早い日までは従前の在留資格で在留可能（「特例期間」）
- **カテゴリー1・2（上場企業・一定規模法人等）に所属する場合（ai推定）**：書類が一部簡略化される場合がある。具体的な書類一覧はISA手続きページを確認
- **転職なし（同じ雇用先での更新）の場合**：カテゴリー別資料を確認。転職後の事業内容・労働条件資料とは別に扱う

## qa_cases

**Q1: 技人国の更新はいつから申請できますか？処理期間はどのくらいですか？**
A: 在留期限の概ね**3か月前**から申請できます。標準処理期間は**2週間〜1か月**です。在留期限が近づく前に早めに必要書類の準備を始めることをお勧めします。申請中は在留期限を過ぎても審査結果が出るまで「特例期間」として在留が認められます。

**Q2: 転職後に技人国を更新する場合、通常の更新と何が違いますか？**
A: 技人国の更新では、共通書類に加えて所属機関のカテゴリーや転職後の状況に応じた資料確認が必要です。新しい会社のカテゴリー、労働条件、事業内容、直近決算などを確認し、必要資料はISA公式ページまたは窓口で確認してください。

**Q3: 技人国更新に必要な基本書類を教えてください。**
A: 基本書類は①在留期間更新許可申請書、②写真1葉（16歳未満不要）、③パスポート・在留カード（提示）、④カテゴリー該当証明書類等です。所属機関代表者申告書、税証明、転職後の労働条件通知書等は、雇用先カテゴリーや転職の有無によって扱いが変わるため、ISA公式ページのカテゴリー別資料を確認してください。
