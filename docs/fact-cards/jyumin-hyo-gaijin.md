## current_effective_fact

中長期在留資格を持つ外国人は、入国後または転居後14日以内に市区町村に転入届を提出する義務がある。転入届提出時に在留カードへの住所記載が行われ、住民票が作成される。

**転入届の手続き（{{TODAY_ISO}} 現在）：**

| 手順 | 内容 |
|------|------|
| 1 | 市区町村の窓口（住民課・市民課等）に行く |
| 2 | 転入届に記入（氏名・住所・在留資格等） |
| 3 | 在留カード・パスポートを提示 |
| 4 | 市区町村が在留カードに住所を記載 |
| 5 | 住民票が作成される |

**住民票の写しの取得：**
- 市区町村窓口：手数料（通常200〜300円）
- コンビニ（マイナンバーカードが必要）：手数料（通常150〜200円）

## exceptions_or_transition

- **短期滞在者（90日以内）**：住民票の登録対象外（住民票なし）
- **引越し時の手続き**：同一市区町村内は転居届、他市区町村は転出届→転入届
- **在留カードの住所欄**：転入届を提出しないと在留カードの住所欄が空白のまま→各種手続きで問題になる可能性

## qa_cases

**Q1: 先週日本に来ました。住民票の登録はどうすればいいですか？**
A: 入国から14日以内に居住する市区町村の窓口（住民課・市民課等）に転入届を提出してください。在留カードとパスポートを持参します。転入届提出時に市区町村が在留カードに住所を記載します。

**Q2: 住民票の写しが必要になりました。どこで取れますか？**
A: 居住する市区町村の窓口で手数料を支払って取得できます。マイナンバーカードがあればコンビニのマルチコピー機でも取得可能です（手数料は市区町村によって異なります）。
