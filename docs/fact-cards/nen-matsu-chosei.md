## current_effective_fact

年末調整は、会社（雇用主）が毎月の給与から源泉徴収した税額を年末に精算する手続き。外国人会社員も「居住者」であれば対象となり、控除申告書を提出することで、払い過ぎた税金の還付を受けることができる。

**年末調整の仕組み：**
1. 毎月の給与から所得税を概算で源泉徴収
2. 年末（12月）に実際の年間所得と控除を計算
3. 払い過ぎ → **還付**（12月の給与または翌年1月に)
4. 不足 → 追加徴収

**提出する書類：**

| 書類 | 内容 |
|------|------|
| **扶養控除等（異動）申告書** | 家族構成・扶養家族の申告（毎年提出） |
| **保険料控除申告書** | 生命保険・地震保険等の控除申告 |
| **基礎控除申告書 兼 配偶者控除等申告書** | 基礎控除・配偶者控除の申告 |

**外国人の注意点：**
- 「居住者」（引き続き1年以上日本在住）であれば日本人と同様に年末調整の対象
- 「非居住者」（在留1年未満・短期滞在等）は年末調整の対象外（別途対応要）

## exceptions_or_transition

- **医療費控除は年末調整で申請できない**：医療費控除は確定申告でのみ申請可能（ai推定）
- **副業収入がある場合**：年収20万円以上の副業収入がある場合は確定申告が必要（年末調整では精算されない）
- **転職・退職した場合**：前職の源泉徴収票を新しい会社に提出して合算精算が必要（ai推定）

## qa_cases

**Q1: 外国人ですが年末調整はやる必要がありますか？**
A: 日本に引き続き1年以上住んでいる「居住者」であれば、日本人と同様に年末調整の対象です。会社から書類（扶養控除等申告書等）が配られますので、必要事項を記入して提出してください。払い過ぎた税金が還付される場合があります。

**Q2: 年末調整で海外にいる家族を扶養として申告できますか？**
A: 2023年以降、海外在住の非居住者親族（30歳以上70歳未満）への扶養控除の適用に制限があります。詳細は税理士または国税庁の情報をご確認ください。申告書には正確な情報を記入する必要があります。

**Q3: 在留して半年しか経っていません。年末調整はありますか？**
A: 日本に引き続き1年以上住んでいない「非居住者」の場合、年末調整の対象外となります。この場合は別途対応が必要になりますので、会社の担当者または税理士に相談してください。
