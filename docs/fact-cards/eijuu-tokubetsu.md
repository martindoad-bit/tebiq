## current_effective_fact

特別永住者は、旧植民地出身者（在日コリアン・在日台湾籍等）の平和条約国籍離脱者およびその子孫が取得できる在留資格。通常の外国人よりも安定した法的地位を持ち、退去強制の事由も限定的。

**特別永住者の特徴（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 対象 | 平和条約国籍離脱者（在日コリアン・在日台湾籍等）およびその子孫 |
| 証明書 | 特別永住者証明書（在留カードとは別の証明書） |
| 有効期間 | 16歳未満：16歳の誕生日まで（または7年）、16歳以上：**7年** |
| 更新 | 有効期間満了2か月前から市区町村窓口で申請 |
| 退去強制 | 内乱・外患罪等の重大犯罪のみ（通常外国人より限定的） |

- **通常の永住者への変更**：特別永住者が通常の永住者への変更を希望することは可能だが、通常は特別永住者の方が保護が強い（ai推定）

## qa_cases

**Q1: 特別永住者証明書の更新手続きを教えてください。**
A: 有効期間満了の2か月前から、居住する市区町村の窓口で更新申請ができます。16歳以上の有効期間は7年です。必要書類の詳細は最寄りの市区町村窓口にご確認ください。

**Q2: 特別永住者と通常の永住者の違いは何ですか？**
A: 主な違いは①対象者（特別永住者は平和条約国籍離脱者の子孫）、②証明書（特別永住者証明書 vs 在留カード）、③退去強制事由（特別永住者は内乱・外患罪等の重大犯罪のみに限定）です。特別永住者の方が通常の外国人よりも安定した法的地位を持ちます。
