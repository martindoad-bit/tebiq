## current_effective_fact

外国人が民間賃貸住宅を借りる場合、人種・国籍のみを理由とした入居拒否は差別的取扱いとして問題とされている。保証人がいない場合は家賃保証会社の利用が一般的。外国人は「住宅確保要配慮者」として国・自治体の支援対象。

**入居審査の基本（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 入居拒否 | 国籍のみを理由とした拒否は差別的取扱い（問題視される） |
| 保証人 | 家賃保証会社で代替可能 |
| 必要書類（ai推定） | 在留カード・収入証明・緊急連絡先等 |
| 支援制度 | 住居確保給付金（離職・廃業等の場合） |

## exceptions_or_transition

- **公営住宅**：都道府県・市区町村によって外国人の申請要件が異なる（koeijutaku-gaijin 参照）
- **住居確保給付金**：離職・廃業等で家賃が払えない場合、市区町村の自立相談支援機関（生活困窮者自立支援窓口）で申請
- **相談窓口**：差別を受けた場合は法務局の「みんなの人権110番」（0570-003-110）に相談可能

## qa_cases

**Q1: 外国人だからという理由でアパートの入居を断られました。どうすればいいですか？**
A: 国籍のみを理由とした入居拒否は差別的取扱いとして問題とされています。法務局の「みんなの人権110番」（0570-003-110）または最寄りの法務局に相談することができます。また、外国人向けに対応している不動産業者を探すことも有効です。

**Q2: 保証人がいません。それでも賃貸を借れますか？**
A: 家賃保証会社を利用することで、保証人なしで賃貸を借りられる場合があります。在留カードと収入証明（給与明細等）があれば審査を受けることが可能です。不動産業者に相談してください。
