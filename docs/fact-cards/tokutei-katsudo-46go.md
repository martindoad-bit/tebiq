## current_effective_fact

特定活動（告示第46号）は、日本の大学・大学院・認定専修学校を卒業したJLPT N1水準の外国人が、飲食・小売・製造・介護等を含む幅広い業種で常勤就労できる制度。技人国では認められない業種も可能だが、JLPT N1（またはBJT480点以上）が必須条件。転職時は届出でなく在留変更申請が必要。

**特定活動46号 主要要件（{{TODAY_ISO}} 現在）：**

| 要件 | 内容 |
|------|------|
| 学歴 | 本邦の大学（4年制）・大学院・認定専修学校専門課程を卒業・修了（日本国内の学校に限定） |
| 日本語能力 | **JLPT N1** または **BJT480点以上**（N2は不可） |
| 就労形態 | **常勤（フルタイム）のみ**（パートタイム・アルバイト・派遣は不可） |
| 業務要件 | 「日本語を用いた円滑な意思疎通を要する業務」を含む活動 |
| 報酬 | 日本人と同等以上 |
| 初回在留期間 | 原則**1年**（留学からの変更・初回更新） |

**対象者（学歴要件）の詳細：**

|------|------|
| 本邦4年制大学 | 卒業・学士学位授与 ✅ |

- ホテルでの**通訳・広報業務兼接客**

## exceptions_or_transition

- **外国大学卒業者は対象外**：日本国内の大学等の卒業が要件（ISA公式明示）。外国の大学を卒業した場合は技人国等の他のビザで対応
- **認定専修学校の判断**：専修学校卒業者の場合、在学中の学校がMEXT認定を受けているか在学中に確認が必要（認定リストは変動する可能性）
- **パートタイム・派遣不可**：常勤フルタイムの雇用契約のみ対象。現在の留学ビザでのアルバイトとは別制度
- **転職のたびに在留変更申請**：技人国と異なり、転職のたびに正式な在留資格変更許可申請が必要（届出のみでは不可）
- **ガイドライン令和8年4月改定**：本カードは最新版（令和8年4月）のガイドラインに基づく

## qa_cases

**Q1: 日本の大学を卒業した後、飲食店で働きたいです。特定活動46号で申請できますか？**
A: はい。本邦の4年制大学卒業者で、JLPT N1またはBJT480点以上を持っていれば特定活動（告示第46号）での就労が可能です。飲食店での業務も「日本語を用いた円滑な意思疎通を要する業務」の要件を満たせば認められます（ISA公式に飲食店での店舗管理・通訳兼接客の事例あり）。ただし常勤フルタイムの雇用契約が必要で、パートタイムや派遣は対象外です。

**Q2: JLPT N2しか持っていません。特定活動46号に申請できますか？**
A: いいえ。特定活動（告示第46号）の日本語要件は「JLPT N1またはBJT480点以上」です（ISA公式）。N2では申請できません。技術・人文知識・国際業務（技人国）への変更を検討するか、N1を取得してから申請することになります。

**Q3: 特定活動46号の会社を辞めて転職したいです。どんな手続きが必要ですか？**
A: 特定活動（告示第46号）での転職は、新しい雇用先を指定する新たな「在留資格変更許可申請」が必要です（ISA公式）。技人国の転職時の14日届出とは異なり、申請・許可を受けてから新しい会社で働き始める必要があります。
