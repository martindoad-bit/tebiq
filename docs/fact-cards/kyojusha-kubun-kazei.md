## current_effective_fact

外国人の日本での所得税は、課税上の「居住者・非永住者・非居住者」の区分によって課税範囲が大きく異なる。

**課税区分の定義と課税範囲（{{TODAY_ISO}} 現在）：**

| 区分 | 定義 | 課税範囲 |
|------|------|---------|
| **居住者** | 日本に住所あり、または1年以上継続居所あり | 全世界所得 |
| **非永住者** | 居住者のうち日本国籍なし + 過去10年のうち日本在住5年以下の外国人 | 国内源泉所得 + 国外所得のうち国内送金分 |
| **非居住者** | 居住者以外（日本滞在1年未満の者等） | 国内源泉所得のみ |

### 来日外国人の典型的な移行パターン（ai推定）

```
来日（就労・留学）
→ 住民票登録・生活本拠確立 → 居住者（非永住者）認定
   ↓
   日本在住通算5年を超えると
→ 一般居住者（全世界所得課税）へ移行
```

### 非永住者の課税範囲の注意点

> 「国外源泉所得で日本国内において支払われ、または**日本国内に送金されたもの**」は課税対象（国税庁 No.2010）

→ 海外の銀行口座に受け取った所得でも、日本の口座に送金した分は課税対象となる。

## exceptions_or_transition

- **「居住者」は在留資格とは別の概念**：在留資格の「永住者」と課税上の「居住者」は異なる。在留資格「技術・人文知識・国際業務」でも来日後に居住者（非永住者）として全世界所得課税される
- **日本国籍者は非永住者にならない**：非永住者の要件に「日本国籍がないこと」が含まれるため、日本国籍取得後は常に一般居住者
- **短期出張・観光で来日**：生活の本拠が日本になく、1年未満の滞在 → 非居住者
- **租税条約**：日本と締結国の間の租税条約が優先適用される場合があり、実際の課税は条約内容による（税理士相談要）

## must_say

- 居住者の定義：日本に住所あり、または「1年以上継続居所」あり→全世界所得課税
- 非永住者：「過去10年のうち日本在住5年以下」の外国人 → 国外所得の国内送金分も課税
- 非居住者：国内源泉所得のみ課税
- 在留資格「永住者」と課税上の「居住者」は別の概念（混同注意）

## must_not_say

- 「日本に来たばかりは非居住者」（住所・生活本拠の確立次第で即座に居住者になる）
- 具体的な税額を断言する（租税条約・個別所得状況による）

## qa_cases

**Q1: 日本に来て何年で海外の収入にも税金がかかりますか？**
A: 日本に生活の本拠（住所）を確立した時点で「居住者」となり、過去10年のうち日本在住通算5年以下の外国人は「非永住者」として扱われます。非永住者の間は、国外所得でも日本国内に**送金した分は課税対象**です。日本在住5年を超えると全世界所得が課税対象となります（ai推定 — `hieijusha_5nen_transition`確認要）。

**Q2: 日本滞在1年未満の外国人は課税されますか？**
A: 日本に住所・生活本拠がなく、継続1年未満の滞在の場合は「非居住者」として日本国内源泉所得のみが課税対象です。ただし生活の本拠が日本に移った時点で居住者となるため、来日後すぐに居住者認定される場合もあります。

**Q3: 在留資格「永住者」になったら課税範囲が変わりますか？**
A: 在留資格の「永住者」と課税上の「居住者・非永住者」は別の概念です。在留資格の永住者取得によって課税区分は変わりません。課税上の非永住者から全世界所得課税（一般居住者）への移行は、過去10年のうち日本在住通算5年を超えた時点で生じます（ai推定・`hieijusha_5nen_transition`確認要）。
