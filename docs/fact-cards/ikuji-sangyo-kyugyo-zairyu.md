## current_effective_fact

外国人（就労資格保有者）は日本人と同様に産前産後休業・育児休業を取得できる（労働基準法・育児・介護休業法は国籍差別を禁止）。雇用保険に加入していれば育児休業給付金（開始から6か月間は67%、以降50%）を受給できる。育休中の在留資格取消リスクについては、ISA公式テキストに「正当な理由」の明文規定がないため注意が必要。

**産前産後休業・育児休業の法定期間（{{TODAY_ISO}} 現在）：**

| 種別 | 期間 | 法的根拠 |
|------|------|---------|
| 産前休業 | 出産予定日前**6週間**（多胎妊娠は14週間）　本人請求制 | 労働基準法第65条第1項 |
| 産後休業 | 出産後**8週間**（産後6週間後は本人請求+医師承認で就業可） | 労働基準法第65条第2項 |
| 育児休業（基本） | 子が**1歳になる前日**まで | 育児・介護休業法 |
| 育児休業（延長①） | 保育所入所不可等の場合 → **1歳6か月**まで | 育児・介護休業法 |
| 育児休業（延長②） | さらに保育所入所不可の場合 → **2歳**まで | 育児・介護休業法 |

**育児休業給付金 支給率（{{TODAY_ISO}} 現在）：**

**受給要件（雇用保険）：**

## exceptions_or_transition

- **「正当な理由」の不確実性**：育休が入管法第22条の4第1項第6号の「正当な理由」に当たるかのISA公式文書は未公開。実務上は認められることが多いが、育休が3か月を大幅に超える場合はISAへの事前確認を推奨（ai推定・`ikukyu_seito_na_riyu_shonin`確認要）
- **雇用保険に未加入の場合**：雇用保険に加入していない（週20時間未満・31日未満雇用等）場合は育児休業給付金を受給できない
- **育休1年未満の雇用者は除外可能**：育児・介護休業法第6条により、雇用開始から1年未満の労働者は労使協定で育休取得対象から除外できる

## qa_cases

**Q1: 技人国ビザで在留中ですが、育休を取ると在留資格が取り消されますか？**
A: 在留資格取消しの3か月ルール（入管法第22条の4第1項第6号）は「正当な理由がある場合を除く」とされており、育児・介護休業法に基づく育休は実務上「正当な理由」として解釈されることが多いとされています（ai推定）。ただし、ISA公式テキストに就労資格の3か月ルールに対する「正当な理由」の具体例は公式には掲載されていません。育休が3か月を超える場合はISAまたは行政書士への確認をお勧めします。

**Q2: 育児休業給付金は外国人でももらえますか？**
A: はい。雇用保険法は国籍・在留資格による除外規定を設けていないため、雇用保険に加入している外国人就労者も育児休業給付金を受給できます。受給要件は育休開始日前2年間に賃金支払基礎日数11日以上の月が12か月以上あることです。支給率は育休開始から180日目まで67%、181日目以降は50%です（MHLW公式）。

**Q3: 育休中に在留期間が切れたらどうすればいいですか？**
A: 育休中であっても在留期限は変わりません。在留期限が近づいたら通常通り更新申請が必要です。育休中であることで更新が拒否されるわけではありませんが（ai推定）、申請には雇用主の協力が必要な書類もありますので早めに準備してください。
