## current_effective_fact

外国人の子どもは公立小学校・中学校に入学を申し出る権利があり、授業料は無償。ただし就学義務（子どもを学校に通わせる法的義務）は日本国民の保護者にのみ課されている。

**外国人の子どもの就学（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 入学の権利 | 公立小・中学校への入学申し出の権利あり |
| 授業料 | **無償**（外国人の子どもも） |
| 就学義務 | 日本国民（保護者）のみに課される |
| 日本語指導 | 「特別の教育課程」による日本語指導あり |
| 手続き先 | 居住する市区町村の教育委員会 |

## exceptions_or_transition

- **高校・大学**：都道府県の公立高校への入学は入試合格が必要（ai推定）。大学は各校の入試制度による
- **外国人学校**：外国人学校（日本語以外で教育する学校）に通わせることも可能
- **不就学の問題**：言語・経済的な壁で就学しない外国人の子どもへの支援が課題（自治体が訪問支援等を実施）

## qa_cases

**Q1: 子どもを日本の小学校に入れたいです。どうすればいいですか？**
A: 居住する市区町村の教育委員会に問い合わせて入学手続きを行ってください。外国人の子どもも公立小学校に入学する権利があり、授業料は無償です。日本語指導が必要な場合は「特別の教育課程」による支援が受けられます。

**Q2: 子どもが日本語を話せません。学校に入れますか？**
A: はい。日本語が話せなくても入学できます。日本語指導が必要な外国人児童生徒への「特別の教育課程」が制度化されており、日本語指導を受けながら学校生活に適応できる支援体制があります。
