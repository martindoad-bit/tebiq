## current_effective_fact

外国人が日本の銀行口座を開設するには、在留カード（または特別永住者証明書）と、マイナンバーの提示が必要。短期滞在（90日以内）では原則として口座開設ができない。

**口座開設に必要な書類（{{TODAY_ISO}} 現在）：**

| 書類 | 内容 |
|------|------|
| 在留カード（必須） | 現住所記載・有効期限内のもの |
| マイナンバー | 個人番号の告知義務あり（2016年以降） |
| 印鑑 | 銀行によっては不要（サインのみも可） |
| その他 | 銀行によっては公共料金の領収書等の追加書類を要求 |

**在留資格別の口座開設可否：**

| 在留資格 | 口座開設 |
|---------|---------|
| 就労・留学・家族滞在等（中長期） | **原則可** |
| 短期滞在（90日以内） | **原則不可** |
| 特別永住者 | **可** |

## exceptions_or_transition

- **ネット銀行の活用**：PayPay銀行・住信SBIネット銀行等はスマホアプリで在留カードをスキャンするだけで口座開設できる場合が多い（ai推定）
- **銀行ごとの対応差**：外国人向けサービスの充実度は銀行によって異なる。事前確認を推奨
- **マイナンバーカード未取得の場合**：マイナンバー通知カード（または住民票）でマイナンバーを告知可能

## qa_cases

**Q1: 技人国ビザで在留中です。銀行口座を開きたいのですが。**
A: 在留カードとマイナンバーがあれば、中長期在留者は原則として口座開設が可能です。必要書類：在留カード・マイナンバー（通知カードまたはマイナンバーカード）・印鑑（銀行による）。各銀行の外国人向け窓口または公式サイトで確認してください。

**Q2: 短期滞在ビザで入国しました。銀行口座は作れますか？**
A: 短期滞在（90日以内）では原則として日本の銀行口座を開設することはできません。就労・留学等の中長期在留資格に変更した後に申請してください。
