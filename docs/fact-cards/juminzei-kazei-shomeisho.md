## current_effective_fact

日本に住所のある外国人は、日本人と同様に**住民税**（都道府県民税＋市区町村民税）の課税対象となる。住民税は**前年課税**が原則であり、当年度の住民税は前年（1月1日〜12月31日）の所得に基づいて計算・課税される。

**住民税の基本（{{TODAY_ISO}} 現在）：**
- **課税対象**：毎年1月1日現在、日本国内に住所がある者（外国人含む）
- **課税基準**：前年の所得（1月1日〜12月31日）に基づく（前年課税原則）
- **税の種類**：都道府県民税 + 市区町村民税（合わせて「住民税」）
- **納付方法**：
  - 特別徴収：給与・年金から天引き（雇用主が代行）
  - 普通徴収：市区町村から届く納付書で自分で納付

**証明書の種類と取得先：**

| 証明書の種類 | 内容 | 発行機関 |
|------------|------|---------|
| 住民税の課税証明書 | 所得額・住民税課税額の証明 | 住民票のある市区町村役場 |
| 住民税の納税証明書 | 住民税の納付実績の証明 | 住民票のある市区町村役場 |
| 国税の納税証明書 その1 | 所得税等の納付すべき税額・納付済額 | 税務署 / e-Tax |
| 国税の納税証明書 その2 | 所得金額の証明 | 税務署 / e-Tax |
| 国税の納税証明書 その3 | 未納の税額がないことの証明 | 税務署 / e-Tax |
| 国税の納税証明書 その4 | 滞納処分を受けたことがない証明 | 税務署 / e-Tax |

**在留申請との関係（ai推定）：**
- 在留更新申請：在留資格によっては税証明書の提出が必要な場合がある
- 永住申請：所得・納税証明書が必要とされる場合がある（具体的な年数・書類種別はeijuu-tsujou-route参照）

## exceptions_or_transition

- **1月2日以降に出国した場合**：1月1日時点で日本に住所があれば、その年の住民税が課税される（年途中出国でも課税）
- **当年度途中に来日した場合**：来日年の翌年に前年所得として課税（来日直後の年は前年の日本国内所得が少ない or なし→非課税または低額の場合が多い）
- **無収入・低所得の場合**：住民税が非課税となる場合がある（「非課税証明書」の発行が可能）
- **確定申告と住民税の関係**：会社員は給与年末調整で完結する場合が多い；自営業・フリーランスは確定申告が住民税の基礎データとなる
- **証明書の有効期限**：在留申請では発行から3か月以内のものが求められる場合がある（ai推定）

- 外国人も日本に住所があれば住民税の課税対象
- 住民税は**前年（1月1日〜12月31日）の所得**に対して当年に課税される（前年課税原則）
- 住民税の課税証明書・納税証明書は**住民票のある市区町村役場**で取得
- 国税の納税証明書（その1〜その4）は**税務署またはe-Tax**で取得

- 「外国人は住民税を払わなくていい」（外国人も課税対象）
- 「今年の収入に対して今年課税される」（前年課税が原則）
- 「住民税の課税証明書は税務署で取得できる」（住民税証明書は市区町村役場）
- 「住民税を未払いにしても在留更新に影響しない」（影響する可能性あり・ai推定）
- 「更新・永住申請に必要な税証明書の年数はすべての在留資格で同じ」（資格ごとに異なる）

## qa_cases

**Q1: 外国人も住民税を払う必要がありますか？**
A: はい。1月1日現在、日本国内に住所がある方（外国人を含む）は住民税の課税対象です。日本人と同じルールが適用されます。

**Q2: 住民税の課税証明書はどこで取得できますか？**
A: 住民票のある市区町村役場（区役所・市役所・町村役場）で取得できます。国税（所得税等）の納税証明書（その1〜その4）は税務署またはe-Taxで取得します。

**Q3: 在留更新に住民税の証明書は必要ですか？**
A: 在留資格によっては、住民税の課税証明書・納税証明書の提出が求められる場合があります（ai推定）。必要な年数・書類の種類は在留資格ごとに異なります。詳しくは出入国在留管理庁の各在留資格の手続きページを確認するか、申請前に入管窓口でご確認ください。

**Q4: 住民税を滞納しています。在留更新に影響しますか？**
A: 住民税の滞納・未納は在留更新審査に影響する可能性があります（ai推定）。早急に市区町村の税務窓口に相談し、納付または分割払いの手続きを行うことを推奨します。
