## current_effective_fact

従来の健康保険証は2024年12月2日以降新規発行が停止された。現在は「マイナ保険証」（マイナンバーカードに健康保険証利用登録をしたもの）が中心となり、マイナ保険証を持たない人には「資格確認書」が無申請・無償で交付される。

**マイナ保険証 制度移行まとめ（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 従来の健康保険証 | 2024年12月2日以降 **新規発行停止** |
| マイナ保険証 | マイナンバーカード ＋ 健康保険証利用登録 |
| 使用方法 | 医療機関・薬局の顔認証付きカードリーダーにかざす |
| カードがない場合 | **資格確認書**（無申請・無償）で対応 |
| スマホ利用 | 2026年9月〜対応医療機関で開始予定 |

**外国人のマイナ保険証：注意点（{{TODAY_ISO}} 現在）：**

| 在留資格 | マイナンバーカード有効期限 | マイナ保険証への影響 |
|---------|------------------------|-------------------|
| 永住者・高度専門職2号・特別永住者 | 発行日から10年 | 長期利用可能 |
| その他（技人国・留学等） | **在留期間の満了日** | ⚠️ 在留更新後は要確認 |

⚠️ 技人国・留学等のマイナンバーカードは在留期間満了でカードが失効 → マイナ保険証も失効する可能性（ai推定・`mynumber_card_renewal_and_maina_hoken`確認要）

**マイナ保険証がない場合の対応（{{TODAY_ISO}} 現在）：**

- 「マイナ保険証を保有していない方等に、当分の間、**無申請で無償で**資格確認書が交付される」（MHLW公式）
- 資格確認書で従来の健康保険証と同様に医療機関を受診できる（ai推定）

## exceptions_or_transition

- **2024年12月2日の意味**：この日以降、雇用主や健保組合が従来型の健康保険証を新規発行できなくなった。既発行のカードは有効期限まで使用可能（経過措置・`existing_card_expiry_transition`確認要）
- **電子証明書失効後の3か月猶予**：マイナンバーカードの電子証明書有効期限が切れていても、失効後3か月間はマイナ保険証として利用可能（MHLW公式）
- **在留期間更新と連動失効**：技人国等の在留者はマイナンバーカードの有効期限が在留期間満了日 → 在留更新後、カードの券面変更 + マイナ保険証継続の確認が必要（ai推定）

## qa_cases

**Q1: 従来の健康保険証はいつ廃止されましたか？今でも使えますか？**
A: 従来の健康保険証は2024年12月2日（令和6年12月2日）以降、新規発行が停止されました。すでに発行済みのものは有効期限まで使用できます（経過措置の詳細は保険者にご確認ください）。

**Q2: マイナンバーカードを持っていない場合、病院はどうすればいいですか？**
A: マイナ保険証（マイナンバーカードの健康保険証利用登録）を持っていない方には、「資格確認書」が無申請・無償で交付されます（MHLW公式）。資格確認書を持参することで医療機関を受診できます。

**Q3: 在留期間を更新したらマイナ保険証はどうなりますか？**
A: 技人国・留学等の在留資格のマイナンバーカードは在留期間満了日が有効期限です。在留期間更新後は市区町村でカードの「券面記載事項変更」手続きが必要です。マイナ保険証としての利用継続については、手続き後に確認が必要です（ai推定・`mynumber_card_renewal_and_maina_hoken`確認要）。
