## current_effective_fact

外国人労働者も国籍・在留資格にかかわらず、未払い賃金・残業代を請求する権利がある。労働基準監督署に申告することで、事業主への調査・是正勧告が行われる。

**未払い賃金の請求の流れ：**
1. 証拠を保全（給与明細・タイムカード・雇用契約書等）
2. 会社に請求（直接交渉）
3. 解決しない場合 → **労働基準監督署に申告**
4. 法的解決が必要な場合 → 弁護士・法テラスへ

**重要な情報：**

| 項目 | 内容 |
|------|------|
| 時効 | **3年**（2020年4月1日以降の賃金） |
| 申告先 | 事業所所在地を管轄する**労働基準監督署** |
| 倒産時 | 未払い賃金立替払い制度（国が最大**80%**を立替払い） |

## exceptions_or_transition

- **時効**：2020年4月1日以降の賃金は時効3年（それ以前の賃金は2年の場合がある）
- **倒産時の立替払い**：退職後6か月以内に破産手続開始等が申立てられた場合が対象。申請先は独立行政法人 労働者健康安全機構（ai推定）
- **不法就労者**：不法就労・資格外就労の状況でも賃金請求権はある（労働基準法は全労働者に適用）（ai推定）

## qa_cases

**Q1: 2か月分の給料が払われていません。どうすればいいですか？**
A: まず証拠（給与明細・雇用契約書・タイムカード等）を保全してください。会社に請求しても支払われない場合は、事業所の所在地を管轄する**労働基準監督署**に申告できます（無料）。監督署が調査・是正勧告を行います。

**Q2: 未払い賃金の時効は何年ですか？**
A: 2020年4月1日以降の賃金については**3年**の時効があります（以前は2年）。時効が来る前に請求・申告することが重要です。

**Q3: 会社が倒産して給料が払われません。どうすればいいですか？**
A: 「未払い賃金立替払い制度」があります。事業主が倒産した場合（退職後6か月以内に破産手続開始等の申立てがあった場合）に、国（独立行政法人 労働者健康安全機構）が未払い賃金の最大**80%**を立替払いしてくれます。労働基準監督署または最寄りの労働局にご相談ください。
