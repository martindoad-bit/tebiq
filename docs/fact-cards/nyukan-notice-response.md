## current_effective_fact

入管から通知を受け取った場合、まず「何の通知か」を確認することが最初のステップ。

**通知対応の基本フレームワーク（{{TODAY_ISO}} 現在・ai推定）：**

1. **通知の表題を確認する**
   - 追加資料提出通知書、補完通知、照会、不許可通知等、表題によって求められる対応が異なる
   - 表題が判別できない場合、申請窓口（地方出入国在留管理局）への問い合わせが有効

2. **期限を確認する**
   - 通知書に記載された期限を確認する
   - 期限内に対応できない場合は、早めに入管窓口に相談することを推奨
   - **期限の記載がない通知の期限をこちらが推測して伝えることはしない**

3. **求められている対応を確認する**
   - 何の書類・情報の提出を求めているかを確認する
   - 提出先・提出方法（窓口持参・郵送・オンライン）を確認する

4. **専門家への相談を検討する**
   - 内容が複雑・不明な場合は行政書士・弁護士への相談を推奨

**重要な区別（ai推定・`refusal_vs_inquiry_distinction`確認要）：**
- **追加資料請求 ≠ 不許可通知**：資料提出を求められることは、申請に問題があることを意味するが、即座の不許可を意味するわけではない
- **補完通知 ≠ 審査終了**：補完・照会への対応後に審査が続く

## exceptions_or_transition

- **通知の内容が外国語（日本語のみ）で理解できない場合**：地方出入国在留管理局の外国人案内窓口・支援センターへの相談が有効（ai推定）
- **期限を過ぎた場合（ai推定）**：期限超過後も早急に対応することが重要。期限超過の影響は通知の種類・入管局の判断による（断定不可）
- **不許可通知の場合**：追加資料請求とは別の問題であり、審査請求・再申請の検討が必要（詳細はPriority Backlog — DOMAIN確認要）
- **申請取次者（行政書士・弁護士）が申請を代行している場合**：通知が代理人に届いている場合があるため確認が必要

## qa_cases

**Q1: 入管から手紙が届きました。どうすればいいですか？**
A: まず封書の**表題（タイトル）**を確認してください。追加資料提出通知・補完通知・照会・不許可通知等、種類によって対応が異なります。次に、記載されている**期限・提出先・提出方法**を確認してください。内容が日本語で理解できない場合、地方出入国在留管理局の窓口に問い合わせるか、行政書士・弁護士への相談を推奨します（ai推定・`notice_types_and_deadlines`確認要）。

**Q2: 入管から「追加資料を提出してください」という通知が来ました。これは不許可ですか？**
A: 追加資料請求は不許可通知ではありません。審査中に確認が必要な書類を求めているものです。通知書に記載された期限・提出先・必要書類を確認の上、期限内に対応することが重要です（ai推定・`refusal_vs_inquiry_distinction`確認要）。

**Q3: 入管通知の期限が書いてありませんでした。いつまでに返答すればいいですか？**
A: 通知書に期限の記載がない場合、弊サービスからは期限を特定できません。申請を行った地方出入国在留管理局に直接問い合わせることを推奨します（ai推定）。
