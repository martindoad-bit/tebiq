## current_effective_fact

日本帰化（国籍取得）は法務大臣の許可が必要（国籍法第4条）。普通帰化は引き続き5年以上の日本在住・18歳以上・素行善良・生計維持等の要件を充足が必要。日本人配偶者は要件が緩和（特別帰化）。二重国籍は原則不可。申請は法務局窓口のみ、手数料無料、処理期間は実務的に1年前後。

**普通帰化の要件一覧（国籍法第5条）（{{TODAY_ISO}} 現在）：**

| 要件 | 内容 |
|------|------|
| 住所要件 | **引き続き5年以上**日本に住所を有すること |
| 年齢要件 | **18歳以上**（本国法でも成人）（2022年4月改正後） |
| 素行要件 | 犯罪歴・納税状況・社会的迷惑を総合考慮 |
| 生計要件 | 「生計を一にする親族単位」で生活困窮なし |
| 国籍要件 | 帰化時に**原国籍を喪失**すること（二重国籍不可） |
| 思想要件 | 日本国憲法・政府を暴力で破壊する意図がないこと |

**特別帰化（住所要件の緩和）：**

| 根拠条文 | 対象 | 緩和内容 |
|---------|------|---------|
| 国籍法第6条第1号 | 日本人の配偶者で**婚姻3年以上**かつ**在住1年以上** | 住所要件 **5年→緩和**（婚姻継続3年+在住1年で足りる） |
| 国籍法第6条第2号 | 日本人の配偶者で**婚姻後3年以上**日本在住 | 住所要件 5年→**3年**に短縮 |
| 国籍法第8条第5号 | **引き続き10年以上居所**を有する者 | 住所要件（居所でも可）の緩和 |

**申請手続き概要（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 申請場所 | 住所地管轄の**法務局または地方法務局**（本人直接出頭） |
| 手数料 | **無料** |
| 郵送・オンライン | **不可**（窓口申請のみ） |
| 標準処理期間 | 公式設定なし（**実務的に1年前後**）（ai推定） |
| 効力発生 | **官報への告示日**が国籍取得日（国籍法第10条） |
| 15歳未満 | 法定代理人が申請可 |

## exceptions_or_transition

- **二重国籍の例外**：外国法により国籍離脱が不可能な場合は国籍法第5条第2項により免除。ただし帰化後に外国国籍を再取得すると日本国籍を失う（国籍法第11条 — ai推定）
- **2022年4月改正**：民法改正により成人年齢が20歳→18歳。帰化の年齢要件が連動変更（公式）
- **永住者と帰化の選択**：永住者も帰化も「長期在留の選択肢」だが別制度。永住者は外国籍を保持したまま日本に住み続けられる。帰化は日本国籍を取得し外国籍を手放す（原則）
- **事前相談の重要性**：書類が揃っていても許可は保証されない。申請前に法務局に事前相談することが実質的に必須（ai推定）

## qa_cases

**Q1: 帰化するには何年日本に住む必要がありますか？**
A: 普通帰化の住所要件は「引き続き5年以上日本に住んでいること」です（国籍法第5条・MOJ公式）。ただし日本人の配偶者の場合は国籍法第6条・第7条の特別帰化により要件が緩和される場合があります。

**Q2: 帰化したら元の国籍も保持できますか（二重国籍）？**
A: 原則として、帰化により元の国籍を喪失することが条件となります。日本は二重国籍を原則認めていません（国籍法第5条）。外国法により国籍離脱が不可能な場合は例外的な取扱いがありますが、詳細は法務局にご確認ください。

**Q3: 帰化申請はどこでできますか？費用はかかりますか？**
A: 住所地を管轄する法務局または地方法務局に本人が直接出頭して申請します。郵送・オンライン申請はできません。手数料は**無料**です（MOJ公式）。審査期間は公式の標準期間の設定がなく、実務的には1年前後かかることが多いとされています（ai推定）。
