## current_effective_fact

日本の会社員は毎月の給与から社会保険料（健康保険・厚生年金・介護保険・雇用保険）と所得税・住民税が天引きされる。外国人労働者も日本人と同様に天引きの対象。

**給与天引きの主な内訳（本人負担分・2024年度の概算）：**

| 項目 | 本人負担率（概算） | 備考 |
|------|----------------|------|
| 健康保険料 | 約**5%** | 標準報酬月額の約10%を労使折半 |
| 厚生年金保険料 | **9.15%** | 18.3%を労使折半 |
| 介護保険料（40歳以上） | 約**0.91%** | 1.82%を労使折半 |
| 雇用保険料 | **0.6%** | 賃金に対して（一般事業） |
| 所得税 | 課税所得の**5〜45%** | 源泉徴収（年末調整で精算） |
| 住民税 | 前年所得の約**10%** | 翌年6月〜（1年目は天引きなし） |

**月収30万円の場合の概算（2年目以降・標準的なケース）：**
- 健康保険料：約15,000円
- 厚生年金：約27,450円
- 雇用保険：約1,800円
- 所得税：約8,000円（概算）
- 住民税：約25,000円（前年所得による）
- **合計：約7〜8万円の天引き → 手取り22〜23万円**

## exceptions_or_transition

- **入社1年目の住民税**：住民税は前年所得に基づくため、入社1年目（日本初就労）は翌年6月まで天引きなし
- **会社規模・組合健保**：加入する健康保険組合によって健康保険料率が異なる（協会けんぽ以外の場合）
- **配偶者・扶養控除**：扶養家族がいる場合は所得税・住民税が軽減される（控除の適用）

## must_say

- 健康保険（本人約5%）・厚生年金（9.15%）・雇用保険（0.6%）が主な社会保険料の天引き
- 所得税（源泉徴収）・住民税（翌年6月〜）も天引き
- 保険料率は毎年度改定 → 最新率は厚生労働省公式で確認要

## must_not_say

- 「天引きは不当」（法律に基づく義務的な控除）
- 「手取りは月収の50%以下」（通常は75〜80%程度が手取り）

## qa_cases

**Q1: 毎月の給与から色々引かれています。何の項目がありますか？**
A: 主に①健康保険料（標準報酬の約5%）②厚生年金保険料（9.15%）③雇用保険料（0.6%）④所得税（源泉徴収）⑤住民税（翌年6月から・前年所得の約10%）が天引きされています。給与明細の「控除」欄で各項目を確認できます。

**Q2: 月収30万円なら手取りはいくらですか？**
A: 標準的なケースで概算すると、社会保険料が約4.5万円・所得税約8千円・住民税約2.5万円（2年目以降）で合計約7〜8万円が天引きされ、手取りは概ね22〜23万円程度です（扶養・各種控除によって異なります）。
