## current_effective_fact

高度専門職1号は、高度な知識・技術を持つ外国人がポイント計算で70点以上を取得することで認定される在留資格。70点以上を3年維持すれば永住申請が可能（通常10年を3年に短縮）、80点以上なら1年で永住申請が可能。

**ポイント計算の構造（{{TODAY_ISO}} 現在）：**

| 項目 | 主な加算要素 |
|------|------------|
| **学歴** | 博士（30点）> 修士（20点）> 学士（10点）＋特定大学加算（10点） |
| **職歴** | 経験年数に応じて加算（10年以上が最高） |
| **年収** | 日本での見込み年収（高いほど加算・年齢補正あり） |
| **年齢** | 若いほど加算（30歳未満が最高） |
| **日本語能力** | JLPT N1（15点）・N2（10点）等 |
| **特例加算** | 日本の大学卒業・研究実績・特許取得等 |

**永住申請の短縮特例：**
- **70点以上** × 3年継続 → 永住申請可（kodo-senmon-shoku-eijuu 参照）
- **80点以上** × 1年継続 → 永住申請可
