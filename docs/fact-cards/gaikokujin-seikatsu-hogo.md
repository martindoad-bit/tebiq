## current_effective_fact

外国人には生活保護法上の受給権はないが、永住者・定住者・日本人配偶者等の一部在留資格者には行政的措置として生活保護に準じた保護が行われる。就労ビザ・留学・家族滞在等の外国人は対象外。

**在留資格別の対応（{{TODAY_ISO}} 現在）：**

| 在留資格 | 生活保護への対応 |
|---------|--------------|
| 永住者・特別永住者 | 行政的措置として**準用あり** |
| 定住者 | 行政的措置として**準用あり** |
| 日本人・永住者の配偶者等 | 行政的措置として**準用あり** |
| 就労ビザ（技人国・特定技能等） | **対象外**（就労による自立が前提） |
| 留学・家族滞在 | **対象外** |

## exceptions_or_transition

- **難民認定者（条約難民）**：生活保護の準用が認められる場合あり（ai推定）
- **緊急の場合**：在留資格に関わらず、緊急の医療費等は行政機関が対応する場合あり
- **生活困窮者支援制度の活用**：就労ビザ等で生活保護対象外の外国人でも、住居確保給付金・生活困窮者自立支援制度等を活用できる場合あり
- **自立相談窓口**：市区町村の自立相談支援機関（生活困窮者向け）に相談可能

## qa_cases

**Q1: 永住者です。生活保護を申請できますか？**
A: 外国人には生活保護法上の受給権はありませんが、永住者には厚生労働省の通知に基づく行政的措置として生活保護に準じた保護が行われる場合があります。市区町村の福祉事務所（生活保護担当）に相談してください。

**Q2: 技人国ビザで在留中です。生活が苦しいです。支援はありますか？**
A: 就労ビザ（技人国等）では生活保護の行政的措置の対象外です。ただし、住居確保給付金（離職等の場合）や生活困窮者自立支援制度を活用できる場合があります。市区町村の自立相談支援機関にご相談ください。
