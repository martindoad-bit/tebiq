## current_effective_fact

健康保険（協会けんぽ・組合健保等）の被保険者が業務外の病気・ケガで仕事を休んだ場合、傷病手当金が支給される。外国人も健康保険被保険者であれば受給対象。

**傷病手当金の概要（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 対象 | 業務外の病気・ケガで仕事に就けない健康保険被保険者 |
| 待期期間 | 連続する3日間（この3日間は支給されない） |
| 支給開始 | 4日目から |
| 支給額 | 標準報酬月額の平均の**約67%**（3分の2）/日 |
| 支給期間 | 支給開始日から通算して**1年6か月** |

**支給対象外：**
- 業務上・通勤中のケガ・病気（→労災保険の対象）
- 国民健康保険の被保険者（協会けんぽ・組合健保の加入者が対象）

## exceptions_or_transition

- **2022年1月1日改正**：「同一の傷病について支給期間が通算1年6か月」に変更。間に仕事に復帰した期間があっても、受給した日数が合計1年6か月を超えると支給終了
- **退職後の継続受給（ai推定）**：退職前に1年以上継続して健康保険に加入しており、退職時に傷病手当金を受給中の場合は、退職後も引き続き受給できる場合がある
- **会社から報酬を受けた場合**：休業中でも報酬を受けた場合は傷病手当金との差額のみ支給

## qa_cases

**Q1: 病気で1か月仕事を休むことになりました。何か手当はもらえますか？**
A: 健康保険（会社の健康保険・協会けんぽ・組合健保等）に加入していれば、業務外の病気で仕事に就けない場合に「傷病手当金」が支給されます。最初の3日間は待期期間として支給されず、4日目から支給されます。支給額は標準報酬月額の平均の約67%（1日あたり）です。

**Q2: 傷病手当金はいつまでもらえますか？**
A: 支給開始日から通算して1年6か月です（2022年1月1日以降）。途中で仕事に復帰しても、復帰後に再び休業した場合は通算された日数が1年6か月に達するまで受給できます。

**Q3: 外国人ですが、傷病手当金はもらえますか？**
A: はい。健康保険の被保険者であれば、国籍・在留資格の種類にかかわらず傷病手当金の受給対象です。申請は会社（事業主）経由で加入している健康保険に行います。
