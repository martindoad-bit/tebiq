## current_effective_fact

日本に居住する外国人（居住者・非永住者）も、一定の条件を満たす場合は日本の所得税の確定申告が必要。年末調整だけで申告が完結しないケースが多い（副業・複数箇所給与等）。

**確定申告が必要なケース（{{TODAY_ISO}} 現在）：**

| ケース | 確定申告の要否 |
|--------|--------------|
| 給与収入2,000万円超 | **申告必要** |
| 給与1か所受取 + 他所得が20万円超（副業等） | **申告必要** |
| 複数の事業所から給与を受ける | **申告必要** |
| 給与1か所受取 + 他所得20万円以下 + 源泉徴収あり | 申告不要（年末調整で完結） |

### 申告期限・手続き基本情報

| 項目 | 内容 |
|------|------|
| 申告受付期間 | 翌年2月16日〜3月15日 |
| 納付期限 | 翌年3月15日 |
| 申告先 | 住所地の所轄税務署 |

## exceptions_or_transition

- **副業所得20万円ライン**：給与所得者の「他所得20万円以下申告不要」は所得税の話。住民税は別途市区町村に申告が必要な場合がある（ai推定）
- **外国人居住者の海外所得**：居住者は全世界所得課税。非永住者は国外所得のうち国内送金分も課税対象（`kyojusha-kubun-kazei`カード参照）
- **e-Tax・スマホ申告**：税務署への持参だけでなくオンライン申告も可能（ai推定 — NTA推奨）
- **申告期限を過ぎた場合**：無申告加算税・延滞税の対象となる可能性がある（ai推定）
- **還付申告**：払い過ぎた税金の還付のための申告は1月1日から受付（申告義務がない場合でも可能）

## qa_cases

**Q1: 外国人でも日本で確定申告が必要ですか？**
A: はい、日本に居住する外国人（居住者・非永住者）も日本人と同様に所得税の確定申告義務があります。ただし、給与1か所受取 + 給与以外の所得20万円以下 + 全額源泉徴収の場合は年末調整だけで完結し確定申告不要です。課税区分（居住者・非永住者・非居住者）については `kyojusha-kubun-kazei` カードをご参照ください。

**Q2: 副業の収入がいくら以上だと確定申告が必要ですか？**
A: 給与所得者の場合、給与所得・退職所得以外の所得（副業・フリーランス等）が**年間20万円を超える**場合は確定申告が必要です。20万円以下であれば所得税の確定申告は不要ですが、住民税については別途申告が必要な場合があります（税務署または市区町村で確認要）。

**Q3: 確定申告の期限はいつですか？**
A: 申告受付期間は**翌年2月16日〜3月15日**、納付期限は**翌年3月15日**です。申告先は住所地の所轄税務署です（e-Taxによるオンライン申告も可能）。
