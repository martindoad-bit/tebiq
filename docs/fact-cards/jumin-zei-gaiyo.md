## current_effective_fact

住民税（個人住民税）は、前年の所得に基づいて翌年に課税される地方税。1月1日時点で住民票のある市区町村が課税する。**来日初年度は前年の日本での所得がないため、翌年分の住民税が非課税または少額となる**（ai推定）。

**住民税の基本情報（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 所得割税率 | **10%**（道府県4%＋市区町村6%）|
| 均等割（年額） | **5,000円**（2024年度以降・森林環境税含む）|
| 課税基準年 | **前年（1月1日〜12月31日）の所得** |
| 課税自治体 | **1月1日時点の住所地**の市区町村 |
| 給与所得者の徴収 | 特別徴収（**6月〜翌5月の12回** 給与天引き） |

### 来日初年度の住民税について（ai推定・`first_year_non_taxable_mechanism`確認要）

**なぜ来日1年目は住民税がかからないのか：**

住民税は「前年の所得」に基づいて課税される後払い制度。来日した年の翌年1月1日時点で日本に住民票があっても、課税対象となる前年の日本での所得がゼロ（または非常に少ない）場合、住民税は課税されないか少額となる。

```
例：2026年4月来日 → 2026年4月〜12月は日本で課税所得が発生
    → 2027年1月1日時点で住民票あり → 2027年度の住民税が課税
    → 2025年（来日前）の日本での所得はゼロ → 2026年度の住民税は非課税
```

### 非課税限度額（東京都23区基準・`municipal_variation`確認要）

| ケース | 非課税条件（所得額） |
|--------|------------------|
| 単身（扶養なし） | 所得45万円以下 |
| 扶養あり | 35万円×（本人＋扶養人数）＋31万円 以下 |

## exceptions_or_transition

- **退職後・転職時の特別徴収→普通徴収の切替え**：退職後は6月〜5月の残額を普通徴収（自分で納付）に切り替えるか、次の勤務先で引き継ぐ。退職時に一括徴収される場合もある（ai推定）
- **海外転出**：住民票を抜いて海外転出した場合、その年度の住民税は1月1日時点の在住自治体が課税（すでに課税が始まっている場合は納付義務あり）
- **国民健康保険料との関係**：国保保険料は住民税の課税情報をもとに計算される自治体が多い（ai推定）
- **均等割の非課税特例（2024年）**：2024年度に実施された定額減税（1人あたり3万円の特別税額控除）などの特例は年度によって異なる（ai推定）

## qa_cases

**Q1: 日本に来た1年目は住民税がかかりませんか？**
A: 住民税は「前年の所得」に基づいて翌年に課税される後払い制度です。来日した年の翌年の課税は、前年（来日前）の日本での所得に基づくため、前年の日本での所得がゼロであれば住民税は課税されません（ai推定 — 詳細は住所地の市区町村で確認要）。ただし来日初年度の途中から就労した所得は翌々年の住民税から課税されます。

**Q2: 外国人も住民税を払う必要がありますか？**
A: はい。1月1日時点で日本に住民票がある方は、国籍に関係なく住民税の課税対象です（ai推定）。住民税の課税は住所地の市区町村（特別区含む）が行います。

**Q3: 住民税の税率はいくらですか？**
A: 所得割は全国標準で**10%**（道府県4%＋市区町村6%）です。また均等割として**5,000円/年**（2024年度以降、森林環境税含む）が課税されます。前年の所得が少ない場合は非課税となる場合があります（単身・扶養なしの場合、所得45万円以下が非課税 — 東京都23区基準）。
