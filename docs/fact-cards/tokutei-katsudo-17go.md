## current_effective_fact

ワーキングホリデービザ（在留資格：特定活動17号）は、日本とワーキングホリデー協定を締結している国の国籍者が、日本での観光・語学学習・就労を目的に1年間滞在できる制度。就労も可能（一部制限あり）。

**ワーキングホリデーの概要（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 在留期間 | 原則**1年**（一部の協定で異なる場合あり） |
| 就労 | **可**（風俗営業等の禁止業種を除く） |
| 更新 | **原則不可**（1回限り） |
| 年齢制限 | 通常18〜30歳（一部35歳まで）（ai推定） |
| 申請場所 | 出国前に自国の日本大使館・領事館で申請 |

**主な協定締結国（{{TODAY_ISO}} 時点・確認要）：**
オーストラリア、ニュージーランド、カナダ、韓国、フランス、ドイツ、英国、アイルランド、台湾、香港等（27か国以上・順次拡大中）

## exceptions_or_transition

- **日本への変更申請**：ワーキングホリデーから就労ビザへの変更は原則として認められていない（帰国後の在外申請が正規ルート）（ai推定 — tankizai-henko 参照）
- **1年経過後**：原則として在留期間の更新・延長はできない。帰国して再申請は同一人物には不可（1回限り）
- **COE不要**：ワーキングホリデーは通常、在留資格認定証明書（COE）なしで自国の日本大使館でビザを取得して入国
