## current_effective_fact

特定技能1号外国人を雇用する受入機関は、11項目の義務的支援を実施しなければならない。この支援は出入国在留管理庁に登録された「登録支援機関」に委託することができる。

**義務的支援11項目の概要（{{TODAY_ISO}} 現在）：**

| # | 支援項目 |
|---|---------|
| 1 | 事前ガイダンス（入国前の生活・業務情報提供） |
| 2 | 出入国時の送迎 |
| 3 | 住居の確保支援 |
| 4 | 生活オリエンテーション（生活・法律・行政の説明） |
| 5 | 公的手続きへの同行支援（住民登録・銀行等） |
| 6 | 日本語学習機会の提供 |
| 7 | 相談・苦情への対応 |
| 8 | 日本人との交流促進支援 |
| 9 | 転職支援（本人が希望する場合） |
| 10 | 定期的な面談・行政への報告 |
| 11 | 非自発的離職時の転職支援 |

## exceptions_or_transition

- **登録支援機関への委託**：受入機関が登録支援機関に全ての支援を委託した場合、受入機関は支援義務から免除される（実施義務を移転）
- **自社で支援する場合**：登録支援機関に委託せず、受入機関が自社で全11項目を実施することも可能（支援計画を作成・届出が必要）
- **特定技能2号**：特定技能2号では義務的支援の対象外（2号は自立した就労が前提）

## qa_cases

**Q1: 特定技能外国人を雇用しました。どんな支援が必要ですか？**
A: 特定技能1号の受入機関として、生活オリエンテーション・住居確保支援・相談対応・定期面談等11項目の義務的支援を実施する義務があります。自社で実施するか、登録支援機関に委託することができます。

**Q2: 登録支援機関に委託すれば、支援の義務はなくなりますか？**
A: はい。登録支援機関に義務的支援の全部を委託した場合、受入機関は支援の実施義務から免除されます。ただし、外国人の就労・生活状況の把握等の一部の責任は引き続き受入機関にあります。
