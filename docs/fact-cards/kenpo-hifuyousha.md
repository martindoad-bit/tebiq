## current_effective_fact

健康保険（協会けんぽ・組合健保等）の被保険者（会社員等）は、一定の条件を満たす家族（配偶者・子等）を「被扶養者」として健康保険に加入させることができる。被扶養者は保険料の追加負担なしで健康保険の給付を受けられる。外国人の家族も日本在住であれば対象。

**被扶養者認定の主な条件（{{TODAY_ISO}} 現在）：**

| 条件 | 内容 |
|------|------|
| 対象者の範囲 | 配偶者・子・孫・直系尊属・兄弟姉妹等（三親等内の親族） |
| 収入要件（同居） | 年収**130万円未満**（60歳以上・障害者は**180万円未満**）かつ被保険者の年収の半分未満 |
| 収入要件（別居） | 年収130万円未満かつ仕送り額より少ない |
| 居住要件 | **日本国内在住**（2020年4月以降・海外在住は原則不可） |
| 保険料 | **無料**（追加の保険料負担なし） |

**申請方法：**
- 会社（事業主）経由で加入している健康保険（協会けんぽ・組合健保）に届け出

## exceptions_or_transition

- **海外在住家族**：2020年4月以降、日本国内に住所を有しない家族は原則として被扶養者認定不可（`kaigai_hifuyousha_2020`確認要）
- **国民健康保険（国保）**：国保には被扶養者制度がなく、家族全員が個別に被保険者として加入・保険料を負担
- **年収の判定**：給与収入だけでなく、パート・アルバイト収入・年金収入等も含めて130万円を判定

## qa_cases

**Q1: 外国人の妻を健康保険の扶養に入れたいのですが、条件は何ですか？**
A: 配偶者（外国人含む）が日本に居住しており、年間収入が130万円未満（かつ被保険者の年収の半分未満）であれば健康保険の被扶養者として認定を受けられます。会社（事業主）経由で加入している健康保険（協会けんぽ等）に届け出てください。被扶養者の保険料は無料です。

**Q2: 妻がパートで働いています。年収はどれくらいまでなら扶養に入れますか？**
A: 年間収入が130万円未満であれば健康保険の被扶養者として認定できます（同居の場合）。ただし、給与収入だけでなくパート・アルバイト・その他すべての収入を合計した年額で判定されます。

**Q3: 海外にいる家族を健康保険の扶養に入れることはできますか？**
A: 2020年4月以降、日本国内に住所を有しない家族は原則として健康保険の被扶養者認定が難しくなりました。詳細は加入している健康保険（協会けんぽ・健康保険組合等）にご確認ください。
