## current_effective_fact

外国人労働者（特別永住者・外交・公用を除く）を雇用・離職させるすべての事業主は、ハローワークへの「外国人雇用状況の届出」が義務。未届出・虚偽届出は**30万円以下の罰金**。雇用保険被保険者は雇入れ翌月10日まで（離職は翌日から10日以内）、非被保険者は翌月末まで。

**外国人雇用状況届出の概要（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 義務者 | **全ての事業主**（独立行政法人・国立大学法人・公社等含む） |
| 届出先 | **ハローワーク（公共職業安定所）** |
| 対象外 | **特別永住者・外交・公用**の在留資格者 |
| 罰則 | **30万円以下の罰金**（未届出・虚偽届出） |
| 法令根拠 | 労働施策総合推進法 |

**届出期限（{{TODAY_ISO}} 現在）：**

| 区分 | 雇入れ | 離職 |
|------|--------|------|
| **雇用保険被保険者** | **翌月10日まで** | **翌日から10日以内** |
| **雇用保険被保険者でない者** | **翌月末日まで** | **翌月末日まで** |
| 国・地方公共団体（通知） | 翌月10日まで（非被保険者は同左） | 翌日から10日以内（非被保険者は同左） |

| 非被保険者 | 外国人雇用状況届出書（様式第3号） | **外国人雇用状況届出システム** |

## exceptions_or_transition

- **特別永住者**：届出義務の対象外（雇用主の届出は不要）
- **外交・公用**：届出義務の対象外
- **国・地方公共団体**：「届出」ではなく「通知」（外国人雇用状況通知書 — 別様式）
- **短時間アルバイト等の非被保険者**：様式第3号で翌月末日までに届出が必要（ai推定）

## qa_cases

**Q1: 外国人を採用しました。ハローワークへの届出はいつまでにすればいいですか？**
A: 雇用保険の被保険者になる外国人の場合は**雇入れの翌月10日まで**、被保険者にならない外国人（短時間アルバイト等）の場合は**翌月末日まで**にハローワークへ届出が必要です（MHLW公式）。未届出は30万円以下の罰金の対象です。

**Q2: 特別永住者を雇用した場合も届出は必要ですか？**
A: いいえ。特別永住者は「外国人雇用状況の届出」の対象外です。在留資格「外交」「公用」も同様に対象外です（MHLW公式）。

**Q3: 外国人社員が退職しました。届出はいつまでにする必要がありますか？**
A: 雇用保険の被保険者であれば**退職の翌日から10日以内**、被保険者でない場合は**翌月末日まで**にハローワークへ届出が必要です（MHLW公式）。
