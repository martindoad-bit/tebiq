**用途**: Codex 后续结构化为产品材料卡的工作底稿

## 数据来源说明

| 标记 | 含义 |
|------|------|
| `direct_source` | 从 ISA 官方 HTML 页面直接读取，文字可引用 |
| `pdf_linked` | ISA 官方 PDF 链接存在，内容无法直接读取（二进制）|
| `related_source` | 来自 ISA 相关页面 / 法令，间接确认 |
| `needs_domain` | 有歧义或实务解释问题，需 DOMAIN-CC 复核 |
| `ai_inferred` | 未在本次官方页面直接读取，属于 FACT 推断，禁止直接展示 |

## V1 范围评估与产品化建议

### 对 V1 范围的一点建议

**留学的PDF问题**：留学清单的全量材料只在 PDF 里，HTML层信息不够独立支撑产品卡。建议：
- 方案A：仅展示共通3件（申请书/照片/护照）+ 告知「需按学校种别下载对应清单」
- 方案B：由 DOMAIN 人工阅读 ISA PDF 后补录

**経営管理の過渡措置**：2025-10改正内容在官方页面可读，但哪些新要件适用于过渡期内的续签申请尚不明确。建议在产品中对経営管理单独标注「改正影响期间，请向行政书士确认适用版本」，而非展示确定性材料清单。
