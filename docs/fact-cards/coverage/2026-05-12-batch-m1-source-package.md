# Batch M1 — 共通材料库 / 高频手续清单 / 法条底座

---

## SECTION A — 共通材料库

> 材料在多个手续中重复出现。每条记录包含：material_id / 日文正式名 / 中文解释 /
> 谁准备 / 去哪取 / 取得方式 / 有效期或建议取得时点 / 常用于哪些手续 /
> 来源 URL / source_status / DOMAIN复核点 / 不适合用户展示的内容。

### M-001 住民票

| 字段 | 内容 |
|------|------|
| **material_id** | M-001 |
| **日文正式名** | 住民票の写し（世帯全員記載 or 本人のみ） |
| **中文解释** | 住民登记证明。证明在日本居住地址及世帯构成。 |
| **谁准备** | 申请人本人（或扶养人） |
| **去哪取** | 申请人居住地的市区町村役所・役场、コンビニ交付対応地域ではコンビニ |
| **取得方式** | 窗口申请 / マイナンバーカード利用のコンビニ交付 |
| **有效期** | 发行后3か月以内（手续窗口によって異なる場合あり） |
| **常用手续** | 技人国更新・変更 / 家族滞在更新・変更 / 日本人配偶者更新 / 永住申请（家族全员） |
| **source_url** | `https://www.moj.go.jp/isa/applications/status/gijinkoku.html` |
| **source_status** | `direct_source` — 各チェックシートに記載確認 |
| **DOMAIN复核点** | — |
| **展示注意** | 「世帯全員記載」か「本人のみ」かは手续によって異なる。永住申请は全員記載が必要。 |
