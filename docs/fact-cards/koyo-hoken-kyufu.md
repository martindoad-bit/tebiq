## current_effective_fact

退職後に雇用保険の基本手当（失業給付）を受給するには、一定の被保険者期間と求職活動の意思・能力が必要。自己都合退職は給付制限期間（原則2か月）がある。

**雇用保険基本手当 受給の基本情報（{{TODAY_ISO}} 現在）：**

| 項目 | 内容 |
|------|------|
| 被保険者期間（一般離職者） | 離職日以前2年間で通算**12か月以上** |
| 被保険者期間（会社都合等） | 離職日以前1年間で通算**6か月以上** |
| 待期期間 | 求職申込み後の失業状態**7日間**は不支給 |
| 給付制限（自己都合） | 待期後**2か月間**不支給（5年間3回以上は3か月） |
| 給付率 | 賃金日額の**50〜80%** |
| 所定給付日数 | **90〜360日**（離職理由・年齢・加入期間による） |
| 受給期間 | 離職翌日から**1年間** |

## exceptions_or_transition

- **会社都合退職（特定受給資格者）**：給付制限なし。所定給付日数も一般より長い（年齢・加入期間で最大330日）
- **特定理由離職者**：やむを得ない自己都合（労働時間の3/4以上削減・ハラスメント等）は特定受給資格者に準じた扱いの場合あり（`special_reason_separation`確認要）
- **傷病**：ハローワーク申請後に病気・ケガで就職困難になった場合、傷病手当として受給可能な場合あり
- **受給期間の延長（疾病等）**：疾病・負傷・妊娠・育児により30日以上就職できない場合、最大3年間の延長申請が可能
- **65歳以上**：2017年1月1日以降は65歳以上も雇用保険の適用対象（高年齢求職者給付金）

## must_say

- 被保険者期間（一般）：離職日以前2年間で通算**12か月以上**
- 待期期間：求職申込み後の失業状態**7日間**は不支給
- 自己都合退職：待期後さらに**2か月間**（5年間3回以上で3か月）給付制限
- 基本手当日額：賃金日額の**50〜80%**
- 受給期間：離職翌日から**1年間**（期間内に申請しないと失効）

## must_not_say

- 「自己都合退職では失業保険はもらえない」（給付制限はあるが受給できる）
- 「外国人は失業保険を受け取れない」（加入・受給ともに国籍不問のai推定 — 公式明示確認要）
- 「退職後いつでも申請できる」（受給期間は離職翌日から1年間で期限あり）
- 具体的な受給額を断言する（賃金日額・年齢・加入期間による計算が必要）

## qa_cases

**Q1: 自己都合退職でも失業保険（基本手当）はもらえますか？**
A: もらえます。ただし求職申込み後の7日間（待期期間）の後、さらに2か月間（過去5年間に2回以上の自己都合退職の場合は3か月間）は支給されません。その後は受給できます。受給資格は離職日以前2年間で通算12か月以上の被保険者期間が必要です。

**Q2: 失業保険の金額はいくらですか？**
A: 基本手当日額は「離職日直前6か月の毎月の給与（賞与除く）合計を180で割った賃金日額」のおよそ50〜80%です（2025年8月1日現在の上限：45歳未満8,870円等）。正確な金額はハローワークでの計算が必要です。

**Q3: 失業保険の申請期限はありますか？**
A: はい。受給期間は「原則として離職の日の翌日から1年間」です。この期間を過ぎると未受給分は失効します。退職後はできるだけ早くハローワークに求職申込みを行うことをお勧めします。
