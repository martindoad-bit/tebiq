## 办公室搬迁（Q032）重写结果

按创始人 brief 8 点要求，Q032 已逐点写入 `action_answer` 各字段：

### 8 点对照覆盖

| brief 要求 | 写入位置 |
|---|---|
| 1. 先确认新办公室是否可用于法人办公和业务使用 | `do_now`（事業使用可契約 + 用途规制 + 看板要件） |
| 2. 租赁合同最好是公司名义 + 说明使用目的 | `do_now`（法人名义 + 事業使用同意書） |
| 3. 本店所在地变更 → 法务局登记 | `how_to_do` 步骤 1-2（取締役会 议事录 + 商業登記） |
| 4. 税务署需要做异动届 | `where_to_go` + `how_to_do` 步骤 4（国税庁 + 都道府県 + 市町村） |
| 5. 社保 / 银行 / 许可 / 合同等地址也要更新 | `how_to_do` 步骤 5-6（年金事務所 / ハローワーク / 銀行 / 各種許認可 / 取引先） |
| 6. 入管续签时要能说明经营场所持续存在 | `conclusion` + `how_to_do` 步骤 8 + `consequences`（持续性立证） |
| 7. 保留新旧租约 + 照片 + 平面图 + 地址变更证明 | `documents_needed`（8 项立证材料） |
| 8. 只改邮寄地址 → 续签时材料不一致 | `consequences`（不一致 → 不许可可能） |

### 关键时间节点

- 法務局 本店移転登記 **2 週間以内**
- 入管 所属機関等届出 **14 日以内**
- 税務 / 社保 / 雇用保険 異動届 **1 ヶ月以内**

### 期限超过后果

- 14 日入管届出 懈怠 → 「届出義務違反」记录 → 下次更新审查不利
- 2 週間 商業登記 懈怠 → 過料（100 万円以下）
- 只改邮寄 + 不办登记 → 登記簿 与实际不一致 → 入管现地調査 → 续签立证不一致

### 高风险题统一格式

```yaml
action_answer:
  conclusion: 这个问题不能简单回答。<具体原因 1-2 句>
  do_now: 用户现在能先确认 1-3 件具体事
  where_to_go: 先不要直接申请。优先咨询：[具体专家类型]
  how_to_do: |
    不能简单回答的原因：[1-3 点]
    需要专家的情况：[1-3 点]
  documents_needed:
    - 需要先整理：[材料清单]
  deadline_or_timing: 客观时间窗口（含「但具体动作需要确认」标记）
  consequences: 客观陈述时间梯度
  expert_handoff: 明确什么情况必须找专家 + 哪种专家
```

### 重点处理

#### Q099 不法滞在 自首
严格区分 3 条独立路径：
- 出国命令制度（入管法 24-3 / 上陸拒否 1 年 / 5 要件）
- 退去強制（24 条 / 上陸拒否 5 年初回 / 10 年再回）
- 在留特別許可（独立路径，涉日本人配偶 / 永住者配偶 / 日本国籍子女）

弁護士 + 行政書士 联合 + 出頭前同行。不评价道德。

#### Q091 老板雇错签证
严格区分 2 条独立线：
- 雇主刑责（不法就労助長罪）
- 雇员处分（资格外活動 → 退去強制）

雇员当事人立即 弁護士（入管 + 刑事系）+ 行政書士。不引导自责。

#### Q065 DV 受害者
严格走制度路径，不情感化：
- DV センター（#8008）+ 警察（#9110）+ 弁護士 + 行政書士 + 入管特例措施
- 公式記録 多源立证（保護命令 / 警察相談 / 医療記録 / シェルター）
- 不写「勇敢」「保护自己」等情感化词

#### Q024 公司倒闭 在留资格
misconception 类，conclusion 客观陈述：「不立即取消，但有 14 日 + 3 个月两条死线」。
expert_handoff = 行政書士 + 社労士 + 必要時弁護士。

## 语气合规检测

### ✗ 全部不出现
- 保证 / 一定（肯定式）/ 绝对成功 / 100% 通过 / 拒签概率（数字化恐吓）
- 赶紧委托 / 我们可以帮你办（营销话术）
- 立即（绝对必须）/ 严重 / 不可逆（非客观）

### ✓ 全部使用
- 通常 / 一般 / 多数情况下 / 实务通说
- 需要先确认 / 需要先整理 / 一般会看
- 客观时间和金额（取代戏剧化）
- 「如果涉及个别事实，建议咨询专业人士」

否定式合法用法（不算违规）：
- 「没有绝对安全数字」「非绝对禁止」「无法保证」「不保证一定能 X」（5 处）— 都是「不 / 非 / 无法」开头的否定，符合 brief 精神
