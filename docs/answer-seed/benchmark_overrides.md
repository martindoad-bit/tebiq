## 命中规则（must_match / must_not_match）

每条 benchmark 含：
- `must_match_keywords`：必须命中的关键词（用户 query 至少含 1 个）
- `must_not_match_keywords`：避开误命中的关键词

具体规则建议给 CCA：
1. 若 user query 含 `must_match_keywords` 中任一项 + 不含 `must_not_match_keywords` → 高置信度命中
2. 若 user query 同时含两个 benchmark 的 must_match → 提示用户选择
3. 命中后优先 render benchmark_overrides 中的 `customer_message` + 展开 `do_now` / `where_to_go` 等

## BM01 — 公司休眠了要不要交国民年金？

> ⚠️ 重点修正：必须回答**个人年金义务**，不要回答经营管理续签作为主答案。

```yaml
question: 公司休眠了，我个人还要不要交国民年金？

aliases:
- 公司休眠了要不要交国民年金
- 公司不营业了社保怎么办

must_match_keywords:

- 公司停了
- 不营业
- 厚生年金
- 国民年金
- 社保切换
- 资格喪失

must_not_match_keywords:
- 经管续签
- 经营管理签证还能续吗
- 经管 改正
- 在留资格 取消

answer_type: workflow
answer_level: L2

conclusion: 公司休眠不等于个人年金义务消失。一旦厚生年金 资格喪失日 確定，你必须 14 日内切换到 国民年金（第 1 号被保険者）+ 国民健康保険。

applies_when: |
  以下任一情况：
  - 公司决议休眠 / 解散 / 停业，你的厚生年金 / 健康保険 即将失效
  - 已收到「健康保険資格喪失証明書」
  - 离职后还没进入新公司（空白期）

do_now: |
  1. 取得「健康保険資格喪失証明書」（公司应当在喪失日起 5 日内交付，未到 → 联络公司 / 元雇主）
  2. 确认 喪失日 是哪一天（这是 14 日窗口起点）
  3. 携 喪失証明書 + 在留カード + マイナンバー 通知書 → 居住地 区役所 / 市役所

where_to_go: |
  居住地 区役所 / 市役所 国保年金課（一站式办理 国民健康保険 + 国民年金）。
  必要时也可单独前往 居住地管轄 年金事務所 办理国民年金部分。

how_to_do: |
  1. 喪失日起 14 日内，持 喪失証明書 + 在留カード + マイナンバー → 居住地 区役所 / 市役所 国保年金課
  2. 同时办理 国民健康保険 加入届 + 国民年金 第 1 号被保険者 加入届
  3. 如收入低 / 失业 / 倒産特例 → 同窗口申请 国保保険料 軽減 + 国民年金 免除申請
  4. 收 新健康保険証（即日 / 数日内邮寄）+ 年金手帳 / 加入記録更新
  5. 如短期内会进入新公司 + 厚生年金加入 → 新公司入职后改回 厚生年金

documents_needed:
- 健康保険資格喪失証明書（公司发行）
- 在留カード
- マイナンバー 通知書 / カード
- 印鑑（自治体可能要求）
- 离職票（如失業 / 倒産特例 軽減申请）
- 通帳（口座振替 申请用）

deadline_or_timing: |
  - 喪失日起 14 日内（国民年金 + 国民健康保険 加入义务）
  - 喪失証明書 取得：公司应在 喪失日起 5 日内发给你
  - 国保保険料 軽減 / 国民年金 免除：同窗口办理或后续补申

consequences: |
  - 14 日内未办：保険料 遡及徴収（最大 2 年分）+ 后日督促状
  - 病院就诊：100% 自費（後日 国保加入後 7 割 還付申請可，但流程繁琐）
  - 永住申請 直近 2 年 公的義務履行 评价负分
  - 国民年金 长期未缴：将来年金额减少 + 老龄给付资格不足

expert_handoff: |
  如果出现以下任一情况，建议咨询 社労士（雇用保険 + 社保）+ 必要时 行政書士（在留资格联动）：
  1. 公司过去 2 年以上 未缴厚生年金（雇主未届出 → 本人加入記録缺失）
  2. 倒産特例給付（雇用保険）+ 国保軽減 同时申请
  3. 配偶扶養加入审查 被拒 / 异议
  4. 公司休眠后你的 経営・管理 在留資格 是否影响（这是另一条线，但建议同时确认）

customer_message: |
  公司休眠不等于个人年金 / 健保义务消失。喪失日确认后 14 日内必须切到 国民年金 第 1 号 + 国民健康保険，否则保险料会被 遡及徴収 最大 2 年分，且影响后续永住审查。今天先去公司要「健康保険資格喪失証明書」，然后带 在留カード + マイナンバー 去居住地 区役所 国保年金課 一站办理。如果同时收入低或申请倒産特例軽減，建议咨询社労士。

source_hint:
```

## BM02 — 办公室搬迁要做哪些手续？

> ⚠️ 表格化：顺序 / 做什么 / 去哪里 / 期限 / 材料

```yaml
benchmark_id: BM02
question: 経営・管理 持有人办公室搬迁要做哪些手续？

aliases:
- 公司搬家 几天内办登记
- 经营管理 事務所変更 手续
- 法人 本店所在地変更 手续
- 公司搬迁后 入管 要不要报告
- 跨都道府県 本店移転 多少钱
- 自宅兼办公室 搬到独立办公室
- 经管 事務所搬迁 完整 checklist
- 我是经管签证 公司换新地方 要做啥

must_match_keywords:
- 办公室搬迁
- 公司搬家
- 事務所変更
- 本店移転
- 经管 搬迁
- 法人 住所変更

must_not_match_keywords:
- 个人搬家
- 在留カード 住所
- 住民票 転入

answer_type: workflow
answer_level: L3

conclusion: 经営・管理 办公室搬迁不是单纯改邮寄地址，而是经营场所资料更新。涉及 4 个机关 7 步联动：法務局 商業登記 + 入管届出 + 税务 / 社保 / 雇用保険 異動届 + 银行 / 许可同步。最关键 3 个时间窗口：法務局 2 週間内 / 入管 14 日内 / 税务社保 1 ヶ月内。

applies_when: |
  以下全部满足：
  - 持 経営・管理 在留資格
  - 公司本店所在地 / 事務所 即将搬迁或已搬迁
  - 想知道完整手续 + 期限 + 不办的后果

do_now: |
  1. 先确认新办公室是否可用于法人办公和业务使用：賃貸物件 是否「事業使用可」/「店舗・事務所 兼用可」+ 用途规制（住居専用地域 → 不可）
  2. 租赁合同最好是公司名义，且能说明使用目的：明记「使用目的：法人 事業所」
  3. 是否涉及本店所在地变更：影响 法務局 登記必要性 + 跨管辖时双方法務局手续

where_to_go: |
  | 机关 | 用途 |
  |---|---|
  | 法務局（管轄 / 跨管辖时双方）| 本店所在地 変更 商業登記 |
  | 出入国在留管理庁 | 所属機関等 に関する届出（オンライン / 郵送 / 持参）|
  | 税務署 + 都道府県税事務所 + 市町村役場 | 法人 異動届 |
  | 年金事務所 | 適用事業所 所在地・名称 変更届 |
  | ハローワーク | 事業主事業所 各種変更届 |
  | 銀行 / 各種許認可機関 / 取引先 | 同步更新地址 |

how_to_do: |
  按 7 步顺序：

  | 顺序 | 做什么 | 去哪里 | 期限 | 材料 |
  |---|---|---|---|---|
  | 1 | 取締役会 / 株主総会 决议本店所在地変更 + 作成議事録 | 公司内部 | 决议日即日 | 議事録 |
  | 2 | 申請本店移転登記 | 法務局（同管辖）/ 双方法務局（跨管辖） | 决议日起 2 週間内 | 議事録 + 登録免許税（同管轄 3 万円 / 跨管轄 6 万円） |
  | 3 | 提交「所属機関等 に関する届出書」 | 出入国在留管理庁 | 14 日内 | 届出書 + 在留カード写し |
  | 4 | 提交 異動届出書 | 税務署 + 都道府県税事務所 + 市町村役場 | 1 ヶ月内 | 異動届 + 新登記事項証明書 |
  | 5 | 提交 適用事業所 所在地・名称 変更届 | 年金事務所 | 5 日内 | 変更届 + 新登記事項証明書 |
  | 6 | 提交 事業主事業所 各種変更届 | ハローワーク | 翌日から 10 日以内 | 変更届 |
  | 7 | 同步更新 銀行 / 各種許認可（食品営業 / 古物商 / 宅建 等）/ 取引先 地址 | 各机关 + 取引先 | 各自规定 | 各自要求 |

documents_needed:
- 株主総会 / 取締役会 議事録（本店移転决议）
- 新办公室 賃貸契約書（法人名义 + 事業使用可明记）
- 房东 印鑑証明書（法人名义合同情况下）
- 新登記事項証明書 + 印鑑証明書
- 新办公室 看板 / 入口 / 内装 / 平面图 照片（入管立证用）
- 旧办公室 解約証明 + 新办公室 入金 / 賃料振込記録
- 入管 所属機関等 届出 控え
- 各机关 異動届 控え
- 各種許認可 地址变更完了書

deadline_or_timing: |
  - 法務局 本店移転登記：决议日起 **2 週間以内**
  - 入管 所属機関等届出：**14 日以内**
  - 年金事務所 適用事業所変更：**5 日以内**
  - ハローワーク 事業主届：**翌日から 10 日以内**
  - 税務 / 都道府県 / 市町村 異動届：**1 ヶ月以内**

consequences: |
  - 14 日入管届出 懈怠 → 「届出義務違反」记录 + 下次更新审查不利
  - 2 週間 商業登記 懈怠 → 過料（100 万円以下）
  - 只改邮寄地址、不办登记 → 登記簿 vs 实际住所 vs 申告住所 不一致 → 入管现地調査 / 续签立证不一致 → 不许可可能
  - 跨法務局 + 期限超过 + 経営場所 立证薄弱 三件套 → 续签风险显著上升

expert_handoff: |
  如果出现以下任一情况，建议同时找 司法書士（商業登記）+ 行政書士（入管届出 + 续签立证）：
  1. 跨法務局搬迁（旧 + 新双方手续）
  2. 兼办公兼自宅 / 共享办公室 → 経営場所 立证 边界 case
  3. 経営・管理 在留期限临近（< 6 ヶ月）+ 同时搬迁
  4. 搬迁同时涉及 役員変更 / 商号変更 / 業種拡大

customer_message: |
  经营管理签的办公室搬迁不是「改个收件地址」就完事。涉及法务局 商業登記（2 週間内）+ 入管届出（14 日内）+ 税务 / 社保 異動届（1 ヶ月内）+ 银行 / 许可同步。今天先确认 3 件事：新办公室是否可用于法人办公 / 合同是否公司名义 / 是否本店所在地変更。其中法务局和入管是硬期限。如果跨法務局或经管在留快到期，建议先和 司法書士 + 行政書士 聊一下，避免登记 / 实际 / 申告地址不一致影响下次续签。

source_hint:
- 本店移転登記（法務局 商業登記）: https://houmukyoku.moj.go.jp/homu/COMMERCIAL_8.html
- 所属機関等に関する届出（出入国在留管理庁）: https://www.moj.go.jp/isa/applications/procedures/16-3.html
- 事業所変更（年金機構）: https://www.nenkin.go.jp/service/kounen/jigyosho/jigyosho/20141204.html
- 異動届出書（国税庁）: https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/hojin/annai/1554_5.htm

requires_review: false
review_notes: source_grade A，制度路径清晰；登録免許税金额可能因法令改正变更，建议每 12 ヶ月复核。
```

## BM03 — 永住申请直近 5 年纳税要完整吗？

```yaml
benchmark_id: BM03
question: 永住申请直近 5 年纳税要完整吗？

aliases:
- 永住要交几年税
- 永住申请要看几年纳税
- 直近 5 年税单要不要
- 永住 5 年 税金完税
- 永住申请 住民税 没交过 怎么办
- 我有 1 年迟交住民税 永住还有救吗

must_match_keywords:
- 永住
- 5 年纳税
- 課税証明
- 納税証明
- 完納

must_not_match_keywords:
- 帰化
- 経営管理 续签
- 公司休眠

answer_type: workflow
answer_level: L2

conclusion: 永住者ガイドライン 要求直近 5 年课税 + 纳税证明，且 5 年内不能有滞納记录。住民税 / 国保 / 国民年金 任何一年迟交都可能成为不许可理由。

applies_when: 计划申请永住 + 想自查直近 5 年 公的義務履行 是否完整。

do_now: |
  1. 居住地市役所 取直近 5 年 課税証明書 + 納税証明書 + 完納証明書
  2. 登 ねんきんネット 看直近 2 年国民年金 / 厚生年金 納付状況
  3. 取健康保険 完納証明書（市役所或健保組合）

where_to_go: |
  - 居住地市役所 課税課 / 国保年金課（住民税 + 国保 完納証明書）
  - 年金事務所 / ねんきんネット（国民年金 / 厚生年金 納付記録）
  - 各健康保険組合（厚生年金加入者 健保 完納証明）

how_to_do: |
  1. 取得直近 5 年 課税証明書 + 納税証明書（市役所 1 处 / 期间内换过市町村 → 各市町村分别取）
  2. 检查每一年是否有迟交 / 滞納记录
  3. 如发现滞納记录 → 立即补缴 + 取得「完納証明書」
  4. 同时核对 国民年金 / 厚生年金 / 健康保険 / 介護保険 直近 2 年完納记录
  5. 全部完納 → 备齐永住申请材料 → 居住地管轄入管递交

documents_needed:
- 直近 5 年 課税証明書（每年一份）
- 直近 5 年 納税証明書（住民税）
- 完納証明書（住民税 + 国保）
- 年金 納付記録（ねんきんネット 打印）
- 健康保険 加入記録（健保組合发行）
- 在留カード + 旅券

deadline_or_timing: |
  - 取得 課税 + 納税証明书：随时（市役所窗口 5-30 分钟）
  - 滞納清算 → 完納証明：1-3 个月（依清算金额 + 自治体处理速度）
  - 永住申请：完納 + 4 大项满足后随时可递

consequences: |
  - 滞納未清直接递永住 → 不许可 + 留下处分记录
  - 任一年课税年収不达门槛 → 在职稳定性评价不利
  - 课税证明跨自治体未集齐 → 入管补正通知 / 审查延长

expert_handoff: |
  如果出现以下任一情况，建议咨询 行政書士：
  1. 直近 5 年内有滞納记录但已清算（要 完納証明書 + 経緯書）
  2. 期间内换过 3 个以上市町村 + 課税証明 取得困难
  3. 个体事业主 / 自营业 → 課税証明 + 確定申告 配套审查复杂

customer_message: |
  永住申请要看你过去 5 年的纳税情况：住民税 / 国保 / 国民年金 都不能滞纳。今天先去市役所拿直近 5 年课税 + 纳税证明 + 完納証明書，对照每年是否完整。如果发现滞纳记录，先补缴 + 取得完納証明書 再申请。如果期间换过几个市町村或有滞納清算，建议先和行政書士聊一下。

source_hint:
- 永住者ガイドライン（出入国在留管理庁）: https://www.moj.go.jp/isa/applications/guide/eijyuu_guide.html
- 課税・納税証明書（国税庁）: https://www.nta.go.jp/

requires_review: false
review_notes: source_grade A，2026/2/24 永住法改正后「公的義務履行」审查口径有调整可能，建议每 12 ヶ月复核。
```

## BM04 — 永住者能不能把父母接来日本长居？

```yaml
benchmark_id: BM04
```

## must_match / must_not_match 逻辑给 CCA

### 命中规则（priority 排序）

1. **强命中**：query 含 ≥ 1 `must_match_keywords` AND 不含 `must_not_match_keywords` → 返回 benchmark_id 作为最终答案
2. **冲突命中**：query 同时强命中两个 benchmark → 提示用户选择（"你想问的是 [BM01] 还是 [BM07]？"）
3. **弱命中**：query 含 `must_not_match_keywords` 但不含 `must_match_keywords` → 不返回该 benchmark
4. **无命中**：退回到 100 条 v1 answer seed 的 alias / question 模糊匹配
