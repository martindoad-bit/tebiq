## Q076

aliases:
- 住民税什么时候交
- 住民税几月交
- 住民税晚交会怎样
- 住民税 4 期 期限
- 住民税 普通徴収

question: 住民税 何时该交？错过期限有什么后果？

answer_level: L1
answer_type: info
review_status: unreviewed
source_grade: A

summary: 住民税 普通徴収 一般 4 期纳付（6 月 / 8 月 / 10 月 / 翌年 1 月）。期限超过未缴 → 督促状 + 延滞金 + 最终差押。具体期限以你收到的纳税通知书为准。

sections:
- heading: 先确认
  body: 你收到的是普通徴収（自己交）还是特別徴収（雇主代扣工资）？普通徴収 → 一年 4 期 + 一括缴纳 选一种。特別徴収 → 雇主每月代扣 12 期。

- heading: 今天可以做什么
  body: 翻出 6 月寄到的「住民税納税通知書」。看每期期限：第 1 期 6 月底前後 / 第 2 期 8 月底 / 第 3 期 10 月底 / 第 4 期 翌年 1 月底（具体日期看通知书）。可以用 Pay-easy / 銀行口座振替 / 役所窗口 / コンビニ / クレジット 缴。

- heading: 不能做什么
  body: 不要任由督促状不处理。期限超过 20 日左右收到督促状（10 円 督促手数料），同时延滞金（年率最大约 8.7%）从期限翌日起算。继续不处理 → 财産調査 → 給与 / 預金 差押。

next_steps:
- 查纳税通知书各期期限
- 用 Pay-easy / 口座振替 / 役所窗口 缴纳
- 收到督促状 → 立即缴纳并联系課税課
- 实在交不了 → 主动联系市役所 申请分割納付

source_hint:
- 総務省 地方税法 第 320 条（普通徴収）
- 各市町村 課税課 / 納税課

boundary_note: 各自治体 延滞金 利率 + 督促手数料 + 分割納付 条件 略有差异。以你所在自治体的纳税通知书 / 督促状 注记为准。

## Q077

aliases:
- 副业要交住民税吗
- 副業 住民税 申报
- 海外汇款 住民税
- 副業 20 万円 確定申告
- 住民税 申告 不要

question: 副業 / 海外汇款收入 要不要申报住民税？

answer_level: L2
answer_type: risk_chain
review_status: unreviewed
source_grade: A

summary: 副業所得「20 万円以下」可以免確定申告（所得税）但住民税申告不可省。海外汇款（赠与 / 资产移转）原则不计为所得，但若是境外劳务报酬 / 利息 / 不动产收入则属课税対象。

sections:
- heading: 先确认
  body: 钱的性质是什么？(1) 副業給与 / 報酬 → 所得，要申报；(2) 親からの 仕送り → 通常非课税（赠与税另算）；(3) 海外不動産 / 配当 / 利子 → 所得，要申报。

- heading: 今天可以做什么
  body: 副業所得 20 万以下 → 仍需向市役所提交「住民税申告書」（4 月前后受付）。20 万超 → 確定申告（3/15 期限）会自动连动到住民税。海外送金 100 万円 超 → 银行会自动向税务署提交「国外送金等調書」。

- heading: 不能做什么
  body: 别以为 20 万以下「全部不用申报」——所得税层是这样，但住民税层从 1 円 就要申报。也不要在沒申告的状态下让公司继续以「給与のみ」的金额特別徴収——副業会被并到住民税通知里，公司从特別徴収金额变化能间接察觉副業存在。

next_steps:
- 区分钱的性质（赠与 / 报酬 / 资产）
- 副業 → 確定申告 或 住民税申告（二选一，不能都不交）
- 怕公司发现副業 → 確定申告 时 住民税选「自分で納付（普通徴収）」

source_hint:
- 国税庁 No.2030 副業
- 総務省 地方税法
- 各市町村 課税課 住民税申告書

boundary_note: 「副業 → 自分で納付」选项 各市町村处理实务略有差异，部分会无视该勾选直接特別徴収。如需保密 → 与所属市町村 課税課 当面确认 + 行政書士 / 税理士 咨询。

## Q078

aliases:
- 住民税 督促状
- 住民税 不交 怎么办
- 住民税 差押
- 住民税 延滞金
- 住民税 滞纳

question: 住民税 滞纳 之后会发生什么？督促状到差押 时间梯度是什么？

answer_level: L2
answer_type: risk_chain
review_status: unreviewed
source_grade: A

summary: 住民税 期限翌日起算 延滞金（年率最大约 8.7%）。期限後 20 日左右 督促状（地方税法 329 条）。督促状发出 10 日経過なお未納 → 法律上即可财産差押。实务上多在 1-3 个月内执行。

sections:
- heading: 先确认
  body: 你最新收到的是 (a) 納税通知書 / (b) 督促状 / (c) 催告書 / (d) 差押予告通知 / (e) 差押調書？阶段不同，应对窗口不同。

- heading: 今天可以做什么
  body: (a)(b) 阶段 → 直接缴纳即可；(c) 阶段 → 立刻联系 市役所 納税課 询问 分納誓約；(d) 阶段 → 当日 联系 + 准备资料；(e) 阶段 → 已被冻结口座 / 给与，必须立即与 納税課 + 弁護士 协商。

- heading: 不能做什么
  body: 不要以为「下次发工资再交」——给与 / 預金 / 自動車 / 生命保険 都可以差押。也不要换地址跑路——住民票 / 在留カード 都有连动，逃不掉。在留更新 / 永住申请 也会因 滞纳 履历而被减分。

next_steps:
- 立即看清楚通知书阶段
- 阶段 (a)(b) → 直接缴
- 阶段 (c)(d)(e) → 联系市役所 納税課 + 准备分納相談
- 永住 / 帰化 视野 → 把 滞纳 履历清干净（5 年完納）

source_hint:
- 地方税法 第 329 条（督促）
- 地方税法 第 331 条（滞納処分）
- 総務省 滞納処分 手引

boundary_note: 「分納誓約」是否被允许 + 期数 由各自治体 納税課 个案判断，不保证一定能分割。永住申请的「住民税完納」要求请见入管 永住許可ガイドライン。

## Q079

aliases:
- 国民年金 厚生年金 区别
- 国民年金 vs 厚生年金
- 第1号 第2号 第3号
- 年金 種類

question: 国民年金 和 厚生年金 有什么区别？我属于哪一种？

answer_level: L1
answer_type: info
review_status: unreviewed
source_grade: A

summary: 国民年金 = 全员强制加入的 1 楼。厚生年金 = 公司员工 / 公務員 在 1 楼上加的 2 楼。被保険者分 第 1 号（自営業 / 学生 / 無職）/ 第 2 号（会社員 / 公務員）/ 第 3 号（第 2 号的配偶 + 年収 130 万以下）。

sections:
- heading: 先确认
  body: 你现在的雇用形态是？(a) 正社員 / フルタイム派遣 → 第 2 号 厚生年金；(b) 個人事業主 / フリーランス / 学生 → 第 1 号 国民年金；(c) 配偶者 是第 2 号且自己年収 130 万以下 → 第 3 号；(d) 公司倒闭 / 离职中 → 第 1 号。

- heading: 今天可以做什么
  body: 保険料：国民年金 一律 月額 約 16,980 円（2024 年度）；厚生年金 標準報酬月額 × 18.3%（公司 + 自己 各半）。給付：厚生年金 在 国民年金（老齢基礎）之上多一層 老齢厚生 + 障害厚生 + 遺族厚生，整体水平显著高于 国民年金。

- heading: 不能做什么
  body: 不要以为「公司有保险所以国民年金不用交」——厚生年金加入者已经自动包含国民年金部分。也不要以为「我是外国人不享受所以不用交」——加入义务和国籍无关，离开日本时可申请「脱退一時金」（滞日 10 年内）。

next_steps:
- 看在留カード 在職証明 / 给料明細 / 健保証 → 判断号别
- 第 1 号 → 国民年金 紙払 / 口座振替 / クレジット
- 第 2 号 → 公司从工资扣，无需自己操作
- 离日 → 脱退一時金 准备出国前办

source_hint:
- 日本年金機構 公式 国民年金 / 厚生年金
- 厚生労働省 年金制度 解説
- 国民年金法 / 厚生年金保険法

boundary_note: 第 3 号被扶养判定（130 万 / 106 万 壁）实务由 健保組合 + 年金事務所 双重审查。具体数字以年度公示为准。

## Q080

aliases:
- 国民年金 免除
- 年金 交不起
- 国民年金 免除申请
- 学生納付特例
- 納付猶予

question: 国民年金 交不起 / 想申请免除 怎么办？

answer_level: L2
answer_type: workflow
review_status: unreviewed
source_grade: A

summary: 国民年金 有 4 档免除（全額 / 4 分の 3 / 半額 / 4 分の 1）+ 納付猶予 + 学生納付特例。按 前年所得 + 世帯人数 判断。承認期間内 給付 部分计入（全額免除 → 1/2 算入），事后 10 年内可追納。

sections:
- heading: 先确认
  body: 你属于：(a) 学生 → 学生納付特例（在校期间专用）；(b) 50 歳未満 + 收入低 → 納付猶予；(c) 通常成年 + 失業 / 收入低 → 申請免除（4 段階）。本人 + 配偶 + 世带主 三方所得 都会被审查。

- heading: 今天可以做什么
  body: 准备：年金手帳 / 基礎年金番号通知書 + 本人確認書類 + 失業証明（離職票 / 雇用保険受給資格者証 / 退職証明書 任一）。在 市役所 国民年金課 或 年金事務所 提交「国民年金保険料免除・納付猶予申請書」。可申請 過去 2 年 1 ヶ月分 + 当年度。

- heading: 不能做什么
  body: 不要不申请就放着不交——「未納」 vs 「免除」 完全不同：未納 期间 障害年金 / 遺族年金 没保障，且永住 / 帰化 时被审查。也不要以为「免除 = 不用还」——10 年内可追納补回 满額。

next_steps:
- 准备 失業証明 + 身份证 + 年金手帳
- 去 年金事務所 / 市役所 国民年金課 提交申请
- 等通知（约 2-3 ヶ月）
- 经济恢复 → 10 年内追納 取回满額

source_hint:
- 日本年金機構 国民年金保険料の免除制度・納付猶予制度
- 国民年金法 第 90 条
- 厚生労働省 年金 Q&A

boundary_note: 申请审查由日本年金機構执行，所得基準 + 認定期間 各年度调整。当年度若 4 月前申请 → 7 月翌年 6 月一年承認；7 月以降申请 → 翌年 6 月。永住申请视野下 「免除期间」是否被算作「未納」由入管个案判断。

## Q081

aliases:
- 公司倒闭 厚生年金
- 公司休眠 年金
- 厚生年金 国民年金 切换
- 退职 年金 怎么办

question: 公司倒闭 / 休眠 → 我的厚生年金 / 健保 怎么办？

answer_level: L3
answer_type: workflow
review_status: unreviewed
source_grade: A
requires_review: true

summary: 公司倒闭 / 休眠 → 厚生年金 + 健保 自动失格（喪失日 = 离职翌日）。本人必须在 14 日内 切换到 国民年金 + 国民健康保険（或 健保任意継続 2 年）。漏切换 → 未加入空白 → 永住 / 帰化 / 在留更新 风险。

sections:
- heading: 先确认
  body: 你手上有：(a) 離職票 / 退職証明 / 雇用保険被保険者離職票 ？(b) 健康保険 資格喪失証明書？没有的话 → 立刻找 元雇主 + 元公司管辖 年金事務所 索取。社長失联 / 公司倒闭 → 直接到 年金事務所 提交「健康保険・厚生年金保険 被保険者資格喪失届」职权请求。

- heading: 今天可以做什么
  body: 切换 14 日内 行动：(1) 持喪失証明 + 在留カード + マイナンバー 到 市役所 国保 課 + 国民年金 課 申请加入；(2) 选项 B：健保任意継続（喪失日 20 日内 申请，最长 2 年，保険料 全額自己负担）；(3) 雇用保険 → ハローワーク 申請 失業給付。

- heading: 不能做什么
  body: 不要在「等下家 offer」期间什么都不做。空白期会算「未加入」「未納」。也不要以为「公司没倒只是暂停」就放任——休眠状态下公司停止支付社会保险料 = 你的资格已被喪失，必须个人切换。

next_steps:
- 取得 健康保険資格喪失証明書 + 離職票
- 14 日内 到市役所 国保 + 国民年金 切换
- 比較 国保 vs 健保任意継続 哪个保険料低
- 雇用保険 → ハローワーク 申請 受給資格決定
- 在留更新 / 永住 视野下 → 行政書士 咨询 空白期影响

source_hint:
- 日本年金機構 資格喪失届
- 全国健康保険協会（協会けんぽ）任意継続 案内
- 厚生労働省 雇用保険 受給手続き
- 入管 永住ガイドライン（公的年金 / 公的医療保険 納付状況）

boundary_note: 雇主未提交「資格喪失届」时，由 年金事務所 / 健保協会 按职权处理，时间会拖延。永住 / 帰化 申请 对「空白期」的容忍度由入管 / 法務局个案判断，建议 行政書士 + 社労士 联合咨询。本回答非法律意见。

## Q082

aliases:
- 国保 社保 重复
- 健保 国保 切换
- 国民健康保険 重複加入
- 健保 重複 返還

question: 国保 / 社保 重複加入 或 切换空白 怎么处理？

answer_level: L2
answer_type: risk_chain
review_status: unreviewed
source_grade: A

summary: 同时加入两种健保 → 必须立即解约其中一个（通常解约 国保）+ 申请保険料返還。空白期 → 一旦发现疾病 / 受伤就医 全額自负。两种情况都需到市役所 + 健保協会双方协调。

sections:
- heading: 先确认
  body: 重複 vs 空白？(a) 入职新公司没去市役所做 国保脱退 → 重複（社保 优先有效，国保 应解约 + 返金）；(b) 离职 → 没在 14 日内做 国保加入 → 空白 → 这期间就医 全额自付。

- heading: 今天可以做什么
  body: 重複情况：到 市役所 国保 課 提交「国民健康保険 喪失届」+ 社保 健保証 复印件 → 申请保険料 過誤納返還（最长追溯 2 年）。空白情况：到 市役所 国保 課 申请「資格遡及取得」+ 提交 喪失証明 → 保険料从空白期起算补缴（也补医療費 7 割還付）。

- heading: 不能做什么
  body: 不要把 国保証 + 社保証 两张都用——医療機関察觉后会要求返還 7 割医療費 → 自费追加。也不要拖到「下个月再去市役所」——补缴期限超过自治体规定就丧失返還權利。

next_steps:
- 整理 健保証 全部历史 + 入职 / 离职日期
- 重複 → 国保脱退届 + 返金申请
- 空白 → 国保資格遡及加入 + 補繳
- 永住 / 帰化 视野 → 把空白期补干净

source_hint:
- 国民健康保険法
- 各市町村 国民健康保険課
- 全国健康保険協会 公式 Q&A

boundary_note: 過誤納返還 时効 一般 2 年。空白期 補繳 是否能全額追溯由市町村个案判断。永住 / 帰化 审查下的「空白期」影响 → 行政書士 个别咨询。

## Q083

aliases:
- 年金 滞纳 永住
- 年金 未納 永住
- 永住 年金 要求
- 帰化 年金
- 年金 補繳

question: 年金 滞纳 怎么清算？永住 / 帰化 申请会被影响吗？

answer_level: L3
answer_type: risk_chain
review_status: unreviewed
source_grade: A
requires_review: true

summary: 年金 滞纳 直接影响永住 / 帰化 审查。入管 2019 年 改正永住ガイドライン 明确「公的年金・医療保険 適切納付」是要件。滞纳 → 追納 时効 2 年（免除期间 10 年）。已超时効的部分 → 行政書士 个案咨询「补救陈述」。

sections:
- heading: 先确认
  body: 拉「ねんきんネット」记录 + 年金事務所「被保険者記録照会回答票」。看：(a) 未納月数 / (b) 免除月数 / (c) 納付月数 / (d) 是否曾经超过 2 ヶ月以上 滞纳。永住 / 帰化 实务中这些数据会被入管 / 法務局 直接调取。

- heading: 今天可以做什么
  body: (1) 时効内（2 年）→ 直接到年金事務所 一括追納；(2) 免除期间 → 10 年内追納；(3) 时効外 → 已无法补缴，但可在永住 / 帰化 申请时附上「理由書」+ 现状改善证明（每月正常缴 + 自动引落）至少 2 年以上。

- heading: 不能做什么
  body: 不要以为「公司一直交厚生年金所以国民年金时期不重要」——离职 / 学生时代 国民年金 滞纳 也算。不要在永住 / 帰化 申请前 6 ヶ月内 才开始补——补完 + 稳定 2 年才比较安全。

next_steps:
- ねんきんネット 拉历史 + 年金事務所 照会

summary: 入管法 第 24 条の 3「出国命令制度」適用条件：(1) 自ら入管に出頭 + (2) 不法滞在以外の罪なし + (3) 過去に退去强制歴・出国命令歴 なし + (4) 速やかに本邦から出国する意思 + (5) 罰金 / 懲役 受けたことがない 等 全て充足 → 上陸拒否 1 年。退去强制 → 5 年（初回）/ 10 年（再回 / 重大違反）上陸拒否。两者法的扱い + 将来 再入国可能性 大幅不同。

sections:
- heading: 先确认
  body: 出国命令 5 要件 全部満たすか？(1) 自首（被摘発前 自分で入管に出頭）；(2) 不法滞在 のみ（不法就労 / 偽造 / 犯罪 等 累計なし）；(3) 退去强制 / 出国命令 履歴 なし；(4) 速やかな出国意思 + 旅費 確保；(5) 罰金 / 懲役 履歴 なし。1 つでも欠ける → 退去强制 路線。

- heading: 今天可以做什么
  body: 出頭前 必須 → (1) 弁護士（入管 / 刑事系）+ 行政書士 联合咨询（法テラス 利用可能）；(2) 5 要件 个案チェック；(3) 出頭 同行依頼；(4) パスポート + 帰国チケット + 帰国準備 整理；(5) 出頭時 弁明書 + 反省文 + 帰国計画書 提出。

- heading: 不能做什么
  body: 不要单独出頭 → 5 要件 微妙なところで 出国命令 適用が崩れて 退去强制 化するリスク。不要 SNS / 知人経由 で 出頭前に話題化 → 摘発リスク 上昇 → 自首扱い 失う。不要在没有专业陪同下 笔录。也不要等「結婚すれば在留資格に変わる」式幻想—不法滞在中 婚姻 → 在留特別許可 申请 別路径，自首 vs 残留 是根本分岐。

- 5 要件 个案チェック（罰金歴 / 過去履歴）

- 出頭後 速やかに 出国（出国命令 = 14 日以内 退去 が一般）
- 上陸拒否 期間 後 再入国 戦略

- 法テラス 公式
