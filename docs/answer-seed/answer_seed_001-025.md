## Q001

aliases:
- 永住要交几年税
- 永住申请要看几年纳税
- 直近 5 年税单要不要
- 永住 5 年税金完税

question: 永住申请直近 5 年纳税要完整吗？

answer_level: L2
answer_type: workflow
review_status: unreviewed
source_grade: A

summary: 永住者ガイドライン明确要求直近 5 年课税 + 纳税证明，且 5 年内不能有滞納记录。任何一年迟交住民税 / 国保 / 国民年金 都会成为不许可理由。

sections:
- heading: 先确认
  body: 你过去 5 年都在日本居住吗？是否有任何一年迟交住民税 / 健康保険料 / 国民年金？是否换过市町村（要逐个市役所取证明）？
- heading: 今天可以做什么
  body: 居住地 市役所 取得 直近 5 年 課税証明書 + 納税証明書（或 e-Tax 在线申请）。年金需向年金事務所 / ねんきんネット 取得 直近 2 年 納付状況。
- heading: 不能这样做
  body: 已知有滞納但未清算前不要硬递永住申请；先补缴 + 取得「完納証明書」+ 等几个月稳定记录再申请。也不要伪造或漏报年金 / 住民税 状况。

next_steps:
- 居住地市役所窗口取得 5 年课税 + 纳税证明
- 年金事務所 取得 2 年納付状況
- 检查每一年是否有迟交记录
- 若有迟交，补缴 + 取得完納証明書
- 滞納清算后 至少等 6-12 个月稳定记录再递交

source_hint:
- 法務省 永住者ガイドライン
- 国税庁
- 各市町村 課税課
- 日本年金機構

boundary_note: 直近 5 年 是 现行ガイドライン基准；2026/2/24 永住法 改正后审查口径有调整可能，具体个案建议咨询行政書士。

## Q002

aliases:
- 永住申请前换工作能不能
- 永住要看在职时长吗
- 直近 1 年同公司是不是硬要求
- 换工作影响永住吗

question: 永住申请前换工作还能不能申请（直近 1 年同公司是不是必须）？

answer_level: L3
answer_type: decision_card
review_status: unreviewed
source_grade: B

summary: 法令上没有「直近 1 年同公司」硬性规定，但实务上 申请时点 在职稳定性 + 直近 1 年课税证明 来自同一收入源 是 入管 通过率显著高的实务通说。换工作后立即申请 风险高。

sections:
- heading: 先确认
  body: 新公司是不是同等以上职位 + 同等以上年収？换工作时间距永住申请有多久？新公司是否已上社保？
- heading: 今天可以做什么
  body: 评估时间窗口：换工作后 至少待满 1 个住民税課税年度（即新工资能体现在新一年课税证明上）再递永住，更稳。新工作的雇用契約書 + 在職証明書 + 直近月份给与明细 提前备齐。
- heading: 不能这样做
  body: 不要 换工作不到 3 个月就递永住（在职稳定性不足）；不要 跳到完全不同行业且年収下降的公司后立即申请；不要 同时申请在留期間更新 + 永住 时机错位。

next_steps:
- 确认换工作后到目前在职月数
- 评估新职位是否同等以上
- 优先等满 1 个课税年度再递
- 备齐新公司雇用契約書 + 在職証明書
- 有疑问咨询行政書士做时间窗口规划

source_hint:
- 法務省 永住者ガイドライン
- 出入国在留管理庁
- 行政書士実務通説

boundary_note: 「直近 1 年同公司」是实务通说而非法令；个案能否通过取决于综合稳定性，建议咨询行政書士。

## Q003

aliases:
- 永住者能不能带父母来日本养老
- 永住者父母想长期住日本怎么办
- 父母移居日本路径
- 永住者帯同父母

question: 永住者能不能把父母接来日本长期居住？

answer_level: L4
answer_type: misconception
review_status: unreviewed
source_grade: A

summary: 这个问题不能简单回答「能」或「不能」。日本无中国「探亲签长居版」对应制度。「家族滞在」明确不含父母。父母可来日的真实路径只有 3 条，且都受严格条件限制。

sections:
- heading: 为什么不能简单回答
  body: 日本入管法 別表第二「家族滞在」对象仅含 配偶者 + 子。父母不在「家族滞在」对象。永住者本身的「家族帯同」也仅限配偶 + 子。
- heading: 真实存在的 3 条路径
  body: 1）短期滞在 90 日 + 招聘理由書（多次入境，最常见）；2）特定活動「老親扶養」ビザ（极严，事实上极少下签，要求父母 70 岁以上 + 在母国无其他亲属可扶养 + 申请人有充分扶养能力）；3）高度専門職 1 号 + 親族帯同（仅高度人材，且子女 7 岁未满或申请人 / 配偶妊娠中育児支援 等限定情形）。
- heading: 不能这样做
  body: 不要 硬以「家族滞在」名义申请父母（100% 不许可）；不要 让父母短期滞在超 90 日 滞留不出境（不法滞在 + 退去強制 + 上陸拒否 5 年）；不要 用「特定活動」轻率赌一把（大量行政書士都不接此类案件）。

next_steps:
- 确认本人是否为高度専門職 1 号
- 评估父母真实健康 + 母国是否还有兄弟姐妹
- 优先走 短期滞在 90 日 + 多次入境策略
- 如必须长居 + 70 岁以上 + 母国无亲属 再考虑老親扶養
- 找有老親扶養实绩的行政書士面谈

source_hint:
- 出入国在留管理庁
- 外務省 査証案内
- 法務省 入管法 別表第二

boundary_note: 「老親扶養 ビザ」实务通过率极低 + 个別審査；具体个案建议咨询有实绩的行政書士。

## Q004

aliases:
- 酒驾还能不能申请永住
- 飲酒運転 影响永住吗
- 酒驾几年后能永住
- 酒后开车永住

question: 有过酒驾还能不能申请永住？

answer_level: L4
answer_type: needs_expert
review_status: unreviewed
source_grade: A

summary: 这个问题不能简单回答「能」或「不能」。酒驾在永住审查中属于「素行善良要件」明显瑕疵，但具体能否通过取决于酒驾类型（酒気帯び / 酒酔い）、是否有罚金 / 刑事处分、距今年数、是否再犯。需个案分析。

sections:
- heading: 为什么不能简单回答
  body: 日本「素行善良」是永住核心要件之一。酒驾包括 酒気帯び運転（呼気 0.15 mg/L 以上）+ 酒酔い運転（明显泥酔）两档，处分轻重 + 是否人身事故 直接决定影响程度。
- heading: 实务参考
  body: 1）酒気帯び 仅罚金、无人身事故、距今 5 年以上：可能可以申请，但要补充充分反省 + 之后无任何交通违反记录；2）酒酔い 或 有人身事故 或 罚金以上刑事处分：5 年内基本不可能通过；3）再犯：即使距今 10 年也极难通过。
- heading: 不能这样做
  body: 不要 隐瞒过去酒驾记录（入管会调阅警察庁数据库，发现即不许可 + 信用破裂）；不要 在罚金 / 刑事処分 完了 5 年内 立即递交；不要 自己判断 案主直接咨询行政書士。

next_steps:
- 取得当时的处分通知書 / 罰金確定書
- 取得运転記録証明書（自动車安全運転センター）确认违反履历
- 算清距今年数
- 最近 5 年内无任何违反才考虑递交
- 案件咨询有酒驾永住实绩的行政書士

source_hint:
- 法務省 永住者ガイドライン 素行善良要件
- 警察庁
- 自動車安全運転センター

boundary_note: 个別審査 + 据案主资料综合判断；强烈建议咨询行政書士或弁護士，自行判断风险高。

## Q005

aliases:
- 公司不交社保还能永住吗
- 老板没给上厚生年金
- 社保未加入 永住
- 自己交国民年金能永住吗

question: 公司不交厚生年金 / 健康保険 还能不能申请永住？

answer_level: L4
answer_type: needs_expert
review_status: unreviewed
source_grade: A

summary: 这个问题不能简单回答。永住要件中 公的義務（含税、年金、健康保険）履行 是硬指标。如果你是 厚生年金強制適用事業所 的全职员工 而公司不上社保，问题在公司不在你；但 你个人是否补缴国民年金 / 国民健康保険 + 是否有滞納记录 直接决定永住通过率。需个案分析。

sections:
- heading: 为什么不能简单回答
  body: 日本《厚生年金保険法》強制適用事業所 中正社員必须加入厚生年金 + 健保。如果公司故意不加入，劳动者可向 年金事務所 + 労働基準監督署 通报。但永住审查只看你「个人是否履行公的義務」，公司违法不能成为你不交年金的免责理由。
- heading: 现实路径
  body: 1）现在立即向 年金事務所 申报「適用調査」要求公司补加入；2）公司未加入期间 你应已加入 国民年金 + 国民健康保険（市役所），无滞納；3）如果你既没上厚生 也没上国民 → 永住申请前 至少 24 个月補納 + 取得 完納証明書；4）已经长期滞納的 通过率 显著降低。
- heading: 不能这样做
  body: 不要 永住申请時 同时存在 年金未納 / 健保未納；不要 以「公司不上」作为永住申请時 的解释（入管不接受）；不要 假装不知道（入管会调 年金事務所 数据）。

next_steps:
- 调取 ねんきんネット 直近 2 年納付状況
- 到 年金事務所 + 市役所 确认未納月数
- 補納能補的部分 + 取得完納証明書
- 必要时向年金事務所申报公司「適用調査」
- 个案咨询行政書士 + 社労士

source_hint:
- 日本年金機構
- 厚生労働省
- 各市町村 国民年金 / 国保 課
- 法務省 永住者ガイドライン

boundary_note: 公司违法 与 个人申请要件 是两件事；永住审查只看个人履行情况，建议先咨询社労士补缴 + 行政書士评估时机。

## Q006

aliases:

- 永住年収门槛
- 安定収入 标准

question: 永住申请的「安定収入」实务标准是多少？

answer_level: L3
answer_type: decision_card
review_status: unreviewed
source_grade: B

summary: 法令未规定具体年収门槛，但永住者ガイドライン要求「独立生計を営むに足る资産または技能」。实务通说：单身 年収 300 万円 是 软门槛；有配偶 / 抚养子女则需 350-400 万円 起；东京 / 大阪 高物价地区门槛实务上更高。

sections:
- heading: 先确认
  body: 你直近 3 年 課税証明書 上的年収 是多少？是否抚养配偶 / 子女？居住地是哪一档物价？
- heading: 实务参考
  body: 1）单身 年収 ≥ 300 万円 + 直近 3 年稳定 + 无负债：通常视为达标；2）抚养 1 人 通常要求 ≥ 350 万円；3）抚养 2 人以上 通常要求 ≥ 400 万円；4）个体事业主 / 自营业 看 確定申告書 + 直近 3 年所得 + 有无赤字。
- heading: 不能这样做
  body: 不要 在年収刚刚跌破前一年时立即申请；不要 把年収以「副業」方式凑数（入管对副業 + 主業比例 + 资格外活動有审查）；不要 漏报抚养亲属。

next_steps:
- 取直近 3 年 課税証明書 看年収实际数
- 算抚养亲属人数
- 单身 < 300 万 / 有抚养 < 350 万 → 等加薪后再递
- 个体事业主 备齐 確定申告書 控 + 直近 3 期决算
- 有疑问咨询行政書士

source_hint:
- 法務省 永住者ガイドライン
- 各市町村 課税証明書
- 行政書士実務通説

boundary_note: 年収门槛是 实务通说不是法令；2026/2/24 永住法改正后口径有微调可能，建议咨询行政書士。

## Q007

aliases:
- 高度人材 1 年永住
- 80 点永住
- 高度専門職 永住路径
- 高度人材 70 点 80 点 区别

question: 高度専門職 1 号 → 永住的 1 年 / 80 点路径是什么？

answer_level: L2
answer_type: workflow
review_status: unreviewed
source_grade: A

summary: 高度専門職 1 号 申請時 + 申请永住時 持续达 80 点以上 + 持续 1 年 → 可申请永住（特例）。70 点以上持续 3 年也可。是日本永住最快路径。

sections:
- heading: 先确认
  body: 你目前在留资格是否为「高度専門職 1 号 イ / ロ / ハ」？最新点数计算多少分？是否在日满 1 年 / 3 年（按点数对应）？
- heading: 今天可以做什么
  body: 1）80 点以上 + 满 1 年：备齐 直近 1 年点数维持证据（年収 + 学历 + 职位 + 研究実績 など）+ 在留资格延长记录 + 一般永住书类（课税 + 纳税 + 年金）；2）70 点 + 满 3 年：同上但 3 年点数维持证据。点数计算表 严格按 法務省 公布版。
- heading: 不能这样做
  body: 不要 中途点数掉到 70 / 80 以下 还硬递（按申请時点数审查）；不要 漏 1 年 / 3 年「高度専門職 在留」连续期；不要 混淆「申请時」和「申请前 1 年 / 3 年」（两个时点都要满足分数）。

next_steps:
- 用 法務省 公布点数计算表 自算
- 备齐学历 + 年収 + 職務 + 研究実績 各项点数证据
- 1 年 / 3 年 在留期間 连续无中断
- 同时达成 一般永住要件（税、年金、健保）
- 直接递永住，无需先转为高度専門職 2 号

source_hint:
- 法務省 高度人材ポイント制
- 出入国在留管理庁 永住許可申請
- 法務省 永住者ガイドライン

boundary_note: 点数计算每年微调，按申请時最新版表计算；建议提前咨询行政書士复核分数。

## Q008

aliases:
- 永住不许可后多久能再申请
- 永住被拒 再申请间隔
- 永住不许可理由 改善
- 永住申请失败再来

question: 永住申请不许可后多久可以再申请？

answer_level: L3
answer_type: workflow
review_status: unreviewed
source_grade: B

summary: 法令无再申请間隔规定，理论上随时可再申请。但实务通说：先取得「不許可理由」（口头说明可申请）+ 针对原因改善 + 至少 6-12 个月再递 通过率才能提升。

sections:
- heading: 先确认
  body: 你拿到不許可通知書了吗？有没有去入管听不許可理由口头说明？原因是 素行 / 收入 / 税金 / 年金 / 在留期間 哪一项？
- heading: 今天可以做什么
  body: 1）持不許可通知書 预约入管 听口头理由（必须本人去）；2）针对理由对症改善（补缴 / 续在留 / 等收入达标）；3）改善状态稳定 6-12 个月后再递；4）准备「前回不許可后改善内容説明書」一并提交。
- heading: 不能这样做
  body: 不要 拿到不許可立即同样书类原样再递（结果一样）；不要 跳过听口头理由 自己猜原因；不要 在留期間更新前夕 才再递永住（同时有两个审查窗口风险）。

next_steps:
- 收到不許可通知後 30 日內 预约入管口头说明
- 整理不許可理由 → 列改善清单
- 改善完成 + 取证据 + 等 6-12 个月稳定
- 重新备齐书类 + 加「改善説明書」
- 有疑问咨询行政書士做策略

source_hint:
- 出入国在留管理庁
- 法務省 永住者ガイドライン
- 行政書士実務通説

boundary_note: 再申请间隔无法令规定但实务通说重要；建议咨询行政書士做时间窗口策略。

## Q009

aliases:
- 永住者出国超 1 年失效
- 永住者长期回国
- 永住 出国 3 年
- 再入国許可超期

question: 永住者出国超过 1 年 / 3 年会不会让永住失效？

answer_level: L2
answer_type: workflow
review_status: unreviewed
source_grade: A

summary: 永住者本身资格不会因出国时长「自动失效」，但 みなし再入国（1 年内有效）/ 再入国許可（最长 5 年）超期出境后 → 在留资格灭失，等于永住资格丧失。再入国制度是关键。

sections:
- heading: 先确认
  body: 你下次出国预计停留多久？目前持有再入国許可（特别再入国 还是 通常再入国）？还是只用 みなし再入国？
- heading: 今天可以做什么
  body: 1）出国 1 年以内：用 みなし再入国（出境时机场提交 EDカード 即可）；2）出国 1 年以上 5 年以内：必须 出国前 入管申请 通常再入国許可（可申请 1-5 年）；3）出国期间 健康保険 / 住民票 / 国民年金 是否要保留 视情况而定。
- heading: 不能这样做
  body: 不要 用 みなし再入国 出境后 超 1 年才回（再入国失效 → 在留资格灭失）；不要 用 通常再入国 5 年期限到了 还在境外（同失效）；不要 误以为永住者「永远不会失效」。
