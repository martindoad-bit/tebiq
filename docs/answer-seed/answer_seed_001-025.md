answer_level: L2
answer_type: workflow

summary: 永住者ガイドライン明确要求直近 5 年课税 + 纳税证明，且 5 年内不能有滞納记录。任何一年迟交住民税 / 国保 / 国民年金 都会成为不许可理由。

question: 永住申请前换工作还能不能申请（直近 1 年同公司是不是必须）？

answer_level: L3
answer_type: decision_card
review_status: unreviewed
source_grade: B

summary: 法令上没有「直近 1 年同公司」硬性规定，但实务上 申请时点 在职稳定性 + 直近 1 年课税证明 来自同一收入源 是 入管 通过率显著高的实务通说。换工作后立即申请 风险高。

sections:
- heading: 先确认
  body: 新公司是不是同等以上职位 + 同等以上年収？换工作时间距永住申请有多久？新公司是否已上社保？
- heading: 今天可以做什么
  body: 评估时间窗口：换工作后 至少待满 1 个住民税課税年度（即新工资能体现在新一年课税证明上）再递永住，更稳。新工作的雇用契約書 + 在職証明書 + 直近月份给与明细 提前备齐。
- heading: 不能这样做
  body: 不要 换工作不到 3 个月就递永住（在职稳定性不足）；不要 跳到完全不同行业且年収下降的公司后立即申请；不要 同时申请在留期間更新 + 永住 时机错位。

next_steps:
- 确认换工作后到目前在职月数
- 评估新职位是否同等以上
- 优先等满 1 个课税年度再递
- 备齐新公司雇用契約書 + 在職証明書
- 有疑问咨询行政書士做时间窗口规划

source_hint:
- 法務省 永住者ガイドライン
- 出入国在留管理庁
- 行政書士実務通説

boundary_note: 「直近 1 年同公司」是实务通说而非法令；个案能否通过取决于综合稳定性，建议咨询行政書士。

action_answer:
  conclusion: 法令未规定「同公司 1 年」硬要求，但实务通说是 换工作后 至少待满 1 个住民税課税年度 + 在职稳定 1 年以上 通过率显著高。
  do_now: 算清换工作后至今的在职月数、确认新公司是否同等以上职位 + 同等以上年収、查明新公司是否已为你上厚生年金 + 健康保険。

  how_to_do: |
    1. 算清换工作日 → 永住预定申请日 的间隔（最理想是已经过 1 个完整課税年度）。
    2. 向新公司人事索取 雇用契約書 + 在職証明書 + 直近 3 个月給与明細。
    3. 确认厚生年金 + 健康保険 加入日（ねんきんネット 或 健保証券面）。
    4. 若间隔 < 6 个月 → 暂缓永住申请，等满 1 课税年度新収入体现在課税証明書后再递。

    - 新公司 在職証明書
    - 直近 3 个月給与明細
    - 直近 3 年 課税証明書
    - 新旧公司社保加入記録（ねんきんネット）
  deadline_or_timing: 一般等满 1 课税年度（即新収入完整体现在新一年課税証明書上）+ 在职 1 年以上 再递永住，更稳。
  consequences: 入职 < 3 个月就递 → 在职稳定性不足，不许可率高 + 留下不许可记录；跨行业 + 年収下降 → 「安定収入」要件被疑。
  expert_handoff: 跨行业转职、年収比前职下降 20% 以上、跳到决算赤字小公司、或在留期間更新和永住申请时机重叠 → 咨询行政書士做时间窗口策略。

    5. 在留剩余 < 3 个月 + 还没找到 → 申「特定活動 6 个月（就職活動）」延长在留（手数料 4,000 円）。
  documents_needed:
    - 在留カード

    - 「特定活動」申請書 + 求职活動記録（如申延長）
    - 健康保険資格喪失証明書
  deadline_or_timing: 14 日内 → 終了届；3 个月节点 → 入管可启动取消程序；在留剩余 < 3 个月 → 立即申特定活動。但具体动作需要确认。
  consequences: 拖着不交終了届 → 罰金 ≤ 20 万円 + 信用破裂；3 个月以上完全不就労不报告 → 在留资格取消程序启动 → 留下处分记录影响一切后续；打黑工凑过渡 → 资格外活動 + 退去强制。
  expert_handoff: 在留剩余 ≤ 3 个月 + 还没找到工作、想申特定活動就職活動延长、公司有未払賃金 / 未支払 雇用保険 / 未支払 社保、或离职票 / 破産公告 取不到 → 同时咨询行政書士（在留）+ 社労士（雇保 + 社保 + 未払賃金）+ 必要時弁護士（不当解雇 / 未払賃金訴訟）。

  - 个案变量多（在留剩余期間 / 公司倒闭确认日 / 是否已交終了届 / 是否已开始转职 / 是否同时申特定活動 / 公司有无未払賃金）

  - 误判后果不可逆（在留资格被启动取消程序后留下处分记录 → 影响后续一切签证申请；打黑工凑过渡 → 退去强制）
