## 优先级判定标准

1. 高频（创始人/客服重复被问）
2. 紧急（14 日届出 / 在留期限 / 不法滞在）
3. 用户容易做错（misconception 高发）
4. 影响签证 / 税保 / 期限（错过窗口 → 不可逆后果）
5. 适合做 Decision Card（多路径分歧但可枚举）
6. 体现 TEBIQ 差异化（档案中心化 / 工具感 / 与 ChatGPT 不同）
