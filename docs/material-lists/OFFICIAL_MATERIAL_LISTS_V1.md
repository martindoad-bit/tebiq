## 一、签证 / 在留資格 6 类

### 1. 技人国（技術・人文知識・国際業務）— 在留期間更新

#### 适用对象
- 持「技術・人文知識・国際業務」在留資格 + 雇用継続中 / 転職中
- 在留期間更新申請（延长 1 年 / 3 年 / 5 年）

#### 官方材料
- 在留期間更新許可申請書（指定様式）
- 写真 1 枚（4cm × 3cm，3 ヶ月以内）
- パスポート（旅券）+ 在留カード 提示
- 雇用契約書（写し）
- 在職証明書（写し）
- 直近 1 年 給与所得 源泉徴収票（写し）
- 直近 1 年 課税証明書 + 納税証明書（市区町村役所）

#### 常见补充材料
- 公司案内 / 业务内容説明書（写し）
- 公司决算書（写し，直近 1 期）
- 法人登記事項証明書（公司提供）
- 公司四季報抜粋 / 上場会社 IR 资料（如适用）
- 業務内容説明書（本人作成，1-2 页）
- 学歴証明書（卒業証明書 / 学位記）

#### 哪里取得
| 材料 | 取得地 |
|---|---|
| 申請書様式 | 出入国在留管理庁 公式網站 |
| パスポート / 在留カード | 本人持有 |
| 雇用契約書 / 在職証明書 / 源泉徴収票 | 公司人事 / 給与担当 |
| 課税証明書 / 納税証明書 | 居住地市区町村 役所 |
| 法人登記事項証明書 | 法務局 |
| 学歴証明書 | 母校事务局（中国 / 日本）|

#### 注意事项
- 在留期限到期前 3 ヶ月开始申请最稳；2 ヶ月内仓促 → 立证容易遗漏
- 转职后立即申更新 → 需附「契約機関に関する届出 控え」
- 業務内容説明書 必须明确属于「自然科学 / 人文科学 / 国際業務」三领域

#### 不要提交 / 容易误解
- ✗ 銀行残高証明書（技人国 不评估「資金能力」，无需提交）
- ✗ 履歴書（不是申请要件，仅補助）
- ✗ 戸口簿翻訳（中国国内文件，技人国 不要求）
- ✗ 父母同意書 / 親族関係 立证（与本人就业无关）

### 2. 経営・管理 — 在留期間更新 / 新規申請

#### 适用对象
- 持「経営・管理」在留資格 持有人 + 公司経営中
- 在留期間更新（延长 1 年 / 3 年 / 5 年）
- 新規申請：在留資格認定証明書 交付申請

#### 官方材料
- 在留期間更新許可申請書 / 在留資格認定証明書交付申請書
- 写真 1 枚（4cm × 3cm，3 ヶ月以内）
- パスポート + 在留カード
- 法人登記事項証明書
- 直近 1 期 法人税申告書（写し，含 别表 1 / 2 / 4）
- 法人決算書（含 貸借対照表 + 損益計算書 + 株主資本等変動計算書）
- 法人住民税 + 事業税 納税証明書
- 公司事業内容説明書（業務内容 + 取引先 + 売上見込）
- 事務所 賃貸契約書（法人名義）
- 役員報酬 給与支払明細書

#### 常见补充材料
- 商業登記簿謄本（履歴事項全部証明書）
- 株主名簿
- 直近 3 期 法人決算書（連続赤字評価时强化）
- 取引先一覽 + 直近契約書（实证立证）
- 事務所 看板 / 入口 / 内装 平面图 + 写真（事務所要件 立证）
- 雇用契約書 + 給与支払台帳（常勤職員 1 名以上 立证）
- 事業計画書（赤字 / 改正过渡期 立证用）

#### 哪里取得
| 材料 | 取得地 |
|---|---|
| 法人登記事項証明書 | 法務局 |
| 法人税 / 法人事業税 / 法人住民税 納税証明書 | 税務署 + 都道府県税事務所 + 市町村役場 |
| 法人決算書 + 法人税申告書 | 顧問税理士 / 法人内 |
| 商業登記簿謄本 | 法務局 |
| 雇用契約書 / 給与支払台帳 | 公司人事 |
| 賃貸契約書 + 事務所写真 | 大家 / 本人撮影 |

#### 注意事项
- 経営・管理 改正（2025/10/16）— 既存持有人有経過措置（截至 2028/10/16），但需补足 資本金 3000 万円 + 常勤職員 1 名以上
- 連続赤字 → 補足 事業計画書 + 役員借入金 補填 + 税理士意見書
- 役員報酬 与 法人決算 整合性 → 入管会比对

#### 不要提交 / 容易误解
- ✗ 個人 銀行残高証明書（看的是法人資本金，不是個人預金）
- ✗ 役員 学歴証明書（経営・管理 无学历要件，但 2025 改正后 経営学修士 等可加分）
- ✗ 配偶者書類（与本人 経営・管理 无关）
- ✗ 取引先 仅口头承诺（必须有書面契約 + 振込記録）

### 3. 家族滞在 — 在留期間更新 / 新規申請（本人持工作签 / 留学）

#### 适用对象
- 配偶者 / 子（実子 / 特別養子）of 工作签 / 留学 / 経営・管理 / 高度専門職 持有人
- 不含父母 / 兄弟姐妹（家族滞在 對象明確不含）

#### 官方材料
- 在留期間更新許可申請書 / 在留資格認定証明書交付申請書
- 写真 1 枚（4cm × 3cm，3 ヶ月以内）
- パスポート + 在留カード（本人 + 扶養者）
- 扶養者 在職証明書 / 在学証明書（写し）
- 扶養者 直近 1 年 課税証明書 + 納税証明書
- 扶養者 給与所得 源泉徴収票（写し）
- 婚姻関係立証（戸籍謄本 / 結婚証明書 + 公証 + 翻訳）/ 出生関係立証（戸籍 / 出生証明書 + 公証 + 翻訳）
- 質問書（指定様式，仅新規申請）

#### 常见补充材料
- 扶養者 雇用契約書（写し）
- 扶養者 公司決算書（経営・管理者 配偶 / 子 場合）
- 同居立证（住民票 + 賃貸契約書 + 公共料金支払）
- 結婚式 写真 / 同居生活 写真（仅新規 + 婚姻立证不足时）
- 婚姻続柄 経緯書（仅新規 + 跨国婚姻）
- 子供 出生証明書（中国 / 日本）+ 母子手帳

#### 哪里取得
| 材料 | 取得地 |
|---|---|
| 申請書様式 | 出入国在留管理庁 公式網站 |
| 扶養者 在職証明書 / 給与関連 | 扶養者 公司人事 |
| 課税 / 納税証明書 | 居住地市区町村 役所 |
| 戸籍謄本（中国侧）| 户籍所在地 公安局 / 大使館 認証 |
| 結婚証 / 出生証明書 翻訳 | 公証役場 + 中国驻日大使馆 認証 |
| 結婚式 / 同居 写真 | 本人持有 |

#### 注意事项
- 跨国婚姻 申請 → 中国侧戸籍 + 結婚証 + 出生証明書 必须 公証 + 中国驻日大使馆 認証 + 日本語翻訳（指定書式）
- 经营管理者 配偶 / 子 申請 家族滞在 → 重点立证扶養者経営実態
- 仅照片 + 同居记录 不构成婚姻立证 → 必须 戸籍 + 結婚証

#### 不要提交 / 容易误解
- ✗ 父母 / 兄弟姐妹 関連書類（不在 家族滞在 对象，硬递不许可）
- ✗ 本人就职用書類（家族滞在 不就労，仅資格外活動許可後可短時間打工）
- ✗ 仅 LINE 聊天记录 / SNS 互动（不是婚姻立证）
- ✗ 同居書類 重複（住民票 + 賃貸契約 + 公共料金 三件套即可，不需要更多）

### 4. 永住者 — 永住許可申請

#### 适用对象
- 在日 10 年以上（含工作签 5 年以上）
- 高度専門職 1 号（70 点 → 在日 3 年 / 80 点 → 1 年快速永住）
- 日本人配偶者等 / 永住者の配偶者等 / 定住者 → 一定期間後 永住

#### 官方材料
- 永住許可申請書（指定様式）
- 写真 1 枚（4cm × 3cm，3 ヶ月以内）
- パスポート + 在留カード
- 直近 5 年 課税証明書 + 納税証明書（住民税）
- 直近 5 年 公的年金 納付状況（ねんきんネット 打印 / 年金事務所「被保険者記録照会回答票」）
- 直近 2 年 公的医療保険 納付状況（健保組合 / 国保 完納証明書）
- 在職証明書（直近 1 年同公司在职 实务通说）
- 雇用契約書 + 直近 1 年 給与所得 源泉徴収票
- 公司案内 / 业务内容説明書
- 法人登記事項証明書（公司）
- 身元保証書 + 身元保証人 在職証明書 + 課税証明書 + 在留資格 関連書類（日本人 / 永住者）
- 永住理由書（A4 1-2 頁）
- 在留歴 + 渡航歴 一覧（自作）

#### 常见补充材料
- 高度専門職 ポイント計算表（1 号 / 80 点 路径）
- 直近 3 年 課税証明書（年収安定立证）
- 賃貸契約書 / 持家 登記事項証明書（生活定着立证）
- 子女 在学証明書 / 在園証明書（家庭定着立证）
- 預貯金通帳 残高証明書（推荐 200 万円 以上）

#### 哪里取得
| 材料 | 取得地 |
|---|---|
| 課税 / 納税証明書 | 居住地市区町村 役所（期间内换市町村 → 各市町村分别取）|
| 年金 納付状況 | ねんきんネット 在线 / 年金事務所 |
| 国民健康保険 完納証明書 | 居住地市区町村 国保年金課 |
| 在職 / 雇用契約 / 給与関連 | 公司人事 |
| 法人登記事項証明書 | 法務局 |
| 身元保証人関連書類 | 身元保証人 提供 |

#### 注意事项
- 永住法 2026/2/24 改正後「公的義務不履行」評価強化 → 直近 5 年 不得有 1 月滞納
- 直近 1 年同公司在職 = 实务通说，非法令明文，但通过率显著高
- 高度専門職 路径 ポイント計算 必须严格立证（年収 + 学历 + 経験 + 年齢 + 所属機関）
- 直近 5 年内 換過市町村 → 必须每个原市町村取証明（缺一不可）

#### 不要提交 / 容易误解
- ✗ 母国 戸口簿 / 父母関連書類（永住 看个人 + 在日生活定着，不需要母国家庭関連）
- ✗ 旅行記録 / 訪問日記（渡航歴 自作一覧即可，不需要详细记录）
- ✗ 銀行外貨送金記録（仅推荐 + 非要件）
- ✗ 帰化 用書類（永住 ≠ 帰化，不要混淆書類）

## 二、常见材料 10 类 — 取得 + 注意

### 1. 年金 納付状況

#### 哪里取得
- 在线：ねんきんネット（https://www.nenkin.go.jp/n_net/）
- 窗口：居住地管轄 年金事務所「被保険者記録照会回答票」申请

#### 注意事项
- 永住申请 → 取直近 2 年（厚生年金 + 国民年金 全期間）
- 帰化申请 → 取直近 5 年
- 跨年度国保 → 必须每年無滞納立证

### 2. 住民税 納付関連

#### 哪里取得
- 課税証明書：居住地市区町村 役所 課税課
- 納税証明書：居住地市区町村 役所 納税課
- 完納証明書：同上（清算后取）

#### 注意事项
- 永住申请 → 直近 5 年（每年 1 份 課税 + 納税 + 完納）
- 期间内换市町村 → 各原市町村分别取（缺一不可）
- 写し手数料 1 件約 300 円

### 3. 健康保険 関連

#### 哪里取得
- 国民健康保険：居住地市区町村 役所 国保年金課（完納証明書 / 加入記録）
- 社会保険（健保組合 / 協会けんぽ）：所属健保組合 / 公司健保担当 / 協会けんぽ支部

#### 注意事项
- 永住 → 直近 2 年完納立证
- 帰化 → 直近 5 年完納立证
- 公司不上社保 → 自己加 国保 + 経緯書

### 4. 住民票

#### 哪里取得
- 居住地市区町村 役所 戸籍住民課

#### 注意事项
- 申请永住 → 「世帯全員 + 続柄記載 + マイナンバー 含 / 不含」明确指定
- 跨自治体搬家时 → 旧住所 取「除票」+ 新住所 取「住民票」（マイナポータル 特例可豁免）
- 写し手数料 1 件約 300 円

### 5. 課税証明書

#### 哪里取得
- 居住地市区町村 役所 課税課

#### 注意事项
- 「市区町村民税 + 都道府県民税 課税証明書」是 standard 申請書類
- 必須年度：1/1 当時居住地で発行
- 期间内換市町村 → 各市町村分别取

### 6. 納税証明書

#### 哪里取得
- 国税（所得税 + 法人税）：税務署
- 住民税：居住地市区町村 役所
- 法人事業税 + 法人住民税：都道府県税事務所 + 市町村役場

#### 注意事项
- 国税 納税証明書 e-Tax 在线申请可
- 法人税 納税証明書 → 経営・管理 続签必备
- 写し手数料 国税 1 件 400 円 / 住民税約 300 円

### 7. 在職証明書

#### 哪里取得
- 公司人事 / 給与担当

#### 注意事项
- 含「就職日 + 役職 + 業務内容 + 給与」4 要素
- 公司印鑑必须（私印不算）
- 跨年度持续在职 → 多份就職時 + 直近 1 件即可

### 8. 雇用契約書

#### 哪里取得
- 公司人事

#### 注意事项
- 必须書面 + 双方署名
- 含「労働条件明示」4 要素（給与 / 勤務時間 / 業務内容 / 期間）
- 派遣 / 業務委託 → 不构成雇用契約（特定技能 / 経営・管理 等不接受）

### 9. 公司 法定調書 / 法人税申告書

#### 哪里取得
- 顧問税理士 / 法人内 経理担当
- 税務署 直近 1 期写し（如丢失）

#### 注意事项
- 経営・管理 续签 → 含 別表 1 / 2 / 4 + 決算書（B/S + P/L + S/E）
- 直近 3 期 連続赤字 → 補足 役員借入金 + 事業改善計画 + 税理士意見書
- 法人税申告書 + 決算書 必须整合性（数字一致）

### 10. 决算書

#### 哪里取得
- 顧問税理士 / 法人内 経理担当

#### 注意事项
- 含 貸借対照表（B/S）+ 損益計算書（P/L）+ 株主資本等変動計算書（S/E）
- 経営・管理 続签 → 直近 1 期主，直近 3 期補强（赤字时）
- 永住 持株会社 / 個人事業主 → 同时提交 個人 + 法人 决算
